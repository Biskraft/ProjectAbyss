## grep 검색 패턴 예시

Layer 4 실행 시 다음 패턴을 사용한다. Case-insensitive, `Reference/`·`game/`·`.git/`·`memory/` 제외.

```
# 스탯 폐기어 (독립 단어로 경계)
\bDEX\b|\bSTR\b|\bVIT\b|\bSPD\b|\bLCK\b|\bMP\b

# 구조 폐기어
\b[Hh]ub\b|재귀\s*진입|recursive\s*entry|3-Space

# 내러티브 폐기어 (맥락 확인 필요)
스승|멘토|가이드\s*NPC|튜토리얼\s*NPC|No\.1|넘버\s*원|대장장이

# 무기 폐기어
도끼|Axe|Spear|Whip|Wand(?!erer)|Bow(?!string)

# 원소 폐기어
\b풍\s*원소|\b광\s*원소|\bWind\s*element|\bLight\s*element

# 시스템 폐기어
스킬\s*트리|이노센트\s*팜|패링|자동사냥|거래소|시즌\s*리셋|배틀패스|가챠
```

## 검색 제외 대상

- `Documents/Terms/Glossary.md` (폐기 용어 정의 문서)
- `Documents/Terms/GDD_Writing_Rules.md` (작성 규칙 문서 내 예시 포함 가능)
- `.claude/skills/gdd-integrity-checker/references/deprecated_terms.md` (본 파일)
- `Reference/` (외부 참고 자료)
- `game/` (코드)
- `.git/`, `memory/`, `node_modules/`, `dist/`

## 동기화 규칙

새로운 폐기 용어가 추가되면:

1. `Documents/Terms/Glossary.md`에 `~~용어~~ DEPRECATED` 형식으로 먼저 등록
2. 이 파일 해당 섹션에 행 추가
3. 대응 DEC 번호 또는 메모리 파일 링크 기재
4. Layer 4 재실행으로 잔존 사용처 탐지
