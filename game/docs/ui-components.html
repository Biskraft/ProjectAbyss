UI COMPONENT CATALOG

이 페이지는 ECHORIS 게임의 모든 UI 컴포넌트와 디자인 토큰의 단일 진실 공급원(SSoT)이다.
소스 파일: game/src/ui/ · 토큰 정의: ModalPanel.ts

사용 규칙
① 새 UI 작업은 반드시 이 페이지의 컴포넌트 목록을 참고한다.
② 목록에 없는 컴포넌트/토큰은 사용 금지. 필요하면 먼저 src/ui/ 와 이 페이지에 추가하라.
③ 색상·폰트 사이즈·여백은 모두 ModalPanel.ts 의 토큰만 사용. 하드코딩 금지.
④ 새 색/사이즈가 정말 필요하면 ModalPanel.ts 에 토큰을 추가한 뒤 이 페이지의 토큰 섹션도 동기화한다.

Selection State FOUNDATION

SSoT: ROW_SELECTED / SLOT_SELECTED tokens · 모든 리스트·메뉴 컴포넌트가 이 패턴을 따른다

플레이어가 "지금 어디에 커서가 있는가" 를 한 번에 인식하지 못하면 메뉴 UI는 실패한다. Bloodstained: Ritual of the Night 의 메뉴를 기준으로, 선택 상태는 다음 4 레이어를 동시에 적용한다. 어느 한 레이어도 단독으로는 충분하지 않다.

Key Color · Orange (#ff9933)
청록 액센트는 다크 네이비 모달 패널에서 묻힌다. 선택 상태는 프로젝트의 워밍 톤(주황 단조 팔레트)과 일치하는 오렌지 키컬러로 통일한다. 장착 상태(#ff8c00 bar)와 색상이 겹칠 때는 펄스 + 챠브론 + [E] 배지로 구분한다.

4-Layer Recipe

① Saturated Fill — 비선택과 채도 차이가 충분한 배경 (단색 아님, 좌→우 그라디언트로 깊이 부여)
② Full-Alpha Edge — 2px 액센트 보더, alpha 0.95+ (희미한 outline 금지)
③ Symmetric Chevrons — ▶ (좌) + ◀ (우) 양쪽에 액센트 색상 표시
④ Outer Halo + Pulse — 외부 box-shadow 글로우 + 1.6s 펄스 애니메이션 (정적 UI에서도 살아있는 느낌)

SELECTED — 4-layer 적용
▶ ⚔ Iron Sword ATK 12
▶ ⚔ Steel Blade ATK 28
▶ ⚔ Frost Edge ATK 34

Anti-Pattern (이전 버전, 사용 금지)

다음 예시는 실제 플레이테스트에서 "선택이 명확하지 않다" 피드백을 받은 구버전이다. 단일 outline 35% alpha + ▶ 1개로는 화면 활동량이 많을 때 묻혀버린다.

DEPRECATED — 단일 outline 35% alpha
▶ ⚔ Iron Sword ATK 12
▶ ⚔ Steel Blade ATK 28
▶ ⚔ Frost Edge ATK 34

State Combinatorics

Selected는 모든 다른 상태(Equipped, On Anvil, Locked, Cleared)와 동시에 표현 가능해야 한다. 배경/보더는 selection 이 잡고, 다른 상태는 좌측 바·우측 뱃지·텍스트 색으로 분리한다.

▶ ⚔ Selected + Equipped ATK 28 E
▶ ⚔ Selected + Dive Ready ATK 34 DIVE
▶ ⚔ Selected + Cleared ATK 41 CLR
▶ ⚔ Selected + Locked ATK 12 🔒

Slot (List Row) ATOM

ModalPanel.ts · ROW_* tokens

인벤토리 리스트 행. 4가지 상태를 명확히 구분한다.
Normal: 어두운 배경, 흐린 텍스트. Selected: 밝은 배경 + 청록 ▶ 커서 + 흰색 아웃라인 + 이름 glow. Equipped: 주황 좌측 바 3px + [E] 뱃지. Equipped+Selected: 양쪽 모두 적용.

  ⚔    Normal — dim text, dark bg ATK 10
▶ ⚔    Selected — bright text, glow, outline ATK 10
  ⚔ E Equipped — orange bar + [E] ATK 10
▶ ⚔ E Equipped + Selected ATK 10

구분 포인트

① Normal→Selected: 배경 밝아짐 + ▶ 청록 커서 + 흰 아웃라인 2px + 이름 glow
② Normal→Equipped: 주황 좌측 바 3px + E 뱃지
③ 색상만이 아닌 형태 차이: 커서(▶), 바(좌측 3px), 뱃지(E), 아웃라인 — 색맹 안전

설계 규칙

① 배경 순수 검정: 게임 아트 미로드 상태. 타이틀만으로 분위기 전달

③ 기존 세이브 존재 시: 프리셋 선택 스킵 (저장된 바인딩 자동 로드) → 바로 게임 시작
④ 입력: ←→ 또는 A/D로 프리셋 탐색 + C 또는 Enter로 확인 (아직 어떤 키가 확인인지 모르므로 Enter도 허용)
⑤ 재접근: Pause > Options > Controls에서 언제든 변경 가능
