## 2. HP Bar 상세

### 2.1. 구조

```
[테두리 1px #444444]
[배경 #222222                                              ]
[HP 채움 ████████████████████████████░░░░░░░░░░░░░░░░░░░░░]
                                     ↑
                              현재 HP 위치
```

- **너비:** 80px
- **높이:** 6px (기존 5 → 6)
- **테두리:** 1px 어두운 회색 (#444444)
- **배경:** #222222
- **채움 색:**

| HP 비율 | 색상 | 의미 |
|:---|:---|:---|
| 50% 이상 | #22AA22 (녹색) | 안전 |
| 25-50% | #AAAA22 (노랑) | 주의 |
| 25% 미만 | #AA2222 (빨강) | 위험 |

### 2.2. HP 변화 연출

#### 데미지

```
[████████████████████████████████████░░░░]  Before
[████████████████████░░░░░░░░░░░░░░░░░░░]  After (즉시 줄어듦)
                     ^^^^^^^^^^^^^^^^
                     빨간 잔상 (200ms 동안 서서히 줄어듦)
```

- 피격 시 HP 즉시 감소
- **빨간 잔상 바:** 감소 전 위치에서 200ms 동안 서서히 줄어듦
- 참고: Dead Cells/Hollow Knight 스타일의 "잔상 HP"

#### 회복 (Flask / 전투 연동 / 보스 클리어)

```
[████████████████████░░░░░░░░░░░░░░░░░░░]  Before
[████████████████████████████████░░░░░░░░]  After
                     ^^^^^^^^^^^^
                     밝은 녹색 채움 (300ms flash)
```

- 회복 시 HP bar 증가 부분이 **밝은 녹색(#44FF44)으로 300ms 플래시** 후 정상 색으로 전환
- 회복량이 클수록 flash 범위가 넓어 시각적으로 "많이 회복됐다" 체감

## 3. Flask 아이콘

### 3.1. 표시

HP bar 바로 아래에 가로로 나열. 충전된 Flask = 채워진 원, 사용한 Flask = 빈 원.

```
● ● ● ○ ○     ← 충전 3개, 사용 2개 (최대 5개)
```

| 상태 | 아이콘 | 색상 | 크기 |
|:---|:---|:---|:---|
| 충전됨 | 채워진 원 ● | #FF8833 (주황, 단조 불꽃) | 6x6 px |
| 사용됨 | 빈 원 ○ | #444444 (어두운 회색) | 6x6 px |

- 간격: 2px
- 최대 표시: 8개 (기본 3 + 확장 최대 5)

### 3.2. Flask 사용 연출

Flask 사용 시 (0.6초 취약 시간):

1. **아이콘 소모 애니메이션:** 마지막 충전 아이콘이 200ms 동안 흰색 flash → 빈 원으로 전환
2. **HP bar 회복 연출:** §2.2 회복 플래시 재생
3. **캐릭터 주변:** 주황 단조 불꽃 파티클 (Phase 0: 주황 사각형 파티클)
4. **피격으로 취소 시:** 아이콘 미소모 (원상 복구). HP 변화 없음

### 3.3. 아이템계 진입 시 충전 연출

아이템계 진입 시 Flask가 레어리티별 충전됨:
- 빈 원들이 왼쪽부터 순서대로 300ms 간격으로 채워진 원으로 전환 (팝 효과)
- 충전 사운드 (Phase 0: 없음)

## 8. 구현 가이드

### 8.1. HUD.ts 변경 사항

| 항목 | 현재 | 변경 |
|:---|:---|:---|
| HP bar 크기 | 50x5 | **80x6** |
| HP bar 위치 | (4, 50) | **(4, 4)** — 좌상단으로 이동 |
| Flask 표시 | 없음 | **추가: HP bar 아래 6x6 원형 아이콘열** |
| Gold 위치 | (4, 4) 좌상단 | **(우상단)으로 이동** |
| Floor text 위치 | (4, 62) | **(4, 350) 좌하단으로 이동** |
| 잔상 바 | 없음 | **추가: 데미지 시 빨간 잔상** |
| 회복 플래시 | 없음 | **추가: 힐 시 녹색 플래시** |

### 8.2. 신규 클래스/메서드

| 항목 | 설명 |
|:---|:---|
| `HUD.updateFlask(current, max)` | Flask 아이콘 갱신 |
| `HUD.flashHeal(amount)` | 회복 시 녹색 플래시 트리거 |
| `HUD.flashDamage(amount)` | 데미지 시 빨간 잔상 트리거 |
| `HUD.flashBossHeal()` | 보스 클리어 회복 금색 플래시 |

### 8.3. 데이터 소스

```typescript
// Player or Game에서 HUD로 전달
hud.updateHP(player.hp, player.maxHp);
hud.updateFlask(player.flaskCharges, player.flaskMax);
hud.updateGold(game.gold);
```

`player.flaskCharges`와 `player.flaskMax`는 `System_Healing_Recovery.md` 구현 시 Player에 추가.
