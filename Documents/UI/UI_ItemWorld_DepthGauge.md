# 아이템계 깊이 표시기 (Item World Depth Gauge)

> **최근 업데이트:** 2026-04-14
> **문서 상태:** Draft
> **관련:** `UI/UI_Minimap.md` (아이템계 미니맵 없음 — Spelunky 모델)
> **기둥:** 탐험

---

## 0. 목적

아이템계에는 미니맵이 없다 (Spelunky 모델). 대신 **현재 지층 깊이**를 항상 보여주는 최소 표시기가 필요하다. "아래로 갈수록 심연으로 간다"는 감각을 수직 게이지로 전달한다.

---

## 1. 사양

### 1.1. 구조

```
■ 1     ← 클리어 (밝은 주황)
|
◆ 2     ← 현재 (흰색 테두리 + pulse)
|
□ 3     ← 미도달 (어두운 회색)
```

### 1.2. 배치

| 항목 | 값 |
|:---|:---|
| 위치 | 좌측 (4, 24) — HP bar + Flask 아래 |
| 방향 | 수직. 위 = 얕은 층, 아래 = 깊은 층 |
| 블록 크기 | 8 x 8 px |
| 숫자 | 블록 오른쪽 2px 간격, 8px 폰트 |
| 연결선 | 블록 사이 수직 1px (#333333), 길이 4px |
| 블록 간 총 간격 | 12px (블록 8 + 연결선 4) |

### 1.3. 블록 상태

| 상태 | 블록 시각 | 숫자 색 |
|:---|:---|:---|
| **클리어** | 채워진 ■, 깊이별 그라데이션 | 동일 색 |
| **현재** | 채워진 + 흰색 테두리 1px + 0.8초 pulse | 흰색 (#FFFFFF) |
| **미도달** | 빈 □ 테두리만 | 어두운 회색 (#333333) |

### 1.4. 깊이별 색 그라데이션

위(얕음)에서 아래(깊음)로 갈수록 주황 → 붉은 → 검붉은:

| 깊이 비율 | 색상 | 의미 |
|:---|:---|:---|
| 0% (최상단) | #FF8833 | 밝은 주황 (표면) |
| 33% | #CC6622 | 어두운 주황 |
| 66% | #993311 | 붉은 |
| 100% (최하단) | #661100 | 검붉은 (심연) |

클리어 블록과 현재 블록 모두 이 그라데이션 적용. 미도달 블록은 항상 #333333.

## 2. 레어리티별 표시

| 레어리티 | 지층 수 | 표시 |
|:---|:---:|:---|
| Normal | 2 | `■1  ◆2` |
| Magic | 3 | `■1  ◆2  □3` |
| Rare | 3 | `■1  ■2  ◆3` |
| Legendary | 4 | `■1  ■2  ◆3  □4` |
| Ancient | 4 | `■1  ■2  ■3  ◆4` |

---

## 3. 연출

### 3.1. 지층 전환 시

이전 블록이 "미도달 → 클리어"로 전환:
- 블록이 #333333에서 해당 깊이 색으로 300ms 동안 채워짐 (fill 애니메이션)
- 동시에 현재 블록 테두리가 다음 블록으로 이동

### 3.2. 현재 블록 pulse

- 주기: 0.8초
- 밝기: 0.7 - 1.0 진동
- 흰색 테두리만 pulse (내부 색은 고정)

### 3.3. 재사이클 시

재사이클(Cycle 2+) 진입 시:
- 모든 블록이 미도달(□)로 리셋
- 첫 블록부터 다시 시작

---

## 4. 좌하단 텍스트

지층 번호는 게이지가 담당. 텍스트에서 **제거**.

기존: `"S1 First Sword Lv5 EXP:120/300"`
변경: `"First Sword Lv5 EXP:180/300"`

재사이클 시: `"C2 First Sword Lv5 EXP:180/300"` (Cycle만 표시)
