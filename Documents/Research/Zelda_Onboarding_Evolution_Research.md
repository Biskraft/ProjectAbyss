## 1. 개별 작품 분석

### 1.1 Ocarina of Time (1998)

#### 첫 5-10분 Beat-by-Beat

| 시간 | 플레이어 행동 | 학습 내용 |
|:-----|:-------------|:----------|
| 0:00 | 컷씬 -- 링크의 악몽, 나비가 깨움 | 세계관 도입, 목적 제시 (데쿠 트리로 가라) |
| 0:30 | 집 밖으로 나옴 | 3D 공간 이동, 카메라 조작 |
| 1:00 | 코키리 마을 자유 탐색 | NPC 대화, 점프(자동), 장애물 인지 |
| 2:00 | 미도가 길을 막음 -- "검과 방패가 필요하다" | 명확한 Gate 제시 (아이템 2개 = 진행 조건) |
| 3:00 | 코키리 검 탐색 -- 구멍 통과, 굴러오는 바위 회피 | 기어가기, 환경 위험 인지, 보물상자 인터랙션 |
| 5:00 | 루피 수집 + 데쿠 방패 구매 | 경제 시스템, 상점 인터랙션 |
| 7:00 | 미도 통과, 데쿠 트리 도착 | Gate 돌파의 쾌감 |
| 8:00 | 데쿠 트리 내부 진입 -- 첫 던전 | 전투(데쿠 바바), 타겟팅(Z-Targeting), 퍼즐 |

#### 온보딩 기법

- **안전한 Sandbox:** 코키리 마을은 적이 없는 폐쇄 공간. 3D 이동을 안전하게 연습
- **NPC Gating:** 미도가 물리적으로 길을 막아 "검+방패" 획득을 강제. 목표가 명확
- **환경 교습:** 코키리 검 획득 경로에 기어가기/바위 회피가 자연스럽게 배치
- **Training Center:** 마을 내 연습장에서 돌/풀 베기로 검술 연습 가능 (선택적)
- **Navi의 힌트:** Z-Targeting, 인터랙션 대상을 말풍선으로 안내. 초기 3D 게임의 필수 보조

#### 핵심 타이밍

- **첫 전투:** 약 8분 (데쿠 트리 내부 데쿠 바바)
- **첫 실질적 에이전시:** 약 2분 (코키리 마을 자유 탐색)
- **첫 던전:** 약 8분 (Inside the Deku Tree)

### 1.2 Majora's Mask (2000)

#### 첫 5-10분 Beat-by-Beat

| 시간 | 플레이어 행동 | 학습 내용 |
|:-----|:-------------|:----------|
| 0:00 | 컷씬 -- 숲 속에서 스컬 키드에게 습격당함 | 전작 지식 전제, 위기 상황 즉시 제시 |
| 1:00 | 데쿠 링크로 변신, 시계탑 내부 | 변신 시스템 도입, 제한된 능력 |
| 2:00 | 시계탑 밖으로 나옴 -- 클럭 타운 진입 | 72시간 타이머 인지, 긴박감 |
| 3:00 | 요정의 안내로 대요정의 동굴 방문 | 길 잃은 요정 수집 + 버블 마법 획득 |
| 5:00 | 봄버즈 숨바꼭질 미니게임 | 데쿠 능력(버블 슛) 실전 활용 |
| 7:00 | 천문대 방문, 스컬 키드 관측 | 망원경 사용, 달 추락 위협 시각화 |
| ~60:00 | 3일차 자정 -- 오카리나 회수, 시간의 노래 연주 | 핵심 메커닉(시간 역행) 체득 |

#### 온보딩 기법

- **전작 지식 레버리지:** OoT 플레이어를 전제. 기본 조작 재교습 생략
- **능력 제한을 통한 교습:** 데쿠 링크는 검 사용 불가 -- 버블/꽃 활용을 강제로 학습
- **Time Pressure Tutorial:** 72시간 제한 자체가 긴장감 있는 학습 동기. "배우지 않으면 달이 떨어진다"
- **미니게임을 통한 실전 연습:** 봄버즈 숨바꼭질로 맵 탐색 + 데쿠 능력 자연스럽게 숙달
- **Diegetic Teaching:** 가면 상인이 게임의 목표를 직접 설명 (가면 회수 + 3일 시한)

#### 핵심 타이밍

- **첫 전투:** 약 3분 (클럭 타운 내 키스 박쥐, 제한적)
- **첫 실질적 에이전시:** 약 2분 (클럭 타운 자유 탐색, 단 데쿠 형태로 제한)
- **첫 던전:** 약 90-120분 (Woodfall Temple -- 3일 사이클 완료 + 늪지 도달 후)

### 1.3 The Wind Waker (2002)

#### 첫 5-10분 Beat-by-Beat

| 시간 | 플레이어 행동 | 학습 내용 |
|:-----|:-------------|:----------|
| 0:00 | 프롤로그 -- 고대 영웅 전설 tapestry 애니메이션 | 세계관 도입 (셀다 전설의 반복 구조) |
| 1:30 | 아릴이 링크를 깨움 (생일) | 캐릭터 관계, 일상의 평화 |
| 2:00 | 망루에서 내려와 마을 탐색 | 이동, NPC 대화, 아웃셋 섬 지형 학습 |
| 3:00 | 할머니에게 영웅의 옷 수령 | 목적 부여 (성인식 전통) |
| 4:00 | 아릴에게 망원경 수령 + 하늘 관측 | 망원경 사용법, 거대 새(Helmaroc King) 목격 |
| 5:00 | 해적 테트라가 새에게 잡혀 숲에 떨어짐 | 서사적 동기 부여, 긴급 상황 |
| 7:00 | 오르카에게 검술 교습 | **명시적 전투 튜토리얼** -- 횡베기, 찌르기, 회전베기, 점프 공격 |
| 9:00 | 숲에서 테트라 구출 + 첫 실전 전투 (보코블린) | 학습한 검술의 즉각 적용 |
| 10:00 | 아릴이 새에게 납치됨 | 강력한 서사적 동기 -- 여동생 구출 |

#### 온보딩 기법

- **서사 통합형 튜토리얼:** 생일 → 망원경 → 납치 사건이 자연스러운 이야기 흐름 속에서 조작을 교습
- **명시적 검술 교습 (Orca):** NPC가 직접 1대1로 검술을 가르침. 구조화된 전투 튜토리얼이지만 서사적 맥락(위기 상황) 내에 배치
- **탐색 유도:** 아웃셋 섬은 작지만 돼지 잡기, 루피 수집 등 선택적 활동이 풍부
- **Emotional Hook:** 아릴 납치가 10분 내에 발생하여 강한 감정적 동기 부여

#### 핵심 타이밍

- **첫 전투:** 약 9분 (숲 속 보코블린)
- **첫 실질적 에이전시:** 약 2분 (아웃셋 섬 자유 탐색)
- **첫 던전:** 약 60-90분 (Forsaken Fortress)

### 1.4 Twilight Princess (2006)

#### 첫 5-10분 Beat-by-Beat

| 시간 | 플레이어 행동 | 학습 내용 |
|:-----|:-------------|:----------|
| 0:00 | 루슬과 대화 -- 목가적 일상 | 세계관, 캐릭터 관계 |
| 2:00 | 에포나 타고 마을 이동 | 말 조작 기초 |
| 3:00 | 염소 몰이 미니게임 (1일차 10마리) | 에포나 조종 숙달, 방향 전환 |
| 5:00 | 마을 아이들과 대화, 심부름 | NPC 퀘스트 구조 학습 |
| 8:00 | 낚시 튜토리얼 | 새총 획득을 위한 전제 조건 |
| 15:00 | 새총 획득 + 사격 연습 | 조준 시스템 |
| 20:00 | 원숭이 추적, 숲 탐색 | 환경 퍼즐, 적 조우 |
| 45:00 | 콜린 구출 이벤트 | 말 위 전투 |
| ~120:00 | 황혼 진입, 늑대 링크 변신 | 변신 시스템, 센서 능력 |
| ~180:00 | Forest Temple 도달 | 첫 정식 던전 |

#### 온보딩 기법

- **확장된 일상 시뮬레이션:** 미야모토의 지시로 1일 → 3일로 연장. 염소 몰이, 낚시, 돼지 구출 등
- **Gated Item Chain:** 새총 → 원숭이 → 부메랑 순서로 아이템 획득이 연쇄적으로 게이팅
- **과도한 NPC 설명:** 거의 모든 메커닉에 대해 NPC가 장문의 설명을 제공
- **Wii 리모컨 적응:** 모션 컨트롤이라는 새로운 입력 방식 때문에 긴 튜토리얼이 정당화(미야모토 판단)

#### 핵심 타이밍

- **첫 전투:** 약 20분 (숲 속 적 조우, 새총 획득 후)
- **첫 실질적 에이전시:** 약 5분 (오르돈 마을 탐색, 단 심부름 강제)
- **첫 던전:** 약 180분 (Forest Temple -- 시리즈 최장 튜토리얼)

### 1.5 Skyward Sword (2011)

#### 첫 5-10분 Beat-by-Beat

| 시간 | 플레이어 행동 | 학습 내용 |
|:-----|:-------------|:----------|
| 0:00 | 컷씬 -- 여신과 봉인전쟁 프롤로그 | 세계관 배경 |
| 2:00 | 젤다가 편지로 링크를 깨움 | 캐릭터 관계 확립 |
| 3:00 | 기사학교 내부 탐색, NPC 대화 | 이동, 대화, 세계 소개 |
| 5:00 | 젤다와 만남, 로프트윙 비행 레이스 준비 | 목표 제시 |
| 8:00 | 로프트윙 찾기 퀘스트 | 스카이로프트 탐색, 기어오르기, 밧줄 매달리기 등 |
| 15:00 | 비행 레이스 (Wing Ceremony) | 로프트윙 조종 + 모션 컨트롤 튜토리얼 |
| 30:00 | 젤다 추락 사건, 여신의 검 획득 | 서사적 전환점 |
| 60:00 | 구름 아래로 강하, 파론 숲 도착 | 지상 세계 첫 진입 |
| ~120:00 | Skyview Temple 도달 | 첫 던전 |

#### 온보딩 기법

- **Fi의 과잉 안내:** 동반자 Fi가 거의 모든 상호작용에서 팝업 설명을 제공. 시리즈 최고 수준의 Hand-holding
- **아이템 설명 반복:** 매 세션 시작 시 아이템 획득 메시지가 재출력되는 설계 결함
- **모션 컨트롤 중심:** Wii MotionPlus 때문에 검 조작 자체가 학습 곡선. 비행/수영도 모션
- **선형 구조:** 스카이로프트 → 파론 숲 → 던전. 거의 일직선 진행
- **HD 리마스터 개선:** 2021년 HD판에서 Fi 개입을 선택적으로 변경, 아이템 메시지 1회 제한

#### 핵심 타이밍

- **첫 전투:** 약 60분 (파론 숲 보코블린 -- 스카이로프트에는 적이 없음)
- **첫 실질적 에이전시:** 약 3분 (스카이로프트 탐색, 단 퀘스트 목표 고정)
- **첫 던전:** 약 120분 (Skyview Temple)

### 1.6 A Link Between Worlds (2013)

#### 첫 5-10분 Beat-by-Beat

| 시간 | 플레이어 행동 | 학습 내용 |
|:-----|:-------------|:----------|
| 0:00 | 대장장이에게 깨어남, 검 배달 심부름 | 세계 소개, 이동 |
| 1:00 | 교회까지 검 배달 | 하이랄 필드 탐색, 탑다운 2D 이동 감각 |
| 2:00 | 유가(Yuga)와 조우, 세레스 납치 목격 | 위기 상황, 벽화 변환 능력 시각적 도입 |
| 3:00 | 유가에게 패배, 벽화 상태로 변환 | 벽화 능력 자연스러운 학습 (플레이어가 직접 경험) |
| 5:00 | 라비오를 만남, 팔찌 수령 | 벽화 능력 정식 해금 |
| 7:00 | 하이럴 성 방문, 임파와 대화 | 서사 진행, 세계 상황 파악 |
| 10:00 | 라비오의 대여점 오픈 -- 활 무료 대여 | **아이템 대여 시스템** 도입 -- 핵심 혁신 |
| 15:00 | Eastern Palace 진입 | 첫 던전 |

#### 온보딩 기법

- **시리즈 관습 전복:** 아이템이 던전에서 획득되는 것이 아니라, 게임 초반에 대여/구매로 전부 접근 가능
- **비선형 던전 순서:** 아이템 대여 시스템 덕분에 던전 순서가 거의 자유. 온보딩이 자유도로 직결
- **빠른 진행:** 튜토리얼이 매우 간결. 15분 내에 첫 던전 진입 가능
- **벽화 능력 체험형 학습:** 유가에게 패배하여 벽화가 되는 것 자체가 능력 교습
- **사망 패널티 = 학습:** 대여 아이템은 사망 시 반납. 위험의 무게를 자연스럽게 체감

#### 핵심 타이밍

- **첫 전투:** 약 3분 (유가 조우 -- 스크립트 이벤트이나 전투 시스템 체험)
- **첫 실질적 에이전시:** 약 10분 (라비오 대여점 오픈 후 던전 선택)
- **첫 던전:** 약 15분 (Eastern Palace)

### 1.7 Breath of the Wild (2017)

#### 첫 5-10분 Beat-by-Beat

| 시간 | 플레이어 행동 | 학습 내용 |
|:-----|:-------------|:----------|
| 0:00 | 부활의 사당에서 눈을 뜸 | 최소한의 컷씬, 즉시 조작 가능 |
| 0:30 | 시카 슬레이트 획득 | 핵심 도구(메뉴/맵) 획득 |
| 1:00 | 사당 밖으로 나옴 -- **하이럴 파노라마** | 환경 스토리텔링의 극치. 세계 전체가 보이는 절벽 위 |
| 1:30 | 절벽을 내려감 | 달리기, 점프, 등반, 스태미나 시스템 |
| 2:00 | 노인과 대화 (선택적) | 최소한의 서사 힌트 |
| 3:00 | 나무 가지/도끼 주움, 첫 적(보코블린) 조우 | 무기 습득, 전투 기초 |
| 5:00 | 자유 탐색 -- 사냥, 요리, 환경 상호작용 | Chemistry Engine (불+풀=화재, 바위 굴리기 등) |
| 7:00 | 첫 번째 사당(Magnesis) 도달 | Rune 능력 획득, 퍼즐 |
| ~60:00 | 4개 사당 완료, 패러세일 획득 | Great Plateau 탈출 = 진정한 자유 |

#### 온보딩 기법

- **Open-Air Design:** Great Plateau는 울타리 없는 오픈 샌드박스. 4개 사당의 순서가 자유
- **환경이 교사:** 추위 → 요리/횃불 필요. 비 → 등반 불가. 번개 → 금속 장비 제거. 시스템이 행동을 가르침
- **"See It, Try It" 원칙:** 절벽 위에서 세계를 보여준 후 즉시 탐색하게 함. NPC 설명 최소화
- **사당 = 격리된 퍼즐 교실:** 각 Rune 능력을 1개 사당에서 집중 학습. 월드에서 응용
- **Chemistry Engine 발견:** 불, 물, 금속, 자석 등의 상호작용을 플레이어가 실험으로 발견
- **Triangle Rule (삼각형 법칙):** 지형을 삼각형으로 설계하여 시야를 자연스럽게 차단/유도

#### 핵심 타이밍

- **첫 전투:** 약 3분 (Great Plateau 보코블린 캠프, 선택적)
- **첫 실질적 에이전시:** 약 1분 (사당 밖으로 나온 즉시)
- **첫 던전:** 약 60분 (Great Plateau 탈출 후 Divine Beast 접근 가능, 단 4사당이 미니던전 역할)

## 5. 개발자 인사이트

### 5.1 CEDEC 2017: Triangle Rule (삼각형 법칙)

**발표자:** 후지바야시 히데마로(디렉터), 요네즈 마코토(TA)

BotW의 지형은 의도적으로 **삼각형(피라미드/원뿔)** 형태로 설계되었다. 이 법칙은 세 가지 스케일로 적용된다:

| 스케일 | 역할 | 예시 |
|:-------|:-----|:-----|
| 대형 삼각형 (산) | Landmark -- 원거리 시각 유도 | 데스 마운틴, 하이럴 성 |
| 중형 삼각형 (언덕) | View Blocker -- 뒤편을 가려 발견의 연쇄를 유도 | 필드의 언덕, 바위 |
| 소형 삼각형 (바위) | Tempo Change -- 플레이어 입력 패턴을 변화시킴 | 전투 중 엄폐물, 보물 은폐 |

**온보딩 관점에서의 핵심:** 중형 삼각형이 "시야 차단 → 이동 → 새로운 발견"의 연쇄를 만들어, 플레이어가 **자기 주도적으로 탐험을 학습**하게 만든다. 튜토리얼 텍스트 없이 "저 너머에 뭐가 있을까?"라는 호기심만으로 진행을 유도한다.

초기 설계에서는 시카 탑(Points)과 도로(Lines)로 플레이어를 유도했으나, 80%의 테스터가 선형 경로를 따르고 20%가 무작위로 배회하는 양극화가 발생했다. 삼각형 법칙을 적용한 후 모든 플레이어가 자연스럽게 핵심 지점에 도달하면서도 자유롭게 탐색하는 이상적인 분포를 달성했다.

> **출처:** [CEDEC 2017 BotW English Summary (Scribd)](https://www.scribd.com/document/361160173/Zelda-BotW-CEDEC-2017-English-Summary), [Matt Walker Translation (GitHub)](https://gist.github.com/idbrii/e39fe96279aa1670319bfa521d907399), [Nintendo Life 보도](https://www.nintendolife.com/news/2017/10/zelda_breath_of_the_wilds_ingenious_design_is_all_about_triangles_apparently)

### 5.2 GDC 2017: "Change and Constant -- Breaking Conventions"

**발표자:** 후지바야시 히데마로(디렉터), 타키자와 사토루(아트 디렉터), 도타 타쿠히로(테크 디렉터)

BotW 팀이 깨뜨린 젤다 관습 중 온보딩과 직접 관련된 항목:

1. **"던전에서 아이템을 얻는다" 파괴:** 모든 Rune 능력을 Great Plateau에서 최초 1시간 내에 제공. 이후 성장은 능력이 아닌 플레이어의 창의성
2. **"정해진 순서로 진행한다" 파괴:** 4개 사당 순서 자유, 이후 Divine Beast 순서 자유
3. **"Chemistry Engine" 도입:** 물리/화학 시뮬레이션이 교습의 핵심. "불이 풀을 태운다"는 물리 법칙을 알면 게임 규칙을 안다
4. **"Multiplicative Gameplay":** 단순한 능력들의 조합이 무한한 해법을 생성. 플레이어가 실험으로 학습

> **출처:** [GDC Vault](https://www.gdcvault.com/play/1024562/Change-and-Constant-Breaking-Conventions), [YouTube](https://www.youtube.com/watch?v=QyMsF31NdNc)

### 5.3 미야모토의 TP 튜토리얼 연장 지시

개발 후반, 미야모토 시게루는 Wii 리모컨의 모션 컨트롤에 플레이어가 충분히 적응하지 못할 것을 우려하여, **오르돈 마을 체류 기간을 1일에서 3일로 연장**하도록 지시했다. 이로 인해:

- 추가 캐릭터, 아이템, 지역, 대화가 급히 제작됨
- 염소 몰이 2회, 낚시, 돼지 구출, 새총 사격 등 미니게임이 누적
- 결과적으로 **약 3시간의 튜토리얼**이 되어 시리즈 최대 비판 포인트가 됨

이 경험이 BotW 개발진에게 "긴 튜토리얼은 해답이 아니다"는 교훈을 남겼고, Great Plateau의 "자유로운 학습 공간" 설계로 이어졌다.

> **출처:** [Engadget -- Miyamoto critical of TP](https://www.engadget.com/2008-10-29-miyamoto-slightly-critical-of-twilight-princess-super-mario-g.html), [WhatCulture](https://whatculture.com/gaming/10-things-you-never-knew-about-zelda-twilight-princess?page=2)

### 5.4 사쿠라이 마사히로의 튜토리얼 철학

사쿠라이는 튜토리얼 설계에 대해 다음 원칙을 제시한다:

1. **"가능하면 실전에 즉시 투입하여 싸우면서 배우게 하라"** -- 연습장보다 실전이 동기 부여와 감정 모두에서 우월
2. **"긴 문장을 설명서처럼 읽히지 마라"** -- 최악의 교습
3. **"능동적으로 놀게 하면서 배우게 하라"** -- 조작을 직접 시켜 기억하게 하는 것이 핵심
4. **이동계 스킬은 "좁은 곳에 가두고 스킬을 쓰지 않으면 진행 불가"가 일반적** -- ECHORIS의 능력 게이트와 정확히 일치

> **출처:** 사쿠라이 마사히로 유튜브, "Making Tutorials Feel Natural", "Teaching Players How to Play"

## 6. ECHORIS 적용점

### 6.1 ECHORIS의 온보딩 제약 조건

ECHORIS는 일반적인 젤다와 다른 고유한 제약을 갖는다:

| 제약 | 내용 | 젤다와의 차이 |
|:-----|:-----|:-------------|
| 대사 0줄 | 에르다는 BLAME! 킬리 오마주. 독백/대화 없음 | 젤다는 NPC 대화로 교습 가능 |
| 아이템계 첫 1분 | 스파이크(아이템 속 던전)를 최대한 빠르게 체험 | 젤다는 던전까지 8분-180분 |
| 웹 브라우저 | 로딩 시간/세션 시간 제약 | 콘솔 전용 타이틀 |
| 2D 횡스크롤 | 카메라 고정, 3D 공간 교습 불필요 | 3D 젤다의 카메라/타겟팅 교습 생략 가능 |
| 환경 스토리텔링 | 모든 서사가 환경으로만 전달 | 젤다도 BotW부터 환경 스토리텔링 강화 |

### 6.2 시대별 적용 교훈

#### 1기 (OoT/MM/WW)에서 배울 것: 안전한 학습 공간 + 명확한 Gate

- **코키리 마을의 "검+방패 Gate"** → ECHORIS의 "ATK 게이트" 직접 대응
  - OoT: 미도가 "검과 방패가 없으면 통과 불가" → 플레이어가 자연스럽게 탐색
  - ECHORIS: ATK 게이트 #1이 "이 수치 이하면 통과 불가" → 모루에서 강화 필요
  - 차이점: OoT는 NPC가 말로 알려줌. ECHORIS는 **시각적 피드백**만으로 전달해야 함
- **MM의 시간 압박** → 아이템계의 지층 제한과 유사. "배우지 않으면 실패한다"는 긴장감
- **WW의 감정 Hook** → ECHORIS에서는 대사 없이 환경으로 감정을 유발해야 함. 발코니에서 심연을 조망하는 순간이 아릴 납치에 대응

#### 2기 (TP/SS)에서 배울 것: 하지 말아야 할 것

- **TP의 3시간 튜토리얼** = 절대 금지. ECHORIS는 첫 1분에 아이템계 진입이 목표
- **SS의 Fi 과잉 개입** = UI 힌트 최소화의 반면교사. ECHORIS는 UI 힌트조차 최소
- **미야모토의 실수에서 배운 것:** "새로운 입력 방식에 대한 불안감"이 튜토리얼 팽창을 유발. ECHORIS는 웹 브라우저(키보드/마우스)라는 익숙한 입력이므로 이 함정을 피할 수 있음
- **반면교사 공식:** `튜토리얼 길이 ∝ 입력 방식의 낯설음` → ECHORIS는 WASD/Arrow + Z/X/C로 충분

#### 3기 (ALBW/BotW/TotK)에서 배울 것: 환경이 교사

- **BotW의 Great Plateau 모델** → ECHORIS 첫 20 Screen의 직접 레퍼런스
  - Great Plateau: 자유로운 울타리 공간 → 4개 능력 학습 → 패러세일로 탈출
  - ECHORIS: 발코니(조망) → 이동/점프/대시/전투(환경) → ATK Gate → 모루 → 아이템계 진입 → 강화 → 돌파
  - **차이점:** BotW는 1시간, ECHORIS는 **5분 이내**에 핵심 루프를 1회전시켜야 함

- **삼각형 법칙의 2D 변환:**
  - 3D에서 삼각형 = 시야 차단 + 호기심 유발
  - 2D 횡스크롤에서는 **수직 단차(Ledge)**가 동일 역할. "저 위/아래에 뭐가 있을까?"
  - 아이템계 지층에서 절차적 생성 시 수직 단차를 의도적으로 배치하여 탐험 유도

- **ALBW의 아이템 대여 시스템 = 아이템계 입구의 영감:**
  - ALBW: 아이템을 빌려서 자유롭게 던전 선택
  - ECHORIS: 아이템을 **획득하여 그 안에 들어감**. 아이템 자체가 콘텐츠

- **TotK의 Ultrahand/Fuse = 사당 교실 모델:**
  - 하나의 능력을 하나의 격리된 공간에서 집중 학습
  - ECHORIS 적용: 첫 아이템계 진입이 "모루 타격 → 강화"라는 단일 메커닉의 격리된 교실

### 6.3 ECHORIS 온보딩 설계 원칙 (젤다 분석에서 도출)

| 원칙 | 젤다 근거 | ECHORIS 적용 |
|:-----|:---------|:-------------|
| **1분 안에 Core Loop 1회전** | BotW 1시간 > TotK 90분이 이미 길다는 비판 | 발코니→이동→Gate→강화→돌파를 5분 내 |
| **환경이 유일한 교사** | BotW Chemistry Engine, 삼각형 법칙 | 대사 0줄이므로 지형/시각 피드백만 사용 |
| **Gate가 곧 교습** | OoT 미도, BotW Rune 사당 | ATK Gate 앞에서 "막힘" 경험 = 강화 필요성 자각 |
| **실패를 허용하라** | BotW 라이넬 조우, MM 달 추락 | 아이템계 사망 → 진행 손실. 위험의 무게가 학습 동기 |
| **Show, Don't Tell의 극단** | BotW 절벽 위 파노라마 | 발코니에서 심연(The Shaft) 조망 = 세계의 스케일 체감 |
| **격리된 교실 → 월드 응용** | TotK 사당, OoT Training Center | 아이템계 = 격리된 강화 교실, 월드 = 응용 공간 |
| **Hand-holding 금지** | TP/SS 반면교사 | UI 팝업, 동반자 설명, 강제 미니게임 전부 배제 |

### 6.4 젤다 온보딩 vs ECHORIS 첫 30분 매핑

```
젤다 BotW (Great Plateau)              ECHORIS (첫 20 Screen)
━━━━━━━━━━━━━━━━━━━━━━━━              ━━━━━━━━━━━━━━━━━━━━━━
부활의 사당에서 눈뜸        ←→         발코니에서 깨어남
절벽 위 하이럴 파노라마     ←→         심연(The Shaft) 조망
절벽 내려가기 (이동 학습)   ←→         이동/점프/대시 환경 교습
보코블린 캠프 (첫 전투)     ←→         첫 적 조우 + 전투
Magnesis 사당 (능력 학습)   ←→         ATK Gate #1 (못 뚫음 = 학습)
4개 사당 클리어             ←→         바닥의 검 → 모루 → 에코 타격
패러세일 획득 = 자유        ←→         아이템계 진입 → 보스 → 강화 → 돌파
                                       (= 스파이크 Core Loop 1회전)
```

**결정적 차이:** BotW는 Great Plateau에서 **능력을 모두 준 후** 자유를 주지만, ECHORIS는 **"막힘→강화→돌파" 루프 자체를 가르친 후** 반복을 유도한다. BotW의 교습 대상은 "능력(What)"이고, ECHORIS의 교습 대상은 "순환(How)"이다.
