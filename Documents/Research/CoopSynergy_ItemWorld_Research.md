## 3. 캐슬바니아 위키 기반 2D 횡스크롤 협동 메커닉

### 3.1. Portrait of Ruin -- 파트너 시스템 완성형

| 메커닉 | 상세 | ECHORIS 적용 |
|:---|:---|:---|
| **Dual Crush** | 두 캐릭터 MP 동시 소모 → 강력 합체 기술. 5초 쿨다운 | 파티원 인접 시 합동 스킬. 무기 조합별 다른 Dual Crush |
| **어깨 밟기 점프** (Acrobat Cube) | 파트너 어깨를 밟고 2단 점프 | 멀티 탐험 퍼즐. 솔로 불가능한 높이에 보상 배치 |
| **HP/MP 분리 소모** | 조작 캐릭터 = HP, AI 파트너 = MP | 멀티에서 탱커/딜러 자원 분리 구조의 원형 |
| **아이템 거래** (Shop Mode) | 플레이어 간 아이템 거래 | 아이템계 내 전리품 교환 |

### 3.2. Harmony of Despair -- 최대 6인 협동의 교훈

| 메커닉 | 상세 | ECHORIS 적용 |
|:---|:---|:---|
| **자유 탐험 + 합류 콤보** | 각자 성을 돌아다니되, 합류하면 Dual Crush 가능 | 아이템계에서 분산 파밍 vs 합류 전투 이중 인센티브 |
| **캐릭터 고유 시스템** | 10명 각각 다른 스킬 체계 | ECHORIS는 무기 카테고리로 대체 |
| **부활 (Revive)** | 죽은 플레이어를 다른 플레이어가 부활 | 접촉 부활 (채널링) |
| **퀵 메시지** | "Got it" / "Thanks" / "Here" 사전 정의 채팅 | 아이템계 간편 커뮤니케이션 |

## 4. 인원수별 시너지 스케일링

### 4.1. "더 많은 인원 = 새로운 가능성" 설계

| 인원 | 해금되는 시너지 | 근거 게임 |
|:---|:---|:---|
| **1인 (솔로)** | 기본 전투 + 콤보 체계 | 베이스라인 |
| **2인 (Phase 3 타겟)** | 원소 퓨전 2조합, 저주 부활, 콤보 가속, 기억 단편 공명 | Gunfire, Windblown |
| **3-4인 (Phase 4+)** | 3-4원소 퓨전, 공명 보너스 확장. 2인 구조에 인원만 추가하는 방식 | -- |

### 4.2. 적 스케일링

| 인원 | 적 HP 배율 | 적 공격력 | 엘리트 빈도 |
|:---|:---|:---|:---|
| 1인 | x1.0 | x1.0 | 기본 |
| 2인 (Phase 3) | x1.75 | x1.0 | +10% |
| 3-4인 (Phase 4+) | x2.5-3.25 | x1.1-1.2 | +20-30% |

> HP는 인원 비례 이상 증가시키되(x1.75, 2.5, 3.25), 시너지 메커닉(콤보 가속, 원소 퓨전)으로 상쇄하여 "시너지를 활용하면 솔로보다 효율적, 활용 안 하면 솔로보다 어려움"을 달성.

## 5. ECHORIS 아이템계 협동 시너지 프레임워크 제안

### 5.1. Tier 1 -- 자동 시너지 (특별한 조작 불필요)

| 시너지 | 메커닉 | 조건 |
|:---|:---|:---|
| **콤보 가속** | 파티원 간 교차 타격 시 콤보 배율 +15% (솔로 +10%) | 같은 적을 2명 이상이 타격 |
| **공명 보너스** | 파티원 무기 카테고리가 전부 다르면 전원 ATK +15% | 파티 구성 시 자동 판정 |
| **기억 단편 공명** | 파티원의 기억 단편 타입이 겹치면 해당 효과 +20% | 파티 구성 시 자동 판정 |

### 5.2. Tier 2 -- 근접 시너지 (가까이 있으면 발동)

| 시너지 | 메커닉 | 조건 |
|:---|:---|:---|
| **팀 어택** | 공격 시 32px 내 파티원이 자동으로 추가 타격 (75% 데미지) | 인접 + 공격 시 |
| **리벤지 게이지** | 파티원 피격 시 전원 게이지 50% 충전. 사망 시 5배 | 파티원 피격/사망 |
| **오라 버프** | 기억 단편가 32px 내 파티원에게 패시브 효과 방사 | 상시 (거리 내) |

### 5.3. Tier 3 -- 의도적 시너지 (특정 행동/조합 필요)

| 시너지 | 메커닉 | 조건 |
|:---|:---|:---|
| **원소 퓨전** | 서로 다른 원소 공격이 같은 적에게 적용되면 퓨전 효과 | 다른 원소 무기로 동일 적 타격 |

> **삭제된 항목 (2026-04-15 피드백):**
> - ~~Dual Crush~~ -- 무기 카테고리 미확정 상태에서 조합별 스킬 설계는 순서 역전. 무기 체계 확정 후 Phase 4+에서 재검토
> - ~~부스트 점프~~ -- 메트로베니아 능력 게이트 설계와 충돌. 솔로 플레이어가 접근 불가능한 영역이 생기면 안 됨
> - ~~부위 파괴 분담~~ -- 무기별 시너지와 동일 이유. 무기 체계 확정 후 검토

### 5.4. 부활 시스템 (Windblown + CV:HoD 혼합)

```
파티원 사망
    ↓
"부식 저주(Corrosion Curse)" 생존자 전원에게 부여
    │  (매 3초 HP 1% 감소)
    ↓
해제 방법 (택1):
    A. 적 5마리 처치 → 저주 해제 + 사망자 부활
    B. 사망 위치에서 3초 채널링 → 부활 (부활자 무방비)
    ↓
부활 시 HP 30%로 복귀. Flask 미소모.
전원 사망 = 아이템계 강제 퇴장 (진행 손실)
```

### 5.5. 루트 시스템

| 방식 | 내용 |
|:---|:---|
| **기본** | 개인 인스턴스 루트 (각자 별도 드롭 테이블) |
| **교환** | 아이템계 내에서 바닥에 놓아 교환 가능 |
| **파티 보너스** | 2인 시 추가 드롭 슬롯 +1 (Phase 4+에서 인원당 +1로 확장) |
| **퓨전 킬 보너스** | 원소 퓨전으로 처치 시 레어리티 1등급 상향 확률 +5% |
| **솔로 보호** | 솔로의 기본 드롭량은 변동 없음. 파티는 추가 레이어 |

## 6. 카메라 / 아군 피격 / 기타 횡스크롤 과제

### 6.1. 카메라

**권장:** 아이템계 방 단위 공유 카메라 + 줌아웃 적응형
- 방이 512x512px로 작아 극단적 분산 문제 적음
- 파티원이 방 경계에서 나뉠 경우: 먼저 이동한 플레이어 기준 전환

### 6.2. 아군 피격

**권장:** 아군 피격 OFF (기본)
- 아군 피격 ON은 원거리 무기 사용을 극도로 제한하여 역할 분화 저해
- Castle Crashers, Rayman Legends 등 성공적 협동 횡스크롤 대부분 OFF

### 6.3. 간편 커뮤니케이션

CV:HoD의 퀵 메시지 차용:
- 사전 정의 메시지 8종: "OK" / "Thanks" / "Help" / "Here" / "Go" / "Wait" / "Nice" / "Sorry"
- 방향키 + 특수 키 조합으로 발신. 캐릭터 머리 위 말풍선 표시

## 7. 스파이크 검증: "이 시너지가 아이템계 경험을 강화하는가?"

| 시너지 | 스파이크 강화 여부 | 근거 |
|:---|:---|:---|
| 콤보 가속 | **O** | 아이템계 전투가 더 빠르고 쾌감 있어짐 |
| 공명 보너스 | **O** | 파티 구성이 아이템 빌드의 일부가 됨 |
| 기억 단편 공명 | **O** | 기억 단편가 솔로뿐 아니라 파티에서도 가치 |
| 원소 퓨전 | **O** | "이 무기를 가져가면 퓨전이 되니까 같이 들어가자" → 아이템계 재진입 동기 |
| Dual Crush | **O** | 협동의 하이라이트 순간. 보스전 클라이맥스 |
| 부스트 점프 | **O** | 솔로에서 못 가는 비밀 영역 → 멀티 전용 탐험 보상 |
| 저주 부활 | **O** | "다음 지층 갈까?" 의사결정에 부활 가능 여부가 영향 |
| 파티 보너스 드롭 | **O** | "같이 가면 더 나오니까" → 파티 플레이 동기 |
