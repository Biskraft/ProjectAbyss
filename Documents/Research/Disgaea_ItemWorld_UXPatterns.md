## 1. 세션 흐름 분석

### 1.1 전형적 세션 구조

```
[진입 결정] → [10층 단위 전투] → [보스 격파] → [탈출 vs 계속 결정]
                     ↑                                        |
                     └────────────────────────────────────────┘
```

10층마다 찾아오는 탈출 포인트가 세션 설계의 심장부.

### 1.2 플레이어 리스크 계산 변수

| 변수 | 탈출 쪽으로 | 계속 쪽으로 |
|:-----|:-----------|:-----------|
| 파티 HP | 잔여 HP 낮음 | 잔여 HP 충분 |
| Mr. Gency 보유량 | 0개 (불안) | 2개+ (안전망 확보) |
| 현재 아이템 레벨 | 이미 충분히 올랐음 | 더 올리고 싶음 |
| 다음 보스 | 킹/신 (강함) | 장군 (약함) |
| 미스터리 룸 기대 | 없음 | 최근 없어서 기대 중 |

## 2. 리스크-리워드 긴장감

### 2.1 긴장감 곡선

```
층수      1  5  10  15  20  25  30  ...  100
                ↑       ↑               ↑
             장군     장군            신(God)
긴장감   ↗↗↗↘  ↗↗↗↘  ↗↗↗  ...
```

10층 단위 스프린트의 마지막 전투가 클라이맥스 → 자연스러운 긴장-이완 사이클.

### 2.2 Mr. Gency's Exit의 심리적 기능

- **보유 시:** "더 깊이 가도 괜찮아" → 모험 욕구 자극
- **미보유 시:** 보스층까지 반드시 도달해야 한다는 압박 → 부정적 긴장감
- **핵심:** 안전망의 희소성이 도전 욕구와 공포 사이의 긴장감 조절 레버

## 3. 세션 길이 패턴

### 3.1 시리즈 진화에서 읽는 세션 길이 교훈

| 버전 | 세션 구조 | 실제 시간 | 플레이어 반응 |
|:-----|:---------|:---------|:-------------|
| D1 | 100층 마라톤, 10층마다 탈출 | 30분~2시간 | "적당히 끊을 수 있음" |
| D5 | 무한 층, 끝점 없음 | 1시간~무제한 | **"끝없는 슬로그" 비판** |
| D6 | 무한 층 + 오토배틀 x32 | 방치 수십 분 | **"방치게임" 비판** |
| D7 | 30층 + 전생 반복 | 15~30분/사이클 | "짧고 명확해서 좋다" |

D5의 "무한 층"은 "언제 끝내야 하는지 모르겠다"는 구조적 불안을 야기. D7의 30층 복귀는 이에 대한 직접적 대응.

### 3.2 이상적 세션 길이 (SDT 관점)

- **자율성:** 10층마다 탈출 선택 가능
- **유능감:** 매 층 아이템 레벨 +1의 즉각적 피드백
- **연결감:** 원본에서는 솔로라 미충족 → **ECHORIS의 멀티플레이가 채울 핵심 기회**

## 4. QoL 개선 이력

### 4.1 버전별 QoL 타임라인

| 버전 | 추가 QoL | UX 인사이트 |
|:-----|:---------|:-----------|
| D1 | 10층마다 탈출, Mr. Gency | 기본 세션 구조 확립 |
| D2 | 미스터리 룸, 이노센트 타운, 해적 | 루프에 '의외성 레이어' 삽입 |
| D4 | 분기 루트 (성장/이노센트/보물) | 목적별 선택이 파밍 집중도 향상 |
| D5 | 보너스 게이지, 이노센트 팜, 무한 층 | **무한 = 세션 목표 소멸** |
| D6 | 오토배틀 x32, D.I. | **핵심 루프 대체 → 존재 이유 위기** |
| D7 | 30층 + 아이템 전생 | 명확한 루프가 리텐션에 유리 |

### 4.2 QoL vs 코어 루프 훼손 경계선

```
[생산적 QoL]                          [코어 파괴적 QoL]
배속 옵션(x2, x4) ───────────→ 오토배틀로 전투 자체 생략
미스터리 룸 (보상 다양성) ──→ 모든 층이 동일한 보너스 룸
탈출 포인트 확대 ───────────→ 언제든 탈출 가능 (리스크 소멸)
분기 루트 (목적별 선택) ──→ 모든 루트가 동등 (선택 무의미)
```

## 5. 멀티플레이어 아이템계: 디스가이아의 현실과 한계

### 5.1 현황

**디스가이아 본편 시리즈는 아이템계 협동 멀티플레이를 한 번도 정식 구현한 적이 없다.**

| 작품 | 멀티 내용 |
|:-----|:---------|
| D1~D5 | 완전 솔로 전용 |
| D4 PC | 비동기: 해적 팀이 타인의 아이템계에 침입 |
| D7 | 로컬 2인 협동 (아이템계 포함, 온라인 미지원) |
| Disgaea Mayhem (2026 예정) | 실시간 액션 + 아이템계 유지 |

### 5.2 동기적 멀티를 피한 이유 (추정)

1. 턴 기반 → 여러 플레이어 동시 행동 시 대기 시간 폭발
2. 파티 전멸 동기화 문제
3. 이노센트 귀속 문제
4. 장비 레벨 차이에 따른 적 레벨 스케일링

→ 이 문제들은 턴 기반 SRPG에서 발생. **ECHORIS는 실시간 액션이므로 다른 해결 가능.**

## 6. 플레이어 동기 패턴

### 6.1 Bartle 유형별

| Bartle 유형 | 아이템계 동기 | 핵심 보상 |
|:-----------|:-------------|:---------|
| 킬러 | 보스 처치 권력감 | 영구 스탯 증가 |
| 달성가 | 아이템 레벨 최대치 완성 | 레벨 MAX, 슬롯 완성 |
| 탐험가 | 미스터리 룸의 예측 불가능성 | 랜덤 발견 |
| 사교가 | **(원본에서 미충족)** | **ECHORIS에서 협동 클리어** |

### 6.2 "육성형" vs "교체형" 아이템 모델

- **Diablo/PoE (교체형):** 더 좋은 드롭 → 기존 것 버림. 빈번한 도파민, 낮은 애착
- **디스가이아 (육성형):** 동일 아이템에 수백 층 투자. 매몰 비용 → 감정적 애착

ECHORIS는 육성형 채택. 장기 리텐션의 핵심.

## 8. 실시간 액션 멀티플레이로의 전환

### 8.1 해결해야 할 구조적 문제

**문제 A: 층 클리어 조건 재정의**
- 4인이 함께할 때 "아무도 포탈에 안 닿은 상태에서 보스를 잡으면?" 등 엣지 케이스

**문제 B: 사망/탈출의 개인화 vs 파티화**
- 모델 1 (파티 공유): 1명이라도 살아있으면 계속. 전원 전멸 시 탈출 선택
- 모델 2 (개인 탈출): 개인 사망 → 대기/관전. 파티 지속

**문제 C: 이노센트 귀속**
- 옵션 1: 최후 일격자 귀속
- 옵션 2: 파티 리더(아이템 오너) 귀속 **(가장 자연스러움)**
- 옵션 3: 기여도 기반 분배

**문제 D: 파티 간 전투력 편차**
- 권장: 아이템 레어리티+층수 기반 적 레벨 (파티원 레벨과 독립)

### 8.2 멀티플레이 UX 기회

1. **이노센트 추격 협동:** 야생 이노센트에 도주 행동 → 4인 포위 추격전
2. **역할 분담 자연 발생:** 포위/방어/보너스 충전 분업
3. **미스터리 룸 파티 의사결정:** 진입 여부를 파티가 함께 결정
4. **파티 탈출 협상:** 보스 후 "계속 vs 탈출" 토론 → 소셜 인터랙션

## 9. ECHORIS를 위한 UX 설계 권장사항

### 9.1 세션 설계

1. **명확한 세션 종료 신호:** 각 레어리티별 층수 명시 + "세션 완료" 연출
2. **10층 단위 체크포인트 유지:** 20년간 검증된 최선의 설계
3. **Mr. Gency 멀티플레이 버전:** 개인 탈출 아이템. 리더 탈출 시 세션 종료

### 9.2 리스크-리워드

4. **멀티플레이 전용 리스크 레이어:** 공유 탈출 아이템으로 blame 방지
5. **안전망 가시화:** 세이브 포인트, 보존 보스 킬, Mr. Gency 잔량을 HUD 상시 표시

### 9.3 이노센트 멀티플레이

6. **귀속 규칙 즉시 표시:** 파티 리더 귀속 원칙 + 동행자 유사 보상
7. **이노센트 사냥 협동:** 도주 행동으로 팀 협력 필수화

### 9.4 QoL

8. **자동화는 보조에만:** 배속 O, 전투 AI 대체 X. "수동 정복 선행" 원칙 유지
9. **층 미리보기 정보:** 다음 층의 위험 수준 힌트 (적 유형, 환경 효과 존재 여부)
10. **세션 요약 화면:** 탈출 후 성과 표시 (이노센트 복종 수, 레벨 증가, 보스 처치)
