## 1. 게임 수직 이동 기믹 분석

### 1.1. 메트로베니아 — 고전적 수직 이동 기믹

#### 캐슬바니아: 월하의 야상곡 (SotN) — 수직 이동 기믹

**엘리베이터 (Outer Wall)**

Outer Wall의 엘리베이터는 SotN 수직 이동의 핵심 예시다.

| 항목 | 내용 |
|:-----|:-----|
| 구조 | 거대한 탑 전체를 수직 관통하는 금속 케이지 엘리베이터 |
| 경유지 | 하단(Marble Gallery) → 중간(Long Library) → 상단(Clock Tower) |
| 활성화 조건 | 탑 꼭대기 부근의 스위치를 먼저 수동으로 활성화해야 사용 가능 |
| 이동 방식 | 탑승 후 문 닫힘 → 빠른 속도로 상승/하강 (자유 낙하 수준의 속도) |
| 진입 방향 선택 | 위층/아래층 버튼으로 선택 가능 |

**설계 원칙 인사이트:**
- 스위치 활성화 필요: 엘리베이터 자체가 보상이다. 탑을 올라서 스위치를 찾아야 "앞으로 이 길이 빨라진다"는 인과적 쾌감 제공
- 3-point 연결: 단순 2점 연결이 아니라 3개 층을 연결해 선택지 다양성 부여
- 수직 축의 내러티브: 탑이라는 건축물이 엘리베이터의 존재를 자연스럽게 정당화

**순간이동 방 (Teleport Rooms)**

SotN은 성 전역에 분산된 순간이동 방을 제공한다.

| 특성 | 내용 |
|:-----|:-----|
| 분포 | 각 주요 구역 1~2개 배치 |
| 사용 조건 | 무조건 무료 사용 가능 (발견만 하면 됨) |
| 발견 방식 | 벽을 통과하거나 숨겨진 문을 찾아야 접근 가능한 경우 다수 |
| 이동 방식 | 진입 → 목적지 방 목록에서 선택 → 순간 전송 |

**설계 원칙 인사이트:**
- 숨겨진 접근 경로: 순간이동 방 자체가 탐험 보상. 찾은 것만 목록에 추가되어 탐험 동기 강화
- 비용 없는 편의: 발견 후에는 무제한 사용. 스트레스 없는 이동 = 탐험 집중 환경 조성

**수직 샤프트 (Vertical Shafts)**

성의 구조 자체가 수직 이동을 위한 기믹이다.

- 긴 수직 공동(Vertical Shaft)이 구역 간 빠른 상/하 이동을 허용
- 낙하로 빠른 하강, 조각상/도약 발판으로 상승
- 음악이 울려퍼지는 거대 홀 → 수직 이동 자체가 분위기 구성 요소

#### 할로우 나이트 (Hollow Knight) — 수직 이동 기믹

**스태그웨이 시스템 (Stagway)**

할로우 나이트에서 가장 정교하게 설계된 수직 이동 시스템이다.

| 항목 | 내용 |
|:-----|:-----|
| 운반 수단 | Last Stag — 유일하게 살아남은 수사슴 NPC |
| 네트워크 | Hallownest 전역 11개 역 연결 |
| 활성화 방식 | 역 발견 → Geo(게임 내 화폐)로 통행료 개방 → 이후 무료 |
| 탑승 방식 | 역에서 종 치기 → Last Stag 도착 → 목적지 선택 → 이동 |
| 수직 연결 | 수직으로 상/하 구역을 모두 연결. 지하 깊은 층부터 지상까지 |

**설계 원칙 인사이트:**
- 캐릭터 부여된 이동 수단: Last Stag이라는 NPC에 이동 기능을 탑재 → 단순 UI가 아닌 세계관 내 살아있는 존재. 이동 중 대화로 세계관 정보 전달
- 최초 유료 + 이후 무료: 소액 비용이 발견 순간의 투자감 부여. 이후 무료는 반복 사용 마찰 제거
- 최초 발견 필수: 역 발견 전까지는 존재 자체를 모름 → 탐험 보상으로서의 이동 기능

**트램 (Tram)**

스태그웨이와 별개로 존재하는 수평/수직 이동 수단이다.

| 항목 | 내용 |
|:-----|:-----|
| 연결 구간 | Failed Tramway 구간 — 서쪽 구석의 Deepnest와 동쪽 Kingdom's Edge 연결 |
| 활성화 조건 | Tram Pass 아이템 획득 (숨겨진 구역에서 발견) |
| 운용 방식 | Tram Pass 사용 → 탑승 → 자동 이동 |

**설계 원칙 인사이트:**
- 아이템 키: 교통 수단 자체를 아이템 획득으로 해금. 아이템이 이동 게이트 역할
- 경쟁 교통수단 서사: Last Stag은 Tram을 "기계가 자신을 대체할 수 없다"고 비하. 두 이동 수단의 서사적 경쟁이 세계관 깊이 추가
- 극단적 거리 커버: 게임 맵의 가장 먼 두 지점을 연결. 최후반에 활용되는 이동 수단

**드림게이트 / 명상 벤치 (Dream Gate + Bench)**

| 항목 | 내용 |
|:-----|:-----|
| 드림게이트 | 특정 지점에 게이트 설치 → 이후 어디서든 해당 지점으로 순간이동 |
| 사용 비용 | 드림 에센스(Dream Essence) 소모 |
| 벤치 | 쉬어가는 저장 지점. 최초 방문 후 맵에 표시 |

**설계 원칙 인사이트:**
- 플레이어가 위치 지정: 고정 목적지가 아닌 플레이어가 앵커 포인트를 직접 설정 → 높은 자율성(Autonomy) 제공
- 자원 소모: 무한 사용 방지. 드림 에센스라는 희소 자원이 남용을 억제

#### 슈퍼 메트로이드 / 메트로이드 시리즈 — 수직 이동 기믹

**엘리베이터 (Area Elevators)**

| 항목 | 내용 |
|:-----|:-----|
| 역할 | 행성 Zebes의 각 구역(Brinstar, Norfair, Maridia 등)을 연결 |
| 활성화 | 없음 — 발견 시 즉시 사용 가능 |
| 시각 언어 | 빛줄기/에너지 기둥으로 표시. 멀리서도 식별 가능 |
| 이동 속도 | 매우 느린 연출 → 목적지 예고 창이 표시되며 도착 |

**설계 원칙 인사이트:**
- 무조건 개방: 진입하면 바로 사용 가능. 탐험 동선에서 병목 없음
- 느린 이동 = 전환 의식(Transition Ritual): 이동 중 구역 이름 표시. 느림이 오히려 "다른 세계로 넘어간다"는 감각 강화

**스피드부스터 샤인스파크 경로 (Shinespark Path)**

| 항목 | 내용 |
|:-----|:-----|
| 메커닉 | 스피드부스터 능력으로 발동 → 특정 각도로 발사체처럼 날아감 |
| 수직 이동 | 수직 샤인스파크 경로를 통해 수십 타일 즉각 상승 |
| 조건 | 스피드부스터 능력 획득 + 특정 지점에서 발동 |

**설계 원칙 인사이트:**
- 숙련 보상 이동: 조작이 어렵지만 성공 시 엄청난 수직 거리를 한 번에 커버. 숙련(Competence) 만족의 극대화
- 환경이 경로 제시: 샤인스파크 블록의 위치 배치 자체가 "여기서 올라갈 수 있다"는 힌트

### 1.2. 플랫폼 게임 — 환경 기반 수직 기믹

#### 셀레스트 (Celeste) — 환경 이동 기믹

**바람 기류 방 (Wind Current Rooms)**

| 항목 | 내용 |
|:-----|:-----|
| 메커닉 | 일정 방향으로 지속 바람 → 플레이어가 해당 방향으로 가속/감속 |
| 수직 적용 | 상승 바람이 가득 찬 방 → 플레이어가 천천히 떨어지거나 상승 |
| 플레이어 역할 | 바람 방향에 저항하거나 이용하여 정밀 이동 |
| 위험 요소 | 바람이 플레이어를 위험 구역으로 밀 수 있음 |

**설계 원칙 인사이트:**
- 기믹과 도전의 통합: 이동 수단이자 장애물. 같은 메커닉이 방 배치에 따라 지원/방해 역할 교체
- 신체 감각 전달: 바람 방향으로 캐릭터 애니메이션 변화 + 바람 SFX → 시각/청각 피드백

**스프링 / 발사 크리스탈 (Springs / Crystal Hearts as Launch Pads)**

| 항목 | 내용 |
|:-----|:-----|
| 메커닉 | 밟거나 접촉 시 특정 방향으로 발사 |
| 수직 적용 | 수직 스프링은 플레이어를 수 화면 높이 즉각 발사 |
| 반복 사용 | 재충전 시간 없이 즉시 재사용 가능 |

**설계 원칙 인사이트:**
- 예측 가능한 결과 + 높은 피드백: 방향/속도가 고정되어 있어 익힌 후 신뢰 가능. 발사 순간의 강한 시각/음향 피드백이 만족감 극대화

**드림 블록 (Dream Blocks)**

| 항목 | 내용 |
|:-----|:-----|
| 메커닉 | 접촉 시 플레이어를 특정 방향으로 빠르게 통과/이동 |
| 수직 적용 | 수직 드림 블록 체인으로 수십 타일 고속 상승 가능 |
| 시각 효과 | 반투명 몽환적 비주얼 → 일반 플랫폼과 즉각 구별 |

### 1.3. 소울류 / 기타 — 수직 이동 기믹

#### 다크 소울 (Dark Souls) — 엘리베이터 숏컷

**언데드 패리시 → 파이어링크 사당 (Undead Parish to Firelink Shrine)**

다크 소울 역사상 가장 유명한 수직 이동 기믹이다.

| 항목 | 내용 |
|:-----|:-----|
| 잠금 해제 조건 | 언데드 패리시 교회 내 레버 활성화 |
| 이동 내용 | 교회 옆 건물 → 파이어링크 사당으로 즉각 하강 |
| 수직 거리 | 게임 내 체감상 수십 미터 하강 |
| 이동 속도 | 느린 엘리베이터 — 세계의 연결성을 인식하는 시간 제공 |

**설계 원칙 인사이트:**

미야자키 히데타카는 "수직 상승하는 진행 구조를 항상 선호한다"고 언급했다. 이 엘리베이터의 설계 비밀은 다음과 같다:

1. **발견의 보상 구조:** 엘리베이터 자체를 "탐험의 보상"으로 설계. 레버를 찾는 것이 먼저이고, 엘리베이터 사용은 그 결과다
2. **공간 인식 재구성:** 이동 도중 창문 너머로 지나온 구역이 보인다. 플레이어의 정신 지도(mental map)가 재구성되는 순간
3. **"아, 이렇게 연결되어 있었구나"의 감각:** Hollow Knight의 Stag와 동일한 원리. 세계가 입체적으로 연결되어 있음을 체감시키는 순간

**두 다리를 걷어차 만드는 사다리 (Ladder Kicks)**

단순하지만 인상적인 수직 이동 기믹이다.

- 사다리를 걷어차 아래로 내림 → 이후 양방향 이동 가능
- "내가 이 연결을 직접 열었다"는 에이전시(Agency) 감각
- 발견 전 내려오는 데 걸리는 긴 시간 vs 발견 후 즉시 이동 → 극적인 시간 대비

### 1.4. 수직 이동 기믹 비교 분석

#### 기믹 유형 분류표

| 기믹 유형 | 대표 사례 | 이동 속도 | 활성화 비용 | 플레이어 에이전시 | 수직 거리 |
|:----------|:---------|:---------|:-----------|:----------------|:---------|
| 엘리베이터/리프트 | SotN, Dark Souls, Hollow Knight | 중간~느림 | 레버/탐험 | 낮음 (탑승만) | 대 |
| 순간이동 방 | SotN, Metroid | 즉시 | 탐험 | 중간 (목적지 선택) | 무제한 |
| NPC 수송 수단 | Hollow Knight Stag | 중간 | 탐험+소액 | 중간 | 대 |
| 바람 기류 | Celeste, Ori | 중간 | 없음 (환경 고정) | 낮음 | 소~중 |
| 발사대/스프링 | Celeste, 메가맨 | 빠름 | 없음 | 낮음 | 중 |
| 로프/사다리 | Spelunky, Dark Souls | 느림 | 소모품/선행 탐험 | 높음 | 소~중 |
| 파이프/튜브 | Dead Cells, Rain World | 빠름 | 없음 | 낮음 | 중~대 |
| 숏컷 해금 | Spelunky | 즉시 | 누적 자원 | 높음 | 전 구간 |

#### 핵심 발견: 수직 이동 기믹의 만족도 요소

1. **발견이 먼저, 사용이 나중:** 가장 만족스러운 기믹은 탐험을 통해 발견하고, 이후 반복 사용 가능하다 (Dark Souls 엘리베이터, SotN 텔레포트, Hollow Knight Stag)
2. **느린 이동 = 세계 인식:** 빠른 순간이동보다 느린 이동이 오히려 세계의 규모와 연결성을 체감시킨다
3. **세계관 내재적 정당화:** 순간이동 UI가 아닌 캐릭터, 건축물, 자연 현상으로 구현된 기믹이 몰입도가 높다
4. **캐릭터 에이전시 부여:** 레버를 올리고, 사다리를 내리고, 스위치를 누르는 행위가 "내가 이 연결을 만들었다"는 소유감 생성
5. **귀환 동선 설계:** 한 방향(하강)이 편해지면 반대 방향(상승)에 흥미 있는 경로를 배치하여 단조로움 방지

## 2. BLAME! 수직 구조 분석

> **주의:** BLAME! 세계관의 기본 구조와 감정 설계는 `BLAME_Biomega_WorldDesign_Research.md`에 상세히 기술되어 있다. 본 섹션은 **수직 이동과 직접 관련된** 요소에 집중한다.

### 2.1. 도시의 수직 구조와 이동 기믹

#### 엘리베이터 — 상대론적 속도

BLAME! 도시 내 엘리베이터는 **상대론적 속도(relativistic speed)**에 도달할 수 있다. 그러나 이는 이동 속도가 아닌 **수직 거리의 비현실적 규모**를 역설적으로 강조하는 장치이다.

| 특성 | 내용 |
|:-----|:-----|
| 이동 속도 | 이론상 광속에 근접 가능 |
| 실제 이동 시간 | 엘리베이터를 타도 한 지층에서 다음 지층까지 **몇 달** 소요 |
| 계단 이동 | 계단으로 이동하면 단일 경로만으로 6,780km 이동 가능 |
| 메가스트럭처 관통 | **GBE(중력자선 방사기)의 직격**만이 메가스트럭처를 관통 가능 |

**서사적 함의:**
- 엘리베이터가 "빠른 이동 수단"임에도 달이 걸린다 → 규모의 비현실성이 오히려 세계의 광활함을 강조
- 빠른 이동조차 느린 세계 → 플레이어/독자가 Killy의 여정의 광활함을 체감

#### GBE — 수직 통로 창조 무기

GBE(Gravitational Beam Emitter)는 단순 무기가 아닌 **수직 이동 경로 창조 도구**이다.

| 특성 | 내용 |
|:-----|:-----|
| 기능 | 메가스트럭처(극도로 내구성 높은 재질) 관통 가능한 유일한 무기 |
| 수직 경로 창조 | 메가스트럭처를 관통하여 지층 간 새 통로 개척 |
| 관통 직경 | 발사체가 지나간 자리에 수십 미터 구멍 생성 |
| 서사 역할 | Killy가 막힌 벽을 뚫고 다음 지층으로 나아가는 유일한 방법 |

**게임 디자인 인사이트:**
- "강력한 무기 = 지형 변형"의 원리: 공격이 이동 수단이 되는 구조
- 힘의 표현이 이동의 수단: Killy의 힘을 가장 잘 보여주는 것이 이동 경로 창조

#### 수직 공동(Void)과 낙하

| 특성 | 내용 |
|:-----|:-----|
| 규모 | 일부 지층 사이 공간은 "목성 크기의 텅 빈 방" — Killy가 가로질러도 수십 km |
| 낙하 이동 | 의도적/비의도적 낙하가 수직 하강의 주요 수단 |
| 교량/현수 구조 | 공동에 걸린 파손된 교량, 매달린 구조물이 이동 경로를 형성 |
| 심층의 어둠 | 아래를 내려다봤을 때 끝이 보이지 않는 어둠 |

#### 빌더 구조물의 우연적 이동 경로

통제를 잃은 빌더의 무질서 건설이 만들어낸 역설적 이동 경로들:

| 구조물 유형 | 이동 기능 |
|:-----------|:---------|
| 끝도 없는 계단 | 수직 이동의 기본 수단. 그러나 목적지 불명 |
| 아무 데도 닿지 않는 파이프/교량 | 우연히 이동 경로를 연결하기도 함 |
| 무작위 배치된 플랫폼 | 절벽과 공동 사이 이동 지점 |
| 붕괴된 구조물 잔해 | 새 경사로/사다리 역할 |

**게임 디자인 인사이트:**
- **절차적 생성의 다이에제틱 정당화:** "왜 매번 방 구조가 다른가?" → "빌더가 무작위로 지어서"
- **이동 경로 자체가 세계관 스토리텔링:** 어디로 이어지는지 모르는 계단 = 통제 상실의 물리적 증거

### 2.2. 시각 언어 — 수직 공간 표현

Nihei의 수직 공간 표현법은 독보적이다. 게임 레벨 디자인에 직접 적용 가능한 요소들이다.

#### 극단적 스케일링 (Figure vs Background)

| 기법 | 내용 | 감정 효과 |
|:-----|:-----|:---------|
| 인물 = 패널의 1% 미만 | 거대한 구조물 앞 Killy의 극소한 실루엣 | 왜소감, 존재의 무의미함 |
| 버드아이 뷰 + 수직 전경 | 높은 곳에서 내려다본 무한한 구조물 | 현기증, 경외 |
| 아래를 보는 Killy | 패널 아래로 펼쳐지는 공동/어둠 | 나락으로의 공포 |
| 위를 보는 Killy | 패널 위로 무한히 연장되는 구조물 | 목표의 아득함 |

#### 패널 구성법 (Panel Composition)

| 기법 | 내용 |
|:-----|:-----|
| 수직 패널 사용 | 수직으로 길게 배치된 패널 → 높이를 직접적으로 전달 |
| 빈 공간의 적극 활용 | 구조물 사이의 공허 = 이야기의 일부. 침묵으로 규모 표현 |
| 작은 인물 + 큰 배경 | 항상 배경이 인물을 압도. 환경이 주인공 |
| 선 원근법 | 구조물이 무한히 소실점으로 수렴 → 깊이감 극대화 |

#### 선 스타일 — 무기적/유기적 충돌

| 요소 | 표현 방식 | 감정 |
|:-----|:---------|:-----|
| 메가스트럭처 | 직선, 반복, 기계적 정밀함 | 질서의 잔해 |
| 생명체/침식 부위 | 불규칙, 유기적, 암세포처럼 퍼지는 선 | 통제 불능의 공포 |
| 충돌 지점 | 두 선 스타일이 공존 | 긴장, 불안정 |

**ECHORIS 2D 적용:**
- 아이템계: Normal 등급 = 정밀한 직선 벽면. Ancient 등급 = 유기적 침식이 벽을 잠식
- 수직 샤프트 배경: 원근법 타일 레이어로 끝없이 내려가는 착시

## 3. 공통 패턴 — 좋은 수직 이동 기믹의 원칙

### 원칙 1: 발견이 투자다 (Discovery as Investment)

좋은 수직 이동 기믹은 발견 즉시 사용 가능하지 않다. **발견 → 활성화/해금 → 반복 사용**의 3단계 구조가 이동 기믹에 의미를 부여한다.

| 단계 | 감정 | 설계 도구 |
|:-----|:-----|:---------|
| 발견 | 호기심, 기대 | 시각적 단서, 숨겨진 위치 |
| 활성화 | 성취, 소유 | 레버/스위치/아이템/자원 납부 |
| 반복 사용 | 편리함, 마스터리 | 무제한 무료 사용 |

**위반 시 문제:** 발견과 동시에 사용 가능한 기믹은 "발견"이 아닌 "발생"에 가깝다. 미학적 만족도가 낮다.

### 원칙 2: 세계관 내재적 정당화 (Diegetic Justification)

UI 기반 빠른 이동("지도에서 클릭")보다 세계관 속 캐릭터/건축/현상으로 구현된 기믹의 몰입도가 월등히 높다.

| 수준 | 예시 | 몰입도 |
|:-----|:-----|:------|
| 순수 UI | 지도 클릭 → 텔레포트 | 낮음 |
| 건축적 | 엘리베이터, 파이프 | 중간 |
| 생태적/서사적 | Hollow Knight Last Stag (NPC), Ori Spirit Well (정령 세계) | 높음 |
| 플레이어 창조 | 사다리 발로 차기, Dark Souls 레버 | 최고 |

### 원칙 3: 속도와 규모 인식의 역설 (Speed-Scale Paradox)

빠른 이동이 항상 더 좋은 경험은 아니다. **느린 이동이 세계의 규모를 체감시키고, 때로는 더 강한 감정을 유발한다.**

- Dark Souls 엘리베이터: 느린 이동 중 지나온 구역이 보임 → "세계가 연결되어 있다"는 인식
- BLAME!: 광속 엘리베이터도 달이 걸림 → 규모의 비현실성이 세계관의 핵심
- Metroid 엘리베이터: 느리지만 구역 이름 표시 → 전환의 의식

**결론:** 이동 중 세계 정보를 제공하는 느린 이동이 순간이동보다 세계관 밀도를 높인다.

### 원칙 4: 위험/위험 해방 이분법 (Danger/Relief Duality)

Rain World의 파이프처럼, 수직 이동 기믹은 **위험한 외부 공간 vs 안전한 이동 채널**의 대비를 만들 때 가장 강렬하다.

- 외부: 포식자, 적, 낙사 위험
- 파이프/엘리베이터 내부: 안전, 이완, 다음 도전 준비
- 이 전환이 반복될수록 이동 기믹이 "안도"라는 감정을 획득

### 원칙 5: 에이전시 연속체 (Agency Continuum)

수직 이동 기믹은 플레이어 에이전시 수준에 따라 다른 감정을 유발한다.

```
낮은 에이전시 ←────────────────────────────── 높은 에이전시
바람에 밀림 → 발사대 → 엘리베이터 탑승 → 사다리 내리기 → 경로 스스로 개척
(놀라움/압도)              (편리함/만족)                (성취/소유)
```

**설계 지침:** 메인 이동 수단은 중간 에이전시. 희귀한 특수 기믹으로 양 극단 제공.

### 원칙 6: 수직 이동은 공간 내러티브다 (Vertical Movement as Spatial Narrative)

가장 기억에 남는 수직 이동 기믹은 이동 중 세계에 대해 무언가를 말한다.

| 기믹 | 서사 메시지 |
|:-----|:----------|
| Dark Souls 엘리베이터 | "이 세계는 수직으로 연결된 하나다" |
| Hollow Knight Stag | "이 왕국에는 한때 교통망이 있었다" |
| BLAME! 엘리베이터 | "광속으로 가도 다음 층이 멀다 — 이 세계는 이해 불가능한 규모다" |
| SotN 텔레포트 방 | "이 성에는 고대의 비밀이 있다" |

## 4. ECHORIS 수직 이동 기믹 제안

### 4.1. 설계 전제

ECHORIS의 수직 이동 기믹은 세 가지 제약 조건을 충족해야 한다:

1. **BLAME! 오마주 세계관 정합성:** 메가스트럭처 산업 미학, 빌더의 무질서 건설, 거대한 수직 공동이라는 세계관에 자연스럽게 녹아야 한다
2. **2-Space 구조 지원:** 월드(탐험) 공간과 아이템계(파밍) 공간 각각에서 다른 역할을 수행한다
3. **Phase별 구현 현실성:** Build 0 → Build 1 → Build 2 단계에서 점진적으로 추가 가능해야 한다

### 4.2. 월드(World) 공간의 수직 이동 기믹

#### 기믹 1: 빌더의 승강 플랫폼 (Builder's Freight Platform)

**세계관 정당화:** 빌더가 자재를 층간 운반하기 위해 설치한 화물 엘리베이터. 현재는 반자동 상태.

| 항목 | 내용 |
|:-----|:-----|
| 외형 | 거대한 금속 화물 플랫폼. 녹과 빌더 마크 각인 |
| 활성화 조건 | 플랫폼 주변의 빌더 터미널 조작 (오염된 명령어 해독) |
| 이동 방식 | 플랫폼에 올라타면 설정된 높이까지 느린 속도로 상승 |
| 이동 속도 | 느림 (약 3~5초) — 이동 중 배경으로 지나온 구역 가시화 |
| 위험 요소 | 플랫폼 가장자리에서 낙사 가능. 탑승 중 적 등장 가능 |
| 연결 지점 | 2점 연결 (하부 거점 ↔ 상부 거점) |

**게임플레이 역할:**
- 주요 구역 간 수직 연결 수단
- 터미널 조작이 탐험 행위 → 발견-투자-보상 사이클 충족
- 느린 이동 중 배경 가시화로 세계 규모 체감

**구현 난이도:** 낮음 (플랫폼 엔티티 + 이동 애니메이션 + 탑승 판정)

**우선순위:** Build 1 (Phase 2)

---

#### 기믹 2: 상승 기류 샤프트 (Updraft Shaft)

**세계관 정당화:** 빌더의 환기 시스템에서 나오는 강한 상승 기류. 일부는 여전히 작동 중.

| 항목 | 내용 |
|:-----|:-----|
| 외형 | 수직으로 뚫린 샤프트. 격자 환기구에서 바람이 뿜어져 나옴 |
| 활성화 조건 | 없음 — 항상 활성 상태 |
| 이동 방식 | 샤프트 내부에 들어가면 빠른 속도로 상승. 조작으로 방향 조절 가능 |
| 이동 속도 | 빠름 — 수십 타일을 수초 내 상승 |
| 위험 요소 | 샤프트 벽에 부딪히면 데미지. 상단 격자에 막히는 경우 존재 |
| 탈출 방법 | 좌/우 이동으로 샤프트 밖으로 나올 수 있음 |

**게임플레이 역할:**
- 탐험 과정에서 자연스럽게 발견하는 상승 수단
- 위험 요소(벽 충돌, 막힌 격자)가 단순 이동을 미니 스킬 체크로 만듦
- Celeste 바람 방의 BLAME! 버전

**구현 난이도:** 낮음 (트리거 존 + 중력 반전/감쇠 물리)

**우선순위:** Build 1 (Phase 2)

---

#### 기믹 3: 붕괴 발판 낙하 경로 (Collapse Chute)

**세계관 정당화:** 오래된 지층의 바닥이 부분 붕괴되어 생긴 자연적 하강 경로.

| 항목 | 내용 |
|:-----|:-----|
| 외형 | 균열 간 틈새. 아래로 어둠이 이어짐. 선택적 진입 표시 |
| 활성화 조건 | 없음 — 항상 진입 가능 |
| 이동 방식 | 낙하 → 여러 층을 한 번에 하강. 아래 안전 착지 지점 배치 |
| 이동 속도 | 빠름 (하강만 가능) |
| 위험 요소 | 잘못된 낙하 시 올바른 경로를 놓쳐 위험 구역 착지 가능 |
| 귀환 경로 | 상승 수단(기류 샤프트, 빌더 플랫폼)을 통해서만 복귀 |

**게임플레이 역할:**
- 상승보다 빠른 하강 경로 제공 → 비대칭 수직 이동
- 한 번 내려가면 돌아오기 어렵다는 구조가 탐험의 경이/불안 강화
- BLAME!의 "목적지 불명의 낙하"를 게임 메커닉으로 번역

**구현 난이도:** 매우 낮음 (특수 낙하 트리거 + 착지 지점)

**우선순위:** Build 0 (Phase 1) — 즉시 구현 가능한 가장 단순한 수직 기믹

---

#### 기믹 4: 에코 반향 경로 (Echo Resonance Path)

**세계관 정당화:** 에르다의 도구 "에코(Echo)"가 발사한 에너지 반향이 빌더의 잠든 시스템을 일시 활성화.

| 항목 | 내용 |
|:-----|:-----|
| 외형 | 빌더 마크가 각인된 수직 벽면. 에코 사용 시 발광 |
| 활성화 조건 | 에코(에르다의 무기)를 벽면에 조준 사격 |
| 이동 방식 | 에코 발사 → 벽면 활성화 → 임시 발판/통로 생성 (10초) |
| 이동 속도 | 중간 — 생성된 발판을 밟고 올라가야 함 |
| 제한 | 에코 사용 쿨다운(Combat 시스템 연동) |

**게임플레이 역할:**
- 에르다의 정체성(격벽 측량사, GBE 오마주) 강화
- 이동과 전투 시스템의 교차점 — 에코가 탐험 도구 역할 수행
- 특정 구역에서만 사용 가능 → 지형지물로서의 탐험 발견 요소

**구현 난이도:** 중간 (에코 발사 이벤트 + 임시 발판 생성 + 타이머)

**우선순위:** Build 1~2 (Phase 2 이후) — 에코 시스템 완성 후

---

#### 기믹 5: 빌더 워프 노드 (Builder Warp Node)

**세계관 정당화:** 빌더의 물류 순간이동 시스템. 일부 노드는 아직 기능 중.

| 항목 | 내용 |
|:-----|:-----|
| 외형 | 빌더 문양이 새겨진 육각형 플랫폼. 활성화 시 에너지 기둥 생성 |
| 활성화 조건 | 양쪽 노드를 모두 탐험으로 발견해야 사용 가능 |
| 이동 방식 | 노드 진입 → 목적지 선택 → 즉시 이동 |
| 이동 속도 | 즉시 |
| 노드 수 | 구역당 1~2쌍 (과도한 빠른 이동 방지) |

**게임플레이 역할:**
- SotN 텔레포트 방의 BLAME! 버전
- 발견 보상: 두 노드를 모두 찾아야 연결 활성화 → 탐험 가치 강화
- 장거리 이동 편의 제공 → 후반 탐험 마찰 감소

**구현 난이도:** 중간 (페어 등록 시스템 + UI)

**우선순위:** Build 2 (Phase 2 후반)

---

### 4.3. 아이템계(Item World) 공간의 수직 이동 기믹

아이템계는 절차적 생성 환경이므로, 수직 이동 기믹은 **고정된 특수 방(Special Room)**으로 구현된다.

#### 기믹 6: 기억의 침강 샤프트 (Memory Descent Shaft)

**세계관 정당화:** 아이템의 기억이 응결된 지층 사이로, 과거 주인의 기억이 상승/하강한다.

| 항목 | 내용 |
|:-----|:-----|
| 방 유형 | 절차적 배치 특수 방 — 지층당 1~2회 등장 |
| 외형 | 수직으로 긴 방. 중심에 반투명 기억 파편이 떠다님 |
| 이동 방식 | 파편에 올라타면 아래 지층 보스 방 근처로 즉각 하강 |
| 위험 요소 | 하강 중 벽에서 적 출현 가능 |
| 제한 | 하강만 가능. 상승 불가 (텔레포트 크리스탈로만 귀환) |

**게임플레이 역할:**
- 지층 내 보스 방까지 빠른 접근 경로
- 세션 리듬 가속 — 아이템계의 "한 층만 더" 중독 강화
- 하강 = 돌아오기 어렵다는 리스크 강화

**구현 난이도:** 낮음 (특수 방 템플릿 + 순간이동 트리거)

**우선순위:** Build 1 (Phase 1 후반)

---

#### 기믹 7: 기억 단편 발사 캐터펄트 (Memory Shard Catapult)

**세계관 정당화:** 아이템 내부를 지배하는 기억 단편 중 "발사 기억 단편"가 플레이어를 위/아래로 쏘아 올림.

| 항목 | 내용 |
|:-----|:-----|
| 방 유형 | 절차적 배치 특수 방 |
| 외형 | 거대한 탄도 발사대. 기억 단편가 주변을 맴돌며 발사 준비 |
| 이동 방식 | 발사대에 올라타면 지정 방향으로 발사. 위/아래 선택 가능 |
| 이동 속도 | 매우 빠름 (즉각) |
| 위험 요소 | 착지 지점에 적이 있을 수 있음 |

**게임플레이 역할:**
- 아이템계 특화 기믹 — 기억 단편 세계관 강화
- Celeste 스프링과 동일한 원리를 BLAME!/디스가이아 미학으로 번역
- 빠른 이동으로 지층 탐험 리듬 가속

**구현 난이도:** 낮음 (발사대 트리거 + 포물선 이동)

**우선순위:** Build 1 (Phase 1 후반)

---

#### 기믹 8: 붕괴 메가스트럭처 낙하 지대 (Megastructure Collapse Zone)

**세계관 정당화:** 아이템 기억이 불안정한 지층. 메가스트럭처 일부가 무너지며 아래 지층으로 통하는 구멍이 열린다.

| 항목 | 내용 |
|:-----|:-----|
| 방 유형 | 이벤트 트리거 방 |
| 외형 | 천장/바닥에 균열. 시간이 지나면 붕괴 |
| 이동 방식 | 붕괴 전 구멍 통과 → 아래 지층으로 직접 이동 |
| 위험 요소 | 붕괴 잔해 데미지. 타이밍 실패 시 일반 진행 경로로 |
| 제한 | 스킵이므로 보상 없음 (중간 지층 보상 건너뜀) |

**게임플레이 역할:**
- 숙련 플레이어를 위한 스피드런/스킵 경로
- 리스크-리워드: 빠른 진행 vs 보상 포기의 선택지
- BLAME!의 "GBE로 벽을 뚫는" 서사를 환경 기믹으로 번역

**구현 난이도:** 중간 (타이밍 기반 붕괴 시퀀스 + 지층 간 이동)

**우선순위:** Build 2 (Phase 2)

### 4.4. 기믹 설계 요약표

| 기믹명 | 공간 | 역할 | 구현 난이도 | Build 우선순위 |
|:-------|:-----|:-----|:----------|:-------------|
| 붕괴 발판 낙하 경로 | 월드 | 빠른 하강, 탐험 긴장 | 매우 낮음 | Build 0 |
| 상승 기류 샤프트 | 월드 | 빠른 상승 이동 | 낮음 | Build 1 |
| 빌더의 승강 플랫폼 | 월드 | 구역 간 수직 연결 | 낮음 | Build 1 |
| 기억의 침강 샤프트 | 아이템계 | 보스 방 빠른 접근 | 낮음 | Build 1 |
| 기억 단편 발사 캐터펄트 | 아이템계 | 지층 내 빠른 이동 | 낮음 | Build 1 |
| 에코 반향 경로 | 월드 | 탐험 발견 + 에코 정체성 | 중간 | Build 2 |
| 붕괴 메가스트럭처 낙하 | 아이템계 | 숙련자 스킵 경로 | 중간 | Build 2 |
| 빌더 워프 노드 | 월드 | 장거리 빠른 이동 | 중간 | Build 2 |

### 4.5. Build 0 즉시 구현 권고

**붕괴 발판 낙하 경로(기믹 3)** 단독으로 Build 0에서 구현을 권고한다.

이유:
1. 구현 난이도 최저 (1~2시간 이내 프로토타입 가능)
2. 수직 하강 기믹이 맵 설계 테스트에 즉시 활용 가능
3. BLAME! 세계관과 완벽한 정합성 — 특수 에셋 없이 기존 타일로 표현 가능
4. 플레이어 수직 이동의 기본 흐름(위험한 하강 vs 안전한 상승)을 가장 단순하게 검증
