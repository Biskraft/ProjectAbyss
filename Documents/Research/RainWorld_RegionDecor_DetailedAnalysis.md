## 공통 시스템 요약 (전 지역 공유)

Rain World의 모든 지역은 **동일한 이펙트 에디터 도구 세트**를 공유한다. 지역 차이는 이펙트 조합, 밀도, 팔레트에 의해 결정된다. [확인함]

### 공통 이펙트 도구

| 이펙트 | 역할 | 적용 패턴 |
|:-------|:-----|:---------|
| **BlackGoo** | 방 외곽 그림자. 기본 전체 적용 후 지우는 역방식 | 전체 |
| **Slime** | 표면을 끈적이고 흘러내리는 모양으로 변형 | Surface |
| **Erosion** | 지형 가장자리 침식/노후화 | Surface |
| **Corrosion** | 금속/구조물 부식 | Surface |
| **Rust** | 금속 표면 녹 | Surface |
| **Grass** | 풀잎 생성 | Grower (바닥->위) |
| **Seed Pods** | 씨앗 꼬투리 | Grower |
| **Cacti** | 선인장형 식물 | Grower |
| **Rain Moss** | 비이끼 | Clinger (벽) |
| **Horse Tails** | 쇠뜨기 식물 | Grower |
| **Circuit Plants** | 회로 식물 | Grower |
| **Feather Plants** | 깃털 식물 | Grower |
| **Thick Roots** | 두꺼운 뿌리 | Hanger (천장->아래) |
| **Shadow Plants** | 그림자 식물 | Hanger |
| **Fungi** | 균류 | Clinger |
| **SlimeMold** | 점액질 곰팡이 | Clinger |
| **Chains** | 배경 분할용 체인 | Structure |
| **Giant Signs** | 거대 표지판 | Structure |
| **Rubble** | 잔해 파편 (Grower 코드 재사용) | Grower |
| **DaddyCorruption** | 크리처 관련 벽 오염 | Special |

[확인함 -- Rain World Effects Editor 문서 기반]

### 공통 셰이더 유니폼

모든 지역에서 동일하게 작동하는 대기 변수: `_RAIN`(사이클 진행도), `_WetTerrain`(습기 반사), `_fogAmount`(안개), `_Grime`(오염도), `_windAngle`(바람). [확인함]

## 1. SU -- Outskirts (외곽 지대)

> **정체성:** 게임의 첫인상. 황량하지만 적대적이지 않은 구조물 외곽.
> **내러티브:** "도시 밖, 문명의 끝자락"

### 팔레트

- **주조색:** 회색~밝은 회색 (콘크리트), 흰 구름 낀 하늘 [확인함]
- **배경색:** 옅은 회색 안개, 밝은 중간 톤 [확인함]
- **악센트:** 밝은 연두색 (이끼/풀 -- 유일한 생명감 있는 색조) [확인함]
- **채도:** 전 지역 중 최저. 의도적으로 눈 피로를 줄임 [확인함]

### 바닥 장식 (Floor Decorations)

| 요소 | 설명 | 이펙트 |
|:-----|:-----|:-------|
| **짧은 풀 (Short Grass)** | 바닥 가장자리에서 자라는 짧고 성긴 풀. 연두색. 밀도 낮음 | Grass (Grower) |
| **벽돌 잔해** | 바닥 표면 자체가 과거 건물의 벽돌 잔해로 구성 [확인함] | Material (Brick) |
| **물 웅덩이** | 평평한 바닥에 얕은 물. 회색 하늘을 반사 | 수면 시스템 |
| **Rubble 파편** | 콘크리트/벽돌 파편이 바닥에 흩어짐 | Rubble (Grower) |
| **Seed Pods** | 드문드문 씨앗 꼬투리. 풀과 혼합 배치 | Seed Pods (Grower) |

### 천장 장식 (Ceiling Decorations)

| 요소 | 설명 | 이펙트 |
|:-----|:-----|:-------|
| **짧은 뿌리** | 콘크리트 틈새에서 내려오는 얇은 뿌리 가닥 | Thick Roots (Hanger), 낮은 밀도 |
| **물방울** | 천장에서 떨어지는 물방울 (비 사이클 연동) | Slime + _WetTerrain |
| **노출 철근** | 부식된 콘크리트에서 삐죽 튀어나온 철근 [확인함] | Prop (수작업) |

### 벽 장식 (Wall Decorations)

| 요소 | 설명 | 이펙트 |
|:-----|:-----|:-------|
| **얇은 이끼** | 벽면에 부착된 밝은 연두색 이끼 패치. 밀도 낮음 | Rain Moss (Clinger) |
| **침식 흔적** | 콘크리트 벽면 가장자리의 불규칙한 침식 | Erosion |
| **슬라임 흘러내림** | 습기 있는 벽면에 어두운 슬라임 줄기 | Slime |
| **벽돌 노출** | 콘크리트 표면이 벗겨져 내부 벽돌이 드러남 | Material 혼합 |

### 배경 요소 (Background / Parallax)

| 레이어 | 요소 | 설명 |
|:-------|:-----|:-----|
| **Layer 2 (중경)** | 콘크리트 벽체 | 방 뒷벽을 이루는 회색 콘크리트. 균열과 녹 패치 포함 |
| **Layer 3 (원경)** | 흐린 하늘 + 거대 구조물 실루엣 | 옅은 회색 하늘 배경에 멀리 보이는 산업 구조물의 희미한 윤곽 [추측임] |
| **최원경** | 안개 그래디언트 | 팔레트 배경색(회색)으로 수렴. 공기원근법 |

### 고유 이펙트

| 이펙트 | 설명 |
|:-------|:-----|
| **흐린 자연광** | 균일하고 방향성 약한 조명. 어두운 구석이 거의 없음 [확인함] |
| **Worm Grass 구역** | 바닥에서 자라는 적대적 벌레 풀. 전용 크리처 이펙트 [확인함] |
| **캐러셀 그림자** | 배경에 느리게 움직이는 큰 그림자 패턴 [확인함] |

### 전환 특성

- **SU -> HI (Industrial Complex):** 풀과 이끼 밀도 감소, 금속 타일 비율 증가, 팔레트가 회색에서 갈색-주황으로 점진적 전환
- **SU -> DS (Drainage System):** 바닥 물 빈도 급증, 풀 사라지고 슬라임 증가, 팔레트가 보라-자주로 전환

## 2. HI -- Industrial Complex (산업 단지)

> **정체성:** 녹슨 거대 기계의 수직 미궁. Rain World에서 가장 높은 방들.
> **내러티브:** "무엇을 제조하던 공장인지 알 수 없는 산업 폐허"

### 팔레트

- **주조색:** 진한 회색-갈색 (산화 금속), 주황-갈색 녹 [확인함]
- **배경색:** 어두운 회색-청색 (깊은 공장 내부) [확인함]
- **악센트:** 주황-황색 (공장 인공 조명)
- **대비:** 강한 명암비. 인공광이 날카로운 경계를 만듦

### 바닥 장식

| 요소 | 설명 | 이펙트 |
|:-----|:-----|:-------|
| **금속 파편** | 녹슨 금속 조각, 기계 부품 잔해 | Rubble (Grower) |
| **기름 웅덩이** | 바닥의 어두운 반사 웅덩이. 물보다 탁함 | 수면 + Grime |
| **녹슨 격자** | 금속 격자/그레이팅 바닥. 아래가 보임 [추측임] | Material (Metal) |
| **Seed Pods (드문)** | 기계 틈새에서 자란 식물. 극히 드문 생명 | Seed Pods, 극저 밀도 |

### 천장 장식

| 요소 | 설명 | 이펙트 |
|:-----|:-----|:-------|
| **체인** | 수직으로 내려오는 산업용 체인. 등간격 링크 패턴 [추측임] | Chains |
| **파이프** | 굵은 산업 배관. 수평/수직 교차 | Prop (Pipe) |
| **녹은 금속 방울** | 천장 접합부에서 늘어진 슬라임/녹 흘러내림 | Slime |
| **환풍구** | 산업용 환기 구조물 [추측임] | Prop |

### 벽 장식

| 요소 | 설명 | 이펙트 |
|:-----|:-----|:-------|
| **심한 녹** | 벽면 전체에 걸친 주황-갈색 산화 패턴 | Rust + Corrosion |
| **노출 기어/톱니** | 기계 장치의 일부가 벽에 박혀 노출 [확인함] | Prop |
| **그래피티 식물** | 기계 틈에서 자란 그래피티 형태 식물 [확인함] | Rain Moss (극저 밀도) |
| **접합 볼트** | 금속판을 고정하는 대형 볼트 헤드 [추측임] | Prop |
| **산업용 팬** | 식물 부패물에 막힌 환풍 팬 [확인함] | Prop |

### 배경 요소

| 레이어 | 요소 | 설명 |
|:-------|:-----|:-----|
| **Layer 2** | 거대 기계 실루엣 | 수직 타워, 교량, 크레인의 어두운 윤곽 [추측임] |
| **Layer 3** | 끝없는 수직 공장 | 위아래로 계속되는 금속 구조물. 안개에 녹아듦 |
| **최원경** | 어두운 청회색 안개 | 공장 내부의 깊이감 |

### 고유 이펙트

| 이펙트 | 설명 |
|:-------|:-----|
| **주황-황색 인공광** | 공장 조명. 강한 방향성, 고대비 그림자 [확인함] |
| **수직 규모감** | 방 자체가 매우 키가 큼. 여러 화면 높이의 수직 공간 [확인함] |
| **기계 소음 분위기** | 시각적이 아닌 청각적이지만, 시각적으로 "기계가 작동하던 흔적"이 있음 |

### 전환 특성

- **HI -> CC (Chimney Canopy):** 금속 비율 유지하되 개방감 증가. 하늘이 보이기 시작
- **HI -> SU:** 금속이 콘크리트로 전환, 녹과 기름이 이끼와 풀로 대체

## 3. DS -- Drainage System (배수 시스템)

> **정체성:** 지하 배수/하수 터널. 습기와 어둠.
> **내러티브:** "도시의 하수 시스템. 물이 모든 것을 지배"

### 팔레트

- **주조색:** 보라-자주색, 어두운 청보라 [추측임]
- **배경색:** 짙은 보라-검정
- **악센트:** 청록색 물, 보라빛 슬라임
- **특이점:** 전 지역 중 가장 독특한 색조. 보라색 팔레트가 "하수"라는 비위생적 인상 강화

| **슬라임 바닥** | 물에 잠기지 않은 바닥의 끈적한 퇴적물 | Slime |
| **균류** | 습한 바닥에서 자라는 작은 균류 군집 [추측임] | Fungi 변형 |
| **배수 격자** | 금속 배수 격자. 아래로 물이 빠짐 [추측임] | Material |

| **파이프 집합** | 배수/하수 배관 밀집. 다양한 굵기 | Prop (Pipe) |
| **두꺼운 뿌리** | 지상 식물의 뿌리가 천장을 관통 | Thick Roots (Hanger) |
| **SlimeMold** | 천장에 달라붙은 점액질 곰팡이 [추측임] | SlimeMold |

### 벽 장식

| 요소 | 설명 | 이펙트 |
|:-----|:-----|:-------|
| **두꺼운 슬라임** | 벽면 전체를 덮는 끈적한 유기물 | Slime (고밀도) |
| **파이프 관통** | 벽을 관통하는 배관 단면 [추측임] | Prop |

| **수위선 자국** | 과거 수위가 남긴 수평 변색선 [추측임] | Slime 변형 |

### 배경 요소

| 레이어 | 요소 | 설명 |
|:-------|:-----|:-----|
| **Layer 2** | 터널 뒷벽 | 파이프가 관통하는 콘크리트/석재 벽. 슬라임 패턴 |
| **Layer 3** | 깊은 터널 암흑 | 끝이 보이지 않는 어두운 터널. 안개 대신 순흑 |
| **수중** | 수중 배경 | 수중에서는 팔레트가 청록-보라로 전환 [추측임] |

## 4. CC -- Chimney Canopy (굴뚝 지붕)

> **정체성:** 수직 굴뚝 구조물 상단. 하늘이 보이는 높은 곳.
> **내러티브:** "구조물의 꼭대기. 바람과 하늘에 노출된 영역"

### 팔레트

- **주조색:** 중간 회색-갈색, 따뜻한 황토 [추측임]

## 지역 간 전환 패턴 요약

Rain World의 지역 전환은 **점진적**이다. Gate(관문) 주변 방들에서 두 지역의 장식 요소가 혼합된다. [확인함 -- devlog]

### 전환 규칙

| 전환 축 | 변화 요소 | 예시 |
|:--------|:---------|:-----|
| **밝기** | 팔레트 명도 점진적 변화 | SU(밝음) -> HI(어두움) -> SH(순흑) |
| **식생 밀도** | Grower/Clinger 밀도 그래디언트 | HI(거의 없음) -> LF(최대) |
| **수분** | 물/슬라임 비율 증가 | SU(건조) -> DS(침수) |
| **구조물 상태** | 침식/부식 강도 변화 | SS(깨끗) -> SH(부식) -> SB(붕괴) |
| **개방감** | 하늘 노출 vs BlackGoo 밀폐 | SI(완전 개방) -> DS(완전 밀폐) |

## ECHORIS 적용 인사이트

### 1. 지역 정체성 = 장식 조합의 가중치

Rain World는 동일한 이펙트 도구를 사용하되 **밀도 비율**로 지역을 구분한다. ECHORIS의 Tier/테마도 동일한 장식 풀에서 가중치만 변경하면 된다.

| Rain World 패턴 | ECHORIS 대응 |
|:---------------|:------------|
| SU의 낮은 밀도 균일 장식 | Tier 1 (천공의 정원) -- 밝고 희소한 장식 |
| HI의 녹/체인/금속 집중 | Tier 4 (지하 수로) T-INFRA -- 산업 장식 최대 |
| DS의 슬라임/물/균류 집중 | Tier 4 수중 구간 -- 습기 장식 최대 |
| SH의 극단적 BlackGoo | Tier 7 (심연의 구) T-DEEP -- 어둠 최대 |
| LF의 식생 폭발 | Tier 1 T-ECOLOGY -- 과성장 |
| SS의 깨끗한 표면 | Tier 5 (연구소) T-RESEARCH -- 장식 최소 |
| SB의 고고학적 지층 | 아이템계 지층 깊이 변형 -- 욕망의 하강 |

### 2. 수직 전환 = Tier 전환

Rain World의 "고도에 따른 환경 변화"는 ECHORIS의 Tier 수직 배열과 직접 대응한다. 높은 곳(밝음, 개방, 식생) -> 낮은 곳(어둠, 밀폐, 부식)의 점진적 전환이 핵심.

### 3. 팔레트만으로 70%의 정체성을 만든다

Rain World의 12개 지역이 동일 타일셋을 공유하면서도 완전히 다른 세계로 느껴지는 이유의 70%는 팔레트이다. ECHORIS의 LUT 팔레트 스왑(DEC-022)이 이 원리를 구현한다.

### 4. "노출된 내장" 원칙의 보편 적용

모든 지역에서 구조물의 내부가 노출되어야 한다. 배관, 배선, 기어, 벽돌 -- 표면 아래의 구조가 항상 보이는 것이 Rain World의 핵심 미학이다. [확인함]

### 5. BlackGoo 강도가 공포 수준을 결정한다

SU(**)에서 SH(*****)까지, BlackGoo 강도가 증가할수록 플레이어의 불안감이 증가한다. ECHORIS의 Tier 7(심연의 구)에서 비네팅을 최대로 적용하는 것이 이 원리.
