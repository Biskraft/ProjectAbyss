## 1. Dead Cells 분석

### 1.1 Rune 시스템 개요

Dead Cells의 이동 능력 해금은 "룬(Rune)" 시스템으로 구현된다. 각 룬은 보스/엘리트를 처치해야 영구 해금되며, 이후 모든 런에서 해당 능력을 사용할 수 있다. 룬은 탐험 게이트 역할을 겸한다 — 룬 없이는 특정 바이옴 입구 자체에 접근할 수 없다.

**설계 핵심:** 룬은 순수한 탐험 해금 도구이다. 전투 무기/스킬과 완전히 분리된 영구 이동 능력으로 구현하여, 파워 인플레이션 없이 탐험 범위만 확장한다.

---

### 1.2 다이브 어택 (Dive Attack / Ground Pound)

#### 메커닉 사양

| 항목 | 내용 |
| :--- | :--- |
| **입력** | 공중에서 점프 버튼 + 아래 방향 (또는 아래 방향 홀드) |
| **발동 조건** | 공중 상태 (점프/낙하 중 언제든 가능) |
| **취소 조건** | 지면에 닿기 전까지 너무 오래 걸리면 자동 취소 |
| **기본 데미지** | 낙하 거리 소 (이중점프 높이 이내): 35 기본 데미지, 반경 1타일 |
| **기본 데미지** | 낙하 거리 대 (이중점프 높이 초과): 90 기본 데미지, 반경 2타일 |
| **판정 유형** | 근접 공격(Melee)으로 분류 — 근접 강화 돌연변이 효과 적용됨 |
| **낙하 가속** | 일반 낙하보다 빠른 하강 속도 |
| **낙하 스턴 제거** | 높은 곳에서 착지 시 발생하는 스턴이 다이브 어택으로 상쇄됨 |

#### Ram Rune 연동 (바닥 뚫기)

Ram Rune 획득 후 다이브 어택의 기능이 확장된다:

- **바닥 파괴:** 오렌지 룬 마킹이 된 바닥 타일 (Breakable Floor)을 뚫을 수 있다
- **잔해 데미지:** 파괴된 바닥 잔해가 아래 적에게 추락하며 추가 피해 적용
- **탐험 해금:** Ram Rune 없이는 접근 불가한 하부 구역 다수 존재
- **Ram Rune 획득처:** Ossuary의 엘리트 Slasher 처치 (Teleportation Rune 선행 필요)

#### 게임 필 (Game Feel) 요소

Dead Cells의 다이브 어택은 게임 전반의 "juicy" 철학이 집약된 동작이다:

- **화면 흔들림:** 착지 시 카메라 셰이크 적용 (강도: 낙하 거리에 비례)
- **파티클:** 착지 지점에서 먼지/파편 파티클 분출 (반경에 맞춘 원형 확산)
- **히트스톱:** 지면/적 충돌 순간 0.05~0.1초 히트스탑 (프리즈 프레임)
- **사운드:** 저주파 임팩트 사운드 + 지면 파괴 시 추가 SFX
- **시각적 충격파:** 착지 지점에서 원형 충격파 스프라이트 재생

#### 전투 유틸리티

다이브 어택은 탐험 도구이자 전투 기술로 이중 기능을 수행한다:

- 공중에서 직하강하며 적에게 광역 근접 피해 적용
- 높이가 있는 전투 구역에서 보스/엘리트에게 선딜 없이 즉시 피해 진입
- 낙하 거리가 길수록 데미지와 반경이 증가 — 높은 위치를 활용하는 플레이를 장려

### 1.3 Dead Cells 전체 Rune 목록 (수직/탐험 관련)

| 룬 이름 | 능력 | 탐험 해금 | 획득처 |
| :--- | :--- | :--- | :--- |
| **Vine Rune** | 지면의 녹색 새싹과 상호작용 시 덩굴이 위로 자라남 (등반 가능) | Toxic Sewers 등 상부 바이옴 입구 | Promenade of the Condemned - 엘리트 Undead Archer |
| **Ram Rune** | 다이브 어택으로 룬 마킹 바닥 파괴, 하부 구역 접근 | 다수의 하부 지하 구역 | Ossuary - 엘리트 Slasher |
| **Spider Rune** | 수직 벽과 천장을 자유롭게 등반 (무한 벽 등반 + 천장 이동) | 벽면/천장에 위치한 숨겨진 구역 | Slumbering Sanctuary 보스 처치 후 |
| **Teleportation Rune** | 지도상의 핑크/보라색 석관과 상호작용, 멀리 떨어진 밀봉 구역으로 순간이동 | 물리적으로 접근 불가한 밀봉 구역 | Toxic Sewers - 엘리트 Caster |
| **Homunculus Rune** | 머리를 분리해 독립 조종. 머리는 벽을 무한 등반하고, 적에게 달라붙어 스턴 적용 | 몸이 들어갈 수 없는 협소 구역 | Hand of the King 처치 후 자동 획득 |

#### 수직 이동 메커닉 매핑

```
상향 이동:   Vine Rune (덩굴 등반) → Spider Rune (벽/천장 등반)
하향 이동:   Ram Rune (바닥 관통) → Dive Attack (낙하 가속 + 데미지)
수평+수직:  Teleportation Rune (공간 초월 이동)
협소 통과:   Homunculus Rune (분리 머리 조종)
```

### 1.4 Dead Cells 설계 인사이트

**인사이트 1 — 탐험 능력과 전투 능력의 분리**
룬은 전투력을 높이지 않는다. 순수하게 탐험 범위만 확장한다. 이로써 파워 게이트(강해야 열림)가 아닌 능력 게이트(할 수 있어야 열림)를 구현한다.

**인사이트 2 — 다이브 어택의 이중 가치**
다이브 어택은 "재미있는 이동 기술"이면서 동시에 "탐험 열쇠"이다. 바닥을 뚫는다는 물리적 행위가 직관적이며, 발견의 쾌감(숨겨진 하부 구역)과 전투 쾌감을 동시에 제공한다.

**인사이트 3 — 낙하 거리 기반 데미지 스케일링**
높이를 활용하는 플레이를 수치로 보상한다. "높이 올라가서 뛰어내리면 더 강하다"는 직관적 규칙이 수직 맵 탐험의 동기를 강화한다.

**인사이트 4 — 룬의 선행 의존 구조**
Ram Rune을 얻으려면 Teleportation Rune이 필요하고, Teleportation Rune을 얻으려면 Vine Rune이 필요하다. 이 선행 체인이 탐험 순서를 자연스럽게 유도한다.

## 2. 캐슬바니아: 월하의 야상곡 (SotN) 분석

### 2.1 SotN 릴릭 시스템 개요

SotN의 릴릭(Relic)은 특정 보스 처치 또는 탐험 중 발견으로 획득하는 영구 능력 아이템이다. 슬롯에 장착해야 효과가 활성화되며, 여러 릴릭을 조합해 추가 효과(조합 기술)를 얻을 수 있다. 이 구조가 Project Abyss 릴릭 시스템의 직접적 원형이다.

---

### 2.2 수직 이동 관련 릴릭 전체 목록

#### Leap Stone (도약석) — 이단 점프

| 항목 | 내용 |
| :--- | :--- |
| **효과** | 공중에서 추가 점프 1회 (이단 점프) |
| **입력** | 공중 상태에서 점프 버튼 재입력 |
| **획득처** | Chapel of Dissonance (코롤라스 이전 구역) |
| **탐험 해금** | 이단 점프 높이로 도달할 수 있는 다수의 상부 플랫폼 및 비밀 방 |
| **체인 가능성** | Gravity Boots 조합 시 더 높은 연속 상승 가능 |

**설계 포인트:** SotN의 이단 점프는 SotN 이전 캐슬바니아 시리즈에 없던 혁신적 추가였다. 이 하나의 릴릭이 성의 수직 탐험 가능성을 근본적으로 바꿨다.

---

#### Gravity Boots (중력의 장화) — 고점프 / 초점프

| 항목 | 내용 |
| :--- | :--- |
| **효과** | 지면에서 대시 + 점프로 일반의 3~4배 높이의 초점프 실행. 천장에 닿으면 잠깐 달라붙음 |
| **입력** | 아래 → 위 + 점프 순서 입력 (방향키 조합) |
| **연속 사용** | 연속 초점프 입력으로 무한 상승 가능 (닿을 천장만 있다면) |
| **획득처** | Olrox's Quarters |
| **탐험 해금** | 일반 점프/이단 점프로 도달 불가한 고도의 구역, 성 상부 비밀 통로 |
| **체인 가능성** | 초점프 → 벽 타기(중간 착지)로 더욱 높은 고도 도달 가능 |

**설계 포인트:** Gravity Boots는 SotN의 "수직 탐험 극대화" 릴릭이다. 연속 초점프로 수직 통로를 고속 상승하는 플레이는 게임에서 가장 직관적으로 쾌감이 느껴지는 이동 경험 중 하나로 꼽힌다.

---

#### Soul of Bat (박쥐의 영혼) — 박쥐 변신 (자유 비행)

| 항목 | 내용 |
| :--- | :--- |
| **효과** | 박쥐로 변신. 자유 비행 (모든 방향 이동). MP 점진 소비 |
| **입력** | 릴릭 슬롯 활성화 토글 |
| **선행 조건** | Leap Stone + Soul of Mist 획득 필수 |
| **획득처** | Long Library 하부 (Lesser Demon 보스 처치 후, Mist Form으로 쇠창살 통과 후) |
| **탐험 해금** | 성의 거의 모든 영역이 개방됨. "Soul of Bat 획득 후 성이 완전히 열린다"고 공식 가이드에서 명시 |
| **조합 기술** | Wing Smash: 박쥐 상태에서 돌진 공격. 명령어 조합(점프 홀드 + 방향키 순환 + 점프 해제)으로 발동 |

**Wing Smash 상세:**
- 박쥐 형태로 화면 가로를 고속 관통하는 돌진 공격
- 이동 중 적 전체에 타격
- 천장이나 지형을 따라 이동하면 상향 벡터를 얻어 수직 상승 가능
- MP 소비 (소량)
- PS1 버전: 일정 시간 후 자동 해제 / Saturn 버전: 지형 충돌 또는 MP 소진까지 지속
- 스피드런에서 "Infinite Wing Smash" 테크닉으로 맵 전체를 고속 비행 가능

#### Soul of Mist (안개의 영혼) — 미스트 변신 (장애물 통과)

### 2.4 SotN 설계 인사이트

**인사이트 1 — 능력 연쇄(Ability Chaining)의 설계 미덕**
각 릴릭이 독립적으로 유용하면서도, 선행 릴릭 없이는 다음 릴릭을 획득할 수 없는 구조가 탐험의 내러티브를 만든다. 플레이어는 릴릭 획득 순서를 통해 자연스럽게 스토리를 경험한다.

**인사이트 2 — 변신(Form Change)의 이동 메타포**
SotN의 변신 능력은 단순한 "더 높이 뛴다"가 아니라 "다른 존재가 된다"는 경험을 준다. 박쥐가 되어 날고, 안개가 되어 통과하는 경험은 능력 게이트를 해금했을 때의 쾌감을 전통적 이동 능력보다 훨씬 크게 만든다. 이것이 메타포로서의 능력 설계다.

**인사이트 3 — 비행(자유 비행)은 게임의 두 번째 절반을 여는 열쇠**
Soul of Bat 획득은 사실상 SotN의 중간 지점이다. 이후 역전된 성(Inverted Castle)까지 포함하면, 비행 능력이 게임 경험의 절반 이상을 담당한다. 이 능력 하나가 게임을 재정의한다.

**인사이트 4 — 슬롯 교환의 전략적 선택**
릴릭은 슬롯에 장착/해제를 플레이어가 선택한다. 박쥐 변신 슬롯을 다른 릴릭으로 교체하면 비행을 포기한다. 이 교환 비용이 빌드 전략 요소를 만든다. 단순한 능력 잠금 해제가 아니라 지속적인 선택 구조를 만든다.

## 3. 기타 메트로베니아 수직 이동 능력 분석

### 3.1 Hollow Knight

| 능력 | 메커닉 | 수직 탐험 해금 | 특이점 |
| :--- | :--- | :--- | :--- |
| **Mantis Claw** | 공중에서 벽에 가까이 붙어 해당 방향 입력 시 매달림. 벽에서 점프로 대각선 이탈 | 벽이 있는 수직 통로 상승, 다수의 수직 비밀 통로 | 벽 매달림이 공중 대시와 이단 점프를 리셋함 — 복합 이동의 핵심 허브 |

| **Shade Cloak** | 대시에 무적 판정 추가. 쉐이드 게이트(보라색 장벽)를 통과 가능 | 쉐이드 게이트로 봉인된 구역 (지도상 분산 배치) | 능력 게이트의 시각적 명시화 사례 — 보라색 장벽이 "여기는 Shade Cloak 없으면 못 간다"를 직관적으로 전달 |

**Hollow Knight 설계 인사이트:**
- 능력 조합(Combo)이 핵심이다. 개별 능력보다 "Mantis Claw + Monarch Wings + Mothwing Cloak" 같은 조합이 이동의 진정한 깊이를 만든다
- 벽 매달림이 공중 자원(대시, 이단 점프)을 리셋하는 규칙이 수직 이동의 리듬을 만든다 — 벽이 "재충전 지점"이 됨
- 능력 게이트의 시각화: Vine Wall(덩굴 벽), Shade Gate(보라 장벽), 가시 구역 등 각 게이트에 시각적 식별자를 부여해 플레이어가 "나중에 돌아오면 열 수 있다"를 직관적으로 인지

**Ori 설계 인사이트:**
- Bash는 메트로베니아 역사상 가장 창의적인 이동 능력 중 하나로 평가받는다. "위험 요소를 이동 수단으로"라는 반전이 핵심
- 모든 능력이 "여러 용도를 갖는다": Bash는 이동/전투/퍼즐 해결에 모두 사용. 이것이 능력 게이트 설계의 이상적 상태 ("능력당 최소 3가지 용도")

### 3.3 Metroid 시리즈

| 능력 | 게임 | 메커닉 | 탐험 해금 |
| :--- | :--- | :--- | :--- |

**Metroid 설계 인사이트:**
- Space Jump는 "이동의 완성"이다. 이 능력을 얻으면 수직 이동의 모든 제약이 사라진다. 따라서 게임 후반부에 배치되며, 이 능력 이후의 게임은 수직 이동이 아닌 다른 종류의 게이트(환경 저항, 특수 형태 등)에 의존한다
- Wall Jump가 공식 튜토리얼 없이 "발견 가능한 테크닉"으로 설계된 것은 Explorer 타입 플레이어에게 극적인 "발견의 쾌감"을 제공한다. Project Abyss의 숨겨진 이동 테크닉 설계에 참고할 수 있다

## 4. Project Abyss 적용 제안 — 릴릭 후보 목록

### 4.0 설계 전제 (Design Premises)

아래 제안은 다음 원칙을 기반으로 한다:

1. **현재 기본 이동 능력:** 대시, 벽 점프, 이단 점프 (System_3C_Character.md)
2. **3대 핵심 판타지:** 탐험가 / 장인 / 모험가
3. **수직 세계관:** 에르다는 거대한 수직 메가스트럭처를 하강하며 탐험
4. **능력 게이트 원칙:** 릴릭은 전투력이 아닌 탐험 범위를 확장 (Dead Cells Rune 모델)
5. **메타포 우선:** 단순한 "더 높이 뛰기"보다 "다른 존재가 된다" (SotN 변신 모델)
6. **릴릭당 최소 3가지 용도:** 이동 / 탐험 게이트 해금 / 전투 유틸리티 (Ori Bash 모델)

| **입력** | 지면의 특수 타일(기억 토양, Memory Soil)에 상호작용 → 덩굴 상승 소환. 덩굴을 따라 점프키로 등반 |
| **이동 효과** | 소환된 덩굴은 5~15타일 높이로 상승. 플레이어가 덩굴을 잡고 올라가며 최상단에서 도약 |
| **탐험 해금** | 기억 토양 타일 위치에서만 소환 가능. 해당 위치는 레벨 디자인으로 명시적으로 배치 (Dead Cells의 녹색 새싹처럼 시각적 식별자 부여) |
| **게임 필** | 덩굴이 자라나는 애니메이션 (프레임별 성장). 잎사귀 파티클. 수직 상승 시 풍압 사운드 |
| **구현 난이도** | ★★☆☆☆ (특수 타일 판정, 덩굴 오브젝트 소환 및 충돌 등록) |
| **세계관 적용** | "기억의 지층" 개념과 연결 — 특정 지점의 기억을 불러내 물리적 경로로 구현 |

### 4.2 릴릭 탐험 게이트 매핑

```
  [R-V05] 속박의 쇄도 ─── 수직 공백(Vertical Gap) → 고속 직상승

수평 + 수직 복합:
  [R-V02] 심연의 날개 ─── 협곡(Chasm) → 안전 활공 횡단

특수 환경 통과:
  [R-V03] 기억의 안개 ─── 안개 게이트(Mist Gate) → 특수 장벽 통과
```

### 5.1 게임별 상승 메커닉 상세 분석

#### Hollow Knight — Mantis Claw (벽 매달림 + 벽 점프)

**벽 점프(Wall Jump)와 벽 매달림(Wall Cling)의 설계적 차이:**

기존 벽 점프(Wall Jump)는 벽에 닿는 순간 즉시 반대 방향으로 점프하는 반응형 입력이다. 플레이어는 벽에 "머무는" 시간이 없다. 반면 Mantis Claw(만티스 발톱)는 다음 구조로 작동한다:

| 구분 | 일반 벽 점프 | Mantis Claw (Hollow Knight) |
| :--- | :--- | :--- |
| **발동** | 벽에 닿는 순간 점프 입력 | 공중에서 벽 방향 입력 → 자동 매달림 |
| **머무는 시간** | 0 (즉각 반응) | 무제한 (슬라이딩 하강하며 대기) |
| **탈출 방향** | 반드시 벽 반대 방향 | 자유 (벽에서 멀어지는 대각선 도약) |
| **공중 자원 리셋** | 없음 | 있음 — 매달림 시 대시+이단점프 재충전 |
| **연속 가능성** | 동일 벽 연속 사용 불가 | 동일 벽 좌우 왕복 무한 상승 가능 |

**핵심 설계 포인트 — 벽이 "재충전 지점"이 된다:**
Mantis Claw의 설계 핵심은 벽에 매달리는 순간 공중 자원(대시, 이단 점프)이 리셋된다는 점이다. 이로써 "벽 → 공중 이단 점프 → 벽 → 공중 이단 점프" 사이클이 가능해져, 수직 통로를 무한 상승할 수 있다. 벽이 단순한 장애물이 아니라 이동 시스템의 재충전 인프라가 된다.

**탐험 게이트 시각화:**
Hollow Knight는 Shade Gate(보라색 장벽), 가시 지형, 특정 점프 높이를 요구하는 플랫폼 배치 등으로 능력 게이트를 명시한다. 벽 매달림이 필요한 수직 구역은 맞닿은 두 벽이 근거리에 배치되어, "벽이 두 개 있으니 여기서 올라가는 거구나"를 시각적으로 암시한다.

#### Celeste — 상승 메커닉 전체 스펙트럼

Celeste는 단일 대시 메커닉에서 파생되는 상승 기술의 층위가 가장 세밀하게 설계된 게임이다.

**기본 대시 (Basic Dash):**
- 8방향 대시. 위 방향 포함 (수직 상승 직접 가능)
- 지면 착지 또는 특정 오브젝트 터치 시 대시 리셋
- 대시 1회가 공중에서 소모되면 착지 전까지 재사용 불가

**월바운스 (Wallbounce) — 상승의 핵심 기술:**
- **입력:** 벽에 인접한 상태에서 위 방향 대시 → 대시 도중 아무 타이밍에 점프
- **효과:** 점프를 대시 종료 직전에 할수록 더 높은 상승 속도를 얻음
- **수치:** 월바운스 시 수평 속도 170, 수직 속도 160 (일반 점프보다 현저히 높음)
- **설계 의도:** 챕터 7 B-Side와 챕터 9에서 공식적으로 가르쳐지며, C-Side 클리어의 필수 기술
- **상승 질:** 일반 대시나 벽 점프보다 수직 이동 속도가 빠름 — 수직 통로를 "폭발적으로" 오르는 느낌 제공

**슈퍼/하이퍼 대시 (Super/Hyper Dash) — 간접 수직 이득:**
- 하이퍼: 지면에서 아래 대각선 대시 후 즉시 점프 → 수평 거리가 극적으로 증가
- 슈퍼: 지면 대시 후 점프 → 장거리 점프
- 이 기술들은 직접 상승이 아니지만, 수평 운동량을 활용해 더 멀리 있는 수직 구조에 닿을 수 있게 함

**벽 등반 (Wall Climb) — 스태미나 소모형:**
- 벽에 달라붙어 방향키 위로 직접 등반 가능
- **스태미나 게이지** 존재: 제한 시간 내에만 등반 가능, 소진 시 자동 낙하
- 스태미나는 지면 착지 시 즉시 회복
- **설계 의도:** 무한 등반이 아닌 리소스 관리형 상승. 압박감(스태미나 소진 경고)과 성취감(시간 내 도달)을 동시에 제공

**Celeste 상승 설계의 핵심 철학 — "Forgiveness (용서)":**
개발자 Maddy Thorson이 직접 서술한 바에 따르면, Celeste의 이동 시스템은 "용서" 원칙으로 설계되었다. 위 방향 점프 시 소모된 스태미나를 부분 환급하고, 벽 점프 타이밍 창을 실제 벽에서 벗어난 이후까지 연장하는 등의 "관대한 판정"이 적용된다. 이는 상승이 어려우면서도 좌절감이 최소화되도록 설계된 것이다.

#### Mega Man X — 벽 킥 + 대시 점프 조합

**벽 킥 (Wall Kick) 기본 구조:**
- 벽에 점프하며 부딪히면 느린 슬라이딩 하강 시작
- 하강 중 점프 입력 시 반대 방향으로 대각선 도약
- 같은 벽과 반대 벽을 왕복하며 좁은 수직 통로를 상승 가능

**대시 벽 킥 (Dash Wall Kick) — 가속 상승:**
- 벽 킥 + 동시 대시 입력 → 일반 벽 킥보다 훨씬 빠른 속도와 거리로 이탈
- 타이밍이 정밀해야 하지만, 숙달 시 수직 통로를 매우 빠르게 오를 수 있음
- Parry Everything 분석에 따르면, 이 두 기술이 "단순하지만 극도로 강력하고 역동적"으로 설계된 것은 의도적이며, 거대한 활용 가능성 공간(possibility space)을 열기 위함이었다

**대시 점프 (Dash Jump):**
- 지면에서 대시 중 점프 → 일반 점프보다 높은 최고 고도 달성
- 대시의 수평 운동량이 점프 높이에 더해짐 (대각선 탄도)
- 수직 게이트를 해금하지 않아도 높은 위치에 달하게 하는 숙련 테크닉

**Mega Man X 상승 설계 인사이트:**
X의 벽 킥은 게임 내 어떤 튜토리얼도 없이 "발견 가능한 테크닉"으로 설계되었다. Metroid의 Wall Jump와 동일한 접근. 이것은 Bartle의 Explorer 타입 플레이어에게 "나만 아는 기술"을 발견했을 때의 극적인 보상을 제공한다. 능력 획득 없이 이동 기술 숙련만으로 상승 가능성이 열리는 설계다.

#### Ori and the Blind Forest / Will of the Wisps — Bash와 Launch

**Bash (Blind Forest) — 역방향 발사 상승:**
- **입력:** 적/발사체/등불에 근접 시 Bash 버튼 → 짧은 시간 정지 + 방향 조준 → 방향 선택 → 적은 선택 방향으로, Ori는 반대 방향으로 고속 발사
- **상승 활용:** 등불이나 공중에 있는 적 위에서 Bash를 위 방향으로 적용 시 Ori가 고속 상승
- **연속 Bash:** 공중에 여러 적/등불이 배치된 구간에서 연속 Bash로 무한 고도 획득 가능
- **설계 반전:** "위험 요소(적)"가 "이동 수단(발판)"으로 전환. 위험 회피에서 위험 활용으로 플레이어 사고방식을 역전시킴
- **시간 정지 효과:** Bash 조준 중 시간이 일시 정지되어 플레이어가 충분히 생각하고 방향을 선택할 수 있음. 이것이 고속 이동 중에도 Bash를 전략적으로 사용할 수 있게 하는 핵심

**Launch (Will of the Wisps) — 충전 후 방향 발사:**
- 에너지를 모아 임의 방향으로 고속 발사
- 약한 벽 파괴 가능 (파괴형 이동 능력)
- 상승 방향으로 발사 시 Bash와 유사한 수직 상승 제공

**Burrow (Will of the Wisps) — 매질 전환형 상승:**
- 모래/흙 지형을 수중처럼 이동
- 수직 모래층에서 위로 이동 후 도약 시 공중으로 발사됨
- **설계 핵심:** 지형 유형 자체가 새로운 이동 매체가 됨. 상승을 "점프"가 아닌 "매질에서 탈출"로 재정의

**Ori 설계 인사이트 — 다목적 능력의 이상:**
Bash는 이동 / 전투 / 퍼즐 해결 세 가지 용도를 모두 갖는다. 이것이 능력 게이트 설계의 이상 상태다. Ori의 공식 분석에 따르면, Bash는 "게임 플레이 방식을 완전히 재정의"한다고 평가된다 — 하나의 능력이 세계를 보는 방식 자체를 바꾸기 때문이다.

#### Castlevania SotN — Gravity Boots 연속 초점프 분석

**Gravity Boots 연속 상승 시퀀스:**
- **입력:** ↓ → ↑ + 점프 (방향키 조합)
- **MP 소비:** 초점프 1회당 4 MP
- **공중 연속 사용:** Leap Stone(이단 점프 릴릭) 보유 시, 공중에서도 추가 초점프 발동 가능
- **연속 상승 조건:** Leap Stone의 이단 점프를 먼저 소비한 후 Gravity Boots 초점프 발동
- **실질 효과:** MP가 허락하는 한 수직 통로를 연속 초점프로 무한 상승 가능
- **천장 상호작용:** 초점프로 천장에 닿으면 Alucard가 뒤집혀 천장에 잠깐 달라붙음 (협소 공간에서 천장 충돌로 정지)

**SotN 초점프의 상승 게임 필:**
연속 초점프는 SotN에서 가장 빠른 수직 이동 수단이다. 수직 통로를 "연속 초점프로 폭발적으로 상승"하는 경험은 월하의 야상곡 전체에서 가장 직관적인 이동 쾌감으로 평가된다. 이는 수직 이동이 MP라는 자원과 직접 연동되기 때문에, 상승이 리소스 관리의 전략적 결정이 되기도 한다.

#### Dead Cells — 상승 특화 분석

Dead Cells는 기본적으로 수평 이동 중심 설계이지만, 상승 전용 룬이 명확하게 존재한다:

**Vine Rune (덩굴 룬) — 소환형 상승 발판:**
- 지면의 녹색 새싹(시각적 식별자)과 상호작용 → 덩굴이 위로 성장 → 등반 가능
- 덩굴은 상호작용 후 즉시 성장, 플레이어가 잡고 올라감
- **설계 포인트:** 상승 자체가 즉각적이고 화려하진 않지만, 덩굴의 성장 애니메이션이 시각적으로 명확한 "여기로 올라가시오" 신호를 제공함

**Spider Rune (거미 룬) — 벽/천장 전면 등반:**
- 수직 벽과 천장을 자유롭게 등반 (무한 벽 등반 + 천장 이동 가능)
- 벽에서 점프하지 않고 직접 기어오름
- Mantis Claw(Hollow Knight)와 달리 공중 자원 리셋 없음 — 순수한 "벽을 오르는" 행동
- **상승 질:** 빠르지 않지만 확실함. 어떤 수직면도 오를 수 있다는 절대적 안정감

### 5.2 "느낌이 좋은" 상승의 공통 요소 — 게임 필 분석

리서치한 게임들에서 상승이 만족스럽게 느껴지는 공통 설계 원칙을 추출한다.

#### 원칙 1 — 상승 가속의 시각화 (Upward Acceleration Visualization)

모든 성공적인 상승 메커닉은 "가속이 시작된다"는 시각적 순간을 명확히 전달한다:
- **Ori Bash:** 타임 슬로우 → 방향 선택 → 폭발적 발사 (준비-실행의 구분)
- **SotN Gravity Boots:** 점프 전 낮은 자세(쭈그림) → 수직 폭발 (예비 동작)
- **Celeste Wallbounce:** 대시 중 벽 접촉 → 폭발적 대각 발사 (운동량 전환)
- **Mega Man X Dash Jump:** 대시 → 도약 (수평 운동량의 수직 전환)

공통점: 상승 전 "예비 동작" 또는 "준비 상태"가 존재하며, 이것이 실제 상승의 임팩트를 증폭시킨다.

#### 원칙 2 — 정점에서의 체공 (Apex Hang)

수직 이동이 만족스럽게 느껴지는 핵심 요소는 상승 정점에서 중력이 일시적으로 약해지는 "체공 구간"이다:
- 점프 정점에서 0.1~0.3초간 중력 감소 → 플레이어가 "최고점에 있다"는 감각을 충분히 느낌
- 이 체공이 없으면 상승이 순식간에 끝나 "뭔가 올라간 것 같기도 하고..."의 불명확한 감각 발생
- 게임 파일 분석에 따르면 Celeste는 점프 정점에서 의도적으로 중력을 감소시키는 변수를 적용함

#### 원칙 3 — 상승 후 안착의 선택권 (Post-Apex Control)

좋은 상승 메커닉은 정점 이후 플레이어에게 충분한 공중 제어권을 준다:
- **Hollow Knight:** 벽 매달림이 이단 점프를 리셋 → 정점 이후에도 다음 이동을 계획 가능
- **Celeste:** 대시를 정점 이후에도 사용 가능 → 상승 후 상황을 즉시 대응
- **Ori:** Bash는 공중에서 다시 Bash 가능 → 연속 상승이 플레이어 판단에 전적으로 의존

공중 제어권이 없으면 상승이 "타는 것(passive)"처럼 느껴진다. 제어권이 있으면 "조종하는 것(active)"처럼 느껴진다.

#### 원칙 4 — 상승과 탐험 게이트의 연동 (Ascent-Gate Coupling)

만족스러운 상승 메커닉은 항상 해당 메커닉만으로 열리는 탐험 게이트와 직접 연동된다:
- Dead Cells Vine Rune → 녹색 새싹이 있는 구역만 열림
- Hollow Knight Mantis Claw → 두 벽이 근거리에 배치된 수직 통로
- SotN Gravity Boots → 일반 점프로 닿지 않는 높이의 플랫폼

게이트가 없으면 상승은 "더 빠른 이동"에 그친다. 게이트가 있으면 상승은 "새로운 세계를 여는 열쇠"가 된다.

#### 원칙 5 — 반응형 피드백의 즉각성 (Immediate Reactive Feedback)

0.5초 이내 피드백 원칙(Csikszentmihalyi 플로우 모델)이 상승에서 특히 중요하다:
- 상승 입력 후 0~0.1초: 예비 애니메이션 또는 파티클 발생 (입력 확인)
- 0.1~0.5초: 실제 상승 이동 시작 (입력이 반응했음을 명확히 인지)
- 0.5초 이후: 상승의 결과물 (새로운 위치, 탐험 가능한 구역) 확인

이 타임라인이 깨지면 "입력이 먹혔는지 안 먹혔는지 모르겠다"는 혼란 발생.

#### 원칙 6 — 상승의 의외성/반전 (Ascent Surprise Factor)

가장 기억에 남는 상승 메커닉은 모두 "이렇게 올라가는 거였어?"라는 반전을 포함한다:
- **Ori Bash:** 적을 발판으로 쓴다 (위험 → 도구의 반전)
- **Mega Man X 벽 킥:** 튜토리얼 없이 발견하는 숨겨진 테크닉
- **Celeste Wallbounce:** 대시 중에 점프를 눌러서 더 높이 오른다 (복합 입력의 시너지)
- **SotN Soul of Bat Wing Smash:** 천장을 따라 이동하며 수직 벡터 획득

이 "반전"은 SDT의 Competence(능숙감) 욕구를 강력하게 충족시킨다 — "나는 이 게임을 이해했다"는 숙달의 쾌감.

| **탐험 게이트** | 공명 게이트(Resonance Gate): 특정 높이의 플랫폼이나 절벽은 최대 스택 초점프로만 닿을 수 있음. 게이트 근처에는 연속 착지를 유도하는 낮은 발판들이 배치 (스택 축적을 유도하는 레벨 설계) |
| **게임 필** | 스택 축적: 착지마다 발 아래에서 원형 파문 이펙트 + 스택 수에 따라 에르다 발광 강도 증가. 초점프 충전: 진동 사운드 + 지면에서 에너지 균열 이펙트. 초점프 발사: 강한 하향 충격파 + 에르다 위에서 상향 방향 빛 기둥 |
| **구현 난이도** | ★★★☆☆ (스택 카운터, 스택별 점프 높이 계수, 충전 상태 머신, 특수 초점프 발동) |
| **3가지 용도** | 1) 스택 축적으로 수직 상승 도달 범위 증가, 2) 공명 게이트 탐험 해금, 3) 최대 스택 초점프로 전투 진입/이탈 |

### 5.6 상승 이동 설계의 핵심 질문에 대한 답

> **"점프/더블점프/벽점프 외에, 상승을 '특별하게' 만드는 메커닉은 무엇인가?"**

리서치 결과 다음 세 가지 설계 원리가 공통적으로 도출된다:

**1. 위험/장애를 도구로 역전 (Risk-to-Tool Conversion)**
- Ori의 Bash: 적(위험) → 발판(도구)
- Mega Man X 벽 킥: 벽(장애물) → 이동 레일(도구)
- Project Abyss 적용: [R-A03] 심연의 반동 — 심연의 적이 상승 수단

**2. 리소스와 상승의 연동 (Resource-Ascent Coupling)**
- SotN Gravity Boots: MP가 곧 상승 거리
- Celeste 벽 등반: 스태미나가 곧 등반 한계
- Project Abyss 적용: [R-A05] 부유의 원환 — 원환 게이지가 곧 체공 시간

**3. 이동이 세계를 재해석하게 만든다 (World Reinterpretation)**
- Soul of Bat: 성 전체가 비행 경로로 재해석됨
- Mantis Claw: 벽 전체가 수직 이동 레일로 재해석됨
- Project Abyss 적용: [R-A02] 기억의 메아리 — 방문했던 모든 위치가 잠재적 귀환점으로 재해석됨

이 세 원리 중 하나 이상을 충족하는 상승 릴릭만이 "특별한" 상승 경험을 제공한다.
