## 1. 핵심 구조

### 층 구조
- **레어리티별 층수:** Common 30층 / Rare 60층 / Legendary 100층 (D1~D4)
- **D5:** 무한 층으로 전환 → **D7:** 30층 + 아이템 전생 반복 루프로 복귀
- **10층마다 보스:** 장군(10,20,40,50,70,80F) → 왕(30,60,90F) → 신(100F)
- 보스 클리어 후 **탈출 vs 계속** 선택 = 세션의 핵심 결정점

### 지오 패널/심볼
- 색깔 타일(패널) + 효과 크리스탈(심볼) 조합으로 **전장 규칙 자체를 물리적으로 배치**
- ATK+50%, 무적, 진입불가, 워프, 즉사 등 30종+ 이펙트
- **지오 체인:** 심볼을 다른 색 패널 위에서 파괴 → 연쇄 폭발 → 대량 보너스 게이지

### 미스터리 룸 (D2~)
- 비보스층에서 ~10~20% 확률로 출현하는 특수 이벤트 방
- 점술사(레벨 ±3~10), 전투방(2턴 클리어 시 +3레벨), 보물방, 상점, 병원 등 20종+
- **"한 층만 더" 심리를 자극하는 서프라이즈 경제학**

### 시리즈 진화 핵심 흐름

```
D1(2003): 원형 확립
→ D2(2006): 미스터리룸 + 해적 + 이노센트타운
→ D3(2008): 레벨구슬 + 역해적
→ D4(2011): 분기루트 + 킬보너스
→ D5(2015): 무한층 + 이노센트팜
→ D6(2021): 오토배틀 x32
→ D7(2023): 30층 + 아이템전생 반복
```

## 2. 이노센트 시스템

### 핵심 메커니즘
- 아이템 내부에 거주하는 NPC → 스탯 보너스 부여
- **야생(Unsubdued):** 반감 효과, 이동 불가, 아이템계에서 적으로 등장
- **복종(Subdued):** 격파 시 레벨 **2배**(D5 헌터팀: **4배**), 이동/합성 가능

### 주요 이노센트 분류

| 카테고리 | 대표 예시 | 소프트 캡 |
|:---------|:---------|:---------|
| 스탯 단일형 (8종) | Gladiator(ATK), Sentry(DEF) 등 | D5: 50,000 |
| 듀얼 스탯형 (7종) | Muscleman(HP+ATK) 등 | 50,000 |
| 특수 획득형 | Statistician(EXP +900%), Broker(HL +300%) | 종류별 상이 |
| 상태이상 부여/내성형 | 독, 수면, 마비 등 각각 | 100 |
| 유니크 이노센트 (40종+) | Combo Maker(통상공격 2회), Overclocker(비행이동) | 고정 효과 |

### 슬롯 제한 = 빌드 설계의 핵심

| 등급 | D2~D5 슬롯 | Project Abyss 기획 |
|:-----|:-----------|:-------------------|
| Common | 4 | 2 |
| Rare | 5 | 4 |
| Legendary | 6 | 6 |
| Ancient | - | 8 |

### 이노센트 팜 (D5~)
- Mediator 방식(D2~D4) → **자동 농장 시스템**으로 대체
- 전투 1회마다 1~7% 확률로 레벨 +1, 자동 번식으로 듀얼 스탯 이노센트 생성
- 능동적 전투 + 수동적 성장의 이중 보상 구조

## 3. 수치/성장 경제

### 스탯 성장 공식

| 시리즈 | 공식 | 최대 배율 |
|:-------|:-----|:---------|
| D1 | 기본스탯 × (1 + Lv × 0.05) | **x6.0** (100층) |
| D4 | 기본스탯 × (1 + Lv × 0.066) + 킬보너스400 | **x24.8** |
| D5 | 기본스탯 × (1 + 레어리티값/100) | **x2.0** (레어리티 기반) |

### 보스 처치 스탯 보너스

```
D1: 장군 = 기본스탯 × (0.1 + 아이템레벨/200)
    왕 = 장군 × 2, 신 = 장군 × 3
D5: 장군 레어리티+1, 왕 +2, 신 +3 (레어리티가 곧 스탯 배율)
```

### 엔드게임 야리코미 루프

```
캐릭터 전생 → 적성치 +5%
→ 아이템계 진입 → 층 클리어(스탯 성장)
→ 이노센트 복종(x2) → 보스 처치(영구 스탯+)
→ 수육계 도전 → 랭크 40 장비 훔치기
→ 새 장비 아이템계 재진입 → 반복
```

### 핵심 설계 원리
- **전 시리즈 순수 선형 성장** (지수 폭발 없음, 예측 가능)
- 이노센트 보너스는 장비 배율과 **독립적으로 합산** → 레어리티 격차 부분 극복 가능
- Mr. Gency's Exit: 10층마다 1개 자동 충전 → **신중한 플레이 시 자급자족**

## 4. 절차적 생성/맵

### 생성 방식 비교

| 항목 | 디스가이아 | 스펠렁키 | 데드셀 |
|:-----|:---------|:--------|:------|
| 방식 | 완전 알고리즘 랜덤 | Template+Critical Path | Concept Graph+Room Pool |
| 경로 보장 | 사후 검증 (약한 보장) | **수학적 완전 보장** | 검증 후 패치 |
| 핸드크래프트 | 없음 | Room Template+Chunk | 바이옴당 50~100+ 프리팹 |
| 전략 레이어 | 지오 패널/심볼 | 함정/장애물 | Rune 게이트 |

### Project Abyss 권장 접근

> **데드셀식 Concept Graph + 스펠렁키식 Critical Path 보장의 하이브리드**
> - 지층별 테마에 맞는 Room Pool로 "핸드메이드 느낌"
> - Critical Path 사전 확정으로 "Always Winnable" 수학적 보장
> - 지오 패널 대응물로 "환경 효과 구역"(중력 역전, 속성 변환 등)

### 아이템계만의 고유 차별점
1. **육성형 모델:** 레벨이 아이템 내부 → 탐험 대상 = 성장 대상
2. **전장 규칙의 물리적 조작:** 지오 심볼 이동/파괴로 규칙 자체를 변경
3. **비소모성 축적:** 죽어도 보스 처치 보너스 보존 → 수백 시간 리텐션의 핵심

## 5. 플레이어 경험/멀티플레이

### 가장 중요한 발견

> **디스가이아는 20년간 한 번도 실시간 협동 아이템계를 구현하지 않았다.**
> 이것이 Project Abyss가 채울 수 있는 **최대의 미충족 수요**이다.

### 시리즈 QoL 교훈

| 실패 사례 | 교훈 |
|:---------|:-----|
| D5 무한 층 | 세션 경계 소멸 → "언제 끝내야 할지 모름" |
| D6 오토배틀 x32 | 핵심 루프를 AI가 대체 → "내가 왜 이 게임을 켜야 하나" |
| D7 아이템 전생 RNG | 수백 시간 투자 후 랜덤 결과 → 야리코미 철학에 위배 |

### 멀티플레이 전환 시 3대 설계 결정

| 결정 사항 | 권장안 |
|:---------|:------|
| **이노센트 귀속** | 파티 리더(아이템 오너) 귀속 + 동행자 별도 보상 |
| **파티 탈출 조건** | 개인 탈출 허용, 단 리더 탈출 시 세션 종료 |
| **적 레벨 스케일링** | 아이템 레어리티+현재 층수 기반 (파티원 레벨과 독립) |

### 멀티플레이 UX 기회

1. **이노센트 추격 협동:** 야생 이노센트에 도주 행동 → 4인 포위 추격전
2. **역할 분담 자연 발생:** 이노센트 포위 / 적 방어 / 보너스 게이지 충전 분업
3. **파티 탈출 협상:** 보스 후 "계속 vs 탈출" 토론 → 소셜 인터랙션의 핵심
4. **세션 요약 화면:** 탈출 후 성과 공유 → 성취감 강화 + 스크린샷 공유

## 6. Project Abyss 기억의 지층에 대한 통합 시사점

| 디스가이아의 검증된 설계 | Project Abyss 적용 |
|:----------------------|:------------------|
| 10층 단위 체크포인트 | 지층 경계 = 보스+안전지대+탈출 선택 |
| 레어리티별 층수 차등 | Normal 2지층 ~ Ancient 4지층+심연 |
| 이노센트 야생→복종 2배 | 아이템계 진입의 핵심 동기 |
| 미스터리 룸 서프라이즈 | 기대+위험 혼합 이벤트 (항상 이득 X) |
| 선형 성장 (지수 폭발 없음) | 예측 가능한 야리코미 곡선 |
| D7의 짧은 루프 + 전생 | 명확한 세션 종료 신호 |
| 수동 정복 선행 원칙 | 자동 사냥은 클리어한 구역에서만 |
