## 핵심 관찰 5가지

1. **메뉴 시스템이 가장 큰 공백.** 타이틀/일시정지/사망/설정 화면이 전무. Phase 1 MVP에서 P0/P1으로 즉시 필요.

2. **캐릭터 스탯 화면이 전무.** ATK/INT/HP 3스탯 + 장비 슬롯 8종이 설계되어 있으나 이를 표시할 전용 화면이 없음. 일시정지 메뉴 내 탭으로 통합 권장.

3. **아이템계 귀환 결과 화면이 설계 공백.** 야리코미의 핵심 보상 확인 순간인데 UI가 없음. 레벨업/기억 단편 변화/스탯 총 변화를 한 화면에 요약하는 결과 화면 시급.

4. **맵/내비게이션은 설계 완성도가 가장 높음 (73%).** 대부분 문서 존재. 구현만 남음.

5. **코옵/소셜 UI는 Phase 3까지 전면 보류 가능.** URL 링크 합류 방식이므로 복잡한 로비/매칭 UI 불필요. P3에서 파티원 HP + 핑 시스템만 추가.
