## 1. 연구 목적과 기존 정의

### 1-1. 연구 질문

> "아이템계 안에서 획득한 아이템의 아이템계에 즉시 중첩 진입(최대 깊이 3)하는 메커닉을, 어떤 원칙으로 설계해야 플레이어에게 의미 있는 리스크-리워드 결정을 제공하면서도 시스템 악용을 방지할 수 있는가?"

### 1-2. 기존 문서에서 이미 정의된 내용

**System_ItemWorld_Core.md (SYS-IW-01):**

| 항목 | 정의 |
|:-----|:-----|
| 최대 깊이 | 3 (IWC-11-A, P2 우선순위) |

## 2. 디스가이아의 "재귀" 분석

### 2-1. 핵심 발견: 디스가이아는 순차적 재귀만 지원

디스가이아 시리즈에서 아이템계의 재귀는 **암묵적(implicit)**, **순차적(sequential)** 형태로만 존재한다.

```
디스가이아:
아이템 A 진입 → 100층 → Item God에서 아이템 B 절도 → 탈출 → 아이템 B 진입 → ...

Project Abyss (제안):
아이템 A 진입 → 지층 2에서 아이템 B 획득 → 재귀의 문 → 아이템 B 아이템계에 즉시 진입 (A 상태 동결)
```

**디스가이아에서는 아이템계 안에서 다른 아이템의 아이템계에 중첩 진입하는 기능이 공식 지원되지 않는다.** 한 번에 하나의 아이템계에만 진입 가능.

### 2-2. 디스가이아의 순차적 재귀 루프

**Item God 절도 체인:**

| 단계 | 행위 | 조건 |
|:-----|:-----|:-----|
| 1 | Rank X 아이템의 아이템계 진입 | 아이템 레벨 80+, 레어리티 50+ |
| 2 | 100층 Item God 도달 | Kill Bonus + Training Bonus 축적 |
| 3 | Item God가 보유한 Rank X+1 아이템 절도 | Thief 클래스 + Hand 장비 |
| 4 | Mr. Gency's Exit로 탈출 | 항상 1개+ 소지 필수 |
| 5 | 절도 아이템의 아이템계에 새로 진입 | 위 과정 반복 → Rank 40까지 |

**딥 런 준비물:**

| 준비물 | 용도 | 중요도 |
|:------|:-----|:------|
| Mr. Gency's Exit (복수) | 비상 탈출 | 필수 |
| Thief/Rogue 캐릭터 | Item God 절도 | 핵심 전략 |
| Land Decimator 스킬 | 전층 적 일소 | D5 최적 |
| 회복 아이템 | 장기전 대비 | 보조 |
| 고레벨 캐릭터 1기 예비 | 안전망 | 보험 |

### 2-3. 미스터리 룸과 재귀의 관계

미스터리 룸은 보스층이 아닌 일반 층에 랜덤 등장하는 특수 포탈. D5에서는 Mystery Route 선택 시 미스터리 룸 등장 확률 증가.

- 미스터리 룸의 보상(이노센트, 레전더리 장비)이 다음 아이템계 진입의 동기 제공
- D4의 분기 루트(Sword/Smiley/Mixed)가 선택적 깊이 제공
- Explorer Evility(3~4명 장착)로 미스터리 룸 빈도 조작 가능

## 3. 비-디스가이아 게임의 중첩/재귀 사례

### 3-1. Noita — 병렬 세계 (수평적 재귀)

| 항목 | 내용 |
|:-----|:-----|
| **메커닉** | 월드 경계 바위벽(Cursed Rock) 굴착 → 병렬 세계 진입 |
| **재귀 특성** | 병렬 세계의 경계벽을 다시 굴착하면 또 다른 병렬 세계. 이론적 512개 |
| **동기** | 각 병렬 세계에 고유 지팡이, 체력 업그레이드, Perk |
| **리스크** | Cursed Rock 통과 시 막대한 데미지. Orbs of Corrupted Knowledge = 체력 44% 지속 데미지 트레이드오프 |
| **방향성** | "수평적 재귀" (옆으로 확장) vs PA의 "수직적 재귀" (깊이로 확장) |

> **교훈:** 재귀 진입 시 보상이 있으되 대가가 따른다는 원칙이 동일.

### 3-2. Superliminal — 꿈 속의 꿈 (위치 불확실성)

| 항목 | 내용 |
|:-----|:-----|
| **메커닉** | 꿈 치료 프로그램에서 꿈 속에서 깨어나면 또 다른 꿈 속 |
| **재귀 퍼즐** | 물체를 확대해서 안으로 들어가면 더 작은 동일 구조물이 반복 |
| **긴장감 원천** | Dr. Pierce: "당신이 몇 번째 꿈 층에 있는지 추적할 수 없다" → 위치 불확실성이 불안 유발 |

> **교훈:** 깊이가 깊어질수록 "지금 내가 어디에 있는지" 파악이 어려워지는 것 자체가 긴장감. PA에서는 HUD로 명확히 표시하되, 환경 왜곡으로 "깊이의 불안감"을 전달.

### 3-3. Psychonauts — 정신 세계 (환경 차별화)

| 항목 | 내용 |
|:-----|:-----|
| **메커닉** | 각 캐릭터의 마음속에 독립적 레벨 존재 |
| **환경 차별화** | 각 정신 세계가 캐릭터의 심리를 반영한 완전히 다른 미학/메커닉 |
| **P2에서의 진화** | 일부 레벨이 마음 속의 더 깊은 층으로 내려가는 구조 |

> **교훈:** 재귀 깊이마다 환경이 시각적으로 극적으로 달라져야 한다. PA의 "기억의 깊이"를 환경으로 표현.

### 3-4. Double Fine의 Stacking — 마트료시카 인형

| 항목 | 내용 |
|:-----|:-----|
| **메커닉** | 작은 인형이 큰 인형 안으로 들어가(stacking) 바깥 인형의 능력 사용 |
| **퍼즐** | 올바른 크기/능력의 인형 조합으로 문제 해결 |

> **교훈:** "안으로 들어간다"는 마트료시카 메타포가 PA의 재귀적 진입과 구조적으로 일치.

### 3-5. FFXIV Deep Dungeon — 딥 던전 (수직적 깊이)

| 항목 | 내용 |
|:-----|:-----|
| **메커닉** | Palace of the Dead 200층, Heaven-on-High 100층 절차적 던전 |
| **진행** | 독립적 레벨링 (진입 시 리셋), 10층 단위 세이브/보스 |
| **리스크-리워드** | 21~30층 파밍 효율 최적. 더 깊으면 희귀 보상 증가 + 전멸 리스크 급증 |

> **교훈:** 10층 단위 체크포인트, 독립적 레벨링, 특수 아이템(Pomander) 생존 자원 관리.

### 3-6. Darkest Dungeon — 스트레스와 깊이

| 항목 | 내용 |
|:-----|:-----|
| **메커닉** | 던전 깊이(Apprentice → Veteran → Champion)에 따라 적 능력치 대폭 상승 |
| **스트레스** | 0~200 게이지. 100 = Affliction/Virtue 판정, 200 = 심장마비(즉사) |
| **원인** | 적 공격, 저조도, 후퇴, 함정, Curio |
| **리스크 관리** | Short vs Long 던전 선택, 보급품 사전 구매 |

> **교훈:** 자원 고갈 + 탈출 불확실성이 깊이에 비례하는 긴장감을 만든다. PA에서 명시적 스트레스 메커닉은 없지만, 포션/MP 고갈이 동일한 역할.

### 3-7. Slay the Spire — 자발적 리스크 테이킹

| 항목 | 내용 |
|:-----|:-----|
| **메커닉** | Act 1→2→3→Final Boss. Elite = 높은 리스크 + Relic 보상 (선택적) |
| **핵심 판단** | "HP를 잃더라도 Elite를 잡아서 Relic을 얻을 것인가, 우회할 것인가" |
| **발견** | 승리한 런일수록 적극적 리스크 테이킹 (정규화 엔트로피 높음) |

> **교훈:** 재귀 진입 자체가 "Elite 전투를 선택하는 것"과 같은 자발적 리스크 테이킹. 선택하지 않아도 기본 클리어는 가능하되, 선택하면 더 높은 보상.

### 3-8. Spelunky — 깊이와 자원의 트레이드오프

| 항목 | 내용 |
|:-----|:-----|
| **메커닉** | Shortcut으로 깊은 구역 즉시 진입 가능하나, 이전 축적물 없는 상태로 시작 |
| **Full Run** | 완전한 런만이 트로피/엔딩 달성 가능 |

> **교훈:** 재귀 진입 시 "상위 깊이의 진행이 중단된다"는 것 자체가 기회비용(시간, 진행 중단).

## 4. 영화/미디어의 중첩 메타포

### 4-1. 인셉션 — 꿈 층 구조

- 5개 꿈 층. 각 층은 시간이 20배씩 느려짐
- 더 깊은 층에서 죽으면 Limbo(무의식의 끝)로 추락
- 강력한 진정제 사용 시 죽음 ≠ 깨어남, 죽음 = Limbo
- 각 층의 색조(color palette)가 명확히 구분

> **교훈:** "더 깊은 곳에서의 실패가 더 큰 대가" + 깊이별 환경의 시각적 구분. PA의 재귀 연출 설계에 이미 반영됨.

### 4-2. 마트료시카 — 구조적 메타포

- "안에 또 안이 있다"는 가장 직관적인 재귀 메타포
- PA의 "아이템 속으로 들어가서, 그 안의 아이템 속으로 다시 들어간다" = 마트료시카 구조

## 5. 재귀적 진입의 보편적 설계 원칙

리서치에서 추출한 7가지 원칙:

| # | 원칙 | 설명 | 근거 |
|:--|:-----|:-----|:-----|
| 1 | **자발적 선택** | 재귀 진입은 항상 플레이어의 선택. 강제되면 그라인딩 | 디스가이아, StS |
| 2 | **깊이 = 리스크 에스컬레이션** | 각 깊이마다 리스크가 가시적으로 증가 | Darkest Dungeon, FFXIV |
| 3 | **고유 보상** | 깊은 재귀에서만 얻을 수 있는 보상 존재 | 디스가이아 Item God 절도, Noita Corrupted Orbs |
| 4 | **명확한 위치 표시** | 현재 재귀 깊이와 위치를 항상 표시 | Superliminal 반례 |
| 5 | **탈출 안전망** | 비상 탈출 수단이 반드시 존재 | 디스가이아 Mr. Gency's Exit |
| 6 | **부분적 손실만** | 전체 진행의 영구 손실은 없어야 | 디스가이아 장비 미소실, PA 장비 보존 |
| 7 | **환경적 차별화** | 각 깊이의 환경이 시각적으로 구분 | Psychonauts, Superliminal, 인셉션 |

## 6. 깊이별 동기 부여와 긴장 구조

### 6-1. 동기 부여 (왜 더 깊이 가는가?)

| 깊이 | 동기 | 보상 유형 | 근거 |
|:-----|:-----|:---------|:-----|
| 깊이 1 (기본) | 장비 강화 | 기본 스탯 증가, 이노센트 | 디스가이아 기본 루프 |
| 깊이 2 | 희귀 자원 | 재귀 전용 이노센트, 중첩 강화 | Item God 절도 체인에서 영감 |
| 깊이 3 (최대) | 극한 보상 | 희귀 강화 소재, 유니크 이노센트, 숨겨진 서사 | Noita Corrupted Orbs (높은 대가의 희귀 보상) |

### 6-2. 긴장감 생성 구조 (깊이별 리스크)

| 긴장 요소 | 깊이 1 | 깊이 2 | 깊이 3 |
|:---------|:------|:------|:------|
| 자원 고갈 | 낮음 | 중간 (상위 진행 중단) | 높음 (이중 중단) |
| 탈출 난이도 | 보스 후 탈출/제단 | 현재 깊이 탈출 → 상위 복귀 | 3단계 순차 탈출 필요 |
| 환경 압박 | 기본 | 팔레트 변화, 채도 증가 | 색수차, 시각적 왜곡 |
| 사망 패널티 | 현재 깊이만 | 현재 깊이만 (상위 보존) | 현재 깊이만 (상위 보존) |
| 시간 투자 | 10~15분 | +10~15분 | +10~15분 (누적 30~45분) |

### 6-3. 수학적 모델 (연계 문서)

> 깊이별 난이도/보상 스케일링의 구체적 수학 모델은 `ItemWorld_DepthReward_RiskBalance_Research.md`에 별도 정리.
>
> 핵심 공식:
> - 몬스터 HP: 지층 간 ×1.8
> - 몬스터 데미지: 지층 간 ×1.4 (비대칭 — 즉사 방지)
> - 전망 이론: 보상이 리스크의 ×2.25여야 도전 유도 (손실 회피 계수 λ ≈ 2.25)
> - 재방문 보상 감쇠: `0.5^(visit-1)` — 자연스러운 파밍 대상 교체

## 7. UX 고려사항

### 7-1. 재귀 깊이 시각 표시

HUD에 마트료시카처럼 중첩된 아이콘으로 현재 깊이 표시:

```
깊이 1: [■]          — 단일 프레임
깊이 2: [■[■]]       — 이중 프레임
깊이 3: [■[■[■]]]    — 삼중 프레임
```

### 7-2. 브레드크럼 네비게이션

경로를 항상 표시:

```
월드 > 롱소드 아이템계(지층 2) > 단검 아이템계(지층 1) > 반지 아이템계(지층 3)
                                  [동결]                [동결]            [현재]
```

### 7-3. 탈출 시 순차 복귀 연출

깊이 3에서 탈출 시 3→2→1→월드 순서로 "상승" 연출:

```
깊이 3 탈출 → 줌아웃 + 반지 축소 → 반지 아이템계에서 단검 아이템계로 복귀 (0.5초)
깊이 2 복귀 → 줌아웃 + 단검 축소 → 단검 아이템계에서 롱소드 아이템계로 복귀 (0.5초)
깊이 1 복귀 → 줌아웃 + 롱소드 축소 → 허브로 복귀 (0.5초)
```

인셉션의 "킥(Kick)"처럼 각 깊이에서 깨어나는 시퀀스.

### 7-4. 깊이별 환경 변형 (이미 정의된 내용 + 보강)

| 깊이 | 시각 변형 | 오디오 변형 (제안) |
|:-----|:---------|:----------------|
| 1 (기본) | 아이템 테마의 기본 팔레트 | 기본 BGM |
| 2 | 팔레트 채도 +20%, 이중 잔상 | BGM 피치 -5%, 리버브 증가 |
| 3 | 색수차(chromatic aberration), 삼중 잔상 | BGM 피치 -10%, 디스토션 + 로우패스 필터 |

### 7-5. 상위 깊이 미니맵

상위 깊이의 미니맵이 작게 축소되어 화면 모서리에 유지:
- "돌아갈 곳이 있다"는 심리적 안전감
- "여기는 그곳이 아니다"는 긴장감
- 두 감정이 동시에 제공

## 8. 밸런스 / 익스플로잇 방지

| 위험 | 대응책 | 근거 |
|:-----|:------|:-----|
| 재귀로 무한 아이템 파밍 | 재방문 보상 50% 감소 (이미 정의) + 깊이별 드랍 테이블 차등 | 디스가이아 더블 킬링 교훈 |
| 저레벨 아이템으로 안전 파밍 | 재귀 가능 아이템 최소 레어리티 제한 (Magic 이상만 재귀의 문 스폰) | PA 고유 밸런스 |
| 깊이 3에서 무한 루프 | 최대 깊이 3 하드캡 + 깊이 3에서 획득 아이템은 재귀 불가 | 시드 충돌 방지 (IWF-11-A) |
| 재귀 전용 보상만 추구 | 기본 아이템계만으로도 핵심 강화 완성 가능. 재귀는 "보너스" | StS Elite 선택 구조 |
| 멀티에서 무의미한 재귀 강요 | 파티 전원 동의 필요. 미동의 시 개인 진입 불가 | 소셜 계약 보호 |

## 9. Project Abyss의 독창적 위치

### 9-1. 핵심 발견

> **웹 리서치 결과, "아이템계 안에서 획득한 아이템의 아이템계에 즉시 중첩 진입"하는 메커닉을 구현한 상용 게임은 확인되지 않았다.**

| 게임 | 재귀 유형 | PA와의 차이 |
|:-----|:---------|:----------|
| 디스가이아 | 순차적 (나와서 다시 들어감) | PA는 즉시 중첩 진입 |
| Noita | 수평적 (옆으로 확장) | PA는 수직적 (깊이로 확장) |
| Superliminal | 내러티브 장치 | PA는 게임플레이 메커닉 |
| Psychonauts | 허브-스포크 구조 | PA는 스택 구조 |
| FFXIV Deep Dungeon | 단일 깊이 수직 진행 | PA는 "깊이의 깊이" |

**Project Abyss의 재귀적 진입은 장르 최초의 시도로서 높은 독창성을 가진다.**

### 9-2. 디스가이아에는 없는 PA 고유 혁신

| 요소 | 디스가이아 | Project Abyss |
|:-----|:---------|:-------------|
| 아이템계 내부 중첩 진입 | **불가** (순차만) | **가능** (최대 깊이 3) |
| 실시간 액션 전투 | 턴제 SRPG | 횡스크롤 액션 |
| 온라인 파티 협동 | 없음 (D7 로컬 2인) | 1~4인 온라인 |
| 깊이별 시각 변형 | 없음 | 팔레트, 채도, 색수차 |
| 재귀 안에서의 서사 | 없음 | 기억의 방 (아이템 역사 조각) |
