## 1. 레퍼런스 게임 분석

### 1-1. Dead Cells — 그라디언트 맵 바이옴 분리 기법

Dead Cells의 아트 디렉션 핵심은 **"같은 타일셋, 다른 팔레트"**다. 개발팀은 모든 패럴랙스 배경 텍스처를 그레이스케일로 먼저 제작한 다음, 바이옴별 그라디언트 맵을 덮어씌워 색상을 결정한다.

| 기법 | 내용 | PA 적용 포인트 |
| :--- | :--- | :--- |
| 그라디언트 맵 기반 팔레트 | 텍스처는 그레이스케일 1벌. 바이옴마다 그라디언트 맵만 교체하면 완전히 다른 색조로 변환. Promenade가 처음엔 오렌지색이었다가 Ramparts와 배치가 겹쳐 게임플레이상 재배치되었을 때, 팀은 그라디언트 맵만 교체하여 시각적 중복 없이 문제를 해결했다. | T-FORGE 타일셋을 그레이스케일 원본으로 제작하고, 테마 ID별 그라디언트 맵을 관리하면 타일 아트 중복 작업 없이 10개 테마를 운영할 수 있다. |
| 라이팅 색상 + 셰이더 | 팔레트 외에 조명 색온도, 연기, 물, 안개, 식생 밀도 파라미터를 개별 조정한다. | 테마 ID마다 `lightColor`, `fogDensity`, `particlePreset` 등을 데이터로 관리. |
| 고채도 유지 원칙 | 채도를 높게 유지하면 어두운 환경에서도 시각적 각성 상태가 유지되어 플레이어 집중도가 높아진다. | PA 아이템계의 어두운 금속 환경에서 테마 포인트 컬러는 항상 채도를 높게 유지한다. |

Dead Cells의 핵심 교훈: **같은 방 레이아웃이라도 팔레트와 앰비언트 이펙트의 조합이 "다른 곳에 있다"는 감각을 만든다.** 플레이어는 타일 형태보다 색온도와 파티클을 먼저 읽는다.

### 1-2. Hollow Knight — 구역별 색상·구조·사운드의 삼위일체

Hollow Knight의 각 구역은 색상 팔레트, 지형 실루엣, 음악 세 요소가 항상 함께 변한다.

| 구역 | 주요 팔레트 | 분위기 기법 | PA 적용 포인트 |
| :--- | :--- | :--- | :--- |
| Greenpath | 짙은 녹색 계열. 에메랄드~황록. 주변보다 밝음 | 식생이 전면에 배치되어 적이 숨어 있다. Forgotten Crossroads에서 전환 시 명도 급상승으로 "살아 있는 구역"을 즉각 인지 | T-FOREST 지층의 넝쿨·뿌리 전경 레이어 아이디어 |
| Crystal Peak | 핑크~연보라. 발광 크리스탈이 유일한 광원 | 크리스탈 자체가 발광하여 에너지의 원천이 환경에 내재함을 보여준다. 단색 발광 반복으로 리듬감 형성 | T-SCHOLAR 지층의 결정·룬 발광 아이디어 |
| Deepnest | 거의 흑백에 가까운 저채도. 단속적인 파란 섬광만 존재 | 미니맵 정보가 극도로 제한되어 시각적 불안을 강화. 동선이 폐쇄적인 횡형 구조 | T-UNDEAD 심층의 어둠·최소 조명 아이디어 |

Hollow Knight의 핵심 교훈: **팔레트 단독이 아니라, 팔레트 + 광원의 출처 + 지형 실루엣 세 요소가 일치할 때 구역 아이덴티티가 완성된다.** 광원이 어디서 오는지(외부광 vs 발광 오브젝트 vs 불꽃)가 분위기를 결정한다.

### 1-3. Hades — 내러티브-팔레트 직결 원칙

Hades는 아티스트와 스크립터가 먼저 "이 구역의 이야기"를 합의한 다음 팔레트를 결정한다. 공간의 내러티브 목적과 색상이 항상 정합된다.

| 구역 | 팔레트 | 내러티브 이유 |
| :--- | :--- | :--- |
| Tartarus | 차가운 회색~역겨운 황록. 타는 붉은 강 포인트 | "영원히 갇힌 영혼의 부패" — 생기 없는 색조가 정체와 부패를 표현 |
| Asphodel | 피빛 적~오렌지. 타오르는 용암 | "피에 젖은 초원" — 위험하고 극적인 붉은 색이 전쟁의 잔상 |
| Elysium | 청백~황금. 차분하고 밝음 | "영웅들의 안식처" — 평온함을 차가운 빛으로 표현 |
| Styx | 보라~검정. 점성의 어둠 | "경계의 강" — 미확정·임박한 전환의 색 |

Hades의 핵심 교훈: **색상은 그 공간의 이야기를 먼저 결정하고, 팔레트는 그 이야기를 시각적으로 번역하는 것이다.** PA의 테마 시스템에서 각 T-ID의 팔레트는 해당 테마의 "기억이 가진 감정"을 번역해야 한다.

### 1-4. Celeste — 감정 상태와 환경 왜곡의 연동

Celeste는 챕터가 진행될수록 환경이 주인공의 심리 상태를 반영하여 기하학적으로 왜곡되고 색조가 변한다. 초반은 밝고 정돈된 환경, 후반은 파편화된 기하와 불안한 색조.

| 기법 | 내용 | PA 적용 포인트 |
| :--- | :--- | :--- |
| 감정 → 기하 변형 | 불안이 증가할수록 플랫폼의 형태가 불규칙해지고 원근이 왜곡된다 | 지층 깊이가 깊어질수록(심층·심연) 타일 형태가 불규칙해지는 기억 퇴화 효과 |
| 파티클로 감정 강화 | 챕터마다 고유한 파티클(꽃잎, 눈, 빛의 파편)이 배경 감정의 레이어로 작동 | 테마 ID마다 고유한 앰비언트 파티클이 기억의 성격을 전달 |
| 제한된 팔레트의 강도 | 12색 팔레트 제한이 오히려 색의 역할을 명확하게 만든다 | PA 아이템계는 테마당 8~12색 팔레트 제한을 권장 |

---

### 1-5. Hyper Light Drifter — 포인트 컬러 반복 기법

Hyper Light Drifter는 각 구역의 **포인트 컬러를 건축 장식과 파티클에 반복** 배치하여 공간의 색상 언어를 통일한다. 구역 입장 직후 몇 초 안에 어떤 구역인지 인지 가능하다.

| 기법 | 내용 |
| :--- | :--- |
| 두 가지 색조 오브젝트 | 오브젝트를 두 톤으로 렌더링하고, 구역별 포인트 컬러를 건축 요소에 반복 배치하여 공간 전체를 통일 |
| 부드러운 그라디언트 배경 | 디더링 대신 소프트 그라디언트를 사용해 깊이감 추가. 픽셀아트 미학을 유지하면서도 명도 변화를 자연스럽게 표현 |
| 포스트 아포칼립스 + 고채도 | "황폐한 세계"임에도 불구하고 구역별로 강렬한 색상을 사용. 어두운 세계가 색을 포기할 필요는 없다 |

PA 적용 포인트: 각 테마 ID의 포인트 컬러를 타일 장식·파티클·광원 세 요소에 모두 반복하면 테마 진입 첫 3초에 정체성을 전달할 수 있다.

### 1-6. Blasphemous — 감정 레이어로서의 건축 실루엣

Blasphemous는 스페인 안달루시아 고딕 건축에서 출발하여, 각 구역의 **건축적 실루엣**이 그 구역의 감정 상태를 표현한다. 첨탑은 압박, 아치는 무게, 좁은 통로는 위협감을 만든다.

| 기법 | 내용 | PA 적용 포인트 |
| :--- | :--- | :--- |
| 전경 실루엣 압박 | 어두운 전경 오브젝트로 화면 공간을 좁혀 심리적 압박감 형성 | T-UNDEAD, T-WAR 지층의 폐허 전경 레이어로 적용 |
| 채도 낮은 마젠타 그림자 | 전면 오브젝트의 그림자에 탈채도 마젠타를 사용하여 이질적 불안감 형성 | PA에서는 특정 테마의 그림자 컬러에 테마 포인트 컬러를 차가운 버전으로 사용 |
| 장식적 디테일 밀도 | 장식이 많을수록 세계관의 집착(광기)을 표현. 장식이 없을수록 황량함을 표현 | 테마 레어리티가 높을수록 장식 밀도가 증가 (기억이 더 풍부함) |

## 2. 픽셀아트 시각 테마 핵심 기법

레퍼런스 분석에서 도출한 PA 아이템계 비주얼 테마 구현을 위한 실용 기법 목록.

### 2-1. 팔레트 스왑 (Palette Swap)

픽셀아트의 전통적 팔레트 스왑은 인덱스 컬러 방식을 기반으로 한다. 현대 게임(PNG 기반)에서는 두 가지 방식이 현실적이다:

| 방식 | 설명 | PA 적합성 |
| :--- | :--- | :--- |
| 셰이더 기반 LUT (Lookup Table) | 그레이스케일 인덱스 → 팔레트 텍스처에서 색상 읽기. 런타임 팔레트 교체 가능 | 적합. PixiJS v8의 필터 시스템으로 구현 가능. 테마 전환 시 LUT 교체만으로 완성 |
| 런타임 그라디언트 맵 | Dead Cells 방식. 그레이스케일 텍스처 + 바이옴별 그라디언트 맵. 패럴랙스 배경에 특히 효과적 | 배경 레이어에 우선 적용 권장 |

구현 원칙: **타일셋 아트는 그레이스케일 또는 최소 색상으로 제작하고, 색상은 데이터(LUT/그라디언트 맵)로 분리 관리한다.** 이렇게 하면 테마 추가 시 아트 자산 재제작 없이 데이터만 추가하면 된다.

### 2-2. 앰비언트 파티클 (Ambient Particles)

| 파티클 유형 | 기상 상태 전달 | 감정 전달 | 구현 특성 |
| :--- | :--- | :--- | :--- |
| 먼지 (Dust) | 건조, 오래됨 | 망각, 쇠퇴 | 저속 랜덤 부유. 불투명도 10~30% |
| 불씨/숯 (Embers) | 고온, 단조 | 긴장, 활동적 | 상승 궤적, 깜박임. 포인트 컬러로 |
| 눈/서리 (Snow/Frost) | 저온, 보존 | 고요, 슬픔 | 낙하 + 약한 수평 이동 |
| 포자 (Spores) | 습함, 유기적 | 으스스함, 자연 | 불규칙 흔들림, 연두~황록색 |
| 물방울 (Droplets) | 수분, 지하 | 고독, 끝없음 | 위에서 아래, 간헐적 |
| 번개/정전기 (Sparks) | 전기, 에너지 | 날카로움, 위험 | 비정기 섬광, 단발 효과 |
| 꽃잎/잎 (Petals) | 식생, 생명 | 덧없음, 아름다움 | 느린 낙하 + 흔들림 |
| 뼛가루 (Bone Dust) | 죽음, 부패 | 공포, 엄숙 | 매우 미세, 흰색~회백색 |

### 2-3. 배경 패럴랙스 레이어 (Background Parallax)

PA의 월드 배경 삼중 레이어 원칙을 아이템계에 적용한 버전:

```
Layer 1 (가장 먼 원경):  다마스커스 기저 — 금속 결의 거시 패턴 (테마와 무관한 PA 공통 레이어)
Layer 2 (중경):          테마 환경 — 해당 T-ID의 특성 배경 (T-FORGE면 용광로 실루엣, T-FOREST면 수목 실루엣)
Layer 3 (근경):          기억의 잔향 — 이전 주인의 흔적 (도구, 가구 파편, 발자국 등)
```

각 레이어는 서로 다른 패럴랙스 속도를 가지며, 플레이어 이동 시 레이어 간 깊이감이 생긴다. 근경 레이어의 "기억 흔적"이 PA 스파이크("아이템의 기억이 던전이 된다")를 지속적으로 강화한다.

### 2-4. 동일 레이아웃, 다른 테마 전략 (Same Layout, Different Theme)

Dead Cells의 핵심 교훈을 PA 아이템계에 직접 적용:

| 단계 | 내용 |
| :--- | :--- |
| 1. 레이아웃 공유 | 4×4 Room Grid는 모든 테마에서 동일한 구조를 사용. 방의 형태(전투방, 보상방, 함정방)는 테마와 무관 |
| 2. 팔레트 전환 | Room 로드 시 해당 테마의 LUT와 그라디언트 맵을 적용. 타일 형태는 같지만 색상이 완전히 다름 |
| 3. Chunk 교체 | 방의 골격은 공유하되, 내부 Chunk(지형 장애물)는 테마별 Chunk 풀에서 선택. T-FORGE는 용광로 기계, T-FOREST는 뿌리/돌 |
| 4. 파티클 + 광원 | 테마별 앰비언트 파티클 프리셋과 광원 색온도를 오버레이. 같은 방이 완전히 다른 감각을 줌 |

## 3. PA 전용 원칙: 다마스커스 기반 테마 시스템

`Design_Art_Direction.md`의 아이템계 핵심 원칙을 테마 시스템과 통합하는 원칙.

### 3-1. 다마스커스 레이어 원칙

모든 테마는 **다마스커스 강철의 단면 위에 기억이 씌워진 것**이다. 어떤 테마라도 다마스커스 금속 결(다마스커스 패턴)이 배경 Layer 1에 존재해야 한다. 이것이 PA 아이템계가 "어떤 아이템이든 같은 세계"임을 보장하는 시각적 공통분모다.

```
기저 (항상 존재):  다마스커스 금속 결  →  PA 아이덴티티 레이어
테마 레이어:        T-ID별 기억 환경   →  아이템 고유성 레이어
레어리티 레이어:   등급별 왜곡/채도    →  강도(intensity) 레이어
지층 레이어:       깊이별 퇴화         →  시간(depth) 레이어
```

### 3-2. 월드-아이템계 팔레트 반전 원칙

`Design_Art_Direction.md` §3.4 정의:
- 월드: 청록(배경) + 주황(구조물)
- 아이템계: 주황(배경·열기) + 청록(디테일·산화)

각 T-ID 테마 팔레트는 이 반전 원칙 위에 **테마 포인트 컬러**를 추가하는 방식으로 구성된다. 테마 포인트 컬러는 기억의 감정적 온도를 표현하는 세 번째 색이다.

```
아이템계 기본 공식:
  배경 주황 (단조열) + 청록 산화 (디테일) + 테마 포인트 컬러 (감정)
```

### 3-3. 기억 단편 색조 규칙

`Design_Art_Direction.md` §5.4에서 정의:
- 야생 기억 단편: 해당 지층 금속 색과 동일 (테마 포인트 컬러 계열)
- 복종 기억 단편: 살아있을 때의 고유색 일부 복원

이 규칙은 각 테마 내에서 일관되게 적용된다. T-FORGE 지층의 야생 기억 단편는 용광로 주황색, T-FOREST 지층의 야생 기억 단편는 수목 녹색 계열이 된다.

## 4. 10개 테마 시각 카탈로그

각 테마는 다음 구조로 정의된다:
- **팔레트:** 배경색 / 전경색 / 포인트 컬러 / 그림자 색
- **타일 미학:** 재질감, 형태 언어, 장식 패턴
- **앰비언트 파티클:** 유형, 색상, 밀도, 움직임

- **광원 설정:** 광원 출처, 색온도, 강도
- **오디오 무드:** BGM 방향, 대표 SFX

- 중경 (Layer 2): 대형 용광로 실루엣, 크레인, 풀무 장치
- 근경 (Layer 3): 반쯤 완성된 금속 조각, 식어가는 단조 금속 — "제작의 순간이 멈춰 있다"

- 대표 SFX: 망치 타격, 금속 냉각 쉬익 소리, 풀무 바람 소리
- 보스 BGM: 망치 리듬이 배경 비트로 전환. 점점 빠른 템포

| Ancient `#00FF00` | 초과채도 (130%) + 왜곡 | 매우 높음 | 전체 + 2차 추상 레이어 | 매우 높음 | 초현실적. 기억이 현실보다 더 생생함 |

### Normal 아이템 — "흐릿한 기억"

Normal 아이템의 아이템계는 테마는 있지만 색상이 탁하고 세부가 지워져 있다. 할아버지의 마지막 부엌칼 한 자루처럼, 너무 오래 되어 기억이 바랜 상태. 파티클이 거의 없고, 배경은 단순한 금속 결 위에 테마 실루엣만 있다.

### Ancient 아이템 — "기억이 넘쳐흐름"

Ancient 아이템의 아이템계는 테마가 현실을 초월한다. 색수차(chromatic aberration)가 적용되어 색상이 분리되어 보이고, 파티클이 화면을 메운다. 배경 레이어 너머에 또 다른 추상 레이어가 보이며, 타일 형태가 살짝 꿈틀거린다. 이것은 "너무 강한 기억"이 물리적 공간을 침식하는 표현이다.

## 6. 지층 깊이별 기억 퇴화 시스템

각 테마의 팔레트와 미학은 지층이 깊어질수록 다음 방향으로 변화한다. `Design_Art_Direction.md` §3.2의 지층 비주얼 변화 원칙과 정합된다.

```
Surface 지층 (지층 1):
  테마 그대로. 기억이 명확. 색상 선명.
  → T-HOME이면 따뜻하고 정돈된 부엌

Mid 지층 (지층 2):
  기억이 겹치기 시작. 배경에 두 번째 기억의 잔상이 비침.
  금속 결이 살짝 불규칙해짐.
  → T-HOME에서 T-MILITARY의 실루엣이 먼 배경에 겹침

Deep 지층 (지층 3, Crossroads):
  두 테마가 적극적으로 혼합. 방 절반은 한 테마, 절반은 다른 테마.
  색조 충돌이 의도적으로 이질적.
  → 왼쪽: T-MILITARY 석재 청회, 오른쪽: T-HOME 황토

Core 지층 (지층 4, Legendary+):
  기원 테마. 처음 만들어진 당시의 기억.
  색상이 원형적이고 원시적. T-FORGE 계열로 회귀 (모든 아이템의 출발점은 단조)
  → 다마스커스 결이 방 전체를 채우고, 그 위에 기원 테마 디테일

심연 (Ancient 전용):
  추상적. 초현실적. 특정 테마 없이 오직 기억의 원형(archetype).
  색수차 + 노이즈 + 왜곡 필터.
  → 에르다와 에코의 관계, 세계관 핵심과 연결되는 비주얼
```

### 기억 퇴화 시각 효과 (Visual Degradation)

심층으로 내려갈수록 적용되는 시각 효과:

| 효과 | Surface | Mid | Deep | Core | 심연 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 팔레트 왜곡 | 없음 | 미미 | 약함 | 중간 | 강함 |
| 색수차 | 없음 | 없음 | 미미 | 약함 | 강함 |
| 타일 흔들림 | 없음 | 없음 | 없음 | 미미 | 중간 |
| 노이즈 오버레이 | 없음 | 없음 | 미미 | 약함 | 강함 |
| 비네팅 강도 | 낮음 | 중간 | 중간 | 높음 | 매우 높음 |
| BGM 피치 시프트 | 없음 | 없음 | -2% | -5% | -10% |

이 효과들은 PixiJS v8 필터 시스템(`ColorMatrixFilter`, `DisplacementFilter`, `CRTFilter`)으로 구현 가능하다. 기술적 명세는 technical-artist에게 위임.

## 7. 크로스로드 테마 혼합 원칙

지층 3(Crossroads)에서 두 테마가 만날 때의 시각 설계 규칙. `Content_Item_Narrative_GrandfatherKitchenKnife.md`에서 "왼쪽 벽: 석조(T-MILITARY), 오른쪽 벽: 황토(T-HOME)"로 이미 방향이 잡혀 있다.

### 공간 분할 방법 3가지

| 방법 | 설명 | 사용 상황 |
| :--- | :--- | :--- |
| 수직 분할 | 방 중앙을 기준으로 왼쪽/오른쪽이 다른 테마 | 두 세계가 충돌하는 긴장 관계 |
| 수평 분할 | 바닥/천장이 다른 테마. 또는 위층/아래층 | 시간의 층위 (더 오래된 것이 아래) |
| 주변부-중심 분할 | 방 가장자리는 한 테마, 중심부는 다른 테마 | 기억의 융합. 중심에서 화해 |

### 크로스로드 전환 파티클

두 테마의 파티클이 공존하되, 방 중앙에서는 두 파티클 유형이 서로 충돌하거나 변환되는 효과를 넣는다. 예: T-HOME 불씨 + T-MILITARY 먼지가 중앙에서 만나 소용돌이.

## 8. 구현 우선순위 및 기술 명세

### Phase 1 (프로토타입) 최소 구현 세트

Phase 1에서는 아이템계 미니(1지층)를 목표로 한다. 최소한 다음 두 테마의 시각적 구분이 필요하다:

| 우선순위 | 테마 | 이유 |

| 2 | T-HOME | 할아버지의 부엌칼(CNT-ITM-001)이 PA 첫 번째 콘텐츠 아이템. 코믹한 톤으로 서비스 |

이 두 테마만으로도 "같은 아이템계인데 다른 기억"이라는 스파이크를 검증할 수 있다.

## 9. 교훈 요약

레퍼런스 분석과 PA 전용 원칙에서 도출한 핵심 교훈.

| 번호 | 교훈 | 출처 | PA 적용 |
| :--- | :--- | :--- | :--- |
| L-01 | 플레이어는 타일 형태보다 색온도와 파티클을 먼저 읽는다 | Dead Cells 그라디언트 맵 | 테마 전환 첫 3초에 파티클과 광원을 우선 변경 |
| L-02 | 광원의 출처(화덕 vs 발광 크리스탈 vs 귀화 불꽃)가 공간의 감정을 결정한다 | Hollow Knight 구역 설계 | T-ID마다 고유한 광원 출처를 정의 |
| L-03 | 내러티브 목적이 팔레트보다 먼저다 | Hades 아트 디렉션 | T-ID 팔레트는 "이 기억의 감정"을 번역한 결과여야 한다 |
| L-04 | 같은 레이아웃을 다르게 느끼게 하는 것은 팔레트 + 파티클 + 광원의 삼위일체다 | Dead Cells | 4×4 Room Grid 공유 전략을 안전하게 실행 가능 |
| L-05 | 제한된 팔레트(8~12색)는 각 색의 역할을 명확하게 만든다 | Celeste 12색 | 테마당 팔레트 제한이 아트 방향성을 보호한다 |
| L-06 | 포인트 컬러를 타일 장식, 파티클, 광원 세 곳에 반복하면 3초 안에 테마를 인지시킨다 | Hyper Light Drifter | 각 T-ID의 포인트 컬러를 세 요소에 일관되게 사용 |
| L-07 | 환경 왜곡은 감정 상태를 기하학으로 번역한다 | Celeste 챕터 진행 | 지층이 깊어질수록 타일이 왜곡 → 기억의 불안정성 표현 |
| L-08 | 건축 실루엣이 심리적 압박을 만든다 | Blasphemous 전경 처리 | T-UNDEAD, T-WAR의 폐허 전경 레이어로 적용 |
