## 1. 엔드게임 루프 비교 분석

### 1.1 디아블로 3 — 시즌 구조와 파라곤 시스템

#### 핵심 엔드게임 루프 구조

디아블로 3의 엔드게임은 **그레이터 리프트(Greater Rift) 티어 상승**을 핵심 축으로 한다. 기본 루프는 다음과 같다:

```
[그레이터 리프트 도전]
       ↓
티어 클리어 → Legendary Gem 업그레이드 (Urshi NPC)
       ↓
Primal Ancient 드랍 확률 = 10% (일반 Ancient 대비)
       ↓
세트 아이템 완성 → 6피스 보너스 활성화
       ↓
더 높은 GR 티어 도전 → 반복
```

| 항목 | 수치 |
|:---|:---|
| 핵심 루프 1회 소요 시간 | GR 70~100: 약 3~8분 |
| Legendary Gem 최대 레벨 | 150 (GR 클리어 티어 +1이 최대 업그레이드 성공률) |
| Primal Ancient 드랍 확률 | 전체 드랍의 1% (GR 70 최초 클리어 후 해금) |
| 시즌 기간 | 약 3개월 (역대 평균 12~14주) |
| 파라곤 포인트 배분 | 무제한 누적 (시즌 종료 시 영구 계정으로 이관) |

#### 시즌 구조와 "Journey" 시스템

디아블로 3 시즌은 **Haedrig's Gift** (무료 세트 아이템 6피스) 지급을 통해 빠른 엔드게임 진입을 보장한다. 시즌 여정(Season Journey)의 단계:

- Chapter I~IV: 세트 파밍용 Haedrig's Gift 지급
- Slayer → Champion → Destroyer → Conqueror → Guardian: 코스메틱 및 Stash Tab 보상
- Guardian 달성 시 시즌 전용 페트/날개 등 영구 코스메틱

**설계 교훈:** 신규 시즌의 "빠른 파워업 구간(rush to endgame)"이 리텐션의 핵심이다. 첫 48~72시간의 경험이 시즌 지속 참여율을 결정한다.

#### 파라곤 시스템의 문제점

파라곤 포인트는 시즌 캐릭터가 비시즌으로 이관 시 **영구 누적**된다. 이로 인해:

- 파라곤 수천 포인트를 보유한 고인물과 신규 시즌 참여자 간의 파워 격차가 시즌 내에서도 형성됨
- 시즌 리더보드는 시간 투자량 = 순위로 수렴하는 경향 (파라곤 포인트당 Main Stat +5, Movement Speed +0.1% 등)
- **D4에서 파라곤 시스템을 전면 개편한 이유**: 무한 누적 방지를 위해 D4는 보드 기반 파라곤으로 교체, 각 노드에 명시적 상한선 부여

#### 디아블로 4 — 성배 던전(Pit of Artificers)과 Masterworking

D4 Season 4(Loot Reborn)에서 도입된 핵심 엔드게임 시스템:

| 시스템 | 설명 | 수치 |
|:---|:---|:---|
| Tempering | 아이템에 2개까지 커스텀 Affix 추가 | 부위별 Temper Pool 5~8종 중 선택 |
| Masterworking | 아이템 스탯 최대 +100% 강화 (12단계) | 4/8/12단계에 특정 Affix 25% 강화 |
| Pit of Artificers | GR 대응 티어 던전 (티어 1~200) | 1회 소요: 4~15분, Masterworking 재료 드랍 |
| Mythic Unique | 극희귀 최강 아이템 (계정 1개) | 게임 내 추정 드랍률 0.03~0.1% |

**D4 설계 전환의 핵심:** "완벽한 아이템 드랍"이 아닌 "아이템을 완벽하게 만드는 과정"으로 엔드게임 콘텐츠를 재설계. 드랍이 목적지가 아닌 재료가 되는 구조.

### 1.2 Path of Exile — 리그 경제와 매핑 시스템

#### PoE 엔드게임의 레이어 구조

PoE 엔드게임은 **수직적 깊이(Atlas 진행도)**와 **수평적 전문화(리그 메카닉 선택)**가 교차하는 구조다:

```
[메인 캠페인 클리어 → 액트 10]
          ↓
[Atlas of Worlds 매핑]
   - Waystones (구 Map) 티어 1~17
   - Atlas Passive Tree 전문화
          ↓
[리그 메카닉 레이어 선택]
   - Breach / Delirium / Expedition / Ritual 등 병행
          ↓
[메이븐 증인 (Maven Witness) + 샤이퍼 보스킬]
          ↓
[Feared Pinnacle Bosses: 아탈라이 코리스트 등]
```

| 항목 | 수치 |
|:---|:---|
| 매핑 1회(T16 맵) 소요 시간 | 3~8분 (빌드에 따라) |
| 일반 T17 맵 1회 | 10~20분 |
| 리그 시즌 기간 | 약 13주 (3개월) |
| Atlas Passive 포인트 | 최대 140개 (시즌마다 초기화) |
| Divine Orb(최상위 재화) 시즌 초 → 말 가치 변화 | 초반 과공급 → 후반 물가 안정 패턴 |

#### PoE의 자원 계층 구조

PoE의 Currency 시스템은 "거래소" 없이 **아이템 자체가 화폐**인 독특한 구조:

```
[하위 재화]           [중위 재화]         [상위 재화]
Orb of Alteration  → Chaos Orb      → Divine Orb
(1티어 강화)         (리롤/가격 기준)   (값비싼 강화)

          ↓ 시즌 내 환율 (2026년 기준 레퍼런스)
1 Divine = 약 180~250 Chaos (시즌 초반)
1 Divine = 약 100~150 Chaos (시즌 후반)
```

**핵심 경제 원칙:** PoE는 아이템 자체를 강화 재료로 소모시켜 **강력한 Sink**를 만든다. 매핑 + 크래프팅을 통한 Faucet(공급)과 크래프팅 시 재화 소비(Sink)가 자연스럽게 균형을 이룬다.

#### 리그 경제 리셋의 건강성

리그 시즌 시작 시 모든 경제가 0에서 리셋된다. 이 구조의 효과:

| 효과 | 설명 |
|:---|:---|
| 공급 인플레이션 방지 | 리그 종료 시 고가 아이템의 대량 공급이 새 리그에 이월되지 않음 |
| 신규/복귀 플레이어 진입 장벽 완화 | 리그 초반은 베테랑도 빈 손으로 시작 |
| 경제 실험 가능 | 리그마다 새 메카닉 추가로 경제 실험 반복 (Bulk Trading, Div Card 등) |
| 빌드 다양성 확보 | 메타가 고정되지 않고 리그마다 재편 |

**설계 교훈:** PoE의 리그 경제는 "인플레이션이 쌓이기 전에 리셋"하는 예방적 경제 정책이다. 리셋 자체가 콘텐츠가 된다는 역설적 장점이 있다.

#### PoE와 D3 거래소 비교 — D3 실물 경매장 실패 사례

| 항목 | D3 실물 경매장(2012~2014) | PoE 트레이드 사이트 |
|:---|:---|:---|
| 방식 | 공식 거래소, 달러 거래 가능 | 비공식 크롤러(poe.trade → pathofexile.com/trade) |
| 결과 | 파밍 동기 완전 소멸 → 2년 만에 폐지 | 건강한 경제 유지 |
| 실패 이유 | 돈으로 최강 장비 즉시 구매 가능 → "왜 직접 파밍해야 하나?" | 직접 파밍 + 크래프팅이 거래보다 비용 효율적인 경우 多 |
| 핵심 교훈 | 거래소가 파밍을 대체하면 게임이 죽는다 | 거래소는 파밍을 보완해야 한다 |

### 1.3 워프레임 — 리셋 없는 누적형 경제

#### 워프레임 엔드게임 구조

워프레임은 리셋이 없는 누적형 진행 구조를 채택한다:

```
[스틸 패스(Steel Path) — 기본 미션 ×2 강화]
          ↓
[아비트레이션(Arbitrations) — 무한 생존 모드]
          ↓
[구인포드(Eidolon) / 오브 마더(Orb Mother) — 보스 레이드]
          ↓
[Zariman / Duviri — 신규 엔드게임 루프]
          ↓
[마스터리 랭크(MR) 30 달성 → Legendary MR]
```

| 항목 | 수치 |
|:---|:---|
| 마스터리 랭크 최대치 | MR 30 (Legendary 1~5) |
| 스틸 패스 1회 미션 | 8~20분 |
| 구인포드 삼중 사냥 1회 | 20~30분 |
| 프라임 워프레임 획득 소요 | 평균 5~40시간 (파밍 운에 따라) |
| 플래티넘(유료 재화) vs 구독 없는 플레이 | F2P로 모든 콘텐츠 접근 가능 |

#### 워프레임의 모딩 경제

워프레임 최강화의 핵심은 **모드 최대 랭크 업** 이다:

```
모드 기본 효과 + Endo(경험치 자원) + Credits(골드) → 최대 랭크 모드
Max Rank 모드 + Forma(리셋 자원) → 극성화(Polarity) → 에너지 절약 → 더 많은 모드 장착
```

| 자원 | Faucet(유입) | Sink(소비) |
|:---|:---|:---|
| Endo | 모든 미션 드랍, 듀얼(해체) | 모드 랭크업 |
| Credits | 모든 미션, 거래 | 모드 업그레이드, 아이템 구매 |
| Platinum(프리미엄) | 구매 or 플레이어 간 거래 | 슬롯 구매, 코스메틱, 가속 |
| Forma | 리워드 로테이션, 블루프린트 제작 | 극성화(소모품) |

**리셋 없는 누적형의 문제점:**
- 인플레이션: Credits, Endo 고인물 보유량이 수억 단위 → 신규 콘텐츠 출시 즉시 최강화
- 진입 장벽: 신규 플레이어가 고인물 따라잡기 사실상 불가능
- 콘텐츠 리텐션: "더 이상 파밍할 것이 없다"는 고인물의 이탈 문제

**설계 교훈:** 리셋 없는 누적형은 충성 유저의 지속 플레이를 보장하지만, 경제 건강성과 신규 유저 유입에 불리하다.

### 1.4 메이플스토리 — 강화 시스템과 파밍 경제

#### 스타포스(Starforce) 시스템 수치 분석

메이플스토리의 스타포스는 장비 강화의 핵심이다. 단계별 성공/실패/파괴 확률:

| 단계 | 성공률 | 실패 확률 (하락) | 파괴 확률 |
|:---|:---|:---|:---|
| 0 → 5성 | 95% | 없음 | 없음 |
| 6 → 10성 | 75~90% | 10~25% | 없음 |
| 11 → 14성 | 45~60% | 39~45% | 1~10% |
| 15 → 16성 | 30% | 59.4% | 0.6% |
| 17 → 20성 | 30% | 60% | 10% |
| 21 → 22성 | 2.1% | 77.9% | 20% |
| 23 → 24성 | 1.1% | 78.9% | 20% |

**기댓값 분석 (22성 달성 기준):**
- 이론적 평균 파괴 횟수: 장비당 약 3~7회 (운 의존도 매우 높음)
- 22성 달성에 드는 평균 메소(골드) 비용: 장비 등급/종류에 따라 수십억~수천억
- "놀이기구 이벤트(5/10/15성 점프)" 없이 순수 스타포스: 22성에 수백 시간 필요

**강화 실패/하락/파괴 메커니즘의 심리학:**
- 간헐적 강화(Variable Ratio Schedule): 언제 성공할지 모르기 때문에 포기하기 어려움
- 손실 회피(Loss Aversion): 파괴는 강력한 부정 감정 유발 → 큐브(재화) 과금 유인
- 선샤인(파괴 방지) 아이템: 파괴 불안을 상품으로 전환하는 수익화 메커니즘

#### 메이플스토리 잠재능력 & 큐브 시스템

잠재능력은 장비에 숨겨진 추가 스탯이다. 큐브(유료 재화)로 재롤(Reroll)하는 구조:

| 등급 | 최대 옵션 | 접근 방법 |
|:---|:---|:---|
| 레어 | 3줄 레어 | 무료 큐브, 장인 큐브 |
| 에픽 | 3줄 에픽 | 모험가의 큐브 (유료) |
| 유니크 | 3줄 유니크 | 명장 큐브 (유료) |
| 레전드리 | 3줄 레전드리 | 넥슨 큐브 (유료), 극히 드물게 수급 가능 |

**3줄 레전드리 핵심 옵션(보공/주공 12%×3) 달성 확률:** 이론상 0.0001% 이하. 실제 소요 큐브 수는 수천~수만 개. 업계에서 P2W의 대표 사례로 인용됨.

#### 핵심 루프 소요 시간

| 활동 | 1회 소요 시간 | 반복 동기 |
|:---|:---|:---|
| 보스(하드 루시드, 하드 빌레노드) | 20~40분 (주간 보스) | 보스 결정석(주간 한정 드랍) |
| 스타포스 이벤트 기간 파밍 | 세션당 1~3시간 | 22성 달성 체크 리스트 |
| 아케인 심볼 일일 콘텐츠 | 30~60분 | 아케인 포스 수치 상승 |
| 헥사 스텟 강화 | 세션당 1~2시간 | 데미지 배율 직접 상승 |

## 2. 자원 순환 설계 (Faucet / Sink)

### 2.1 자원의 유입 경로 (Faucet) — 업계 사례

#### 자원 Faucet의 분류

```
[즉시형 Faucet]         [지연형 Faucet]          [활동 보상형 Faucet]
몹 드랍 골드            일일/주간 퀘스트           보스 처치 보상
아이템 판매             시즌 패스 보상             업적 달성 보상
미션 클리어 보상        오프라인 누적 보상         멀티플레이 협동 보너스
```

| 자원 유형 | D3/D4 | PoE | 워프레임 | 메이플 | 디스가이아 |
|:---|:---|:---|:---|:---|:---|
| 골드(주 화폐) | Gold: 몹/NPC 드랍 | - (없음) | Credits: 모든 미션 | 메소: 몹/미션 | HL: 몹/아이템계 |
| 강화 재화 | Legendary Gem | Orb of Alteration | Endo | 스타포스 메소 | 이노센트 |

| 소모품 | Crafting Materials | Fossils/Resonators | Forma | 스타포스 점프권 | Mr. Gency's Exit |

#### 자원 인플레이션 사례와 교훈

**디아블로 3 Gold 인플레이션 (2012):**
- 런칭 초 1골드 = 실제 0.00001달러 수준
- 6개월 후 골드 수십억 보유자 속출 → 실물 경매장에서 골드 가치 붕괴
- 해결책: Gold Sink 강화 (수리비 대폭 인상, 소모품 비용 증가) + 실물 경매장 폐지

**메이플스토리 메소 인플레이션:**
- 메소 메이킹 콘텐츠 추가마다 메소 공급 급증
- 거래소 가격 지속 상승 → "강화 비용도 오르고, 아이템 구매 비용도 오르는" 악순환
- 해결책: 메소 획득 한도(일일 캡), 강화 비용 자체 증가, 메소 시세 직접 개입

### 2.2 자원의 소멸 경로 (Sink) — 설계 원칙

효과적인 Sink의 3가지 조건:
1. **자발적 소비**: 강제 소모가 아닌, 더 강해지기 위한 선택적 소비
2. **비가역성**: 소비한 자원은 돌아오지 않음 (파괴, 소모 등)
3. **규모 확장성**: 강해질수록 더 많은 자원이 소비됨 (선형 or 지수 스케일)

| Sink 유형 | 사례 | 소비량 특성 |
|:---|:---|:---|
| 강화 비용 | 스타포스, D4 Masterworking | 지수적 증가 (고단계일수록 폭발적 증가) |
| 강화 실패 소멸 | 스타포스 파괴, PoE 크래프팅 실패 | 리스크 기반 |
| 소모품 소비 | PoE Orb 소비, 워프레임 Forma | 사용 즉시 소멸 |
| 거래 수수료 | PoE 비공식: 없음, D3 경매장: 15% | 거래량 × 수수료율 |
| 수리비 | D3 Gold, 메이플 메소 | 활동량 비례 |
| 아이템 분해 | 워프레임 중복 모드 해체 | 중복 아이템을 Endo로 변환 |

### 2.3 인플레이션 제어 전략 — 업계 검증된 방법론

#### 전략 1: 듀얼 커런시 시스템 (PoE 모델)
가장 가치 있는 자원(Divine Orb)의 공급을 극도로 제한하고, 일반 자원(Chaos Orb)은 자유롭게 흘러다니게 한다. 자연스러운 가격 계층 형성.

#### 전략 2: 활동 기반 한도 (메이플 모델)
일일/주간 콘텐츠 도전 횟수 한도 → 무한 파밍 방지 → 자원 공급량 예측 가능

#### 전략 3: 소비 확장형 Sink (D4 Masterworking 모델)
장비를 강하게 만들수록 더 많은 재료가 필요 → 강해질수록 더 많이 파밍해야 → Faucet과 Sink가 자동으로 균형

#### 전략 4: 완전 경제 리셋 (PoE 리그 모델)
3개월마다 경제 리셋 → 인플레이션 누적 차단 → 매 시즌 균등한 출발점

### 2.4 Project Abyss 자원 흐름 설계 방향

```
[월드 (World)] ─────────────────────────────
   Faucet:                    Sink:
   - 몹 처치 → HL(골드)       - 소모품 구매 (탈출 아이템 등)
   - 몹 드랍 → 장비 아이템     - 허브 상점 구매
   - 보스 처치 → 희귀 재료     - 아이템계 진입 비용 (논의 필요)
   - 탐험 완료 → 맵 소재       - 이노센트 이식 비용 (Phase 2)

[아이템계 (Item World)] ──────────────────────
   Faucet:                    Sink:
   - 이노센트 드랍 (야생)       - 이노센트 복종 비용 (HL 소모)
   - 보스 처치 → 레벨 구슬     - 탈출 아이템 소모
   - 층 클리어 → 이노센트 EXP  - 아이템계 재진입 시 입장 재화
   - 미스터리 룸 → 특수 보상   - 강화 실패 없음 (선형 성장 철학)

[허브 (Hub)] ────────────────────────────────
   Faucet:                    Sink:
   - 거래소 → 아이템 판매 HL   - 거래소 수수료 (5~10% 권장)
   - 장비 분해 → 소재 변환     - 합성/강화 비용
   - 퀘스트/이벤트 보상        - 이노센트 이식 비용 (Phase 2)
```

**월드 → 아이템계 → 허브 순환의 경제적 균형 핵심 규칙:**
1. 아이템계는 HL을 소비하는 주요 Sink (이노센트 복종 = HL 소모)
2. 월드는 HL의 주요 Faucet (몹 드랍, 보스 처치)
3. 허브는 HL의 균형 교환소 (거래 수수료로 유통 HL 흡수)

## 3. 강화/성장 곡선 설계

### 3.1 성장 곡선 유형 비교

#### 선형 성장 (Linear Growth)
```
레벨 1: 스탯 100
레벨 2: 스탯 110 (+10)
레벨 3: 스탯 120 (+10)
레벨 N: 스탯 100 + (N-1) × 10
```
**장점:**
- 플레이어가 성장을 직관적으로 예측 가능
- "1층 더 클리어하면 스탯이 얼마나 오르나"를 정확히 인지
- 밸런싱이 단순하고 예측 가능
- 디스가이아 전 시리즈가 채택한 모델 (층당 +5~6.6%)

**단점:**
- 후반 성장의 "돌파감"이 약함
- 초반과 후반의 성장 체감 차이 없음

**채택 사례:** 디스가이아 1~7, Project Abyss MVP 단계

#### 지수 성장 (Exponential Growth)
```
레벨 1: 스탯 100
레벨 2: 스탯 120 (+20%)
레벨 3: 스탯 144 (+20%)
레벨 N: 스탯 100 × 1.2^(N-1)
```
**장점:**
- 후반 성장이 폭발적 → 강해지는 쾌감 극대화
- 고인물과 신규 플레이어 파워 격차 자연 형성 → 목표 의식

**단점:**
- 후반 숫자가 관리 불가능하게 커짐 (메이플스토리 데미지 = 수십억)
- 밸런싱 복잡도 폭발적 증가
- 콘텐츠 수명 단축 (고인물이 신규 콘텐츠를 즉시 솔로 클리어)

**채택 사례:** 메이플스토리, 디아블로 3 파라곤 후반

#### 로그 성장 (Logarithmic Growth)
```
레벨 1: 스탯 100
레벨 10: 스탯 200 (+100%)
레벨 100: 스탯 300 (+50%)
레벨 1000: 스탯 400 (+33%)
```
**장점:**
- 초반 성장이 빠르고 후반 성장이 점진적 → "실력 기반" 체감
- 고인물과 신규 플레이어 격차에 자연스러운 상한선
- 레벨이 높아질수록 추가 투자의 한계효용 감소 → 경쟁 보존

**단점:**
- 야리코미 플레이어에게 성장 한계의 좌절감
- "후반에 투자해봐야 의미 없다"는 인식 형성 가능

**채택 사례:** 대부분의 MMO 레벨링 곡선, 워프레임 마스터리 랭크

#### Project Abyss 권장 곡선: 선형 + 구간 보너스 하이브리드

```
기본 성장: 선형 (이노센트 복종 시 직접 스탯 +N)
           ↓
구간 보너스: 10층마다 보스 처치 시 기존 스탯의 X% 추가
           ↓
희귀 이노센트: 완전 복종 시 레어리티별 배율 보너스
```

이 구조는 디스가이아의 선형 성장 철학을 유지하면서, GDC 인사이트의 "Make it Bumpy" 원칙을 적용한다. 매 10층은 가속 구간(보스 보너스), 나머지는 안정 구간(이노센트 복종)으로 리듬을 형성한다.

### 3.2 강화 실패/하락 메커니즘의 심리학

#### 실패 메커니즘의 심리학적 분석

**메이플스토리 스타포스 — 부정적 강화(Negative Reinforcement) 모델:**

```
[심리 사이클]
성공 기대 → 실패 → 좌절/하락 → 메소/큐브 소모 → 성공 → 일시적 쾌감
    ↑_______________________________________↑
              반복 (가변 비율 강화 스케줄)
```

- 파괴는 최악의 경험이지만 "보호 아이템 구매" 동기를 만듦
- 하락(성공 실패 시 성물 단계 감소)은 진행도 손실 → 더 많은 플레이 시간 요구
- 업계 비판: "심리적 조작을 통한 수익화"로 규제 대상이 되는 추세 (벨기에, 네덜란드 규제)

**디스가이아 — 무실패 선형 성장 모델:**

```
[심리 사이클]
층 진입 → 클리어 → 아이템 레벨 +1 → 더 강해짐 → 다음 층 도전
    ↑___________________________________________________↑
              반복 (고정 비율 강화 스케줄 → 명확한 기대감)
```

- 실패가 없으므로 좌절감 없이 "얼마나 더 투자할 것인가"의 자발적 선택
- 야리코미의 핵심: "내가 투자한 만큼 강해진다"는 확실성이 극한 투자를 가능케 함
- 위험은 아이템계 탈출 실패 시 **진행도 손실**로 구현 (Mr. Gency's Exit 없이 탈출 불가)

**Project Abyss 무실패 원칙과 긴장감 균형:**

| 요소 | 실패 없음 | 긴장감 원천 |
|:---|:---|:---|
| 이노센트 복종 | 항상 성공, 복종 수치만 오름 | 희귀 이노센트 출현 확률 (RNG) |
| 층 클리어 | 클리어 시 반드시 스탯 증가 | 사망 시 진행 층 손실 (긴장감 유지) |
| 레어리티 강화 | 기존 이노센트 유지 | 강화에 드는 시간/아이템계 재도전 |
| 아이템 드랍 | 드랍률은 고정 (LCK 보정) | "갓 롤" 이노센트 출현 = 서프라이즈 |

### 3.3 성장 체감 구간(Plateau) 설계

성장 곡선에서 플레이어가 "정체감"을 느끼는 구간을 의도적으로 설계하고, 돌파 시 강한 보상을 제공하는 것이 야리코미 동기 유지의 핵심이다.

**GDC 인사이트 "Make it Bumpy" 원칙 적용:**

```
[아이템계 층별 성장 밀도 설계]

1~9층:   이노센트 복종 보너스 (낮은 밀도)
10층:    ★ 아이템 장군 처치 → 스탯 +보너스 (가속 구간)
11~19층: 이노센트 복종 보너스 (낮은 밀도)
20층:    ★ 아이템 장군 처치 + 미스터리 룸 확률 증가 (가속 구간)
...
30/60/90층: ★★ 아이템 왕 처치 → 스탯 +보너스×2 (강한 가속)
100층:   ★★★ 아이템 신/대신 처치 → 큰 보상 (하이라이트)
```

**체감 구간 돌파 보상 설계:**

| 구간 | 정체감 원인 | 돌파 보상 |
|:---|:---|:---|
| 1~9층 → 10층 보스 | 이노센트 소량 복종, 작은 증가 | 보스 처치 스탯 보너스 (즉각 체감) |
| 지층 1 → 지층 2 | 지층 간 전환 준비 시간 | 신규 이노센트 종류 해금, 난이도 상승 |
| 일반 이노센트 → 희귀 이노센트 | 반복 파밍 | 희귀 이노센트 출현 → 강한 서프라이즈 |
| Legendary → Ancient | Ancient 드랍까지 긴 파밍 | x3.0 배율 + 8 이노센트 슬롯 (극적 파워업) |

## 4. 드롭률/파밍 효율 설계

### 4.1 레어리티별 드롭률 — 업계 표준 비교

| 게임 | 가장 희귀 등급 | 드랍률 | 파밍 방법론 |
|:---|:---|:---|:---|
| 디아블로 3 | Primal Ancient | ~1% (Ancient 대비) | GR 100+ 반복 |
| 디아블로 4 | Mythic Unique | ~0.03~0.1% | Uber 보스 처치 |
| Path of Exile | Mirror of Kalandra | 0.001%대 (역사적) | 대량 매핑 or 거래 |
| 워프레임 | 특정 Riven Mod | 0.2~2% | 미션 타입별 상이 |
| 메이플스토리 | 아케인 드랍 일부 | 0.01~0.1% | 보스 주간 한도 |
| 디스가이아 1 | Legendary 아이템 | 0.8% (Legendary + 랭크별) | 수육계 도적 훔치기 |

**Project Abyss 레어리티 드랍률 기준값 (System_Equipment_Rarity.md 기준):**

| 등급 | 드랍 비중 | 업계 비교 |
|:---|:---|:---|
| Normal | 60% | D3 Uncommon(70%)보다 낮음 → 적절 |
| Magic | 25% | D3 Magic(20%)과 유사 → 적절 |
| Rare | 10% | D3 Rare(7%)보다 약간 높음 → 관대한 편 |
| Legendary | 4% | D3 Legendary(2%)보다 높음 → 관대한 편 |
| Ancient | 1% | D3 Primal(~0.1%)보다 월등히 높음 → 매우 관대 |

**권장 조정 방향:** Phase 1 MVP에서는 현행 유지. Phase 2에서 LCK 스탯 보정 시스템이 도입되면, 기본 드랍률을 약간 낮추고 LCK에 따른 보정폭을 넓혀 파밍 투자의 의미를 강화하는 것을 권장.

### 4.2 파밍 효율 차등: 수동 vs 자동

**Project Abyss "수동 정복 선행" 원칙의 경제적 함의:**

자동 사냥은 직접 클리어한 구역에서만 해금된다. 이 원칙이 경제에 미치는 영향:

| 영향 요소 | 수동 플레이 | 자동 사냥 |
|:---|:---|:---|
| 드랍 효율 | 100% (기준) | 50~70% (권장 차등) |
| 희귀 아이템 드랍 | Legendary/Ancient 드랍 가능 | Legendary 이하만 (Ancient 불가 권장) |
| 이노센트 종류 | 전 종류 | 일반 이노센트만 |
| 골드 획득 | 100% | 70~80% |
| 경험치 | 100% | 60~70% |

**자동 사냥 효율 차등의 근거:**

1. **그라인드 방지**: 자동 사냥이 수동보다 효율적이면 게임의 재미가 파괴됨
2. **수동 플레이 인센티브 유지**: 보스 처치, 비밀방 발견 등 수동에서만 가능한 고가치 보상
3. **GDC Albion Online 원칙 적용**: "가장 재미있는 콘텐츠"와 "가장 높은 보상"이 일치하도록

**Legendary 이상 자동 사냥 드랍 금지 규칙 (Project_Vision_Abyss.md 준수):**
- 최고 등급 아이템은 반드시 수동 플레이 중에만 드랍
- 자동 사냥으로 인한 최강 아이템 누적 차단
- "Ancient를 얻으려면 직접 플레이해야 한다"는 명확한 인센티브

### 4.3 타겟 파밍 vs 랜덤 파밍의 균형

#### 타겟 파밍 (Target Farming)
특정 보스나 구역에서 특정 아이템이 높은 확률로 드랍되는 시스템.

**장점:** 파밍 목표가 명확해 집중력 유지. 달성 가능성에 대한 기대감.
**단점:** 특정 구역만 반복 플레이 → 다른 콘텐츠 소외.
**사례:** D3의 Specific Bounty 보상, 메이플스토리의 보스별 전용 드랍.

#### 랜덤 파밍 (Random Farming)
어느 구역/몹에서도 모든 아이템이 드랍될 수 있는 시스템.

**장점:** 어디서나 기대감 유지. 모든 콘텐츠에 가치 부여.
**단점:** "최강 아이템이 언제 드랍될지 모름"으로 인한 무기력감 가능.
**사례:** D3 기본 파밍, 워프레임 비전용 드랍.

**Project Abyss 권장 균형:**

```
[구역별 특화 드랍 풀]     [공통 드랍 풀]
지하 수로 → DEX 특화    +  Normal/Magic 아이템 (어디서나)
빙결 동굴 → VIT 특화    +  공통 소모품 (탈출 아이템 등)
묘지/카타콤 → LCK 특화  +  HL(골드) (어디서나)
천공의 탑 → SPD 특화

[보스 전용 드랍]          [아이템계 전용 드랍]
각 구역 보스             이노센트 (야생 상태)
→ 해당 스탯 고정 옵션    희귀 이노센트
  Legendary 이상          레벨 구슬
```

이 구조는 "어디서나 파밍 가능하지만, 원하는 스탯을 빠르게 얻으려면 특정 구역을 공략해야 한다"는 타겟 파밍의 전략성과 랜덤 파밍의 기대감을 동시에 제공한다.

### 4.4 "갓 롤" 확률과 야리코미 동기 유지

#### "갓 롤"의 정의와 설계

갓 롤(God Roll)은 이론적 최대치에 근접한 이노센트 조합을 의미한다. Project Abyss에서의 갓 롤:

```
[갓 롤 조건 예시 — Ancient 검 기준]
이노센트 슬롯 8개 모두 채움
+ STR 특화 이노센트 3개 (최고 등급)
+ ATK% 이노센트 2개 (최고 등급)
+ Statistician(경험치 증가) 이노센트 1개
+ LCK 이노센트 1개 (드랍률 보정)
+ 특수 이노센트(고유 효과) 1개

= 동일 레어리티 평균 대비 150~200% 이상 효율
```

**갓 롤 확률 목표값:**
- 단일 이노센트 슬롯에 원하는 최고 등급 이노센트 배치 확률: ~1~3%
- 슬롯 8개 모두 원하는 구성 확률: 수천~수만 분의 1
- 이 확률이 야리코미의 궁극 목표가 되어 극한 반복 플레이를 동기화

**배드 럭 프로텍션 (Bad Luck Protection) — 업계 사례:**

| 시스템 | 게임 | 작동 방식 |
|:---|:---|:---|
| 천장 시스템(Pity) | 가챠 게임 | N회 미획득 시 확정 획득 |
| 스파크(Spark) | 원신, HSR | 일정 획득 수 누적 → 선택 획득 |
| 페이트 포인트 | D4 Uber Uber Unique | 2회 연속 미드롭 → 3번째 확정 |
| 통계적 보정 | PoE 다수 아이템 | 실제 드랍률이 장기 평균으로 수렴하도록 조정 |

**Project Abyss 권장 배드 럭 보정:**

```
[희귀 이노센트 배드 럭 보정]
연속 N층 클리어 시 희귀 이노센트 미출현 → 다음 층 출현 확률 점진적 상승
상한: 기본 확률의 최대 3배 (예: 기본 2% → 최대 6%)
리셋: 희귀 이노센트 출현 시 카운터 초기화

[Ancient 드랍 배드 럭 보정]
해당 구역에서 N회 연속 Legendary 이하 드랍 시
다음 드랍에 Ancient 확률 소폭 증가
상한: 기본 1% → 최대 3%
```

## 5. 거래소/경제 시스템

### 5.1 자유 거래 vs 제한 거래 vs 귀속 시스템 비교

| 방식 | 자유 거래 | 부분 거래 (귀속 혼합) | 귀속(BoP) |
|:---|:---|:---|:---|
| 사례 | PoE, Albion Online | 메이플스토리, 워프레임 | D3 (Seasons 이후), D4 일부 |
| 파밍 동기 | 파밍 = 경제 참여 → 유지됨 | 파밍 + 거래 병행 | 직접 파밍 강제 |
| RMT 위험 | 높음 | 중간 | 낮음 |
| 격차 | 돈 많은 유저 유리 | 중간 | 플레이 시간 = 파워 |
| 커뮤니티 | 경제 참여로 강한 소속감 | 중간 | 솔로 플레이 성향 |

#### 디아블로 3 실물 경매장 실패 심층 분석

2012년 5월 런칭한 D3 Real-Money Auction House(RMAH)는 2014년 3월 폐지됐다. 폐지 이유:

1. **파밍 동기 완전 소멸**: "파밍하는 것보다 경매장에서 사는 게 효율적" → 게임 핵심 루프 붕괴
2. **아이템 드랍 밸런스 왜곡**: 경매장 수익을 위해 드랍률을 의도적으로 낮추었다는 의혹 (Blizzard 내부 인정)
3. **사행성 논란**: 아이템 강화 결과물을 현금으로 거래 = 사실상 도박
4. **시장 독점**: 소수의 파워 유저가 고가 아이템 시장 장악 → 신규 유저 진입 비용 폭등

**핵심 교훈**: "거래소는 파밍의 동기를 약화시키지 않는 방향으로 설계해야 한다." 파밍으로 얻은 아이템의 가치가 거래소 구매보다 높아야 한다 — 최소한 체감적으로라도.

#### Path of Exile 비공식 거래의 건강성

PoE는 공식 거래소 없이 크롤러 기반 비공식 거래(pathofexile.com/trade)를 운영한다. 이 구조의 장점:

- 거래 불편함이 파밍 동기를 유지 (편리한 거래소가 없으므로 직접 파밍이 종종 더 빠름)
- League 경제가 리셋되므로 독점 방지
- Bulk Trading(대량 거래)만 편의성 지원 → 소량 고가 아이템은 직접 거래가 필수

### 5.2 거래소가 파밍 동기에 미치는 영향

**거래소 설계의 황금률 (Albion Online GDC 강연 기반):**

```
파밍 동기 = f(파밍 효율 - 거래 가격)

파밍 > 구매 → 파밍 동기 유지
파밍 ≒ 구매 → 취향에 따라 선택
파밍 < 구매 → 파밍 동기 소멸 (D3 RMAH 문제)
```

**효과적인 거래소 설계 원칙:**
1. **수수료**: 거래 금액의 5~10% 수수료로 HL Sink 형성
2. **등재 한도**: 동시에 올릴 수 있는 아이템 수 제한 (독점 방지)
3. **가격 투명성**: 모든 거래 내역 공개 (조작 방지)
4. **귀속 아이템 구분**: Ancient, 특정 퀘스트 아이템 등은 거래 불가 (최강 장비는 직접 파밍 필수)

### 5.3 RMT 방지 전략

| 전략 | 설명 | 사례 |
|:---|:---|:---|
| 시즌 리셋 | 계정 간 자산 이월 차단 | PoE, D3 Seasons |
| 최고 등급 귀속 | Ancient 급은 거래 불가 | D4 Unique BoA |
| 일일 거래 한도 | 계정당 일일 거래 횟수/금액 제한 | 메이플스토리 |
| 아이템 교환 이력 추적 | 거래된 아이템 출처 추적 | Albion Online |
| IP/기기 제한 | 다중 계정 RMT 봇 차단 | 대부분의 온라인 게임 |
| 실명 인증 | 한국 게임 법적 의무화 | 메이플스토리 |

**Project Abyss 권장 RMT 방지 설계:**
- Ancient 등급 장비 귀속 (직접 파밍 필수)
- 이노센트는 아이템 귀속 (아이템 거래 시 이노센트 소멸 or 거래 불가)
- 시즌 단위 리셋으로 자산 누적 차단

### 5.4 Project Abyss 허브 거래소 설계 방향

**거래 가능 아이템:**

| 아이템 등급 | 거래 가능 여부 | 이노센트 포함 여부 |
|:---|:---|:---|
| Normal | 가능 | 이노센트 소멸 후 거래 |
| Magic | 가능 | 이노센트 소멸 후 거래 |
| Rare | 가능 | 이노센트 소멸 후 거래 |
| Legendary | 제한 거래 (획득 후 N시간 이내만) | 이노센트 소멸 후 거래 |
| Ancient | 귀속 (거래 불가) | 해당 없음 |

**이노센트 소멸 거래 방식의 설계 의도:**
- 이노센트를 키워 강화한 아이템은 본인만 사용 가능 → 파밍 개인 성취 보존
- 새로 드랍된 (이노센트 미복종) 아이템만 거래 → 드랍 자체에 경제적 가치 부여
- 이노센트 이식(Phase 2)이 도입되면 이식 비용(HL) 소모 후 거래 가능하도록 확장

## 6. 시즌/리셋 메커니즘

### 6.1 디아블로 시즌 리셋 — 설계 철학 분석

#### 왜 리셋을 하는가?

**공식 이유 (Blizzard):**
- 경제 인플레이션 방지 (너무 많은 고급 아이템이 누적되면 신규 드랍의 의미 퇴색)
- 신규/복귀 플레이어 진입점 제공 (시즌 시작 = 모두가 동일한 출발선)
- 시즌 테마 메카닉 실험 (임시 메카닉으로 피드백 수집 → 상설화 결정)

**플레이어 반응 패턴:**
- 시즌 시작 후 1~2주: 최고 동접 유지 (파워 레이스)
- 시즌 중반(4~6주): 엔드게임 달성 후 이탈 시작
- 시즌 후반(10~12주): 핵심 고인물만 잔류 (리더보드 경쟁)
- 시즌 종료 직전: 시즌 여정 미완료자 복귀 (마감 효과)

**D3 시즌 리셋 후 비시즌 이관:**
- 시즌 캐릭터 → 비시즌으로 이관
- 파라곤 포인트 누적 → 고인물과의 영구 격차 문제

**D4 개선:** 시즌 캐릭터 이관 시 일부 진행도만 영구 반영 (Account Power), 경제적 자산은 이관되지 않음.

### 6.2 PoE 리그 — 경제 리셋의 건강성

**PoE 리그의 핵심 설계 원리:**

```
리그 시작
   ↓
새 메카닉 학습 (리그 고유 시스템)
   ↓
파워 성장 + 경제 성장 동시 진행
   ↓
리그 엔드게임 달성
   ↓
리그 종료 → 영구 서버(Standard)로 이관
   ↓
다음 리그 시작 (경제 완전 리셋)
```

**리그 경제의 건강성 요인:**

| 요인 | 설명 |
|:---|:---|
| 완전 리셋 | 인플레이션 누적 차단. 고인물도 리그마다 0에서 시작 |
| Standard 이관 | 리그에서 키운 캐릭터/아이템 영구 보존 → 노력 가치 유지 |
| 리그 메카닉 실험 | 매 리그 새 경제 메카닉 실험 → 성공 시 상설화 |
| 커뮤니티 경제 연구 | 리그 초반 경제 분석이 커뮤니티 콘텐츠가 됨 |

### 6.3 워프레임 — 리셋 없는 누적형의 장단점

**장점:**
- 투자한 시간/노력이 영원히 보존됨 → 충성 유저에게 강한 보상
- "내가 몇 년에 걸쳐 키운 워프레임"이라는 소유감
- 신규 콘텐츠 즉시 접근 가능 (강한 캐릭터로 신규 지역 탐험)

**단점:**
- 신규 유저 진입 장벽 (고인물 장비 차이 극복 불가)
- 인플레이션: Credits, 구식 아이템의 가치 폭락
- "더 이상 파밍할 것이 없다"는 콘텐츠 부족 문제 → DE가 지속적 신규 콘텐츠로 대응
- 고인물-신규 격차가 멀티플레이 균형을 파괴 (MR 20+ vs MR 1 파티)

### 6.4 Project Abyss 적합한 모델 — 하이브리드 추천

**추천 모델: "소프트 리셋 + 영구 레거시" 하이브리드**

```
[시즌 구조]
시즌 기간: 3~4개월
               ↓
[시즌 진행]
월드 시드 재생성 (새 지도 탐험)
+ 새 시즌 테마 아이템/이노센트 추가
+ 시즌 전용 화폐(레거시 코인) 적립 가능
               ↓
[시즌 종료 시 처리]
캐릭터 레벨: 영구 보존 (비시즌 서버로)
장비 아이템: 영구 보존 (창고로 이관)
이노센트: 장비에 귀속이므로 함께 보존
HL(골드): 일부 리셋 (상한 제한, 초과분 레거시 코인으로 전환)
시즌 전용 화폐: 영구 화폐(레거시 코인)로 전환
               ↓
[다음 시즌 시작]
새 월드 시드로 탐험 재개
새 시즌 테마 이벤트 + 아이템
HL 리셋으로 경제 신선도 유지
```

**이 모델의 근거:**

| 요소 | 리셋하는 이유 | 보존하는 이유 |
|:---|:---|:---|
| HL(골드) | 인플레이션 방지 | - |
| 월드 지도 | 탐험 신선도 유지 | - |
| 캐릭터 레벨 | - | 레벨업 노력 보존 |
| 장비/이노센트 | - | 아이템계 야리코미 가치 보존 |
| 능력(렐릭) | - | 탐험 성취 보존 |

## 7. Project Abyss 3-Space 순환 경제 설계 시사점

### 7.1 월드 → 아이템계 → 허브 순환의 경제적 균형

**순환 구조도 (경제 흐름)**

```
[월드] ──아이템 드랍──▶ [아이템계]
  ↑                            │
  │                      이노센트 획득
  │                      장비 강화
  │                            │
  │  스탯 게이트 해금           ↓
  └──────────────── [허브] ◀──거래/합성
                         │
                  HL 소비(수수료/합성비)
                         │
                    HL Sink 형성
```

**핵심 경제 지표 목표값:**

| 지표 | 목표값 | 측정 방법 |
|:---|:---|:---|
| 시간당 HL 획득 (수동 플레이) | 기준값 X | 구역별 몹 HL 드랍 합산 |
| 시간당 HL 획득 (자동 사냥) | 기준값 0.6X~0.7X | 수동 대비 70% 미만 |
| 이노센트 완전 복종까지 소요 시간 | 2~5시간/지층 | 아이템계 탐험 시간 기반 |
| Ancient 아이템 최초 획득까지 | 30~100시간 | 드랍률 × 파밍 효율 |
| 아이템계 Ancient 갓 롤까지 | 100~500시간+ | 야리코미 콘텐츠 |

### 7.2 "더 깊은 아이템계를 위해 더 넓은 월드를 탐험해야 하는" 순환 강화 전략

이것이 Project Abyss의 핵심 순환 루프이자 가장 중요한 경제적 설계 원칙이다:

**순환 강화 단계별 분석:**

```
Step 1: 월드 신규 구역 탐험 → Legendary 장비 드랍
Step 2: Legendary 아이템계 4지층 도전 → 강화 이노센트 확보
Step 3: 강화된 장비 착용 → STR/VIT/INT 등 스탯 상승
Step 4: 스탯 게이트 해금 → 다음 월드 구역 접근 가능
Step 5: 새 구역 탐험 → 더 높은 기반 스탯 Ancient 드랍
Step 6: Ancient 아이템계 4지층+심연 도전 → 극강 이노센트
Step 7: 극강 장비로 고레벨 월드 보스 처치 → 최고 희귀 드랍
    ↑_______________________________↑
            야리코미 핵심 루프
```

**각 단계별 경제적 밸브(Valve):**

| 밸브 위치 | 역할 | 튜닝 지점 |
|:---|:---|:---|
| 스탯 게이트 수치 | 진행 속도 조절 | 너무 낮으면 아이템계 불필요, 너무 높으면 진입 장벽 |
| 아이템계 이노센트 복종 효율 | 강화 속도 조절 | 복종 속도 = 스탯 게이트 돌파 빈도 |
| Ancient 드랍률 | 최강 등급 접근 속도 | 너무 빠르면 엔드게임 소진, 너무 느리면 이탈 |
| 이노센트 슬롯 상한 | 파워 상한선 | 무한 성장 방지 |

### 7.3 1~4인 멀티플레이에서의 보상 분배 공정성

**파티 보상 분배 원칙 (업계 사례 기반):**

| 방식 | 사례 | 장단점 |
|:---|:---|:---|
| 개인 드랍(Personal Loot) | D3, D4 | 각자 파밍 → 협동 필요성 약화 |
| 자유 드랍(FFA Loot) | 메이플스토리 구형 | 경쟁 → 갈등, 하지만 강한 긴장감 |
| 투표 분배(Loot Council) | MMORPG 레이드 | 길드 내 공정성, 밖에서는 부적합 |
| 기여도 기반 분배 | Albion Online | 공정하지만 계산 복잡 |

**Project Abyss 권장 방식:**

```
[아이템계 협동 보상 구조]

이노센트 드랍: 개인 드랍 (각자 아이템계에서 나온 이노센트는 본인 귀속)
HL 드랍: 파티원 균등 분배 (협동 인센티브)
아이템 드랍: 개인 드랍 (본인 아이템계에서 드랍된 아이템)
보스 처치 보너스: 파티 전체 스탯 보너스 (협동 보상)
```

**파티 플레이 인센티브 (솔로 대비):**

| 인원 | 이노센트 출현 보너스 | HL 획득 보너스 | 보스 처치 보너스 |
|:---|:---|:---|:---|
| 솔로 (1인) | 기준 100% | 기준 100% | 기준 100% |
| 2인 파티 | +10% | +20% | +10% |
| 3인 파티 | +15% | +40% | +20% |
| 4인 파티 | +20% | +60% | +30% |

이 수치는 파티 플레이가 솔로보다 시간 효율이 높지만, 아이템계의 난이도도 파티 인원에 따라 상승하도록 설계해야 균형이 맞는다.

### 7.4 야리코미 유저와 캐주얼 유저의 성장 격차 관리

**격차 구조 분석:**

```
[캐주얼 유저]          [미드코어 유저]        [야리코미 유저]
일일 1~2시간          일일 2~4시간           일일 4시간+
Normal~Rare 파밍      Legendary 파밍         Ancient 극강화
이노센트 기본 복종    이노센트 최적화        이노센트 갓 롤 추구
월드 일부 탐험        월드 대부분 탐험       월드 완전 탐험 + 재귀 아이템계
```

**격차 관리 전략:**

| 전략 | 설명 | 적용 방법 |
|:---|:---|:---|
| 스탯 게이트 중간 설정 | 캐주얼도 대부분의 월드를 탐험 가능 | Legendary 없이 도달 가능한 게이트 수 = 전체의 70% |
| 콘텐츠 수직 분리 | 야리코미 콘텐츠 = 아이템계 심층 + 수육계 | 캐주얼은 참여 안 해도 월드 클리어 가능 |
| 자동 사냥 차등 | 캐주얼의 오프라인 진행을 어느 정도 보장 | 자동 사냥 Normal~Rare 구간에서 작동 |
| 배드 럭 보정 | 장시간 파밍 없이도 최소한의 Legendary 획득 보장 | N회 Legendary 미드랍 시 확률 증가 |

**Power Gap 목표 수치 (Albion Online GDC 원칙 적용):**

```
캐주얼 유저의 최대 파워: 기준 100%
미드코어 유저의 최대 파워: 기준 150~180%
야리코미 유저의 최대 파워: 기준 300~400%

캐주얼 vs 야리코미 최대 파워 비율: 1:3~1:4 이내로 제한
```

이 비율을 초과하면 캐주얼 유저가 멀티플레이에서 짐이 되는 느낌을 받아 이탈한다. D3가 이 원칙을 잘 지켜 시즌마다 캐주얼 유저의 참여율이 높았다.

## 8. 경제 건강 지표 (Economic Health Metrics)

### 8.1 모니터링 지표 목록

| 지표 | 정상 범위 | 이상 신호 |
|:---|:---|:---|
| 시간당 평균 HL 획득 | 기준값 ±20% | +50% 이상: 인플레이션 위험 |
| Legendary 드랍 1회당 소요 시간 | 30~60분 | 10분 이하: 희소성 붕괴 |
| Ancient 드랍 1회당 소요 시간 | 30~100시간 | 10시간 이하: 야리코미 콘텐츠 소진 |
| 이노센트 갓 롤 달성 유저 비율 | 상위 1~5% | 10% 초과: 난이도 하향 필요 |
| 허브 거래소 HL 거래량 | 일일 총 HL의 10~20% | 50% 초과: 거래소 의존도 과도 |
| 자동 사냥 vs 수동 플레이 비율 | 수동 60%+ | 수동 40% 미만: 자동화 너무 강함 |
| 시즌 중반 이탈률 | 30~40% | 60% 초과: 콘텐츠 소진 너무 빠름 |

### 8.2 경제 이상 신호 대응 프로토콜

```
[인플레이션 감지]
HL 획득량 기준값 +50% 이상 지속 → 즉시 대응
  ├→ 단기: 상점 가격 상승, Sink 강화 (이노센트 복종 비용 상향)
  ├→ 중기: 새 HL Sink 콘텐츠 추가
  └→ 장기: 다음 시즌 Faucet 조정

[희소성 붕괴 감지]
Ancient 드랍 소요 시간 10시간 미만 → 즉시 대응
  ├→ 단기: 해당 드랍 구역 일시 점검 (버그 여부 확인)
  ├→ 중기: 드랍률 하향 패치
  └→ 장기: Ancient 상위 등급(Mythic) 추가로 목표 이동

[파밍 동기 소멸 감지]
시즌 중반 일일 활성 유저 40% 이상 이탈 → 긴급 대응
  ├→ 단기: 긴급 이벤트/보너스 기간 (드랍률 x2 등)
  ├→ 중기: 신규 이노센트 종류 추가
  └→ 장기: 다음 시즌 콘텐츠 볼륨 증대
```

## 9. 종합 설계 권장사항 (Project Abyss 직접 적용)

### 9.1 Phase 1 MVP 적용 우선순위

| 우선순위 | 시스템 | 설계 원칙 | 근거 |
|:---|:---|:---|:---|
| P1 | 무실패 선형 성장 | 이노센트 복종 = 항상 스탯 증가 | 디스가이아 철학 + 야리코미 동기 |
| P1 | 층당 보상 리듬 | 10층 보스 = 가속 구간, 나머지 = 안정 구간 | GDC "Make it Bumpy" |
| P1 | 수동/자동 효율 차등 | 수동 100%, 자동 60~70% | 수동 정복 선행 원칙 |
| P1 | 구역별 특화 드랍 풀 | 스탯별 타겟 파밍 가능 | 파밍 목표 명확화 |
| P2 | 배드 럭 보정 | 연속 N층 미출현 시 희귀 이노센트 확률 증가 | GDC Pity System |
| P2 | HL 거래소 수수료 | 거래 금액의 5~10% HL Sink | 인플레이션 방지 |
| P2 | 파티 보상 보너스 | 인원 증가마다 HL/이노센트 출현 보너스 | 협동 인센티브 |
| P3 | 시즌 리셋 구조 | 소프트 리셋(HL) + 영구 보존(캐릭터/장비) | 하이브리드 모델 |
| P3 | 경제 모니터링 | 시간당 HL 획득, Ancient 소요 시간 등 | 경제 건강 지표 관리 |

### 9.2 핵심 수치 기준값 (Phase 1 검증용)

```
[아이템계 경제 기준값]
1지층(Normal, 2층짜리) 완료 소요 시간: 20~40분
1지층(Legendary, 4층짜리) 완료 소요 시간: 60~120분
이노센트 야생→복종까지 소요 HL: 기준값 Z (초반 HL 획득 기준 1~2시간분)
층당 평균 이노센트 야생 출현 수: 1~3개
희귀 이노센트 출현율: 2~5%

[드랍 기준값]
Normal 아이템 1분당 드랍 수 (수동): 0.5~1개
Legendary 아이템 획득까지 평균 소요 시간 (수동): 30~60분
Ancient 아이템 획득까지 평균 소요 시간 (수동): 30~50시간

[경제 순환 기준값]
일일 플레이어 평균 HL 획득 (수동 2시간): 기준값 X
허브 거래소 거래 수수료: 7%
이노센트 복종 비용(HL): 복종당 기준값 X × 0.01 ~ 0.05
```

### 9.3 최종 설계 원칙 요약

| 원칙 | 내용 | 근거 |
|:---|:---|:---|
| **무실패 성장** | 강화 실패 없음. 리스크는 사망 시 진행 손실로만 구현 | 디스가이아 야리코미 철학 |
| **선형 + 구간 보너스** | 기본 성장은 선형, 10층 보스 구간에서 강한 보너스 | "Make it Bumpy" 원칙 |
| **수동 정복 선행** | Legendary/Ancient는 수동에서만 드랍 | 파밍 핵심 루프 보호 |
| **거래 제한** | Ancient 귀속, 이노센트 아이템 귀속 | RMT 방지 + 파밍 동기 보존 |
| **배드 럭 보정** | 연속 미획득 시 확률 점진 증가 (상한 있음) | 박탈감 방지 |
| **파티 인센티브** | 파티 플레이 시 HL/이노센트 보너스 | 멀티플레이 동기 부여 |
| **소프트 리셋** | HL 리셋, 장비/캐릭터 영구 보존 | 경제 신선도 + 노력 보존 |
| **Low Power Gap** | 야리코미 vs 캐주얼 최대 파워 비율 1:4 이내 | Albion Online GDC 원칙 |
