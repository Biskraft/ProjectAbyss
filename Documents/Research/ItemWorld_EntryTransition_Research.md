## 1. 현재 Project Abyss의 아이템계 진입 정의

### 1-1. 내러티브 프레이밍 (기존 문서 정리)

> "아이템계에 들어간다는 것은 아이템의 **기억 속으로 다이빙**하는 것이다."
> — `Design_Narrative_Worldbuilding.md`

| 요소 | 현재 정의 |
|:-----|:---------|
| 진입 은유 | 아이템의 기억 속으로 다이빙 |
| 진입 위치 | 허브의 "기억의 사서(Memory Archivist)" NPC |
| 내부 공간 | 기억의 지층(Memory Strata) — 깊을수록 오래된 기억 |
| 지층 시각 변화 | 1층=따뜻/밝은 색조 → 심층=추상적/원형적 |
| 지층 보스 | 기억의 수문장 → 군주 → 신 → 핵 |

### 1-2. 현재 미정의 영역

다음 항목이 아직 구체적으로 명세되어 있지 않다:

- 허브 → 아이템계 진입 시 **시각적 전환 연출**
- 지층 간 전환 시 **시각적 연출**
- 아이템계 → 허브 탈출 시 **시각적 연출**
- "다이빙"이라는 은유의 **구체적 시각 표현**

## 2. 게임 레퍼런스

### G-01. 슈퍼 마리오 64 — 그림 속으로 뛰어들기

| 항목 | 내용 |
|:-----|:-----|
| **게임** | Super Mario 64 (1996) |
| **진입 오브젝트** | 벽에 걸린 그림 |
| **시각 묘사** | 그림 표면이 **물처럼 일렁이는(ripple)** 텍스처 애니메이션. 마리오가 점프하여 부딪히면 화면 전체가 물결 효과로 전환. 사인파 기반 UV 디스토션으로 고정된 2D 이미지가 액체 포탈처럼 보인다 |
| **핵심 감정** | 경이감과 호기심 — "벽에 걸린 그림이 다른 세계로 통한다" |
| **결정적 요소** | 캐릭터가 **능동적으로 뛰어든다**는 행위. 포탈이 아닌 **일상적 오브젝트**(그림)가 세계 입구 |

> **적용 포인트:** 아이템 스프라이트가 물결치며 포탈화 → 캐릭터가 아이템 속으로 빨려 들어감. "그림 → 세계"를 "장비 → 기억의 지층"으로 치환.

### G-02. 할로우 나이트 — 드림 네일 (Dream Nail)

| 항목 | 내용 |
|:-----|:-----|
| **게임** | Hollow Knight (2017) |
| **진입 트리거** | 드림 네일로 대상을 베는 행위 |
| **시각 묘사** | 밝은 **백색 플래시** → **물결 같은 웨이브 디스토션**이 화면 전체를 왜곡 → 색상 팔레트가 어두운 톤에서 **에테르한 밝은 톤**으로 극적 전환 → 반투명 입자가 부유하는 꿈의 영역 |
| **핵심 감정** | 초월과 신비 — "현실의 베일을 찢는다" |
| **결정적 요소** | **무기 사용이라는 의식적 행위**로 진입. 플레이어의 에이전시가 강하다 |

> **적용 포인트:** "기억의 베일을 가른다"는 콘셉트와 완벽 부합. 화면이 물결치며 색상이 변하는 연출은 내러티브적으로 정합. 진입 트리거를 아이템에 특수 행동(검으로 베기? 손을 얹기?)으로 만들면 몰입도 상승.

### G-03. 페르소나 5 — 메타버스 진입

| 항목 | 내용 |
|:-----|:-----|
| **게임** | Persona 5 / Royal (2016/2019) |
| **진입 트리거** | 메타버스 내비게이터 앱 활성화 |
| **시각 묘사** | **빨간색+검은색의 물결(ripple)**이 화면을 지배. 현실 세계가 **물에 빠지듯 일그러지며** 색상이 포화/왜곡되어 팔레스로 전환. 주인공이 마스크를 벗고 의상이 변하는 컷인 삽입 |
| **핵심 감정** | 각성과 반항 — 현실이 문자 그대로 "뒤틀린다" |
| **결정적 요소** | 리플 색상이 세계관의 핵심 색상(빨강). **색상 코딩으로 차원 전환을 각인** |

> **적용 포인트:** 레어리티별 리플 색상 차별화. Normal(흰색), Magic(파란), Rare(노란), Legendary(주황), Ancient(초록). 진입 순간부터 아이템 등급이 시각적으로 각인된다.

### G-04. 셀레스테 — 거울 사원 진입 & 드림 블록

| 항목 | 내용 |
|:-----|:-----|
| **게임** | Celeste (2018) |
| **진입 오브젝트** | 거울 / 드림 블록 |
| **시각 묘사** | 거울이 캐릭터를 **빨아들이며** 화면이 반전/왜곡. 드림 블록 통과 시 **사이키델릭한 색상 전환** + 중력/이동 규칙 변화. 픽셀아트 게임에서 셰이더 이펙트의 효과가 검증된 사례 |
| **핵심 감정** | 불안과 자기 대면 — 자신의 내면으로 빨려 들어감 |
| **결정적 요소** | 픽셀아트에서의 셰이더 기반 전환이 충분히 효과적임을 증명 |

> **적용 포인트:** 아이템 표면이 거울처럼 반짝이기 시작 → 캐릭터의 반사된 모습이 비침 → 반사 속으로 빨려 들어감. "기억의 지층이 아이템의 내면"이라는 내러티브와 정합.

### G-05. 바스티온 — 세계 생성/붕괴

| 항목 | 내용 |
|:-----|:-----|
| **게임** | Bastion (2011, Supergiant Games) |
| **진입 방식** | 캐릭터가 걸어가면 발 앞의 땅이 조립됨 |
| **시각 묘사** | 타일 조각이 **하늘에서 떨어져 내려와 지형을 형성**. 에어리어 이동 시 반대로 **바닥 타일이 아래로 떨어져 붕괴**. 대격변으로 부서진 세계가 주인공의 존재에 반응하여 재구성되는 연출 |
| **핵심 감정** | 복원과 불안정 — 세계가 만들어지지만 언제든 무너질 수 있는 긴장 |
| **결정적 요소** | **트랜지션 자체가 내러티브를 전달**하는 완벽한 예시. "세계가 만들어지는 과정"을 보여줌 |

> **적용 포인트:** 허브 세계의 타일이 산산이 부서져 내리고, 파편들이 재조합되어 기억의 지층을 형성하는 연출. 절차적 생성이라는 시스템 특성을 내러티브적으로 정당화 — "기억이 조각에서 세계로 재구성된다."

### G-06. Enter the Gungeon — 총알 구멍 낙하

| 항목 | 내용 |
|:-----|:-----|
| **게임** | Enter the Gungeon (2016) |
| **진입 방식** | 건전(Gungeon) 입구로 자유낙하 |
| **시각 묘사** | 게임 시작 시 캐릭터가 **자유낙하**. 플로어 간 이동은 **엘리베이터로 하강**. 점점 더 깊이 들어간다는 수직 감각 |
| **핵심 감정** | 담대한 돌진 — 낙하가 "돌이킬 수 없는 도전"을 상징 |
| **결정적 요소** | **낙하라는 물리적 행위**가 진입의 은유와 직결 |

> **적용 포인트:** 아이템 속으로 "낙하"하는 연출. 바닥에 구멍이 열리며 캐릭터가 떨어짐. 각 지층 이동에도 낙하 트랜지션 → "점점 더 깊이" 야리코미 감각 강화.

### G-07. 아이작의 번제 — 함정문 낙하

| 항목 | 내용 |
|:-----|:-----|
| **게임** | The Binding of Isaac: Rebirth (2014) |
| **진입 방식** | 보스 처치 후 바닥에 함정문(trapdoor) 개방 |

> **적용 포인트:** 지층 간 이동 시 활용. 특수 이노센트가 다음 지층의 프리뷰를 보여주는 설계 가능.

### G-08. 사이코너츠 — 정신 세계 진입

| 항목 | 내용 |
|:-----|:-----|
| **게임** | Psychonauts 1 & 2 (2005/2021) |
| **진입 트리거** | 사이코 포탈을 대상 머리 위에 올려놓음 |
| **시각 묘사** | 캐릭터가 아스트랄 프로젝션 상태로 전환 → **나선형 소용돌이(spiral swirl)**가 화면을 축소 → 대상의 내면 세계가 완전히 다른 아트 스타일로 펼쳐짐. 각 정신 세계는 독자적 비주얼(카지노, 사이키델릭 축제, 사막 등) |
| **핵심 감정** | 탐구와 발견 — 남의 머릿속에 들어간다는 강렬한 호기심 |
| **결정적 요소** | **각 대상마다 완전히 다른 비주얼 세계**. 매번 놀라움이 있음 |

> **적용 포인트:** 아이템 종류/내러티브에 따라 지층 비주얼이 달라지는 콘셉트. 무기=불꽃/전투적, 방어구=결정체/광물, 액세서리=에테르/마법적. 소용돌이 진입 이펙트는 "빨려 들어간다"는 감각에 적합.

### G-09. Noita — 성스러운 산 붕괴

| 항목 | 내용 |
|:-----|:-----|
| **게임** | Noita (2019) |
| **진입 방식** | Holy Mountain 출구 터널 통과 |
| **시각 묘사** | 출구 경사면 지나면 **천장이 물리 시뮬레이션으로 실시간 붕괴** → 잔해가 통로를 막음. 빨간 파티클이 역류 방지. 모든 픽셀이 물리 시뮬레이션 |

> **적용 포인트:** 타일 깨짐 + 파티클 낙하로 단순화 적용. "표면 세계가 부서지면 숨겨진 기억의 지층이 나타난다"는 내러티브.

### G-10. FEZ — 차원 회전

| 항목 | 내용 |
|:-----|:-----|
| **게임** | FEZ (2012) |
| **진입 방식** | 시점을 90도 회전 |
| **시각 묘사** | 2D 평면 세계에서 **시점을 90도 회전**하면 숨겨진 3차원이 드러남. 3D 큐브의 면이 스무스하게 회전하며 이전에 보이지 않던 플랫폼/경로 출현 |
| **핵심 감정** | 인식의 확장 — "차원이 하나 더 있었다"는 깨달음 |
| **결정적 요소** | **관점의 전환**만으로 완전히 새로운 세계가 드러남 |

> **적용 포인트:** 아이템을 회전시키면 내부 면이 드러나고, 그 세계로 진입하는 메타포. 기술적 부담이 크므로 진입 컷씬에서만 부분 적용.

### G-11. 디스가이아 — 아이템계 진입 (반면교사)

| 항목 | 내용 |
|:-----|:-----|
| **게임** | 마계전기 디스가이아 1~7 (2003~2023) |
| **진입 방식** | 아이템계 안내인 NPC → 아이템 선택 → 로딩 |
| **시각 묘사** | **화면 페이드아웃 → 암전 → 스테이지 출현**의 간결한 흐름. 시리즈 전반 시각 연출은 최소한(minimal). D5 이후 아이템 2D 일러스트가 짧게 표시된 뒤 전환 |
| **핵심 감정** | 효율적 반복 — SRPG 특성상 빠른 반복 진입이 우선 |
| **결정적 요소** | **연출 부재가 반면교사**. "아이템 안에 세계가 있다"를 시스템으로만 전달하고 시각으로 보여주지 않음 |

> **적용 포인트:** ARPG인 Project Abyss는 행동의 쾌감이 중요하므로, **물리적으로 빨려 들어가는 감각**을 시각적으로 전달해야 한다. 단, 야리코미 반복 진입 시 **스킵 가능해야 한다**는 교훈은 그대로 적용.

### G-12. Dead Cells — 바이옴 간 전환

| 항목 | 내용 |
|:-----|:-----|
| **게임** | Dead Cells (2018) |
| **시각 묘사** | 바이옴 출구의 문/튜브 통과 → 극적으로 다른 색상 팔레트와 환경으로 전환 (감옥 → 하수도 → 성벽 → 시계탑). 간결하지만 세련된 전환 디테일 |
| **핵심 감정** | 전진과 미지 — 레이아웃은 모르지만 바이옴은 아는 긴장 |
| **결정적 요소** | **팔레트 시프트 + 환경 극변**의 조합이 최소 비용으로 최대 임팩트 |

> **적용 포인트:** 지층 간 이동 시 팔레트 시프트 + 환경 극변. 같은 아이템계 내에서도 깊어질수록 색감이 변하면 깊이감 강화.

### G-13. 젤다: 신들의 트라이포스 — 아이리스 와이프

### 4-2. 기법별 GLSL 핵심 코드

**T-01 아이리스 와이프:**
```glsl
float dist = length(uv - center);
float mask = smoothstep(radius - edge, radius + edge, dist);
gl_FragColor = mix(sceneColor, vec4(0,0,0,1), mask);
```

**T-03 소용돌이:**
```glsl
vec2 centered = uv - 0.5;
float dist = length(centered);
float angle = atan(centered.y, centered.x);
angle += dist * spiralStrength + time * speed;
vec2 newUV = vec2(cos(angle), sin(angle)) * dist + 0.5;
gl_FragColor = texture2D(uSampler, newUV);
```

**T-04 픽셀화:**
```glsl
float blockSize = mix(1.0, maxBlock, progress);
vec2 blockUV = floor(uv * resolution / blockSize) * blockSize / resolution;
gl_FragColor = texture2D(uSampler, blockUV);
```

**T-05 디더링:**
```glsl
// 4x4 Bayer 매트릭스
int x = int(mod(gl_FragCoord.x, 4.0));
int y = int(mod(gl_FragCoord.y, 4.0));
float threshold = bayer[y * 4 + x] / 16.0;
float alpha = step(threshold, progress);
gl_FragColor = mix(sceneColor, vec4(0,0,0,1), alpha);
```

**T-09 파편화:**
```glsl
// Voronoi 셀 기반 — 각 셀이 독립적으로 낙하
vec2 cellCenter = voronoiCenter(uv, cellCount);
float fallOffset = hash(cellCenter) * progress * fallSpeed;
vec2 newUV = uv + vec2(0.0, fallOffset);
float cellMask = step(progress, hash(cellCenter) + 0.3);
gl_FragColor = texture2D(uSampler, newUV) * cellMask;
```

### 5-3. 제안 C: "아이템 다이브 (Item Dive)"

> **콘셉트:** 마리오 64처럼 아이템 표면이 포탈화되어 캐릭터가 물리적으로 다이빙한다. (마리오 64 + 앤트맨 참조)

**시퀀스 (약 2초):**

```
Phase 1 — 포탈화 (0.5초)
┌─────────────────────────────┐
│  아이템 스프라이트 확대          │
│  표면이 물결치기 시작 (리플)     │
│  레어리티 색상 글로우            │
└─────────────────────────────┘

Phase 2 — 다이빙 (0.8초)
┌─────────────────────────────┐
│  캐릭터가 아이템을 향해 돌진     │
│  충돌 순간 → 화면 플래시        │
│  잔상(disco trail) 남김         │
└─────────────────────────────┘

Phase 3 — 줌인 모핑 (0.7초)
┌─────────────────────────────┐
│  아이템 텍스처가 프랙탈 줌으로   │
│  확대 → 기억의 지층 지형으로     │
│  모핑(morphing)                │
│  캐릭터 착지                    │
└─────────────────────────────┘
```

**사용 기법:** T-02(리플) + T-10(잔상) + T-07(줌 로테이션) + T-08(팔레트)
**감정 곡선:** 기대 → 돌진의 쾌감 → 스케일 전환의 경이

### 5-6. 지층 간 전환 (보스 클리어 후)

아이템계 내부에서 지층 간 이동 시 별도의 짧은 전환:

```
보스("기억의 문") 처치
  → 바닥에 빛나는 균열/함정문 개방 (Isaac 참조)
  → 캐릭터 낙하 (0.5초)
  → 낙하 중 팔레트 시프트 (Dead Cells 참조)
  → 다음 지층 착지
```
