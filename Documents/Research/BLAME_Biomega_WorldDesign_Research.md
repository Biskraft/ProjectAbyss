## 1. BLAME! — 도시의 물리적 구조

### 1.1. 규모

"도시(The City)"는 태양계 전체를 점유하는 거대 인공 구조물. 직경 **목성 궤도 = 약 16억 km** (지구 지름의 12만 5천 배).

- **동심원 다이슨 구**가 겹겹이 쌓인 형태
- 각 층의 하부가 빛을 방출 → 아래 층에 주/야간 사이클 제공
- 층과 층을 분리하는 것이 **메가스트럭처** — 도시의 하중 지지 프레임이자 넷스피어 연산 기판
- 메가스트럭처 내부에 "N차원으로 접힌" 컴퓨터 칩이 매립

### 1.2. 지층(Strata) 구조

- 도시는 수직으로 쌓인 **거대 지층**으로 나뉨
- 각 지층은 위아래로 메가스트럭처에 의해 분리
- **두 지층 사이 이동에 최대 한 달 소요**
- 메가스트럭처는 **중력자선 방사기(GBE)의 직격**만이 관통 가능

### 1.3. 빌더의 무질서 건설

통제 상실 후 빌더의 프로그래밍이 퇴화:
- 아무 곳에도 이르지 않는 계단
- 1피트마다 문이 가로막는 복도
- 기능 없는 기둥, 파이프, 부서진 기계로만 구성된 지층 전체
- 광활한 공허(void)가 너무 흔해져 지층 간 구별이 거의 불가능

> 이것이 **절차적 생성의 서사 내재적(diegetic) 정당화** — "왜 방 구조가 매번 다른가?"에 대한 답.

### 1.4. 폐소공포 ↔ 광장공포의 교차

| 요소 | 폐소공포적 | 광장공포적 |
|:-----|:---------|:---------|
| **형태** | 좁은 터널, 파이프 통로, 문마다 막힌 복도 | 끝이 안 보이는 공동, km 단위 수직 낙하 공간 |
| **감정** | 압도, 질식, 갇힘 | 나락, 무의미, 존재의 미소함 |
| **기능** | 긴장의 축적 | 긴장의 해소 + 경외 |

## 2. BLAME! — 세력 구조와 내러티브 장치

### 2.1. 세 세력

| 세력 | 정체 | 도시와의 관계 |
|:-----|:-----|:------------|
| **세이프가드** | 넷스피어의 자동화 보안 시스템 | 도시의 면역 체계. Gene 없는 인간 말살 |
| **실리콘 라이프** | 적대적 사이보그 종족 | 도시의 기생자. Gene 자체를 소멸시키려 함 |
| **인간/포스트-휴먼** | 분산된 소규모 집단 | 도시에 갇힌 존재. 물리적 격리로 하위 종으로 분화 |

킬리: 세 세력 어디에도 속하지 않는 고독한 여행자. 유일한 목적 = **Net Terminal Gene 보유자 찾기**.

### 2.2. Net Terminal Gene = 능력 게이트의 원형

- 넷스피어(도시 중앙 제어) 접속을 위한 유전적 인증 마커
- 테러 조직이 레트로바이러스로 모든 인간에게서 삭제
- Gene 없이 접근 시도 → 세이프가드가 살해 = **게이트 미충족 시 진행 불가**

### 2.3. 토하 중공업 = 허브

- 도시 내부의 원통형 메가스트럭처, 예외적 안전지대
- 중앙 AI(Mensab) + 13개 하위 AI가 관리
- 인간 공동체 "식물재배자(Planters)" 거주

## 3. BLAME! — 감정 설계

### 3.1. 감정 팔레트

| 감정 | 구현 방식 | 강도 |
|:-----|:---------|:-----|
| **고독** | 대사 극소, 거대 빈 공간, 인구 부재 | 지배적 정서 |
| **왜소감** | 인물이 패널의 1% 미만, 건축물이 프레임 지배 | 시각적 핵심 |
| **경외** | 갑작스러운 광활한 공동, 이해 불가능한 규모 | 간헐적 폭발 |
| **공포** | 무엇이 나올지 모르는 어둠, 세이프가드의 갑작스런 출현 | 지속적 저음 |

> *"니헤이는 깔끔한 SF 시스템을 넘어, 혼자라는 것 — 측량 불가능하게 혼자라는 것의 무드를 만들고 있다."*

### 3.2. 건축이 이야기하는 방식

- 대사/설명이 극소화 → **공간 자체에서 의미를 읽어야**
- 한때 기능했던 구조물의 잔해 → 과거 문명의 흔적
- 빌더의 무작위 건설물 → 통제 상실의 물리적 증거
- 점점 이질적으로 변하는 환경 → 중심(넷스피어)에서 멀어짐

## 4. BIOMEGA — 코드 월드와 세계 변형

### 4.1. 역변형 폴리머 → 세계 재구성

이야기 후반부에서 지구가 파괴되고 **거대한 코드형 세계**가 출현:
- 길이 **약 48억 km** (태양-해왕성 거리)
- 재질: 좀비 육체 + 폴리머로 이루어진 우주 서식지
- 독자적 인구와 생태계 보유
- 내부 주민들도 드론화(dronification)에 굴복 시작

### 4.2. 구조적 의미

- 익숙한 지구 → 완전히 이질적인 선형 구조물로 교체
- 기존의 모든 지리/건축/생태계가 소멸, 새 규칙 적용
- **"더 깊이 들어갈수록 더 낯설어진다"**의 물리적 극한

## 5. 니헤이의 디자인 철학

### 5.1. 배경

- 실제 건설업 종사 경험
- 파슨스 디자인 스쿨(뉴욕) 11개월 수학
- 일본 건축 설계 사무소 근무 후 만화가로 전향

### 5.2. 핵심 영향 — 메타볼리즘의 역전

일본 메타볼리즘 건축 운동(인공+자연 통합 메가스트럭처)을 뒤집어 **"통합이 실패한 후의 메가스트럭처"**를 그림.

### 5.3. 시각 언어 — 무기적 vs 유기적

| 요소 | 무기적(Inorganic) | 유기적(Organic) |
|:-----|:----------------|:--------------|
| 선 | 경직, 정밀, 반복적 | 감염적, 불규칙, 침식적 |
| 패턴 | 블록, 그리드, 대칭 | 규칙 없음, 비대칭, 잠식 |
| 감정 | 질서의 잔해 | 통제 불능의 성장 |

> **교훈:** 두 시각 언어의 충돌이 니헤이 미학의 핵심. 레어리티/깊이에 따라 유기적 비율이 증가.

## 6. ECHORIS 매핑

### 6.1. 구조적 대응

| BLAME! | ECHORIS | 적용 |
|:-------|:-------------|:-----|
| 도시의 수직 지층 | 기억의 지층(Memory Strata) | 각 스트라텀이 전환 연출로 분리 |
| 빌더의 무한 무질서 건설 | 절차적 방 생성 | "깨진 기억이 끝없이 자기복제/재구성" = 절차적 생성의 다이에제틱 정당화 |
| 깊어질수록 이질적 | 깊이별 시각/오디오 변형 | 재귀 깊이 1→2→3에서 채도/색수차/BGM 극적 변화 |
| Net Terminal Gene | 스탯 게이트 + 능력 게이트 | 자격 없이 접근 → 거부/사망 |
| 세이프가드 | 지층 내 적 시스템 | 깊은 지층 = 더 강한 면역 반응 |
| 실리콘 라이프 | 야생 기억 단편 | 독자적 목표를 가진 서식 존재 |
| 토하 중공업 | 허브 / 탈출 제단 | 적대적 구조 속 안전 거점 |
| 메가스트럭처 관통 | 재귀의 문 | GBE만이 관통 = 재귀의 문만이 다음 깊이 허용 |
| 빌더 프로그래밍 퇴화 | 레어리티별 지층 변이 | Normal=규칙적, Ancient=완전히 퇴화한 기괴한 건축 |

### 6.2. 감정 설계 매핑

| BLAME! 감정 | 아이템계 구현 수단 |
|:-----------|:----------------|
| **고독** | BGM이 점점 미니멀. 파티원 있어도 환경이 압도적 |
| **왜소감** | 배경 레이어 극단적 스케일링. 캐릭터가 화면의 극히 작은 부분 |
| **경외** | 지층 전환 시 거대한 수직 공간이 드러나는 줌아웃 연출 |
| **공포** | 조명 감소, 시야 제한, 예고 없는 적 출현 빈도 증가 |

### 6.3. 2D 횡스크롤 적용 — 핵심 시각 기법

**수직성:**
- 5~7개 레이어 다층 패럴랙스 배경으로 끝없는 수직 깊이 시사
- 지층 간 이동 시 긴 낙하/상승 시퀀스
- 배경에 거대한 기둥, 파이프, 폐구조물이 시차 스크롤

**대비:**
- 좁은 1x1 통로 방 → 거대 2x2+ 보이드 방 전환
- 어두운 통로 → 밝은 대공간 조명 전환
- 밀집된 파이프/기계 vs 텅 빈 공간 교차

**건축적 무의미:**
- 배경에 끊긴 계단, 목적 없는 문, 연결 안 되는 통로
- 동일 기둥이 끝없이 반복되는 배경 레이어
- 정교한 무기적 구조물을 잠식하는 유기적 성장물

**스케일 커뮤니케이션:**
- 특정 방 진입 시 카메라 줌아웃 → 주변 구조의 거대함 노출
- 먼 배경 레이어에서 빌더(유사 존재)가 건설하는 실루엣

### 6.4. BIOMEGA → 재귀적 진입 매핑

| BIOMEGA 요소 | 재귀적 진입 적용 |
|:------------|:---------------|
| 지구 → 코드 월드 변형 | 깊이 1(익숙) → 깊이 2(변형) → 깊이 3(완전 이질) |
| 48억 km 선형 구조 | 재귀 깊이 증가 → "세계가 길어지고 좁아지는" 시각 왜곡 |
| 좀비 육체로 재구성 | 기억의 지층이 깊어질수록 기억 자체가 해체/재구성 |
| 드론화 | 깊은 지층의 기억 단편/적이 더 변형되고 이질적 |

### 6.5. 핵심 교훈 6가지

1. **건축이 곧 내러티브** — 공간만으로 이야기를 전달
2. **무한 스케일은 감정** — 규모 자체가 고독/경외/공포
3. **통제 상실 = 절차적 생성의 정당화** — "시스템 폭주로 무한 건설"
4. **폐소↔광장의 리듬** — 긴장과 해소의 교차
5. **단일 목표의 힘** — 무한 세계에서의 방향성
6. **무기적 vs 유기적의 충돌** — 깊이에 따라 유기적 비율 증가
