## 1. 아이템계 층 구조 (Floor Structure)

### 1.1 기본 구조

아이템계는 아이템 내부에 존재하는 **랜덤 생성 던전**으로, 크기·형태·적 배치·지오 이펙트가 모두 진입할 때마다 새로 생성된다. 층과 층 사이의 이동은 맵 위에 놓인 차원 포탈(Dimensional Gate)을 통해 이루어지며, 포탈 도달 또는 맵 내 모든 적 전멸로 해당 층을 클리어한다.

### 1.2 아이템 레어리티별 층수 (D1~D4)

| 등급 | 최대 층수 | 최대 아이템 레벨 |
|:-----|:---------|:----------------|
| 커먼 (Common) | 30층 | 30 |
| 레어 (Rare) | 60층 | 60 |
| 레전더리 (Legendary) | 100층 | 100 |

D5부터는 층수 제한이 폐지되고 **무한 층** 탐험이 가능해졌다. D7에서는 다시 30층으로 복귀하되 "전생(Item Reincarnation)"으로 루프를 반복하는 구조로 전환되었다.

### 1.3 보스층 (Boss Floor) 구조

모든 게임에서 공통으로, **매 10층이 보스층**이다.

| 층 번호 | 보스 유형 | 특이사항 |
|:--------|:---------|:---------|
| 10, 20, 40, 50, 70, 80층 | 아이템 장군 (Item General) | EXP·HL x1.5 |
| 30, 60, 90층 | 아이템 왕 (Item King) | EXP·HL x2 |
| 100층 (랭크 39 이하) | 아이템 신 (Item God) | EXP·HL x3 |
| 100층 (랭크 40) | 아이템 신 2 (Item God 2) | EXP·HL x3.5 |

보스층 클리어 후에는 다음 두 가지 선택지가 주어진다:
- **탈출 (Exit):** 아이템을 가지고 허브로 귀환. 현재 층까지의 강화 결과를 보존.
- **계속 탐험 (Continue):** 다음 층으로 진행.

보스층 클리어 후 일정 확률로 **이노센트 타운(Innocent Town)** 이 등장한다(D2에서 도입). 이노센트 타운에서는 파티 회복, 아이템 의회(Item Assembly) 법안 통과, 이노센트 관리, 탈출이 가능하다.

### 1.4 탈출 (휴식층 없는 구조)

디스가이아의 아이템계에는 별도의 "휴식층"은 없다. 대신 보스층 이후에 이노센트 타운이 확률적으로 등장하는 방식으로 휴식을 제공한다. 이노센트 타운이 사실상의 "체크포인트+휴식 구간"이다.

## 2. 지오 패널 & 지오 심볼 시스템

### 2.1 기본 작동 원리

```
지오 패널 (Geo Panel):
  맵 바닥의 특정 영역을 덮는 색깔 타일.
  자체로는 아무 효과 없음.

지오 심볼 (Geo Symbol / D3~D4에서 Geo Block):
  이동/파괴 가능한 크리스탈 오브젝트.
  같은 색 패널 위에 놓이면 → 해당 패널 전체에 심볼 효과 적용.

예시:
  빨간 패널 (맵 우측 절반) + ATK+50% 심볼 (빨간) =
  → 빨간 패널 위의 모든 유닛이 ATK +50% 획득
  → 심볼을 옮기거나 파괴하면 효과 즉시 소멸
```

### 2.2 전체 지오 이펙트 목록

**전투 강화 계열:**
- ATK +50% / ATK -50%
- DEF +50% / DEF -50%
- Ally Boost +50% (아군 전용)
- Enemy Boost +50% / Enemy Turbo x3 (적 전용)
- Enemy Level Up 10% (적이 매 턴 레벨 +10%)
- Attacks +1 (모든 행동이 2회 실행)
- Evade (회피율 +50%)

**이동/행동 제한 계열:**
- Move +1 / Move -1
- No Entry (해당 패널 진입 불가)
- No Lifting (들어올리기 불가)
- No Close Combat (근접 공격 불가)
- No Ranged Attacks (원거리 공격 불가)

**데미지/생존 계열:**
- Damage 20% (매 턴 최대 HP의 20% 감소)
- Ally Damage 20% (아군에게만 적용)
- Heal 20% (매 턴 최대 HP의 20% 회복)
- Invincibility (직접 데미지 면역)
- Deathblow (0 이상 데미지를 즉사로 변환)
- Reverse Damage (공격이 회복, 힐이 데미지로)
- Disperse Damage (피해를 같은 팀의 해당 이펙트 유닛에게 분산)

**특수 메커닉 계열:**
- Clone (랜덤 유닛의 클론 생성)
- Fusion (2개의 랜덤 적 유닛 합체)
- Warp (턴 종료 시 같은 이펙트 패널로 순간이동)
- Encroach (패널이 인접 4칸으로 색상 전파)
- Stage Clear (아군이 이 이펙트 패널에서 턴 시작 시 스테이지 즉시 클리어)
- Game Over (이 패널에서 턴 종료 시 패배)

**리소스 계열:**
- EXP +50% (처치 시 경험치 +50%)
- Mana +50% (처치 시 마나 +50%)

### 2.3 지오 체인 (Geo Chain)

```
발동 조건:
  지오 심볼을 "자신과 다른 색의 패널" 위에서 파괴

체인 진행:
  1. 파괴된 심볼의 색과 같은 모든 패널이 연쇄 폭발
  2. 각 폭발은 패널 위 유닛에게 데미지
  3. 폭발로 다른 심볼이 파괴되면 → 그 심볼의 패널도 연쇄 폭발
  4. 체인이 연결될수록 폭발 데미지 증폭

데미지 공식:
  D1~D4: (2 × 심볼 레벨 + 3) × 체인 크기
  D5:    (5 + 최대HP × 4%) × 체인 크기

투명 심볼(Transparent Symbol) 특수 규칙:
  체인 완성으로 맵의 모든 패널이 사라지면 →
  맵 전체 유닛에게 마지막 체인과 같은 데미지의 대폭발

보너스 게이지 연동:
  체인 카운터 수에 비례하여 보너스 게이지 대량 충전
```

## 3. 보너스 게이지 (Bonus Gauge)

매 층의 우상단에 표시되는 게이지로, 적 처치·지오 심볼 파괴·지오 체인으로 충전된다.

| 보상 순위 (1위 = 최고) | 내용 |
|:----------------------|:------|
| 1위 | Mr. Gency's Exit (탈출 아이템) 확정 지급 |
| 2위 | 레전더리 등급 장비 |
| 3위 | 랜덤 이노센트 (복종 상태) |
| 4~8위 | 일반 아이템·재화 등 하위 보상 |
| 9위 (Rank 0) | 아이템 레벨 +1 (D5에서 보상 체계 개편 후 여기서 레벨 획득) |

## 4. 미스터리 게이트 & 미스터리 룸

### 4.1 기본 개념 (D2~)

미스터리 게이트는 보스층이 아닌 일반 층에 **랜덤으로 등장**하는 특수 포탈이다. 진입 시 특수 이벤트 방(미스터리 룸)으로 이동한다. DD2부터는 층 클리어/스킵 후 자동 랜덤 진입 방식(미스터리 아일랜드)으로 전환되었다.

### 4.2 미스터리 룸 종류

**D2 기본 룸:**

| 방 이름 | 내용 |
|:--------|:------|
| 보물 방 | 보물 상자 3개 (일반 2 + 레전더리 1) |
| 아이템 상점 | 회복 아이템·장비 구매 |
| 병원 | 파티 전원 HP·SP 회복 |
| 점술사 | 아이템 레벨 ±3 또는 ±10 랜덤 (기대값 +3/방문) |
| 전투 방 | 2턴 내 적 전멸 시 아이템 레벨 +3 |
| 로스트 레지던트 | 콜렉터 이노센트 획득 (레어리티 즉시 상승) |

**D3 추가 룸:**

| 방 이름 | 내용 |
|:--------|:------|
| 스칼퍼 | X-디멘션 맵 해금 티켓 판매 (최고 100억 HL) |
| 몬스터 룸 (B31~99F) | 2턴 내 전멸 시 아이템 +3 레벨 |
| 디에즈 젠틀맨 룸 (B31~99F) | 2턴 내 전멸 시 아이템 +5 레벨 |
| 레벨 구슬 수집가 | 수집한 레벨 구슬 교환 보상 |

**DD2 미스터리 아일랜드:**

| 방 이름 | 내용 |
|:--------|:------|
| 아이템 보스 섬 | 장군/왕/신이 기본 스탯 5/10/20% 무료 부스트 |
| 이노센트 상점 | 희귀 이노센트 1개 구매 가능 |
| 이노센트 광장 | 이노센트 대화 → 레벨 대폭 상승 |
| 층 스킵 | 5층 또는 10층 스킵 (유료) |

## 5. 해적 시스템 (Pirates)

### 5.1 기본 규칙
- 아이템계에서 최소 1턴 경과 후 랜덤 확률로 등장
- 플레이어와 별개로 맵에 추가 스폰

### 5.2 시리즈별 변화

**D2 (16종 해적):**
- 1층~: Ambling Pirates, Chicken Pirates(보물상자), Dancing Pirates(이노센트 동반)
- 21층~: Gate Pirates(미스터리 게이트 추가), Wealthy Pirates(레전더리 풀장착)
- 41층~: Ninja Pirates, Prism Pirates
- 61층~: Z Pirates
- 91층~ (수육계): Baal Pirates

**D4:** 루트별 출현 분화. 함선 파츠 드랍. **Pirate Editor** 도입으로 커스텀 함선 제작.

## 6. 탈출 메커니즘 (Exit Mechanics)

### 6.1 Mr. Gency's Exit

"Mr. Gency" = "Emergency(긴급)"의 말장난.

**핵심 특징:**
- 아이템계 내 어느 층에서든 사용 가능 (보스층 포함)
- 사용 시 현재 층 진행 상황과 모든 강화 내용이 저장됨
- 다음 진입 시 중단했던 층부터 재시작 가능

**획득 방법:**
- 10의 배수 층 클리어 후 보너스 게이지 1위 보상으로 확정 지급
- Common 30층 완전 탐험 = 최소 3개 획득
- Legend 100층 = 최소 10개 획득

### 6.2 전체 탈출 경로

| 탈출 방법 | 조건 | 진행도 보존 |
|:---------|:-----|:-----------|
| 보스층 클리어 후 선택 | 10의 배수 층 보스 처치 | 완전 보존 |
| Mr. Gency's Exit | 소비 아이템 1개 사용 (언제든 가능) | 현재 층 기록 보존 |
| 이노센트 타운 탈출 | 보스층 클리어 후 타운 진입 시 | 완전 보존 |
| 파티 전멸 (Mr. Gency 없음) | 전원 HP 0 | 보스 처치 보너스만 보존, 나머지 손실 |

## 7. 시리즈 진화 요약 (D1~D7)

### D1 — Hour of Darkness (2003, 원형 확립)
- 기본 구조 확립: 커먼 30층 / 레어 60층 / 레전더리 100층
- 최대 아이템 레벨 100, 레벨당 스탯 증가 5%
- 이노센트(Specialist) 야생→복종 메커니즘 도입

### D2 — Cursed Memories (2006, 대폭 확장)
- 최대 아이템 레벨 200으로 상향
- **미스터리 게이트/룸 도입** (14종+)
- **이노센트 타운** 도입 (보스층 후 확률적 등장)
- **해적 시스템** (16종 해적 그룹 랜덤 침입)
- **더블 킬링** 테크닉 등장

### D3 — Absence of Justice (2008)
- **레벨 구슬(Level Sphere)** 도입: 들고 전멸 시 +5레벨
- 레벨당 스탯 증가 6.6%로 상향
- **역해적(Reverse Pirating)** 도입
- 지오 심볼 → **지오 블록** 전환

### D4 — A Promise Unforgotten (2011)
- 최대 아이템 레벨 300으로 상향
- **분기 루트** 도입: 5층마다 검(성장)/웃는얼굴(이노센트)/혼합(보물) 루트 선택
- **킬 보너스:** 최대 400레벨 추가 스탯 보너스

### D5 — Alliance of Vengeance (2015, 패러다임 전환)
- **무한 층** (층수 제한 폐지)
- 레벨링 방식 전환: 층 클리어만으로 레벨 불가, 보너스 게이지 보상으로만 획득
- **레어리티 기반 스탯 시스템** (레어리티 값 1~100이 스탯 배율 직접 결정)
- **이노센트 팜** 도입
- 이노센트 소프트 캡 50,000

### D6 — Defiance of Destiny (2021, 자동화)
- **오토배틀 x32** 도입
- **D.I. (Demonic Intelligence):** AI 매크로 프로그래밍
- 커뮤니티: "방치 게임으로 변질" 비판

### D7 — Vows of the Virtueless (2023, 세션 최적화)
- **30층 복귀 + 아이템 전생** 반복 루프
- 전생 시 능력 일부 계승, 레벨 1 리셋, 무한 반복
- 오토배틀 유지 (x32)
- "100층 마라톤"에서 **"30층 스프린트 반복"** 으로 설계 철학 전환

## 9. ECHORIS 설계 시사점

### 채택 가치가 높은 메커니즘
1. **10층(지층) 단위 체크포인트 구조:** 짧은 세션(10층)과 긴 세션(100층) 모두 수용 가능
2. **보스층 + 이노센트 타운 분리:** 클라이맥스 → 안전지대 → 탈출 결정의 자연스러운 리듬
3. **레어리티 기반 깊이 차등:** 아이템 등급 간 가치 격차를 구조적으로 강제
4. **이노센트 복종 시 2배 효과:** 전투 참여 동기 강화
