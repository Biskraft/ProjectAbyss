### 1.2. 설계 원칙 도출

| 원칙 | 근거 게임 | 설명 |
|:---|:---|:---|
| **HUD 최소화** | Hollow Knight, Spelunky | 숫자 대신 아이콘. 정보는 일시정지 메뉴에서 상세 확인 |
| **색상으로 상태 전달** | Dead Cells Rally (녹→주황→소멸) | 텍스트 없이 색상 변화만으로 복잡한 시스템 교육 |
| **보스 HP = 선택적** | HK(없음) vs DC/Hades(있음) | 탐험 게임은 미표시가 몰입감 강화. 파밍 게임은 표시가 효율 판단에 필수 |

- **월드:** Hollow Knight식 마스크/아이콘 HP + 최소 HUD. 탐험 몰입 우선
- **아이템계:** Dead Cells식 Rally 색상 바 + 보스 HP 표시. 야리코미 효율 판단 필요
- **보스 HP:** 월드 보스 = 미표시(긴장감), 아이템계 보스 = 표시(파밍 효율 판단)

### 2.2. SotN 맵 상세 (위키 기반)

- **PAUSE 메뉴 전체 맵:** 방문=컬러, 미방문=회색 (Castle Map 보유 시)
- **세이브룸=빨강, 워프룸=노랑/파랑** 즉시 식별
- **OoE 혁신:** 단일 대형 맵 대신 세계 지도 + 개별 던전 맵 분리. ECHORIS 2-Space와 구조적 유사

### 2.3. Hollow Knight 맵 상세

- **Cornifer → Quill → 벤치 갱신** 3단계 맵 획득 과정이 탐험 자체를 보상으로 만듦
- **Map Pin 시스템:** 색상별 핀을 직접 꽂는 커스텀 마킹. 공식 의미 없이 플레이어가 자체 코드 생성
- **맵 스테이션:** 환경 내 물리적 안내도 (다이어제틱 UI)

### 2.4. ECHORIS 적용 인사이트

- **월드:** SotN/HK 혼합. 세이브포인트(대장간)=빨강, 스탯 게이트=주황, 능력 게이트=초록 색상 코딩
- **아이템계:** Dead Cells식 절차적 맵. 4x4 그리드 구조를 미니맵에 방 단위로 표시. 보스방/시작방 아이콘 구분
- **OoE 참조:** 2-Space 전환 시 "월드 전체 맵 ↔ 아이템계 지층 맵" 분리 표시

### 3.2. SotN 장비 화면 상세 (위키 기반)

PAUSE 메뉴 5탭 구조:
1. **Equip** -- 무기(좌/우), 투구, 갑옷, 망토, 기타(링 2개)
2. **Item** -- 소모 아이템 (사용 가능)
3. **Relics** -- 능력 항목 (켜기/끄기 토글)
4. **Familiar** -- 5종 중 1개 선택
5. **Status** -- 스탯 수치

### 3.3. Disgaea 인벤토리 상세 (위키 기반)

- **Rarity 색상 코딩:** Common(1-24) / Rare(25-49) / Legendary(50-99) / Epic(100)
- **Same Rarity Bonus:** 같은 카테고리 장비 복수 장착 시 스탯 보너스 (D5: Rare 2%, Legendary 4%, Epic 5%)
- **이노센트 슬롯:** 아이템 상세에서 거주 중인 이노센트 목록 표시. 야생=흰색, 복종=레벨 2배 표시
- **Innocent Warehouse (D:D2+):** 이노센트 전용 별도 창고

### 3.4. ECHORIS 적용 인사이트

- **장비 화면:** SotN 5탭 구조 참조. ECHORIS 버전: Equipment / Items / Relics(능력) / Innocents / Status
- **레어리티 색상:** CLAUDE.md 기존 정의 (흰/파/노/주/초) 그대로 유지. Disgaea의 Same Rarity Bonus 개념 검토 가치
- **이노센트 표시:** Disgaea의 야생/복종 시각 구분 직접 차용. 슬롯 수는 레어리티별 차등 (2/3/4/6/8)
- **아이템 비교:** Dead Cells식 자동 비교 팝업. 아이템계에서 빠른 장비 교체 판단 지원

### 6.2. Dead Cells Rally 상세

- 피격 시 손실 HP의 80%가 주황색으로 표시
- 주황 구간 중 적 공격 시 가한 피해의 12% HP 회수 (1회 상한 = 원래 손실의 20%)
- 재피격 시 이전 주황 구간 소멸
- **색상 신호:** 녹(안전) → 주황(지금 반격하면 기회) → 소멸(실패) 의 3단계를 색상만으로 교육

### 6.3. ECHORIS 적용 인사이트

- **데미지 숫자:** 월드 = 없음 (HK식 몰입), 아이템계 = 있음 (야리코미 효율 판단)
- **Rally 검토:** Dead Cells Rally를 "단조열 봉합"으로 테마 변환. 피격 후 반격 시 HP 일부 회수
- **경고 시스템:** Dead Cells ! 아이콘 채택. 보스 텔레그래프 + 일반 적 경고 이중 구조

### 7.2. Hollow Knight Shade 시스템 상세

- 사망 위치에 Shade 생성. 잃은 Geo 보유 + 공격적 행동
- Soul 미터 용량 2/3로 제한 (Shade 처치 전까지)
- 지도에 해골 아이콘으로 위치 표시
- Shade는 플레이어 업그레이드를 반영 (주문 학습 시 Shade도 사용)

### 7.3. Hades 사망 처리 상세

- 사망 → Hypnos가 사망 방식별 다른 대사 (21,000+ 보이스 라인)
- 사망이 "실패"가 아닌 "이야기의 다음 장"
- Death Defiance: 치사량 피격 시 50% HP 부활 (최대 3회). 파란 방패 아이콘으로 잔여 횟수 표시

### 7.4. ECHORIS 적용 인사이트

- **월드:** HK Shade 변형 검토. 사망 시 Gold 일부 손실 + 사망 위치에 "기억의 잔영" 생성 → 회수 시 Gold 복구
- **아이템계:** Disgaea Mr. Gency's Exit식 "탈출 판단". 사망 = 아이템계 강제 퇴장 + 진행 손실. Hades Death Defiance를 이노센트로 제공 가능 ("기억 방패" 이노센트)

## 11. 아이템계(Item World) 고유 UI (Disgaea 위키 기반)

ECHORIS 스파이크에 직결되는 Disgaea 아이템계 UI 상세 정리.

### 11.1. 아이템계 진입 플로우

1. 기지 내 Item World Guide NPC에게 접근
2. 인벤토리/창고에서 진입할 아이템 선택
3. 무작위 생성 배틀 맵 연속 진행 (층별)
4. 매 층 시작 전 현재 층수(Floor Number) 표시

### 11.2. 층 구조 UI

- **클리어 방법:** 전 적 격파 OR Skip Gate 진입
- **아이템 레벨:** 층 클리어마다 +1 상승
- **보스:** 10층마다 등장 (General/King/God/God2 4등급)
- **이노센트 타운:** 보스 클리어 후 랜덤 출현. 내부: 힐러/게이트키퍼/Item Assembly NPC
- **Mr. Gency's Exit:** 언제든 탈출 가능한 소모 아이템

### 11.3. Bonus Gauge UI

- 0-9 레벨 바. 전투 중 화면 모서리에 표시
- 충전: 공격, 킬, 콤보, Geo Chain
- 레벨별 보상 사전 열람 가능 (EXP, HL, 아이템)

### 11.4. 분기 경로 UI (D4+)

- 5층마다 출구 포탈 2개: 칼 아이콘(성장 루트) / 스마일 아이콘(이노센트 루트)
- 선택 조합에 따라 3가지 경로 분기: Item Growth / Innocent Boost / Treasure

### 11.5. ECHORIS 적용 인사이트

- **진입 플로우:** 세이브포인트(대장간)에서 장비 선택 → 아이템계 진입. Disgaea와 동일 구조
- **Bonus Gauge:** 콤보/킬 스트릭에 연동되는 보너스 게이지. 아이템계 방 클리어 보상 품질에 영향
- **분기 경로:** 4x4 그리드에서 보스 방 도달 경로에 따라 보상 유형 차등. Disgaea D4 칼/스마일 아이콘 참조
- **Mr. Gency's Exit:** 아이템계 "귀환의 돌" 소모 아이템. 초반 진입장벽 낮추기 + 탈출 의사결정 핵심 도구
