## 1. 아이템 레벨 시스템

### 1.1 시리즈별 최대 아이템 레벨 상한

| 시리즈 | 최대 레벨 | 비고 |
|:-------|:---------|:-----|
| D1 | 100 | 레전더리 100층 = 레벨 상한 |
| D2 | 200 | 미스터리 룸으로 추가 레벨 획득 확장 |
| D3 | 200 | 레벨 구슬 도입 (+5/회) |
| D4 | 300 | 킬 보너스로 최대 400레벨 추가 획득 가능 |
| D5 | 9,999 | 무한 층, 레벨 구슬 +2, 레벨 피쉬 추가 |
| D6 | 9,999 | 오토 배틀 x32 지원 |
| D7 | 없음 (전생 루프) | 30층 후 아이템 전생으로 리셋 반복 |

### 1.2 층 클리어와 레벨 획득

**D1~D4:** 포탈 도달 또는 적 전멸 → 아이템 레벨 +1 자동 적용

**D5:** 적 전멸 시 보너스 게이지 경유로만 레벨 획득
- Rank 0 보상: 레벨 +1
- Rank 9 보상: 레벨 +1
- 한 층 완전 클리어 시 최대 +2레벨

### 1.3 레벨 구슬 (Level Sphere)

| 시리즈 | 출현 조건 | 획득 방법 | 레벨 보너스 |
|:-------|:---------|:---------|:-----------|
| D3 | 21층 이후 랜덤 | 들고(Lift) 전멸 클리어 | +5 레벨 |
| D5 | 랜덤 출현 | 구슬 파괴(Destroy) | +2 레벨 |
| D5 (레벨 피쉬) | 랜덤, 복수 등급 | 들고 클리어 | 등급별 상이 |

## 2. 스탯 성장 공식

### 2.1 D1 원본 공식

```
레벨당 증가량 = InitialStat / 20 + 1
강화 스탯 = 기본 스탯 × (1 + L × 0.05)
```
- 레벨 1당 기본 스탯의 5% 증가
- 레벨 100(최대)에서 기본 스탯의 **6배 (600%)**

### 2.2 D4 강화 공식

```
레벨당 증가량 = InitialStat / 15 + 1
강화 스탯 = 기본 스탯 × (1 + L × 0.066)
```
- D1 대비 약 33% 상향

### 2.3 시리즈별 최대 스탯 배율

| 레벨 | D1 (5%/Lv) | D2 (5%/Lv, Lv200) | D4 (6.6%/Lv + Kill) |
|:-----|:-----------|:-------------------|:--------------------|
| 10 | x1.50 | x1.50 | x1.66 |
| 50 | x3.50 | x3.50 | x4.30 |
| 100 | **x6.00** | x6.00 | x7.60 |
| 200 | - | **x11.00** | x14.20 |
| 300+Kill400 | - | - | **x24.80** |

> **핵심:** D1~D4 전 시리즈 모두 **순수 선형 성장**. 지수 폭발이 없어 예측 가능.

### 2.4 D5 레어리티 기반 스탯 배율

```
최종 스탯 = 기본 스탯 × (1.0 + 레어리티값 / 100)
```
- Common (1~24): x1.01 ~ x1.24
- Rare (25~49): x1.25 ~ x1.49
- Legendary (50~99): x1.50 ~ x1.99
- Epic (100): x2.00 (고정)

## 3. 아이템 레어리티 등급 체계

### 3.1 D1 레어리티 (0~255 범위, 낮을수록 희귀)

| 등급 | 값 범위 | 스탯 보너스 | 이노센트 슬롯 | 아이템계 층수 |
|:-----|:--------|:-----------|:-------------|:-------------|
| Legendary | 0~7 | +50% | 최대 13 | 100층 |
| Rare | 8~31 | +25% | 최대 10 | 60층 |
| Common | 32~255 | 없음 | 최대 7 | 30층 |

### 3.2 동일 레어리티 보너스

D1:
- 같은 레어리티값 장비 2개 장착: +10%
- 3개 장착: +20%
- 4개 장착: +30%

D5: 같은 등급이면 매칭 성립. Rare: +2%, Legendary: +4%, Epic: +5% (적성치에 가산)

### 3.3 등급 승급 (DD2)

DD2에서는 Common 전 층(30층) 완료 시 Rare로 자동 승급, Rare 완료 시 Legendary로 승급. 단, 승급 시 현재 층이 1로 리셋.

## 4. 아이템 보스 체계와 영구 스탯 보너스

### 4.1 보스 출현 규칙

| 보스 종류 | 출현 층 | EXP/HL 배율 |
|:---------|:--------|:-----------|
| Item General (장군) | 10, 20, 40, 50, 70, 80층 | x1.5 |
| Item King (왕) | 30, 60, 90층 | x2.0 |
| Item God (신) | 100층 (랭크 39 이하) | x3.0 |
| Item God 2 (신 2) | 100층 (랭크 40) | x3.5 |

### 4.2 보스 처치 스탯 보너스 공식

**D1:**
```
장군 = 기본스탯 × (0.1 + 아이템레벨/200)
왕 = 장군 × 2
신 = 장군 × 3
```

예시: 기본 ATK 100, 아이템 레벨 50 → 장군 처치 시 ATK +35

**D4 (상향):**
```
장군 = 기본스탯 × (0.1 + 아이템레벨 × 2/300)
```

**D5:** 보스 처치 시 레어리티 값 +1(장군) / +2(왕) / +3(신)

### 4.3 더블 킬링 (Double Killing, D2~D4)

1. 보스 처치 → Mr. Gency's Exit으로 탈출 (보너스 저장됨)
2. 재진입 → 같은 보스 다시 처치 → 보너스 2중 획득
3. D4 패치 이후: 처치 횟수 제한 완전 해제 → 이론적 무한 반복

## 5. 이노센트의 최종 스탯 기여

### 5.1 직접적 스탯 기여 (D5 기준)

- 스탯형 8종 각 레벨 50,000 = 캐릭터 스탯에 각 +50,000 고정 추가
- Statistician 레벨 900 = EXP 획득 +900%
- 이노센트 보너스는 장비 스탯 배율이나 적성치에 **영향받지 않고 직접 합산**

### 5.2 최종 스탯 공식 (전체 레이어)

```
최종 스탯 = 기본 스탯 (레벨업)
          + 장비 스탯 × 적성치% × 무기숙련도 보너스%
          + 이노센트 레벨 합산 보너스
          + 지오 이펙트 (전장 임시)
          + 에빌리티 효과 (패시브)
```

## 6. 훔치기 메커니즘 (Steal)

### 6.1 일반 클래스 확률 공식

```
확률 = (자기레벨 × 방향보정 + 핸드HIT/2) - (적레벨 + 아이템랭크×3 + 레어리티보정)
최대 50% 캡
```

방향보정: 정면 1.0 / 측면 1.1 / 배후 1.2
레어리티보정: Common 0 / Rare 10 / Legendary 30 / Legendary(0) 100

### 6.2 도적 클래스

```
확률 = (자기레벨×2 + 핸드HIT + 25) - (적레벨 + 아이템랭크×3 + 레어리티보정)
최소 1%, 최대 99%
```

### 6.3 전략적 위치

훔치기는 **랭크 40 최강 장비를 획득하는 사실상 유일한 방법**. 수육계 아이템 갓 2가 랭크 40 장비를 장착하므로 도적으로 훔치기 필수.

## 7. 엔드게임 야리코미 루프

### 7.1 전체 성장 순환도

```
[캐릭터 레이어]        [장비 레이어]
레벨 1 → 9999          아이템계 진입
      ↓                       ↓
전생 (천재 품질)         층 클리어 (스탯 +5~6.6%/Lv)
      ↓                       ↓
보너스 포인트 축적       이노센트 복종 (효과 ×2)
      ↓                       ↓
적성치 +5%              보스 처치 (스탯 +5~20%)
      ↓                       ↓
더 높은 캐릭터 스탯     더 강한 장비
      ↓                       ↓
      └──────→ 수육계 도전 ←──┘
                    ↓
            랭크 40 장비 훔치기
                    ↓
            새 장비 아이템계 재진입
                    ↑_____________↑
```

### 7.2 D5 최적화 파워 레벨링

```
Statistician 900 (EXP ×10)
+ 치트샵 EXP 배율 (최대 ×10)
+ 부트캠프 스쿼드 (EXP 35% 공유)
+ 엘리트 포 스쿼드 (성장률 +100%)

= Carnage Dimension 1스타에서 레벨 1 → 9999를 1회 전투로 달성
```

### 7.3 수육계 (Land of Carnage)

```
적 레벨 = 기본 레벨 × 21 + 200 (+ 스탯 2배)
예: 기본 레벨 300 → 수육계 레벨 6,500
```
- 랭크 35~40 장비의 유일한 획득처
- DD2 LoC Mode: 적 처치마다 캐릭터 스탯 직접 흡수 (HP 0.025%, 스탯 0.25%)

## 8. 시간 투자 대비 보상 곡선

### 8.1 탐험 시간 기준표

| 목표 | D1~D4 수동 | D5 수동 (원턴킬) | D6 오토 ×32 |
|:-----|:-----------|:----------------|:-----------|
| 30층 클리어 | 30~60분 | 15~25분 | 10~20분 |
| 100층 클리어 | 2~4시간 | 50분~1.5시간 | 30분~1시간 |
| 장비 1개 완전 최적화 | 30~50시간+ | 30~50시간+ | 10~20시간 |

### 8.2 보상 체감 주기

| 보상 유형 | 주기 | 체감 강도 |
|:---------|:-----|:---------|
| 아이템 레벨 +1 | 매 층 | 낮음 (누적) |
| 미스터리 룸 출현 | 5~15층마다 | 높음 (서프라이즈) |
| 보스 처치 + 스탯 보너스 | 매 10층 | 중간~높음 |
| 이노센트 복종 | 랜덤 | 높음 (특히 희귀종) |
| 새 랭크 장비 훔치기 | 100층 돌파 시 | 매우 높음 (세션 하이라이트) |

### 8.3 Mr. Gency's Exit 소모품 경제

**획득:** 10의 배수 층마다 보너스 게이지 1위로 확정 지급
- Common 30층 = 최소 3개
- Legend 100층 = 최소 10개

**소비:** 장시간 탐험 시 자급자족. 고위험 탐험(더블 킬링)은 소모 빠름.

## 9. 경제 설계 총평 — Project Abyss 적용 시사점

### 핵심 수치 공식 요약 (참조용)

```
아이템 레벨당 스탯 증가:
  D1: InitialStat/20 + 1 (5%/레벨)
  D4: InitialStat/15 + 1 (6.6%/레벨)

보스 처치 보너스 (D1):
  장군 = 기본스탯 × (0.1 + 아이템레벨/200)

레어리티 스탯 배율 (D5):
  배율 = 1.0 + (레어리티값/100)

훔치기 확률:
  = (자기레벨 × 방향보정 + 핸드HIT/2) - (적레벨 + 랭크×3 + 레어리티보정)

수육계 적 레벨:
  = 기본레벨 × 21 + 200 (+ 스탯 2배)
```

### 설계 원리

| 원리 | 디스가이아의 해법 | Project Abyss 적용 포인트 |
|:-----|:----------------|:------------------------|
| 아이템 애착 | 교체형이 아닌 육성형 | 기억의 지층 = 소유한 아이템을 키우는 구조 |
| 리스크-보상 | 10층 단위 탈출 + Mr. Gency | 지층 단위 탈출 + 긴급 탈출 소모품 |
| 무한 성장의 천장 | 이노센트 소프트 캡 50,000 | 이노센트 타입별 상한 설정 필수 |
| 세션 길이 유연성 | 10층 ≈ 15~30분 | 지층 수 = 레어리티별 세션 길이 설계 |
| 성장 체감 주기 | 매 층(연속) + 매 10층(보스) | 연속 소보상 + 주기적 대보상 이중 구조 |
| 재화 자급자족 | Mr. Gency는 탐험으로 자급 | 소모품이 콘텐츠로 자급되는 구조 권장 |
