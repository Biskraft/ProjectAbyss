# 환경 교습 패턴 카탈로그 — 말 없는 튜토리얼 설계 리서치

> **작성일:** 2026-04-06
> **담당:** 게임 디자이너
> **목적:** 텍스트·대사·UI 힌트 없이 플레이어를 가르치는 기법 체계화, Project Abyss (에르다 0발화) 첫 30분 설계에 직접 적용
> **분류:** Research / 환경 교습 / 사일런트 튜토리얼

## 0. 연구 배경 및 제약 조건

### Project Abyss의 설계 공리

| 공리 | 규칙 |
|:--|:--|
| 에르다 0발화 | 주인공 에르다는 독백, 생각 풍선, 대사 일절 없음 |
| NPC 최소화 | 첫 30분 동안 NPC는 존재하지 않음 (그림자 시장 이전) |
| UI 튜토리얼 금지 | "Z키로 공격" 같은 프롬프트 없음 |
| 세계가 스스로 설명 | 건축, 조명, 적 배치, 아이템 배치가 곧 튜토리얼 |

이 제약은 선택이 아닌 설계 원칙이다. BLAME! / 킬리 아키타입에서 오는 톤 — 압도적 침묵, 행동만으로 존재를 증명하는 캐릭터 — 이 PA의 플레이어 판타지와 직결된다. 따라서 환경 교습 기법의 학습은 선택 사항이 아니라 필수 기반이다.

## 1. 레퍼런스 게임 분석

### 1.1 Hollow Knight — King's Pass의 침묵 교습

**핵심 기법: 공간 배치에 의한 행동 유도 + 즉각 부정 피드백**

King's Pass는 Hollow Knight의 튜토리얼 구역으로, 단 하나의 텍스트 없이 다음을 가르친다.

| 가르치는 것 | 방법 |
|:--|:--|
| 낙하 데미지 없음 | 첫 장면에서 에르다가 높은 곳에서 낙하하며 스폰됨. 데미지 없음을 즉시 경험. |
| 이동 + 점프 | 좁은 갭으로 구성된 복도. 통과하려면 점프가 필수. 설명 없이 필요성이 강제됨. |
| 공격 동사 | 왼쪽 비밀 구역 탐험 시 석재 봉인을 공격으로 깨야 진행 가능. 문이 막혀 있으면 공격 외 대안이 없다. |
| 속도의 중요성 | 낙하하는 바위 — 느리게 움직이면 피격. "빠르게 움직이고 위험을 감수하는 것이 더 낫다"는 게임 철학을 첫 2분 안에 신체로 학습시킴. |
| 분위기/세계관 | 오프닝 낙하 장면, 최소 HUD, 주변 사운드, Vengefly 비행 — 설명 없이 세계의 고딕 분위기를 즉각 전달. |

**핵심 통찰:** 왼쪽 비밀 구역은 필수 경로가 아니다. 하지만 탐험 욕구를 가진 플레이어가 자연스럽게 발견하도록 공간이 설계되어 있다. 선택적 교습 구역이 강제 교습보다 더 깊은 학습을 유발한다.

**PA 적용 포인트:** 모루의 빛나는 마법 검 → "저걸 집어야 한다"는 공간 유도. 봉인된 격벽 문 → "나중에 돌아와야 한다"는 메트로베니아 훅을 텍스트 없이 심기.

### 1.2 INSIDE (Playdead) — 완전 무언의 경험 설계

**핵심 기법: 조작 최소화 + 음향/조명 피드백 + 메커닉-내러티브 통합**

INSIDE는 2버튼 + 스틱으로 전체 게임을 운영한다. UI 없음, 튜토리얼 없음, 텍스트 없음.

| 기법 | 구체적 적용 |
|:--|:--|
| 조작 원시화 | 달리기, 점프, 잡기 — 이 세 가지만 존재. 조작이 단순하므로 튜토리얼 자체가 불필요. |
| 음향 반응 | 조명 변화에 사운드 이펙트가 동기화됨. 플레이어는 시각+청각 신호의 일치를 통해 "이 환경이 반응하고 있다"는 것을 즉시 이해. |
| 메커닉 톤 반영 | 스토리 톤이 전환되는 지점에서 메커닉 자체가 변화. 내러티브를 설명하지 않고 조작 경험으로 전달. |
| 색조 언어 | 어두운 채도 낮은 색 = 위험/적대적 공간. 밝음 = 안전/중립 공간. 색이 곧 안내판. |

**핵심 통찰:** 조작 수를 줄이면 튜토리얼의 필요성이 줄어든다. 단순한 동사 체계는 환경 교습의 전제 조건이다.

**PA 적용 포인트:** Z(공격) / X(회피) / C(에코 활성) — 3버튼 체계를 첫 30분에서 하나씩 순차 도입. 전투 전 공격만. 아이템계 진입 전 에코는 사용 불가 상태로 잠금. 복잡도를 시간에 따라 레이어링.

### 1.3 Another World (Eric Chahi, 1991) — HUD 없는 원조

**핵심 기법: 씬 연출과 실패를 통한 학습 + 시네마틱 공간 언어**

Another World는 HUD와 UI를 완전히 제거한 선구적 설계. Eric Chahi의 의도: "언어 없이 이야기하는 것."

| 기법 | 구체적 적용 |
|:--|:--|
| 영화적 컷인 | 위험 요소가 먼저 화면에 등장한 후 플레이어가 진입. "저것은 죽임"이라는 정보를 컷씬으로 주입. |
| 즉사와 재시작 루프 | 최초 몇 분에 즉사 이벤트가 연속. 빠른 재시작 속도로 죽음을 교습 메커닉으로 정상화. |
| 배경 요소 활용 | 배경의 움직임, 캐릭터 반응 애니메이션이 환경 위험을 설명. |
| 보편적 시각 기호 | 새장, 총, 문 — 문화권과 언어를 초월하는 시각 기호만 사용. |

**핵심 통찰:** 죽음이 페널티가 아닌 교습 도구가 되려면 재시작 속도가 빨라야 한다. Another World는 즉사가 잦지만, 재시작이 1초 이내라 죽음이 분노가 아닌 "아, 그렇구나"로 느껴진다.

**PA 적용 포인트:** 슬라임은 느리고 2히트 처치로 설계 — 첫 전투는 실패해도 재도전 비용이 낮다. ATK 게이트에서 에코가 튕겨나는 붉은 섬광은 Another World식 즉각 부정 피드백.

### 1.4 Hyper Light Drifter — 픽토그램 언어 시스템

**핵심 기법: 이미지 기반 커뮤니케이션 + 미야자키식 모션 표현**

Hyper Light Drifter는 메인 메뉴와 일시정지 화면 외 게임 내 문자 없음. NPC 대화는 픽토그램 말풍선으로만 표현.

| 기법 | 구체적 적용 |
|:--|:--|
| 픽토그램 말풍선 | NPC가 이미지 시퀀스로 정보 전달. 언어를 모르는 플레이어도 의도를 추론 가능. |
| 마이크로 인터랙션 교습 | 기초 게임 메커닉을 초반 마이크로 상호작용으로 가르침. 명시적 튜토리얼 없이 게임 플레이를 통한 학습. |
| 모션 스토리텔링 | 캐릭터의 표정, 신체 언어, 행동의 의도가 애니메이션으로 전달됨. 대사가 없어도 감정이 전달됨. |
| 환경 레거시 서술 | 폐허, 잔해, 남겨진 오브젝트들이 과거 서사를 암시. 플레이어가 퍼즐 조각을 스스로 맞춤. |

**핵심 통찰:** 개발자 Alex Preston은 미야자키 애니메이션에서 영감을 받았다 — "아름다운 애니메이션과 디자인이 세계에 생명을 불어넣는다. 정보는 동작과 표정을 통해 전달된다." 이것이 PA의 에르다 설계 원칙과 정확히 일치한다.

**PA 적용 포인트:** 에르다의 애니메이션이 내러티브를 대체. 검을 살피는 동작 → "이것이 목표다." 아래를 응시하는 마지막 장면 → "저리 가야 한다." 대사 없이 캐릭터의 시선 방향과 신체 언어로 정보 전달.

### 1.5 Mega Man X — Egoraptor Sequelitis 분석: 인트로 스테이지 교습

**핵심 기법: 동사를 순서대로 고립 도입 + 환경이 테스트를 제공**

Arin Hanson(Egoraptor)의 Sequelitis 분석은 게임 디자인 강의 소재로 사용될 만큼 영향력이 크다. 핵심: Mega Man X의 인트로 스테이지는 새로운 메커닉을 게임 내에서 순서대로 가르친다.

| 교습 순서 | 방법 |
|:--|:--|
| 달리기 | 긴 수평 통로. 달리지 않으면 느리다. 자연스럽게 달리기를 선택. |
| 점프 | 갭 앞에 적 배치. 점프하지 않으면 피격. |
| 벽 점프 | 특정 지점에서 떨어지면 벽만 보임. 벽에서 점프하면 살아남음. 실수가 교습. |
| 대시 | 대시 부츠 획득 직후 대시가 필요한 갭 등장. 즉시 적용 테스트. |
| 차지샷 | 첫 보스 직전 체력이 많은 적 등장. 일반 샷이 비효율적임을 경험. |

**핵심 통찰:** 동사를 한 번에 하나씩 고립 환경에서 도입하고, 즉시 해당 동사를 테스트하는 상황을 제공. 이것이 Celeste의 "스크린 당 하나의 개념"과 동일한 원리. 메커닉 소개 → 안전한 연습 → 조합 테스트 — 이 3단계가 핵심.

**PA 적용 포인트:** 첫 30분 스크린 순서 자체가 이 구조를 따름. 스크린 1(이동) → 스크린 2(점프 + 갭) → 스크린 3(전투) → 스크린 4(에코) — 각 스크린이 하나의 메커닉을 고립 도입.

### 1.6 Celeste — 스크린 단위 개념 격리

**핵심 기법: 1스크린 = 1개념 + 복잡도 점진적 레이어링 + 실패 비용 최소화**

Celeste의 프롤로그는 3개 스크린으로 구성. 각 스크린은 정확히 하나의 메커닉을 가르친다.

| 스크린 | 가르치는 것 | 방법 |
|:--|:--|:--|
| 스크린 1 | 달리기 + 점프 | 조작 느낌, 물리, 기초 이동 체화 |
| 스크린 2 | 벽 타기 | 벽 앞에서 점프 외 대안이 없는 상황 |
| 스크린 3 | 대시 | 절망적 낙하 상황에서 대시가 유일한 탈출구 |

**메커닉 도입 3단계:**
1. **발견 (Discovery):** 새 메커닉이 필요한 상황에 플레이어를 배치. 정보 없이.
2. **연습 (Practice):** 위험 없이 새 메커닉을 쓸 수 있는 단순한 맥락 제공.
3. **조합 (Combination):** 이전 메커닉과 새 메커닉의 조합이 요구되는 상황 등장.

**핵심 통찰:** Celeste는 게임 전체가 튜토리얼이다 — 모든 스크린에서 새로운 것을 배운다. 단 "가르치고 있다는 느낌"을 주지 않는다. 학습은 항상 플레이로 위장된다.

**PA 적용 포인트:** 봉인된 복도 스크린은 "이동 + 점프"만 있는 순수 이동 학습 구간. 적 없음, 위협 없음. 이 구간이 끝난 후에야 슬라임이 등장. 단계가 명확히 분리됨.

### 1.7 Super Metroid — 에테쿤과 행동 시연 교습

**핵심 기법: NPC가 메커닉을 직접 시연 + 환경이 테스트 장소를 즉시 제공**

슈퍼 메트로이드의 에테쿤(Etecoon)은 사무스에게 벽 점프를 가르치는 크리처다.

| 단계 | 내용 |
|:--|:--|
| 시연 | 에테쿤이 벽 사이에서 연속 점프를 반복. 직접 보여줌. |
| 맥락 제공 | 에테쿤이 있는 공간 자체가 벽 점프 연습장으로 설계됨. |
| 즉시 테스트 | 출구가 위쪽에 있음. 벽 점프 없이는 탈출 불가. |

**핵심 통찰:** 벽 점프는 공식 매뉴얼에 기재되지 않은 숨겨진 메커닉이다. 그럼에도 플레이어는 에테쿤을 통해 자연스럽게 발견한다. "숨겨져 있지만 발견 가능"의 설계 — 이것이 Explorer 유형 플레이어에게 최고의 보상.

**PA 적용 포인트:** 에르다의 에코 타격 애니메이션은 "이것이 어떻게 작동하는지"를 직접 보여준다. 에코가 검에 닿는 순간, 바닥이 균열되는 순간 — 이 시퀀스 자체가 "에코 = 공간을 바꾸는 도구"임을 시연.

### 1.8 Dark Souls — 에실 데몬의 두 번의 만남

**핵심 기법: 도전 → 회피 → 재도전(유리한 위치) → 숙달 3막 구조**

다크 소울의 에실 데몬(Asylum Demon)은 튜토리얼 보스로, 최초 만남 시 플레이어는 장비가 없고 적은 압도적으로 크다.

| 만남 | 상황 | 교습 |
|:--|:--|:--|
| 1차 만남 | 장비 없음, 도어 있음 | "이 위협을 회피하는 것이 선택지다" |
| 중간 | 장비 획득, 공격 패턴 관찰 | 준비, 패턴 읽기 |
| 2차 만남 | 위에서 낙하 공격 가능 | 환경 활용, 위치 우위의 가치 |

**핵심 통찰:**
- 회피는 비겁함이 아니라 생존 전략이라는 것을 첫 보스로 가르침.
- 동일한 적을 다른 조건에서 재만남 → "내가 성장했다"는 증거.
- 위에서 낙하 공격 옵션은 필수가 아님 — 발견한 플레이어에게 보상을 주는 숨겨진 우위.

**PA 적용 포인트:** 기억의 수문장은 패턴 1(수평 돌진) → 패턴 2(점프 내려찍기) 순서로 공격. 각 공격 전 0.5초 준비 동작이 있어 2~3회 시도면 학습 가능. 죽음은 아이템계 입구로 리스폰 → 재시작 비용 낮음 → 죽음이 교습 도구가 됨.

### 1.9 Dead Cells — 첫 런이 전체 튜토리얼

**핵심 기법: 하이브리드 레벨 구조 + 바이옴별 단계적 메커닉 도입 + 포워드 모멘텀**

Dead Cells는 핸드크래프트 섹션과 절차적 생성을 혼합. 첫 번째 런이 전체 게임의 튜토리얼 역할을 한다.

| 기법 | 구체적 내용 |
|:--|:--|
| 포워드 모멘텀 | 레벨 구조가 플레이어를 항상 앞으로 밀어붙임. 멈추면 답답함을 느끼도록 페이스 설계. |
| 바이옴별 교습 | Toxic Sewers = 포지셔닝 학습 (독 웅덩이). Ramparts = 수직 전투 학습. 각 바이옴이 새 교습 테마를 가짐. |
| 환경 스토리텔링 | 닫힌 문 뒤에서 들리는 목소리, 메모, 흔적 — 텍스트 없이 이야기를 조각조각 제공. |
| 적 배치가 안내 | 적이 있는 방향 = 진행 방향. 적이 없는 방향 = 탐험 보상. |

**핵심 통찰:** 절차적 생성과 환경 교습의 긴장 관계 — 매 런이 달라지면 "고정된 교습 순서"를 만들기 어렵다. Dead Cells의 해결책은 바이옴(생물군계) 단위로 교습을 고정. 방 배치는 랜덤이어도 "이 바이옴에서 배울 것"은 고정됨.

**PA 적용 포인트:** 아이템계의 절차적 생성은 Dead Cells 문제와 동일한 도전. 해결책: 첫 번째 방(착지 방)은 고정 레이아웃. 야생 이노센트 첫 조우는 고정 방에서 보장. 절차적 다양성은 2번째 방부터 시작.

## 2. 환경 교습 기법 분류 (패턴 카탈로그)

리서치에서 추출한 기법을 8개 패턴으로 분류한다.

---

### 패턴 A: 공간 유도 (Spatial Funnel)

**정의:** 플레이어가 특정 행동을 하도록 공간 형태가 강제하는 기법. 문, 갭, 플랫폼의 배치가 곧 지시문.

**작동 원리:**
- 진행 가능한 방향이 하나 또는 두 개로 제한됨
- 목표 오브젝트가 공간 내 가장 눈에 띄는 위치에 배치됨
- 플레이어가 "저기로 가야겠다"를 자연스럽게 결정

**사례:**
- Hollow Knight: 왼쪽 비밀 구역 입구가 시야에 들어오도록 배치
- Celeste: 벽이 막혀 있고 위쪽에 출구 — 벽 타기가 유일한 선택
- Super Metroid: 에테쿤 공간의 출구가 위쪽에만 있음

**PA 적용:**

| PA 스크린 | 공간 유도 설계 |
|:--|:--|
| 대장간 (S1) | 방 중앙 모루 위 빛나는 검 — 가장 밝은 오브젝트. 시선이 자동으로 향함. |
| 봉인된 복도 (S2) | 직선 복도 + 2타일 갭들 + 봉인된 문. 진행 방향 하나, 막힌 것 하나. |
| 아이템계 착지 (S5) | 좁은 착지 공간 + 오른쪽 하나의 출구. 선택지가 없으므로 튜토리얼이 필요 없음. |

---

### 패턴 B: 시각적 차이화 (Visual Differentiation)

**정의:** 색, 빛, 재질, 크기의 차이로 "이것은 중요하다" 또는 "이것은 위험하다"를 전달.

**작동 원리:**
- 상호작용 가능 오브젝트는 주변보다 밝거나 색이 다름
- 위험 요소는 붉은색, 날카로운 형태, 날카로운 사운드
- 레어리티 색상 시스템이 아이템의 가치를 즉시 전달

**사례:**
- Hyper Light Drifter: 색조 언어 — 어두운 채도 = 위험
- INSIDE: 밝음 = 안전, 어두움 = 위험의 일관된 조명 언어
- Hollow Knight: 위험한 가시들은 환경에서 시각적으로 분리됨

**PA 적용:**

| PA 요소 | 시각적 차이화 설계 |
|:--|:--|
| 마법 검 | 파란빛 (#6969FF) 발광 — "이것은 특별하다" |
| 야생 이노센트 | 붉은 구체 — "이것은 적대적이다" |
| ATK 게이트 통과 실패 | 에코가 붉은 섬광과 함께 튕겨남 — 즉각 부정 피드백 |
| 레어리티 색상 | Normal 흰색 → Magic 파란색 → Rare 노란색 — 가치 서열이 색으로 표현 |
| 증식 구역 새 벽 | 더 밝은 색 석재 + 균열 — "여기는 새로 생겼다" |

---

### 패턴 C: 행동 시연 (Behavioral Demonstration)

**정의:** NPC, 환경 오브젝트, 또는 게임 자체가 메커닉을 직접 보여줌. 플레이어는 보고 따라함.

**작동 원리:**
- 가르칠 행동이 플레이어 눈앞에서 발생
- 플레이어는 같은 행동을 요구하는 상황에 즉시 배치됨
- "본 것 → 따라한 것 → 성공" 흐름

**사례:**
- Super Metroid 에테쿤: 벽 점프를 반복 시연 → 즉시 테스트 공간 제공
- Dark Souls Asylum Demon: 낙하 공격 가능 위치를 환경이 만들어둠
- Mega Man X: 대시 부츠 획득 직후 대시가 필요한 갭 등장

**PA 적용:**

| PA 순간 | 행동 시연 설계 |
|:--|:--|
| 에코 타격 시퀀스 (S4) | 에르다가 에코로 검을 치는 애니메이션이 "에코 = 파괴 도구"를 직접 보여줌 |
| 두 번째 다이브 (S11) | 첫 다이브에서 플레이어가 본 것 → 이번엔 직접 시작. 시연에서 자발적 실행으로 전환 |
| 아이템계 EXP 카운터 | 적 처치 시마다 카운터가 오름. "싸우면 강해진다"를 숫자로 실시간 시연 |

---

### 패턴 D: 죽음을 교습 도구로 (Death as Teacher)

**정의:** 실패(죽음)가 페널티가 아닌 정보 습득 수단이 되도록 재시작 비용을 낮추는 설계.

**작동 원리:**
- 재시작이 1~3초 이내
- 사망 지점에서 가까운 곳에서 재시작
- 사망 원인이 명확히 시각화됨 (텔레그래프 애니메이션)
- 실패 직후 "아, 그렇구나" 반응을 유발

**사례:**
- Another World: 잦은 즉사 + 1초 내 재시작 → 죽음이 분노가 아닌 학습
- Celeste: 사망 파티클이 즉각적이고 명확 + 거의 즉시 리스폰
- Dark Souls: 사망 시 소울 회수 기회 존재 → 죽음에서 배울 동기 제공

**PA 적용:**

| PA 요소 | 죽음 교습 설계 |
|:--|:--|
| 슬라임 (S3) | 느리고 2히트 처치. 피격해도 대미지 낮음. 첫 전투 실패 비용이 작음. |
| 아이템계 사망 | 보스 방 앞 체크포인트 존재. 보스에게 사망 시 보스 방 입구 리스폰. |
| 기억의 수문장 텔레그래프 | 공격 전 0.5초 준비 동작. 2~3회면 패턴 읽기 가능. |
| ATK 게이트 거부 | 죽음이 아닌 "거부" — 에코 튕김 + 붉은 섬광. 페널티 없이 정보만 전달. |

---

### 패턴 E: 환경 서사 (Environmental Narrative)

**정의:** 건축, 유물, 배치, 잔해가 과거와 세계를 설명. 플레이어가 텍스트 없이 이야기를 추론.

**작동 원리:**
- 사용된 흔적, 부서진 오브젝트, 봉인된 문 — 과거의 물리적 증거
- 플레이어는 탐정이 되어 퍼즐 조각을 맞춤
- 결론을 "말해주지 않는 것"이 immersion을 높임

**사례:**
- Hyper Light Drifter: 폐허, 잔해가 문명 붕괴를 암시
- Dead Cells: 닫힌 문 뒤 목소리, 남겨진 메모
- INSIDE: 텅 빈 건물, 방치된 기계 — 무슨 일이 있었는지를 유추하게 만듦

**PA 적용:**

| PA 요소 | 환경 서사 설계 |
|:--|:--|
| 대장간 균열 지도 (S1) | 여러 마커에 X표 + 하나에 ?표. "에르다가 이 일을 오래 해왔다" + "아직 해결 안 된 것이 있다"를 지도 하나로 전달. |
| 선반의 도구들 (S1) | 격벽 조각 샘플, 에코 충전대 — 에르다의 직업과 습관을 사물로 설명. |
| 증식 구역 새 벽 (S9) | 더 밝은 색 석재, 따뜻한 이음새 균열 — "이 벽은 최근에 자랐다"를 건축 질감으로 전달. |
| 대공동 전망 (S14) | 끝없는 수직 낙하, 저 아래 다른 층위 — "이 세계는 광대하다. 아직 모르는 것이 많다." |

---

### 패턴 F: 보상의 직접성 (Immediate Reward Signal)

**정의:** 올바른 행동에 즉각적이고 명확한 피드백을 제공. 숫자, 파티클, 사운드 — 추론 없이 이해 가능한 신호.

**작동 원리:**
- 행동 → 결과가 0.5초 이내 시각화
- 숫자 상승, 빛 효과, 소리의 세 채널 동시 활용
- 피드백이 클수록 행동이 좋은 것임을 즉시 전달

**사례:**
- Hollow Knight: 히트스탑 + 사운드 — 타격감을 즉각 전달
- Dead Cells: 킬 기어 (연속 처치 시 버프) 게이지 상승 — 적극적 전투를 즉시 보상
- Mega Man X: 체력 아이템 획득 시 HP 바 상승 + 소리

**PA 적용:**

| PA 요소 | 즉각 보상 설계 |
|:--|:--|
| 히트스탑 (S3) | 첫 공격 3프레임 화면 정지. "이 타격감 좋은데." 즉각적 감각 보상. |
| 보스 사망 후 "ATK +8" (S7) | 화면 중앙 금색 큰 텍스트 + 섬광. 강해졌다는 것을 1초 만에 전달. |
| 아이템 EXP 카운터 | 처치 시마다 HUD 소형 카운터 상승. 행동의 축적이 실시간으로 가시화. |
| 게이트 통과 성공 | 에코 타격 → 격벽 균열 애니메이션 → 통로 개방. 힘이 세졌다는 것을 시각화. |

---

### 패턴 G: 메커닉의 순차 레이어링 (Sequential Mechanic Layering)

**정의:** 복잡한 시스템을 한 번에 노출하지 않고, 시간 순서에 따라 하나씩 추가. 각 레이어가 정착된 후 다음 레이어 추가.

**작동 원리:**
- 초기: 핵심 동사 1개만 활성화
- 중반: 핵심 동사가 안정화되면 2번째 동사 추가
- 후반: 두 동사의 조합을 요구하는 상황 등장

**사례:**
- Celeste: 달리기 → 점프 → 벽타기 → 대시 순서로 스크린별 도입
- Mega Man X: 달리기 → 점프 → 벽점프 → 대시 순서로 스테이지별 도입
- INSIDE: 달리기/점프 → 잡기 → 특수 능력 순서

**PA 적용:**

| 시점 | 활성화된 메커닉 |
|:--|:--|
| S1 (대장간) | 이동 (좌/우), 상호작용 (↑) |
| S2 (봉인된 복도) | + 점프 |
| S3 (첫 전투) | + 공격 (Z키) |
| S4 (에코 타격) | + 에코 활성화 — 단, 자동 트리거 (플레이어가 버튼 외움 전에 게임이 시연) |
| S6 (아이템계) | + 아이템 EXP 개념 도입 (HUD 카운터) |
| S7 (보스전) | + 보스 패턴 읽기 (텔레그래프 있음) |
| S11 (2차 다이브) | 모든 메커닉 자발적 조합 — 플레이어가 스스로 선택하고 실행 |

---

### 패턴 H: 선택적 고급 교습 (Optional Advanced Teaching)

**정의:** 필수 경로가 아닌 곳에 숨겨진 고급 메커닉을 배치. 탐험한 플레이어만 발견. 강요하지 않으므로 발견이 더 가치 있음.

**작동 원리:**
- 기본 루트에서 약간 벗어난 선택적 공간
- 해당 공간에서 고급 메커닉의 시연 또는 보상
- 발견하지 못해도 게임 진행에 지장 없음 — 하지만 발견하면 강력한 우위

**사례:**
- Hollow Knight King's Pass: 왼쪽 비밀 구역 — 필수 경로가 아니지만 전투 교습
- Super Metroid 에테쿤: 필수 경로가 아닌 숨겨진 방에 존재
- Dark Souls: 2차 만남에서 낙하 공격은 선택 — 발견한 플레이어에게 보상

**PA 적용:**

| PA 선택적 교습 요소 | 내용 |
|:--|:--|
| 증식 구역 막다른 길 (S10) | 필수 경로가 아님. 탐험한 플레이어만 "벽이 자라고 있다"는 위기를 직접 목격. |
| 대공동 전망 (S14) | 필수 통과 구역이지만 멈춰서 바라보는 것은 선택. 경외감은 능동적으로 받아들여야 함. |
| 숨겨진 갈림길 | 아이템계 지층에서 보물방으로 가는 숨겨진 문. 벽 공격으로 발견 가능. |

## 3. PA 첫 30분 환경 교습 매핑

### 3.1 스크린별 적용 패턴 매트릭스

| 스크린 | 교습 목표 | 적용 패턴 | 구체적 메커니즘 |
|:--|:--|:--|:--|
| S1 대장간 | 이동, 상호작용 | A (공간 유도) + B (시각 차이화) + E (환경 서사) | 모루 위 빛나는 검 → 시선 유도. 균열 지도 → 에르다의 직업 맥락. 방 구조 → 탐험 강제. |
| S2 봉인된 복도 | 점프, 메트로베니아 훅 | A (공간 유도) + B (시각 차이화) | 바닥 갭이 점프 강제. 봉인된 문 + 균열 → "나중에 올 수 있겠지." |
| S3 첫 전투 | 공격, 아이템 드롭 | D (죽음 교습) + F (즉각 보상) | 슬라임 느림 + 2히트 → 실패 비용 낮음. 히트스탑 → 즉각 감각 보상. 드롭 → 아이템 획득 교습. |
| S4 에코 타격 | 에코 = 특별한 도구 | C (행동 시연) + F (즉각 보상) | 에르다가 애니메이션으로 시연. 바닥 붕괴 + 백색 섬광 → "이것은 전투 공격과 다르다." |
| S5 아이템계 착지 | 아이템계 = 검 안의 공간 | A (공간 유도) + B (시각 차이화) | 벽의 결정 무늬 = 검의 칼날 무늬. 파란빛 = Magic 검 색상. 연결 고리를 텍스트 없이 색으로 전달. |
| S6 아이템계 전투 | 야생 이노센트, EXP | G (순차 레이어링) + F (즉각 보상) | 기존 전투 메커닉과 동일 조작. 새 요소 = EXP 카운터만 추가. 익숙한 것에 새 레이어 하나. |
| S7 기억의 수문장 | 보스 패턴 읽기 | D (죽음 교습) + F (즉각 보상) | 0.5초 텔레그래프 → 반응 가능한 예고. 사망 = 보스 앞 리스폰. ATK +8 화면 중앙 표시. |
| S8 탈출 제단 | 아이템계 탈출 메커닉 | A (공간 유도) + B (시각 차이화) | 빛나는 제단. 방 안 유일한 발광 오브젝트. "여기가 출구다." |
| S9 귀환 + 발견 | 세계 변화 가시화 | E (환경 서사) + B (시각 차이화) | 없던 벽이 생겼다. 더 밝은 색 석재. "내가 없는 동안 세계가 변했다." |
| S10 증식 구역 | 증식 = 위기 | H (선택적 고급 교습) + E (환경 서사) | 증식하는 벽이 화면에서 팽창. 막다른 길. "이것이 왜 싸워야 하는지"의 이유. |
| S11 두 번째 다이브 | 자발적 루프 선택 | C (행동 시연) + G (순차 레이어링) | 이번엔 플레이어가 직접 시작. 첫 다이브에서 본 것을 스스로 실행. 주체성 확립. |
| S12 ATK 게이트 | 강화 = 새 영역 | F (즉각 보상) + D (죽음 교습 변형) | 게이트 통과 = 에코 타격 → 균열 → 개방. 실패 = 에코 튕김 + 붉은 섬광 (죽음 없이 정보만). |
| S14 대공동 전망 | 세계의 광대함 | H (선택적 고급 교습) + E (환경 서사) | 끝없는 수직 낙하. 저 아래 Tier 3, 4가 보임. 경외감 = 동기의 원천. |
| S16 그림자 시장 | 세이브, NPC, 소속감 | E (환경 서사) + A (공간 유도) | NPC들이 보임. 등불. 북적임의 오디오. "혼자가 아니다." |
| S17 발코니 결의 | 루프 재진입 동기 | C (행동 시연) + B (시각 차이화) | 에코 공명. 아래를 보는 에르다. 한 발짝 앞으로. 페이드 아웃. "내려가야 한다." |

### 3.2 PA 설계 공리와 패턴의 정합성 검증

| 설계 공리 | 담당 패턴 | 검증 결과 |
|:--|:--|:--|
| 에르다 0발화 | 패턴 C, E, B | 에르다의 행동(C)과 환경(E)이 대사를 대체. 색과 빛(B)이 UI를 대체. |
| NPC 없음 (첫 20분) | 패턴 A, B, D, F | 공간, 시각, 즉각 피드백, 실패 교습이 NPC 지시를 대체. |
| UI 튜토리얼 금지 | 패턴 B, F | 빛, 색, 파티클, 숫자가 UI 프롬프트를 대체. "Z키로 공격" 대신 슬라임이 에르다에게 다가옴. |
| 세계가 스스로 설명 | 패턴 E, A, H | 건축(E), 공간 배치(A), 숨겨진 발견(H)이 세계 서사를 전달. |

모든 설계 공리가 최소 2개 이상의 패턴으로 커버됨. 정합성 확인.

## 4. 위험 요소 및 주의사항

### 4.1 환경 교습의 실패 조건

환경 교습은 강력하지만, 아래 조건에서 실패한다:

| 실패 조건 | 증상 | PA 위험 지점 |
|:--|:--|:--|
| 시각 언어 불일치 | 플레이어가 중요한 오브젝트를 배경으로 오인 | 검의 파란빛이 너무 약하면 모루 위 검이 배경처럼 보임 |
| 피드백 지연 | 행동과 결과 사이 0.5초 이상 딜레이 | ATK 게이트 거부 피드백이 느리면 "내가 잘못 누른 것인지" 불명확 |
| 복잡도 과부하 | 첫 스크린에서 너무 많은 것을 동시에 요구 | 에코 타격 시퀀스에서 조작 입력이 필요하다면 너무 이른 복잡도 |
| 선택지 없는 학습 | 모든 것이 강제 — 발견의 쾌감이 없음 | 전체 첫 30분이 완전 선형이면 탐험 판타지 훼손 |
| 사망 비용 과다 | 죽음이 교습이 아닌 처벌 | 기억의 수문장에서 사망 시 아이템계 입구 리스폰 반드시 보장 |

### 4.2 에르다 0발화의 특수 위험

PA는 주인공이 완전 침묵이므로 일반 게임보다 더 강한 시각 언어가 요구된다:

- **위험:** 에코 타격이 "실수"처럼 느껴질 수 있음 — 의도한 행동임을 어떻게 전달?
  - **해결:** S4에서 에르다의 시선이 의도적으로 모루를 향함. 준비 동작 애니메이션이 "이것은 계획된 행동"임을 시연.

- **위험:** 아이템계가 "벌칙 공간"처럼 느껴질 수 있음
  - **해결:** 착지 시 파란빛과 결정 무늬 = 검의 내부 → 신비롭고 아름다운 공간. BGM이 월드와 다른 경이로운 톤으로 즉시 전환.

- **위험:** ATK 게이트 "강화 후 귀환" 루프를 설명할 수 없음
  - **해결:** 패턴 F — 보스 후 ATK +8이 화면에 크게 표시. 귀환 후 게이트 앞에서 에코를 자연스럽게 시도. 이전에 거부된 것이 이번엔 통과됨. "강해지면 길이 열린다"를 행동으로 전달.

## 5. 구현 우선순위 권고

리서치를 바탕으로 PA 첫 30분에서 구현 우선도가 높은 환경 교습 요소:

| 우선도 | 요소 | 근거 |
|:--|:--|:--|
| P0 (즉시) | 에코 타격 시퀀스 애니메이션 완성도 | S4가 전체 게임의 핵심 "THE MOMENT". 시각 시연 품질이 아이템계 진입 교습의 전부. |
| P0 (즉시) | 보스 사망 후 ATK +8 피드백 | 패턴 F의 핵심. "아이템계 = 성장"이라는 루프를 처음으로 전달하는 순간. |
| P1 (1주) | 시각 언어 일관성 확인 | 파란빛 검, 붉은 이노센트, 보스 텔레그래프 색상 — 일관된 시각 문법 필요. |
| P1 (1주) | 사망/리스폰 속도 최적화 | 기억의 수문장 사망 시 3초 이내 보스 방 앞 리스폰 보장. |
| P2 (2주) | 증식 구역 증식 벽 애니메이션 | "세계가 좁아지고 있다"를 화면에서 직접 보여줘야 E(환경 서사)가 작동. |
| P2 (2주) | 대공동 전망 패럴랙스 | S14의 경외감 = "더 깊이 가야 한다"는 동기의 시각적 근거. |
| P3 (3주) | 대장간 환경 소품 완성 | 균열 지도, 선반 도구 — 에르다의 맥락을 전달하는 E(환경 서사). |
