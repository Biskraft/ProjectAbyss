# 스파이크 피처 & 경쟁 환경 분석 — ECHORIS 비판적 평가

> **작성일:** 2026-04-03
> **담당:** 연구원
> **목적:** GDC 스파이크 피처 이론을 학습하고, 이를 기반으로 ECHORIS의 경쟁적 위치를 냉정하게 평가
> **참조:** GDC Vault (Klei "Be Spiky", Subnautica Postmortem), Steam 시장 데이터

## 1. 스파이크 피처란 무엇인가

### 1.1. 정의

> "Spiky features are the differentiators — the things that make a game not necessarily good, but **different**."

게임을 "좋게" 만드는 것이 아니라 **"다르게"** 만드는 차별화 요소. 한두 가지를 극단적으로 밀어붙여 강렬한 인상을 남기는 설계 철학.

### 1.2. GDC 발표

**Jamie Cheng (Klei, 2016):** "Be Spiky — A Decade of New Ideas"
- "모든 사람을 기쁘게 하려 하면 영혼 없는 게임이 된다. 비전에 충실하라."
- Don't Starve: 외적 보상을 완전히 배제, 내적 보상에 올인하는 "뾰족한" 결정

**Jonas Boetel (Unknown Worlds, 2019):** "Subnautica Postmortem"
- **"뾰족한 거품(Spiky Bubble)"** 비유:
  - 뾰족한 부분은 일부 플레이어를 밀어내지만, 좋아하는 플레이어는 **더 강하게** 끌어당김
  - 이 방정식은 양의 합(positive sum)
- "자신의 스파이크를 살펴보고, 돋보이는 것에 집중하고, 그것을 더 하라"
- 매 업데이트마다 "하나의 눈에 띄는 피처" → 스트리머의 관심 → 세일즈

### 1.3. 성공 게임의 스파이크

| 게임 | 스파이크 | 결과 |
|:-----|:---------|:-----|
| Subnautica | 수중 탐험+공포 (전투 의도적 배제) | 500만+ |
| Hollow Knight | 거대 세계 + $15 대비 압도적 콘텐츠 | 1000만+ |
| Animal Well | 퍼즐의 극한 깊이 + 커뮤니티 수수께끼 | 96% 긍정 |
| Nine Sols | 세키로급 패리 + 타오펑크 미학 | 95% 긍정 |
| Vampire Survivors | 극한 접근성 + $3 + 중독성 | 1000만+ |

### 1.4. 왜 뾰족함 > 균형인가

- "괜찮은 게임"은 이제 기본값. OK = 아무것도 아닌 것
- Steam에 매일 수십 개 출시. 7/10 게임은 묻힘. 한 가지 10/10이 입소문
- 스트리머/유튜버: "이 게임에서 이것만큼은 미쳤다"라고 말할 수 있어야 콘텐츠
- 2025 데이터: 리뷰 1000개+ 달성 비율 2.99%. 대다수는 리뷰 10개도 못 받음

## 2. 픽셀아트 + 메트로베니아 시장 (2023-2026)

### 2.1. 주요 출시작

| 게임 | 출시 | Steam | Metacritic | 스파이크 |
|:-----|:-----|:------|:-----------|:---------|
| **Nine Sols** | 2024.05 | 95% | 87 | 타오펑크 + 세키로급 패리 |
| **Animal Well** | 2024.05 | 96% | 91 | 다층 퍼즐 + 커뮤니티 미스터리 |
| **Ender Magnolia** | 2025.01 | 98% | 89 | 아름다운 아트 + 감정 스토리 |
| **Blasphemous 2** | 2023.08 | 90% | 84 | 극한 고딕 비주얼 |
| **Minishoot** | 2024.04 | 98% | 87 | MV + 트윈스틱 융합 |
| **Ultros** | 2024.02 | Mixed | 79 | 사이키델릭 아트 (전투 약함) |
| **Moonscars** | 2022.09 | 83% | ~75 | 없음 → "영혼 없다" 평가 |

### 2.2. 성공 vs 실패

**성공 공통점:**
1. 명확한 스파이크 (Nine Sols=패리, Animal Well=퍼즐)
2. 독자적 비주얼 아이덴티티 (타오펑크, 뫼비우스 영감)
3. 가격 대비 콘텐츠 비율이 높음

**실패 교훈:**
- Prince of Persia Lost Crown: 86 메타크리틱인데 130만 카피만. "왜 이 게임이어야 하는가?"가 없으면 실패
- Ultros: 혁신적 아트만으로 부족. 전투/이동이 뒷받침되어야
- Moonscars: "소울즈라이크 MV"라는 포지셔닝이 너무 흔함

### 2.3. 시장 포화

- MV 시장: 2024년 $1.5B → 2033년 $4.2B (CAGR 15.5%)
- 인디가 전체 MV의 70%
- **"또 하나의 픽셀아트 메트로베니아"로는 절대 부족**

### 3.1. 경쟁자

| 게임 | 평균 플타 | 리텐션 비결 |

### 3.2. 직접 경쟁자

> **웹 기반 횡스크롤 액션 RPG + 아이템계 = 확인되지 않음.** 직접 경쟁자는 사실상 없다.

기회이자 경고 — 시장이 없는 건 "아무도 성공 못 했다"는 뜻이기도.

## 4. ECHORIS 비판적 평가

### 4.1. 가장 큰 위험: "모든 것을 하려는" 설계

현재 PA는 동시에:
- 메트로베니아 (탐험 + 능력 게이트)
- 소울즈라이크 (타격감 전투)
- 디스가이아 (아이템계 + 야리코미)
- MMORPG (멀티플레이 + 허브 + 거래)
- 로그라이크 (절차적 생성)
- 자동 사냥 (모바일/방치형)

**6가지를 다 하겠다 = "둥근 거품"의 정의.** Subnautica GDC의 핵심 교훈을 정면 위반.

### 4.2. 비주얼 레드오션

PA의 톤: "고딕, 다크 판타지, 하이디테일 2D"

= Blasphemous, Moonscars, Crowsworn, Ender Lilies와 동일한 포지셔닝. Moonscars가 "영혼 없다"는 비판을 받은 이유가 이 동질성. Nine Sols가 성공한 이유는 "타오펑크"라는 전혀 다른 미학.

### 4.3. 웹 플랫폼의 기술적 천장

- Nine Sols/Blasphemous 2 수준의 타격감을 브라우저에서 구현하기 극도로 어려움
- 프레임 타이밍 불안정성, GC 히치, 입력 지연
- 플레이어는 "브라우저 치고 괜찮다"로 만족 안 함

### 4.4. 타겟 오디언스의 모순

"야리코미 마니아(코어)"를 메인 타겟으로 삼으면서 "웹 브라우저의 캐주얼 접근성"을 강조하는 것은 근본적 모순.

### 4.5. 경쟁 포지션 요약

| 측면 | 강점 | 약점 |
|:-----|:-----|:-----|
| 독창성 | 아이템계 재귀 진입 = 유일무이 | 겉에서 보면 "또 하나의 MV" |
| 접근성 | 웹 = 설치 불필요 | 웹 = 성능/타격감 한계 |
| 깊이 | 야리코미 설계가 매우 정교 | 설계 문서만 정교, 구현은 Build 0 |
| 비주얼 | 고딕 다크 판타지 | 레드오션 비주얼 |
| 경쟁 | 웹 기반 아이템계 = 직접 경쟁자 없음 | 간접 경쟁자(PoE, Diablo)가 너무 강함 |

## 5. 개선 제안

### 5.1. 스파이크를 하나 골라 극단까지 밀어라

**"아이템계 재귀적 진입"이 가장 강력한 후보.**

조건:
- **첫 30분 안에 체험 가능해야.** Phase 2에 배치하면 그 전에 이탈
- **"아이템 속으로 들어간다"가 시각/감각적으로 충격적이어야.** "아, 이건 처음이다" 반응 필수
- **이 메커닉을 중심으로 나머지를 정리.** 자동 사냥? 삭제. 거래소? 후순위. 멀티? 아이템계 안에서만

### 5.2. 비주얼 아이덴티티를 재고하라

"고딕 다크 판타지"는 Blasphemous/Crowsworn과 싸우는 것. "기억의 지층"은 훨씬 독자적인 비주얼 언어를 가질 수 있음:
- 현실적 → 인상주의적 → 추상적으로 변형되는 미학 진행
- 이것이 아이템계 재귀 진입과 결합되면 그 자체가 스파이크

### 5.3. 웹 플랫폼을 스파이크로 만들어라

"브라우저 치고 괜찮은 MV"가 아니라:
> **"URL 하나로 친구에게 아이템계를 공유하고, 클릭 한 번으로 합류하는 유일한 게임"**

Steam 게임이 절대 따라할 수 없는, 웹만의 스파이크.

### 5.4. 스코프를 대폭 축소하라

- Vampire Survivors: 하나의 메커닉으로 시작
- Subnautica: 매 업데이트마다 "하나의 스파이크"에 집중
- Minishoot: 2인 팀이 하나의 장르 융합에 집중 → 14만 카피

**Phase 1에서 증명할 것은 단 하나:** "아이템 안으로 들어가는 경험이 재미있는가?"

이것만 증명되면 나머지는 따라옴. 이것이 안 되면 나머지는 전부 의미 없음.
