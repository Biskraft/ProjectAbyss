## 1. 개요 (Concept)

### 1.1. 설계 의도 (Intent)

> "듀얼 이노센트는 두 가지 속성을 하나의 슬롯에 압축한 존재다. 단일 이노센트보다 각 속성의 효율은 낮지만, 슬롯 1개를 절약하는 대가로 빌드의 방향성을 결정하는 열쇠가 된다. 이 선택은 수치 최적화가 아니라 전투 스타일의 질적 전환이다."

듀얼 이노센트 시스템은 이노센트 성장의 두 번째 레이어다. 첫 번째 레이어(SYS-INC-01)가 "야생 이노센트를 찾아 복종시켜라"라면, 두 번째 레이어는 "복종시킨 이노센트를 결합하여 새로운 존재로 만들어라"다.

디스가이아의 이노센트 합성 구조와 Project Abyss의 슬롯 제약 철학을 결합한 결과물이다. 단순 레벨 합산(동종 합성)보다 난이도가 높고 리스크가 있지만, 성공 시 빌드 구성의 새로운 차원이 열린다.

에르다의 세계관 언어로: "두 기억 조각을 충분히 두드려 녹인 뒤 하나의 틀에 부어 새 형태를 만드는 것 — 성공하면 두 기억이 하나로 응결된다. 실패하면 두 조각 모두 흩어진다."

## 2. 핵심 규칙 (Core Rules)

### 2.1. 듀얼 이노센트 정의

듀얼 이노센트는 **두 가지 단일 스탯 또는 효과를 하나의 이노센트 슬롯에서 제공**하는 특수 이노센트다. 일반 단일 이노센트와 구분되는 속성:

- **슬롯 점유:** 1개 (단일 이노센트와 동일)
- **제공 효과:** 두 종류의 스탯 또는 효과 (각각 단일 대비 `Dual_Stat_Ratio`배 적용, 기본값 0.7)
- **합성 방식:** 이종 합성(레시피 필요) 또는 팜 번식으로만 생성. 동종 합성 불가
- **야생 조우:** 아이템계 2지층 이상에서만 출현. 1지층에서는 등장하지 않음
- **서사 예외:** 넘버 원(No.1)은 어떠한 듀얼 합성에도 사용 불가

### 2.2. 듀얼 이노센트 생성 경로

듀얼 이노센트를 획득하는 방법은 세 가지다:

```
경로 A (이종 합성): 재료 이노센트 A + B (레시피 일치, 복종 상태)
                    → 확률 판정 (Dual_Synthesis_Success_Rate 기본값 60%)
                    → 성공: 듀얼 이노센트 생성
                    → 실패: 재료 두 개 소멸 + Mana_Crystal 일부 반환

경로 B (팜 번식): 팜에 이종 이노센트 2개 배치 + 아이템계 전투 횟수 누적
                  → Farm_Breed_Chance (기본값 2% / 전투)
                  → 성공: 신규 듀얼 이노센트 탄생 (팜 빈 슬롯 필요)
                  → 실패: 누적만 됨 (소멸 없음)

경로 C (야생 조우): 아이템계 2지층 이상 방 클리어 시
                    → Dual_Wild_Spawn_Rate (기본값: 지층 번호 × 2%)
                    → 조우 및 격파 시 복종 전환
```

### 2.3. 이종 합성 상세 규칙

#### 합성 조건

합성을 진행하려면 다음 조건을 모두 충족해야 한다:

1. 레시피에 정의된 두 종류의 이노센트 보유 (타입 일치 필수)
2. 두 재료 이노센트 모두 복종(Subdued) 상태
3. Hub 합성 NPC 접근 (허브 내 전용 위치)
4. Mana_Crystal `Dual_Synthesis_Cost`개 보유 (기본값 50개)

#### 합성 결과 계산

```yaml
# 이종 합성 성공 시 결과값
성공_듀얼_이노센트:
  타입: [타입A + 타입B 조합명 — 레시피 테이블 참조]
  레벨: (A.Level + B.Level) × Dual_Level_Ratio  # 기본값 0.4
  상태: 복종 (Subdued)
  등급: 재료 중 낮은 등급 기준 (예: Gleam + Radiant → Gleam)
  배율: 각 스탯 × Dual_Stat_Ratio  # 기본값 0.7

# 예시 계산
재료: Gladiator Lv.80 (복종, Radiant) + Guardian Lv.60 (복종, Gleam)
결과_레벨: (80 + 60) × 0.4 = 56
결과_등급: Gleam (낮은 쪽 기준)
결과_타입: Iron Fist (STR + VIT)
STR_보너스: 56 × 0.7 = 39.2 → 39 (소수점 내림)
VIT_보너스: 56 × 0.7 = 39.2 → 39 (소수점 내림)
```

#### 합성 실패 처리

```yaml
# 이종 합성 실패 시
실패_처리:
  재료_상태: 두 이노센트 모두 영구 소멸
  비용_환급: Mana_Crystal × Dual_Fail_Refund  # 기본값 10개 (소비 50개의 20%)
  기타: 쿨다운 없음. 즉시 재시도 가능 (재료 조달 필요)
```

#### 합성 성공률 조정 인자

기본 성공률 `Dual_Synthesis_Success_Rate`(기본값 60%)는 다음 인자에 따라 조정된다:

| 조정 인자 | 효과 | 스택 방식 |
| :--- | :--- | :--- |
| 재료 레벨 합산 낮음 (합산 < 50) | 성공률 -15%p (낮은 레벨 재료는 합성 안정성 낮음) | 가산 |
| 재료 레벨 합산 높음 (합산 > 200) | 성공률 +10%p (충분히 성장한 재료) | 가산 |
| 재료 등급 불일치 (예: Dim + Radiant) | 성공률 -10%p (이질적 기억 구조) | 가산 |
| 재료 등급 일치 | 성공률 +5%p | 가산 |
| 유니크 이노센트 "Lucky Star" 장착 시 | 성공률 +20%p | 별도 가산 |

성공률 상한: `Dual_Synthesis_Max_Rate` (기본값 85%). 하한: `Dual_Synthesis_Min_Rate` (기본값 20%).

### 2.4. 듀얼 스탯 수치 공식

```yaml
# 듀얼 이노센트 스탯 계산 공식
dual_stat_formula:
  효과_A: Dual.Level × Dual_Stat_Ratio  # 단일 스탯 기준 레벨당 효과 × 0.7
  효과_B: Dual.Level × Dual_Stat_Ratio
  소프트_캡: 단일 이노센트 소프트_캡 × Dual_Cap_Ratio  # 기본값 0.833
            # 예: Gladiator 캡 300 → Iron Fist의 STR 캡 = 300 × 0.833 = 249 ≈ 250
  캡_이후_감소율: 기본 스탯형 감소율(0.2/레벨) × Dual_Decay_Ratio  # 기본값 0.5
                # 캡 이후 듀얼은 더 빨리 감소하여 단일의 캡 초과 가치 보존

# 슬롯 효율 비교 (레벨 200 Gladiator 기준)
단일_Gladiator_Lv200:
  STR: +200 (캡 300 이하)

듀얼_Iron_Fist_Lv200:
  STR: 200 × 0.7 = +140
  VIT: 200 × 0.7 = +140
  총합_효율: STR +140 + VIT +140 = 두 단일 이노센트(Lv200 각각)의 합 대비 70%
  슬롯_효율: 슬롯 1개로 두 스탯 동시 확보 (슬롯이 제약인 장비에서 핵심 가치)
```

### 2.5. 팜 번식 규칙

팜 번식은 의도적 합성과 별개의 "우연한 생성" 경로다. SYS-INC-02 팜 시스템의 번식 메커니즘을 상속하며, 듀얼 이노센트 관점에서의 상세 규칙을 다음과 같이 정의한다.

```
번식 발생 조건:
  1. 서로 다른 종류의 이노센트 2개가 팜 슬롯에 배치됨
  2. 두 이노센트 모두 복종(Subdued) 상태
  3. 팜에 빈 슬롯이 최소 1개 존재
  4. 아이템계 전투 1회 완료 시마다 확률 판정

번식 확률 = Farm_Breed_Chance × 등급_계수 × 레벨_계수
  Farm_Breed_Chance: 기본값 0.02 (2% / 전투)
  등급_계수: Dim×Dim = 1.0 / Gleam 포함 = 1.5 / Radiant 포함 = 2.0 / Myth 포함 = 3.0
  레벨_계수: (A.Level + B.Level) / 200 (상한 3.0)

번식 결과:
  탄생 이노센트 타입: 레시피에 해당 조합이 있으면 그 타입, 없으면 번식 불발
  탄생 레벨: (A.Level + B.Level) × Farm_Breed_Level_Ratio (기본값 0.05)
  탄생 등급: 재료 중 낮은 등급
  탄생 상태: 복종 (Subdued) — 팜에서 태어난 이노센트는 이미 복종 상태

예시:
  Gladiator Lv.60 (Gleam) + Guardian Lv.40 (Gleam) 팜 배치
  전투 1회 종료
  확률 = 0.02 × 1.5 × (100/200) = 0.015 = 1.5%
  성공 시: Iron Fist Lv.5 탄생 (레벨 = (60+40)×0.05 = 5), Gleam 등급
```

번식으로 탄생한 이노센트는 레벨이 매우 낮지만, 재료 이노센트가 소멸하지 않는다는 점에서 이종 합성과 본질적으로 다른 경로다. 번식은 "덤으로 얻는 보상"이고 합성은 "목표를 향한 투자"다.

### 3.3. 금지 조합 (Forbidden Combinations)

아래 조합 쌍은 이종 합성 시도 불가다. 레시피 UI에서 "공명 불가" 메시지와 함께 합성 버튼이 비활성화된다.

```yaml
# 금지 조합 — 속성 충돌 또는 밸런스 붕괴 위험
forbidden_combinations:
  - [Guardian, Dietician]   # VIT + MaxHP 직접 중복 — 생존 인플레이션
  - [Guardian, Scholar]     # VIT + MaxMP 중복 — 과도한 자원 확보
  - [Sprinter, Drifter]     # SPD + 이동속도 직접 중복 — 속도 폭주
  - [Sprinter, Dasher]      # SPD + 대시거리 직접 중복 — 기동성 폭주
  - [Trickster, Scavenger]  # LCK + 드롭률 직접 중복 — 파밍 인플레이션 (Lucky Reaper로 별도 정의)
  - [Critical Eye, Executioner]  # 크리 확률 + 크리 배율 직접 조합 — 크리 피해 폭주
  - [Statistician, Broker]  # EXP + 골드 직접 조합 — 경제 인플레이션
  - [서사_이노센트, 모든_타입]  # 넘버 원(No.1) 합성 불가
```

> **설계 노트:** 금지 조합은 "이 두 속성은 결합하면 안정적이지 않다"는 세계관 논리로 설명된다. 에르다의 대사: "두 기억 조각이 서로 같은 성질을 향하면 합쳐지는 게 아니라 터진다."

### 3.4. 레시피 발견 메커니즘

듀얼 이노센트 레시피는 처음부터 전체 공개가 아닌, 탐험으로 발견하는 정보다.

```
레시피 발견 경로:
  A. 아이템계 특수 방 "이노센트 타운(Innocent Town)" 내 NPC 대화
     → 특정 레시피 1종 힌트 제공 (타입명만 공개, 등급/조건 미공개)

     → 해당 이노센트의 레시피 자동 등록 (격파 시 "[타입명] 레시피 발견" 알림)

     → 이노센트 결정(재화) 10개 소모로 레시피 1종 구매
  D. 아이템계 보스 처치 보상 (일부 보스)
     → 고급 레시피(복합 듀얼형) 보상으로 드롭

초기 해금:
  게임 시작 시 기본 레시피 2종 자동 해금: Iron Fist, Arcane Body
  (가장 직관적인 조합으로 이종 합성 시스템의 존재를 자연스럽게 소개)
```

## 4. 듀얼 이노센트 시각 차별화 (Visual Differentiation)

### 4.1. 아이콘 표현

듀얼 이노센트는 단일 이노센트와 시각적으로 명확히 구분되어야 한다. 디자인 방향은 다음과 같다:

| 요소 | 단일 이노센트 | 듀얼 이노센트 |
| :--- | :--- | :--- |
| 아이콘 배경 | 단색 (속성 색상) | 두 속성 색상의 대각 그라디언트 |

| 레벨 표시 | 흰색 숫자 | 금색 숫자 (듀얼 강조) |
| 효과 툴팁 | 단일 효과 행 | 두 효과를 나란히 배치 + "듀얼 이노센트" 뱃지 |

### 4.2. 등급별 추가 표현

```
Dim 듀얼: 그라디언트 채도 50% (흐릿한 두 색상)

Myth 듀얼: 황금 테두리 + 두 속성 파티클 방출 이펙트
```

## 5. 레어리티 연동 (Rarity Implications)

### 5.1. 장비 레어리티와 듀얼 이노센트의 가치 관계

듀얼 이노센트의 핵심 가치(슬롯 절약)는 장비 레어리티(슬롯 수)에 따라 달라진다. 이 관계가 "어떤 장비에 어떤 이노센트를 넣을 것인가"의 전략적 깊이를 만든다.

| 장비 등급 | 슬롯 수 | 듀얼 이노센트 가치 | 전략적 판단 |
| :-------- | :-----: | :--- | :--- |
| Normal    | 2       | 매우 높음 — 슬롯 1개 절약이 전체의 50% | 듀얼 이노센트 1개로 두 역할 확보가 핵심 |
| Magic     | 3       | 높음 — 슬롯 1개 절약이 33% | 듀얼로 만든 여유 슬롯에 유니크 추가 가능 |
| Rare      | 4       | 중간 — 전략적 빌드 설계 시 유용 | 세 역할(STR+VIT+유니크+파밍) 중 우선순위 선택 |
| Legendary | 6       | 낮음~중간 — 유니크 중심 빌드 보조 | 유니크 2개 이상 채울 여유를 만드는 역할 |
| Ancient   | 8       | 낮음 — 단일 이노센트 극대화가 더 유리 | 특수 목적(파티 시너지 빌드)에서만 사용 |

### 5.2. 이노센트 등급(Dim~Myth)과 듀얼의 관계

듀얼 이노센트의 등급은 재료 중 낮은 등급을 따른다. 이 규칙은 "고등급 재료를 통일해야 고등급 듀얼이 나온다"는 조건을 형성한다.

```
등급 상승 효과 (이노센트 자체 등급):
  Dim 듀얼: 기본 레벨 범위 1~20, 복종 후 ×2.0 레벨
  Gleam 듀얼: 기본 레벨 10~60, 복종 후 ×2.5 레벨
  Radiant 듀얼: 기본 레벨 40~100, 복종 후 ×3.0 레벨
                + 추가 효과: 두 속성 각각 소프트 캡 +20% 상향
  Myth 듀얼: 기본 레벨 100~250, 복종 후 ×4.0 레벨
             + 추가 효과: 두 속성 각각 소프트 캡 +40% 상향
             + 복수 Myth 이노센트 장착 보너스 연동 (2개 = +5%, 4개 = +10%)
```

### 5.3. 야리코미 깊이와 듀얼 이노센트의 연결

야리코미(극한 파밍)의 단계별로 듀얼 이노센트가 다른 역할을 한다:

```
Phase 1 (초기): 단일 이노센트 복종 학습 단계. 듀얼 시스템 미해금.

Phase 2 (중기 야리코미):
  - 아이템계 2지층 도달 후 첫 야생 듀얼 이노센트 조우로 시스템 발견
  - 기본 레시피(Iron Fist, Arcane Body) 해금
  - Normal~Magic 장비에서 슬롯 절약 가치 체감

Phase 3 (고급 야리코미):
  - Legendary 장비에서 복합 듀얼형 레시피 활용
  - 유니크 이노센트와 듀얼의 조합으로 빌드 완성도 극대화
  - 소프트 캡 도달 후 Myth 등급 업그레이드 시도

Phase 4 (극한 야리코미 — Ancient 장비 심연):
  - Myth 등급 파티 시너지형 듀얼 합성 도전
  - 순환 진입(고레어리티 아이템계 반복) + Depth Seeker(U-19) 이노센트 조합으로 강화
  - 장비 세트 전체를 아우르는 듀얼+유니크 빌드 완성
```

## 6. 데이터 & 파라미터 (Parameters)

모든 조정 수치는 외부 CSV로 관리한다. GDD 본문의 수치는 기본값 참고용이며, 실제 밸런스 SSoT는 `Sheets/Innocent_Dual_Params.csv` (Phase 2 생성 예정)다.

```yaml
# Innocent_Dual_Params.csv 예정 필드 목록

# 이종 합성 기본 파라미터
Dual_Synthesis_Success_Rate: 0.60      # 기본 합성 성공률 (60%)
Dual_Synthesis_Cost: 50                # 합성 당 Mana_Crystal 비용
Dual_Fail_Refund: 10                   # 실패 시 Mana_Crystal 반환 (20%)
Dual_Synthesis_Max_Rate: 0.85          # 성공률 상한
Dual_Synthesis_Min_Rate: 0.20          # 성공률 하한
Dual_Level_Ratio: 0.40                 # 합성 결과 레벨 = (A+B) × 0.4
Dual_Stat_Ratio: 0.70                  # 각 스탯 단일 대비 배율

# 소프트 캡 조정
Dual_Cap_Ratio: 0.833                  # 단일 캡 × 0.833 = 듀얼 각 스탯 캡
Dual_Decay_Ratio: 0.5                  # 캡 이후 감소율 배율 (단일보다 빠르게 감소)

# 팜 번식 파라미터
Farm_Breed_Chance: 0.02                # 전투당 기본 번식 확률 (2%)
Farm_Breed_Level_Ratio: 0.05           # 번식 결과 레벨 = (A+B) × 0.05

# 성공률 조정 인자
Rate_Adjust_Low_Level: -0.15           # 재료 합산 레벨 < 50 시 패널티
Rate_Adjust_High_Level: +0.10          # 재료 합산 레벨 > 200 시 보너스
Rate_Adjust_Grade_Mismatch: -0.10      # 재료 등급 불일치 패널티
Rate_Adjust_Grade_Match: +0.05         # 재료 등급 일치 보너스

# 야생 듀얼 이노센트 드롭
Dual_Wild_Spawn_Rate_Base: 0.02        # 지층 번호 × 2% (1지층 0%, 2지층 4%...)
Dual_Wild_Min_Stratum: 2               # 최소 등장 지층 (2지층 이상)

# 이종 합성 결과 등급 결정
Dual_Grade_Rule: "min"                 # 재료 중 낮은 등급 기준 ("min" 또는 "avg")
```

## 7. 예외 처리 (Edge Cases)

| 상황 | 처리 규칙 |
| :--- | :--- |
| 넘버 원(No.1) 합성 시도 | UI 차단. "에코에 귀속된 기억은 분리할 수 없습니다" 메시지 표시. 합성 창에서 넘버 원은 선택 불가 상태 |
| 재료 이노센트 중 하나가 팜에 배치된 상태 | 팜에서 먼저 회수 후 합성 가능. "이노센트가 팜에 있습니다. 회수 후 합성하세요" 안내 |
| 합성 결과 레벨이 해당 타입 소프트 캡 초과 | 소프트 캡에서 절사. 예: Iron Fist 캡 250인데 계산 레벨 280 → 250으로 생성 |
| 팜 번식 시도 중 팜 슬롯이 꽉 찬 경우 | 번식 기회 판정은 계속되지만, 결과 탄생 불가. 빈 슬롯 생성 후 다음 전투에서 재판정 (누적 소멸 없음) |
| 이종 합성 성공 후 아이템 슬롯이 꽉 찬 경우 | 생성된 듀얼 이노센트를 "임시 보관함(Inbox)"에 저장. 허브에서 슬롯 정리 후 이동 가능 |
| 동일 조합 레시피가 없는 타입 쌍 합성 시도 | "이 두 기억은 함께 응결되지 않습니다" 메시지. 합성 버튼 비활성화 (재료 소모 없음) |
| 멀티플레이 중 파티원이 듀얼 이노센트를 획득한 경우 | 이노센트는 개인 귀속. 동행자 보상(이노센트 크리스탈/조각)으로 간접 기여 |
| Myth 등급 듀얼 이노센트 이식 시도 | 이식 가능 (복종 이노센트). 단, 이식 비용 골드(`Myth_Transfer_Cost`)가 일반 이노센트 이식 대비 2배 |
| 듀얼 이노센트의 한쪽 스탯이 소프트 캡 도달 후 다른 스탯이 미달인 경우 | 캡 미달 스탯은 계속 성장. 캡 도달 스탯은 감소율 적용. 두 스탯이 독립적으로 관리됨 |
| 서버 접속 끊김 중 합성 확률 판정 발생 시 | 서버 측에서 결과 확정 후 재접속 시 결과 표시. 클라이언트 결과 선행 계산 금지 (서버 권한) |

## 9. 수용 기준 (Acceptance Criteria)

### 9.1. 기능 기준 (Functional)

- [ ] 이종 합성: 레시피에 정의된 두 타입의 복종 이노센트 + Mana_Crystal 비용으로 합성 시도 가능
- [ ] 합성 성공률 60% 판정 작동 및 성공/실패 결과 올바르게 처리
- [ ] 실패 시 재료 두 개 소멸, Mana_Crystal 10개 반환 확인
- [ ] 합성 결과 레벨 = (A.Level + B.Level) × 0.4 계산 정확성 검증
- [ ] 듀얼 스탯 각 0.7배 배율 적용 및 단일 이노센트와 별도 소프트 캡 작동 확인
- [ ] 금지 조합(forbidden_combinations) 시도 시 합성 버튼 비활성화
- [ ] 넘버 원(No.1) 합성 시도 완전 차단
- [ ] 팜 번식: 이종 이노센트 2개 배치 후 전투 횟수 누적에 따른 번식 확률 판정 작동
- [ ] 야생 듀얼 이노센트: 2지층 이상에서만 스폰, 격파 시 복종 전환
- [ ] 레시피 발견 메커니즘: 최초 격파 시 자동 등록, NPC 구매, 보스 드롭 각각 작동
- [ ] 듀얼 이노센트 아이콘 그라디언트 표시 (두 속성 색상 대각 그라디언트)

### 9.2. 체험 기준 (Experiential)

- [ ] **합성 의사결정 긴장감:** 고레벨 재료 2개를 이종 합성에 투입하는 순간 "과연 성공할까?"라는 긴장이 느껴지는가?
- [ ] **슬롯 절약 체감:** Normal/Magic 등급 장비에서 듀얼 이노센트 장착 후 "여유 슬롯에 뭘 더 넣을까"라는 추가 선택이 자연스럽게 생기는가?
- [ ] **실패 후 동기 유지:** 합성 실패로 재료를 잃었을 때 "다시 파밍해서 재도전하겠다"는 회복 의지가 좌절보다 강한가? (실패 환급과 즉시 재시도 가능성이 이를 뒷받침하는가?)
- [ ] **레시피 발견의 보상감:** 아이템계에서 새 레시피를 발견하거나 야생 듀얼 이노센트를 처음 마주치는 순간이 "탐험의 발견 순간"으로 느껴지는가?
- [ ] **빌드 정체성:** 듀얼 이노센트를 중심으로 구성한 빌드가 단순 수치 최적화와 구별되는 "내 스타일"이라는 감각을 주는가?
