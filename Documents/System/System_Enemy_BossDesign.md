## 1. 개요 (Concept)

### 1-1. 설계 의도 (Design Intent)

이 문서는 ECHORIS 월드(World) 공간에 배치된 **6개 진행 보스 + 6개 숨겨진 보스**의 패턴, 내러티브 정체, 능력 게이트 연동 보상을 정의한다.

월드 보스는 단순한 전투 관문이 아니다. **보스 하나하나가 심연 전쟁(Era 3)의 살아있는 증언자**다. 수로의 파수꾼은 100년 전 주인의 명령을 아직도 수행하고 있고, 수호단의 잔영은 전사한 대장의 기억이 갑옷에 붙어 있는 것이다. 에르다는 이들을 처치하는 것이 아니라, 그들의 기억을 "종결"시키는 역할을 맡는다.

보스 설계의 핵심 원칙:

> "처음 마주쳤을 때 압도당하고, 세 번 만나면 패턴이 보이고, 처치하는 순간 이 존재가 왜 여기 있었는지를 이해한다."

이것은 `BossDesign_SideScrolling_Research.md` 섹션 1.3의 Cuphead 학습 사이클(첫 시도 압도 → 3-5회 패턴 파악 → 숙달 → 완전한 숙달감)과 섹션 1.5의 SotN 서사 연동("보스가 그 공간의 탐험 완결을 의미한다") 원리를 통합한 것이다.

### 1-2. 설계 근거 (Reasoning)

| 결정 | 근거 |
| :--- | :--- |
| 보스 = 층위 역사에서 탄생 | 역사적 맥락 없는 보스는 "generic 몬스터"다. 세계관 내러티브가 전투 경험을 감정적으로 만든다. World Bible Layer 1-3 기반 |
| 진행 보스 6개 (층위당 1개) | 메트로베니아 문법상 층위 클리어 = 보스 처치. 처음 만나는 경험이 최고조이므로 1회성 핵심 서사에 집중 |
| 숨겨진 보스 6개 (층위당 1개) | 탐험형 플레이어(Explorer)를 위한 보상. Critical Path와 무관하게 배치. 더 높은 난이도, 고유 보상 |
| 능력 게이트 보상 = 보스 처치 연동 | 메트로베니아 핵심: 보스가 "문"이 된다. 능력 획득의 서사적 의미가 각 보스의 역사와 연결됨 |
| 고정 루틴 + 후반 랜덤 변주 | 리서치 섹션 2.4 권장 방식. 1회성 월드 보스는 순수 학습이 가능해야 하므로 기본 고정. 분노 페이즈에서 가중 랜덤 |
| 에르다의 반응 대사 필수 | 에르다의 독백으로 보스의 정체를 전달. NPC 설명 금지. 디스가이아 1의 캐릭터 감정 표현 방식 오마주 |

### 1-3. 3대 기둥 정렬 (Pillar Alignment)

| 기둥 | 월드 보스 디자인에서의 구현 |
| :--- | :--- |
| 메트로베니아 탐험 | 보스 처치 = 능력 게이트 해금. 처치하지 않으면 접근 불가 층위가 존재. 보스가 탐험 진행의 주요 문 |
| 아이템계 야리코미 | 보스 처치 후 해당 층위에 필드 제단 활성화. 야리코미의 동기가 월드 탐험과 연결 |
| 온라인 멀티플레이 | 월드 보스는 기본 솔로 설계. 단 2인 협동 탐험 시 에르다의 부담 경감 + 대사 변주 존재 |

### 1-4. 저주받은 문제 (Cursed Problems)

| 문제 | 상충하는 요구 | 설계 해법 | 희생/제약 |
| :--- | :--- | :--- | :--- |
| 서사 보스 vs 반복 플레이 | 1회성 경험에 최적화하면 재도전이 지루하고, 반복에 최적화하면 첫 만남의 감동이 약해짐 | 진행 보스는 1회성 서사에 최적화. 재도전은 허용하나 보상은 첫 처치만 최대. 숨겨진 보스는 반복 도전 가능 구조 | 재도전 플레이어에게 첫 감동과 동일한 경험 제공 불가 |
| 보스 난이도 vs 접근성 | 너무 어려우면 이탈, 너무 쉬우면 탐험의 관문이 의미 없어짐 | 스탯이 낮아도 패턴 학습으로 클리어 가능하고, 스탯이 높으면 더 빠르게 클리어 가능한 SotN 방식. 패턴 숙달 ≥ 스탯 의존 | 스탯 편차가 매우 크면 패턴을 무시한 딜 버로우 가능 |
| 패턴 학습 vs 공정성 | 패턴이 예측 불가능하면 불공정, 완전 고정이면 공략집 의존 | 기본 패턴 고정 + 분노 페이즈 가중 랜덤. Cuphead 원칙(모든 공격에 Tell) 준수 | 공략집 공유 시 고정 패턴 구간은 빠르게 메타화됨 |
| 서사 보스 마지막 대면(카엘) vs 전투 | "기억"이 주제인 게임에서 최종 대면이 순수 전투이면 주제와 충돌 | 카엘 오르스는 전투 + 선택으로 끝나는 구조. 전투 자체가 기억의 재생이며, 최종 순간은 전투가 아닌 선택 | 액션 중심 플레이어에게 최종 보스가 "약하다"는 인상 줄 수 있음 |

### 1-5. 위험과 보상 (Risk & Reward)

| 행동 | 위험 (Risk) | 보상 (Reward) |
| :--- | :--- | :--- |
| 진행 보스 처치 | 사망 시 세이브 포인트 복귀. 보스 방 재진입 필요 | 능력 게이트 해금(이동 능력 렐릭), 층위 역사 단편, 층위 내 필드 제단 활성화 |
| 숨겨진 보스 처치 | 진행 보스보다 높은 난이도. 사망 패널티 동일 | 고유 아이템, 이노센트 고급 드랍, 세계관 심층 단편, 숨겨진 마르타의 흔적 |
| 보스 분노 페이즈 돌입 | 패턴 빨라짐, 안전 윈도우 감소 | 분노 돌입 = 보스 처치 직전 구간. 최대 카타르시스 직전 |
| 카엘 오르스 전투 | 4페이즈. 에르다의 서사 완결 전에 실패 시 감정적 좌절 | 최종 선택지 열림. 계시 1-5 완결. 에르다의 감정 아크 결말 |

## 2. 공통 메커닉 (Common Mechanics)

### 2-1. 월드 보스 공통 규칙

모든 월드 보스는 다음 공통 규칙을 따른다:

1. **Tell 의무:** 모든 공격은 반드시 `Tell_Duration` 이상의 예고 동작을 갖는다. Tell 없는 공격은 허용하지 않는다.
2. **페이즈 기반 전투:** 모든 진행 보스는 최소 2페이즈를 갖는다. 페이즈 전환은 HP 임계점 기준이며, 전환 시 무적 연출 + 새 패턴 소개가 있다.
3. **체크포인트:** 보스 방 진입 전 30초 이내 위치에 세이브 포인트가 배치된다. 보스 방 내부에는 체크포인트가 없다.
4. **안전 윈도우:** 기본 페이즈 안전 윈도우 `Safe_Window_Normal`. 분노 페이즈 `Safe_Window_Rage`. 연속 패턴 최소 간격 `Pattern_Gap_Min`.
5. **에르다의 반응 대사:** 보스 첫 조우 시, 보스 처치 시 에르다 독백 2회 의무. NPC 설명으로 대체 불가.
6. **첫 처치 보상:** 보스 처치 시 최대 보상은 첫 처치에만 지급된다. 재도전 시 일반 드랍만 지급.

### 2-2. 페이즈 전환 연출 (Phase Transition)

```yaml
phase_transition:
  trigger: HP 임계점 도달
  sequence:
    - 보스 비발동 상태 전환 (0.5초)
    - 특수 애니메이션 + 이펙트 재생 (1~2초)
    - BGM 변주 또는 새 BGM 시작
    - 새 페이즈 첫 패턴 느린 텔레그래프로 시작 (패턴 소개 구간)
  rules:
    - HP 회복 없음 (리서치 섹션 2.3 원칙)
    - 전환 연출 중 플레이어는 행동 가능
    - 전환 연출 중 보스는 피격 불가 (불공정 방지)
```

### 2-3. Tell 시간 기준 (Telegraph Timing)

| 공격 유형 | Tell 최소 시간 | 텔레그래프 방법 |
| :--- | :--- | :--- |
| 근접 단타 | 300ms | 예비 동작 애니메이션 |
| 근접 광역 | 500ms | 예비 동작 + 색상 강조 |
| 원거리 투사체 | 400ms | 발사 모션 + 조준선 |
| 장판/지속 공격 | 700ms | 바닥 이펙트 예고 |
| 즉사급 공격 | 1000ms 이상 | 위험 구역 표시 + 음향 |
| 분노 페이즈 패턴 | 400ms (최소) | 기본과 동일, 속도만 증가 |

`BossDesign_SideScrolling_Research.md` 섹션 2.1 기준 적용.

### 2-4. 보스 처치 후 능력 드랍 연출

```yaml
ability_drop_sequence:
  trigger: 보스 처치
  sequence:
    1. 보스 사망 이펙트 재생 (1~2초)
    2. 층위 역사 단편 환경 연출 (3~5초 — 벽에 새겨진 기억의 파편, 잔상 등)
    3. 에르다 처치 후 독백 (보이스 오버 없음, 텍스트 독백)
    4. 능력 렐릭 오브젝트 스폰
    5. 플레이어가 렐릭을 획득하면 능력 게이트 해금 연출
    6. 에르다 능력 획득 독백 (마르타/세계관 연결)
```

## 3. 보스별 상세 설계 (Individual Boss Design)

### 3-1. 수로의 파수꾼 (Guardian of the Veinway)

> 구역: 지하 수로 / 유형: 기믹 보스 / 페이즈: 2 / 난이도: ★★☆☆☆

#### 내러티브 정체

번영기에 생명의 수로(Veinway)를 지키기 위해 만들어진 방어 골렘. 최고급 기억 결정체를 동력으로 작동한다. 주인이 심연 전쟁에서 사망한 후에도 마지막 명령("수로를 지켜라")만을 기억하며 100년간 수호를 계속하고 있다. 의지도, 분노도, 적대감도 없다. 다만 명령이 있을 뿐이다.

**에르다의 첫 조우 반응:**
> "...이게 골렘이야? 100년 전 거야? 아직도 작동하고 있다고? 대체 어떤 장인이 만든 거야, 이걸... 잠깐, 이 구동 기억 결정체, 스승님 솜씨 아닌가?"

**에르다의 처치 후 반응:**
> "명령이 끝났다. 이제 쉬어도 돼. 100년은 충분히 오래 지켰잖아."

#### 아레나 설계

- 크기: 40×14타일 (넓고 낮은 수평 아레나)
- 특징: 좌우 각 끝에 수문 레버. 중앙에 낮은 수위의 물 구간(4타일 폭)
- 물 구간에서는 이동 속도 `Water_Move_Penalty` 감소

#### 페이즈 1 패턴

HP `GUARDIAN_P1_HP_RANGE`(파라미터 참조) 구간.

### 3-4. 관측의 눈 (The Observing Eye)

> 구역: 천공의 탑 / 유형: 기믹+군체 혼합 보스 / 페이즈: 3 / 난이도: ★★★★☆

#### 내러티브 정체

천공의 탑(Augurium)을 봉쇄할 때 관측자들이 남긴 최후의 감시 장치. 100년 전 탑을 봉쇄한 관측자들은 "탑 안의 예언이 세상에 누설되면 안 된다"는 이유로 자발적으로 탑에 남아 감시를 계속했다. 그들이 죽은 후, 남긴 집합적 의지가 이 관측 장치에 깃들었다. 단순한 기계가 아니라, 관측자들의 마지막 서약이 물질화된 존재다.

형태는 거대한 발광하는 눈. 수많은 보조 감시 렌즈(작은 눈)가 주변을 맴돈다. 플레이어를 "침입자"로 규정하고 배제하려 하지만, 전투가 깊어질수록 대사에서 "보지 마라, 알아서는 안 된다"는 절박함이 드러난다.

**에르다의 첫 조우 반응:**
> "감시 장치? 이 탑을 봉쇄한 관측자들이 직접 만든 거야? 이게 100년 전 것인데 아직도 작동한다고... 대체 뭘 그렇게 못 보게 하려는 거야."

**에르다의 처치 후 반응:**

#### 내러티브 정체

100년 전 심연 전쟁의 영웅. "심연의 검"을 들고 심연 속으로 뛰어들어 협약을 맺은 장본인. 심연 속에 기억으로 남아 협약의 닻 역할을 하고 있다. 이미 육체는 없다. "카엘 오르스의 기억"이 심연 안에서 응결되어 실체화된 것이다.

에르다와의 대면은 전쟁이 아니다. 카엘은 에르다를 적으로 보지 않는다. 다만 "아직 여기에 들어오면 안 된다"는 협약의 수호자로서 막을 뿐이다. 전투가 깊어질수록 카엘의 대사에서 피로함, 고독, 그럼에도 계속 지켜야 한다는 의지가 드러난다.

최종 순간은 전투가 아닌 선택이다. 에르다가 카엘을 "이길" 때 전투가 끝나지 않는다. 에르다가 카엘의 기억을 진심으로 이해하고 받아들이는 순간, 카엘이 스스로 멈추고 선택지를 내민다.

**에르다의 첫 조우 반응:**

**전투 중 카엘의 대사 예시:**
> (페이즈 2 전환 시): "돌아가라. 네가 알기엔 너무 무거운 것들이 여기 있다."

### 4-1. 설계 원칙

숨겨진 보스는 Critical Path 밖 비경로 탐험 또는 특정 능력 조건에서 진입 가능한 전용 방에 배치된다. 진행 보스보다 전투 시간이 짧지만 난이도가 높고, 고유한 고레어리티 보상을 제공한다.

| 진입 조건 | 비경로 탐험 또는 특정 능력(능력 게이트 통과 후 해당 능력 활용) |
| 보상 | 진행 보스 드랍보다 고레어리티 보장 + 고유 이노센트 드랍 |

### 4-3. 물의 기억 — 간략 설계

진입 조건이 수중 호흡 획득 후이므로, 전투 아레나가 전부 수중이다. 수중에서 에르다의 이동 방식이 완전히 달라지는 환경을 활용한다.

- 유형: 수중 전투 특화 대형 고정 보스
- 핵심 기믹: 수중 전류를 이용한 이동 패턴. 전류 방향을 읽어야 보스에 접근 가능
- 보상: 물 속성 이노센트 (아이템 해당 속성 강화)

### 4-4. 별의 거인 — 간략 설계

역중력 렐릭 획득 이후에만 접근 가능한 층위에 존재. 이 보스는 역중력 메커닉을 최대로 활용하는 전투를 요구한다.

- 유형: 역중력 전투 특화 보스. 보스 자체도 역중력으로 이동
- 핵심 기믹: 역중력 구간에서 에르다의 상하 방향이 역전된 상태로 전투
- 보상: Legendary 등급 보장. 심연 전쟁 이전 별의 기억을 담은 서사 단편

| 카엘 오르스 | 이해 → 선택 → 성장 | Act 3-엔딩 (고쳐지지 않아도 실재했다) | 자동 (최종 대면에서 에코 공명) |

## 8. 검증 기준 (Acceptance Criteria)

### 8-1. 기능 검증

- [ ] 모든 보스의 모든 공격에 `Tell_Duration_Min` (300ms) 이상의 예고 동작이 있는가
- [ ] 모든 진행 보스의 페이즈 전환 시 HP 회복이 발생하지 않는가
- [ ] 보스 방 진입 전 30초 이내 체크포인트가 배치되어 있는가
- [ ] 첫 처치 보상(능력 렐릭)이 재도전 시 재지급되지 않는가
- [ ] 카엘 오르스 제3의 길 선택지가 세 조건 미충족 시 비활성화 상태인가
- [ ] 숨겨진 보스 진입 경로가 해당 능력 없이는 물리적으로 통과 불가한가

### 8-2. 경험 검증 (플레이테스트 기준)

- [ ] 플레이테스터가 보스를 처음 만났을 때 패턴을 이해하지 못해도 "불공정하다"는 반응 없이 "다시 해볼게"라는 반응을 보이는가
- [ ] 3회 도전 이내에 플레이테스터가 각 패턴의 Tell을 인식하는가
- [ ] 10회 이내에 진행 보스 클리어 달성이 가능한가 (기본 스탯 기준)
- [ ] 보스 처치 후 에르다의 독백이 보스의 정체를 별도 설명 없이 이해하게 돕는가
- [ ] 카엘 오르스 최종 대면에서 플레이테스터가 선택지 앞에서 "어느 쪽을 선택해야 하나"를 고민하는가 (정답 없음)
- [ ] 수로의 파수꾼 기믹(수문 레버)을 플레이테스터가 5회 이내 자연스럽게 발견하는가
