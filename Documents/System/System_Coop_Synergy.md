## 0. 설계 원칙

### 0.1. 핵심 철학

> "같이 가면 더 재밌고, 더 효율적이지만, 혼자서도 전부 할 수 있다."

솔로 플레이어가 접근 불가능한 콘텐츠는 존재하지 않는다. 협동은 **효율과 쾌감의 보너스**이지 필수가 아니다.

### 0.2. 핵심 결정

| 결정 | 근거 |
|:---|:---|
| Phase 3에서 **2인**만 지원 | 2인 시너지가 검증되면 3-4인은 인원 추가만으로 확장 가능 |
| **클래스 없음** — 무기로 역할 분화 | Terraria/Monster Hunter 모델. 장비 기반 역할은 야리코미와 정합 |
| **아군 피격 OFF** | 원거리 무기 사용 제한 방지. Castle Crashers/Rayman 검증 |
| **개인 인스턴스 루트** | 전리품 분쟁 방지. 디스가이아 개인 드롭과 정합 |
| URL 링크 합류 | 로비/매치메이킹 없이 즉시 합류. 접근성 최대화 |

## 1. 시너지 3계층

복잡도 순. 아래로 갈수록 플레이어의 의도적 행동이 필요.

### Tier 1: 자동 시너지 (특별한 조작 불필요)

파티를 구성하는 것만으로 발동. "같이 있으면 자동으로 좋다."

#### 1.1. 콤보 가속

| 항목 | 값 |
|:---|:---|
| 트리거 | 같은 적을 2명이 교차 타격 |
| 효과 | 콤보 배율 +15% (솔로 +10%) |
| 윈도우 | 1초 내 교차 히트 |

솔로에서도 콤보 배율은 존재하지만 +10%씩. 파티원과 교차하면 **+15%로 가속**. "이 적을 같이 때리면 더 세다."

#### 1.2. 공명 보너스 (무기 다양성)

| 항목 | 값 |
|:---|:---|
| 트리거 | 파티원의 무기 카테고리가 전부 다름 |
| 효과 | 전원 ATK +15% |
| 판정 | 아이템계 진입 시 자동. 무기 교체 시 재판정 |

같은 무기끼리 가면 보너스 없음. 다른 무기면 보너스. "너 활 가져와, 나 검 가져갈게."

#### 1.3. 이노센트 공명

| 항목 | 값 |
|:---|:---|
| 트리거 | 파티원의 이노센트 타입이 겹침 |
| 효과 | 겹치는 이노센트 효과 +20% |
| 판정 | 아이템계 진입 시 자동 |

같은 이노센트를 양쪽 다 장착하면 공명. "나도 Wind Memory 있어? 공명이다." → 이노센트 파밍 동기 강화.

---

### Tier 2: 근접 시너지 (가까이 있으면 발동)

같이 싸우는 것을 장려. "흩어지지 말고 붙어서 싸워라."

#### 2.1. 팀 어택

| 항목 | 값 |
|:---|:---|
| 트리거 | 파티원이 32px 내에 있을 때 공격 |
| 효과 | 인접 파티원이 **자동으로 추가 타격** (본인 ATK의 75%) |
| 쿨다운 | 0.5초 |
| 시각 | 추가 타격에 에코 잔상 이펙트 |

디스가이아 팀 어택의 실시간 변환. 행동권 소모 없이 "가까이 있으면 자동 합류." 32px = 약 2타일. 바로 옆에 서 있을 때.

#### 2.2. 리벤지 게이지

| 항목 | 값 |
|:---|:---|
| 게이지 | 0-100%. HUD에 작은 바로 표시 |
| 충전 | 파티원 피격 시 전원 50% 충전. 파티원 사망 시 100% (즉시 만충) |
| 발동 | 게이지 100% → 자동 발동 5초간 |
| 효과 | ATK +30%, 크리티컬 확률 +50%, 공격속도 +20% |
| 소모 | 5초 후 게이지 0%로 리셋 |

"동료가 맞으면 분노한다." 파티원 사망 시 **즉시 리벤지 모드** → 복수의 5초. 드라마틱한 순간 연출.

---

### Tier 3: 의도적 시너지 (특정 행동/조합 필요)

의식적으로 조합해야 발동. "이 원소 조합 쓰려고 같이 들어간다."

#### 3.1. 원소 퓨전

서로 다른 원소 공격이 같은 적에게 적용되면 퓨전 효과 발생.

| 원소 조합 | 퓨전 효과 | 데미지 |
|:---|:---|:---|
| **화 + 빙** | Steam Blast (증기 폭발) — 범위 데미지 | 양측 ATK 합산의 150% |
| **화 + 뇌** | Plasma Surge (플라즈마) — 단일 고데미지 + 스턴 1초 | 양측 ATK 합산의 200% |
| **빙 + 뇌** | Cryo Shock (극저온 충격) — 3초 슬로우 80% | 양측 ATK 합산의 120% + 강력 슬로우 |

**발동 조건:**
- 적에게 원소 A 상태이상이 걸린 상태에서 원소 B 공격 적중
- 퓨전 발동 시 양쪽 상태이상 소모 (리셋)
- 퓨전 쿨다운: 적 1체당 3초

**솔로에서도 가능?:** 가능하지만 어려움. 원소 전환(인챈트 변경)에 시간이 걸리므로, 2인이 각각 다른 원소를 쓰는 것이 압도적으로 편함. **파티의 가치를 퓨전 효율로 증명.**

## 2. 적 스케일링

| 인원 | 적 HP 배율 | 적 ATK 배율 | 엘리트 빈도 |
|:---|:---:|:---:|:---|
| 1인 (솔로) | x1.0 | x1.0 | 기본 |
| 2인 (Phase 3) | x1.75 | x1.0 | +10% |
| 3-4인 (Phase 4+) | x2.5-3.25 | x1.1-1.2 | +20-30% |

**설계 의도:**
- HP만 올리고 ATK는 유지 → 죽는 속도는 같지만 죽이는 데 더 오래 걸림
- 시너지(콤보 가속, 팀 어택, 원소 퓨전)를 활용하면 솔로보다 효율적
- 시너지 미활용 시 솔로보다 비효율 → "시너지 쓰는 법을 배워야 한다"

## 3. 저주 부활 시스템

### 3.1. 흐름

```
파티원 사망
    ↓
생존자 전원에게 "부식 저주(Corrosion Curse)" 부여
    │  (매 3초 HP 1% 감소)
    ↓
해제 방법 (택1):
    A. 적 5마리 처치 → 저주 해제 + 사망자 부활
    B. 사망 위치에서 3초 채널링 → 부활 (부활자 무방비)
    ↓
부활 시 HP 30%로 복귀. Flask 미소모.
전원 사망 = 아이템계 강제 퇴장 (진행도 손실)
```

### 3.2. 설계 근거

| 요소 | 이유 |
|:---|:---|
| 저주 = 시간 압박 | "빨리 살려야 한다" 긴장감. 무한 대기 방지 |
| 적 5마리 처치 | 전투 계속 유도. "싸우면서 살린다" |
| 3초 채널링 대안 | 적이 없는 구간/보스 처치 후에도 부활 가능 |
| HP 30% 복귀 | 부활 후 취약. Flask 판단 필요 |
| 전원 사망 = 퇴장 | 무한 부활 방지. 진짜 위험이 있어야 긴장감 유지 |

## 7. 간편 커뮤니케이션

텍스트 채팅 없이 빠른 소통. CV:HoD 퀵 메시지 모델.

**사전 정의 메시지 8종:**

| # | 메시지 | 용도 |
|:---|:---|:---|
| 1 | "OK" | 동의/준비 완료 |
| 2 | "Thanks" | 감사 |
| 3 | "Help" | 위기/지원 요청 |
| 4 | "Here" | 이쪽으로 와 |
| 5 | "Go" | 출발/전진 |
| 6 | "Wait" | 대기 |
| 7 | "Nice" | 칭찬 |
| 8 | "Back" | 후퇴/돌아가 |

**입력:** 방향키 + 전용 키 조합 (미정)
**표시:** 캐릭터 머리 위 말풍선, 2초 표시 후 fade
