## 1. 개요 (Concept)

### 1.1. 설계 의도 (Intent)

> **"재귀적 진입은 기억 속의 기억으로 들어가는 것이다. 에르다는 아이템 안에서 또 다른 아이템을 발견하고, 그 아이템 속으로 다시 뛰어내린다. 더 깊이 갈수록, 기억은 더 원초적이고 더 추상적이며, 돌아오는 길은 더 멀어진다."**

본 문서(SYS-IW-04)는 아이템계 내부에서 획득한 장비 아이템의 아이템계에 즉시 중첩 진입하는 **재귀적 진입 메커닉**의 전체 규칙을 정의한다. 이 메커닉은 디스가이아의 순차적 재귀(나와서 다시 들어감)를 넘어서는 **동시적 중첩 구조**를 세계 최초로 구현하며, 야리코미 공간을 수직적으로 확장하는 핵심 차별 장치다.

리서치(`ItemWorld_RecursiveEntry_Research.md`)에서 확인된 바와 같이, **아이템계 안에서 다른 아이템의 아이템계에 즉시 중첩 진입하는 기능을 구현한 상용 게임은 존재하지 않는다.** Project Abyss의 재귀적 진입은 장르 최초의 시도이며, 이 설계가 작동하기 위해서는 명확한 스폰 조건, 깊이별 스케일링, 상태 동결 규칙, UX 가이드가 모두 정밀하게 정의되어야 한다.

### 1.2. 설계 근거 (Reasoning)

| 설계 결정 | 채택 이유 | 기각된 대안 |
| :--- | :--- | :--- |
| **즉시 중첩 진입 (동시적 재귀)** | 야리코미 공간의 입체적 확장. "내 안에 내가 있다"는 마트료시카 메타포가 아이템의 기억 내러티브와 완벽 정합. 디스가이아의 순차 재귀보다 몰입도 높음 | 순차 재귀(탈출 후 재진입) — 상위 상태 유지 불가, 연속성 단절로 긴장감 소멸 |
| **최대 깊이 3 하드캡** | 시드 충돌 방지(SYS-IW-02 IWF-11-A), 메모리 과부하 방지, 플레이어가 이해 가능한 중첩 한계. 깊이 3만으로도 45분+ 세션 달성 가능 | 무제한 재귀 — 기술적 시드 충돌, UX 위치 혼란, 메모리 폭발 |
| **Magic+ 아이템만 재귀의 문 스폰** | Normal 아이템으로 안전한 저위험 재귀 파밍 방지. 레어리티 상승 동기 강화. "더 좋은 아이템 속에 더 깊은 기억이 있다" 내러티브 일관성 | 모든 레어리티 허용 — 저위험 반복 파밍으로 보상 체계 붕괴 |
| **상위 깊이 완전 동결 (시간 정지)** | "내가 떠난 기억은 멈춰 있다"는 내러티브 정합성. 적 리스폰 없이 복귀 시 안전 보장. UX 복잡도 최소화 | 상위 깊이 실시간 진행 — 멀티에서 동기화 불가, 복귀 시 상황 예측 불가, 플레이어 불안 증폭 |
| **순차 복귀 UX (3→2→1→월드)** | 인셉션의 "킥" 구조를 게임적으로 구현. 각 깊이에서 한 번씩 "깨어나는" 감각이 귀환의 감동을 만든다. 재귀 깊이만큼의 보상 확정 순간 제공 | 즉시 전체 탈출 — 중간 깊이 보상 확정 과정의 생략으로 성취감 소멸 |
| **깊이별 시각/오디오 변형** | Psychonauts, Superliminal, 인셉션의 공통 교훈: 각 재귀 깊이의 환경이 극적으로 달라야 플레이어가 "다른 곳에 있다"를 체감 | 동일 환경 유지 — 깊이감 소멸, 재귀 진입의 의미 약화 |

### 1.3. 3대 기둥 정렬 (Pillar Alignment)

| 기둥 | 재귀 시스템에서의 구현 |
| :--- | :--- |
| **메트로베니아 탐험** | 재귀의 문은 지층 탐험 중 발견하는 숨겨진 가능성이다. "이 방 저 구석에 재귀의 문이 있을지도"라는 탐색 동기가 지층 내 비선형 탐험을 강화한다 |
| **아이템계 야리코미** | 재귀는 야리코미의 수직 확장이다. 깊이 1의 기본 강화 루프 위에 깊이 2와 3이 중첩되어 파밍 공간이 입체화된다. 깊이 2/3 전용 보상은 야리코미의 다음 목표가 된다 |
| **온라인 멀티플레이** | 깊이 3은 솔로로도 가능하나 파티가 현실적 우위를 가진다. 재귀 진입 투표 메커닉이 파티 내 협의와 사회적 계약을 형성하며, 함께 깊이 3에서 복귀하는 경험은 강한 연대감을 만든다 |

### 1.4. 저주받은 문제 검증 (Cursed Problem Check)

| 긴장 | 위험 A | 위험 B | 설계의 선택 |
| :--- | :--- | :--- | :--- |
| **야리코미 심화 vs 세션 길이** | 재귀가 세션을 무한정 연장 → 피로 유발 | 재귀 보상이 미약 → 진입 동기 소멸 | 깊이별 세션 추가 시간 10~15분. 깊이 3까지 총 30~45분 추가. 각 깊이 탈출 가능으로 중단점 보장 |
| **위치 추적 vs 몰입감** | 재귀 위치가 명확하면 불안감 소멸 | 위치 불명확하면 Superliminal식 혼란 | HUD로 정확한 위치 표시 + 환경 왜곡으로 "깊이의 불안감" 동시 제공. 정보는 주되 감각적 압박은 유지 |
| **솔로 vs 파티 재귀** | 솔로 재귀 불가 → 진입 장벽 | 솔로가 더 효율 → 멀티 동기 소멸 | 솔로 재귀 전면 허용. 파티는 깊이 3 생존 마진 우위. 재귀 진입 투표는 강제 아닌 협의 |
| **보상 독점성 vs 접근성** | 재귀 보상이 너무 강 → 재귀 의무화, 그라인딩으로 전락 | 재귀 보상이 미약 → "선택"의 가치 소멸 | 기본 강화는 재귀 없이 완성 가능. 재귀 전용 보상은 "더 강한 최적화"를 위한 것. Slay the Spire Elite 선택 구조 참조 |

### 1.5. 설계 의도 — 분해·분석·재구축 (Deconstruct–Analyze–Rebuild)

#### 분해 (Deconstruct)

| 참고 게임 | 재미 요소 |
| :--- | :--- |
| 디스가이아 시리즈 | 아이템계 → 탈출 → 획득한 아이템의 아이템계 재진입이라는 순차적 체인. "더 강한 아이템을 먹으러 다시 다이빙"이 야리코미의 핵심 루프를 형성 |
| 인셉션 (영화) | 5개 꿈 층 구조. 더 깊은 층에서의 실패가 더 큰 대가. 층마다 시간 감각이 달라짐. "킥"으로 깨어나는 순차 복귀 |
| Noita | 세계 경계를 뚫으면 병렬 세계에 진입. 재진입마다 독특한 보상과 대가 |
| Superliminal | 꿈 속의 꿈. 위치 불확실성 자체가 긴장감 원천 |
| Double Fine Stacking | 작은 인형이 큰 인형 안에 들어가는 마트료시카 구조. 능력 중첩 |

#### 분석 (Analyze)

| 재미의 본질 | 리스크와 리턴 분석 |
| :--- | :--- |
| 재귀 자체의 성취감 | "다시 들어간다"는 행위가 장인의 탐구심과 일치. 더 깊이 파고드는 것이 성취의 증거가 된다 |
| 인셉션식 긴장감 | 깊을수록 돌아오는 길이 멀어진다. 탈출이 단순하지 않다는 인식이 현재 수집물의 심리적 가치를 2~2.5배 증폭한다 (전망 이론 λ ≈ 2.25) |
| 위치 추적 불안과 HUD 안전감의 공존 | Superliminal의 위치 불명확성은 패닉을 만들지만, HUD로 정확한 위치를 제공하면서 환경 왜곡으로 압박을 주면 "알지만 불안한" 더 성숙한 긴장을 만든다 |
| 순차 복귀의 감동 | 인셉션 "킥" 구조에서 검증됨. 각 층에서 한 번씩 깨어나는 경험이 보상 확정의 의식을 만들고, 귀환 자체가 클라이맥스가 된다 |

#### 재구축 (Rebuild)

| 원형의 한계 | Project Abyss의 독창적 변형 |
| :--- | :--- |
| 디스가이아: 탈출 후 순차 진입 — 상위 상태가 보존되지 않아 재귀의 연속성이 단절됨 | 즉시 중첩 진입: 상위 아이템계를 동결하고 그 안에서 다시 뛰어내린다. 상위 세계가 "기다리는" 느낌이 마트료시카 내러티브와 완벽 정합 |
| 인셉션: 일방향적 구조 — 플레이어가 선택이 아닌 서사에 끌려감 | 자발적 선택 재귀: 재귀의 문(Magic+ 드랍)을 발견했을 때 진입 여부를 플레이어가 결정. "할 수 있지만 해야 할 필요는 없다"가 Slay the Spire Elite 선택 구조와 동일 |
| Superliminal: 위치 파악 불가가 공황 유발 | HUD 마트료시카 아이콘 + 브레드크럼으로 정확한 위치 제공. 단, 환경 채도/색수차로 감각적 압박을 별도 채널로 전달 |

> **독창성 요약:** 세계 최초의 "동시적 중첩 재귀 아이템계" 구조. 디스가이아(순차), Noita(병렬), Psychonauts(허브-스포크), FFXIV(단일 수직)와 구별되는 **스택형 수직 재귀**로, 야리코미 공간을 입체화하는 핵심 차별 장치다.

| 깊이 2 진입 | 난이도 대폭 증가(적 HP ×1.8, ATK ×1.4, 보스 등급 +1), 탈출 시 순차 복귀 필요, 상위 상태 동결 | 재귀 이노센트 Tier B(+30~50%), 기억의 잔광 소재, 재방문보다 높은 보상률 |
| 깊이 3 진입 | 최고 난이도(적 HP ×3.24, ATK ×1.96, 보스 등급 +2), 삼중 복귀 시간 투자, 색수차+비네팅 환경 압박 | 재귀 이노센트 Tier A(+60~100%), 유니크 이노센트, 기억의 끝 내러티브 단편 |
| 재귀 중 탈출 포기 (전멸) | 현재 깊이 획득 이노센트 손실, 시간 투자 소멸 | 상위 깊이 보존 (장비 아이템+영구 보너스 보존). 다음 시도를 위한 정보 |
| 재귀 파티 투표 거부 후 단독 진입 | 파티 분리로 잔류 파티 난이도 재조정, 재합류까지 협동 이득 소멸 | 개인 재귀 보상 독점. 솔로 파밍 리듬 유지 |

## 2. 핵심 규칙 (Core Rules)

### 2.1. 재귀의 문 (Gate of Recursion) — 스폰 조건

> **핵심 원칙:** "재귀의 문은 아이템계 지층 내에서 또 다른 기억으로 들어가는 균열이다. 아이템계 안에서 획득한 장비 아이템을 에코로 두드리면 그 자리에 새로운 바닥이 열린다."

재귀의 문은 별도의 오브젝트가 아니다. 아이템계 내부에서 획득한 **장비 아이템을 에코로 두드리는 행위** 자체가 재귀 진입 트리거다. 아이템계 탈출 제단 위에서, 또는 지층 내 임의 위치에서 인벤토리의 장비 아이템을 선택해 에코로 두드리면 즉시 진입 시퀀스가 시작된다.

#### 재귀 진입 가능 조건

| 조건 항목 | 세부 규칙 |
| :--- | :--- |
| **대상 아이템 레어리티** | Magic 이상 필수. Normal 아이템은 재귀 진입 불가 |
| **현재 재귀 깊이** | 0~2에서만 진입 가능. 깊이 3에서는 재귀 불가 (하드캡) |
| **대상 아이템 상태** | 아이템계 내부에서 드랍 또는 인벤토리에 소지 중인 장비 아이템. 현재 진입 중인 아이템은 재귀 대상 불가 |
| **멀티플레이 조건** | 파티 진행 중인 경우 전원 동의 필요. [섹션 2.5] 참조 |
| **진입 행위** | 해당 장비 아이템을 인벤토리에서 선택 → 에코로 두드림. SYS-IW-01의 Echo Strike와 동일한 행위 |

#### 재귀의 문 드랍 확률 테이블

아이템계 내부에서 획득하는 장비 아이템이 재귀 진입 가능 조건(Magic+)을 만족하는 확률은 드랍 테이블에 의해 결정된다. 별도의 "재귀의 문" 오브젝트가 스폰되는 것이 아니라, 드랍 아이템의 레어리티가 재귀 가능 여부를 자동 결정한다.

| 현재 지층 깊이 | Normal 확률 | Magic+ 확률 (재귀 가능) |
| :---: | :---: | :---: |
| 깊이 1 (기본 아이템계) | 70% | 30% |
| 깊이 2 | 50% | 50% |
| 깊이 3 (최대) | - | 진입 불가 (드랍 아이템 무관) |

> 수치 데이터: `Sheets/Content_ItemWorld_RecursionDrop.csv` 참조 (Phase 2 제작)

추가적으로, 지층 내 보스를 처치하면 보상 아이템의 Magic+ 드랍률이 상향된다.

| 레어리티 | Magic+ 드랍 가중치 증가량 |
| :--- | :--- |
| 현재 진입 아이템이 Magic | +5% |
| 현재 진입 아이템이 Rare | +10% |
| 현재 진입 아이템이 Legendary | +15% |
| 현재 진입 아이템이 Ancient | +20% |

### 2.2. 진입 연출 — 재귀 깊이별 바닥 붕괴 변형

SYS-IW-01에 정의된 기억의 두드림(Echo Strike) 연출을 재귀 깊이에 따라 다음과 같이 변형 적용한다.

| 재귀 깊이 | 연출 변형 | 내러티브 의미 |
| :--- | :--- | :--- |
| **깊이 1 (기본)** | SYS-IW-01 기본 시퀀스 그대로 적용 | 기억 속으로의 첫 번째 다이빙 |
| **깊이 2** | 이중 바닥 붕괴 — 아래로 떨어지는 순간, 위쪽 깊이 1의 바닥 타일도 동시에 무너지며 이중 파편 이펙트. 팔레트 채도 +20%. 잔상 이펙트(에코의 잔상이 위에 0.3초 남음) | "기억 속의 기억. 더 깊은 곳에 원래 기억이 있다" |
| **깊이 3** | 삼중 붕괴 — 깊이 1, 2, 3의 바닥이 순차적으로 무너지는 세 겹의 파편 이펙트. 낙하 중 색수차(chromatic aberration) 적용. 슬로우모션(0.7×, 0.5초) + 화면 가장자리 보라색 비네팅 | "This is the end of memory. Going deeper may mean no return." (에르다 독백) |

**반복 진입 단축 규칙:** 재귀 진입에도 SYS-IW-01의 반복 진입 단축 규칙이 동일하게 적용된다. 단, 깊이 3은 항상 최소 0.8초 이상의 연출을 유지한다 (경고 연출이 생략되면 안 된다).

### 2.3. 상위 깊이 동결 규칙 (Upper Depth State Freeze)

재귀 진입 시 상위 깊이의 아이템계는 **완전 동결** 상태로 전환된다. 내러티브적으로 "에르다가 떠난 기억은 멈춰 있다."

#### 동결 시 처리 항목

| 항목 | 동결 처리 내용 |
| :--- | :--- |
| **적 상태** | 모든 적의 위치, HP, AI 상태를 그대로 보존. 에르다 부재 중 이동, 공격, 리스폰 없음 |
| **타이머** | 상위 깊이 내에 기억의 침식(시간 경과 피해) 등 시간 기반 메커닉이 있을 경우 완전 정지 |
| **인벤토리/드랍** | 동결 전에 지층 바닥에 있던 드랍 아이템은 위치 그대로 보존 |
| **탈출 제단** | 탈출 제단의 발견 여부 상태 그대로 유지. 미발견 상태가 동결된 경우 복귀 후에도 미발견 |
| **방문 기록** | `ItemWorldProgress.visitedRooms` 상태 보존. 복귀 후 미방문 방 그대로 유지 |
| **보스 상태** | 보스를 처치하지 않고 재귀 진입한 경우, 복귀 시 보스가 그 자리에 그대로 존재 |

> 상태 저장 구현: `RecursionStateStack` 참조. [섹션 3.1] 데이터 구조 정의

#### 동결 시간 상한

단일 재귀 진입의 최대 체류 시간 제한은 없다. 플레이어가 깊이 2에서 원하는 만큼 탐험 후 자발적으로 복귀한다. 단, 멀티플레이에서 상위 깊이의 파티원이 무한 대기하는 상황을 방지하기 위해 파티 분리 규칙이 적용된다. [섹션 2.5] 참조.

### 2.4. 탈출 및 순차 복귀 (Exit and Sequential Return)

재귀 진입 중 탈출은 **현재 깊이에서 한 단계씩 순차적으로 복귀**한다. 즉시 전체 탈출은 없다.

#### 탈출 방법

| 방법 | 사용 조건 | 도달 위치 |
| :--- | :--- | :--- |
| **탈출 제단 사용** | 현재 깊이에서 탈출 제단 발견 시 | 한 단계 상위 깊이로 복귀. 연출: 잔상 효과와 함께 상승 |
| **보스 처치 후 탈출 선택** | 현재 깊이 보스를 처치한 후 30초 내 탈출/계속 선택 화면 | 한 단계 상위 깊이로 복귀 |
| **전멸** | HP 0 | 현재 깊이에서 획득한 이노센트 손실. 한 단계 상위 깊이로 강제 복귀 (세이프티 메커닉) |

> 탈출 시 현재 깊이에서 획득한 아이템(이노센트 제외)은 인벤토리에 보존된다. 전멸 시에는 현재 깊이에서 획득한 이노센트만 손실이며, 장비 아이템은 보존된다.

#### 순차 복귀 연출 시퀀스

각 단계 복귀 연출은 0.5초씩 진행된다. 총 복귀 시간은 깊이에 비례한다.

```
[깊이 3에서 탈출 시]

① 깊이 3 탈출 트리거
   → 화면 전체가 0.3초간 흰색 플래시
   → 에르다 위로 떠오르는 애니메이션 + 색수차 해제
   → 복귀 중 깊이 3에서 확정된 보상이 HUD에 표시 (이노센트, 드랍 아이템)
   → [0.5초 후] 깊이 2 아이템계 재진입 지점에 착지

② 깊이 2 복귀
   → 채도 +20% 환경에서 에르다 착지 연출
   → 동결된 적들이 다시 움직이기 시작 (AI 재개 0.3초 딜레이)
   → 이 지점에서 계속 깊이 2 탐험 가능 (탈출 안 해도 됨)
   → 탈출 선택 시 다시 상위로 복귀

③ 깊이 2 탈출 시 깊이 1 복귀
   → 채도 정상화 + 기본 팔레트로 전환
   → 동결된 적들 재개 (AI 재개 0.3초 딜레이)

④ 깊이 1 탈출 시 원래 아이템계로 복귀
   → 탈출 제단 또는 보스 처치 후 허브/월드로 귀환
```

스킵 설정: 설정 메뉴에서 복귀 연출을 0.1초 페이드 전환으로 단축 가능. 단, 보상 확정 HUD 표시는 생략되지 않는다.

### 2.5. 멀티플레이 재귀 진입 규칙

파티 진행 중 특정 플레이어가 재귀 진입을 시도할 경우 다음 규칙이 적용된다.

#### 재귀 진입 투표 (Recursion Vote)

| 단계 | 규칙 |
| :--- | :--- |
| **1. 투표 시작** | 재귀 진입 시도 플레이어가 파티 UI에 "재귀 진입 요청" 발송. 모든 파티원에게 팝업 알림 표시 |
| **2. 동의 기간** | 30초 내에 전원이 수락해야 파티 전체 진입 가능 |
| **3. 전원 수락** | 파티 전체가 함께 재귀 진입. 현재 깊이 상태 동결 |
| **4. 1명 이상 거부** | 파티 분리 발생 ([섹션 2.5 파티 분리] 참조) |
| **5. 시간 초과** | 미응답자는 거부로 처리. 파티 분리 발생 |

#### 파티 분리 (Party Split)

| 항목 | 규칙 |
| :--- | :--- |
| **분리 조건** | 재귀 진입 투표에서 과반 미달 또는 1명 이상 거부 |
| **진입 플레이어** | 재귀 진입 요청자 + 수락한 플레이어만 재귀 진입 |
| **잔류 플레이어** | 현재 깊이에서 계속 진행. 재귀 진입한 플레이어는 잠시 현재 깊이에서 사라짐 |
| **잔류 플레이어의 난이도** | 적 HP 스케일링이 잔류 인원 수에 맞게 즉시 재조정 (`InCombat` 상태 외에서만 적용) |
| **재합류 조건** | 재귀 진입 플레이어가 현재 깊이로 복귀(탈출)하면 자동 재합류. 파티 복귀 연출 재생 |
| **잔류 파티의 전멸** | 잔류 파티 전멸 시 잔류 플레이어들만 사망 패널티 적용. 재귀 진입 파티는 영향 없음 |

> 단, 재귀 진입 플레이어가 보유 중인 파티의 유일한 헤일러/탱커인 경우, 잔류 파티는 난이도 재조정으로 솔로 모드에 준한 난이도가 적용된다.

## 3. 수치 설계 (Data & Parameters)

### 3.1. 데이터 구조 — RecursionStateStack

기존 `ItemWorldProgress`에 재귀 스택을 추가하여 상위 깊이 상태를 보존한다.

```typescript
// 단일 깊이 상태 스냅샷
interface RecursionLayerState {
  itemId:           string          // 이 깊이를 진행 중인 아이템 ID
  itemRarity:       Rarity          // 아이템 레어리티
  currentStratum:   number          // 현재 지층 번호 (1-based)
  clearedStrata:    number[]        // 보스 처치 클리어된 지층 목록
  visitedRooms:     string[]        // 방문한 방 ID 목록
  seed:             number          // 현재 지층 생성 시드
  escapeAltarFound: boolean         // 탈출 제단 발견 여부
  enemyStates:      EnemySnapshot[] // 동결된 적 상태 배열
  droppedItems:     ItemSnapshot[]  // 동결된 바닥 드랍 아이템
  frozenAt:         number          // 동결 시각 (timestamp)
}

// 재귀 스택 확장 ItemWorldProgress
interface ItemWorldProgress {
  // ... 기존 SYS-IW-01 필드 유지 ...
  recursionDepth:    number                  // 현재 재귀 깊이 (0=기본, 최대 3)
  recursionStack:    RecursionLayerState[]   // 동결된 상위 깊이 상태 스택 (최대 2개)
  // recursionStack[0] = 깊이 1 상태 (깊이 2 진입 시 저장)
  // recursionStack[1] = 깊이 2 상태 (깊이 3 진입 시 저장)
}
```

> SSoT: `game/src/data/StrataConfig.ts` 확장 예정

### 3.2. 깊이별 적 스케일링 공식

재귀 깊이에 따른 적 HP와 ATK 배율은 기본 아이템계(깊이 1)를 기준 ×1.0으로 설정하고 아래 계층식 배율을 적용한다.

리서치(`ItemWorld_DepthReward_RiskBalance_Research.md`)의 구간별 복합 스케일링 모델을 따른다. HP는 지수 증가로 야리코미의 체감 성장을 표현하고, ATK는 비대칭 증가(HP보다 낮은 배율)로 즉사 방지를 보장한다.

```yaml
# Sheets/Content_ItemWorld_DepthScaleConfig.csv 참조 (Phase 2)
depth_scaling:
  depth_1:
    enemy_hp_multiplier: 1.0       # 기준값
    enemy_atk_multiplier: 1.0      # 기준값
    boss_tier_delta: 0             # 보스 등급 변화 없음
    comment: "기본 아이템계. SYS-IW-03 등급 그대로 적용"

  depth_2:
    enemy_hp_multiplier: 1.8       # 기준 대비 ×1.8
    enemy_atk_multiplier: 1.4      # 기준 대비 ×1.4 (비대칭 — 즉사 방지)
    boss_tier_delta: +1            # 보스 등급 1단계 상향 (장군→왕, 왕→신, 신→대신)
    comment: "DiabloIII HP ×1.17/GR에서 영감. 단 2계층이므로 1.8로 압축"

  depth_3:
    enemy_hp_multiplier: 3.24      # 깊이 2 대비 ×1.8 = 기준 대비 ×3.24
    enemy_atk_multiplier: 1.96     # 깊이 2 대비 ×1.4 = 기준 대비 ×1.96
    boss_tier_delta: +2            # 보스 등급 2단계 상향 (장군→신, 왕→대신, 신=대신 고정)
```

### 3.4. 재방문 보상 감쇠 (Revisit Diminishing Returns)

동일 아이템의 아이템계를 반복 방문할 경우, 재귀 진입 보상도 동일한 감쇠 규칙을 따른다. 이는 특정 아이템을 무한 반복 파밍하는 익스플로잇을 방지하고 파밍 대상의 자연 교체를 유도한다.

```
재귀_보상_배율(v) = 0.5^(v-1)

v = 해당 아이템계 재귀 진입 횟수 (1부터 시작)

v=1: 100% (최초)
v=2: 50%
v=3: 25%
v=4: 12.5%
v=5+: ~6% (수렴)
```

> 감쇠는 이노센트 드랍 확률과 소재 드랍 확률에 적용된다. 적 HP/ATK 스케일링과 보스 등급에는 감쇠 적용 없음 (난이도는 항상 동일하게 유지).

## 4. 흐름 설계 (Flow Design)

### 4.1. 재귀 진입 전체 플로우

```
[깊이 1 아이템계 탐험 중]
  ↓
[지층 내 장비 드랍 획득]
  ↓
[드랍 아이템 레어리티 판정]
  ├── Normal → 재귀 불가. 일반 아이템으로 소지
  └── Magic+ → 재귀 진입 가능 (플레이어 선택)
       ↓
[플레이어가 인벤토리에서 해당 아이템 선택 → Echo Strike]
  ↓
[멀티 여부 판단]
  ├── 솔로 → 즉시 재귀 진입 연출 시작
  └── 파티 → 재귀 진입 투표 팝업 (30초)
       ├── 전원 수락 → 파티 전체 재귀 진입
       └── 1인 이상 거부 → 파티 분리, 수락자만 진입
          ↓
[현재 깊이 1 상태 → RecursionStateStack에 저장 (동결)]
  ↓
[깊이 2 진입 연출: 이중 바닥 붕괴 + 채도 +20% + 잔상]
  ↓
[깊이 2 아이템계 탐험 (깊이 2 전용 드랍 테이블 적용)]
  ↓
[탈출 결정 (제단/보스 처치 후)]
  ↓
[순차 복귀 연출: 깊이 2 → 깊이 1 (0.5초)]
  ↓
[깊이 1 동결 해제 (적 AI 재개, 드랍 아이템 복원)]
  ↓
[깊이 1 계속 탐험 또는 최종 탈출]
```

## 6. 예외 처리 (Edge Cases)

### 6.1. 깊이 3에서 재귀 시도

- **처리:** 인벤토리에서 장비 아이템을 선택해 에코를 사용하려 해도 Echo Strike 입력이 무효화된다.
- **플레이어 피드백:** UI에 "End of Memory. Cannot go deeper." 팝업 1.5초 표시. SFX: 망치가 허공을 치는 공허한 음향.

### 6.2. Normal 아이템에 Echo Strike 시도 (아이템계 내부)

- **처리:** 재귀 진입이 아닌 일반 아이템계 진입 시도로 처리된다.
- **처리 결과:** "This item holds no memory." 팝업. 진입 불가. 단, 허브나 필드 제단에서 사용하면 SYS-IW-01 기본 진입 규칙대로 Normal 아이템계 진입은 가능.

### 6.3. 네트워크 단절 (멀티플레이 재귀 중)

- **처리:** 단절된 플레이어는 해당 재귀 깊이에서 빠진 것으로 처리. 잔류 파티의 적 HP 스케일링 즉시 재조정.
- **재접속:** 재접속 시 해당 깊이의 재귀 진입 지점(아이템계 입구)으로 복귀. 상위 깊이 상태는 동결 유지. 획득 아이템(이노센트 제외)은 보존.
- **단절 플레이어의 보상:** 재접속하지 않고 파티가 탈출 완료한 경우, 단절 전에 확정된 보상만 지급.

### 6.4. 재귀 진입 대상 아이템이 현재 진입 중인 아이템과 동일

- **처리:** 진입 불가. "Already inside this memory." 팝업. 자기 자신의 아이템계 안에서 같은 아이템을 또 두드리는 것은 무한 루프이므로 하드 차단.

### 6.5. 재귀 중 상위 깊이 파티원 전멸

- **처리:** 재귀 진입 파티에게 UI 알림 표시. "Party wiped in upper memory." 재귀 진입 파티의 진행에는 영향 없음. 상위 파티원은 해당 깊이에서 전멸 패널티 적용 후 그 상위 깊이로 복귀.

### 6.6. 깊이 3에서 취득한 아이템에 Echo Strike (재귀 시도)

- **처리:** [6.1]과 동일한 하드캡 처리. "End of Memory. Cannot go deeper."

### 6.7. 재귀 스택 손상 (비정상 종료)

- **처리:** 서버 측 `RecursionStateStack` 스냅샷을 5분 간격으로 백업. 비정상 종료 시 최근 스냅샷으로 복원. 복원 불가 시 재귀 스택 강제 초기화 후 깊이 1 아이템계 입구로 귀환. 아이템 EXP와 확정된 영구 보너스는 보존.

## 7. 내러티브 통합 (Narrative Integration)

### 7.1. 재귀 = 기억 속의 기억

재귀적 진입은 단순한 중첩 던전이 아니다. 내러티브적으로, 아이템의 기억 속에 존재하는 또 다른 아이템은 **원래 소유자가 함께 사용했던 다른 무기나 장비의 흔적**이다. 검사의 기억 속에서 발견된 단검은 그 검사가 젊었을 때 쓰던 도구이고, 그 단검의 기억 속에는 검사가 처음 싸우는 법을 배우던 아득한 원초적 순간이 담겨 있다.

깊이가 깊어질수록, 기억은 더 원초적이고 추상적이 된다. 깊이 1은 아이템 소유자의 성숙한 기억(구체적, 장소와 얼굴이 있음), 깊이 2는 소유자의 초기 기억(감정만 남고 세부가 흐릿), 깊이 3은 기억의 원형(추상적 색채와 충동만 남은 원초적 인상)이다.

| 재귀 깊이 | 내러티브 의미 | 에르다의 심리 |
| :--- | :--- | :--- |
| 깊이 1 | 장인으로서의 호기심. "이 무기 안에 뭔가 더 있다." | 탐구심, 전문가적 관심 |
| 깊이 2 | 점점 낯설어지는 기억. "이 공간이 조금... 불편하다." | 경계심, 미묘한 불안 |
| 깊이 3 | 기억의 가장 깊은 곳. "이곳은 기억의 끝이야." | 외경심, 두려움과 경이가 뒤섞임 |

### 7.2. 마르타 연결 서사

깊이 3은 단순한 최고 난이도 구간이 아니다. 스토리 진행에 따라(Act 2 이후), 에르다가 깊이 3에 처음 도달할 때 특수 이벤트가 발생한다.

> **[Act 2 이후 최초 깊이 3 도달 트리거]**
>
> 깊이 3의 특정 방에서 기억의 잔광이 에르다를 향해 다가온다. 그 잔광은 사람 형태를 하고 있다. 에르다가 가까이 가면 잔광이 희미한 목소리를 남긴다:
>
> *"...에르다. 여기까지 왔구나."*
>
> 에르다: "스승님...? 마르타 스승님이에요?"
>
> 잔광은 대답 없이 더 깊은 곳을 가리키고 사라진다.
>
> 에르다 독백: "스승님은 자발적으로 이 안에 남은 거야. 기억을 지키기 위해서. 그렇다면... 기억의 끝에는 무엇이 있는 걸까?"

이 이벤트는 1회만 발생한다. 이후 깊이 3 재방문 시에는 잔광 없이 일반 탐험으로 진행된다. 기억의 끝 단편 수집 요소(IWR-04-B)를 통해 마르타의 자발적 유배 이유가 조각조각 밝혀진다.

### 7.3. 기억의 끝 단편과 Revelation 5

깊이 3 곳곳에 배치된 "기억의 끝 단편" 수집 요소는 게임 전체의 핵심 미스터리와 연결된다.

> **연결 포인트:** Revelation 5 — "세계 자체가 재귀적 구조를 가지고 있다."
>
> 깊이 3에서 수집되는 단편들은 아이템의 기억이 단순히 개인의 역사가 아니라, 세계의 구조 자체와 연결되어 있음을 암시한다. 아이템 안에 기억이 있고, 그 기억 안에 또 기억이 있다는 재귀적 구조는 — 실은 세계 자체의 구조다. 아이템은 세계의 축소판이며, 에르다가 아이템 안에서 발견한 것은 세계의 본질이다.
>
> 마르타가 "기억을 지키기 위해" 아이템 안에 자발적으로 들어간 것은, 세계의 재귀적 구조 안에서 무언가를 막거나 보존하기 위한 행위였을 것이다.

## 8. 수용 기준 (Acceptance Criteria)

### 8.1. 기능적 기준 (Functional Criteria)

- [ ] Normal 아이템에 아이템계 내부에서 Echo Strike 시도 시 재귀 진입이 차단되고 UI 피드백이 표시된다
- [ ] Magic+ 아이템에 Echo Strike 시 `recursionDepth` 가 1 증가하고 상위 상태가 `recursionStack`에 저장된다
- [ ] 깊이 3에서 Echo Strike 시도 시 하드캡이 작동하여 진입이 불가하다
- [ ] 깊이 2/3 진입 시 적 HP가 각각 ×1.8, ×3.24 배율로 적용된다
- [ ] 깊이 2/3 진입 시 보스 등급이 규칙에 따라 상향된다
- [ ] 재귀 탈출 시 순차 복귀(3→2→1)가 순서대로 진행되며 각 단계에서 상위 상태가 복원된다
- [ ] 전멸 시 이노센트는 손실되지만 장비 아이템과 영구 보너스(아이템 EXP)는 보존된다
- [ ] 멀티 재귀 투표 UI가 표시되며 30초 타임아웃이 정상 작동한다
- [ ] 파티 분리 후 잔류 파티의 적 HP 스케일링이 인원 수에 맞게 재조정된다
- [ ] 재방문 감쇠가 `0.5^(v-1)` 공식으로 이노센트/소재 드랍에 적용된다

### 8.2. 시각/연출 기준 (Visual Criteria)

- [ ] 깊이 2 진입 시 채도 +20%와 3프레임 잔상이 적용된다
- [ ] 깊이 3 진입 시 색수차, 보라색 비네팅, 5프레임 잔상이 적용된다
- [ ] 깊이 3 진입 연출 시 에르다 독백 "이곳은 기억의 끝이야."가 재생된다
- [ ] 순차 복귀 연출에서 각 단계별 0.5초의 상승 애니메이션이 재생된다
- [ ] HUD의 마트료시카 아이콘이 현재 재귀 깊이를 정확히 표시한다
- [ ] 브레드크럼 네비게이션에 현재 깊이 경로가 표시된다

### 8.3. 체험적 기준 (Experiential Criteria — 플레이테스트 검증)

- [ ] 플레이테스터가 재귀 진입을 자발적으로 선택한다 (강제됐다고 느끼지 않는다)
- [ ] 깊이 3의 난이도가 "무섭지만 도전하고 싶다"는 반응을 유발한다 (불합리하다고 느끼지 않는다)
- [ ] 순차 복귀 연출이 "한 층씩 깨어나는" 감각을 제공한다
- [ ] 깊이 3 첫 도달 시 마르타 이벤트가 "이 공간이 그냥 던전이 아니었다"는 감정적 전환을 유발한다
- [ ] 재방문 감쇠가 "이 아이템보다 저 아이템을 파야겠다"는 자연스러운 파밍 대상 교체를 유도한다
- [ ] 파티 분리 규칙이 "같이 갈까, 나만 갈까"라는 사회적 협의를 형성한다
