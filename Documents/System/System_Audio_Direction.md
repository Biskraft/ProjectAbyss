## 1. 개요 (Concept)

### 설계 의도

> "ECHORIS의 오디오는 단순한 배경이 아니다. 대사 없는 에르다의 감정, 아이템 속 기억의 무게, 대장간의 유일한 온기를 청각으로 전달하는 서사 도구다."

오디오 시스템이 해결하는 핵심 문제:

1. **타격감 절반의 공백**: `System_Combat_HitFeedback.md`의 사쿠라이 8기법은 시각/운동감각 채널을 구현했으나, CMB-07-R-V(SFX 레이어 4종)는 전부 대기 상태다. 타격감의 30-50%는 사운드에서 온다. 이 공백이 최우선 과제다.
2. **침묵 주인공의 감정 전달**: 에르다는 대사 0줄이다. 그녀의 성격과 감정 상태는 전적으로 이동 SFX, 에코 사운드, 타격음을 통해 표현된다.
3. **스파이크 강화**: "아이템 속으로 들어간다"는 순간은 게임 전체에서 가장 중요한 단일 오디오 이벤트여야 한다.
4. **2-Space 정체성 분리**: 월드(탐험)와 아이템계(야리코미)는 오디오만으로도 서로 다른 공간임을 즉각 인식시켜야 한다.

### 설계 근거

| 결정 | 근거 | 대안 대비 장점 |

## 2. 사운드 팔레트 (Sound Palette)

### 2-1. 월드 음향 정체성

ECHORIS의 월드는 **메가스트럭처 + 빌더의 유산 + 탐험가의 고독**이다. 음향 팔레트:

- **기조**: 산업적 금속음 + 광활한 잔향 + 저음 드론
- **대비**: 세이브 포인트의 화덕 온기 vs 외부 구조물의 냉정한 기계음
- **금지**: 고딕 다크 판타지 음향 (교회 성가, 악마적 코러스). Blasphemous/Moonscars와 차별화

### 2-2. 아이템계 음향 정체성

아이템계는 **기억이 던전이 된 공간**이다. 시각이 팔레트 반전(주황 배경 + 청록 디테일)이듯, 오디오도 반전:

- **기조**: 금속 타격 리듬 루프 + 베이스 드론 (리드미컬, 반복적, 긴장감)
- **테마 레이어**: 각 테마의 기억을 암시하는 포인트 악기 1종 오버레이
- **대비**: 월드의 멜로딕한 BGM vs 아이템계의 리드미컬+타악기 중심

### 2-3. 주요 레퍼런스 트랙 방향

| 공간 | 음악 방향 | 참조 레퍼런스 |
| :--- | :--- | :--- |
| 월드 탐험 | 고딕 산업 앰비언트. 고독하고 광활함. 멜로딕하되 무겁다 | Hollow Knight (Christopher Larkin) — 미니멀 오케스트라 |
| 아이템계 | 금속 긴장 루프. 반복 리듬. 테마별 포인트 악기 | Dead Cells (Yoann Laulan) — 전자음+오케스트라 하이브리드 |
| 세이브 포인트 | 대장간 테마. 따뜻하고 리드미컬. 유일한 "집" | Hades (Darren Korb) — 공통 리듬의 지역 변형 |
| 최심층 (Tier 7) | 앰비언트 텍스처. 구조적 음악의 해체 | Hyper Light Drifter (Disasterpeace) — Music as Texture |

## 3. 히트 SFX 메커닉 (Combat Audio)

### 3-1. 3레이어 히트 구조

모든 근접 히트 SFX는 3개의 독립 레이어가 시간차를 두고 겹치는 구조를 따른다. CMB-07-R/S/T와 직접 대응한다.

| 레이어 | 역할 | 타이밍 | 음역 | CMB 연동 |
| :--- | :--- | :--- | :--- | :--- |
| L1. 슬래시 (Whoosh) | 무기 궤적. 공격 모션 시작과 함께 | 0ms | 중고음 (2-8 kHz) | CMB-07-R |
| L2. 임팩트 (Thud/Crack) | 금속이 살/갑옷에 닿는 순간 | 히트 판정과 동시 | 저중음 (200-1,200 Hz) | CMB-07-S |
| L3. 피격 반응 (Grunt/Echo) | 적의 피격 반응. 임팩트에서 지연 | +30-50ms | 중음 + 소멸 잔향 | CMB-07-T |

**히트 컨펌 원칙:** L2 임팩트음에서 150-400 Hz 저음 성분이 충분히 커야 타격감이 살아난다. 저가 노트북 스피커 환경에서도 이 대역이 체감되도록 설계한다.

**콤보 피치 계단 원칙:** 같은 무기의 콤보는 타격이 누적될수록 임팩트음 피치가 미세하게 상승한다 (3-7%). 피니셔 직전에 "완성"을 청각으로 예감시킨다.

### 3-2. 무기별 SFX 프로파일

| 무기 | 슬래시 사운드 | 임팩트 사운드 | 특수 SFX | 설계 레퍼런스 |
| :--- | :--- | :--- | :--- | :--- |
| 검 | 중간 "휙" (금속 마찰) | 1타 "챙" / 2타 "챙+" / 3타 "쾅" (피치 상승 계단) | 3타 피니셔 공명 잔향 0.3초 | Hollow Knight Nail |
| 대검 | 묵직한 "쉬익" + 공기 압력 | "쿵" 저주파 중심. 땅이 울리는 느낌 | 2타 충격파 "파동" 저음 레이어 | Dead Cells 대검 |
| 단검 | 매우 짧은 "틱" x 4 | 건조하고 날카로운 "탁" | 배후 강타: 피치 하강 "쾅" 1회 | Hyper Light Drifter |
| 활 | 현 당김 "흐음" (차징) | 발사 "탁" + 충돌 "퍽" | Dead Cells 활 3단 구조 | Dead Cells 활 |
| 지팡이 | 마법 충전 "윙-" | 원소별 충돌음 (화염/냉기/전격 분기) | 투사체 비행음 "씽" | Hades 마법 계열 |

## 4. 앰비언트 / 환경 오디오 (Ambient Audio)

### 4-1. 앰비언트 3레이어 구조

모든 구역의 앰비언트는 3개의 독립 레이어로 구성된다. Hollow Knight 앰비언트 시스템 적용.

```
Layer A: 환경 기저 (Environmental Bed)
  — 항상 재생. 공간 특성을 정의.

Layer B: 거리 이벤트 (Distance Events)
  — 랜덤 간격으로 발생. "저 멀리 뭔가 있다" 암시. 시각화 없음.

Layer C: 근접 트리거 (Proximity Triggers)
  — 특정 오브젝트/구역 진입 시 활성화.
```

**잔향(Reverb) 원칙:** 잔향 길이는 공간 규모의 직접 번역이다. The Shaft의 깊은 층위일수록 잔향이 길어지고("광활한 공동"), 아이템계 지층 내부는 잔향이 짧아진다("금속 상자 안").

**전환 원칙:** 구역 간 이동 시 앰비언트는 스냅 전환이 아닌 크로스페이드(2-3초) 방식. Dead Cells 바이옴 전환 기법 적용.

### 4-2. 월드 층위별 앰비언트 프로파일

| 층위 | 공간 특성 | 앰비언트 기저 | 거리 이벤트 | 잔향 설정 |
| :--- | :--- | :--- | :--- | :--- |
| Tier 2 (세이브) | 따뜻한 대장간. 유일한 안전 지대 | 화덕 소리, 망치 원거리 타격, 낮은 기계 허밍 | 동료 소리, 거래 소리, 물 방울 | 중간 잔향 (돌 공간) |
| Tier 3-4 | 산업 구조물. 탐험 시작 | 금속 구조물 진동, 통풍구 바람, 케이블 긴장음 | 멀리서 들리는 기계 작동음 | 긴 잔향 (대공동) |
| Tier 5-6 | 구조물 노후화. 이질감 증가 | 금속 부식 소리, 저음 맥동, 간헐적 진동 | 원인 불명의 스크래핑 | 매우 긴 잔향 + 플랜지 |
| Tier 7 (최심층) | 빌더의 영역. 세계의 끝 | 거의 침묵. 에르다의 발소리만 들림 | 없음 | 극장급 잔향 (무한 공동) |

**세이브 포인트 오디오 정체성:** 진입 순간, 외부의 산업적 앰비언트가 페이드아웃되고 화덕 소리와 낮은 대장간 허밍이 페이드인된다. 이것이 "이곳은 안전하다"를 전달하는 가장 강력한 신호다.

| T-MILITARY | 금속 갑옷 울림, 먼 행군 소리 | 문 잠금, 발걸음 메아리 | 중간 (석조 복도) | 긴장, 경계 |

| T-FOREST | 바람, 나뭇잎, 물, 벌레 소리 | 낙엽 밟기, 가지 흔들림 | 중간 (야외형) | 고요, 야생의 경계 |
| T-SCHOLAR | 펜 긁는 소리, 낮은 서가 삐걱임 | 책 닫히는 소리, 먼지 파열 | 긴 건식 잔향 (도서관) | 집중, 고독 |
| T-SACRED | 파이프 오르간 드론, 석재 공명 | 촛불 깜박임, 종소리 원거리 | 매우 긴 잔향 (성당) | 경외, 압박 |
| T-MARKET | 낮은 인파 허밍, 거래 소음 | 동전 소리, 저울 소리 | 중간 (도시 골목) | 활기, 혼잡 |
| T-PRISON | 물 방울, 먼 비명 (언어 불명), 쥐 소리 | 철창 소리, 족쇄 끌림 | 긴 습식 잔향 (지하 감옥) | 공포, 고립 |
| T-NATURE | 강 흐름, 바람, 원거리 천둥 | 돌 구름, 흙 무너짐 | 중간-긴 (동굴형) | 원초적, 무심한 자연 |
| T-WAR | 먼 전투 소음 (인식 불가 거리), 포성 잔향 | 무기 충돌 원거리, 깃발 바람 | 매우 긴 잔향 (야전) | 비극, 공허 |

**보스 방 전환 원칙:** 아이템계 방에서 보스 방 진입 시, 앰비언트가 0.5초 내 다운믹스(-6dB)된다. 보스 BGM 시작 시 앰비언트는 거의 사라진다.

## 5. 음악 디렉션 (Music Direction)

### 5-1. BGM 2-Space 분리

| 공간 | 음악 방향 | 핵심 악기 | 강도 시스템 |
| :--- | :--- | :--- | :--- |
| 월드 탐험 | 고딕 산업 앰비언트. 고독하고 광활함. 멜로딕하되 무겁다 | 첼로/비올라 + 산업적 타악기 + 신디사이저 패드 | 탐험/전투 2레이어 크로스페이드 |
| 아이템계 | 금속 긴장 루프. 반복 리듬. 테마별 포인트 악기 레이어 | 금속 타악기 + 베이스 드론 + 테마 포인트 악기 | 지층 깊이에 따라 긴장도 상승 |
| 세이브 포인트 | 대장간 테마. 따뜻하고 리드미컬. 유일한 "집" | 목관악기 + 망치 리듬 타악기 + 낮은 피아노 | 단일 고정 테마. 강도 변화 없음 |
| 보스 | 기존 BGM의 타악기 중심 강도 버전 | 기존 악기 + 공격적 드럼 레이어 | 보스 HP 단계에 따라 레이어 추가 |

**BGM 2-Space 연결 원칙:** 아이템계 BGM은 월드 BGM과 공통 리듬 패턴을 공유하되, 음색(Timbre)이 완전히 다르다. Hades "수퍼제트 아그리안" 장르 블렌딩 기법 적용.

### 5-2. 월드 층위별 BGM 감정 목표

"심리적 날씨 우선" 작업 지시서 포맷. Hollow Knight 크리스토퍼 라킨 접근 적용.

| 층위 | 감정 목표 | 음악 방향 |
| :--- | :--- | :--- |
| Tier 2 | 준비, 안도, 인간의 온기 | 대장간 테마 변형. 따뜻한 화성 |
| Tier 3-4 | 기대, 발견, 경이로움 | 멜로딕 탐험 테마. 현악기 주도 |
| Tier 5 | 긴장, 이질감 | 불협화음 증가. 타악기 비중 상승 |
| Tier 6 | 압박, 단호함 | 화성 구조 붕괴. 드론 중심 |
| Tier 7 | 고독, 경외, 무한 | 앰비언트 텍스처. 구조적 음악의 해체 |

**루프 원칙:** 모든 BGM은 루프 포인트를 음악적으로 해결한다 (멜로디 종지 후 루프). SotN 미치루 야마네 접근.

### 5-3. 아이템계 테마별 BGM 방향

기조(금속 타격 리듬 + 베이스 드론)는 모든 테마 공통. 테마 고유성은 포인트 악기 레이어 1종으로만 표현.

| 테마 ID | 포인트 악기 레이어 | BGM 변화 핵심 |
| :--- | :--- | :--- |
| T-HOME | 목관악기 (따뜻한 멜로디) | 리듬이 요리 리듬 (왈츠풍) |
| T-MILITARY | 군악 스네어 드럼 | 행진 리듬. 정확한 박자감 |
| T-FORGE | 대형 앤빌 타격 (베이스 리듬) | 리듬이 망치 소리 자체가 됨 |
| T-FOREST | 타악 없음. 자연 앰비언트 | 리듬 구조 해체. 자연음이 BGM 대체 |
| T-SCHOLAR | 하프시코드 + 낮은 현악 | 중세 학술 느낌. 느린 템포 |
| T-SACRED | 파이프 오르간 드론 | BGM 거의 없음. 오르간이 앰비언트화 |
| T-MARKET | 만돌린 + 짧은 멜로디 | 가볍고 빠른 템포. 거래 리듬 |
| T-PRISON | 없음 + 저음 맥동 | BGM 극소화. 앰비언트 압도 |
| T-NATURE | 타악 기반 원시 리듬 | 박자 불규칙. 자연의 비주기성 |
| T-WAR | 전쟁 북 + 현악 불협 | 리듬 붕괴 직전의 긴장 |

### 5-4. 음악이 없어야 할 때

음악의 부재 자체가 음악이다. BGM이 페이드아웃되어야 하는 순간:

1. Tier 7 최심층 탐험
2. T-FOREST 지층에서 오래 머물 때 (자연이 BGM을 대체)
3. 아이템계 진입 직전 1초 (침묵이 진입의 무게를 만든다)
4. 에르다가 기억 오브젝트를 마주칠 때 (내러티브 순간)

## 6. 스파이크 사운드 — 아이템계 진입 SFX

아이템계 진입은 ECHORIS의 가장 중요한 단일 오디오 이벤트다. `Documents/Research/ItemWorldEntry_TechImplementation_Research.md`의 8페이즈 연출과 동기화된다.

### 6-1. 진입 SFX 시퀀스

```
Phase 1: Echo Strike 충전 (0~0.5초)
  SFX: 에너지 충전 "윙~" (피치 상승)

Phase 2: 바닥 균열 생성 (0.5~1.0초)
  SFX: 금속 균열 "쩍쩍" + 저음 공명

Phase 3: 바닥 붕괴 (1.0~2.0초)
  SFX: 금속 파열 + 낙하 바람 소리 급증

Phase 4: 에르다 다이빙 (2.0~2.5초)
  SFX: 바람 소리 최대 + 아이템 고유 "흡수" 음

Phase 5: 씬 전환 블랙아웃 (2.5~3.4초)
  SFX: 모든 소리 페이드아웃 0.5초
  이후: 새 테마 앰비언트 페이드인
```

**"흡수" 음 구현 방향:** 에르다가 아이템 속으로 빨려 들어가는 순간, 모든 외부 소리에 급격한 LP 필터(저역 통과)를 걸면서 동시에 음량 감소. "귀가 멍해지는" 기압 변화 느낌. 기술 구현은 `lead-programmer`에게 위임.

### 6-2. 레어리티별 진입 SFX 강도

| 레어리티 | 균열 SFX | 다이빙 SFX | 흡수 음 배율 |
| :--- | :--- | :--- | :--- |
| Normal | 작고 건조한 균열 | 짧은 바람 | x1.0 |
| Magic | 전기 스파크 추가 | 보통 바람 | x1.2 |
| Rare | 황금빛 "챙" 추가 | 강한 바람 | x1.5 |
| Legendary | 불꽃 폭발 추가 | 포효 수준 바람 | x2.0 |
| Ancient | 현실 왜곡 (피치 시프트 모든 SFX) | 차원 붕괴 수준 | x2.5 + 반향 |

## 8. 침묵 주인공 오디오 (Silent Protagonist Audio)

### 8-1. 에르다의 비언어 발성

에르다는 대사 0줄이다. 신체 반응에 해당하는 비언어적 발성만 허용된다.

| 상황 | 비언어 발성 | 음량/피치 | 목표 |
| :--- | :--- | :--- | :--- |
| 대시 | 짧은 숨 내뱉기 ("후") | 작음, 낮음 | 에너지 소비 표현 |
| 피격 | 짧은 충격 숨소리 ("읏") | 중간 | 피해 인식. 과하지 않게 |
| 치명타 피격 | 더 강한 충격음 ("윽") | 중간-큰 | 위험 신호 |
| 사망 | 짧은 쓰러짐 소리 | 중간 | 드라마틱하되 길지 않게 |
| 높은 곳 착지 | 충격 숨소리 + 무릎 구부림 관절음 | 작음 | 물리적 무게감 |
| 아이템계 진입 다이빙 | 숨 참기 (무음) → 진입 후 공기 흡입 | 없음 → 중간 | 긴장감 극대화 |

**절제 원칙:** 에르다의 비언어 발성은 아껴야 의미가 생긴다. 너무 자주, 너무 크게 사용하면 "말 없는 주인공"의 정체성이 희석된다.

### 8-2. 발소리 시스템

발소리는 에르다가 어떤 공간에 있는가를 지속적으로 전달한다.

| 지면 재질 | 발소리 특성 | 적용 층위/테마 |
| :--- | :--- | :--- |
| 금속 격자 | 딱딱한 "탁탁". 짧은 금속 공명 | 월드 구조물 전반 |
| 돌/콘크리트 | 단단하고 낮은 "툭툭" | Tier 5-7 |
| 흙/유기물 | 부드럽고 흡수적인 "퍽퍽" | T-FOREST, T-NATURE |
| 금속 판 (대형) | "탁" + 공명이 길게 울림 | T-FORGE, T-MILITARY |
| 나무 | "삐걱" + 건조한 "탁" | T-HOME |
| 물/진흙 | "첨벙" + 낮은 퍽 | T-NATURE 일부 |

**달리기/걷기 차이:** 달리기는 발소리 간격 짧고 충격 강함. 걷기(잠입)는 음량 -12dB.

### 8-3. 에코의 사운드 정체성

에코는 에르다의 감정과 의도를 표현하는 주요 오디오 채널이다.

| 에코 상태 | 사운드 | 빈도 |
| :--- | :--- | :--- |
| 평상시 대기 | 낮고 안정적인 전기 허밍. 거의 들리지 않음 | 상시 (음량 매우 낮음) |
| 새 구역 스캔 | 짧은 "핑" + 잔향 (소나 효과) | 탐험 중 가끔 |
| 균열 탐지 | 빠른 "비이-" 알림음 | 균열 근처 |
| 아이템계 진입 충전 | 에너지 상승 "윙-" (충전 중 피치 상승) | 진입 트리거 시 |
| 원소 인챈트 활성 | 원소별 지속 허밍 (화염=따뜻한 징잉, 냉기=차갑고 높은 윙) | 인챈트 지속 중 |
| 전투 중 | 허밍 볼륨 약간 상승. 타격 시 각 히트에 에코 공명 추가 | 전투 중 |

**에코 음색 원칙:** 순수 전자음이 아닌 따뜻한 유기적 울림이 있는 전기음. "이것은 단순한 기계가 아니다"를 사운드로 표현.

### 8-4. 기억 오브젝트 오디오 내러티브

에르다가 아이템계 내 기억 오브젝트를 마주칠 때의 오디오 이벤트. 대사 없이 "에르다가 이 기억을 인식했다"를 전달한다.

1. 배경 BGM이 점차 낮아지며 앰비언트 레이어로 이행
2. 이전 주인과 관련된 "소리의 잔향"이 짧게 재생 (T-HOME: 아이 웃음소리 1초, T-MILITARY: 불명확한 군인 목소리)
3. 에르다 발소리가 잠시 느려짐 (이동 속도와 무관한 발소리 레이어)
4. 에코가 낮은 공명음 ("감지했다")

## 11. 오디오 아키텍처 요약 (Audio Architecture)

### 11-1. 오디오 레이어 구조

```
L1: BGM (탐험/전투 2레이어, 크로스페이드)
L2: 앰비언트 루프 (기저/거리이벤트/근접트리거 3레이어)
L3: 에르다 이동 SFX (발소리, 점프, 착지, 대시)
L4: 에코 상태 SFX (허밍, 스캔, 인챈트)
L5: 전투 SFX (슬래시, 임팩트, 피격반응, 킬)
L6: 게임플레이 SFX (아이템 획득, 스탯게이트, 레어리티드랍)
L7: UI SFX (메뉴, 커서, 장착)
```

### 11-2. 오디오 스프라이트 번들 전략

HTTP 요청 최소화를 위한 카테고리별 번들. Howler.js `sprite` 옵션 활용. 기술 구현은 `lead-programmer`에게 위임.

| 번들 파일 | 포함 SFX | 예상 크기 | 로드 타이밍 |
| :--- | :--- | :--- | :--- |
| `sfx_combat_bundle.ogg` | 무기 슬래시 5종 x 3변형, 임팩트 5종 x 3변형, 킬 SFX | 2MB 내외 | 게임 시작 시 |
| `sfx_ui_bundle.ogg` | 메뉴 조작, 아이템 장착, 레벨업, 드랍 SFX | 0.5MB 내외 | 게임 시작 시 |
| `sfx_world_bundle.ogg` | 발소리 (지면 재질 8종), 점프, 착지, 대시 | 1MB 내외 | 게임 시작 시 |
| `sfx_itemworld_bundle.ogg` | 진입 시퀀스 전체, 이노센트 SFX, 보스 전환 | 1.5MB 내외 | 첫 아이템계 진입 전 |

### 11-3. 동시 재생 풀 설정

아이템계 4인 파티 전투 기준.

| 사운드 카테고리 | 권장 pool 크기 |
| :--- | :--- |
| 무기 임팩트 | 8 |
| 무기 슬래시 | 8 |
| 피격 반응 | 8 |
| 환경 SFX | 4 |
| 앰비언트 루프 | 3 |
| BGM | 2 |

### 11-4. Phase 1 구현 우선순위

1. L5 전투 SFX — 검 4종 (슬래시, 임팩트, 킬, 에코스트라이크) — 히트스탑과 동기화
2. L3 에르다 발소리 — 금속 격자 2종 — 존재감 확립
3. L1 BGM — 세이브 포인트, Tier 3 탐험, 아이템계 기본 3트랙
4. L2 앰비언트 — 세이브 포인트 1종
5. L6 아이템계 진입 SFX — Normal 레어리티 1종
6. L7 UI SFX — 메뉴, 선택, 장착 최소 세트

## 12. 예외 처리 (Edge Cases)

| 상황 | 처리 방침 |
| :--- | :--- |
| Web Audio 레이턴시 (30-100ms) | AudioContext 초기화 시 `baseLatency + outputLatency` 측정 후 SFX 트리거 오프셋 보정. AudioContext 사전 워밍(Pre-warming)으로 콜드 스타트 제거 |
| 백그라운드 탭 오디오 정지 | Howler.js `visibilitychange` 이벤트 처리. 재포커스 시 첫 SFX 지연 허용 범위 내 |
| 4인 파티 동시 히트 SFX 충돌 | Howler.js `pool` 설정으로 동시 인스턴스 관리. 과부하 시 낮은 우선순위 SFX(L7 UI)부터 탈락 |
| BGM 루프포인트 불연속 | 모든 BGM 에셋 제작 단계에서 루프포인트 음악적 처리(멜로디 종지 후 루프) 검수 의무화 |
| 탐험/전투 BGM 2레이어 동기화 실패 | 두 트랙은 정확히 같은 BPM, 같은 마디 수, 같은 루프포인트를 가져야 함. 에셋 납품 검수 기준에 포함 |
| 아이템계 진입 중 접속 끊김 | 오디오 상태는 재연결 시 현재 씬 앰비언트로 리셋. 진입 SFX 재실행 없음 |
