## 1. 개요 (Concept)

### 1.1. 설계 의도 (Intent)

> **"아이템계는 모든 장비 아이템이 내부에 품고 있는 절차적 기억의 공간이다. 플레이어는 장비 속으로 다이빙하여 그 기억의 지층을 하나씩 정복함으로써 장비를 영구적으로 강화하고, 그 강함으로 월드의 새로운 영역을 열어낸다."**

본 문서(SYS-IW-01)는 아이템계의 **WHAT**과 **WHY**를 정의한다. 지층이 어떻게 생성되는지(HOW)는 `System_ItemWorld_FloorGen.md` (SYS-IW-02)에서 전담한다.

### 1.2. 설계 근거 (Reasoning)

| 설계 결정 | 채택 이유 | 기각된 대안 |
| :--- | :--- | :--- |
| **기억의 지층(Memory Strata) 컨셉** | 메트로베니아의 공간적 깊이 표현과 일치. "아이템의 기억 속으로 다이빙"이라는 내러티브가 야리코미 동기를 감성적으로 강화. 각 지층이 독립된 미니 메트로베니아 맵으로 기능하여 장르 일관성 유지 | "층(Floor)" 단위 선형 구조 — 횡스크롤에서 "n층"이 공간적으로 부자연스럽고 메트로베니아 비선형 탐험과 충돌 |
| **레어리티별 고정 지층 수** | 사이클 예측 가능성이 "한 번만 더" 심리를 작동시킴. D5 무한층 슬로그 교훈. D7이 30층 고정 복귀로 검증 | 무한 층 구조 — 단기 목표 소실, 이탈 유발 |
| **보스(기억의 문) = 클리어 조건** | 보스를 지층 완료 경계로 설정하면 자연스러운 중단점 형성. 보스 처치 후 "탈출 vs 계속" = 리스크-리턴 판단의 핵심 순간 | 전멸 조건 — 세션 길이 예측 불가, 중단점 부재 |
| **탈출 제단 희소성** (25% 확률) | 탈출 수단 항상 보장 시 리스크 소멸. 25%는 발견 시 안도감과 미발견 시 긴장감 동시 생성. 보스 후 항상 탈출 가능으로 최후 안전망 보존 | 항상 탈출 가능 — "한 층 더" 판단 무의미화 |
| **솔로 + 1~4인 듀얼 모드** | 솔로=개인 야리코미 몰입 보장, 파티=깊은 지층 생존 가능성 확대. 동일 구조 공유로 시스템 복잡도 미증가 | 멀티 전용 — 솔로 이탈 유발 |
| **재귀적 진입 (최대 깊이 3)** | 야리코미 공간의 입체적 확장. 깊이 3 제한은 시드 충돌 방지(SYS-IW-02 IWF-11-A)와 UX 이해 가능성 보장 | 무제한 재귀 — 시드 충돌, 메모리 과부하, 위치 혼란 |

### 1.3. 3대 기둥 정렬 (Pillar Alignment)

| 기둥 | 아이템계에서의 구현 |
| :--- | :--- |
| **메트로베니아 탐험** | 각 지층이 독립된 비선형 Room Grid 미니 맵. 분기 경로, 비밀방, 막다른 길이 존재하며 탈출 제단 위치를 탐색. "이 방 먼저? 크리티컬 패스를 따라갈까?"의 공간적 판단이 매 지층 반복 |
| **아이템계 야리코미** | 레어리티 = 지층 수 = 야리코미 깊이. Normal 2지층부터 Ancient 4+심연까지. 보스 처치가 장비 영구 강화 → 월드 스탯 게이트 해금. 재귀적 진입으로 파밍 공간 입체 확장 |
| **온라인 멀티플레이** | 깊은 지층(3~4)은 파티가 현실적으로 유리. 탈출 제단 위치 공유, 보스 역할 분담. 개인 탈출 허용으로 플레이 스타일 차이 존중 |

### 1.4. 저주받은 문제 검증 (Cursed Problem Check)

| 긴장 | 위험 A | 위험 B | 설계의 선택 |
| :--- | :--- | :--- | :--- |
| **세션 길이 vs 깊이** | 짧으면 야리코미 몰입 소실 | 길면 피로 유발 | 보스 처치를 중단점으로 설정. 지층 1개 = 만족스러운 단위 |
| **솔로 vs 멀티** | 솔로 불리 → 이탈 | 솔로가 더 효율 → 파티 동기 소멸 | 솔로=지층 1~2 핵심, 멀티=지층 3~4 생존 마진. 보상은 동일 |
| **반복 피로 vs 야리코미** | 패턴 피로 발생 | 반복 억제 시 파밍 루프 붕괴 | 재방문 보상 50% 감소로 파밍 대상 자연 교체. 이노센트 랜덤성(Phase 2)이 변수 추가 |
| **안전망 vs 긴장감** | 탈출 어려우면 진입 기피 | 탈출 쉬우면 리스크 소멸 | 탈출 제단(25%) + 보스 후 탈출 이중 안전망. 제단 미발견 시 보스 방향 진행 = "리스크 감수" |

## 2. 핵심 규칙 (Core Rules)

### 2.1. 진입 규칙 (Entry Rules)

> **핵심 원칙:** "내 무기를 두드려서 안에 들어가 강화한다." 에르다가 무기를 모루/제단 위에 올리고 에코로 두드리면 무기의 기억이 깨어나며 균열이 열린다. 이것이 아이템계 진입이다.
>
> **내러티브 근거:** 심연 전쟁(Era 3)이 현실과 아이템 기억 사이의 경계를 갈라놓았다. 100년이 지난 지금도 세계 곳곳에 그 상처가 "기억의 균열"로 남아 있다. 에르다의 망치 에코는 이 균열을 열어 무기의 기억 속으로 들어가는 열쇠다.

#### 진입 공통 조건

| 조건 항목 | 세부 규칙 |
| :--- | :--- |
| **대상 아이템** | 인벤토리의 장비 아이템 (Normal 이상). 소비/재료 아이템 진입 불가 |
| **진입 행위** | 무기를 모루/제단에 올리고 에코로 두드린다 (어떤 진입 장소든 동일) |
| **장착 해제** | 진입 시 자동 장착 해제. 체류 중 잠금 (거래/드랍 불가) |
| **소지품** | 현재 장비·스킬 세트 유지. 내부 획득 아이템은 귀환 시 인벤토리 반영 |
| **재진입** | 사망 탈출 시 진행도 보존. 클리어한 지층 다음부터 재개 |
| **귀환 위치** | 진입한 장소로 복귀 (허브 진입 → 허브, 필드 제단 진입 → 해당 제단) |

#### 진입 장소 3가지

##### 방법 1: 허브 대장간 (Hub Forge)

| 항목 | 내용 |
| :--- | :--- |
| **위치** | 허브(중앙 성채) 내 에르다의 작업대 또는 기억의 사서(Memory Archivist) 시설 |
| **조건** | 없음. 항상 사용 가능 |
| **행위** | 무기를 모루에 올리고 에코로 두드린다 |
| **귀환** | 허브로 복귀 |
| **특징** | 풀 서비스. 이노센트 복종/합성/이식 등 고급 기능은 허브에서만 가능 |
| **내러티브** | 에르다의 대장간이자 직장. 가장 안정적인 진입 방법 |

##### 방법 2: 필드 균열 제단 (Field Fissure Altar)

| 항목 | 내용 |
| :--- | :--- |
| **위치** | 월드 각 구역에 1~2개 고정 배치. 심연 전쟁의 상흔이 안정화된 옛 대장간 폐허/전쟁 제단 |
| **조건** | 제단 발견 (탐험으로 해금. 한 번 발견하면 이후 항상 사용 가능) |
| **행위** | 무기를 제단에 올리고 에코로 두드린다 (허브와 동일한 행위) |
| **귀환** | 해당 제단 위치로 복귀 |
| **특징** | 허브까지 돌아가지 않아도 됨. 탐험 중 즉시 진입 가능. 단, 이노센트 고급 기능 불가 |
| **내러티브** | 심연 전쟁이 남긴 균열이 안정화된 곳. 옛 대장장이들이 전쟁 중 무기를 수리하던 야전 대장간의 흔적 |

**필드 제단 배치:**

| 구역 | 제단 수 | 위치 설명 |
| :--- | :--- | :--- |
| 중앙 성채 | 1 | 기억의 사서 시설 (= 허브 대장간과 겸용) |
| 지하 수로 | 1 | 수로 중간 지점의 야전 대장간 폐허 |
| 마법 연구소 | 1 | 기억 연구원들의 실험용 제단 |
| 빙결 동굴 | 1 | 냉동 보존소 입구 근처 균열 |
| 천공의 탑 | 1 | 탑 중층의 관측 제단 |
| 묘지/카타콤 | 2 | 수호단 야전 대장간 + 카타콤 깊은 곳 봉인 제단 |
| 심연의 구 | 1 | 카엘 오르스가 심연의 검을 벼렸던 최후의 모루 |

##### 방법 3: 기억의 방랑자 (Memory Wanderer)

| 항목 | 내용 |
| :--- | :--- |
| **출현** | 월드 필드에 랜덤 출현하는 유령형 NPC. 인간 실루엣. 적대적이지 않음 |
| **행동** | 천천히 걸어다니다 일정 시간(30초) 후 소멸 |
| **접촉** | 에르다가 접촉하면 방랑자가 멈추고 그 자리에 **임시 균열 제단** 생성 (60초 유지) |
| **행위** | 임시 제단에 무기를 올리고 에코로 두드린다 (동일한 행위) |
| **귀환** | 방랑자가 있던 위치로 복귀 (임시 제단은 소멸) |
| **특징** | **보상 강화 아이템계** — 이노센트 출현율 ×1.5, 보스 드랍 보너스 |
| **출현 빈도** | 평균 30~45분에 1회. 구역 난이도가 높을수록 출현 확률 증가 |

**내러티브:**
기억의 방랑자는 과거에 아이템 속으로 들어갔다 돌아오지 못한 자들의 잔상이다. 이들은 현실과 기억의 경계에 걸쳐 떠돌며, 자신이 들어갔던 균열의 흔적을 무의식적으로 재현한다. 에르다가 접촉하면 방랑자의 기억이 일시적으로 균열을 열어주는 것이다.

> **Act 3 트리거:** 게임 후반부, 특정 조건 달성 후 방랑자 중 하나가 **마르타 벤-나흐트의 실루엣**으로 나타난다. 에르다가 이를 알아차리는 순간이 Act 2 → Act 3 전환의 감정적 트리거. 이 방랑자는 접촉해도 사라지며, 임시 제단을 남기지 않는다. 대신 에르다의 독백만 남는다: "...스승님?"

**출현 연출:**
- 방랑자 출현 시 특유의 SFX (먼 곳에서 울리는 에코 소리)
- 화면 가장자리에 미세한 필터 효과 (기억이 스며드는 시각 표현)
- 플레이어에게 "저기 방랑자가 있다!" → 30초 안에 접촉해야 하는 긴장감

#### 진입 행위 공통 연출: 기억의 두드림 (Echo Strike)

> **콘셉트:** 에르다가 에코로 무기를 두드리면 무기의 기억이 깨어나 균열이 열린다. 대장장이의 일상 행위 = 아이템계 진입 의식. 이것이 프로젝트의 루도내러티브 하모니 핵심.

**시퀀스 (총 2.0초, 4 Phase):**

| Phase | 시간 | 연출 | 사용 기법 |
| :--- | :--- | :--- | :--- |
| 1. 두드림 | 0.0~0.5초 | 에르다가 에코를 들어올려 무기를 내리친다. **히트스탑 6프레임**. 무기 표면에 빛나는 균열이 퍼져나감. 레어리티 색상 글로우 | 히트스탑, DisplacementFilter, 레어리티 ColorTint |
| 2. 균열 확장 | 0.5~1.3초 | 균열이 무기를 넘어 공간으로 확장. 포탈이 열린다. 에르다가 빛 속으로 끌려 들어간다. **1프레임 화면 플래시** + 캐릭터 뒤에 잔상 3~5개 | 스프라이트 복제 + alpha 감쇄, 화면 플래시 |

### 2.2. 지층 진행 규칙 (Strata Progression)

#### 레어리티별 지층 테이블

| 레어리티 | 지층 수 | Grid 구성 | 야리코미 깊이 |
| :--- | :---: | :--- | :--- |
| Normal | 2 | 4×4 → 4×4 | 기본 |
| Magic | 3 | 4×4 → 4×4 → 4×4 | 중급 |
| Rare | 3 | 4×4 → 4×4 → 4×4 | 상급 |
| Legendary | 4 | 4×4 → 4×4 → 4×4 → 4×4 | 고급 |
| Ancient | 4+심연 | 4×4 → 4×4 → 4×4 → 4×4 + Anchor 무한 | 극한 |

#### 지층 클리어 조건

- 보스(기억의 문) 격파로 클리어
- 사전 조건 없음 — 크리티컬 패스 따라 보스 방 도달 즉시 도전 가능
- 보스 미처치 사망 시 해당 지층 미클리어

#### 지층 간 전환

```
보스 처치 → [선택 UI] → "탈출" → 허브 귀환 (진행도 저장)
                       → "계속" → 다음 지층 생성 (새 시드)
```

**지층 간 전환 연출 (0.8초):**

| Phase | 시간 | 연출 |
| :--- | :--- | :--- |
| 1. 균열 | 0.0~0.2초 | 보스 처치 후, 보스 방 바닥 중앙에 **빛나는 균열** 발생. 레어리티 색상으로 빛남 |
| 2. 개방 | 0.2~0.4초 | 균열이 확대되어 **함정문(trapdoor)** 형태로 열림. 아래에서 다음 지층의 빛이 새어 나옴 |
| 3. 낙하 | 0.4~0.7초 | 캐릭터가 함정문으로 **자유낙하**. 낙하 중 배경이 **팔레트 시프트**로 다음 지층 색조로 전환 |
| 4. 착지 | 0.7~0.8초 | 다음 지층 시작 방에 착지. 먼지 파티클 + 약한 화면 흔들림(2px) |

**탈출 연출 (1.0초):**

| 경로 | 연출 |
| :--- | :--- |
| 보스 후 탈출 | 진입의 **역재생**: 캐릭터가 위로 상승 → 아이템 텍스처로 줌아웃 → 아이템 축소 → 허브 복귀 |
| 탈출 제단 | 제단에 손을 올리면 **백색 빛 기둥**이 캐릭터를 감싸며 상승 → 페이드아웃 → 허브 |
| 사망 탈출 | 화면 **적색 틴트 + 디더링 페이드아웃** → 허브에서 쓰러진 채 등장 → 일어남 |

### 3.3. 탈출 제단 확률

기본 확률 25%에 지층 깊이 보너스가 가산된다. 깊이가 깊어질수록 탈출 수단이 조금 더 자주 등장하여 극단적 진행을 완화한다.

| 지층 | 탈출 제단 스폰 확률 | 설계 의도 |
| :---: | :---: | :--- |
| 1 | 25% | 기준 |
| 2 | 28% | 미미한 완화 |
| 3 | 30% | 중간 보정 |
| 4 | 32% | 최심층 안전망 강화 |
| 심연 | 35% (Phase 2) | 무한 진행 안전망 |

> 배치는 시드 기반 결정적(deterministic) 생성. 동일 아이템·지층은 항상 동일 결과.

### 3.4. 사망 패널티 파라미터

| 파라미터 | 수치 | 비고 |
| :--- | :---: | :--- |
| EXP 손실 비율 | 30% | 현재 지층 진입 이후 획득분 기준 |

## 4. 흐름 (Flow)

### 4.1. 전체 플로우

```
[허브]
  │
  ▼ 기억의 사서 NPC 상호작용 + 장비 아이템 선택
  │
[아이템계 진입]
  │  (currentStratum = lastSafeStratum 또는 1)
  │
  ├─────────────────────────────────────────┐
  │                                         │
  ▼                                         │
[지층 N 탐험]                               │
  │                                         │
  ├── 탈출 제단 발견 ──────────────────────►│
  │                                         │
  ├── 보스 처치 ──► [선택 UI]              │
  │                   ├── "탈출" ──────────►│
  │                   └── "계속" ──┐        │
  │                                │        │
  │                    (최종 지층이면 강제 탈출)
  │                                ▼        │
  │                        [지층 N+1 탐험]  │
  │                           (반복)        │
  │                                         │
  ├── 사망 ──────────────────────────────►│
  │                                         ▼
  │                                    [허브 귀환]
  │                                    진행도 저장
  │                                    보상 지급
  └─────────────────────────────────────────┘
```

### 4.2. 단일 지층 플로우

```
[지층 N 입구 (Start Room)]
  │
  ▼ 적 탐색 + 방 탐험 시작
  │
  ├── 크리티컬 패스 방 진입
  │     ├── 25% 확률: 탈출 제단 등장 → [탈출 옵션]
  │     └── 75% 확률: 일반 전투 방
  │
  ├── 분기 경로 탐색 (선택)
  │     ├── 비밀방 발견 → 보너스 게이지 +50
  │     └── 막다른 방 → 추가 드랍
  │
  ▼ 보스 방 (End Room) 도달
  │
  [보스 전투: 기억의 문]
  │
  ├── 승리 → 영구 스탯 보너스 지급
  │         → 아이템 EXP 지급 (×3~5)
  │         → [선택 UI: 탈출 / 계속]
  │
  └── 패배 → 사망 패널티 적용 → 허브 귀환
```

### 4.4. 사망 처리 흐름

```
[HP 0 도달]

  ▼ 사망 연출 (화면 암전, 아이템 파편 이펙트)
  │
  ├── 긴급 탈출 소모품 보유 시: [소모품 사용 여부 팝업]
  │     ├── 사용 → HP 30% 회복 + 현재 지층 유지
  │     └── 미사용 → 패널티 적용
  │
  └── 패널티 적용:

  │
  └── 전원 사망 시 → 파티장 아이템 기준 패널티 적용
```

### 5.6. Design_Yarikomi_Philosophy.md와의 정렬

| 야리코미 원칙 | 아이템계에서의 구현 |
| :--- | :--- |
| **예측 가능한 목표** | 레어리티별 고정 지층 수로 "몇 번 더 하면 된다"가 항상 명확 |
| **점진적 깊이** | Normal 2층 → Ancient 4+심연. 진입 장벽 없이 자연스러운 야리코미 심화 |
| **리스크-리턴 판단** | 보스 후 탈출/계속 선택 + 탈출 제단 발견 여부 = 매 지층 리스크 평가 |
| **한 번만 더 심리** | 지층 1개 클리어 = 영구 보상 획득. 짧은 보상 사이클이 재진입 유도 |
| **무한 파밍 동기** | 재귀 진입(깊이 3)으로 파밍 대상 이론상 무제한 확장 |

### 5.7. Design_CoreLoop_Circulation.md와의 정렬

아이템계는 핵심 순환 구조의 **중심 엔진**이다.

```
월드 탐험
  → 아이템 획득
  → [아이템계 진입] ← 본 문서
      → 지층 정복
      → 장비 영구 강화
  → 스탯 게이트 해금
  → 새 구역 탐험
  (반복)
```

| 순환 구조 요소 | IWC 연동 |
| :--- | :--- |
| 입력 | 월드에서 획득한 장비 아이템 |
| 출력 | 영구 스탯 증가 + 이노센트(Phase 2) |
| 동기 | 보스 처치 보너스 → 스탯 게이트 해금 |
| 재진입 동기 | 더 높은 레어리티 아이템 획득 → 더 깊은 지층 탐험 |

- [ ] 지층 1 탈출 제단 발견 → 귀환 → 재진입 시 지층 1에서 시작 (lastSafeStratum = 1)
