## 1. 시스템 개요

### 1.1. 설계 의도

> "환경은 텍스트 없이 서사를 말한다. 바닥 타일 하나, 조명 색조 하나가 아이템의 기억을 체험하게 한다."

각 테마 팔레트는 5가지 시각 요소(바닥, 벽면, 조명, 색조, 파티클)와 5가지 범용 환경 오브젝트, 그리고 8개의 40초 이벤트 풀로 구성된다.

아이템 서사 문서는 테마 코드를 선택한 뒤:
1. 팔레트 5요소를 그대로 사용하거나 아이템 고유 세부사항을 덮어쓴다.
2. 환경 오브젝트 5개 중 아이템과 무관한 것을 교체할 수 있다.
3. 이벤트 풀 8개 중 5개를 자동 선택하되, 서사적으로 중요한 이벤트는 고정(Pinned) 처리한다.

### 1.2. 지층별 팔레트 원칙

| 지층 | 팔레트 원칙 | 비고 |
| :--- | :--- | :--- |
| 지층 1 | 가장 따뜻하거나 밝은 색조. 아이템의 가장 최근 기억이므로 친숙한 공간 | 플레이어가 적응하는 공간 |
| 지층 2 | 지층 1보다 어둡거나 차가운 색조. 과거로 거슬러 올라가는 느낌 | 중간 전환 공간 |
| 지층 3 | 가장 원형적이거나 추상적인 공간. 두 팔레트가 혼합될 수 있음 | 서사 정점. 혼합 팔레트 규칙 적용 |

### ENV-HOME: 가정 / 주방

#### 2.1. 시각 요소 5종

| 요소 | 상세 | 서사적 기능 |
| :--- | :--- | :--- |
| **바닥 타일** | 나무 마루. 반들반들하게 닳아 있음. 곳곳에 칼자국, 기름 자국 | 오랫동안 같은 자리에서 같은 일이 반복되었다 |
| **벽면** | 황토 또는 흰 석회 벽. 레시피 메모지, 달력, 가족 그림 등이 붙어 있음 | 생활의 흔적이 벽에 쌓인다 |
| **조명** | 화덕 또는 벽난로의 따뜻한 오렌지 빛. 창문에서 들어오는 자연광 병행 | 따뜻하고 안전한 공간의 신호 |

| **파티클** | 요리 김, 화덕 불꽃 파편, 공기 중 밀가루 입자, 말린 허브 가루 | 공간이 살아 있다는 신호 |

| **색조** | 청녹색-짙은 남색 기조. 깊이에 따라 어두워짐 |
| **파티클** | 물거품, 기포, 소금 결정, 해초 조각 |

**보스 템플릿:** 거대한 문어형 또는 유령 선박. 처치 연출: 서서히 심해로 가라앉으며 마지막으로 닻이 남음.

## 3. 혼합 팔레트 규칙 (지층 3 전용)

### 3.1. 혼합 적용 조건

지층 3에서 두 테마 팔레트를 혼합하는 것은 아이템의 핵심 서사 갈등(예: 요리사의 삶 vs. 경비병의 삶)을 공간으로 구현할 때 사용한다. 모든 지층 3이 혼합 팔레트를 사용해야 하는 것은 아니다.

| 혼합 사용 권장 상황 | 혼합 불필요 상황 |
| :--- | :--- |
| 아이템 소유자가 두 삶 사이에서 선택한 기억이 있을 때 | 아이템의 기원이 단일 테마로 명확할 때 |
| 지층 1과 지층 2의 테마가 대조적일 때 | 지층 1~2가 같은 테마 계열일 때 |
| T-PHANTOM 등장이 서사적으로 강조될 때 | 단순한 배경 아이템 서사일 때 |

### 3.2. 혼합 팔레트 시각 규칙

| 요소 | 혼합 방식 |
| :--- | :--- |
| **바닥 타일** | 한 테마의 바닥이 다른 테마의 바닥으로 점진적으로 전환됨. 경계선에서 두 타일이 섞임 |
| **벽면** | 방의 절반씩 각 테마의 벽면을 사용. 또는 두 테마 오브젝트가 혼재 |
| **조명** | 두 테마의 조명이 공간 내에서 분리. 예: 한쪽은 횃불(MILITARY), 반대쪽은 화덕 빛(HOME) |
| **색조** | 두 테마의 색조가 화면 중앙에서 경계를 이루며 블렌딩 |
| **파티클** | 두 테마의 파티클이 공존. 수량은 각 50%씩 |

### 3.3. 보스 방 혼합 규칙

지층 3 보스 방은 두 테마의 요소가 가장 명확하게 대립하는 공간으로 설계한다. T-PHANTOM이 등장하는 사전 구역과 보스 방을 분리하여 배치한다.

| 구역 | 테마 처리 |
| :--- | :--- |
| T-PHANTOM 구역 (보스 방 직전) | 테마 색조 완전히 제거. 흰색-회색 무채색 공간 |
| 보스 방 진입 시 | 두 테마 색조가 동시에 강하게 표현 |
| 보스 처치 후 | 색조가 서서히 단일 색 (따뜻한 중성 색)으로 수렴. 서사 해소 표현 |

## 4. 지층 구성 자동화 규칙

아이템 서사 문서를 작성할 때 이 문서의 팔레트를 적용하는 기본 흐름:

```
1. 지층별 테마 코드 선정 (몬스터 풀 문서와 동일한 코드 사용)
2. 해당 팔레트의 5요소를 그대로 적용
3. 범용 오브젝트 5개 중 "반드시 교체" 항목만 아이템 고유 내용으로 대체
4. 이벤트 풀 8개 중 5개 선택:
   - Pinned 이벤트는 반드시 포함
   - 나머지는 서사적 우선순위에 따라 선택
5. 보스 템플릿 이름과 처치 연출을 아이템 기원과 연결하여 교체
6. 지층 3이면 혼합 팔레트 필요 여부 판단 후 적용
```
