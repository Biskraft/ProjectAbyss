## 1. 개요 (Concept)

### 1-1. 설계 의도 (Design Intent)

Project Abyss의 월드는 **대공동(The Shaft)**이라 불리는 거대한 수직 구조물이다. 대장장이 에르다는 중앙 성채(Tier 2, 허브)에서 출발하여, 심연 전쟁의 상흔이 남은 7개 층위를 **아래로 하강**하며 스승 마르타의 흔적을 쫓고, 최종적으로 최심부 심연의 구(Tier 7)에서 세계의 진실과 마주한다. 후반부에는 허브 위쪽의 천공의 정원(Tier 1)으로 올라가는 역방향 경로가 열린다.

이 구조는 세 가지 설계 목표를 동시에 달성해야 한다:

1. **수직 하강의 긴장감**: 아래로 내려갈수록 강해지는 위협과 깊어지는 비밀. "심연으로의 하강"이라는 물리적 체험이 내러티브와 일치
2. **에르다의 감정 아크**: 코미디(의뢰 수행) → 전환(기억의 진실 목격) → 진지(마르타의 흔적 추적)가 층위 하강과 함께 자연스럽게 전개
3. **순환 구조 연결**: 층위 탐험 → 아이템 획득 → 제단에서 에코로 두드려 아이템계 진입 → 장비 강화 → 스탯 게이트 해금 → 다음 층위 하강

### 1-2. 설계 근거 (Reasoning)

| 결정 | 근거 |
| :--- | :--- |
| 숲길 → 대장간 → 중앙 성채 시작 동선 | 첫 30분 플로우(`CNT-EXP-001`)에서 에르다는 숲길을 걸어 대장간에 도착하고, 에코 공명 사건 후 허브(중앙 성채)로 이동. 게임 시작점이 허브가 아니라 숲길 |
| 수직 하강 + 가지 경로 토폴로지 | 수직 방향의 명확한 진행감을 제공하면서, 각 층위 내부에서 횡방향 메트로베니아 탐험을 보장. 하강할수록 깊어지는 공포와 경외감이 "심연" 테마와 일치 |
| 7개 층위 = 심연 전쟁의 7개 흔적 | 각 층위는 단순 바이옴이 아니라 심연 전쟁의 상흔이 남은 역사적 장소. World Bible Layer 1-3과 직접 연결 |
| 이중 게이트 (능력 + 스탯) | 능력 게이트만으로는 아이템계 진입 동기가 약함. 스탯 게이트가 "무기를 두드려 강화하러 가자" 순환을 자연스럽게 창출 |
| 격벽(Bulkhead) 기반 능력 게이트 | 층위 간 경계에 격벽을 배치하여, 특정 능력 없이는 다음 층위로 하강 불가. 물리적 구조가 게이트를 자연스럽게 정당화 |
| 층위별 마르타의 흔적 배치 | 각 층위에 마르타가 과거에 방문했던 흔적(작업 자국, 메모, 수리 흔적)이 있다. 에르다의 장기 동기이자 세계 탐험의 내러티브 보상 |
| 후반부 상승(T1) | 하강 일변도를 피하고, "정상에서 세계를 내려다보는" 반전 경험 제공. 최종 층위 전 시야 확장 |

### 1-3. 3대 기둥 정렬 (Pillar Alignment)

| 기둥 | 월드 맵 구조에서의 구현 |
| :--- | :--- |
| **탐험가** | 능력 게이트(격벽)가 층위 하강을 정의. 각 능력 획득 시 다음 층위와 기존 층위의 잠긴 가지 경로가 해금. 마르타의 흔적이 탐험의 내러티브 보상 |
| **장인** | 스탯 게이트가 아이템계로의 자연스러운 통로. "저 격벽을 부수려면 장비가 더 강해야 한다" → 근처 필드 제단에서 무기를 두드려 아이템계로 |
| **모험가** | 허브(중앙 성채)가 수직 구조의 거점. 포탈/제단을 통해 허브 귀환. 멀티플레이는 아이템계에서, 월드는 솔로의 고독한 하강 |

### 1-4. 저주받은 문제 (Cursed Problems)

| 문제 | 상충하는 요구 | 설계 해법 | 희생/제약 |
| :--- | :--- | :--- | :--- |
| 수직 선형성 vs 탐험 자유도 | 수직 하강은 선형적이지만, 메트로베니아는 비선형이어야 함 | 층위 간 이동은 선형(T2→T3→…→T7)이지만, 각 층위 내부는 횡방향 메트로베니아 가지 경로로 넓은 탐험 자유도 제공. 이전 능력으로 상위 층위 잠긴 경로 해금 가능 | 층위 순서 자체는 고정. 자유도는 층위 내부에서 발생 |
| 파밍 효율 vs 탐험 동기 | 아이템계에서 무한 파밍하면 월드 탐험을 하지 않으려 함 | 스탯 게이트만 파밍으로 해결 가능. 능력 게이트(격벽)는 월드에서만 획득. 마르타의 흔적은 월드에서만 발견 | 플레이어는 양쪽 콘텐츠를 모두 경험해야 함 |
| 하강 후 귀환의 번거로움 | 깊은 층위에서 허브까지 물리적으로 올라가는 것은 비현실적 | 포탈/제단을 통한 즉시 허브 귀환. 한번 도달한 격벽 앞에는 빠른 이동 지점 설치 | 물리적 역행 경험은 포기. 빠른 이동에 의존 |
| 절차적 생성의 반복감 vs 역사적 장소감 | 절차적 생성하면 "심연 전쟁이 여기서 일어났다"는 장소 정체성 손실 위험 | 층위별 핵심 랜드마크(보스방, 마르타 흔적, 서사 오브젝트)는 핸드크래프트 고정. Room 내부 Chunk만 절차적 변화 | 일부 층위에서 반복 구조감 잔존 |
| 필드 제단 vs 허브 중심 설계 | 필드 균열 제단에서 아이템계 진입 → 현장 복귀하면 허브를 거치지 않음 | 필드 제단은 진입+강화만 가능. 이노센트 복종/합성 등 고급 기능은 허브 대장간에서만 가능 | 필드에서 빠른 강화는 가능하지만, 심화 강화는 허브 방문 필요 |

**수직 하강 + 가지 경로 구조 설명:**
- **수직 축 (Vertical Spine):** 허브(T2) → T3 → T4 → T5 → T6 → [대격벽] → T7 순서로 하강. 각 층위 사이에 격벽이 있으며, 해당 능력을 획득해야 다음 층위로 내려갈 수 있다
- **후반부 상승 분기:** 허브(T2) → T1(천공의 정원). 역중력 능력 획득 후 허브 위쪽으로 올라가는 후반부 경로
- **횡방향 가지 경로:** 각 층위 내부는 좌우로 넓은 메트로베니아 탐험 구조. 잠긴 가지 경로는 이전/이후 층위에서 획득한 능력으로 해금
- **귀환 시스템:** 각 층위에 포탈/제단 배치. 허브로의 즉시 귀환 가능 (물리적 역등반 불필요)
- **빠른 이동:** 한번 도달한 격벽 앞에 빠른 이동 지점 자동 활성화

**숲길 + 대장간 (진입 시퀀스):**
- 게임 최초 시작 전용. 첫 30분 플로우 이후에는 대장간 ↔ 중앙 성채를 자유 이동
- 숲길과 대장간은 Tier 2(허브) 외곽의 진입 동선이며, 독립 층위가 아님

### 2-2. 층위별 정체성 — 세계관 연결 (Tier Identity)

| **T6: 빙결 보존소** | 냉동 보존소 (Cryo-Archive) | 냉동 시스템 오작동. 극저온+해동 혼재 | 소실 위기의 고대 기억 복원. 마르타의 수리 흔적 | 수중 호흡 (격벽 ④) |
| **T7: 심연의 구** | 밀봉지 (Sigil Point) | 협약 이후 불안정. 최근 침식 재개 | 카엘 오르스의 흔적. 계시 1-5. 제3의 길 | 대격벽 (심연 저항) |
| **T1: 천공의 정원** | 공중 정원 / 관측소 (Canopy) | 100년 전 봉쇄. 최근 정원에서 빛 관측 | 관측자들의 마지막 예언. 세계의 진실에 가까운 단서 | 역중력 (후반부 상승) |

### 2-3. 마르타의 흔적 시스템 (Marta's Trail)

> 에르다의 스승 마르타 벤-나흐트는 10년 전 사라지기 전에 여러 층위를 방문했다. 각 층위에 마르타가 남긴 흔적이 1개씩 있으며, 이것이 에르다의 장기 탐험 동기이자 감정 아크의 구동력이다.

| 층위 | 마르타의 흔적 | 에르다의 반응 (감정 아크) | 발견 조건 |
| :--- | :--- | :--- | :--- |
| **대장간** | 에코 공명 후 드러나는 쪽지 | "스승님도 이걸 알고 있었어?" (호기심) | 첫 30분 내 자동 |
| **T3: 묘지/카타콤** | 묘비 사이에 마르타가 새긴 에코의 각인 | "에코의 흔적이야. 스승님이 여기서 무언가를 하고 갔어." (추적 시작) | 비경로 탐험 |
| **T4: 지하 수로** | 수로 벽에 남은 망치 자국과 수리 흔적 | "스승님이 여기도 왔었구나. 이 수로를 고치려 한 건가?" (의문 심화) | 비경로 탐험 |
| **T5: 연구소 폐허** | 아르카나 리브르의 연구실에 마르타의 메모 | "스승님이 기억 연구원의 기록을 읽었다? 뭘 알아낸 거지?" (직업적 공감) | 보스 처치 후 |
| **T6: 빙결 보존소** | 냉동 보존소에서 마르타가 수리한 고대 아이템 | "이 아이템... 스승님이 고친 거야. 왜 여기까지 와서?" (충격) | 비경로 탐험 |
| **T1: 천공의 정원** | 관측자의 기록에 "벤-나흐트의 장인"이라는 언급 | "스승님의 이름이 여기 있어. 관측자들이 스승님을 알고 있었다?" (확신) | 보스 처치 후 |
| **T7: 심연의 구** | 심연 입구에 에코와 동일한 공명 자국 | "스승님이 여기까지 왔다. 그리고... 들어갔다." (결심) | 최종 층위 진입 |

**설계 원칙:**
- 마르타의 흔적은 Critical Path에 배치하지 않는다 (선택적 발견)
- 발견 순서에 무관하게 각 흔적은 독립적으로 의미가 있다
- 모든 흔적을 모으면 마르타가 왜 Ancient 아이템 안으로 들어갔는지 추론 가능
- 흔적 발견 시 에르다의 독백만으로 전달. NPC 설명 없음

### 2-5. Critical Path 정의

**수직 하강 경로:**

```
[숲길] → [대장간] → [에코 공명 사건] → [중앙 성채 / T2 허브]
  ↓ (에르다의 첫 의뢰)
  ↓ (T3 격벽 ① 앞에서 이단 점프 획득 이벤트)
[T3: 묘지/카타콤] — 이단 점프로 격벽 ① 통과
  ↓ (보스 처치 → 벽 타기 획득)
[T4: 지하 수로] — 벽 타기로 격벽 ② 통과
  ↓ (보스 처치 → 안개 변신 획득)
[T5: 연구소 폐허] — 안개 변신으로 격벽 ③ 통과
  ↓ (보스 처치 → 수중 호흡 획득)
[T6: 빙결 보존소] — 수중 호흡으로 격벽 ④ 통과
  ↓ (보스 처치 → 역중력 획득)
  ↓
  ├──→ [T1: 천공의 정원] — 역중력으로 허브 위 상승 (후반부 분기)
  ↓
[대격벽] — 심연 저항 획득 (최종 조건)
  ↓
[T7: 심연의 구] → [카엘 오르스와의 대면] → [선택]
```

**원칙:**
- Critical Path는 수직 하강 순서를 따른다 (T2→T3→T4→T5→T6→T7)
- 각 격벽은 해당 능력 없이는 통과 불가 (능력 게이트)
- 스탯 게이트는 Critical Path에 배치하지 않는다 (가지 경로에만 배치)
- 마르타의 흔적은 Critical Path 밖에 배치 (선택적 탐험 보상)
- 모든 시드에서 시작점 → 최종 보스 경로가 보장된다
- T1(천공의 정원)은 Critical Path의 선택적 분기 (최종 층위 진입 전 후반부 콘텐츠)

### 2-6. 능력 게이트 (Ability Gate) 배치 규칙

| 능력 | 획득 위치 | 통과 격벽 | 새로 접근 가능한 층위/구간 | 내러티브 의미 |
| :--- | :--- | :--- | :--- | :--- |
| **이단 점프** | T2(허브) 하단 이벤트 | 격벽 ① | T3 진입 + 상위 층위 높은 가지 경로 | 에르다가 에코의 힘으로 도약하는 법을 깨달음 |
| **벽 타기** | T3 보스 처치 | 격벽 ② | T4 진입 + 이전 층위 수직 벽면 가지 경로 | 수호단 갑옷의 부적에서 얻는 능력 |
| **안개 변신** | T4 보스 처치 | 격벽 ③ | T5 진입 + 1타일 너비 틈새 진입 | 수로 깊은 곳의 물안개에서 습득 |
| **수중 호흡** | T5 보스 처치 | 격벽 ④ | T6 진입 + 수중 통로 진입 | 연구원이 기록한 기억 호흡법 응용 |
| **역중력** | T6 보스 처치 | T1 상승 경로 | T1(천공의 정원) 진입 + 역중력 구간 | 냉동 보존소 깊은 곳의 고대 부적에서 획득 |
| **심연 저항** | 최종 조건 | 대격벽 | T7(심연의 구) 진입 | 에르다의 에코가 심연의 기억과 공명하여 얻는 최종 능력 |

**배치 원칙:**
- 능력 게이트는 층위 간 격벽에 배치 (상하 층위 이동 차단)
- 각 층위의 보스 처치가 다음 층위 격벽 해제의 열쇠
- 능력 게이트 위치는 모든 시드에서 동일 (매크로 구조 고정)
- 각 능력은 해당 층위의 역사와 의미가 연결된다
- 획득한 능력으로 이전 상위 층위의 잠긴 가지 경로를 역방문 해금 가능

### 2-7. 스탯 게이트 (Stat Gate) 배치 규칙

| 스탯 | 임계값 구간 | 배치 위치 | 목적 |
| :--- | :--- | :--- | :--- |
| **STR** | 10-20 - 30 | 각 층위 가지 경로 보상 영역 | 높은 방어도의 적/물체 파괴 |
| **INT** | 10-20 - 30 | 연구소 폐허(T5) 중심 | 마법 저항 적 처치, 마법 함정 해제 |
| **DEX** | 10-20 - 30 | 타이밍 기반 함정, 빠른 적 | 정밀 점프, 빠른 이동 구간 |
| **VIT** | 5-15 - 25 | 고난이도 적 밀집 구간 | 높은 피해 환경 생존 |
| **SPD** | 5-10 - 15 | 시간 제한 구간 | 타이밍 플랫포밍, 시간 챌린지 |
| **LCK** | 10-20 - 30 | 숨겨진 경로 | 숨겨진 아이템/이노센트/마르타 흔적 발견 |

**배치 원칙:**
- 스탯 게이트는 층위 내부 가지 경로(비경로 탐험 구간)에 배치
- Critical Path(수직 하강 경로) 위에 스탯 게이트를 배치하지 않는다 (진행 보장)
- 각 층위마다 3-5개의 스탯 게이트
- 스탯 게이트 임계값은 층위 깊이에 비례 (아래로 갈수록 높은 스탯 요구)
- **LCK 게이트 뒤에 마르타의 흔적이 배치될 수 있다** — 탐험적 플레이어에 대한 보상

## 4. 흐름 (Flow)

### 4-1. 에르다의 여정 흐름도 (Erda's Journey Flowchart)

```
[숲길 — 일상 세계]
    ↓
[대장간 — 에코로 의뢰 검을 두드림 → 첫 균열 (소명)]
    ↓
[아이템계 첫 진입 (시련)]
    ↓
[대장간 복귀 — 거부 + 마르타 쪽지 (스승)]
    ↓
[중앙 성채 / T2 허브 — 오렌 만남, 첫 의뢰 (관문 통과)]
    ↓
[이단 점프 획득 → 격벽 ① 통과]
    ↓
[T3: 묘지/카타콤] ── 벽 타기 획득 + 마르타 에코 각인 + 필드 제단 발견
    ↓
[T4: 지하 수로] ── 안개 변신 획득 + 마르타 수리 흔적 + 필드 제단 발견
    ↓
    ↓ (각 층위에서 필드 제단 발견 → 허브 안 돌아가도 아이템계 진입 가능)
    ↓ (기억의 방랑자 랜덤 출현 → 보상 강화 아이템계)
    ↓ (장비 강화 → 스탯 게이트 해금 → 가지 경로 탐험)
    ↓
[T5: 연구소 폐허] ── 수중 호흡 획득 + 마르타 메모 + 필드 제단 발견
    ↓
[T6: 빙결 보존소] ── 역중력 획득 + 마르타가 수리한 고대 아이템 + 필드 제단 발견
    ↓
    ├──→ [T1: 천공의 정원] — 역중력으로 허브 위 상승 (후반부 분기)
    ↓                       관측자의 예언 + 마르타의 이름 + 필드 제단 발견
    ↓
[대격벽 — 심연 저항 획득]
    ↓
[T7: 심연의 구 — 최종 층위]
    ↓ (카엘 오르스의 기억과 대면)
    ↓ (계시 1~4 수집)
    ↓
[선택: 협약 유지 / 협약 파기 / 제3의 길]
    ↓ (계시 5: 이 세계도 무기의 내부)
    ↓
[에르다: 망치 에코로 세계의 경계를 두드린다]
```

### 4-2. 세션 구조 (Session Structure)

**목표 평균 세션: 45분**

| 단계 | 시간 | 활동 | 에르다의 상태 |
| :--- | :--- | :--- | :--- |
| 1. 층위 탐험 | 20-25분 | 현재 층위 메인 경로 + 가지 경로 탐험 | 의뢰 수행 + 마르타 흔적 탐색 |
| 2. 보스/챌린지 | 5-10분 | 층위 진행 보스 또는 보상 챌린지 | 역사 단편 목격 |
| 3. 제단에서 무기 두드림 → 아이템계 | 15-20분 | 획득한 장비 강화 | 장인 모드 |
| 4. 귀환/정리 | 5분 | 포탈로 허브 복귀, 장비 정렬, 오렌 의뢰 확인 | 다음 목표 설정 |

### 4-3. 장기 진행 곡선 (Long-term Progression)

**추정 콘텐츠 기간:**

| 목표 | 시간 | 에르다의 감정 아크 |
| :--- | :--- | :--- |
| Critical Path만 클리어 (T2→T7) | 3-4시간 | Act 1 코미디 → Act 2 전환 |

| 완전 정복 (계시 5 + 제3의 길) | 30-40시간 | 계시 5 + 엔딩 |

     ↓
[아이템계 진입 → 지층 클리어 → 무기 영구 강화]
     ↓

## 6. 검증 체크리스트 (Acceptance Criteria)

### 기능적 검증

- [ ] 모든 시드에서 숲길 → 심연의 구(T7)까지 Critical Path로 도달 가능한가?
- [ ] 각 격벽이 해당 능력 없이는 통과 불가능한가?
- [ ] 능력 게이트 배치가 각 능력의 획득 위치(이전 층위 보스)와 일치하는가?
- [ ] 스탯 게이트가 Critical Path(수직 하강 경로)에 배치되지 않았는가?
- [ ] 모든 필드 균열 제단에서 진입 → 해당 제단 복귀가 작동하는가?
- [ ] 기억의 방랑자 임시 제단에서 진입 → 해당 위치 복귀가 작동하는가?
- [ ] 포탈/제단을 통한 허브 즉시 귀환이 모든 층위에서 작동하는가?
- [ ] T1(천공의 정원)이 역중력 획득 전에는 접근 불가능한가?

### 내러티브 검증

- [ ] 각 층위의 보스가 해당 층위의 역사에서 비롯된 존재인가?
- [ ] 마르타의 흔적 7개가 순서 무관하게 독립적으로 의미가 있는가?
- [ ] 에르다의 감정 아크(코미디 → 전환 → 진지)가 층위 하강과 자연스럽게 정렬되는가?
- [ ] 계시 1-5의 단서가 적절한 층위에 배치되어 있는가?

### 경험 검증

- [ ] 하강할수록 "더 깊고 위험한 곳으로 내려가고 있다"는 긴장감이 느껴지는가?
- [ ] 스탯 게이트에서 막혔을 때 "아이템계를 더 돌면 된다"는 인식이 자연스러운가?
- [ ] 마르타의 흔적 발견이 하강의 추가 동기로 작동하는가?
- [ ] 후반부 T1 상승이 "반전"으로 느껴지는가?

## 7. 예외 처리 (Edge Cases)

| 상황 | 처리 |
|:-----|:-----|
| Critical Path 단절 (절차적 생성 실패) | Room Grid 생성 후 Critical Path 검증 알고리즘을 실행한다. 시작점→종료점 경로가 존재하지 않으면 시드를 변경하여 재생성한다. 최대 3회 재시도 후 실패 시 Fallback 템플릿(검증 완료된 고정 배치)을 사용 |
| 시드 충돌 (동일 시드에서 다른 결과) | 시드는 `hash(itemID + itemLevel + stratumNumber)` 결정론적 공식으로 생성한다. 플랫폼 간 부동소수점 차이로 인한 충돌 시 정수 연산만 사용하는 해시 함수로 보장. 서버가 시드를 최종 검증 |
| 방 연결 실패 (출입구 불일치) | Room Template의 출입구 메타데이터(상/하/좌/우 Door 태그)를 사전 검증한다. 인접 Room 간 Door 태그가 쌍으로 일치하지 않으면 해당 연결을 재배치한다. 재배치 불가 시 해당 Room을 삭제하고 대체 Room 삽입 |
| 층위 간 격벽에서 능력 미획득 상태로 접근 | 격벽 오브젝트가 능력 게이트 판정을 수행한다. 능력 미보유 시 통과 불가 + UI 안내("이단 점프 능력이 필요합니다"). 우회 경로 진입 시도 시에도 동일한 격벽이 존재하여 스킵 불가 |
| 워프/귀환 중 월드 상태 불일치 | 워프 실행 시 현재 Room 상태를 서버에 저장한 후 목적지로 전환한다. 전환 중 접속 끊김 시 마지막 저장된 세이브 포인트로 복원. Room 내 처치한 적/획득한 아이템은 서버 기록 기준으로 판정 |
