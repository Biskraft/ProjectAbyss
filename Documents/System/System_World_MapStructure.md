## 1. 개요 (Concept)

### 1-1. 설계 의도 (Design Intent)

Project Abyss의 월드는 **대장장이 에르다의 여정**을 물리적 공간으로 구현한 것이다. 에르다는 성채 외곽 대장간에서 출발하여, 심연 전쟁의 상흔이 남은 7개 구역을 탐험하며 스승 마르타의 흔적을 쫓고, 최종적으로 심연의 구에서 세계의 진실과 마주한다.

이 구조는 세 가지 설계 목표를 동시에 달성해야 한다:

1. **메트로베니아 비선형성**: 플레이어가 탐험 순서를 결정하되, 어떤 순서를 택해도 반드시 깰 수 있는 구조 (Always Winnable)
2. **에르다의 감정 아크**: 코미디(의뢰 수행) → 전환(기억의 진실 목격) → 진지(마르타의 흔적 추적)가 구역 진행과 함께 자연스럽게 전개
3. **순환 구조 연결**: 월드 탐험 → 아이템 획득 → 제단에서 에코로 두드려 아이템계 진입 → 장비 강화 → 스탯 게이트 해금 → 새 구역 진입

### 1-2. 설계 근거 (Reasoning)

| 결정 | 근거 |
| :--- | :--- |
| 숲길 → 대장간 → 중앙 성채 시작 동선 | 첫 30분 플로우(`CNT-EXP-001`)에서 에르다는 숲길을 걸어 대장간에 도착하고, 에코 공명 사건 후 허브(중앙 성채)로 이동. 게임 시작점이 허브가 아니라 숲길 |
| 허브 & 스포크 + 원형 순환 토폴로지 | 월하의 야상곡의 선형 구역 연결과 할로우 나이트의 극도의 비선형성 사이의 중간 지점. 충분한 자유도를 제공하면서도 방향성을 유지 |
| 7개 구역 = 심연 전쟁의 7개 흔적 | 각 구역은 단순 바이옴이 아니라 심연 전쟁의 상흔이 남은 역사적 장소. World Bible Layer 1~3과 직접 연결 |
| 이중 게이트 (능력 + 스탯) | 능력 게이트만으로는 아이템계 진입 동기가 약함. 스탯 게이트가 "무기를 두드려 강화하러 가자" 순환을 자연스럽게 창출 |
| 구역별 마르타의 흔적 배치 | 각 구역에 마르타가 과거에 방문했던 흔적(작업 자국, 메모, 수리 흔적)이 있다. 에르다의 장기 동기이자 세계 탐험의 내러티브 보상 |

### 1-3. 3대 기둥 정렬 (Pillar Alignment)

| 기둥 | 월드 맵 구조에서의 구현 |
| :--- | :--- |
| **탐험가** | 능력 게이트가 세계 진행을 정의. 각 능력 획득 시 접근 가능한 구역이 확장. 마르타의 흔적이 탐험의 내러티브 보상 |
| **장인** | 스탯 게이트가 아이템계로의 자연스러운 통로. "저 벽을 부수려면 장비가 더 강해야 한다" → 근처 필드 제단에서 무기를 두드려 아이템계로 |
| **모험가** | 허브(중앙 성채)가 모든 구역의 중심. 멀티플레이는 아이템계에서, 월드는 솔로의 고독한 발견 |

### 1-4. 저주받은 문제 (Cursed Problems)

| 문제 | 상충하는 요구 | 설계 해법 | 희생/제약 |
| :--- | :--- | :--- | :--- |
| 자유로운 탐험 vs 내러티브 진행 | 플레이어가 원하는 순서로 탐험해야 하는데, 에르다의 감정 아크는 방향성이 있음 | 각 구역에 독립적인 서사 단편 배치. 순서 무관하게 각 구역이 완결된 이야기를 가짐. 마르타의 흔적만 누적되어 감정 곡선 형성 | 특정 순서에서만 최적인 감정 경험이 존재하지만, 다른 순서에서도 서사적으로 파탄나지 않음 |
| 파밍 효율 vs 탐험 동기 | 아이템계에서 무한 파밍하면 월드 탐험을 하지 않으려 함 | 스탯 게이트만 파밍으로 해결 가능. 능력 게이트는 월드에서만 획득. 마르타의 흔적은 월드에서만 발견 | 플레이어는 양쪽 콘텐츠를 모두 경험해야 함 |
| 절차적 생성의 반복감 vs 역사적 장소감 | 절차적 생성하면 "심연 전쟁이 여기서 일어났다"는 장소 정체성 손실 위험 | 구역별 핵심 랜드마크(보스방, 마르타 흔적, 서사 오브젝트)는 핸드크래프트 고정. Room 내부 Chunk만 절차적 변화 | 일부 구역에서 반복 구조감 잔존 |
| 필드 제단 vs 허브 중심 설계 | 필드 균열 제단에서 아이템계 진입 → 현장 복귀하면 허브를 거치지 않음 | 필드 제단은 진입+강화만 가능. 이노센트 복종/합성 등 고급 기능은 허브 대장간에서만 가능 | 필드에서 빠른 강화는 가능하지만, 심화 강화는 허브 방문 필요 |

## 2. 핵심 규칙 (Core Rules)

### 2-1. 월드 토폴로지 (World Topology)

```
                      [숲길] (게임 시작 — 에르다의 출근길)
                         |
                      [대장간] (에코 공명 사건 발생)
                         |
      [마법 연구소] ─── [중앙 성채] ─── [천공의 탑]
      (기억 연구원        (허브)        (관측자들의
       본거지 폐허)         |            봉쇄된 탑)
                      [지하 수로]
                      (전쟁 전 생명의
                       인프라 잔해)
                         |
                      [묘지/카타콤]
                      (수호단 최후의
                       방어선 잔해)
                         |
                      [심연의 구]
                      (카엘 오르스가
                       뛰어든 곳)
```

**원형 순환 (Circular Loop) 구조:**
- **숲길 + 대장간**: 게임 최초 시작 전용. 첫 30분 플로우 이후에는 대장간 ↔ 중앙 성채를 자유 이동
- **중앙 성채 (Central Keep / 허브)**: 번영기의 세계 의회 "Concordia". 심연 전쟁 중 유일하게 함락되지 않은 중립지. 모든 구역의 중심
- **인접 구역 (Tier 1)**: 중앙 성채와 직접 연결. 마법 연구소, 빙결 동굴(미표시 — 중앙 성채 서쪽), 천공의 탑, 지하 수로
- **심연 구역 (Tier 2)**: 묘지/카타콤과 심연의 구. 여러 경로를 통해 도달

**경로 선택의 자유도:**
- Tier 1 구역 4개를 자유로운 순서로 탐험 가능
- 묘지/카타콤은 최소 2개 Tier 1 구역 클리어 후 접근 가능
- 심연의 구는 4개 Tier 1 구역 모두 클리어 또는 특정 조건 만족 후 접근 가능

### 2-2. 구역별 정체성 — 세계관 연결 (Zone Identity)

> 각 구역은 단순한 바이옴이 아니라 **심연 전쟁(Era 3)의 상흔이 남은 역사적 장소**다. `Content_World_Bible.md` Layer 1~3 참조.

| 구역 | 번영기(Era 2) 용도 | 전쟁 후(Era 4) 상태 | 에르다에게 의미하는 것 | 진입 조건 |
| :--- | :--- | :--- | :--- | :--- |
| **숲길** | 성채 외곽 삼림 | 몬스터 서식 시작. 관리 방치 | 출근길 (일상 세계) | 게임 시작 |
| **대장간** | — | 마르타 → 에르다가 물려받은 독립 공방 | 에코 공명 사건의 장소 (소명) | 숲길 통과 |
| **중앙 성채** | 세계 의회 (Concordia) | 허브. 장인 길드, 수호회, 상점, 기억의 사서 | 의뢰를 받고 돌아오는 거점 | 대장간 통과 |
| **지하 수로** | 생명의 수로 (Veinway) | 방치 후 미지의 생물 서식. 물 공급 일부 기능 | 첫 본격 탐험. 수중 환경 학습 | 기본 능력 |
| **마법 연구소** | 기억 연구원 (Memoria Arcana) | 해방파 본거지 폐허. 폭주 실험체 배회 | 아르카나 리브르의 연구 기록. "아이템계란 무엇인가"의 단서 | 기본 능력 |
| **빙결 동굴** | 냉동 보존소 (Cryo-Archive) | 냉동 시스템 오작동. 극저온+해동 혼재 | 소실 위기의 고대 기억 복원. 마르타의 수리 흔적 | 기본 능력 |
| **천공의 탑** | 관측탑 (Augurium) | 100년 전 봉쇄. 최근 탑 꼭대기에서 빛 관측 | 관측자들의 마지막 예언. 세계의 진실에 가까운 단서 | 기본 능력 + STR 게이트 10 |
| **묘지/카타콤** | 추모원 (Memoria) | 전쟁 사상자 무질서 매장. 원혼 소용돌이 | 수호단 마지막 대장의 기억. "기억을 지키는 것"의 의미 | 2개 Tier 1 클리어 |
| **심연의 구** | 밀봉지 (Sigil Point) | 협약 이후 불안정. 최근 침식 재개 | 카엘 오르스의 흔적. 계시 1~5. 제3의 길 | 4개 Tier 1 클리어 |

### 2-3. 마르타의 흔적 시스템 (Marta's Trail)

> 에르다의 스승 마르타 벤-나흐트는 10년 전 사라지기 전에 여러 구역을 방문했다. 각 구역에 마르타가 남긴 흔적이 1개씩 있으며, 이것이 에르다의 장기 탐험 동기이자 감정 아크의 구동력이다.

| 구역 | 마르타의 흔적 | 에르다의 반응 (감정 아크) | 발견 조건 |
| :--- | :--- | :--- | :--- |
| **대장간** | 에코 공명 후 드러나는 쪽지 | "스승님도 이걸 알고 있었어?" (호기심) | 첫 30분 내 자동 |
| **지하 수로** | 수로 벽에 남은 망치 자국과 수리 흔적 | "스승님이 여기도 왔었구나. 이 수로를 고치려 한 건가?" (추적 시작) | 비경로 탐험 |
| **마법 연구소** | 아르카나 리브르의 연구실에 마르타의 메모 | "스승님이 기억 연구원의 기록을 읽었다? 뭘 알아낸 거지?" (의문 심화) | 보스 처치 후 |
| **빙결 동굴** | 냉동 보존소에서 마르타가 수리한 고대 아이템 | "이 아이템... 스승님이 고친 거야. 왜 여기까지 와서?" (직업적 공감) | 비경로 탐험 |
| **천공의 탑** | 관측자의 기록에 "벤-나흐트의 장인"이라는 언급 | "스승님의 이름이 여기 있어. 관측자들이 스승님을 알고 있었다?" (충격) | 보스 처치 후 |
| **묘지/카타콤** | 묘비 사이에 마르타가 새긴 에코의 각인 | "에코의 흔적이야. 스승님이 여기서 무언가를 하고 갔어." (확신) | 비경로 탐험 |
| **심연의 구** | 심연 입구에 에코와 동일한 공명 자국 | "스승님이 여기까지 왔다. 그리고... 들어갔다." (결심) | 최종 구역 진입 |

**설계 원칙:**
- 마르타의 흔적은 Critical Path에 배치하지 않는다 (선택적 발견)
- 발견 순서에 무관하게 각 흔적은 독립적으로 의미가 있다
- 모든 흔적을 모으면 마르타가 왜 Ancient 아이템 안으로 들어갔는지 추론 가능
- 흔적 발견 시 에르다의 독백만으로 전달. NPC 설명 없음

| **지하 수로** | 중앙 성채 남쪽 | 기본 능력 | 보스 처치 |
| **마법 연구소** | 중앙 성채 동쪽 | 기본 능력 | 보스 처치 |
| **빙결 동굴** | 중앙 성채 서쪽 | 기본 능력 | 보스 처치 |
| **천공의 탑** | 중앙 성채 북동쪽 | 기본 능력 + STR 10 | 보스 처치 |
| **묘지/카타콤** | 지하 수로 남쪽 | 2개 Tier 1 클리어 | 보스 처치 |
| **심연의 구** | 묘지 최심부 | 4개 Tier 1 클리어 또는 최종 능력 | 최종 보스 처치 |

### 2-5. Critical Path 정의

**기본 능력만으로 도달 가능한 경로:**

```
[숲길] → [대장간] → [에코 공명 사건] → [중앙 성채]
  ↓ (에르다의 첫 의뢰)
[지하 수로] 또는 [마법 연구소] 또는 [빙결 동굴] 진입
  ↓
[보스 처치, 능력 획득]
  ↓
[다음 구역 → 4개 Tier 1 클리어]
  ↓
[묘지/카타콤]
  ↓
[심연의 구] → [카엘 오르스와의 대면] → [선택]
```

**원칙:**
- Critical Path는 기본 능력(이단 점프, 벽 타기, 변신 없음)만으로 통과 가능해야 한다
- 능력 게이트와 스탯 게이트는 Critical Path에 배치하지 않는다
- 마르타의 흔적은 Critical Path 밖에 배치 (선택적 탐험 보상)
- 모든 시드에서 시작점 → 최종 보스 경로가 보장된다

### 2-6. 능력 게이트 (Ability Gate) 배치 규칙

| 능력 | 획득 위치 | 새로 접근 가능한 구역/구간 | 내러티브 의미 |
| :--- | :--- | :--- | :--- |
| **이단 점프** | 마법 연구소 보스 처치 | 마법 연구소 상단, 다른 구역 고도 높은 영역 | 기억 연구원이 연구한 "기억의 도약" 기술. 기억으로 도달할 수 없던 곳에 닿는다 |
| **벽 타기** | 빙결 동굴 보스 처치 | 빙결 동굴 절벽, 지하 수로 수직 통로 | 냉동 보존소의 벽에 새겨진 고대 부적에서 얻는 능력 |
| **안개 변신** | 묘지/카타콤 중반 보상 | 1타일 너비 틈새 진입 | 죽은 자들의 기억이 안개처럼 흐르는 카타콤에서 습득 |
| **수중 호흡** | 지하 수로 중반 이노센트 | 수중 통로 진입 | 생명의 수로에 깃든 물의 이노센트가 부여하는 능력 |
| **역중력** | 천공의 탑 보스 처치 | 천공의 탑 상단부, 심연의 구 역중력 구간 | 관측자들이 하늘의 기억을 읽기 위해 개발한 기술 |
| **심연 저항** | 최종 조건 | 심연의 구 심층 진입 | 에르다의 에코가 심연의 기억과 공명하여 얻는 최종 능력 |

**배치 원칙:**
- 능력 게이트는 구역 경계에만 배치 (구역 간 이동 차단)
- 같은 구역 내 두 번째 능력은 배치하지 않는다
- 능력 게이트 위치는 모든 시드에서 동일 (매크로 구조 고정)
- 각 능력은 해당 구역의 역사와 의미가 연결된다

### 2-7. 스탯 게이트 (Stat Gate) 배치 규칙

| 스탯 | 임계값 구간 | 배치 위치 | 목적 |
| :--- | :--- | :--- | :--- |
| **STR** | 10 ~ 20 ~ 30 | 각 구역 고급 보상 경로 | 높은 방어도의 적/물체 파괴 |
| **INT** | 10 ~ 20 ~ 30 | 마법 연구소 중심 | 마법 저항 적 처치, 마법 함정 해제 |
| **DEX** | 10 ~ 20 ~ 30 | 타이밍 기반 함정, 빠른 적 | 정밀 점프, 빠른 이동 구간 |
| **VIT** | 5 ~ 15 ~ 25 | 고난이도 적 밀집 구간 | 높은 피해 환경 생존 |
| **SPD** | 5 ~ 10 ~ 15 | 시간 제한 구간 | 타이밍 플랫포밍, 시간 챌린지 |
| **LCK** | 10 ~ 20 ~ 30 | 숨겨진 경로 | 숨겨진 아이템/이노센트/마르타 흔적 발견 |

**배치 원칙:**
- 스탯 게이트는 구역 내부 비경로 탐험 구간에 배치
- Critical Path 위에 스탯 게이트를 배치하지 않는다 (진행 보장)
- 각 구역마다 3~5개의 스탯 게이트
- 스탯 게이트 임계값은 구역 난이도에 비례
- **LCK 게이트 뒤에 마르타의 흔적이 배치될 수 있다** — 탐험적 플레이어에 대한 보상

### 2-8. 보스 배치 규칙 (Boss Placement Rules)

| 구역 | 진행 보스 | 내러티브 정체 | 숨겨진 보스 |
| :--- | :--- | :--- | :--- |
| **중앙 성채** | 없음 (Hub) | — | 아르카나 리브르의 환영 (선택) |
| **지하 수로** | 수로의 파수꾼 | 전쟁 중 수로 방어를 위해 만들어진 골렘. 주인이 사라진 후에도 경비 중 | 물의 기억 |
| **마법 연구소** | 폭주한 실험체 | 기억 연구원들의 실험이 전쟁 후 통제를 벗어난 존재 | 마법의 정령 |

| **심연의 구** | — | — | 카엘 오르스의 기억 (최종 대면) |

**보스 설계 원칙:**
- 모든 보스는 그 구역의 역사에서 비롯된 존재여야 한다 (generic 몬스터 금지)
- 보스 처치 시 해당 구역의 역사 단편이 환경 연출로 전달된다
- 숨겨진 보스는 비경로 탐험 또는 특정 능력 조건 달성 후 진입 가능
- 최종 대면(카엘 오르스)은 전투가 아닌 선택으로 끝날 수 있다

## 4. 흐름 (Flow)

### 4-1. 에르다의 여정 흐름도 (Erda's Journey Flowchart)

```
[숲길 — 일상 세계]
    ↓
[대장간 — 에코로 의뢰 검을 두드림 → 첫 균열 (소명)]
    ↓
[아이템계 첫 진입 (시련)]
    ↓
[대장간 복귀 — 거부 + 마르타 쪽지 (스승)]
    ↓
[중앙 성채 — 오렌 만남, 첫 의뢰 (관문 통과)]
    ↓
[Tier 1 구역 자유 탐험]
    ├─→ [지하 수로] ── 수중 호흡 획득 + 마르타 수리 흔적 + 필드 제단 발견
    ├─→ [마법 연구소] ── 이단 점프 획득 + 마르타 메모 + 필드 제단 발견
    ├─→ [빙결 동굴] ── 벽 타기 획득 + 마르타가 수리한 고대 아이템 + 필드 제단 발견
    └─→ [천공의 탑] ── 역중력 획득 + 마르타의 이름 발견 + 필드 제단 발견
    ↓
    ↓ (각 구역에서 필드 제단 발견 → 허브 안 돌아가도 아이템계 진입 가능)
    ↓ (기억의 방랑자 랜덤 출현 → 보상 강화 아이템계)
    ↓ (장비 강화 → 스탯 게이트 해금)
    ↓
[묘지/카타콤 — 안개 변신 + 수호단의 기억]
    ↓ (에르다: "기억을 지킨다는 건 이런 뜻이었구나")
    ↓
[심연의 구 — 최종 구역]
    ↓ (카엘 오르스의 기억과 대면)
    ↓ (계시 1~4 수집)
    ↓
[선택: 협약 유지 / 협약 파기 / 제3의 길]
    ↓ (계시 5: 이 세계도 무기의 내부)
    ↓
[에르다: 망치 에코로 세계의 경계를 두드린다]
```

## 6. 검증 체크리스트 (Acceptance Criteria)

### 기능적 검증

- [ ] 모든 시드에서 숲길 → 심연의 구까지 기본 능력만으로 도달 가능한가?
- [ ] 4개 Tier 1 구역 진행 순서가 자유로운가?
- [ ] 능력 게이트 배치가 각 능력의 획득 위치와 일치하는가?
- [ ] 스탯 게이트가 Critical Path에 배치되지 않았는가?
- [ ] 모든 필드 균열 제단에서 진입 → 해당 제단 복귀가 작동하는가?
- [ ] 기억의 방랑자 임시 제단에서 진입 → 해당 위치 복귀가 작동하는가?

### 내러티브 검증

- [ ] 각 구역의 보스가 해당 구역의 역사에서 비롯된 존재인가?
- [ ] 마르타의 흔적 7개가 순서 무관하게 독립적으로 의미가 있는가?
- [ ] 에르다의 감정 아크(코미디 → 전환 → 진지)가 구역 진행과 자연스럽게 정렬되는가?
- [ ] 계시 1~5의 단서가 적절한 구역에 배치되어 있는가?

### 경험 검증

- [ ] 플레이어가 "다음에 어디를 탐험할지" 선택할 수 있는 자유도를 느끼는가?
- [ ] 스탯 게이트에서 막혔을 때 "아이템계를 더 돌면 된다"는 인식이 자연스러운가?
- [ ] 마르타의 흔적 발견이 탐험의 추가 동기로 작동하는가?
