## 1. 개요 (Concept)

### 1-1. 설계 의도 (Design Intent)

각 층위는 단순한 바이옴이 아니다. 층위는 에르다의 여정이 통과하는 **심연 전쟁의 상흔**이자, 플레이어가 이중 게이트(능력+스탯)를 통해 세계의 깊이를 발견하는 무대다. 세계는 수직 샤프트 구조로 7개 층위(Tier 1 천공의 정원 ~ Tier 7 심연의 구)가 위에서 아래로 쌓여 있다.

본 문서는 `System_World_MapStructure.md`가 정의한 토폴로지와 게이트 배치 규칙을 각 층위 단위로 구체화한다. 각 층위는 다음 5가지 요소를 반드시 갖는다.

1. **바이옴 정체성**: 시각·청각·환경 기믹으로 식별되는 고유 분위기
2. **난이도 곡선**: 층위 입구에서 보스방까지의 긴장 상승 패턴
3. **이중 게이트 배치**: 능력 게이트와 스탯 게이트의 정확한 위치와 조건
4. **내러티브 밀도**: 마르타의 흔적, 환경 서사, 보스 역사가 분포하는 방식
5. **리스크-리턴 구조**: 비경로 탐험에 비례하는 보상 체계

### 1-2. 설계 근거 (Reasoning)

| 결정 | 근거 |
| :--- | :--- |
| 각 층위마다 고유 기믹 1개 이상 | 월하의 야상곡이 층위별로 "이 층위에서만 경험하는 것"을 두어 탐험의 기억을 선명하게 만든 원칙. 빙결 보존소는 미끄러지는 바닥이 있고, 지하 수로는 물이 있다 |
| 마르타의 흔적은 Critical Path 밖 비경로 배치 | 탐험 동기를 내러티브로 강화하되, 강요하지 않는다. SDT의 자율성 원칙. 흔적을 발견하는 것은 보상이지 의무가 아니다 |
| 필드 균열 제단은 층위 후반부 배치 | 층위를 어느 정도 탐험한 뒤 아이템계로 빠지는 것이 자연스럽다. 층위 초입에 배치하면 탐험 없이 즉시 아이템계로 이탈할 수 있다 |
| 보스방은 층위 최심부에 고정 | 목표 방향성 제공. 플레이어는 "더 깊이 가면 보스"를 직관적으로 안다 |
| 각 층위 탐험 시간 20~30분 목표 | 45분 평균 세션에서 1층위 탐험 + 아이템계 1회로 세션이 완결되도록 설계 |

### 1-3. 3대 기둥 정렬 (Pillar Alignment)

| 기둥 | 층위 설계에서의 구현 |
| :--- | :--- |
| **탐험가** | 층위별 고유 기믹과 비경로 탐험 보상이 탐험 의욕을 지속적으로 자극한다. 마르타의 흔적이 발견의 감정 보상 역할 |
| **장인** | 각 층위에 필드 균열 제단 1~2개 배치. 탐험 중 획득한 장비를 현장에서 아이템계로 강화 가능. 스탯 게이트가 야리코미 동기를 제공 |
| **모험가** | 각 층위 보스는 협동 도전의 자연스러운 목표. 2인 탐험 전용 숨겨진 방/보상이 일부 층위에 배치 |

### 1-4. 저주받은 문제 (Cursed Problems)

| 문제 | 상충하는 요구 | 설계 해법 | 희생/제약 |
| :--- | :--- | :--- | :--- |
| 자유 탐험 vs 방향 제시 | 플레이어가 자유롭게 탐험해야 하는데, 길을 잃으면 좌절한다 | "비경로 탐험은 보상, Critical Path는 명확"의 원칙. 환경 시각 단서로 방향 암시 | 비경로를 너무 강조하면 Critical Path가 불분명해질 수 있다 |
| 층위 고유성 vs 절차적 반복성 | 절차적 생성하면 층위의 고유 분위기가 희석된다 | 매크로(보스방, 랜드마크, 마르타 흔적 위치)는 고정. 마이크로(Room 내부 Chunk)만 절차적 | 절차적 구간에서 장소성 약화는 불가피 |
| 난이도 자유 vs 적정 순서 | 자유 탐험 가능 층위를 자유롭게 탐험하지만, 순서에 따라 너무 쉽거나 어려울 수 있다 | 스탯 게이트 임계값을 자유 탐험 층위 내에서는 Critical Path 위에 배치하지 않는다. 비경로 탐험 보상에만 스탯 게이트 적용 | 일부 순서에서 보스가 비교적 쉬울 수 있다 |

### 1-5. 리스크-리턴 분석 (Risk & Reward)

| 행동 | 리스크 | 리턴 |
| :--- | :--- | :--- |
| Critical Path 진행 | 보스 전까지 누적되는 피해, MP 소모 | 층위 보스 처치, 능력 게이트 해금, 다음 층위 진입 |
| 비경로 탐험 | 사망 시 세이브 포인트 복귀, 시간 소모 | 마르타의 흔적, 희귀 아이템, 이노센트, 숨겨진 보스 |
| 스탯 게이트 돌파 | 아이템계 파밍 시간 투자 | 비경로 고급 보상 경로 해금 |
| 필드 균열 제단 사용 | 아이템계 진입 후 사망 시 진행 손실 | 현장 장비 강화, 이노센트 포획 |

## 2. 메커닉 (Mechanics)

### 2-1. 공통 층위 구성 요소

모든 층위는 다음 구성 요소의 조합으로 설계된다.

| 구성 요소 | 수량 | 배치 원칙 | 구현 방식 |
| :--- | :---: | :--- | :--- |
| 입구 Room | 1 | Critical Path 시작점. 능력/스탯 게이트가 입구 바로 위 또는 아래 | 핸드크래프트 고정 |
| Critical Path Room | 층위별 상이 | 입구 → 보스방 최단 가용 경로 | 절차적 생성 + 고정 와이어프레임 |
| 비경로 Room | Critical Path의 30~50% | 탐험 보상 배치. 스탯 게이트 뒤에 배치 가능 | 절차적 생성 |
| 보스 전실 Room | 1 | 보스방 직전 안전 구역. 회복 가능 | 핸드크래프트 고정 |
| 보스방 (Boss Room) | 1 | 층위 최심부 고정 | 핸드크래프트 고정 |
| 세이브 포인트 | 2~3 | Critical Path 위. 보스 전실 포함 | 고정 배치 |
| 필드 균열 제단 | 1~2 | 층위 중반 이후 (Room Grid 60% 지점 이후) | 고정 배치 |
| 마르타의 흔적 | 1 | Critical Path 밖 비경로. LCK 게이트 뒤 가능 | 고정 배치 |
| 랜드마크 | 1~2 | 층위 정체성을 시각적으로 표현하는 특수 방 | 핸드크래프트 고정 |
| 숨겨진 보스방 | 0~1 | 조건 달성 후 진입 (능력 게이트 또는 아이템 조건) | 핸드크래프트 고정 |

### 2-2. 층위별 고유 기믹 시스템

각 층위는 해당 층위에서만 작동하는 **고유 환경 기믹**을 1개 이상 보유한다. 기믹은 전투와 탐험 양쪽에 적용된다.

| 층위 | 고유 기믹 | 전투 영향 | 탐험 영향 |
| :--- | :--- | :--- | :--- |
| 숲길 | 울창한 숲, 시야 제한 | 원거리 적이 나무 뒤에서 기습 | 비선형 분기가 많아 은닉 경로 존재 |
| 중앙 성채 | 허브. 전투 없음 | — | NPC 상호작용 중심 |
| 지하 수로 | 수중 구간, 조류 흐름 | 수중에서 이동속도 제한, 수중 적 등장 | 수중 통로에 숨겨진 방 다수 |
| 마법 연구소 | 기억 잔상 함정, 마법 장벽 | 기억 실체화 적 등장 | 마법 장벽은 INT 게이트 또는 아이템으로 해제 |
| 빙결 동굴 | 미끄러지는 빙판, 냉기 대미지 구간 | 빙결 상태이상, 해동 적 | 빙판 슬라이딩으로 통과 가능한 숨겨진 경로 |
| 천공의 탑 | 강풍 구간, 이동 방해 | 날아오는 풍압 공격 | 강풍이 없는 구간에서만 정밀 점프 가능 |
| 묘지/카타콤 | 어둠 구간 (시야 50% 제한), 원혼 소환 | 어둠 속 원혼 증원 | 횃불 아이템 또는 안개 변신으로 은밀 통과 |
| 심연의 구 | 심연 침식 타일 (밟으면 서서히 붕괴), 역중력 구간 | 발판 소멸, 심연 몬스터 | 균열 타이밍 점프가 경로의 일부 |

### 2-3. 난이도 곡선 패턴 (Sawtooth Model)

각 층위 내부의 난이도는 `LevelDesign_ProgressionShape_Research.md`의 **톱날(Sawtooth) 패턴**을 따른다.

```
층위 내 긴장도
  높음 │
       │           [보스방]
       │       ╱       │
       │    ╱─╲    ╱───┘
       │  ╱    ╲╱
       │╱
  낮음 └──────────────────
       입구   중반   최심부
```

- **입구~첫 세이브 포인트**: 낮은 위협. 층위 적과 기믹 소개
- **첫 세이브~중반**: 기믹 복합 사용. 적 조합 다양화
- **중반 세이브~보스 전실**: 긴장 최고조. 복합 기믹 + 밀집 적 배치
- **보스 전실**: 완전 이완. 회복 가능, 안전 지대
- **보스방**: 층위 기믹의 종합 도전

#### 3-9. 층위 규칙

중앙 성채는 World 공간이지만 **전투가 없는 예외 층위**이다. 3-Space 모델에서 Hub 기능을 담당한다.

| 기능 | 상세 |
| :--- | :--- |
| 세이브 포인트 대장간 | 에코로 장비를 두드려 아이템계 진입. 이노센트 복종/합성은 여기서만 가능 |

| 기억의 사서 | 세계관 단편, 발견한 마르타 흔적에 대한 맥락 제공 |
| 워프 포인트 | 중앙 성채에서 탐험한 층위 입구로 빠른 이동 가능 |

#### 3-10. 랜드마크

- **Concordia 의회 홀**: 번영기 세계 의회 건물. 현재는 장인 길드가 사용 중. 거대한 원형 홀에 각 층위 대표의 빈 의자가 남아 있다
- **기억의 탑**: 기억의 사서가 있는 탑. 발견한 아이템계 기억 파편이 이곳에 수집되어 세계관을 시각화
- **야전 대장간 광장**: 에르다의 허브 대장간이 있는 광장. 성채 중심부. 다른 플레이어들의 장비를 볼 수 있는 소셜 공간
- **전쟁 기념비**: 심연 전쟁 희생자들의 이름이 새겨진 거대 석판. 카엘 오르스의 이름은 없다 — 에르다가 나중에 이를 이상하게 여긴다

#### 3-44. 난이도 곡선

묘지/카타콤은 **어둠 기믹**과 **안개 변신 게이트**가 핵심이다. 상층 묘지(Tier 2 허브 바로 아래)와 하층 카타콤(심화 탐험)으로 2단 구조다.

| 구간 | 공간 패턴 | 적 유형 | 기믹 |
| :--- | :--- | :--- | :--- |
| 지상 묘지 | `open` — 개방 묘지 | 원혼 정령 | 낮 시간. 시야 제한 없음. 예비 긴장 |
| 묘지 중심 | `network` — 묘비 미로 | 수호단 졸병 언데드 | 묘비가 시야 차단. 원거리 적 위협 증가 |
| 카타콤 입구 | `funnel` — 좁아지는 통로 | 원혼 중형 | 어둠 기믹 시작. 시야 50% 제한 |
| 카타콤 심부 | `loop` — 복잡한 지하 통로 | 수호단 정예 언데드 | 안개 변신 게이트 다수. 좁은 틈새 |
| 카타콤 최심부 | `arena` — 지하 의식 홀 | 수호단의 잔영 (보스) | 완전 어둠 + 원혼 소환 페이즈 |

#### 3-45. 게이트 배치

| 게이트 유형 | 위치 | 조건 | 뒤에 있는 것 |
| :--- | :--- | :--- | :--- |
| 능력 게이트 (안개 변신) | 카타콤 심부 다수 위치 | 카타콤 중반 보상으로 획득 | 카타콤 심층 전체. 1타일 너비 틈새 통과 |
| 스탯 게이트 (STR 15) | 지상 묘지 봉인된 석묘 | 두꺼운 석관 파괴 | 수호단 보물고 (Legendary 아이템) |
| 스탯 게이트 (INT 15) | 카타콤 중반 마법 봉인 | 마법 봉인 비문 해석 | 수호단 역사 기록실. 마르타 흔적 접근 |
| 스탯 게이트 (LCK 15) | 카타콤 완전 어둠 비경로 | 거의 보이지 않는 숨겨진 구멍 | 마르타의 흔적 위치 |

#### 3-46. 안개 변신 획득 방식 (카타콤 전용)

카타콤에서 안개 변신 능력은 다른 층위의 보스 처치 보상과 다르게 획득한다. 카타콤 **중반 탐험 보상**으로 얻는다.

| 단계 | 내용 |
| :--- | :--- |
| 1. 카타콤 중반 비경로 방 탐험 | 수호단 최후 대장이 남긴 유물 방 발견 |
| 2. 유물 조사 | 에르다: "죽은 자들의 기억이 안개처럼 흐르는 곳... 이걸 배울 수 있을까?" |
| 3. 안개 변신 해금 | 에르다가 카타콤의 안개 기억을 흡수하여 능력 획득. 보스 처치와 무관 |
| 4. 즉시 활용 가능 | 남은 카타콤 구간에서 안개 변신이 탐험에 필수적으로 사용됨 |

#### 3-47. 마르타의 흔적

| 항목 | 내용 |
| :--- | :--- |
| 위치 | LCK 15 게이트 통과 후 카타콤 완전 어둠 구간의 묘비 하나 |
| 내용 | 일반 묘비들 사이 하나에 에코의 각인이 새겨져 있다. 마르타가 에코로 직접 새긴 흔적 |
| 에르다 반응 | "에코의 흔적이야. 스승님이 여기서 무언가를 하고 갔어." (확신 단계. 마르타가 무언가를 기억하려 했다는 확신) |
| 발견 조건 | 안개 변신 획득 + LCK 15 스탯 게이트 통과 |

## 5. 연동 (Integration)

### 5-1. System_World_MapStructure.md와의 연동

본 문서는 `System_World_MapStructure.md` (SYS-WLD-01)의 **마이크로 계층 구현 명세**다. 두 문서의 역할 분담은 다음과 같다.

| 항목 | System_World_MapStructure.md | System_World_ZoneDesign.md (본 문서) |
| :--- | :--- | :--- |
| 층위 토폴로지 | SSoT — 층위 간 수직 연결 그래프 정의 | 참조만 |
| 게이트 배치 규칙 | SSoT — 배치 원칙과 임계값 정의 | 층위별 구체적 위치와 뒤에 있는 것 정의 |
| 마르타 흔적 시스템 | SSoT — 시스템 규칙과 감정 아크 | 층위별 구체적 위치, 내용, 에르다 반응 정의 |
| 보스 배치 규칙 | SSoT — 보스 종류와 서사 정체 | 층위별 전투 설계와 연출 정의 |
| 층위 크기/시간 | SSoT — 수치 정의 | 참조만 |

### 5-2. System_World_ProcGen.md와의 연동

| 항목 | 본 문서 역할 | ProcGen 문서 역할 |

각 층위의 필드 균열 제단은 `System_ItemWorld_Core.md`의 진입 규칙을 따른다. 층위 설계에서 정의하는 것은 **제단의 위치와 서사적 맥락**이며, 아이템계 내부 규칙은 ItemWorld 문서가 정의한다.

| 층위 제단 | 서사적 맥락 | 아이템계 진입 규칙 |
| :--- | :--- | :--- |
| 대장간 제단 | 에코 공명 사건으로 처음 발견 | `System_ItemWorld_Core.md` 참조 |
| 지하 수로 제단 | 전쟁 중 야전 대장간 폐허 | 동일 |
| 마법 연구소 제단 | 기억 연구원의 아이템계 실험 장치 | 동일 |
| 기타 층위 제단 | 각 층위 역사와 연결된 폐허 | 동일 |

## 6. 예외 처리 (Edge Cases)

| 상황 | 처리 |
|:-----|:-----|
| 층위 간 이동 중 접속 끊김 | 층위 전환은 서버 확인 후 실행한다. 전환 시작 시 출발 층위 상태를 저장하고, 목적지 층위 로드 완료 응답을 받은 후 플레이어를 이동시킨다. 응답 없이 타임아웃(5초) 시 출발 층위의 마지막 세이브 포인트로 복원 |
| 능력 게이트 보스 회피 (보스를 처치하지 않고 능력 획득 시도) | 능력은 반드시 해당 층위 보스 처치 보상으로만 획득된다. 보스방 이외의 경로로 능력을 얻는 방법은 존재하지 않는다. 보스방 입장 시 후방 문이 잠기며, 보스 처치 전까지 탈출 불가 |
| 환경 기믹 판정 실패 (빙판 슬라이딩으로 구멍 추락 등) | 추락 구간에 Kill Zone을 배치한다. Kill Zone 접촉 시 HP의 20%를 소모하고 최근 안전 지점(Room 입구 또는 발판)으로 리스폰. HP 0 이하 시 사망 처리 |
| 환경 대미지 구간에서 VIT 부족 (냉기/독 등) | 환경 대미지는 초당 고정 HP 감소이다. VIT가 없어도 Critical Path 통과에 필요한 최소 시간 내 사망하지 않도록 대미지를 설정한다. 비-크리티컬 경로의 고강도 환경 구간만 VIT 요구 |
| 동일 층위 재방문 시 절차적 생성 결과 변경 | 월드의 매크로 구조(층위 배치, 격벽, 보스방, 랜드마크)는 핸드크래프트 고정이므로 변경 불가. 마이크로 구조(Room Grid 내 Chunk 배치)는 서버 고정 시드로 생성되므로 동일 시드에서는 동일 결과 보장 |
