## 1. 개요 (Concept)

### 1.1. 설계 의도 (Intent)

> **"이노센트는 아이템 내부에 거주하는 주민이다. 야생 상태에서는 절반의 힘만 발휘하지만, 아이템계에서 격파하여 복종시키는 순간 효과가 2배가 되고, 그 순간부터 다른 아이템으로 옮길 수 있으며, 같은 종류와 합성하여 한계 없이 강해질 수 있다. 이노센트는 야리코미의 또 다른 레이어다."**

이노센트 코어 시스템은 아이템계 내에서 플레이어가 만나는 **파밍 상대**이자, 장비 강화의 **핵심 성장 엔진**이다. 디스가이아의 이노센트/스페셜리스트 시스템을 실시간 액션 + 메트로베니아 + 협동 멀티플레이에 맞게 적응시킨다.

에르다에게 이노센트 정련(복종)은 대장장이의 행위와 같다 — 혼돈의 재료를 두드려 쓸 수 있는 형태로 만드는 것.

### 1.2. 설계 근거 (Reasoning)

| 설계 결정 | 채택 이유 | 기각된 대안 |
| :--- | :--- | :--- |
| **야생 → 복종 이분법** | 격파(리스크)에 따른 2배 보상(리턴)이 아이템계 진입 동기. 복종 후 다른 아이템 이식 가능이 수집 목적 창출 | "처음부터 복종" — 격파 긴장감 소실, 야리코미 동기 약화 |
| **슬롯 제약 (Normal 2~Ancient 8)** | 제한된 슬롯이 "어떤 이노센트를 선택할 것인가"라는 전략적 판단을 유발. 장비의 정체성 형성 | 슬롯 무제한 — 모든 보너스를 스태킹하는 최적해 존재, 선택 압박 소실 |
| **단순 합성 규칙 (레벨 합산)** | 저레벨 이노센트 여러 개를 모아 합성하는 반복이 "수집의 보상 순간" 형성 | "비용-효율 합성" — 계산 복잡화, 취향 선택보다 수학적 최적화 |
| **6종 타입 (+ Phase 2 확장)** | MVP 단계에서 핵심 이노센트만 정의. 데이터 구조는 무제한 확장성 보장 | "모든 타입 한 번에" — 밸런싱 불가, 초기 테스트 혼란 |

### 1.3. 3대 기둥 정렬 (Pillar Alignment)

| 기둥 | 이노센트 시스템에서의 구현 |
| :--- | :--- |
| **메트로베니아 탐험** | 이노센트 ATK/INT 합산으로 장비 스탯 향상 → 월드 ATK/INT 게이트 해금. 더 높은 이노센트를 얻기 위해 더 깊은 지층으로 도전 |
| **아이템계 야리코미** | 야생 이노센트 격파가 아이템계 진입의 또 다른 목표. 복종 2배 보상 + 합성을 통한 무한 성장이 파밍 동력 |
| **온라인 멀티플레이** | 깊은 지층의 강화된 이노센트는 파티 협력으로 획득. 이노센트 완성도가 팀의 전투력을 좌우하는 사회적 동기 |

### 1.4. 저주받은 문제 검증 (Cursed Problem Check)

| 긴장 | 위험 A | 위험 B | 설계의 선택 |
| :--- | :--- | :--- | :--- |
| **복종 리스크 vs 보상** | 복종 전투 난이도 높음 → 진입 기피 | 복종 너무 쉬움 → 성취감 소실 | 보스 전투와 동등한 난이도. 복종 시 즉각 2배 체감 |
| **합성 인플레 vs 성장 천장** | 무한 합성으로 인플레 발생 | 캡 너무 낮으면 야리코미 이탈 | 시리즈별 소프트 캡 설정. Phase 2에서 경제 설계 확정 |
| **슬롯 부족 vs 복종 동기** | 슬롯 꽉 차면 새 이노센트 거부 | 슬롯 많으면 선택 의미 소실 | 레어리티별 선택지: 모든 슬롯 채우거나 최적화할 것인가 |
| **야생 절반 효과 vs 초보자 불편** | 초보는 야생만으로 충분할 수도 | 절반은 너무 낮은 페널티인가 | 스탯 게이트는 복종 강제 안 함. 자발적 선택이 야리코미 진입 |

## 2. 핵심 규칙 (Core Rules)

### 2.1. 야생 vs 복종 상태

#### 야생 상태 (Unsubdued)
- **초기 상태:** 아이템 획득 시 모든 이노센트는 야생
- **효과 배율:** 보너스 50% 적용 (절반 효과)
- **제약:** 다른 아이템으로 이식 불가. 같은 종류끼리 합성 불가
- **아이템계 역할:** 아이템계 내 적 NPC로 등장하여 플레이어를 공격
- **표시:** UI에서 노란 느낌표 마크

#### 복종 상태 (Subdued)
- **획득 조건:** 아이템계 내에서 해당 이노센트 적 NPC를 격파
- **즉각 효과:** 복종 시점에 이노센트 레벨 **2배** (예: 레벨 10 → 20)
- **해제 조건:** 이식 가능 (다른 아이템으로 옮김). 동종 합성 가능
- **보너스 배율:** 100% 적용 (전체 효과)
- **표시:** UI에서 초록 체크마크

### 2.2. 이노센트 슬롯 시스템

#### 레어리티별 슬롯 정의

| 레어리티 | 이노센트 슬롯 수 | 추천 야리코미 깊이 |
| :--- | :---: | :--- |
| Normal | 2 | 기본 (1~2명 솔로) |
| Magic | 3 | 초급 (협력 권장) |
| Rare | 4 | 중급 (2인 파티) |
| Legendary | 6 | 고급 (3~4인 파티) |
| Ancient | 8 | 극한 (4인 전체 협력) |

#### 슬롯 운영 규칙
- **슬롯 초과:** 모든 슬롯이 찬 상태에서 새 이노센트 복종 시 기존 것과 교체 또는 무시 선택
- **슬롯 효율:** 슬롯이 비어 있으면 스탯 손실. "이 슬롯에 뭘 넣을까"가 장비 설계의 핵심 선택

### 2.3. 이노센트 타입 (5분류 15종)

> **설계 변경:** ATK/INT/HP 3스탯 체계 전환에 따라 이노센트 분류를 전면 개편. 기본형 3종 + 행동형 6종 + 원소형 3종 + 파밍형 2종 + 서사형 1종 = 총 15종.
>
> **DEPRECATED:** Sentry (보초, DEF+) — DEF는 장비에서 직접 제공. 이노센트로 보정하지 않음.
> **DEPRECATED:** Collector (수집가) — Scavenger(수집가)에 통합. 드랍률 보정으로 기능 흡수.

#### 행동형 (Behavioral) — 6종

> **설계 의도:** 행동형 이노센트는 스탯 수치가 아닌 **플레이 방식**을 변화시킨다. 같은 무기라도 어떤 행동형 이노센트를 장착하느냐에 따라 전혀 다른 전투 경험이 된다.

| 타입명 | 효과 (복종) | 야생 = 50% | 플레이 변화 |
| :--- | :--- | :--- | :--- |
| Berserker (광전사) | 공격 속도 +15% | 7.5% | 빠른 콤보, 높은 DPS |
| Vampire (뱀파이어) | 적 처치 시 HP 5% 회복 | 2.5% | 공격적 플레이 = 생존 |
| Ironclad (철벽) | 넉백 -50% | -25% | 보스전 안정, 진지 사수 |
| Ghost (유령) | 대시 i-frame +50ms, 대시 거리 +20% | 50% | 회피 특화 |
| Sprinter (질주자) | 이동 속도 +15% | 7.5% | 탐험/스피드런 |
| Leech (거머리) | 공격 적중 시 HP 1 회복 (0.5초 쿨다운) | HP 0.5 | 보스 지구전 |

#### 원소형 (Elemental) — 3종

> **핵심 규칙: 원소 공격은 에코(Echo)의 인챈트로 가능하다.** 에르다가 인챈트 액션(↑+공격)으로 무기에 원소를 입히면 해당 무기의 모든 공격이 해당 원소 속성이 된다. 인챈트 미적용(무 속성) 시 물리 데미지만 발생한다. 무기 고유 속성으로는 원소 공격이 불가능하다. 인챈트 시스템 상세는 `System_Combat_Action.md` §2.2-A 참조.
>
> **원소 이노센트의 역할:** 원소 이노센트는 원소 공격의 조건이 아니라 **에코 인챈트 효과를 강화**하는 역할이다. 이노센트 없이도 인챈트로 원소 공격이 가능하지만, 해당 원소 이노센트 장착 시 데미지 배율과 상태이상 지속시간이 강화된다.
>
> **원소 적중률 100%:** 모든 원소 공격은 항상 적용된다 (확률 기반 아님). 데미지는 INT × ElementMultiplier로 결정되며, 상태이상(화상/빙결/감전)도 항상 발동한다. 상태이상의 지속시간/강도가 INT에 비례하여 스케일된다.
>
> **화(Fire) 인챈트 기본 해금:** 에코는 대장간 화로에서 단련된 망치로, 게임 시작부터 화(Fire) 인챈트가 사용 가능하다. 이로 인해 **게임 시작부터 화 원소 반응(거미줄 소각, 얼음 해빙 등)이 가능**하다. 이것은 BotW에서 횃불이 시작 장비인 것과 같은 원리 — 가장 직관적인 원소를 가장 먼저 경험하게 한다.

| 타입명 | 원소 | 효과 (복종) | 획득처 |
| :--- | :--- | :--- | :--- |
| Burner (버너) | 화 (Fire) | 화 인챈트 데미지 +20%. 화상 지속시간 +30% | **아이템계 랜덤 드랍 전용** |
| Freezer (프리저) | 빙 (Ice) | 빙 인챈트 데미지 +20%. 빙결 지속시간 +30% | **아이템계 랜덤 드랍 전용** |
| Shocker (쇼커) | 뇌 (Thunder) | 뇌 인챈트 데미지 +20%. 감전 지속시간 +30% | **아이템계 랜덤 드랍 전용** |

> **원소 이노센트는 월드 보스에서 확정 드랍되지 않는다.** 원소 해금은 에코가 보스 처치 시 원소를 흡수함으로써 이루어진다 (아래 에코 흡수 순서 참조). 이노센트는 해금된 인챈트의 효과를 강화하는 수단이며, 아이템계 랜덤 드랍으로만 획득한다.

**원소 이노센트 규칙:**
- 장비에 장착 시 해당 장비에 에코 인챈트 적용 중 해당 원소의 강화 효과 발동
- 복수 원소 이노센트 장착 시: 현재 인챈트 원소와 일치하는 이노센트만 효과 발동 (다른 원소 이노센트는 인챈트 전환 전까지 대기)
- 원소 이노센트는 야생 상태로 드랍. 복종 처리 필요.

| 순서 | 원소 | 에코 흡수 시점 | 새로 열리는 것 |

| 2 | 빙 (Ice) | 지하 수로 보스 | 물→얼음 변환, 용암 표면 동결, 화+빙 연쇄 |

- 빙 인챈트 (Freezer 이노센트로 강화 가능) → 물(ID 2) 동결, 용암(ID 13) 표면 동결
- 뇌 인챈트 (Shocker 이노센트로 강화 가능) → 물/얼음/산성 감전 확산(flood-fill)

### 2.3-N. 넘버 원 (No.1) — 특수 이노센트

**분류:** 서사 이노센트 (Story Innocent)

**외형:** 에르다의 해머 에코(Echo)에 붙어 있는 작은 발광 구체. 분리 불가.

**행동 패턴:**
- P1 (비주얼 전용): 전투 위기 시 에코 뒤로 숨는 듯한 위치 이동. 복종 성공 시 빠르게 진동 및 발광.
- P2 (게임플레이 연동): 서사 진행 이벤트와 연결된 반응 추가 예정.

**스탯 효과:** P1 단계에서 스탯 보너스 없음. 넘버 원의 가치는 수치가 아니라 서사적 존재감에 있다.

**진실의 정체:**
> 넘버 원은 에르다의 스승 마르타가 남긴 이노센트다. 에르다가 처음 아이템계에 진입한 순간, 마르타의 금속 조각 안에 있던 이노센트가 에코로 이주해 왔다. 마르타와 에르다를 10년간 이어준 보이지 않는 끈이 이노센트의 형태로 현현한 것이다.

**3막 공개:**
> 마르타의 실루엣이 기억의 방랑자로 등장하는 순간, 넘버 원이 격렬하게 반응한다. 에르다가 깨닫는 것: 마르타가 남겨둔 금속 조각이 아니라, 넘버 원 자체가 마르타와의 실제 연결이었다.

**구현 단계:** P1 — 비주얼 전용 (에코 부착 발광 구체, 상황별 위치 반응) → P2 — 서사 트리거 연동

방 입장 → 모든 적 처치 → [확률 판정]

### 2.5. 복종 전투 규칙

#### 조우 및 격파 메커니즘
- **대상:** 스폰된 야생 이노센트 또는 보스층의 이노센트 적(Phase 2)
- **난이도:** 일반 방 적보다 약간 강함. 보스 등급과 동등
- **격파 조건:** HP 0 = 복종 확정 (실패 없음)

#### 복종 후 배치
- **자동 배치:** 아이템의 비어있는 첫 슬롯에 배치
- **수동 배치:** 배치 완료 후 Hub의 이노센트 관리 UI에서 재배치 가능 (Phase 2)

## 4-N. 기억의 방랑자와 이노센트의 관계

### 4-N.1. 방랑자의 정체

기억의 방랑자(Memory Wanderer)는 아이템계에 진입했다가 탈출하지 못한 사람들의 의식이 남긴 존재다. 이들은 아이템계에 오래 머무는 과정에서 아이템 내부의 이노센트와 의식이 뒤섞였고, 현실과 기억의 경계에 걸쳐 있는 반생반사(半生半死) 상태로 남아 있다.

> **정확히는:** 이노센트가 된 것도 아니고, 살아있는 사람도 아니다. 아이템의 기억 구조가 사람의 의식을 지층으로 흡수하면서 발생하는 비의도적 융합 현상이다.

### 4-N.2. 임시 제단과 이노센트 에너지

기억의 방랑자가 임시 제단을 생성할 때, 제단에서 제공되는 보너스 보상은 방랑자 안에 잔류하는 이노센트 에너지에서 유래한다. 방랑자는 살아있을 당시 복종시킨 이노센트들의 흔적을 여전히 품고 있으며, 그 잔향이 에르다에게 일시적으로 전달된다.

**게임플레이 의미:** 방랑자를 외면하지 않고 접촉하는 행위는 단순한 보상 획득이 아니다. 갇혀 있는 의식에게 잠시 현실과의 접점을 주는 행위다.

### 4-N.3. 마르타의 경우 — 자발적 잔류

일반 방랑자들이 탈출에 실패해 방랑자가 된 것과 달리, 마르타는 Ancient 등급 아이템의 기억 구조 안에 **자발적으로 남기를 선택**했다. 그 아이템의 기억이 보존될 수 있도록, 그 안의 이노센트들이 야생 상태로 돌아가지 않도록.

> 마르타는 아이템의 기억을 지키는 방랑자가 됐다. 그것이 에르다에게 아무런 설명 없이 사라진 이유다.

**3막 서사 트리거:** 에르다가 해당 Ancient 아이템의 심연 지층에 도달하면 마르타의 실루엣이 기억의 방랑자 형태로 등장한다.

### 4-N.4. 서사적 무게: 이노센트 복종의 재해석

이 맥락에서 에르다가 이노센트를 복종시키는 행위는 단순한 스탯 획득 이상의 의미를 가진다.

| 행위 | 게임플레이 의미 | 서사적 의미 |
| :--- | :--- | :--- |
| 야생 이노센트 복종 | 스탯 2배, 이식 가능 | 방황하는 기억 조각을 두드려 형태를 준다 |
| 기억의 방랑자 접촉 | 임시 제단 보너스 보상 | 갇힌 의식에게 현실 접점을 열어준다 |
| 3막 마르타 조우 | 서사 트리거 | 에르다가 스스로 방랑자가 됐음을 이해하다 |

## 7. 수용 기준 (Acceptance Criteria)

### 7.1. 기능 기준 (Functional)

- [ ] 야생 이노센트는 50% 스탯 효과 적용, UI에 표시
- [ ] 복종 이노센트는 100% 스탯 효과 적용, 복종 즉시 레벨 2배
- [ ] 이노센트 이식 가능 (복종만)
- [ ] 합성 UI 구현: 같은 타입 + 복종 선택 → 합성 가능
- [ ] 레어리티별 슬롯 수 적용
- [ ] 야생 이노센트 조우 15% 확률 작동
- [ ] 복종 격파 시 자동 슬롯 배치

### 7.2. 체험 기준 (Experiential)

- [ ] **야생 → 복종 느낌:** 격파 직후 "이제 이걸 강화할 수 있다"는 해방감이 느껴지는가?
- [ ] **슬롯 제약 느낌:** "이 아이템에는 뭘 넣을까"라는 선택이 의미 있게 느껴지는가?
- [ ] **합성의 보상:** 저레벨 여러 개를 모아 합성한 고레벨 이노센트가 쾌감을 주는가?
- [ ] **야리코미 동기:** 더 높은 레어리티 아이템의 이노센트를 얻기 위해 더 깊이 들어가고 싶은 욕구가 생기는가?
