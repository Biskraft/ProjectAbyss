## 1. 개요 (Concept)

### 1.1. 설계 의도 (Intent)

> **"지층 탐험 도중 예상치 못한 방으로 들어서면, 전투가 아닌 다른 무언가가 기다린다. 보상이 될 수도, 대화가 될 수도, 극한 도전이 될 수도 있다. 미스터리 룸은 야리코미의 리듬에 숨 쉬는 공간을 만든다."**

지층은 기본적으로 전투와 탐험의 반복이다. 이 리듬에 단절 없이 다른 종류의 보상과 긴장을 삽입하는 장치가 미스터리 룸이다. 발견했을 때의 "이게 뭐지?" 반응과, 들어가서 얻는 예상 밖의 결과가 아이템계 세션을 "전투만 하는 파밍"에서 "이야기가 있는 탐험"으로 격상시킨다.

이 문서(SYS-IW-05)는 미스터리 룸의 스폰 규칙, 이벤트 종류, 보상 테이블, NPC 대화, 그리고 에르다·기억의 방랑자·마르타의 서사 트리거와의 연결을 정의한다.

### 1.2. 설계 근거 (Reasoning)

| 설계 결정 | 채택 이유 | 기각된 대안 |
| :--- | :--- | :--- |
| **5% 기본 스폰 확률** | 희소성이 발견의 쾌감을 만든다. 흔하면 루틴이 되고, 너무 드물면 존재감이 소멸한다. 4×4 Room Grid 기준 16방 중 평균 0~1개 등장 — 세션당 0~2회로 기대값 설정 | 10% 이상 — 매 지층 등장이 확실해지며 특별함 소멸 |
| **최대 1개/지층** | 2개 이상 등장 시 "미스터리 룸 파밍"이 최적 전략이 되어 일반 방 탐험 가치 저하. 탐험의 균형 보호 | 제한 없음 — 이벤트 집중으로 일반 전투 공동화 |
| **시작 방·종료 방 제외** | 크리티컬 패스의 첫/마지막 방은 전투 리듬을 유지해야 함. 보스 방 직전에 이벤트가 끼어들면 긴장감 단절 | 모든 방 포함 — 보스 전 미스터리 룸으로 리듬 파괴 |
| **아이템 EXP를 이벤트 화폐로** | 아이템계 고유 화폐를 통일하여 허브 경제(골드)와 구분. 이벤트 참여가 파밍 목표(아이템 레벨 상승)와 직결 | 전용 이벤트 화폐 — 시스템 복잡도 증가, 환전 UX 필요 |
| **서사와 기계적 보상 동시 제공** | 이벤트가 순수 보상 자판기가 되면 야리코미 도구로만 인식됨. 에르다의 대사·기억의 단편이 삽입되면 "이 아이템의 이야기" 감성이 강화됨 | 기계적 보상만 — 야리코미 효율 도구로 전락 |
| **재귀의 문을 미스터리 룸으로** | 재귀 진입(`IWC-11-A`)의 입장은 예상치 못한 순간에 나타나야 "결단"의 감각이 산다. 예측 가능한 버튼이 되면 리스크-리턴 감각 소멸 | 별도 버튼/UI — "눌러야 하는 선택지"로 재귀의 신비감 소멸 |

## 2. 메커닉 (Mechanics)

### 2.1. 미스터리 룸 스폰 메커닉

지층 생성 시(SYS-IW-02), Room Grid의 각 방이 배치될 때 미스터리 룸 여부를 판정한다.

**판정 대상 방:**
- 크리티컬 패스 위의 방 (보조 분기 방은 별도 확률 적용)
- 시작 방(Start Room), 보스 방(End Room), 탈출 제단 방(Escape Altar Room) 제외

**스폰 판정 흐름:**

```
지층 생성 시 → 각 방 배치 단계 → 미스터리 룸 후보 방 추출
→ 이미 이 지층에 미스터리 룸이 1개 생성됐는가?
    → YES: 스킵
    → NO: 스폰 확률 계산 → PRNG 판정 → 확률 이상이면 미스터리 룸 배정
```

**스폰 확률 계산:**

```
spawnChance = BASE_MYSTERY_CHANCE
            + (stratumIndex - 1) × DEPTH_BONUS
            + (itemRarity - NORMAL) × RARITY_BONUS
            + (playerLCK / LCK_DIVISOR)
```

| 변수 | 기본값 | 범위 | 비고 |
| :--- | :---: | :--- | :--- |
| `BASE_MYSTERY_CHANCE` | 0.05 | 고정 | `Sheets/IWE_EventWeightTable.csv` SSoT |
| `DEPTH_BONUS` | 0.005 | 지층당 가산 | 지층 2=+0.5%, 지층 4=+1.5% |
| `RARITY_BONUS` | 0.01 | 레어리티 등급당 | Normal=0, Magic=+1%, Legendary=+3% |
| `LCK_DIVISOR` | 500 | — | LCK 500당 +1% |

예시: Rare 아이템, 지층 3, LCK 0 → 5% + 1% + 2% = 8%

**보조 분기 방 확률 보정:**
크리티컬 패스에서 벗어난 분기 방에는 `BASE_MYSTERY_CHANCE × 0.5`를 적용한다. 탐험 보상이지만 크리티컬 패스 미스터리 룸과 중복 억제를 위해 반감한다.

### 2.2. 미스터리 룸 외형 메커닉

미스터리 룸은 일반 방과 구분되는 시각 언어로 표시된다.

| 상태 | 미니맵 표시 | 입구 타일 | 진입 전 표시 |
| :--- | :--- | :--- | :--- |
| 미발견 | 표시 안 됨 | 일반 입구 | 없음 |
| 발견(입구 인접) | `?` 아이콘 (레어리티 색상) | 특수 문양 프레임 | 단거리 BGM 변화 (2초) |
| 진입 후 | 특수 방 아이콘 | — | — |

> **설계 의도:** 발견은 탐험 과정에서 자연스럽게 이루어져야 한다. 미니맵에서 미리 보이지 않으므로 플레이어는 방을 직접 돌아다니며 발견해야 한다. LCK가 높을수록 미스터리 룸 자체가 더 자주 등장하여 "탐험 스탯"으로서의 가치를 부여한다.

### 2.3. 이벤트 선택 메커닉

미스터리 룸에 진입하면 이벤트가 결정적으로 배정된다. 동일 아이템·동일 지층은 항상 동일 이벤트(시드 기반).

## 3. 규칙 (Rules)

### 3.1. P1 이벤트 규칙

#### EVT-01: 기억의 상인 (Memory Merchant)

**개념:** 아이템 기억 속에 자발적으로 거주하는 상인 NPC. 허브 상점에 없는 희귀 소모품·이노센트 단편을 거래한다.

**에르다 대사 (진입 시):**
> "어, 여기에도 사람이...? 아니, 이 분위기는 — 기억 속에 사는 사람인가요?"

**NPC 대사 (상인):**
> "이 안에 있으면 조용해서 좋아. 손님 오면 팔고, 아니면 그냥 기억이나 보지 뭐."

**규칙:**
- 상품 3종이 무작위로 진열된다. 상품 풀은 `Sheets/IWE_MerchantPool.csv` 참조.
- 화폐: 아이템 EXP (`currentItemEXP`에서 즉시 차감).
- 플레이어는 0개에서 3개까지 자유롭게 구매한다. 강제 구매 없음.
- 미구매 상품은 사라지고 다음 진입 시 재생성.
- 1회 방문당 각 상품 1개 한도.

**상품 풀 분류:**

| 등급 | 상품 예시 | 가격 (아이템 EXP) | 등장 조건 |
| :--- | :--- | :---: | :--- |
| 일반 | 응급 포션 소 (HP 30% 회복) | 150 | Normal 이상 |
| 일반 | 기억의 향로 (EXP +20% 버프, 다음 방까지) | 200 | Normal 이상 |
| 희귀 | 봉인된 이노센트 조각 (랜덤 이노센트 ×1, Phase 2) | 500 | Magic 이상 |
| 희귀 | 기억의 결정 (아이템 레벨 +1 소재) | 800 | Rare 이상 |
| 고급 | 재귀의 열쇠 (이번 지층 한정, 재귀의 문 확률 +20%) | 1,200 | Legendary 이상 |

#### EVT-02: 기억의 점술사 (Memory Oracle)

**개념:** 아이템의 기억 파편에서 미래와 과거를 읽는 은자형 NPC. 아이템의 레벨(기억의 깊이)을 조작할 수 있다.

**에르다 대사 (진입 시):**
> "이 아이템 안에... 이런 사람도 살 수 있구나. 아, 저는 이 아이템의 주인인데요 —"

**NPC 대사 (점술사):**
> "기억은 흘러. 위로도, 아래로도. 내가 흘려드릴까요, 거슬러드릴까요?"

**규칙:**
- 플레이어에게 두 가지 선택지 제공:
  - **"기억을 강화한다"** → 아이템 레벨 +2 (최대 레벨 초과 불가)
  - **"기억을 되돌린다"** → 아이템 레벨 -2 (최소 레벨 1, 보정은 복구 불가)
- 선택 후 이벤트 종료. 취소 가능(선택 전 퇴장 시 이벤트 소멸 — 재방문 불가).
- 서사 단편 1줄이 추가로 표시된다. 내용은 아이템의 `itemNarrativeTheme` 테마 연동(`System_ItemNarrative_Template.md` §2.1 테마 ID 참조).

> **설계 의도:** "기억을 되돌린다"를 선택하는 플레이어는 없을 것처럼 보이지만, 이미 최대 레벨에 가까운 아이템에 쓸 공간이 없을 때 다른 방향의 서사 접근이 될 수 있다. 실질적 페널티로 의도된 선택지이므로 명확하게 표기한다.

#### EVT-03: 도전 방 (Challenge Room)

**개념:** 방 전체에 강화된 적 무리가 배치되어 있다. 제한 시간 내 전멸을 달성하면 고보상을 얻는다.

**진입 연출:**
- 방 진입 시 철문이 잠긴다 (탈출 불가).
- 화면 상단에 `[도전 방: 00:60]` 타이머 표시.
- 적들은 일반 지층 적의 레벨 ×1.5.

**규칙:**
- 타이머: `CHALLENGE_TIME_LIMIT` 초 (기본 60초). `Sheets/IWE_EventWeightTable.csv` 파라미터.
- 성공 조건: 타이머 종료 전 방 내 모든 적 전멸.
- 성공 시:
  - 아이템 EXP `CHALLENGE_SUCCESS_EXP` (기본 600)
  - 추가 드랍 확률 `CHALLENGE_BONUS_DROP_CHANCE` (기본 30%) — 레어리티 +1 등급 아이템 드랍
- 실패 조건: 타이머 만료 OR 에르다 사망.
- 실패 시: 철문 개방. 보상 없음. 진행 손실 없음 (사망 패널티 별도 적용).
- 멀티플레이: 타이머는 동일. 적 HP는 SYS-IW-01 §3.5 인원별 배율 적용.

**에르다 대사 (성공):**
> "후... 이 긴장감, 나쁘지 않네. 이 아이템도 꽤 날카롭군."

#### EVT-04: 기억 저장고 (Memory Vault)

**개념:** 방 안에 보물상자 3개가 배치되어 있다. 전투 없이 열 수 있다.

**규칙:**
- 보물상자 3개. 각각 레어리티 색상으로 구분.
- 보상 테이블:

| 상자 등급 | 등장 조건 | 보상 |
| :--- | :--- | :--- |
| 일반 (흰색) | 항상 | 소모품 1~2종 or 소량 골드 |
| 고급 (파란색/노란색) | Magic 이상 | 아이템 EXP 200~500 or 이노센트 조각 (Phase 2) |
| 특수 (레어리티 색상) | 항상 1개 | 현재 아이템 레어리티 +1 등급 장비 아이템 드랍 |

- 상자는 전부 공개 타입 (함정 없음). 야리코미 리듬 단절 최소화.
- 파티 플레이: 특수 상자는 개인 인스턴스 드랍. 일반/고급 상자는 첫 상호작용 플레이어에게 귀속.

> **설계 의도:** 가장 단순한 이벤트. 탐험 보상의 순수한 형태. 전투 피로가 쌓인 시점에 "그냥 좋은 것" 을 준다. 디스가이아 보물 방의 핵심 감성.

#### EVT-05: 방랑자의 제단 (Wanderer's Altar)

**개념:** 아이템계 내부에 기억의 방랑자의 흔적이 남겨진 제단. 방랑자의 잔상이 에르다와 대화를 나눈다.

**규칙:**
- 방 중앙에 제단이 있다. 방랑자 NPC 잔상이 제단 앞에 서 있다.
- 에르다가 상호작용하면 짧은 대화 시퀀스(3~4줄) 자동 진행. 스킵 가능(A/입력).
- 대화 후 제단에서 보상 1종 획득:
  - 탐험 게이지 즉시 `WANDERER_GAUGE_BONUS` 점 증가 (기본 100점, 탐험 게이지 §3.7 연동)
  - 10% 확률: 서사 단편 — 해당 아이템의 전 주인 기억 1절 해금
- 방랑자 잔상은 대화 후 소멸. 사망하지 않음.

**서사 특수 규칙 (SYS-IW-01 §2.1 기억의 방랑자 연동):**
- 아이템 서사 테마가 `T-LOSS`(상실) 또는 `T-WAR`(전쟁)인 아이템에서는 방랑자 대사에 해당 테마 키워드가 삽입된다.
- **마르타 트리거 조건** (Act 3 서사 연동): 게임 진행도 조건 충족 시 (`Act3Trigger = true`), Legendary 이상 아이템의 지층 3~4 방랑자의 제단에서 방랑자 잔상이 **마르타 벤-나흐트의 실루엣**으로 대체될 수 있다. 확률 `MARTA_TRIGGER_CHANCE` (기본 15%). 이 경우 에르다는 대화 대신 독백만 한다.

**방랑자 대사 예시 (일반):**
> "...이 안에 오래 있었어. 돌아가려 했는데, 더 좋은 기억이 있었거든. 당신은 돌아가려고 하나요?"

**에르다 대사 (마르타 트리거 시):**
> "...스승님? 아니, 달랐다. — 왜 저한테 그 얼굴을 보여준 거야..."

### 3.2. P2 이벤트 규칙

#### EVT-06: 이노센트 광장 (Innocent Gathering)

> **의존성:** Phase 2 이노센트 시스템(`System_Innocent_Core.md`) 구현 후 활성화.

**개념:** 아이템 내부에서 야생 이노센트 여러 마리가 집결해 있는 특수 공간. 즉시 전투 없이 이노센트를 관리할 수 있다.

**규칙:**
- 방 내에 야생 이노센트 3~5마리가 배치된다. 적대적이지 않음.
- 에르다가 이노센트에게 접촉하면 선택지 제공:
  - **전투:** 즉시 1:1 전투. 승리 시 복종 상태로 전환.
  - **관찰:** 해당 이노센트의 스탯 미리보기. 전투 유발 없음.
  - **무시:** 통과.
- 파티 플레이: 이노센트는 먼저 접촉한 파티원에게 귀속. 타 파티원은 해당 이노센트 상호작용 불가.

#### EVT-07: 재귀의 문 (Recursive Gate)

> **의존성:** SYS-IW-01 `IWC-11-A` 재귀적 진입 구현 후 활성화.

**개념:** 아이템계 내부에 현재 아이템계보다 더 깊은 공간으로 통하는 균열 문. 진입하면 현재 아이템계 상태가 동결되고, 새 아이템계 깊이 1이 시작된다.

**스폰 조건:**
- 최소 레어리티: Rare
- 최소 지층: 2
- 현재 재귀 깊이 < 3 (`recursionDepth < 3`)
- 스폰 확률: `BASE_MYSTERY_CHANCE × 0.5` (미스터리 룸 내 이벤트 가중치 10)

**규칙:**
- 방 중앙에 균열 문이 서 있다. 에르다가 접촉하면 선택 UI 표시.
- 선택 UI:

```
┌────────────────────────────────────────────────┐
│  재귀의 문 — 이 안에 또 다른 기억이 있습니다.  │
│                                                │
│  현재 깊이: [N] / 최대: 3                      │
│  경고: 이 아이템계의 진행이 일시 동결됩니다.    │
│                                                │
│  [ 진입한다 ]          [ 돌아간다 ]            │
└────────────────────────────────────────────────┘
```

- 진입 시: 인벤토리에서 미강화 장비 아이템 1개 선택 → 해당 아이템의 아이템계 진입 (SYS-IW-01 진입 규칙 동일).
- 재귀 진입 연출: SYS-IW-01 §2.1 재귀적 진입 연출 그대로 적용.
- 완료 후: 상위 아이템계 동결 지점으로 자동 복귀.
- 재귀 깊이 3에서 재귀의 문이 스폰되어 있다면 — 문이 봉인된 상태로 표시된다. 에르다 대사: *"더는 못 들어가겠어. 이 안에서 길을 잃으면..."*

#### EVT-08: 망각의 시련 (Oblivion Trial)

**개념:** 기억이 희미해지는 고위험 전투 공간. 진입하면 에르다의 스탯 일부가 봉인된다.

**규칙:**
- 진입 시 스탯 봉인 효과 발동: ATK 또는 DEF 중 무작위 1종 -30% (이번 지층 탈출까지 유지).
- 방 내 적: 현재 지층 기준 ×2.0 HP, ×1.5 ATK.
- 전멸 성공 시: 아이템 레벨 `OBLIVION_LEVEL_BONUS` (기본 +1) 영구 증가 + 봉인 즉시 해제.
- 전멸 실패 / 탈출(A버튼): 봉인 해제. 보상 없음.
- 봉인 상태는 HUD 상단에 적색 자물쇠 아이콘으로 명시 표시.

> **리스크-리턴 설계:** 아이템 레벨 +1 영구 증가는 강력한 보상이지만, 봉인으로 인해 즉각 다음 방이 위험해진다. 고보상을 원하면 약해진 상태에서 계속 탐험해야 한다는 의미 있는 결단.

#### EVT-09: 기억의 메아리 (Memory Echo)

**개념:** 방 안에 공중에 떠 있는 기억 조각들이 에르다의 과거 장면을 반복 재생하고 있다. 전투 없음.

**규칙:**
- 방 내에 기억 구체 3개가 배치된다. 각각 에르다 서사의 단편을 담고 있다.
- 에르다가 구체에 접촉하면 텍스트 단편 1절이 자동 표시 (2~3줄, 스킵 가능).
- 구체 3개 모두 접촉 시: 탐험 게이지 `ECHO_GAUGE_BONUS` (기본 150점) 획득.
- 구체 내용은 `itemNarrativeArchetype`(서사 아키타입)에서 파생. 아이템마다 다른 에르다 기억이 등장.
- 단편 내용 예시 (T-HOME 테마):
  > "스승님 대장간의 모루 소리. 아침마다 그 소리로 잠을 깼다. 언제부터인가 그 소리가 들리지 않았다."

#### EVT-10: 이노센트 여인숙 (Innocent Inn)

> **의존성:** Phase 2 이노센트 시스템 구현 후 활성화.

**개념:** 복종 상태 이노센트들이 운영하는 작은 여인숙. 에르다가 잠깐 쉬어갈 수 있다.

**규칙:**
- 방 내부에 침대 오브젝트가 있다. 에르다가 상호작용하면 HP `INN_HEAL_AMOUNT` (기본 최대 HP의 30%) 회복.
- 여인숙 이노센트 NPC가 1마리 배치되어 있다. 대화 시 아이템 레어리티·테마에 맞는 대사 1줄.
- HP 회복 후 이벤트 종료. 재방문 시 HP 회복 불가 (이노센트만 대화 가능).

## 5. 예외 처리 (Edge Cases)

### 5.1. 진입 예외

| 상황 | 처리 |
| :--- | :--- |
| 미스터리 룸 진입 중 접속 끊김 | 재접속 시 미스터리 룸 입구 앞에 위치. 이벤트 상태는 진입 전으로 초기화. 이미 획득한 보상은 보존(아이템 보존 원칙, SYS-IW-01 §2.3) |
| 미스터리 룸 내에서 사망 | 일반 사망 처리와 동일(SYS-IW-01 §4.4). 도전 방 진행 중 사망 시 해당 이벤트 종료 (재도전 불가). 보상 없음 |
| 파티 중 일부 파티원이 미스터리 룸 미진입 | 미진입 파티원에게 입구 방향 아이콘 표시. 이벤트는 진입한 파티원만 참여. 타이머 이벤트(도전 방)의 타이머는 첫 진입 파티원 기준 시작 |
| 이벤트 조건 미충족으로 폴백 발생 | EVT-04(기억 저장고)로 대체. 플레이어에게 별도 알림 없음 |

### 5.2. 도전 방 예외

| 상황 | 처리 |
| :--- | :--- |
| 타이머 0초 시 에르다가 마지막 적을 타격 중 | 히트스탑(6프레임) 우선 처리. 히트스탑 종료 후 타이머 만료 판정. 이 프레임에서 적이 사망하면 성공 처리 |
| 타이머 만료와 마지막 적 사망이 동시 | 성공 처리 우선 (플레이어 유리 원칙) |
| 파티 중 일부 사망 후 잔여 멤버가 성공 | 성공 보상은 전원 지급. 사망 멤버는 사망 패널티 별도 적용 |
| 철문 내 에르다가 미스터리 룸 입구로 이동 시 | 입구 방향 이동 불가 (철문 물리 콜라이더 활성 상태 유지). 이벤트 진행 중 강제 탈출 없음 |

### 5.3. 재귀의 문 예외

| 상황 | 처리 |
| :--- | :--- |
| 재귀 진입 선택 후 인벤토리에 진입 가능 아이템 없음 | "진입할 수 있는 아이템이 없습니다" 알림. 문 상호작용 취소. 이벤트 대기 상태 유지 (재시도 가능) |
| 재귀 아이템계 진행 중 상위 아이템계가 타임아웃 | 타임아웃 없음. 아이템계에 세션 제한 없음. 재귀 완료 후 상위 복귀 항상 보장 |
| 재귀 깊이 3에서 EVT-07 스폰됨 | 문이 봉인 상태로 표시. 상호작용 시 에르다 독백 출력 후 이벤트 종료. 보상 없음 |
| 멀티플레이 중 파티장이 재귀 진입, 파티원이 미진입 | 파티원은 현재 지층에 그대로 남음. 재귀 진입은 파티장 개인 인스턴스. 파티장 복귀 후 파티 재결합 |

### 5.4. 방랑자의 제단 — 마르타 트리거 예외

| 상황 | 처리 |
| :--- | :--- |
| `Act3Trigger`가 false인 상태에서 조건 달성 | 일반 방랑자 대사로 진행. 마르타 실루엣 없음 |
| 마르타 트리거 발동 후 파티 플레이 중 | 마르타 독백은 파티장(아이템 소유자)에게만 표시. 타 파티원은 "방랑자가 에르다에게 조용히 다가갔다" 로그 표시 |
| 마르타 트리거가 동일 플레이스루에서 재발동 | `martaTriggerFired = true` 플래그로 1회만 발동. 이후 동일 조건에서는 일반 방랑자 대사 |

### 5.5. 경계값 처리

| 상황 | 처리 |
| :--- | :--- |

| 아이템 레벨이 최대일 때 점술사 "기억을 강화한다" 선택 | 선택지 비활성화 (회색 표시). 툴팁: "이 아이템의 기억은 이미 가득 찼습니다" |
