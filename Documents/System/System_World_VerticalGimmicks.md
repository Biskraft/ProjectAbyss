## 1. 개요 (Concept)

### 1-1. 설계 의도

수직 이동 기믹은 플레이어의 이동 능력(렐릭)과 독립적으로 존재하는 **환경 장치**로, The Shaft의 거대한 수직 구조를 체감시키고 탐험 속도를 조율한다. 이 시스템은 플레이어 능력 기반 수직 이동(`System_World_AbilityGating.md`)과 병렬로 존재하며, 두 시스템이 결합하여 수직 공간의 입체적 탐험을 만든다.

환경 기믹은 능력 게이트가 아니다. 능력 없이도 사용할 수 있되, 특정 렐릭을 보유하면 더 안전하거나 더 효율적으로 사용할 수 있다.

> **스파이크 검증:**
> - q1: 이 시스템이 아이템계 진입 경험을 강화하는가? — YES (아이템계 기믹이 파밍 세션 리듬을 가속하고, 월드 기믹이 아이템 획득 동기를 유지시킨다)
> - q2: 이 시스템이 없으면 스파이크가 약해지는가? — YES (수직 기믹 없이는 수직 대공동이라는 세계관이 평면적으로 느껴진다)
> - q3: 이 시스템이 스파이크를 희석시키지 않는가? — YES (기믹은 탐험과 파밍 루프를 강화하며 분리된 시스템을 추가하지 않는다)

### 1-2. 설계 근거

| 설계 결정 | 이유 |
| :--- | :--- |
| 기믹을 능력 게이트와 분리 | 능력 게이트는 Critical Path. 기믹은 탐험 보조. 두 시스템을 혼용하면 "이 기믹을 쓰려면 무슨 렐릭이 필요한가"가 불명확해진다 |
| 월드/아이템계 기믹을 별도 설계 | 월드는 핸드크래프트 배치, 아이템계는 특수 방(Room Template) 배치. 기믹의 물리적 구현은 같더라도 게임플레이 맥락이 다르다 |
| 세계관 내재적 정당화 필수 | 모든 기믹은 BLAME! 세계관(빌더 유산, 아이템의 기억)으로 설명 가능해야 한다. UI 기반 텔레포트는 채택하지 않는다 |
| 발견-활성화-반복 3단계 구조 | 기믹은 발견 즉시 사용 가능해서는 안 된다. 탐험 행위가 투자가 되어야 한다 (예외: 상승 기류 샤프트 — 항상 활성) |
| 비대칭 수직 이동 | 상승보다 하강이 빠른 구조. 아래로 내려가는 것은 쉽지만 돌아오는 것은 어렵다 — BLAME!의 수직 공동 서사 반영 |

### 1-3. 3대 기둥 정렬

| 기둥 | 기여 방식 |
| :--- | :--- |
| **탐험가** | 수직 기믹 자체가 탐험 보상. 빌더 터미널 찾기, 워프 노드 양쪽 발견 등 탐험 행위가 기믹 해금으로 이어진다 |
| **야리코미** | 아이템계 기믹(침강 샤프트, 캐터펄트)이 파밍 세션 리듬을 가속. 보스 방 접근 속도를 높여 반복 플레이 마찰 감소 |
| **모험가** | 멀티플레이 시 파티원 한 명이 빌더 터미널을 활성화하면 파티 전체가 이동 가능. 협동 탐험 가치 부여 |

## 2. 기믹 카탈로그 (Gimmick Catalogue)

### 2-1. 월드 기믹

#### VGM-W0-1: 붕괴 발판 낙하 경로 (Collapse Chute)

**세계관:** 오래된 지층의 바닥이 부분 붕괴되어 생긴 자연적 하강 경로. 빌더가 의도하지 않은 통로.

| 항목 | 내용 |
| :--- | :--- |
| 외형 | 균열 바닥 틈새. 아래로 어둠이 이어지는 수직 개구부 |
| 활성화 조건 | 없음 — 항상 진입 가능 |
| 이동 방향 | 하강 전용 |
| 이동 속도 | 빠름 (자유낙하 물리) |
| 착지 | 반드시 안전 착지 지점 배치 (abyss_void 착지 금지) |
| 귀환 | 상승 수단 없으면 귀환 불가 (비대칭 수직 이동 의도적 설계) |
| 시각 단서 | 균열 텍스처 + 아래를 향한 파티클 흐름 + 열린 구멍에서 바람 소리 |

**BLAME! 오마주:** 목적지 불명의 낙하. Killy가 수직 공동에 떨어지는 장면의 게임 번역.

#### VGM-W1-1: 상승 기류 샤프트 (Updraft Shaft)

**세계관:** 빌더의 환기 시스템에서 나오는 강한 상승 기류. 일부는 여전히 작동 중.

| 항목 | 내용 |
| :--- | :--- |
| 외형 | 수직으로 뚫린 금속 샤프트. 하단 격자 환기구에서 바람 분출 |
| 활성화 조건 | 없음 — 항상 활성 상태 (updraft IntGrid 타일 영역) |
| 이동 방향 | 상승 전용 |
| 이동 속도 | 빠름 (updraft 강도: `Updraft_Force_Multiplier`) |
| 탈출 | 좌/우 이동 입력으로 샤프트 밖 이탈 가능 |
| 위험 요소 | 샤프트 벽 충돌 시 데미지 (`Wall_Collision_Damage`). 상단 격자 막힘 가능 |
| 렐릭 연동 | 이단 점프: 상승 중 추가 공중 조작 가능. 대시: 샤프트 내 수평 대시로 벽 회피 |

**타일 구현:** 기존 updraft(4) IntGrid를 수직으로 연속 배치하여 샤프트를 구성. 별도 Entity 불필요.

#### VGM-W1-2: 빌더의 승강 플랫폼 (Builder's Freight Platform)

**세계관:** 빌더가 자재를 층간 운반하기 위해 설치한 화물 엘리베이터. 현재 반자동 상태.

| 항목 | 내용 |
| :--- | :--- |
| 외형 | 거대한 녹슨 금속 플랫폼. 빌더 마크 각인. 상하 레일 구조 |
| 활성화 조건 | 플랫폼 주변의 빌더 터미널 조작 (탐험 행위로 해금) |
| 이동 방향 | 양방향 (상승/하강 버튼 선택) |
| 이동 속도 | 느림 (`Platform_Move_Speed` — 약 3~5초/구간) |
| 탑승 판정 | 플랫폼 상단에 grounded 상태일 때 탑승. 탑승 중 이동 가능 |
| 위험 요소 | 플랫폼 가장자리 낙사. 이동 중 적 등장 가능 |
| 연결 지점 | 2점 연결 기본 (하부 거점 ↔ 상부 거점). 3점 연결 가능 (`SotN 엘리베이터` 참조) |
| 이동 중 | 배경으로 지나온 구역 가시화. 세계 규모 체감 의도 |

**BLAME! 오마주:** 빌더가 도시 건설에 사용한 산업 엘리베이터의 잔재. 광속 엘리베이터도 달이 걸리는 규모의 세계관을 느리게 이동하며 체감.

#### VGM-W2-1: 에코 반향 경로 (Echo Resonance Path)

**세계관:** 에르다의 도구 에코(Echo)가 발사한 에너지 반향이 빌더의 잠든 시스템을 일시 활성화.

> **의존성 주의:** 에코 전투 시스템(`System_Combat_Echo.md`) 완성 후 구현. Phase 2 이전 미구현.

| 항목 | 내용 |
| :--- | :--- |
| 외형 | 빌더 마크가 각인된 수직 벽면. 에코 조사 시 발광 |
| 활성화 조건 | 에코를 마크에 조준 사격 |
| 이동 방식 | 에코 발사 → 벽면 활성화 → 임시 발판/통로 생성 (`Echo_Ramp_Duration`초) |
| 이동 방향 | 상승 (생성된 발판을 밟고 오름) |
| 이동 속도 | 중간 (직접 이동해야 함) |
| 제한 | 에코 쿨다운 연동 (`Echo_Cooldown_s`). 특정 벽면에만 마크 배치 |
| 렐릭 연동 | 이단 점프 + 에코 발판 조합으로 더 높은 구간 도달 가능 |

**설계 의도:** 에르다의 정체성(격벽 측량사, GBE 오마주)을 이동에 연결. 전투 도구가 탐험 도구로 전환.

#### VGM-W2-2: 빌더 워프 노드 (Builder Warp Node)

**세계관:** 빌더의 물류 순간이동 시스템. 일부 노드는 아직 기능 중.

| 항목 | 내용 |
| :--- | :--- |
| 외형 | 빌더 문양이 새겨진 육각형 플랫폼. 활성화 시 에너지 기둥 생성 |
| 활성화 조건 | **양쪽 노드를 모두 탐험으로 발견**해야 연결 활성화 |
| 이동 방식 | 노드 진입 → 목적지 선택 → 즉시 이동 |
| 이동 속도 | 즉시 |
| 노드 수 | 구역당 1~2쌍 (과도한 빠른 이동 방지) |
| 발견 구조 | 노드 A 발견 → "연결 미완성" 상태 표시 → 노드 B 발견 → 연결 활성화 |

**설계 원칙:** `SotN` 순간이동 방의 BLAME! 버전. 단방향 해금이 아닌 양방향 발견 조건이 추가 탐험 동기를 부여.

### 2-2. 아이템계 기믹

아이템계 기믹은 절차적 생성 환경이므로 **특수 방 템플릿(Special Room Template)**으로 구현된다. 지층 당 등장 빈도는 `Content_ItemWorld_SpawnTable.csv`에서 관리.

#### VGM-I1-1: 기억의 침강 샤프트 (Memory Descent Shaft)

**세계관:** 아이템의 기억이 응결된 지층 사이. 과거 주인의 기억 파편이 하강한다.

| 항목 | 내용 |
| :--- | :--- |
| 방 유형 | 절차적 배치 특수 방. 지층당 `DescentShaft_SpawnRate` 확률 |
| 외형 | 수직으로 긴 방 (높이 8~12타일). 중심에 반투명 기억 파편 부유 |
| 이동 방식 | 파편에 올라타면 다음 지층 보스 방 근처로 하강 |
| 이동 방향 | 하강 전용 |
| 위험 요소 | 하강 중 벽에서 적 출현 가능 (`DescentShaft_EnemySpawn_Rate`) |
| 귀환 | 하강 후 귀환 불가. 텔레포트 크리스탈(소모품)만 귀환 수단 |
| 선택 | 파편에 올라타는 것은 선택. 일반 진행 경로로도 보스 방 도달 가능 |

**세션 설계:** 파밍 세션의 "한 층만 더" 심리 강화. 보스 방 접근 마찰 감소 → 반복 플레이 리듬 가속.

---

#### VGM-I1-2: 이노센트 발사 캐터펄트 (Innocent Catapult)

**세계관:** 아이템 내부를 지배하는 이노센트 중 일부가 플레이어를 위/아래로 쏘아 올린다.

| 항목 | 내용 |
| :--- | :--- |
| 방 유형 | 절차적 배치 특수 방 |
| 외형 | 거대한 탄도 발사대. 이노센트가 주변 맴돌며 발사 준비 |
| 이동 방식 | 발사대 탑승 → 방향 선택 (위/아래) → 발사 |
| 이동 속도 | 매우 빠름 (즉각 발사) |

| 렐릭 연동 | 이단 점프: 공중 중 추가 조작 가능. 안개 변신: 착지 직전 변신으로 틈새 통과 |

## 3. 기믹 × 렐릭 연동 매트릭스

각 기믹과 렐릭의 상호작용 관계. "필수"는 렐릭 없이 진행 불가, "강화"는 렐릭이 있으면 더 효율적이거나 더 안전하게 사용 가능, "-"는 관련 없음.

| 기믹 | 대시 | 이단 점프 | 벽 타기 | 안개 변신 | 수중 호흡 | 역중력 |
| :--- | :---: | :---: | :---: | :---: | :---: | :---: |
| **붕괴 발판 낙하 경로** | 강화¹ | 강화² | - | 강화³ | - | - |
| **상승 기류 샤프트** | 강화⁴ | 강화⁵ | 강화⁶ | - | - | - |
| **빌더의 승강 플랫폼** | - | - | - | - | - | - |
| **에코 반향 경로** | - | 강화⁷ | - | - | - | - |
| **빌더 워프 노드** | - | - | - | - | - | - |
| **기억의 침강 샤프트** | - | - | - | - | - | - |
| **이노센트 발사 캐터펄트** | - | 강화⁸ | - | 강화⁹ | - | 강화¹⁰ |
| **붕괴 메가스트럭처 낙하 지대** | 강화¹¹ | - | - | 강화¹² | - | - |

**주석:**
1. 대시: 낙하 중 수평 대시로 착지 지점 미세 조정 가능
2. 이단 점프: 낙하 중 이중 점프로 낙하 경로 전환 (옆 통로 진입 가능)
3. 안개 변신: 착지 직전 변신으로 좁은 착지 지점 통과
4. 대시: 샤프트 내 수평 대시로 벽 충돌 회피
5. 이단 점프: 상승 중 추가 공중 조작으로 샤프트 탈출 지점 정밀 선택
6. 벽 타기: 샤프트 벽에서 상승 속도 조절 가능 (벽 슬라이드로 감속)
7. 이단 점프: 에코 발판 + 이중 점프 조합으로 더 높은 구간 도달
8. 이단 점프: 발사 후 공중에서 추가 점프로 착지 위치 조정
9. 안개 변신: 발사 후 좁은 틈새 착지 통과
10. 역중력: 아래 방향 발사 시 역중력으로 천장 착지 가능 (특수 조합)
11. 대시: 타이밍 창 내 빠른 진입을 위한 대시 활용
12. 안개 변신: 잔해 데미지 회피 (변신 중 무적 프레임 또는 통과)

**능력 게이트와의 분리 원칙:** 상기 매트릭스에서 "필수" 항목이 없음을 확인. 모든 기믹은 렐릭 없이 기본 사용 가능. 렐릭은 사용 효율을 높이지만 기믹 자체의 접근을 막지 않는다.

### 5-2. 숏컷 설계 원칙

기믹이 만들어내는 숏컷은 다음 원칙을 따른다:

- **숏컷은 우회이지 생략이 아니다:** 숏컷을 사용해도 메인 경로에서 발생하는 핵심 보상(능력 렐릭, 보스 처치)은 건너뛸 수 없다. 숏컷은 이미 클리어한 구간의 재방문 마찰을 줄이는 용도.
- **숏컷 해금 = 탐험 보상:** 빌더 터미널 조작, 노드 양쪽 발견 등 숏컷 활성화 자체가 탐험 행위에 대한 보상이다.
- **붕괴 낙하 경로 예외:** 이 기믹만이 아직 클리어하지 않은 구간을 건너뛸 수 있는 진정한 숏컷. 리스크(귀환 불가)가 보상(스킵)을 정당화.

### 5-3. BLAME! 스케일 연출 원칙

기믹이 BLAME!의 수직 대공동 감각을 강화하는 방식:

| 기믹 | 스케일 연출 방법 |
| :--- | :--- |
| 빌더의 승강 플랫폼 | 느린 이동 중 배경 레이어에 지나온 구역 가시화. 원근법 타일로 깊이감 강조 |
| 붕괴 발판 낙하 경로 | 낙하 시 아래가 보이지 않는 어둠. 착지 전까지 목적지 불명 |
| 상승 기류 샤프트 | 샤프트 출구까지 올라가는 동안 양쪽 벽에 지층 단면 배경 노출 |
| 기억의 침강 샤프트 | 하강 중 아이템의 기억 파편이 양쪽에서 지나감 (몽환적 연출) |

| 상황 | 처리 | 근거 |

| 캐터펄트 발사 중 안개 변신 입력 | 변신 실패 (공중 상태에서 트리거 불가) 또는 즉시 변신 중 한 가지 — 구현 시 결정 필요 | 안개 변신 시스템 규칙 확정 후 이 항목 갱신 |
| 역중력 + 캐터펄트 아래 방향 발사 | 역중력 중 아래 발사 = 천장으로 이동. 천장에 착지 판정 처리 필요 | 레어 시너지지만 물리 불일치 방지를 위해 명시 |
| 아이템계에서 상승 기류 샤프트 진입 시 산소 게이지 | 아이템계는 산소 게이지 미적용 (`System_World_TileSystem.md §5` 동일) | 아이템계는 짧은 세션 구조 — 생존 압박 최소화 |

### 8-2. LDtk 통합

- **월드 기믹:** LDtk Entities 레이어에 `BuilderPlatform`, `CollapseChute`, `BuilderTerminal`, `WarpNode` Entity 등록
- **아이템계 기믹:** `ItemWorldTemplates.ts` 내 RoomType 추가: `DescentShaft`, `InnocentCatapult`, `CollapseZone`
- **상승 기류 샤프트:** LDtk 레이어에 updraft(4) IntGrid를 수직 연속 배치하는 것으로 충분. 별도 Entity 불필요
- **LDtk 필드명 규칙:** PascalCase 준수 (`BuilderPlatform`, `WarpNodeID`, `CollapseChute_ExitY`)

## 9. 검증 기준 (Acceptance Criteria)

### 9-1. 기능적 검증

- [ ] 상승 기류 샤프트에서 렐릭 없이 상승 가능한가
- [ ] 빌더 플랫폼이 터미널 조작 전에는 비활성 상태인가
- [ ] 붕괴 낙하 경로에서 낙하 후 올라오는 수단이 없을 때 아이템계/월드 규칙에 따라 처리되는가
- [ ] 워프 노드가 양쪽 발견 전에는 "연결 미완성" 상태를 표시하는가
- [ ] 침강 샤프트 하강 후 텔레포트 크리스탈로만 귀환 가능한가
- [ ] 캐터펄트 발사 방향이 플레이어 선택을 받는가
- [ ] 붕괴 낙하 지대의 타이밍 창이 `CollapseZone_Window_s` 값과 일치하는가
- [ ] 렐릭 연동 매트릭스의 "강화" 항목들이 실제로 기믹 사용 효율을 높이는가

### 9-2. 경험적 검증

- [ ] 상승 기류 샤프트를 처음 만났을 때 "이걸로 올라갈 수 있겠다"가 즉시 인지되는가
- [ ] 빌더 플랫폼이 느리게 이동하는 동안 세계의 수직 규모를 체감할 수 있는가
- [ ] 붕괴 낙하 경로 앞에서 "내려가면 어떻게 되지?"라는 긴장감이 느껴지는가
- [ ] 침강 샤프트 발견이 "지름길이 있었다"는 쾌감을 주는가
- [ ] 캐터펄트 발사 순간의 피드백(시각/청각)이 만족스러운가
- [ ] 붕괴 낙하 지대 타이밍 성공 시 "내가 해냈다"는 숙련 성취감이 있는가
- [ ] 플레이테스트 5회 기준 월드 기믹이 없는 루트와 있는 루트의 이동 시간 차이가 `15~30%` 범위인가
