> **용도:** 월드/아이템계 공용 placeholder 비주얼. 실제 적 데이터(SSoT)는 `Sheets/Content_Stats_Enemy.csv` 와 `Documents/Content/Content_Monster_Bestiary.md` 참조.
> **아트 SSoT:** `Documents/Design/Design_Art_Direction.md` §12 (breakable_01.png 절제 톤). 본 문서는 §12 팔레트·화법 표준을 그대로 따른다.

---

## 1. 개념

슬라임이 아니라 **풍화된 봉합천 + 의례 향유의 응고체**. BLAME!의 유기 폐기물(biomass)을 본·세피아 절제 톤으로 옮긴 것. "젖은 점액"이 아니라 "마른 천이 굳어 기어다닌다".

---

## 2. 외형 특징

| 항목 | 사양 |
|:-----|:-----|
| 셀 크기 | 32×32 |
| 프로포션 | 폭 24px / 높이 16~18px. 위로 솟은 둥근 돔이 아니라 **납작한 더미** |
| 외피 | 본 파치먼트 **수의 천 다발**. 단일 면이 아니라 4~5겹 천 레이어가 미세 디더링으로 구분 |
| 코어 | 중심부 **잉크 흑 응결점** 1점. 핵이자 약점. 공격 텔 시에만 노출 |
| 누액 | 바닥 접지면에 **세피아 엄버(#3A2818) 1px 자국** 1~2개. 향유가 새어 굳은 흔적. 움직일 때만 잠깐 늘어났다 끊김 |
| 악센트 | 가일트 봉인 인장 1점이 천 표면에 박혀 있음 ("누군가가 봉인했지만 풀린 것") |
| 돌출부 | 둥근 윤곽이 아니라 **천 끝자락이 4~6개 늘어진 누더기 실루엣**. BLAME! 폐기물의 비대칭성 |

## 4. 애니메이션 텔

| 상태 | 시각 신호 |
|:-----|:---------|
| Idle | 천 자락 1개가 1px 흔들림. 코어 점은 항상 잉크 흑(은폐 상태) |
| Move | 위아래 호흡 없음. **옆으로 미끄러지듯 끌리는 이동**. 바닥 누액 자국이 잠깐 늘어남 |
| Tell (공격 직전) | 코어가 잉크 → **버건디 1프레임 노출** → 천이 부풀어오름 |
| Attack | 천 자락 한 가닥이 채찍처럼 펼쳐짐 (또는 가일트 점 1개 탄환 발사) |
| Death | 천 결합 풀리며 파치먼트 조각 4~5개로 분해, 누액 자국 1~2초 잔류 |
