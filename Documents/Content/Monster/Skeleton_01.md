# Skeleton — `husk_surveyor` (본 측량사)

> **용도:** 월드/아이템계 공용 placeholder 비주얼. 실제 적 데이터(SSoT)는 `Sheets/Content_Stats_Enemy.csv` 와 `Documents/Content/Content_Monster_Bestiary.md` 참조.
> **아트 SSoT:** `Documents/Design/Design_Art_Direction.md` §12 (breakable_01.png 절제 톤). 본 문서는 §12 팔레트·화법 표준을 그대로 따른다.

---

## 1. 개념

격벽 측량 작업 중 굳어버린 인간형 잔존체. BLAME! Safeguard 하급 익스터미네이터의 실루엣을 본·세피아 톤으로 미라화한 형태이다. 의례적으로 봉인되었으나 시간이 흘러 다시 움직이기 시작한 측량 분견대의 잔존.

**전투 어휘:** Hollow Knight Husk/Crawlid 계열 **바디 터치 접촉형(Contact)**. 별도 공격 프레임 없음. 위협 가독성은 실루엣·자세·상시 가일트 슬릿이 담당한다.

---

## 2. 외형 특징

| 항목 | 사양 |
|:-----|:-----|
| 셀 크기 | **32×32** (Erda 동일 스케일, 1:1 PPU 정합. 2×2 타일 점유) |
| 프로포션 | ~3등신 (Erda 정합). 4등신 원화는 컨셉 SSoT로만 보존 |
| 두부 | 두개골이 아닌 **수의-감긴 두건 머리**. 본 파치먼트 천이 안면을 감싸고 슬릿 한 줄(눈 자리) 노출. **슬릿은 가일트(#A88A48) 상시 1~2px** — 활성 위협 신호 |
| 몸통 | 갈비뼈가 아니라 **풍화된 측량 하니스**. 잉크 흑 끈 + 세피아 가죽 벨트. 메달리온 1~2px로 압축 |
| 사지 | 건조된 인대 + 강철 지지대. 산화 청회색(#4A5660) 1px 점이 무릎에 박힘 (32×32 한도로 팔꿈치는 생략) |
| 무기 | 한쪽 팔이 짧은 **부러진 측량봉**으로 융합 (4~5px 둔단). 별도 손잡이 무기 없음. 봉 자체는 공격 도구가 아니라 식별자 |
| 자세 | 상체 1~2px 앞으로 기울임. 앞발에 무게, 뒷발 살짝 끌림 — "다가오는 중" 모션 시사 |
| 표면 | 본 파치먼트 지배 + 세피아 디더링으로 마름/금. 풀 아웃라인 사용 금지 |

## 3. 색상 분배

| 부위 | 색상 | 면적 |
|:-----|:-----|:----:|
| 면적 60%+ | 본 파치먼트 #C8B89A ~ #B5A580 | 주조 |
| 그림자/내부 | 세피아 엄버 #5A3E28 ~ #3A2818 | 디더 |
| 아웃라인 | 잉크 흑 #1A1410 (부분 끊김) | 윤곽 |
| **가일트 슬릿** | **#A88A48 (눈 슬릿 1~2px 상시)** | **상시 발광** |
| 가일트 강조 (옵션) | #A88A48 (이마 점) | 0~1점 |
| 산화 청회색 | #4A5660 (무릎 1px) | 2점 |

> **버건디(#6E2C20) 폐기:** 공격 텔이 사라졌으므로 본 자산에서 사용 안 함. 보스 슬롯 전용 색으로 격하.

## 4. 애니메이션 세트

> **공격 프레임 없음.** 바디 터치 접촉형이므로 Attack 애니가 존재하지 않는다. Hollow Knight Husk·Crawlid 표준.

| 상태 | 프레임 | 시각 신호 |
|:-----|:---:|:---------|
| Idle | 2~3 | 1~2px 위아래 호흡 (좀비 흔들림). 슬릿 가일트 상시 |
| Walk | 4 | 무릎이 거의 굽지 않는 **삐걱대는 직립 보행**. 후드 자락 1~2px 뒤로 트레일. 무릎 청회색 점이 1프레임만 깜빡 (기계 관절 시사) |
| Death | 3~4 | 좌측으로 쓰러지며 두건 천 1조각 분리, 마지막 프레임에 슬릿 가일트 소멸 |

→ Attack 프레임 작업 0. 애니 작업량 표준 잡몹 대비 33% 감소.

## 5. 전투 패턴 (Contact 아키타입)

| 단계 | 동작 |
|:---|:---|
| Idle | 정지/미세 호흡. 슬릿 가일트 상시 발광 |
| Detect | 플레이어 감지. 슬릿 1프레임 밝아짐 (또는 후드 1px 흔들림) |
| Chase | 지상 보행으로 플레이어에게 접근 |
| Damage | **신체 접촉 시 데미지 발생.** 별도 공격 프레임 없음. 데미지 빈도는 AttackCooldown으로 제어 (`Sheets/Content_Stats_Enemy.csv`) |
| Death | HP 0 → Death 애니 |

**아키타입:** `Content_Monster_Bestiary.md` §1 신규 **Contact** 슬롯. Residue(수동)·Sentinel(능동 텔 공격)과 구분.
