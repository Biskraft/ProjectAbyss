## 1. 개요 (Concept)

### 1.1. 설계 의도 (Intent)

> **"말하지 않는 사람이 벽을 두드리고, 검 속에 빨려 들어가고, 강해져서 돌아와 벽을 부순다. 그것을 보고 플레이어가 스스로 이해한다."**

첫 30분은 단 한 줄의 대사도 없다. NPC도, 퀘스트도, 튜토리얼 텍스트도 없다. 에르다는 킬리가 메가스트럭처를 걷듯이 대공동을 내려가고, 플레이어는 그녀의 행동을 관찰하며 세계와 규칙을 이해한다.

이 30분이 끝났을 때 플레이어는 다음을 체득해야 한다:
1. 이동, 점프, 공격의 조작감
2. "검 속에 들어갈 수 있다"는 핵심 판타지
3. "아이템계에서 싸우면 장비가 강해진다"는 핵심 루프
4. "강해지면 못 가던 곳을 갈 수 있다"는 메트로베니아 훅
5. "이것을 반복하면 더 깊이 갈 수 있다"는 루프 인식

이 모든 것을 대사 한 마디 없이 전달한다.

### 1.2. 설계 근거 (Reasoning)

| 결정 | 근거 |
| :--- | :--- |
| 대사 완전 삭제 | 에르다는 킬리 오마주 과묵 주인공. "보여주되 설명하지 않는다" 원칙. BLAME! vol.1 첫 화에 대사가 거의 없듯이 |
| UI 힌트 없는 환경 교습 | 레벨 구조가 조작을 가르친다. ICO, Journey, INSIDE의 방법론. 힌트가 없을 때 플레이어는 더 주의 깊게 관찰한다 |
| ATK 게이트 2회 경험 | 1회: "막혔다". 2회: "아, 또 이거네. 강화하면 되겠다." — 반복으로 루프를 가르침 |
| 아이템계 진입 전에 게이트 먼저 노출 | "왜 강해져야 하는지"를 먼저 심어야 아이템계 보상이 의미를 갖는다 |
| 모루 = 마르타의 흔적 | 모루 텍스처에 다른 사람의 망치 자국이 있다. 지금은 의미 없지만, 마르타를 알게 되면 소급적으로 깨닫는다 |
| 첫 아이템계 사망 특례 | 첫 진입에서 추방하면 재시도 마찰이 너무 높다. 보스 HP 유지 + 내부 리스폰으로 반드시 클리어 가능하게 |

### 1.3. 3대 기둥 정렬

| 기둥 | 이 30분이 기여하는 방식 |
| :--- | :--- |
| 메트로베니아 탐험 | ATK 게이트 2회 노출. "못 가는 곳"과 "강해져서 뚫는 쾌감"을 30분 안에 2회 경험 |
| 아이템계 야리코미 | 검 속에 들어가서 싸우고, 강화되어 나오는 핵심 경험. "다시 들어가고 싶다"가 30분 끝의 감정 |
| 온라인 멀티플레이 | 이 30분에서는 다루지 않음. 솔로 경험에 집중 |

### 1.4. 감정 곡선

```
[고요/경외]    발코니에서 심연을 내려다봄
→ [호기심]      걷기 시작. 이 사람은 누구인가
→ [학습]        점프, 전투를 환경에서 배움
→ [좌절]        ATK 게이트 — 못 뚫음. 돌아감
→ [발견]        텅 빈 바닥의 검. 줍는다
→ [의문]        모루. 에코로 두드린다. 왜?
→ [충격/경외]   바닥 붕괴. 아래로 떨어짐
→ [긴장]        낯선 공간. 기억의 풍경. 전투
→ [도전/몰입]   보스전. 사망. 재도전. 패턴 학습
→ [성취]        보스 처치. "ATK +8" — 강해졌다
→ [쾌감]        이전에 3타 걸리던 적을 1타 컷!
→ [돌파감]      ATK 게이트 파괴. 새 공간!
→ [다시 좌절]   ATK 게이트 #2 — 또 못 뚫음
→ [이해/갈망]   바닥에 또 아이템. 모루로 돌아감. "아, 이 루프구나"
```

## 2. 시간대별 상세 설계

### Phase A — 도입: 고독한 걸음 (0:00 ~ 3:00)

#### SCREEN 1 — 발코니 (The Shaft 조망)

**LDtk 레벨:** `World_Level_0`

검은 화면에서 페이드 인. 에르다가 대공동 가장자리 발코니에 서 있다. 카메라는 먼저 심연의 깊이를 보여준 뒤, 천천히 에르다에게 포커스한다.

| 항목 | 내용 |
| :--- | :--- |
| 연출 | 5초간 패럴랙스 배경 — 아래로 끝없이 이어지는 대공동. 안개와 희미한 빛 |
| 에르다 행동 | 난간에 기대어 아래를 내려다봄 → 고개를 들고 → 돌아서서 걷기 시작 |
| 대사 | 없음 |
| 조작 | 에르다가 돌아선 후 플레이어에게 조작권 이전 |
| 설계 의도 | 세계의 규모감 각인. "이 사람은 왜 아래를 보고 있었는가?" — 답은 게임 전체를 통해 |
| 카메라 | entireLevel = true. 발코니 전체를 보여주는 넓은 앵글 |

---

#### SCREEN 2 — 성채 복도 (첫 걸음)

**LDtk 레벨:** `World_Level_17`

발코니에서 안쪽으로 이어지는 성채 복도. 평지. 장애물 없음. 창문 사이로 대공동의 빛이 스며든다.

| 항목 | 내용 |
| :--- | :--- |
| 교습 목표 | 이동 (Walk) |
| 난이도 압력 | 없음. 순수 이동 |
| 환경 디테일 | 벽에 금이 간 석재. 오래된 횃불 거치대 (불 꺼짐). 먼지 파티클 |
| 설계 의도 | 플레이어가 이동에 익숙해지는 안전 구간. 환경으로 "이곳은 오래되었고, 사람이 적다"를 전달 |

**LDtk 레벨:** `World_Level_32`

복도 끝에 계단이 아래로 이어지지만, 중간이 무너져 있다. 2~3타일 간격.

| 항목 | 내용 |
| :--- | :--- |
| 교습 목표 | 점프 (Jump) |

#### SCREEN 4 — 좁은 틈 (교습: 타이밍 점프)

**LDtk 레벨:** `WorldEntrance` 전반부

통로가 좁아지며, 천장에서 주기적으로 돌이 떨어지는 구간. 타이밍 맞춰 지나가야 한다. 대시 렐릭 획득 후 재방문하면 대시로 쉽게 통과 가능 — 메트로베니아 재방문 보상.

| 항목 | 내용 |
| :--- | :--- |
| 교습 목표 | 타이밍 점프. 위험 요소 인지 |
| 위험 요소 | 낙석 — 피격 시 HP 10% 감소 (치명적이지 않음) |
| 설계 의도 | "관찰 → 타이밍 → 통과"를 환경 위험으로 가르침. 적 없이 |
| 재방문 보상 | 대시 렐릭 획득 후 대시로 간단히 통과 가능 |
| 주의 | 키 바인딩(Z/X)은 화면 우하단에 작고 투명하게 표시. 첫 조작 후 자동 페이드 아웃 |

---

**LDtk 레벨:** `WorldEntrance` 후반부

넓은 방. 벽 쪽에 슬라임 1마리가 배회. 에르다를 감지하면 느리게 접근.

| 항목 | 내용 |

| 히트 피드백 | 히트스탑 2f + 넉백 4px + 화면 흔들림 1px. `System_Combat_HitFeedback.md` 참조 |

### Phase C — 첫 번째 막힘 (8:00 ~ 10:00)

#### SCREEN 7 — ATK 게이트 #1

**LDtk 레벨:** `World_Level_31` 후반부 → `World_Level_33` 입구

통로 끝에 거대한 결정 장벽. 에르다가 에코로 두드리면 "킹-" 소리와 함께 튕겨남. 파편 하나 떨어지지 않는다.

| 항목 | 내용 |
| :--- | :--- |
| 게이트 조건 | ATK >= 25 (현재 에르다 기본 ATK: 10. 돌파 불가) |
| 거부 연출 | 에코가 장벽에 부딪히면 불꽃이 튀고 에르다가 1타일 뒤로 밀려남. 장벽에 ATK 수치 요구량이 희미하게 빛남 |
| 에르다 행동 | 장벽을 한 번 더 두드려봄 → 같은 결과 → 잠시 멈춤 → 돌아감 |
| 대사 | 없음 |
| 설계 의도 | "지금은 못 간다." 메트로베니아 핵심 훅 설치. ATK 수치가 빛나므로 플레이어는 "ATK를 올리면 되겠다"를 직감 |
| 시스템 참조 | `System_World_StatGating.md` |

### Phase D — 발견: 바닥의 검 (10:00 ~ 13:00)

#### SCREEN 8 — 텅 빈 방, 바닥의 검

**LDtk 레벨:** `World_Level_29`

게이트에서 돌아온 에르다가 다른 경로로 진입하는 방. 텅 비어 있다. 방 중앙 바닥에 검 하나가 놓여 있다.

| 항목 | 내용 |
| :--- | :--- |
| 아이템 | 낡은 검 (Old Sword). Magic 등급. 기본 ATK 20 |
| 배치 | 방 정중앙 바닥. 빛이 위에서 내려와 검만 비춘다 |
| 상호작용 | 접근 → 자동 줍기 (또는 하단 상호작용 키) |
| 환경 디테일 | 검 주변 바닥에 긁힌 자국. 누군가 여기서 오랫동안 검을 끌고 다닌 흔적 |
| 설계 의도 | 텅 빈 공간에 검 하나만 있는 구도. "이것은 누구의 것이었는가?"를 묻지 않아도 느끼게 |

---

#### SCREEN 9 — 모루 (The Anvil)

**LDtk 레벨:** `WorldEntrance2` (중앙 대형 방)

에르다가 검을 들고 이동하면 도달하는 공간. 방 한쪽에 모루가 있다. 모루 표면에 수많은 망치 자국이 새겨져 있다 — 에르다의 것이 아닌, 다른 사람의 흔적.

| 항목 | 내용 |
| :--- | :--- |
| 모루 상호작용 | 모루 앞에서 상호작용 → 에르다가 검을 모루 위에 올려놓고 에코를 든다 |
| 마르타의 흔적 | 모루 텍스처에 에르다의 에코와 다른 크기/각도의 타격 자국. 지금은 의미 없음. 나중에 소급적으로 깨달음 |
| 설계 의도 | 모루는 숨겨진 균열 제단. 마르타가 10년간 이 위에서 아이템계에 진입했기 때문에 균열이 형성됨 |

### Phase E — 핵심 사건: 아이템계 (13:00 ~ 22:00)

#### SCREEN 10 — 기억의 두드림: 바닥 붕괴

에르다가 에코로 모루 위의 검을 두드리는 순간.

```yaml
echo_strike_sequence:
  context: "모루 = 마르타가 남긴 숨겨진 균열 제단. 첫 기억의 두드림."

  step_1:
    action: "에코가 검에 닿음 (타격)"
    duration_frames: 1
    note: "에르다의 평범한 행위. 두드리는 것이 이 사람의 습관"

  step_2:
```

### Phase F — 돌파 (22:00 ~ 26:00)

#### SCREEN 15 — 강화 체감

**LDtk 레벨:** `WorldEntrance2` → `WorldEntrance` 방향

#### SCREEN 16 — ATK 게이트 #1 돌파

**LDtk 레벨:** `World_Level_31` 후반부 → `World_Level_33`

에르다가 SCREEN 7에서 막혔던 게이트로 다시 간다.

| 항목 | 내용 |
| :--- | :--- |
| 게이트 조건 | ATK >= 25. 현재 에르다 ATK: 18 (10 기본 + 8 강화) |
| 돌파 방법 | 에코로 두드림 → 결정 장벽에 금이 감 → 두 번째 타격 → 장벽 파괴 |
| 파괴 연출 | 파편이 폭발적으로 흩어짐. 카메라 흔들림. 뒤에 새 공간이 보인다 |
| 설계 의도 | "막힘→강화→돌파" 첫 사이클 완성. 쾌감의 정점 |

> **밸런스 노트:** 기본 ATK 10 + 강화 8 = 18. 게이트 요구 25. 수치상 부족하지만, 아이템계에서 획득한 이노센트(Gladiator Lv.10, Tamed → ATK +10)로 총 ATK 28 달성 가능. 또는 게이트 요구치를 18로 조정하여 이노센트 없이도 돌파 가능하게 할 수 있음. 플레이테스트로 결정.
