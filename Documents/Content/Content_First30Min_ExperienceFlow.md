## 1. 개요 (Concept)

### 1.1. 설계 의도 (Intent)

> **"말하지 않는 사람이 벽을 두드리고, 검 속에 빨려 들어가고, 강해져서 돌아와 벽을 부순다. 그것을 보고 플레이어가 스스로 이해한다."**

첫 30분은 단 한 줄의 대사도 없다. NPC도, 퀘스트도, 튜토리얼 텍스트도 없다. 에르다는 킬리가 메가스트럭처를 걷듯이 대공동을 내려가고, 플레이어는 그녀의 행동을 관찰하며 세계와 규칙을 이해한다.

이 30분이 끝났을 때 플레이어는 다음을 체득해야 한다:
1. 이동, 점프, 대시, 공격의 조작감
2. "검 속에 들어갈 수 있다"는 핵심 판타지 (스파이크)
3. "아이템계에서 싸우면 장비가 강해진다"는 핵심 루프
4. "강해지면 못 가던 곳을 갈 수 있다"는 스탯 게이트
5. "이것을 반복하면 더 깊이 갈 수 있다"는 루프 인식
6. "벽을 타거나 바닥을 찍을 수 있으면 갈 수 있는 곳이 있다"는 메트로베니아 훅 (복선)

이 모든 것을 대사 한 마디 없이 전달한다.

### 1.2. 설계 근거 (Reasoning)

| 결정 | 근거 |
| :--- | :--- |
| 대사 완전 삭제 | 에르다는 킬리 오마주 과묵 주인공. "보여주되 설명하지 않는다" 원칙 |
| 대시 = 기본 능력 | 30분 데모에서 렐릭 획득 과정을 넣으면 스파이크(아이템계) 집중도가 희석됨. 대시는 전투+탐험의 기본 도구 |
| 렐릭 없음 | 30분은 "무기 강화" 루프에만 집중. 능력 해금은 Phase 2에서 |
| ATK 게이트 2회 경험 | 1회: "막혔다". 2회: "아, 또 이거네. 강화하면 되겠다." — 반복으로 루프를 가르침 |
| 벽 타기/바닥 찍기 힌트만 | 높은 벽에 아이템이 보이지만 올라갈 수 없음, 금 간 바닥이 보이지만 깰 수 없음 — 메트로베니아 "나중에 와야지" 훅 |
| 모루 = 마르타의 흔적 | 모루 텍스처에 다른 사람의 망치 자국. 나중에 소급적으로 깨달음 |
| 첫 아이템계 사망 특례 | 첫 진입에서 추방하면 재시도 마찰이 너무 높다. 보스 HP 유지 + 내부 리스폰 |

### 1.3. 3대 기둥 정렬

| 기둥 | 이 30분이 기여하는 방식 |
| :--- | :--- |
| 아이템계 야리코미 | 검 속에 들어가서 싸우고, 강화되어 나오는 핵심 경험. "다시 들어가고 싶다"가 30분 끝의 감정. **스파이크 집중** |
| 메트로베니아 탐험 | ATK 게이트 2회 + 벽 타기/바닥 찍기 힌트. "못 가는 곳"의 존재를 인식 |
| 온라인 멀티플레이 | 이 30분에서는 다루지 않음. 솔로 경험에 집중 |

### 1.4. 감정 곡선

```
[고요/경외]    발코니에서 심연을 내려다봄
→ [호기심]      걷기 시작. 이 사람은 누구인가
→ [학습]        점프, 대시, 전투를 환경에서 배움
→ [좌절]        ATK 게이트 — 못 뚫음. 돌아감
→ [발견]        텅 빈 바닥의 검. 줍는다
→ [의문]        모루. 에코로 두드린다. 왜?
→ [충격/경외]   바닥 붕괴. 아래로 떨어짐
→ [긴장]        낯선 공간. 기억의 풍경. 전투
→ [도전/몰입]   보스전. 사망. 재도전. 패턴 학습
→ [성취]        보스 처치. "ATK +8" — 강해졌다
→ [쾌감]        이전에 걸리던 적이 더 빨리 죽는다!
→ [돌파감]      ATK 게이트 파괴. 새 공간!
→ [호기심]      높은 벽/금 간 바닥 — "나중에 올 수 있을까?"
→ [다시 좌절]   ATK 게이트 #2 — 또 못 뚫음
→ [이해/갈망]   바닥에 또 아이템. 모루로 돌아감. "아, 이 루프구나"
```

## 2. 시간대별 상세 설계

### Phase C — 첫 번째 막힘 (8:00-10:00)

**LDtk 레벨:** `World_Level_31` 후반부

통로 끝에 거대한 결정 장벽. 에코로 두드리면 튕겨남.

| 항목 | 내용 |
| :--- | :--- |

| 에르다 행동 | 장벽을 한 번 더 두드려봄 → 같은 결과 → 잠시 멈춤 → 돌아감 |
| 대사 | 없음 |
| 설계 의도 | "지금은 못 간다." ATK 수치가 빛나므로 "ATK를 올리면 되겠다"를 직감 |

### Phase E — 핵심 사건: 아이템계 (13:00-22:00)

#### SCREEN 10 — 아이템계 진입

에르다가 에코로 모루 위의 검을 두드리는 순간. 진입 연출(MemoryDive)이 재생되고 아이템계로 전환.

```yaml
echo_strike_sequence:
  context: "모루 = 마르타가 남긴 숨겨진 균열 제단."

  step_1:
    action: "에코가 검에 닿음 (타격)"
    duration_frames: 1

  step_2:
    action: "히트스탑 + 공명 시작"
    duration_frames: 10
    effect: "화면 정지. 고주파 공명음. 모루 표면에 빛나는 균열 문양"

  step_3:
    action: "MemoryDive 연출 시작"
    effect: "아이리스 축소 + 색상 드레인 + 아이템계 전환"
    audio: "BGM 전환 — 성채 앰비언트 → 아이템계 음악"
```

## 5. 검증 체크리스트

### 시스템 기준 (Mechanical)
- [ ] 대시가 기본 능력으로 작동하는가? (렐릭 없이)
- [ ] 발코니 → 교습(점프+대시+전투) → 게이트 → 검 발견 → 모루 → 아이템계 → 보스 → 귀환 → 돌파 → 2차 게이트 전체 흐름이 막힘 없이 진행되는가?
