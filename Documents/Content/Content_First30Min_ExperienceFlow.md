## 1. 개요 (Concept)

### 1.1. 설계 의도 (Intent)

> **"첫 30분은 에르다가 대장장이를 그만두는 이야기가 아니라, 대장장이이기 때문에 세계로 나가게 되는 이야기다."**

첫 30분은 플레이어에게 게임의 규칙을 가르치는 튜토리얼이 아니다. 영웅 서사의 원형(Hero's Journey)을 따라 에르다가 자신의 정체성과 부딪히는 사건을 보여주면서, 그 사건이 자연스럽게 게임의 3대 핵심 루프(탐험 → 아이템계 → 강화 → 탐험)를 가르치도록 설계한다.

셀레스트의 첫 다리 씬처럼 — 드라마적 사건이 게임의 핵심 감각을 처음으로 전달하는 순간이어야 한다.

### 1.2. 설계 근거 (Reasoning)

| 설계 결정                                    | 이유                                                                                           |
| :------------------------------------------- | :--------------------------------------------------------------------------------------------- |
| 바닥 붕괴 대신 에코 공명 → 검 포탈로 진입   | 시스템 일관성: 에코가 아이템계 진입 도구라는 규칙과 완전히 정합. 연출 비용도 낮음             |
| Magic 등급 검이 의뢰 아이템                  | "동화 30개짜리인데 이상하다"는 에르다의 감지가 플레이어에게 신뢰감을 줌. 더 깊은 첫 경험 제공 |
| 허브 도착을 30분 후반으로 배치               | 아이템계 → 귀환 → 내러티브 씨앗 → 허브 순서로 루프 개념을 먼저 경험시킨 뒤 소셜 공간 도입   |
| 능력 게이트를 숲길에서 먼저 노출             | "지금은 못 가지만 나중엔 갈 수 있다"는 메트로베니아 훅을 핵심 사건 이전에 심어 기대값 형성   |
| STR 게이트를 첫 아이템계 직후 경험           | 아이템계 루프가 어떻게 월드 탐험과 연결되는지 플레이어가 스스로 깨닫도록 순서 배치           |
| No.1 씨앗을 최소 연출로 처리                 | Act 3의 반전을 위한 씨앗. 지금은 "이상한 부착물" 수준. 과도한 강조는 Act 1 톤을 망침         |

### 1.3. 3대 기둥 정렬 (Pillar Alignment)

| 기둥             | 이 30분이 기여하는 방식                                                                 |
| :--------------- | :-------------------------------------------------------------------------------------- |
| 메트로베니아 탐험 | 숲길 능력 게이트 노출, STR 15 게이트 노출 → "아직 못 가는 곳" 개념 2회 심기           |
| 아이템계 야리코미 | 첫 아이템계 진입 → 보스 처치 → "공격력 +8 영구 강화" UI로 핵심 루프 체험             |
| 온라인 멀티플레이 | 허브 공간 도입 + 기억의 사서 NPC + 오렌 NPC로 사회적 공간의 존재 암시                 |

### 1.4. 감정 곡선 목표

```
[호기심/친근함] 숲길 이동
→ [긴장/낯섦] 적 첫 등장
→ [당황/드라마] 에코 공명 사건
→ [집중/파밍 쾌감] 아이템계 전투
→ [성취/발견] 보스 처치 + 강화 UI
→ [궁금증] 구체 씨앗 + 마르타 쪽지
→ [안도/따뜻함] 허브 도착
→ [의욕/자발] "이번엔 내가 직접"
```

## 2. 영웅 서사 구조 매핑 (Hero's Journey Mapping)

첫 30분은 Joseph Campbell의 영웅 서사 원형 중 첫 7단계를 따른다.

| 단계 | 영웅 서사 명칭            | 시간         | 장소           | 에르다의 상태                                  |
| :--- | :------------------------ | :----------- | :------------- | :--------------------------------------------- |
| 1    | 일상 세계 (Ordinary World) | 0:00 ~ 8:00  | 숲길           | 대장장이 에르다. 의뢰 수행 중. 평범한 아침     |
| 2    | 모험의 부름 (Call)         | 8:00 ~ 12:00 | 벤-나흐트 대장간 | 에코가 공명. 검 포탈로 빨려 들어감             |
| 3    | 시련 (Tests)               | 12:00 ~ 18:00 | 아이템계 내부  | 낯선 공간. 전투. 기억의 수문장 처치            |
| 4    | 귀환 (Return)              | 18:00 ~ 19:00 | 대장간         | 강화된 검 확인. 에코에 구체 부착              |
| 5    | 거부 (Refusal)             | 19:00 ~ 22:00 | 대장간         | "오늘은 끝이야." 수리 완료. 평상심 유지 시도   |
| 6    | 스승 등장 (Mentor)         | 22:00 ~ 24:00 | 대장간         | 마르타의 쪽지 발견. 에코의 비밀 암시. 셀라 목격 |
| 7    | 첫 관문 돌파 (Threshold)   | 24:00 ~ 30:00 | 허브           | 허브 입성. 오렌과 만남. STR 게이트 발견. 자발적 의지 |

## 3. 씬별 상세 설계 (Screen-by-Screen Design)

### 3.1. Act 1 — 일상 세계 (0:00 ~ 8:00)

#### SCREEN 0 (도입부)

**형식:** 검은 화면 텍스트

```
의뢰서: 낡은 검 수리.
선불 30동화.
장소: 성채 외곽 벤-나흐트 대장간.
의뢰인 이름: 없음.
```

**설계 의도:** 플레이어에게 에르다의 현재 상황과 동기를 30자 안에 전달한다. "이름 없는 의뢰인" 디테일은 Act 2의 미스터리를 심는 씨앗이기도 하다.

#### SCREEN 9 — 기억의 두드림: 첫 균열 (THE MOMENT)

이 씬은 게임 전체에서 가장 중요한 단일 연출 순간이다. 셀레스트의 첫 다리 씬에 해당하는 게임의 감정적 닻점이다.

> **내러티브 근거:** 에르다의 대장간 모루는 스승 마르타가 사용하던 것이다. 마르타는 이 모루 위에서 수많은 아이템계에 진입했다. 10년간의 반복된 진입이 모루에 **기억의 균열**을 남겼고, 이 모루는 사실상 **숨겨진 균열 제단**이 되어 있었다. 에르다가 에코로 두드린 순간, 모루에 잠들어 있던 균열이 Magic 등급 검의 기억에 반응하여 처음으로 열린 것이다.
>
> 에르다는 이 사실을 모른다. 나중에(Act 2~3) 필드 균열 제단을 발견하고 "이것도 모루랑 같은 원리네"라고 깨달으면서 점차 이해하게 된다. 그리고 마르타의 쪽지("에코가 처음 울렸다면")가 이 모루의 존재를 마르타가 알고 있었다는 증거가 된다.

**연출 순서 (구현 명세):**

```yaml
first_echo_strike_sequence:
  context: "에르다의 모루 = 마르타가 남긴 숨겨진 균열 제단. 첫 기억의 두드림."

  step_1:
    action: "에코가 검에 닿음 (타격)"
    duration_frames: 1
    note: "평범한 수리 행위의 시작. 에르다도 플레이어도 이것이 진입 트리거가 될 줄 모른다"

  step_2:
    action: "히트스탑"
    duration_frames: 10
    effect: "화면 정지. 타격 사운드 컷. 고주파 공명음 시작"
    note: "평범한 수리와 확연히 다른 반응. '뭔가 이상하다'의 첫 신호"

  step_3:
    action: "모루 반응"
    effect: "모루 표면에 희미한 균열 문양이 빛나기 시작. 에코가 진동"
    note: "모루에 잠들어 있던 균열이 깨어나는 연출. 에르다가 놀라서 에코를 놓으려 하지만 진동이 멈추지 않음"

  step_4:
    action: "검의 기억 반응"
    effect: "모루의 균열이 검으로 전이. 검 표면에 빛의 균열이 퍼짐"
    note: "모루(제단)의 균열 에너지가 검(아이템)의 기억을 깨운 것. SYS-IW-01 § 2.1 '기억의 두드림' 시퀀스와 동일 원리"

  step_5:
    action: "포탈 생성"
    effect: "검의 균열이 확장 → 빛의 포탈 분출"
    camera: "화면 흔들림 (진폭 큼)"

  step_6:
    action: "에르다 흡입"
    effect: "에르다가 포탈 방향으로 당겨짐. 모루 가장자리 잡음. 손을 놓침"
    audio: "에르다: 'ああああ—!'"
    note: "필수 대사. 유머와 공포가 동시에 있어야 함. 첫 진입이므로 비자발적 — 이후 진입은 에르다가 능동적으로 두드림"

  step_7:
    action: "씬 전환"
    effect: "화면 완전 백색 → BGM 컷 → 무음 0.5초 → 아이템계 BGM 시작"
```

**설계 의도:**

| 요소 | 설명 |
| :--- | :--- |
| 구현 복잡도 | 낮음. 히트스탑 + 스프라이트 발광 + 카메라 흔들림 + 씬 전환. 커스텀 쉐이더 불필요 |
| 시스템 정합 | `SYS-IW-01` § 2.1 "기억의 두드림" 연출과 동일 원리. 모루 = 제단, 에코 = 열쇠, 검 = 대상 아이템 |
| 첫 진입 vs 이후 진입 | 첫 진입만 비자발적(끌려 들어감). 이후 모든 진입은 에르다가 스스로 무기를 올리고 두드리는 능동적 행위 |
| 모루 = 제단 복선 | 이 시점에서 플레이어는 모루가 제단이라는 것을 모른다. 이후 필드 제단을 발견하고 "모루랑 같은 원리"임을 깨달음 |
| 마르타 연결 | 모루에 균열이 있는 이유 = 마르타가 10년간 이 위에서 아이템계에 진입했기 때문. Screen 16 마르타 쪽지의 근거 |

#### SCREEN 20-21 — 지하 균열 구역 (STR 게이트 노출)

**배경:** 지하 균열 구역. 스켈레톤 전투 1~2회 (숲길에서 만난 적과 동일).

**소재 수집:**

| 항목          | 내용                                                        |
| :------------ | :---------------------------------------------------------- |
| 수집 목표     | 소재 아이템 3개                                             |
| 전투 난이도   | 숲길과 동일하거나 약간 높음. 충분히 해결 가능               |
| 설계 의도     | 새로운 공간에서 배운 것을 활용하는 구간                     |

**STR 15 게이트 — 핵심 루프 클릭 포인트:**

```yaml
str_gate_encounter:
  gate_type: "스탯 게이트 (STR 15 요구)"
  visual: "잠긴 문. STR 15 수치 표시"
  erda_monologue:
    line: "STR 15. 나는 아직... 아까 아이템계에서 검 수치가 올랐으니까, 더 돌면 이것도 열리겠지."
    mandatory: true
    design_note: "에르다가 스스로 핵심 루프를 언어화한다. 플레이어가 '아 저 문을 열려면 아이템계를 더 해야 하는구나'를 확인하는 순간"
```

**설계 의도:** 이 독백이 울리는 순간, 플레이어는 게임의 핵심 순환 구조를 완전히 이해한다. 튜토리얼 텍스트 없이, 에르다의 추론을 통해 플레이어도 같은 결론에 도달한다.
