## 0. 컨텍스트

플레이어는 Phase A~C (SCREEN 1~7)를 이미 경험했다:
- 발코니에서 심연을 조망 (세계 규모감)
- 이동/점프/대시/전투를 환경에서 학습
- ATK 게이트 #1에 막힘 (ATK >= 30 필요, 현재 Starter Blade ATK 25)
- 게이트에서 돌아와 다른 경로를 탐색 중

**플레이어 상태:** 조작은 익숙해졌지만 진행이 막힌 상태. "어떻게 강해지지?"

## SCENE 1: 검의 발견 (픽업)

**LDtk 레벨:** `World_Level_29`
**추정 플레이타임:** 10:00~11:00

### 환경

텅 빈 방. 방 중앙 바닥에 낡은 검이 놓여 있다. 위에서 빛 한 줄기가 내려와 검만 비춘다. 검 주변 바닥에 긁힌 자국 -- 오래전 누군가가 여기서 검을 끌고 다닌 흔적.

### 시퀀스

| 단계 | 트리거 | 연출 | 조작 |
| :--- | :--- | :--- | :--- |
| 1 | 플레이어가 검에 접근 | 자동 줍기. 에르다가 검을 집어 든다 | 조작 일시 정지 (1초) |
| 2 | 픽업 완료 | **첫 아이템 획득 연출**: 화면 중앙에 아이템 카드 팝업. "Rustborn / Normal / ATK 15" 표시. 검의 실루엣이 빛나며 회전 | 확인 버튼 또는 자동 진행 (3초) |
| 3 | 아이템 카드 닫힘 | **Ego 각성**: 검날에 빛이 스친다. 고주파 공명음 | 대사 시작 |

### 첫 아이템 획득 연출 상세

```yaml
item_acquire_cutscene:
  trigger: "Rustborn 픽업"
  scope: "핸드크래프트 무기 최초 획득 시에만. 드롭 무기는 이 연출 없음"

  phase_1_card:
    duration: 3s
    visual: "화면 중앙 아이템 카드. 검 실루엣 + 이름 + 등급 + ATK"
    audio: "아이템 획득 효과음 (금속 공명)"
    input: "아무 키 → 카드 닫힘. 또는 3초 자동 진행"

  phase_2_ego_wake:
    duration: 0.5s
    visual: "카드 닫힌 직후. 에르다의 손에 든 검날이 한 번 빛남"
    audio: "고주파 공명 -- 짧고 맑은 소리. BGM과 별개"
    camera: "약간 줌인 (에르다 상반신 + 검)"
```

### Ego 각성 대사

검날이 빛나며 텍스트 박스 표시. 대사는 게임 화면 하단에 표시되며, 검날 빛이 발화에 동기화.

```
Rustborn: "밝아."

Rustborn: "오랜만이야, 빛."
```

> 2문장. 텍스트 박스 자동 진행 (1.5초 간격). 에르다는 답하지 않는다.
> 바이올렛 1화의 "소령님의 눈동자와 같은 색입니다" -- 감정이 아니라 감각을 보고하는 패턴.

**대사 종료 후:** 조작권 즉시 반환. 플레이어는 자유롭게 이동 가능.

## SCENE 2: 검과의 첫 대화 (이동 중)

**LDtk 레벨:** `World_Level_29` → `WorldEntrance2` 방향 이동 중
**추정 플레이타임:** 11:00~11:30

### 트리거

SCENE 1 종료 후 플레이어가 **첫 발걸음을 내딛는 순간** (이동 입력 감지).

### 시퀀스

| 단계 | 트리거 | 대사 | 비고 |
| :--- | :--- | :--- | :--- |
| 1 | 이동 시작 | Rustborn: "안이 이상해. 원래 이렇지 않았던 것 같은데." | 이동 중 대사. 플레이어 조작 차단 없음 |
| 2 | 1번 대사 종료 (2초 후) | Rustborn: "너는 왜 여기 있는 거야?" | 에르다에게 직접 묻는다 |
| 3 | 2번 대사 종료 (2초 후) | (대사 없음. 에르다는 답하지 않는다. 침묵이 흐른다) | 1.5초 빈 간격 |
| 4 | 침묵 후 | Rustborn: "너도 말 안 하는 쪽이야." | 에르다의 과묵함을 인식하고 받아들인다 |

> **설계 의도:** 이동 중에 대사가 나온다. 플레이어는 멈출 필요가 없다. Transistor의 Unknown이 Red가 걷는 동안 말하는 것과 동일한 구조. 이동과 대화가 분리되지 않는다.
>
> **조작:** 차단 없음. 대사는 하단 텍스트 박스에 자동 진행. 전투가 시작되면 대사 큐 일시 정지.

## SCENE 3: 앤빌 발견 + 사용법 교습

**LDtk 레벨:** `WorldEntrance2`
**추정 플레이타임:** 11:30~13:00

### 환경

방 한쪽에 모루(Anvil)가 있다. 모루 위에 [UP] 키 프롬프트가 표시된다.

### 시퀀스

| 단계 | 트리거 | 연출/대사 | 기능 |
| :--- | :--- | :--- | :--- |
| 1 | 앤빌에 접근 (proximity 범위 진입) | Rustborn: "여기. 이걸 알고 있어." | 앤빌 인식 |
| 2 | [UP] 키 입력 | **앤빌 UI 오픈**: 인벤토리에서 아이템 선택 화면 | Anvil Interaction |
| 3 | Rustborn 선택 + 확인 | **MemoryDive 연출 즉시 시작** | 아이템계 진입 |

### 앤빌 UI 상세

```yaml
anvil_ui:
  open_trigger: "앤빌 proximity + UP키"
  layout: "인벤토리 아이템 목록. 현재 Rustborn 1개만 있으므로 선택지 1개"

  rustborn_slot:
    display: "Rustborn / Normal / ATK 15"
    ego_indicator: "아이템 이름 옆에 작은 빛 아이콘 (Ego 무기 표시)"

  confirm:
    trigger: "아이템 선택 후 확인 버튼"
    action: "앤빌 UI 닫힘 → MemoryDive 연출 즉시 시작"
```

### MemoryDive 전환 연출

```yaml
memory_dive:
  trigger: "앤빌 UI에서 아이템 선택 + 확인"

  step_1:
    action: "공명 시작"
    duration_frames: 10
    visual: "화면 정지. 모루 표면에 빛나는 균열 문양"
    audio: "고주파 공명음"

  step_2:
    action: "색상 드레인 + 아이리스 축소"
    duration_ms: 800
    visual: "화면이 모노크롬으로 변하며 검 중심으로 수축"

  step_3:
    action: "화면 전환"
    visual: "검은 화면 0.5초 → 아이템계 배경 페이드 인"
    audio: "BGM 전환 -- 성채 앰비언트 → 아이템계 음악"
```

## SCENE 4: 아이템계 진입 직후

**맵:** 아이템계 1지층, 절차 생성. **첫 Normal 특례: 3x3 그리드**
**추정 플레이타임:** 13:00~13:30

### 환경

착지. 낯선 공간. 아이템 기억으로 이루어진 세계 -- 황갈색 색조, 흙길, 나무 판벽의 흔적이 구조물 형태로 존재. 조명이 어둡다.

### 시퀀스

| 단계 | 트리거 | 대사 | 연출 |
| :--- | :--- | :--- | :--- |
| 1 | 착지 완료 (0.5초 후) | Rustborn: "여기야. 내 안." | 카메라가 살짝 넓게 -- 공간을 보여줌 |
| 2 | 1번 대사 종료 (1.5초 후) | Rustborn: "어두워. 원래는 이렇지 않았어." | 환경이 어둡다는 것을 Ego가 확인 |
| 3 | 플레이어가 첫 적 시야 범위 진입 | Rustborn: "저건 있으면 안 되는 것들이야." | 왜곡 몬스터를 가리킴 |

> 대사 3은 첫 적이 화면에 보이는 순간 트리거. 적과의 거리가 아니라 카메라 프레임 진입 기준.
> 조작 차단 없음. 대사는 이동/전투 중에도 표시.

### 환경 교습 (비대사)

| 요소 | 교습 내용 | 방식 |
| :--- | :--- | :--- |
| 문 마커 | 초록 발광하는 문 -- 다음 방으로 이동 | 시각적 어포던스 |
| 탈출 제단 | 빛나는 원형 패턴 -- 월드로 귀환 | SCREEN 마지막에 사용 |
| 방 구조 | 3x3 그리드의 첫 방. 적 1~2마리 | 최소 난이도 |

## SCENE 5: 아이템계 전투 (3방)

**추정 플레이타임:** 13:30~18:00

### 방 1: 첫 전투

| 항목 | 내용 |
| :--- | :--- |
| 적 | 슬라임 1 (HP 80). Starter Blade ATK 25 기준 4타 처치 |
| 트리거 대사 | 적 처치 후 -- Rustborn: "없어졌어. 지금 네가 한 건 나를 위한 거야?" |
| 설계 의도 | 바이올렛 패턴: 타인의 행동을 관찰하고 그 의도를 질문한다. 감사가 아니라 사실 확인 |

> 이 대사가 핵심이다. "나를 위한 거야?"는 Rustborn이 **에르다의 행동에 의미를 부여하기 시작하는 순간**이다. 바이올렛이 "사랑한다를 알고 싶습니다"로 시작했듯, Rustborn은 "나를 위한 거야?"로 시작한다.

### 방 2: 공간 전투

| 항목 | 내용 |
| :--- | :--- |
| 적 | 스켈레톤 1 (HP 100, 상단 발판) + 슬라임 1 |
| 트리거 대사 | 없음. 전투에 집중하게 둔다 |
| 환경 | 높이 차이가 있는 발판 구조. 점프 + 전투 복습 |

### 방 3: 기억 단편 조우

| 항목 | 내용 |
| :--- | :--- |
| 적 | 슬라임 2 + 스켈레톤 1. **기억 단편 NPC 1체가 방 구석에 있음** |
| 기억 단편 상태 | 형태 불안정. 반투명. 가까이 가면 뒤로 물러남 |
| 주변 왜곡 | 기억 단편 반경 3타일 이내에 슬라임 1이 배회 |

**기억 단편 조우 시퀀스:**

| 단계 | 트리거 | 대사/연출 |
| :--- | :--- | :--- |
| 1 | 기억 단편가 화면에 보임 | Rustborn: "저건 내 일부야." |
| 2 | 1번 대사 종료 (1.5초 후) | Rustborn: "떨고 있어. 원래는 저렇지 않았는데." |
| 3 | 기억 단편 주변 왜곡 몬스터 전멸 | **안정화 연출**: 기억 단편 형태가 선명해짐. 색이 맑아짐 |
| 4 | 안정화 완료 | 기억 단편: "괜찮아졌어. 고마워." |
| 5 | 기억 단편 대사 후 | Rustborn: "따뜻해졌어. 이 아이 때문인 것 같아." |
| 6 | 시스템 | **기억 단편 안정화 UI**: "Wind Memory stabilized. ATK +5%" 표시 |

```yaml
memory shard_stabilize:
  trigger: "기억 단편 주변 반경 3타일 내 모든 왜곡 몬스터 전멸"

  visual_sequence:
    - "기억 단편의 형태가 0.5초에 걸쳐 안정된다"
    - "반투명 → 불투명. 경계선이 선명해짐"
    - "기억 단편 주변에 작은 빛 파티클 발생"

  audio: "정화 효과음 -- 맑은 종소리계"

  ui_popup:
    text: "Wind Memory stabilized"
    subtext: "ATK +5%"
    duration: 2s
    position: "화면 중앙 상단"

  mechanic: "기억 단편 스탯 보너스 즉시 적용. Rustborn의 ATK에 영구 반영"
```

> **설계 의도:** 플레이어는 기억 단편를 직접 처치하지 않는다. 주변의 왜곡을 없애면 자연스럽게 안정화된다. "싸우는 이유"가 명확해진다 -- 주민을 구하기 위해. 바이올렛이 편지를 대필하면서 타인의 감정을 배웠듯, Rustborn은 기억 단편 안정화를 통해 감정을 배운다.

## SCENE 6: 보스 -- 기억의 수문장

**추정 플레이타임:** 18:00~22:00

### 보스 방 진입

| 단계 | 트리거 | 연출/대사 |
| :--- | :--- | :--- |
| 1 | 보스 방 문 통과 | 문이 뒤에서 잠김. 넓은 아레나. 중앙에 그림자 검사 실루엣 |
| 2 | 보스 등장 연출 (1초) | 그림자 검사가 형태를 갖추며 검을 든다. 얼굴 없음 |
| 3 | 연출 종료 직후 | Rustborn: "크다." |
| 4 | 보스 HP 바 표시 | 하단 중앙 200x6. "Guardian" 이름 표시 |

> 1문장. 사실만 말한다. 바이올렛 패턴 -- 두려움을 표현하는 법을 모르니까 크기만 보고한다.

### 보스 전투 상세

```yaml
boss_guardian:
  name: "Guardian"
  grade: "아이템 장군 (지층 1 전용)"

  stats:  # 첫 Normal 특례 적용
    hp: 420  # 기본 600 x 0.7
    atk: 30
    def: 5

  patterns:
    1_charge: "좌우 돌진. 벽에 부딪히면 1초 경직"
    2_slam: "점프 내려찍기. 착지점 2타일 범위"
    3_swipe: "근접 스와이프. 전방 3타일"

  first_entry_special:
    enrage: false  # 첫 진입 격노 없음
    death_penalty: false  # 사망 시 내부 리스폰
    respawn_point: "SCENE 4 착지점"
    respawn_hp: "50%"
    mob_respawn: false  # 잡몹 리스폰 안 함
    boss_hp_retain: true  # 보스 HP 유지
```

### 전투 중 대사

| 트리거 | 대사 | 조건 |
| :--- | :--- | :--- |
| 보스 HP 50% | (대사 없음 -- 검날 빛이 강하게 한 번 깜빡) | 시각 피드백만 |
| 플레이어 사망 시 | Rustborn: "괜찮아?" | 리스폰 직후 1회만 |

> 전투 중 대사는 최소화. 전투 집중을 방해하지 않는다. "괜찮아?"는 Rustborn이 에르다의 상태를 **처음으로 걱정하는** 대사다. 바이올렛이 타인의 울음을 처음 인식한 순간과 대응.

### 보스 처치 시퀀스

| 단계 | 타이밍 | 연출/대사 | 시스템 |
| :--- | :--- | :--- | :--- |
| 1 | 처치 즉시 | 그림자 부서짐. 금색 섬광 + 12f 히트스탑 | -- |
| 2 | 처치 후 1초 | **강화 UI**: 화면 중앙 큰 텍스트 "Rustborn -- ATK +8 (Permanent)" | ATK 반영 |
| 3 | UI 표시 후 1초 | Rustborn: "없어졌어." | -- |
| 4 | 3번 대사 후 1.5초 | Rustborn: "이게 뭐라고 하는 거지, 이런 느낌." | **첫 번째 감정적 전환점** |

```yaml
boss_clear:
  kill_visual: "그림자가 부서진다. 파편이 금색 빛으로 변하며 위로 흩어짐"
  kill_audio: "파괴 SFX + 1초 뒤 정적 + 맑은 공명음"
  hitstop: 12  # frames

  reward_ui:
    delay: 1000  # ms after kill
    text: "Rustborn"
    subtext: "ATK +8 (Permanent)"
    format: "화면 중앙. 큰 폰트. 주황색 (#ff8833)"
    duration: 3000  # ms
    audio: "레벨업 효과음"

  ego_dialogue:
    line_1:
      delay: 1000  # ms after UI
      text: "없어졌어."
    line_2:
      delay: 1500  # ms after line_1
      text: "이게 뭐라고 하는 거지, 이런 느낌."

  post_clear:
    environment: "방이 밝아진다. 어두웠던 조명이 따뜻한 색으로 전환"
    door_open: "탈출 제단으로 향하는 문이 열림"
```

> **"이게 뭐라고 하는 거지"** -- 바이올렛의 "'이런 것'을 뭐라고 부르면 좋을까요"(1화)와 동일한 구조. 감정이 발생했으나 이름을 붙이지 못한다. 안도인지, 기쁨인지, 감사인지 -- 플레이어가 채운다.

## SCENE 8: 월드 귀환 + 강화 체감

**LDtk 레벨:** `WorldEntrance2` → 이전 구간
**추정 플레이타임:** 23:00~26:00

### 귀환 직후

| 단계 | 트리거 | 연출 |
| :--- | :--- | :--- |
| 1 | 모루 앞 복귀 | 에르다가 모루 앞에 선다. Rustborn 장착 상태 유지 |
| 2 | 복귀 후 1초 | (대사 없음. 검날 빛이 잔잔하게 깜빡 -- Ego가 살아있다는 시각적 표시) |

> **Ego 없음.** 귀환 후에는 Rustborn이 말하지 않는다. 월드에서는 침묵. 아이템계에서만 말한다 -- 아이템계가 Rustborn의 "집"이니까.

### 강화 체감

플레이어가 이전 구간으로 돌아가면 리스폰된 적과 다시 싸운다.

| 적 (Lv.1) | 강화 전 (Starter Blade ATK 25) | 강화 후 (Rustborn ATK 23 = 15+8) | 비고 |
| :--- | :--- | :--- | :--- |
| 슬라임 (HP 80) | 4타 | 4타 | Rustborn 단독으로는 아직 약함 |
| 스켈레톤 (HP 100) | 5타 | 5타 | Starter Blade가 여전히 강함 |

> **주의:** Rustborn은 Normal ATK 15 + 강화 8 = ATK 23. Starter Blade ATK 25보다 **아직 약하다.** 그러나 기억 단편 보너스(ATK +5%)와 향후 추가 아이템계 클리어로 Rustborn이 Starter Blade를 역전한다. 이것이 "아이템계를 다시 돌려야 하는" 동기다.
>
> ATK 게이트 #1 조건 = ATK >= 30. Rustborn(23) + 기억 단편 보너스로는 아직 부족. **2회차 아이템계 클리어가 필요.** 이것이 루프를 완성한다.

## SCENE 9: ATK 게이트 돌파 (2회차 클리어 후)

**추정 플레이타임:** 26:00~30:00 (2회차 아이템계 포함)

### 2회차 아이템계 진입

| 항목 | 내용 |

| 대사 | Rustborn: "다시 왔어." (1문장만. 대사 감쇠 규칙) |

| 보스 | 동일 Guardian, **정상 규칙 적용** (HP 600, 격노 있음) |

2회차 클리어 후 Rustborn ATK = 23 + 8 = **31**. ATK 게이트 #1 (>= 30) 돌파 가능.

### 게이트 돌파

| 단계 | 트리거 | 연출 |

| 2 | 돌파 직후 | (대사 없음. Rustborn은 월드에서 말하지 않는다) |
| 3 | 새 공간 진입 | Tier 2 심부 환경. 새로운 적, 새로운 아이템 |

> **"막힘 → 강화 → 돌파" 첫 사이클 완성.** 플레이어는 이 루프를 체득한다.
