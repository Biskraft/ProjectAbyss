# 첫 30분 경험 흐름 — v2: 대공동 (The Shaft)

> **MVP 경험 플로우.** BLAME! 오마주 메가스트럭처. 과묵한 주인공.
> 대체 문서: `Content_First30Min_ExperienceFlow.md` (v1, 다화형 에르다)

---

## 0. 설계 공리

| 공리 | 규칙 |
|:--|:--|
| 에르다는 말하지 않는다 | 독백 없음. 생각 풍선 없음. 플레이어 캐릭터 대사 0줄. |
| 세계가 스스로 설명한다 | 건축, 조명, 적 배치, 아이템 배치 — 이것들이 곧 튜토리얼이다. |
| 폐소감 우선 | 모든 방은 좁게 시작한다. 넓은 공간은 획득하는 것이다. |
| 위기는 눈에 보여야 한다 | 플레이어는 메가스트럭처가 증식하고, 벽이 좁혀오고, 없던 복도가 생기는 것을 직접 목격해야 한다. |
| 아이템계 = 계시 | 첫 다이브가 플레이어가 "이 게임이 뭔지" 이해하는 순간이다. |

## 1. 월드 맵 (메가스트럭처 단면도)

```
                    ┌─────────────────────────────┐
                    │  Tier 2: 중앙 성채            │
                    │  (세이브 포인트 — 그림자 시장)   │
                    │                              │
  ┌─────┐  ┌───────┤  [에르다의 대장간]─[시장]      │
  │증식 │  │봉인된 │  [발코니]──[대홀]             │
  │구역 │  │복도   │  [문서고]─[이렌의 서재]       │
  │(새) │  │(게이트)│                              │
  └──┬──┘  └───┬───┤──────────────────────────────┘
     │         │                    │
     │    [격벽 2]             [대공동 전망]
     │    (STR 게이트)         (수직 낙하,
     │         │                Tier 3이
     │         ▼                저 아래 보임)
     │   ┌──────────┐
     │   │ Tier 3:  │
     │   │ 카타콤   │  ← MVP에서 잠김
     │   └──────────┘
     │
  [증식 복도]
  새 벽이 자라나고 있음.
  슬라임이 새 콘크리트에서
  기어나옴.
  막다른 길 — 증식하는
  벽에 막힘.
```

**첫 30분 크리티컬 패스:**

```
대장간 → 봉인된 복도 → 대장간 (막힘)
  → 모루 타격 → 바닥 붕괴 → 아이템계
  → 보스 클리어 → 대장간 귀환
  → 증식 구역 발견 (없던 복도가 생김)
  → 격벽 2에서 STR 게이트 확인
  → 그림자 시장 도달
  = 30분 종료
```

## 2. 스크린별 상세 설계

### Act 1: 대장간 (0:00 ~ 3:00)

#### SCREEN 1 — 에르다의 대장간 (스폰)

어두운 방. 석벽. 중앙에 모루. 벽에 도구들. 램프 하나.
에르다가 모루 옆에 서 있다. 에코를 오른손에 들고 있다.

| 항목 | 내용 |
|:--|:--|
| 튜토리얼 | 없음. 플레이어가 방을 탐험하며 이동을 스스로 익힌다. |
| 아이템 | 모루 위에 검이 놓여 있다. 격벽 틈에서 꺼낸 것 (Magic 등급). |
| 상호작용 | 모루 근처에서 ↑ → 에르다가 검을 집어 살핀다. |
| 시각 단서 | 검이 희미하게 파란빛을 낸다 (Magic 레어리티). |
| 환경 스토리텔링 — 균열 지도 | 벽에 대공동 단면 지도가 핀으로 꽂혀 있다. 격벽 위치마다 마커가 있고, 여러 개에 X표(봉합 완료), 하나에 ?표(미조사). 의뢰서 따위는 없다. |
| 환경 스토리텔링 — 도구 | 선반에 격벽 조각 샘플, 균열 측정 도구, 에코 충전대. 에르다가 이 일을 오래 해왔다는 것을 보여줌. |
| 키리이 대응 | 키리이의 망막 분석 정보 = 에르다의 균열 지도. 둘 다 "찾고 있다"는 사실을 한 장면에 보여줌. |

대사 없음. 텍스트 프롬프트 없음. 플레이어가 움직이고, 검을 보고, 상호작용한다. 에르다가 누구인지, 왜 이 일을 하는지 설명하지 않는다. 행동만 있다.

---

#### SCREEN 2 — 봉인된 복도 (교습: 이동 + 점프)

대장간 문을 지나면: 오른쪽으로 뻗은 복도. 천장에 석재 파이프. 어두운 산업 조명.

| 항목 | 내용 |
|:--|:--|
| 교습 | 이동 (좌/우), 점프 (바닥 틈새, 2타일 너비) |
| 적 | 없음. 순수 이동 학습 구간. |
| 건축 | BLAME! 스타일: 거대한 파이프, 리벳 박힌 금속 벽, 에르다를 왜소하게 만드는 스케일 |
| 복도 끝 | 봉인된 격벽 문. 가장자리에 균열이 보인다. |

플레이어가 봉인된 문에 도달한다. 통과 불가. 대장간으로 돌아가야 한다.

**설계 의도:** 플레이어는 이제 막힌 길이 있다는 것을 안다. 이것이 메트로베니아 훅 — "나중에 다시 오면 열 수 있겠지."

---

#### SCREEN 3 — 대장간 귀환 + 첫 전투 (교습: 전투)

대장간으로 돌아오면 복도 입구 근처에 슬라임 2마리가 나타나 있다.

| 항목 | 내용 |
|:--|:--|
| 교습 | 공격 (Z키). 슬라임은 느리고 2히트에 처치. 첫 히트스탑 경험. |
| 감각 | 플레이어의 첫 타격이 3프레임 동안 화면을 멈춘다. "이거 좋은데." |
| 아이템 드롭 | 슬라임 하나가 Normal 검을 드롭. 위를 지나가면 획득. |

슬라임 처치 후, 플레이어는 모루 위에 의뢰 검이 있는 대장간으로 돌아온다.

---

### Act 2: 다이브 (3:00 ~ 8:00)

#### SCREEN 4 — 에코 타격 (THE MOMENT)

플레이어가 의뢰 검이 있는 모루에 접근한다. ↑으로 상호작용.

에르다가 검을 모루 위에 올린다. 에코를 든다. 내려친다.

```yaml
echo_strike_sequence:
  step_1:
    action: "에코가 모루 위의 검을 타격"
    duration_frames: 1

  step_2:
    action: "히트스탑"
    duration_frames: 10
    effect: "화면 정지. 모든 사운드 컷. 고주파 공명음 시작."
    note: "이건 평범한 타격이 아니다. 뭔가 이상하다."

  step_3:
    action: "모루 균열 발광"
    effect: "모루 표면의 미세 균열이 빛나기 시작. 에코가 진동."
    duration_ms: 1500

  step_4:
    action: "바닥 붕괴"
    effect: "FloorCollapse.ts — 모루 아래 타일이 아래로 부서짐"
    camera: "화면 흔들림 (진폭: 대)"
    note: "에르다가 떨어진다. 검이 떨어진다. 모든 것이 떨어진다."

  step_5:
    action: "백색 섬광 → 씬 전환"
    effect: "완전 백색 → 0.5초 무음 → 아이템계 BGM 시작"
```

**대사 없음. 설명 없음.** 바닥이 무너진다. 에르다가 떨어진다. 플레이어는 이제 검 안에 있다.

---

#### SCREEN 5 — 아이템계 착지 (공간 도입)

에르다가 금속성 공간에 착지한다. 벽에 결정 무늬가 있다 — 검의 칼날과 같은 무늬.

| 항목 | 내용 |
|:--|:--|
| 시각 | 벽이 파란빛으로 일렁인다 (Magic 레어리티 색상). 좁은 복도. |
| 플레이어 인식 | "검 안에 있는 거야...?" — 검과 일치하는 시각 디자인만으로 전달. |
| 첫 방 | 작다. 오른쪽으로 출구 하나. |

---

#### SCREEN 6 — 아이템계 전투 (3~4개 방)

4x4 그리드 지층. 절차적 생성 방에 적 배치.

| 항목 | 내용 |
|:--|:--|
| 적 | InnocentNPC (적대, 야생). 에르다에게 돌진하는 붉은 구체. |
| 전투 | 월드와 동일한 조작. 히트스탑, 넉백, 데미지 넘버. |
| 진행 | 방 클리어 → 아이템 EXP 축적 (HUD에 작은 카운터로 표시). |
| 방 수 | 보스 전 3~4개 방. 좁은 복도, 발판, 수직 통로. |

---

#### SCREEN 7 — 기억의 수문장 보스

아레나 방. 에르다 뒤로 문이 잠긴다.

| 항목 | 내용 |
|:--|:--|
| 보스 | 기억의 수문장 (아이템 장군). HP 80, ATK 12. |
| 패턴 | 패턴 1: 수평 돌진 (좌우 왕복). 패턴 2: 점프 내려찍기. |
| 텔레그래프 | 각 공격 전 0.5초 준비 동작. 2~3회 시도면 학습 가능. |
| 사망 | 보스가 파티클로 폭발. |

**보스 클리어 보상:**

```yaml
boss_clear:
  ui_display:
    text: "ATK +8"
    format: "화면 중앙 큰 텍스트. 금색. 화면 섬광."
    timing: "보스 사망 1초 후 표시"
  mechanical_effect:
    item_exp: 600  # Lv0 → Lv2 (ATK 15 → 23)
  feel: "숫자가 나타나는 그 순간이 THE moment. 말은 필요 없다."
```

---

#### SCREEN 8 — 탈출 제단 (귀환)

보스 방 직후: 빛나는 제단. 100% 스폰 (첫 방문 보장).

| 항목 | 내용 |
|:--|:--|
| 상호작용 | 제단 근처 ↑ → 백광이 에르다를 감쌈 → 씬 전환 |
| 귀환 장소 | 에르다의 대장간 (진입 장소). |
| 시각 | 떨어졌던 것의 역재생 — 빛이 위로 올라간다. |

---

### Act 3: 증식 (8:00 ~ 20:00)

#### SCREEN 9 — 대장간 귀환 + 발견

에르다가 대장간에 돌아왔다. 모루에 타격한 자리에 균열이 있다.

**핵심 발견:** 원래 단단한 벽이었던 곳에 새 복도가 나타났다. 새 콘크리트. 에르다가 검 안에 있는 동안 벽이 자란 것이다.

| 항목 | 내용 |
|:--|:--|
| 시각 | 새 벽면 — 더 밝은 색 석재, 아직 따뜻함. 이음새를 따라 균열. |
| 적 | 슬라임 2마리가 새 벽 재료에서 기어나온다. |
| 환경 스토리텔링 | 메가스트럭처가 자라고 있다. 이것이 위기다. |

---

#### SCREEN 10 — 증식 구역 (새 영역)

새 복도는 전에 없던 구역으로 이어진다. BLAME! 스타일: 동일한 복도, 반복되는 패턴, 방향 감각 상실.

| 항목 | 내용 |
|:--|:--|
| 건축 | 새 콘크리트 벽. 아무 데로도 이어지지 않는 파이프. 반쪽짜리 방. |
| 적 | 슬라임 + 스켈레톤 1마리 (첫 실전급 위협). |
| 교습 | 스켈레톤이 먼저 공격. 회피/반격 패턴 학습. |
| 막다른 길 | 복도가 증식하는 벽에서 끝남 — 플레이어가 콘크리트가 천천히 팽창하는 것을 본다. |
| 아이템 | 스켈레톤이 Rare 검을 드롭 (VS 플로우용 보장). |

**설계 의도:** 플레이어가 위기를 직접 본다. 세계가 좁아지고 있다. 벽이 자라고 있다. 에르다가 더 깊이 가야 하는 이유가 이것이다.

---

#### SCREEN 11 — 두 번째 다이브 (플레이어 주도)

플레이어에게 Rare 검이 있다. 대장간으로 돌아온다. 모루는 균열이 있지만 사용 가능.

**이번에는 플레이어가 직접 타격을 선택한다.** Rare 검 장착 상태에서 모루 앞 ↑.

같은 에코 타격 시퀀스이나 더 빠르게 (플레이어가 이제 예상하므로). 바닥 붕괴. Rare 검의 아이템계로 낙하.

| 항목 | 내용 |
|:--|:--|
| 시각 | Rare 검 = 황색 톤 벽. 더 강한 적. 더 깊은 느낌. |
| 지층 | 3지층 (Rare 등급). 플레이어는 지층 1, 어쩌면 지층 2까지 클리어. |
| EXP | Rare ATK_PER_LEVEL = 8. 지층 1 클리어 → Lv2 (ATK 26 → 42). |
| 탈출 | 각 지층 보스 후 제단. 플레이어가 떠날 시점을 선택. |

---

#### SCREEN 12 — 귀환 + STR 게이트 발견

대장간 복귀. 더 강한 무기.

Screen 2의 봉인된 복도가 이제 증식 구역을 통해 도달 가능 — 증식하는 벽이 우연히 우회 경로를 열었다.

| 항목 | 내용 |
|:--|:--|
| 새 경로 | 증식 구역 → 옆 복도 → 격벽 2 부근으로 연결. |
| STR 게이트 | 격벽 2 보임. 표시: "STR 40". |
| 플레이어 ATK | Rare Lv2 = 42 ATK면 간신히 통과. |
| 게이트 상호작용 | 접근 → 에르다가 에코로 타격 → 격벽에 균열 → 통로 개방. |
| 또는 | ATK 부족 시 → 에코가 튕겨남. 붉은 섬광. 더 강해져야 함. |

---

#### SCREEN 13-15 — 그림자 시장으로 가는 길

격벽 2 너머: 하강. 더 어둡고. 더 좁고. 그러다 갑자기 — 넓어진다.

| 스크린 | 내용 |
|:--|:--|
| 13 | 좁은 하강로. 스켈레톤 (2). 고스트 (1, 원거리). 첫 다수 적 전투. |
| 14 | 거대한 공동이 열린다. 대공동이 보인다 — 끝없는 수직 낙하, 먼 곳에 다른 층위가 보임. 경외의 순간. |
| 15 | 그림자 시장 입구. 등불. 군중의 소리 (오디오 추가 시). NPC들이 보인다. |

---

### Act 4: 그림자 시장 (20:00 ~ 30:00)

#### SCREEN 16 — 그림자 시장 (세이브 포인트 도착)

성채 하층에 숨겨진 시장. 비공식적. 좁은 노점, 매달린 등불, 증기 파이프.
에르다가 걸어 들어간다. 사람들이 에르다를 힐끗 본다. 에르다는 아무도 보지 않는다.

| 항목 | 내용 |
|:--|:--|
| NPC | 오렌 (정보상), 이렌 다스 (학자), 상인들 |
| 상호작용 | NPC 근처 ↑ → NPC가 말한다 (에르다는 침묵) |
| 키리이식 상호작용 | 에르다는 NPC에게 한 가지만 원한다: **균열 정보.** 오렌에게 접근하면 오렌이 "균열? 아래쪽에서 새로운 게 열렸다는 소문이 있어." 에르다 반응: 침묵. 아래를 본다. |
| 이렌 다스 | "증식이 가속화되고 있어. 격벽 3이 위험해." (정보 제공. 의뢰가 아님.) |
| 상점 | 무기 매매. 강화 재료. (MVP: 플레이스홀더) |
| 키리이 대응 | 키리이가 모든 사람에게 "넷 터미널 유전자를 가지고 있나?"만 묻듯, 에르다는 균열 정보만 찾는다. 돈도 안 받고, 의뢰도 안 받는다. |

#### SCREEN 17 — 30분 종료

에르다가 시장을 지나 대공동이 보이는 발코니에 선다.

아래가 보인다. 끝없는 어둠. Tier 3, 4, 5... 보이지 않는 아래.
에르다의 균열 지도에 ?표가 남아있다. 아래쪽.

에코가 희미하게 진동한다. 아래에서 무언가가 올라오는 것 같은 — 공명.

에르다가 아래를 본다. 한 발짝 앞으로.

**페이드 아웃.** 텍스트 없음. 통계 화면 없음. 침묵과 어둠뿐. 그리고 타이틀: **PROJECT ABYSS**.

**30분 종료.** 플레이어는 이제:
- 핵심 루프를 경험했다 (탐험 → 다이브 → 강화 → 게이트 → 새 영역)
- 위기를 보았다 (증식하는 벽)
- 세이브 포인트에 도달했다 (그림자 시장)
- 계속할 동기가 있다: **아래에 더 있다. 에코가 반응한다. 내려가야 한다.**
- 의뢰를 받은 게 아니다. 에르다가 스스로 가는 것이다.

## 6. v1 대비 변경점

|:--|:--|:--|

## 7. 구현 참고

### v1에서 재사용 가능
- FloorCollapse.ts (모루 바닥 붕괴)
- Anvil.ts (모루 엔티티)
- DialogueBox.ts (NPC 대화 전용, 에르다는 사용 안 함)
- Slime.ts, Skeleton.ts, Ghost.ts, GoldenMonster.ts
- InnocentNPC.ts (아이템계 적)
- ItemWorldScene.ts (절차적 지층)
- Portal/Altar 시스템

### 신규 필요
- 증식 구역 레벨 디자인 (증식 벽 시각 효과가 있는 LDtk 방)
- 증식 벽 엔티티 (시각: 콘크리트가 천천히 팽창, 상호작용 불가)
- 그림자 시장 레벨 (NPC 배치가 있는 LDtk 세이브 포인트 영역)
- STR 게이트 / 격벽 엔티티 (StatGate 스펙과 유사)
- NPC 대화 데이터 (오렌, 이렌 — 그들이 말하고, 에르다는 침묵)
- 발코니 씬 (대공동 전망 + 에코 공명 연출 + 페이드 아웃)
- 대공동 전망 배경 (끝없는 수직 낙하를 보여주는 패럴랙스 배경)

### MVP에서 제외
- 마르타 쪽지 (Act 2+ 이연)
- 세라 실루엣 (Act 2+ 이연)
- No.1 이노센트 씨앗 에코 부착 (이연)
- 의뢰/코인 시스템 (완전 제거. 에르다는 고용되지 않는다)
- 오디오 (별도 태스크, 이 스펙에 미포함)
