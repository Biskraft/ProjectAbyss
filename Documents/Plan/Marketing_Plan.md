### Phase 2: 불씨 키우기 (Ignite) — 데모 출시-Steam 페이지 오픈

**목표:** 데모 플레이어 확보 → Steam 위시리스트 축적 시작.

| 채널 | 행동 | KPI |
|:--|:--|:--|
| **itch.io 데모** | 데모 공개 + 피드백 수집 | 다운로드 1,000+ |
| **X** | 데모 출시 발표 + 플레이어 반응 RT | 팔로워 300+ |
| **Reddit** | r/indiegaming "Demo released" 포스트 | 업보트 100+ |
| **YouTube/TikTok** | 15-30초 스파이크 영상 | 조회수 1,000+ |
| **Steam 스토어 페이지** | 위시리스트 수집 시작 | 위시리스트 500+ (첫 달) |
| **인디 게임 미디어** | 메일 피칭 (프레스킷 + 데모 링크) | 커버리지 1-2건 |

**전환 퍼널:**

```
X/Reddit 포스트 (스파이크 GIF)
    ↓ 클릭
itch.io 데모 (즉시 플레이)
    ↓ 플레이 15분+
"Steam 버전에서 더 깊이" CTA
    ↓ 클릭
Steam 위시리스트 등록
```

**핵심 전환 포인트:**
- itch.io 데모 → Steam 위시리스트 전환율 목표: **15%+**
- 데모 플레이타임 15분+ 유저의 위시리스트 등록률: **25%+**

## 5. 바이럴 메커닉 설계

### 5.1. URL 즉시 플레이 (Phase 1)

```
플레이어가 X에 게임 링크 공유
→ 클릭한 사람이 5초 내 플레이 시작
→ "이게 뭐지?" → 스파이크 체험
→ Steam 위시리스트 또는 친구에게 공유
```

**Steam 게임은 이것이 불가능.** 설치→실행→튜토리얼→게임 시작까지 10분+. 웹은 5초.

### 5.2. 아이템계 URL 초대 (Phase 3)

```
플레이어 A가 아이템계 클리어
→ "친구 초대" 버튼 → URL 생성
→ 친구 B가 URL 클릭 → 즉시 파티 합류
→ 친구 B도 Steam 구매 동기 발생
```

**각 플레이어가 마케팅 엔진이 된다.** Among Us 모델.

### 5.3. 강화 결과 공유 (Phase 2)

```
아이템 강화 완료 시
→ "결과 공유" 버튼 → 이미지 자동 생성
→ "Rusty Sword Lv.1 → Legendary Blade Lv.47" 비포/애프터
→ X/Discord에 공유 → 야리코미 자랑 문화
```
