# Phase 1 프로토타입 — 구현 우선순위

> **작성일:** 2026-04-05
> **목적:** 스파이크 피처 검증을 위한 Phase 1 구현 순서 정의
> **핵심 질문:** "아이템 속으로 들어가는 30초가 재미있는가?"
> **선행 문서:** `Documents/Research/SpikeReview_PostRedesign_2026-04-05.md`

---

## 0. 원칙

1. **스파이크 우선:** 아이템계 진입 경험을 가장 먼저 만든다. 나머지는 이 경험을 뒷받침하는 만큼만.
2. **최소 루프 완성:** Tier 0~2까지 완성하면 핵심 순환이 돌아간다. 이 시점에서 재미를 판단한다.
3. **후순위는 재미 증명 후:** Tier 3은 Tier 0~2의 재미가 확인된 후에만 진행한다.

---

## 1. Tier 0 — 이것 없이는 30초가 불가능

> 아이템계에 들어가서 싸우고 나올 수 있는 최소 조건.

| 순서 | 시스템 | 구현 내용 | 참조 문서 | 완료 기준 |
|:----:|:-------|:----------|:----------|:----------|
| **1** | 3C (이동/점프) | 에르다 이동, 점프. 대시는 렐릭 해금 후 활성화 | `System_3C_Character.md`, `System_3C_Control.md` | 플랫폼 위를 자유롭게 이동하고, 점프로 장애물을 넘을 수 있다 |
| **2** | 검 전투 | 3타 콤보, 히트스탑 (2~4f), 넉백, 데미지(ATK-DEF) | `System_Combat_Action.md`, `System_Combat_Damage.md`, `System_Combat_Weapons.md` | 적을 때렸을 때 "맞았다"는 감각이 명확하다 |
| **3** | 아이템계 진입/탈출 | 장비 선택 → 진입 연출 → 내부 로드 → 탈출 | `System_ItemWorld_Core.md` | "아이템 속으로 들어간다"는 경험이 시각적으로 전달된다 |
| **4** | 절차적 방 생성 | 4×4 Room Grid, 템플릿 기반 방 조립, Critical Path 보장 | `System_ItemWorld_FloorGen.md` | 매번 다른 맵이 생성되고, 시작→끝 경로가 항상 존재한다 |
| **5** | 기본 적 3~5종 | 잡몹 AI (순찰/추적/공격), 피격/사망 애니, 아이템 드랍 | `System_Enemy_AI.md` | 적이 플레이어를 인지하고 공격하며, 처치 시 장비를 드랍한다 |

**Tier 0 완료 시점의 플레이:**
```
에르다가 장비를 들고 아이템계에 진입한다
→ 절차적으로 생성된 지층을 탐험한다
→ 적을 검으로 베어 처치한다
→ 아이템계를 탈출하여 돌아온다
```

## 2. Tier 1 — 30초를 "재미있게" 만드는 것

> 아이템계 안에서 "의미 있는 보상"을 얻고, 강해지는 느낌을 받는다.

| 순서 | 시스템 | 구현 내용 | 참조 문서 | 완료 기준 |
|:----:|:-------|:----------|:----------|:----------|
| **6** | 이노센트 (야생→복종) | 아이템계 내 야생 이노센트 조우, 격파 시 복종 (효과 2배) | `System_Innocent_Core.md` | 야생 이노센트를 격파하면 복종되고, 장비에 장착하면 효과가 체감된다 |
| **7** | 장비 ATK/INT/HP | 장비 스탯 표시, 이노센트 슬롯 UI, 장착/교체 | `System_Equipment_Slots.md`, `System_Growth_Stats.md` | 더 좋은 장비를 얻으면 "강해졌다"가 숫자로 보인다 |
| **8** | 아이템계 보스 1종 | 지층 끝 보스, 패턴 2~3개, 처치 시 보상 | `System_ItemWorld_Boss.md` | 보스를 쓰러뜨리면 "해냈다"는 달성감이 있다 |
| **9** | 레어리티 3등급 | Normal(흰)/Magic(파랑)/Rare(노랑), 스탯 배율 차등 | `System_Equipment_Rarity.md` | 파란 아이템이 뜨면 "오!" 하는 반응이 나온다 |

**Tier 1 완료 시점의 플레이:**
```
아이템계에서 야생 이노센트를 복종시킨다
→ Gladiator 이노센트를 장비에 장착 → ATK +5
→ 보스를 격파 → Magic 등급 검 드랍!
→ "이 검의 아이템계는 어떨까?" → 다시 진입 동기
```

## 3. Tier 2 — "한 번 더"를 만드는 것

> 순환 루프가 완성되어 "한 번만 더"가 발생한다.

| 순서 | 시스템 | 구현 내용 | 참조 문서 | 완료 기준 |
|:----:|:-------|:----------|:----------|:----------|
| **10** | 스탯 게이트 1개 | 월드에 "ATK ≥ 50" 장벽 1개 배치 | `System_World_StatGating.md` | 장벽 앞에서 "ATK가 부족하다"를 확인하고, 아이템계에서 강화 후 돌파한다 |
| **11** | 월드 방 2~3개 | 대장간(스폰/세이브) + 통로 + ATK 게이트 방 | `System_World_MapStructure.md` | 대장간 → 아이템계 → 강화 → 게이트 돌파의 전체 루프가 돌아간다 |
| **12** | 순환 진입 루프 | 아이템계에서 획득한 장비를 월드로 가져와 그 장비의 아이템계에 진입하는 순환 | `System_ItemWorld_Core.md` | 아이템계 → 월드 귀환 → 더 좋은 장비의 아이템계 진입 순환이 "끝없이 더 깊이" 느낌을 준다 |

**Tier 2 완료 시점의 플레이 — 최소 순환 루프:**
```
대장간 스폰 → 검으로 적 처치 → 장비 드랍
→ 장비의 아이템계 진입 (스파이크!)
→ 절차적 지층 탐험 → 이노센트 복종 → 보스 격파
→ 강화된 장비로 귀환 → ATK 게이트 돌파
→ 새 구역 진입 → 더 좋은 장비 드랍
→ 다시 아이템계 진입 (순환!)
→ 더 좋은 장비의 아이템계에 다시 진입 (순환!)
```

**이 루프가 재미있으면 PA의 스파이크가 증명된다.**

---

## 4. Tier 3 — 재미 증명 후 확장

> Tier 0~2가 재미있다는 확신이 생긴 후에만 진행한다.

| 시스템 | 구현 내용 | 조건 |
|:-------|:----------|:-----|
| 무기 +4종 (대검/단검/활/지팡이) | 각 무기별 콤보/히트박스/정체성 | 검 전투가 재미있을 때 |
| 에코 인챈트 (화/빙/뇌) | INT 스케일링, 원소 데미지, 100% 적중 | ATK 빌드가 작동할 때 |
| 행동형 이노센트 6종 | Berserker/Vampire/Ironclad/Ghost/Sprinter/Leech | 기본 이노센트(Gladiator/Dietician)가 재미있을 때 |
| 아이템 서사 (기억의 지층) | 무기별 테마/몬스터풀/환경풀 | 절차적 생성이 작동할 때 |
| 멀티플레이 (URL 초대) | WebSocket, 1~4인 아이템계 협동 | 솔로 아이템계가 재미있을 때 |
| 능력 게이트 (이단점프) | 이단점프 획득, 월드 수직 확장 | 월드 기본 구조가 작동할 때 |
| INT 게이트 (마법 봉인) | 지팡이/원소 빌드 전용 분기 | ATK 게이트가 작동할 때 |
| 추가 레어리티 (Legendary/Ancient) | 고레어리티 장비, 추가 이노센트 슬롯 | 3등급이 재미있을 때 |

## 5. 구현 판단 기준 (Go/No-Go)

### Tier 0 → Tier 1 진행 조건
- [ ] 아이템계 진입 연출이 "들어간다"는 감각을 전달하는가?
- [ ] 검 타격감이 "맞았다"는 피드백을 주는가?
- [ ] 절차적 생성 맵이 매번 다르면서도 플레이 가능한가?

### Tier 1 → Tier 2 진행 조건
- [ ] 이노센트 복종 시 "강해졌다"가 체감되는가?
- [ ] 보스 격파 후 "해냈다"는 달성감이 있는가?
- [ ] Magic/Rare 드랍 시 "좋은 거 나왔다"는 흥분이 있는가?

### Tier 2 → Tier 3 진행 조건 (스파이크 증명)
- [ ] 대장간 → 아이템계 → 강화 → 게이트 순환이 자연스럽게 반복되는가?
- [ ] 아이템계에서 더 좋은 장비를 얻어 다시 진입하는 순환에서 "끝이 없네" 반응이 나오는가?
- [ ] 플레이어가 자발적으로 "한 번만 더"를 하는가?

---

## 6. 예상 구현 순서 (시간 기준)

| 주차 | 목표 | 산출물 |
|:----:|:-----|:-------|
| 1~2주 | Tier 0 (#1~#2) | 에르다 이동 + 검 전투 |
| 3~4주 | Tier 0 (#3~#5) | 아이템계 진입 + 절차적 생성 + 적 |
| 5~6주 | Tier 1 (#6~#9) | 이노센트 + 장비 스탯 + 보스 + 레어리티 |
| 7~8주 | Tier 2 (#10~#12) | 월드 루프 + 스탯 게이트 + 순환 진입 |

> **8주 = 스파이크 증명 가능 시점.** 이 시점에서 Go/No-Go 결정.
