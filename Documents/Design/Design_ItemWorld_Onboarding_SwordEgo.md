## 1. 스파이크 재정의

### 1.1 기존 스파이크

> "아이템에 들어가면, 그 아이템의 **기억**이 던전이 된다"

이 정의에서 아이템계는 과거의 재현이다. 반복 진입할 때마다 같은 기억이 다른 형태로 재생된다. 이 프레임의 문제: 플레이어가 "왜 들어가야 하는가"에 대한 감정적 답이 없다. 기억은 이미 끝난 것이다. 끝난 것을 반복 체험할 동기는 약하다.

### 1.2 새 스파이크

> **"아이템에 들어가면, 그 안에 살아있는 세계가 있다"**

아이템계는 아이템 내부에 **지금 존재하는** 세계다. 이 세계에는 주민(기억 단편)이 살고 있고, 왜곡된 기억이 몬스터의 형태로 이 세계를 침식하고 있다. 플레이어가 들어가는 것은 과거를 감상하는 것이 아니라 **현재의 위기를 해결하는 것**이다.

### 1.3 두 스파이크의 관계

기존 스파이크를 폐기하는 것이 아니다. **포함 관계로 확장한다.**

```
기존: 아이템의 기억이 던전이 된다
  ↓ (확장)
변경: 아이템 안에 살아있는 세계가 있다.
      그 세계의 풍경은 아이템의 기억으로 이루어져 있다.
      그러나 세계 자체는 지금 살아있고, 주민이 있고, 위기가 있다.
```

**비유:** 아이템계는 "영화 상영관"이 아니라 "기억으로 지어진 마을"이다. 마을의 건물과 거리는 과거의 기억이지만, 마을 자체는 지금 살아있다. 주민이 있고, 문제가 있고, 도움이 필요하다.

### 1.4 야리코미와의 정합성

반복 진입이 서사적으로 성립하는 이유:

| 기존 (기억) | 변경 (살아있는 세계) |
| :--- | :--- |
| "같은 기억을 왜 또 보는가?" | "세계가 계속 침식당하고 있으니까 다시 도와야 한다" |
| 반복 = 기억의 다른 층위 | 반복 = 세계의 다른 영역이 왜곡에 감염됨 |
| 종결 = 모든 기억을 본다 | 종결 = 세계의 왜곡을 완전히 정화한다 |

## 2. 검의 Ego 설계

### 2.1 Ego란 무엇인가

Ego는 아이템이 가진 자의식이다. 모든 아이템에 Ego가 있는 것은 아니다. 오래되고, 깊은 역사를 가지며, 충분한 기억이 축적된 아이템만이 Ego를 발현한다.

**세계관적 위치:** 기억 단편가 "기억의 파편이 의지를 가진 존재"라면, Ego는 "아이템 전체의 기억이 하나의 자의식으로 수렴한 존재"다. 기억 단편가 주민이라면, Ego는 그 세계의 정령 또는 의지 그 자체다.

**Ego의 조건: 레어리티가 아니라 핸드크래프트 여부**

Ego는 레어리티와 무관하다. **월드에 핸드 플레이스된 무기(5~15개)만 Ego를 가진다.** 절차적으로 드롭되는 무기(285+)는 레어리티가 높더라도 Ego가 없다.

| 구분 | Ego | 수량 | 대사 |
| :--- | :--- | :--- | :--- |
| 핸드크래프트 무기 (월드 고정 배치) | **있음** | 5~15개 | 직접 작성 (무기별 고유) |
| 드롭/파밍 무기 (절차적 생성) | 없��� | 285+ | 없음 |

> Ego의 희소성이 감정적 무게를 만든다. 모든 무기가 말하면 아무 무기도 특별하지 않다.

**무기 수집 모델 (원신 참조):**

| 원칙 | 설명 |
| :--- | :--- |
| **Ego 무기 = 영구 수집** | 인벤토리에서 삭제/판매 불가. 컬렉션에 영구 보관 |
| **장착 교체 자유** | 전투 최적화를 위해 더 강한 무기로 자유 교체. 감정 부담 없음 |
| **미장착 Ego 무기도 진입 가능** | 장착하지 않은 Ego 무기도 세이브 포인트에서 아이템계 진입 가능 |
| **강화는 무기별 독립** | 각 Ego 무기는 자기 아이템계(살아있는 세계)를 가짐. 독립 강화 |
| **Ego와 대화** | 세이브 포인트에서 인벤토리의 Ego 무기 선택 시 대화 가능 |

> 플레이어는 "내러티브 캐릭터(Ego 무기 컬렉션)"와 "전투 캐릭터(장착 무기)"를 분리해서 관리한다. 강한 무기를 끼되, Ego 무기와의 관계는 유지된다.

### 2.2 첫 검의 Ego

#### 아이템 이름 변경

| 항목 | 기존 | 변경 |
| :--- | :--- | :--- |
| 이름 | 격벽 틈에서 꺼낸 낡은 검 | **Rustborn** |
| Ego 호칭 | 없음 | 자기 이름을 모른다. 플레이어가 픽업한 순간 의식이 깨어남 |
| 등급 | Magic | **Normal** (온보딩 난이도 정합. Normal이지만 핸드크래프트이므로 Ego 있음) |

> **"Rustborn"**: 녹에서 태어난. 격벽 속에서 오래 녹슬며 잠들어 있다가 에르다가 집어든 순간 깨어난 검. Ego 자체도 자기 과거를 완전히 기억하지 못한다. 이 불완전함이 캐릭터다.

#### Ego 성격

**캐릭터 롤모델:** Violet Evergarden + 아야나미 레이 혼합
- 바이올렛: "무기"였던 존재가 감정을 배워가는 과정. 에르다의 행동을 관찰하며 감정의 의미를 이해하려 함.
- 레이: 감정이 없는 게 아니라 표현하는 법을 모름. 말이 건조하지만 가끔 우연히 깊은 말을 함.

| 특성 | 설명 |
| :--- | :--- |
| **성별 뉘앙스** | 여성적. 목소리(텍스트 톤)가 차분하고 낮다. 감정을 과장하지 않는다 |
| **어조** | 건조하고 담담하다. 감정이 아니라 사실을 말한다. 문장이 짧고 군더더기가 없다 |
| **핵심 특질** | 감정 표현을 모른다. 오래 잠들어 있어서 감정이라는 것 자체를 잊었다. 에르다의 행동을 보며 하나씩 재학습한다 |
| **태도** | 경계가 아니라 **무관심에 가까운 관찰**. 에르다가 뭘 하는지 지켜본다. 점차 "이 사람은 왜 나를 돕는가"에 호기심을 가진다 |
| **핵심 감정** | 자기 존재 가치를 모른다. "나는 교체 가능한 도구"라고 생각한다. 에르다가 행동으로 그렇지 않다는 걸 증명해줄 때 변한다 |
| **한계** | 자기 과거를 모른다. 기억이 아니라 **감각**만 남아있다. "따뜻한 게 있었어. 뭔지는 모르겠어." |
| **에르다에 대한 인식** | 에르다가 말하지 않는다는 것을 **자연스럽게 받아들인다**. 자기도 말이 적으니까. "너도 말 안 하는 쪽이야." |
| **감정 각성 곡선** | 건조 → 호기심 → 당혹 ("이게 뭐지, 이 느낌") → 소중함의 자각 |

> **에르다 캐릭터 레퍼런스:** Half-Life 2의 Gordon Freeman + BLAME!의 키리이. 과묵하지만 행동으로 의지를 보여주는 주인공. 검 Ego가 말하면 에르다는 답하지 않고 앞으로 걷는다 -- 그것이 응답이다.
>
> **에르다 × Rustborn 관계:** 둘 다 말이 없다. 그래서 서로의 **행동**을 읽는다. Rustborn은 에르다의 행동에서 감정을 배우고, 에르다는 Rustborn의 세계를 지키는 것으로 답한다. 두 침묵 사이의 대화.

#### Ego 발화 규칙

| 상황 | 발화 | 빈도 제한 |
| :--- | :--- | :--- |
| **첫 획득 (픽업 직후)** | 각성 대사 | 1회 |
| **아이템계 첫 진입** | 세계 설명 + 도움 요청 | 1회 |
| **아이템계 내 트리거** | 특정 지점에서 짧은 코멘트 | 첫 진입에만. 재진입 시 침묵 |
| **기억 단편 최초 조우** | 기억 단편가 뭔지 설명 | 최초 1회 |
| **보스 방 진입** | 보스에 대한 반응 | 매 진입 1회 (짧게) |
| **보스 처치 후** | 정화 반응 + 감사 | 매 진입 1회 |
| **재진입 (2회차 이후)** | **침묵** 또는 1문장 이하 | Fi 반면교사. 반복 피로 방지 |
| **지층 최초 도달** | 새 지층에 대한 반응 | 각 지층 첫 방문만 |

> **원칙: "첫 경험에만 말하고, 반복에서는 침묵한다."** 야리코미 100회차에도 검이 말하면 Fi의 실수를 반복한다.

### 3.1 SCREEN 8 (개정) -- 검의 각성

**기존:** 방 중앙 바닥에 검. 접근 → 자동 줍기.

**변경:**

텅 빈 방. 방 중앙 바닥에 낡은 검이 놓여 있다. 위에서 빛 한 줄기가 내려와 검만 비춘다.

```
[환경]
텅 빈 방. 바닥에 검 하나.
빛이 위에서 내려와 검만 비춘다.

[검 획득]
플레이어가 접근하면 자동 줍기.
첫 아이템 획득 연출 (아이템 카드 팝업).

[Ego 각성 연출]
아이템 카드가 닫히면 검날에 빛이 스친다. 고주파 공명음.
```

**Ego 각성 대사** (검날이 빛나며 텍스트 표시):

```
"밝아."

"오랜만이야, 빛."
```

> 2문장. 오래 어둠 속에 있다가 빛을 본 것에 대한 감각적 반응. 감정이 아니라 사실을 말한다. 망설이지 않는다 -- 감각을 있는 그대로 보고한다.
> 텍스트 박스, 자동 진행(1.5초 간격). 검날 빛이 대사에 동기화.
> 에르다는 답하지 않는다. Rustborn은 그것을 자연스럽게 받아들인다.

### 3.2 SCREEN 8.5 (신규) -- 검과의 첫 대화

Ego 각성 직후, 검이 계속 말한다. 에르다가 검을 들고 서 있는 상태.

```
"안이 이상해. 원래 이렇지 않았던 것 같은데."

"너는 왜 여기 있는 거야?"

"들어올 수 있어. 안에."
```

> 3문장. 도움을 "요청"하지 않는다 -- 사실을 진술하고 가능성만 제시한다. 바이올렛처럼 감정적 호소를 할 줄 모른다.
> 납치감 3요소 해결: 문맥(안이 이상함) + 예고(들어올 수 있음) + 동의(진입은 플레이어 선택)

**행동 유도:**
검이 말한 직후, 화면에 모루 방향으로 **희미한 빛의 흔적**이 보인다. 검이 모루의 위치를 알고 있다는 암시. 플레이어가 모루로 이동하도록 환경 유도.

### 3.3 SCREEN 9 (개정) -- 모루에서 진입

**기존과 동일한 물리적 흐름**, 그러나 Ego가 보충한다.

```
[모루 접근 시]
"여기. 이걸 알고 있어."

[앤빌 UI에서 Rustborn 선택 + 확인]
(대사 없음 -- 검날의 빛이 강해진다)

[MemoryDive 연출 + 아이템계 전환]
```

### 3.4 SCREEN 10-11 (개정) -- 아이템계 진입 직후

아이템계에 착지. 검이 첫 반응을 한다.

```
"여기야. 내 안."

"어두워. 원래는 이렇지 않았어."

"저건 있으면 안 되는 것들이야."
```

> "저건" = 몬스터(왜곡). 첫 적이 시야에 보이는 타이밍에 이 대사가 나온다. 감정이 아니라 사실 판단.

### 3.5 SCREEN 12 (개정) -- 아이템계 전투 + 트리거 대사

전투 중 Ego는 **짧은 1문장 코멘트만** 한다.

| 트리거 | 대사 | 조건 |
| :--- | :--- | :--- |
| 첫 적 처치 | "없어졌어. 지금 네가 한 건 나를 위한 거야?" | 첫 진입 1회만 |
| 첫 피격 | (대사 없음 -- 검날의 빛이 흔들린다) | 첫 진입 1회만 |
| 방 2 진입 | (침묵 -- 전투에 집중하게 둔다) | -- |
| 방 3 클리어 후 | "밝아졌어, 조금." | 첫 진입 1회만 |

> **원칙:** 전투 중에는 말하지 않는다. 전투 시작 전 또는 종료 후에만. Transistor의 맥락 분리 원칙.

### 3.6 기억 단편 최초 조우 (신규 트리거)

아이템계 탐험 중 **기억 단편 NPC를 발견하는 순간**.

기억 단편는 야생 상태 -- 형태가 불안정하고 경계하고 있다. 기존 설계에서 야생 기억 단편는 적으로 등장했지만, **"살아있는 세계" 프레임에서 기억 단편는 주민**이다.

```
[기억 단편 시야 진입 시 -- Ego 대사]
"저건 내 일부야."

"떨고 있어. 원래는 저렇지 않았는데."
```

> 2문장. "가까이 가봐" 같은 지시는 하지 않는다. Rustborn은 명령하는 성격이 아니다. 플레이어가 스스로 접근하도록 둔다.

**기억 단편 상호작용 재설계:**

| 기존 (야생 → 복종) | 변경 |
| :--- | :--- |
| 야생 기억 단편 = 적으로 등장, 처치하면 복종 | 야생 기억 단편 = **NPC로 등장**, 주변의 왜곡 몬스터를 처치하면 안정화 |
| 처치 = 복종 | **구원 = 안정화** (주변 왜곡 클리어 → 기억 단편가 안정 상태로 전환) |
| 복종 대사: 안전 구역에서 발화 | 안정화 직후 감사 대사 1회 + 이후 안전 구역에서 추가 대사 |

> **핵심 변경:** 기억 단편를 직접 때려서 복종시키는 것이 아니라, 기억 단편를 둘러싼 왜곡(몬스터)을 처치하여 구원한다. 이것이 "칼을 휘두르는 명분"의 정수다. 플레이어는 적을 죽이는 것이 아니라 **주민을 구한다**.

**기억 단편 안정화 연출:**

```yaml
memory shard_stabilize:
  trigger: "기억 단편 주변 반경 내 왜곡 몬스터 전멸"
  visual: "기억 단편의 형태가 안정된다. 색이 맑아지고 경계가 선명해진다."
  memory shard_line: "괜찮아졌어. 고마워."
  ego_line: "따뜻해졌어. 이 아이 때문인 것 같아."
  mechanic: "기억 단편 스탯 보너스 적용. 기존 '복종' 상태와 동일한 수치 효과."
```

### 3.7 SCREEN 13 (개정) -- 보스: 기억의 수문장

보스 방 진입 시 Ego 반응:

```
"크다."
```

> 1문장. 사실만 말한다. 두려움을 표현하는 법을 모른다.

**보스 처치 후:**

```
[기존 UI: "Rustborn -- ATK +8 (Permanent)"]

[Ego 대사]
"없어졌어."

"이게 뭐라고 하는 거지, 이런 느낌."
```

> 2문장. 바이올렛이 "사랑이 뭔지 알고 싶다"고 했듯, Rustborn은 감정에 이름을 붙이지 못한다. "이런 느낌"이 무엇인지 플레이어가 채운다 -- 안도, 감사, 혹은 기쁨. 이것이 첫 번째 감정적 전환점이다.

### 3.8 SCREEN 14 (개정) -- 탈출 + 귀환

탈출 제단 접근 시:

```
[Ego]
"나아졌어, 조금."

"또 올 거야?"
```

> 기대하지 않는 톤. 하지만 기다릴 것이라는 게 느껴진다. 바이올렛이 편지를 기다리듯, Rustborn은 에르다가 다시 오기를 기다린다. 그것이 "기다림"이라는 감정인지 본인은 아직 모른다.

## 4. 반복 진입 시 Ego 행동

### 4.1 대사 감쇠 (Dialogue Decay)

| 진입 횟수 | Ego 대사량 | 예시 |
| :--- | :--- | :--- |
| 1회차 (첫 진입) | 풀 대사 (위 시퀀스 전체) | 각성, 설명, 트리거, 보스 반응 |
| 2회차 | 진입 시 1문장 + 보스 시 1문장 | "다시 왔어." / "아직 남아있어." |
| 3회차 | 진입 시 1문장만 | "고마워." |
| 4회차 이후 | **침묵** | -- |
| **새 지층 첫 도달** | 풀 대사 복귀 (지층별 1회) | 지층 2 첫 도달 시 새 서사 |

> Fi 반면교사: 4회차 이후 침묵. 야리코미 파밍에서 검은 도구로 돌아간다. 그러나 새 지층에 도달하면 다시 서사가 열린다.
> Normal = 2지층 (4x4 고정). 스탯 배율 x1.0, 기억 단편 슬롯 2.

### 4.2 부정의 역전 구조 (바이올렛 1화→7화 패턴)

바이올렛: "불타고 있지 않습니다" (1화) → "불타고 있습니다...!" (7화)

Rustborn에 심어둘 부정-역전 쌍:

| 초반 (부정) | 후반 (역전) | 트리거 |
| :--- | :--- | :--- |
| "아무것도 느끼지 못해." (각성 직후) | "느끼고 있어." (지층 2 클리어) | 친밀도 3 도달 |
| "나는 도구야. 교체하면 돼." (새 무기 획득 시) | "기다렸어." (재방문 시) | 미장착 상태에서 아이템계 재진입 |

> 단 3줄의 발화가 전체 서사 아크를 압축한다. (바이올렛 분석 에이전트 결론)

### 4.3 처분 요청 -- 자기를 물건으로 인식 (바이올렛 1화 패턴)

플레이어가 더 강한 무기를 찾아 장착을 교체할 때, Rustborn이 발화하는 특수 대사:

```
"더 좋은 거야, 그거."

"교체해도 돼. 나는 도구니까."
```

> 바이올렛의 "필요없어졌다면 처분되어야 합니다. 버려주십시오."와 동일한 구조.
> 플레이어가 이를 무시하고 나중에 Rustborn의 아이템계를 다시 방문하면,
> Rustborn은 "왔어."라고만 한다. 이 한 마디가 "기다렸다"는 감정의 전부다.

### 4.4 친밀감 시스템 (Affinity)

명시적 수치 UI는 **없다**. 그러나 내부적으로 친밀도가 추적되며, 특정 임계점에서 **Ego의 태도가 변한다**.

| 친밀도 단계 | 조건 | Ego 태도 변화 |
| :--- | :--- | :--- |
| 0 -- 관찰 | 첫 획득 | "밝아." / 감각만 보고. 감정어 0 |
| 1 -- 감각 | 첫 아이템계 클리어 | "이게 뭐라고 하는 거지, 이런 느낌." / 감정이 발생했으나 이름을 모른다 |
| 2 -- 인식 불가 | 기억 단편 2개 이상 안정화 | "너는 계속 오는 거야. 왜? 어떤 심정인지는 알고 있어도, 그것이 내 안에서 솟아나고 있는지를 알 수가 없어." / 바이올렛 6화 패턴 |
| 3 -- 각성 | 지층 2 (최종) 클리어 | "느끼고 있어, 조금. 너 때문인 것 같아." / 초반 "아무것도 느끼지 못해"의 역전. 자기 이름을 되찾는다 |

> 친밀도 단계 전환 시에만 특수 대사가 발생한다. 그 외에는 침묵 원칙 유지.
> Normal = 2지층이므로 친밀도 최종 단계는 지층 2 클리어 시. "Rustborn"�� 기억을 잃은 상태의 이름이다.

## 5. 기억 단편 재프레임: "왜곡된 기억 속 주민"

### 5.1 기존과의 관계

| 기존 설정 | 유지/변경 | 새 프레임 |
| :--- | :--- | :--- |
| "기억의 파편이 의지를 가진 존재" | **유지** | 기억 단편는 아이템 세계의 주민이다. 기억에서 태어났지만 지금은 살아있다 |
| "야생 = 기억의 혼돈" | **재해석** | 야생 = 왜곡에 의해 불안정해진 상태. 겁먹고 있거나 혼란 중 |
| "복종 = 기억의 정련" | **재해석** | 안정화 = 왜곡이 걷히고 본래의 의지를 되찾는 것 |
| "처치 → 복종" | **변경** | 주변 왜곡 몬스터 처치 → 안정화 |

### 5.2 왜곡(Distortion)이란 무엇인가

왜곡은 아이템의 기억이 부패하거나 뒤틀린 상태가 물리적으로 구현된 것이다.

| 요소 | 설명 |
| :--- | :--- |
| **정체** | 아이템의 기억 중 고통, 분노, 후회가 변질되어 생긴 존재 |
| **형태** | 기존 몬스터 풀과 동일 (비주얼 변경 불필요) |
| **행동** | 아이템 세계를 침식한다 -- 기억 단편를 공격하고, 환경을 어둡게 만든다 |
| **처치 효과** | 처치하면 해당 영역의 왜곡이 걷힌다 -- 밝아지고 기억 단편가 안정화된다 |

> **루도나러티브 정합성:** 왜곡은 기억이 원천이므로 반복 진입 시에도 재발생이 자연스럽다. "기억은 한 번 정화해도 다시 뒤틀릴 수 있다." 야리코미 반복 구조와 충돌하지 않는다.

### 5.3 기억 단편 대사 구조 (CNT-ITM-002 개정 방향)

기존 기억 단편 대사 구조는 보존하되, 프레임만 변경:

| 기존 | 변경 |
| :--- | :--- |
| 야생 대사 (전투 중) | **불안정 대사** (플레이어 접근 시 겁먹은 상태) |
| 복종 대사 (안전 구역) | **안정화 대사** (왜곡 제거 후 감사 + 기억 이야기) |

기존 "바람의 기억" 대사 재해석 예시:

| 기존 야생 대사 | 변경 (불안정 대사) |
| :--- | :--- |
| "또 싸우는 거야." | "또 와." (놀라지 않는다. 관찰한다) |
| "도망쳐. 괜히 묶이지 마." | "여기 있으면 뒤틀려. 너도." (사실을 말한다) |
| "이길 수 있어. 그게 문제지." | "할 수 있어?" (기대 없이 묻는다) |

| 기존 복종 대사 | 변경 (안정화 대사) |
| :--- | :--- |
| "그 사람은 한 곳에 오래 있지 않았어." | 동일 유지 -- 안정화 후 기억이 선명해지며 자기 과거를 이야기 |
| "이름을 안 남기면 불려오지 않는다고..." | 동일 유지 |

## 5.5 Rustborn 대사 작법 가이드 (바이올렛 분석 기반)

### 대사 작법 5원칙

| # | 원칙 | 바이올렛 근거 | Rustborn 예시 |
| :--- | :--- | :--- | :--- |
| 1 | **감정어 대신 물리 용어** | "가슴이 이상합니다" | "날이 무거워." / "떨려." |
| 2 | **사실 진술로 감정 탐색** | "어떤 심정인지는 알고 있어도 그것이 저에게 솟아나고 있는지를 알 수가 없습니다" (6화) | "이 느낌이 뭔지는 알고 있어. 내 안에서 일어나고 있는지를 모르겠어." |
| 3 | **완전한 이해를 선언하지 않는다** | "조금은 이해하게 되었습니다" (13화) | "알 것 같아, 조금." |
| 4 | **침묵이 대사다** | 가장 강렬한 순간에 이름만 부름 ("소령님...") | 보스 처치 후 "없어졌어." 한 마디, 그 뒤 침묵 |
| 5 | **위기 시 도구 언어로 퇴행** | "I am the Major's weapon!" (극장판) | "교체해도 돼. 나는 도구니까." (새 무기 획득 시) |

### 금지 사항

- **줄임표("...") 남용 금지** -- Rustborn은 망설이지 않는다. 말이 짧을 뿐이다. 간격이 필요하면 텍스트 박스 전환으로 처리. 줄임표는 대사당 최대 1회, 문장 중간 끊김에만 허용
- "고마워", "미안해" 같은 직접적 감정 표현은 **친밀도 3(각성) 이후에만** 허용
- 느낌표(!) 사용 금지 -- Rustborn은 감정을 폭발시키지 않는다
- 에르다에게 명령/지시하는 대사 금지 -- "가봐", "싸워" 같은 것은 Rustborn의 성격이 아니다
- 자기 감정을 정확히 명명하는 대사 금지 -- "슬퍼", "기뻐" 대신 항상 우회

## 6. 감정 곡선 (개정)

기존 CNT-EXP-001의 감정 곡선을 검 Ego 도입으로 재구성:

```
[고요/경외]    발코니에서 심연을 내려다봄
 [호기심]      걷기 시작. 이 사람은 누구인가
 [학습]        점프, 대시, 전투를 환경에서 배움
 [좌절]        ATK 게이트 -- 못 뚫음. 돌아감
 [발견]        격벽 틈의 빛. 검을 꺼낸다
 [놀라움]      검을 줍자 검이 **말한다** -- "밝아."              ← NEW
 [호기심]      "안이 이상해." / "너는 왜 여기 있는 거야?"       ← NEW
 [동의]        앤빌 UI에서 Rustborn 선택 + 확인              ← 문맥+동의 확보
 [경외]        MemoryDive -- 아이템계 진입
 [이해]        "내 안. 저건 있으면 안 되는 것들이야."           ← NEW
 [명분]        왜곡 처치 → "지금 네가 한 건, 나를 위한 거야?" ← NEW
 [공감]        기억 단편 구원 -- "따뜻해졌어."                  ← NEW
 [도전]        보스전. "...크다."
 [당혹]        보스 처치. "이게 뭐라고 하는 거지, 이런 느낌."    ← NEW
 [여운]        "또 올 거야?" -- 기다림을 모르면서 기다린다      ← NEW
 [쾌감]        ATK 게이트 돌파. 새 공간 개방
 [루프 인식]   ATK 게이트 #2 -- "또 이거네. 새 검 찾아서 강화하면 되겠다"
```

핵심 변화: 기존의 [발견] → [충격] 사이에 **[놀라움] → [공감] → [동의]** 3단계가 삽입된다. 이것이 납치감 3요소(문맥/예고/동의)를 정확히 해결한다.
