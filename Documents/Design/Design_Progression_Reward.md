## 1. 개요 (Overview)

### 1.1. 문서 목적

본 문서는 Project Abyss의 성장 곡선이 "어떤 모양이어야 하는가", 보상이 "왜 지금 이 방식으로 주어지는가", 그리고 플레이어가 "왜 멈추지 않는가"를 정의하는 상위 철학 문서이다.

난이도 두 축(숙련도/수치)의 상호작용은 `Design_Difficulty_Progression.md`가, 야리코미의 세션 리듬과 "한 지층만 더" 심리는 `Design_Yarikomi_Philosophy.md`가 다룬다. 이 문서는 그 두 문서가 다루지 않는 영역에 집중한다.

- **D-11의 고유 영역:** 성장 곡선의 형태와 체감, 보상 스케줄의 심리학, 드랍 설계의 원칙, 파워 판타지와 도전의 균형, 전생 루프의 철학.
- **D-11이 참조하는 것:** 순환 구조(D-02), 야리코미 동기(D-05), 난이도 공정성(D-03). 해당 내용은 이 문서에서 반복하지 않는다.

### 1.2. 한 줄 요약

> "성장은 느껴져야 하고, 보상은 예측 불가능하게 찾아와야 하며, 전생은 리셋이 아니라 재탄생이어야 한다."

## 2. 설계 의도 (Design Intent)

### 2.1. 분해 (Deconstruct)

성장과 보상 설계에서 검증된 핵심 재미 요소를 분석한다.

| 참고 게임 | 성장 곡선 특성 | 보상 심리 특성 | 핵심 교훈 |
| :--- | :--- | :--- | :--- |
| 디아블로 2 | 초반 급성장 → 후반 희귀 아이템 사냥 | 랜덤 드랍의 가변비율 강화 | "다음 몬스터를 죽이면 나올지도"가 세션을 연장한다 |
| Path of Exile | 리그 시작마다 초반 재경험 + 무한 빌드 깊이 | 아이템 조합의 내재적 탐구 동기 | 목표가 다양할수록 보상의 의미도 다양해진다 |
| 디스가이아 5 | 전생 루프 기반 무한 성장 천장 | 이노센트 복종의 점진적 보상 누적 | 리셋이 페널티가 아닌 도구가 될 때 야리코미가 지속된다 |
| 원신 | 스토리 → 엔드게임 명확한 전환점 | 보장 천장(Pity) + 가변 드랍 혼합 | 기대감과 보장이 공존할 때 이탈률이 낮아진다 |
| 메이플스토리 | 전직 기반 급성장 구간 반복 | 스타포스 강화의 니어미스 효과 | 실패도 "거의 됐다"로 느껴지면 재시도 동기가 된다 |

### 2.2. 분석 (Analyze)

| 재미의 본질 | 리스크와 리턴 분석 |
| :--- | :--- |
| 체감 성장 | 수치가 올라도 전투에서 차이가 느껴지지 않으면 성장이 존재하지 않는 것과 같다. 숫자는 체감의 수단이지 목적이 아니다 |
| 드랍 기대감 | 언제 나올지 모르는 보상이 확정 보상보다 더 강력한 지속 동기를 만든다. 슬롯머신은 결과가 예측 불가능하기 때문에 손을 놓지 못한다 |
| 전생의 역설 | 리셋 직후에는 약해 보이지만, 전생 이전보다 더 빠르게 강해진다. 이 역설이 성립할 때 플레이어는 자발적으로 리셋을 선택한다 |
| 파워 판타지 | 압도적으로 강해지는 순간(카타르시스)이 존재해야 강화 활동의 의미가 완성된다. 항상 팽팽한 난이도는 성장의 쾌감을 앗아간다 |

### 2.3. 재구축 (Rebuild)

참조 게임들의 성공 요소를 Project Abyss의 3-Space 구조에 맞게 재구축한다.

| 원형 | Project Abyss의 독창적 변형 |
| :--- | :--- |
| 디아블로의 랜덤 드랍 | 3-Space 분리로 드랍 원천을 분화. 월드는 장비 드랍, 아이템계는 강화 소재와 이노센트 드랍. 공간마다 기대하는 보상이 다르다 |
| 디스가이아의 전생 루프 | 아이템 전생을 서사적 진화와 연결. 아이템이 강해지는 것이 아니라, 아이템의 "기억"이 깊어지는 것으로 설계 |
| 원신의 보장 천장 | 연속 미획득 시 드랍률이 점진적으로 상승하는 Pity 시스템 적용. 단, Legendary 이상은 보장하지 않고 확률 상한만 제공하여 희소성 유지 |
| 메이플스토리의 강화 피드백 | 니어미스 연출(강화 실패 시 "99%에서 멈춘" 시각 피드백) 없이, 이노센트 복종 성공/실패의 즉각적 시청각 피드백으로 대체. 수치 조작 없이 체감으로 설계 |
| PoE의 빌드 다양성 | 6대 스탯(STR/INT/DEX/VIT/SPD/LCK)의 특화 빌드가 서로 다른 스탯 게이트를 열도록 설계. 동일 장비도 어떤 스탯을 주력으로 키웠느냐에 따라 열리는 세계가 달라진다 |

### 2.4. 3대 기둥 정렬 (Pillar Alignment)

| 기둥 | 성장과 보상 철학에서의 구현 |
| :--- | :--- |
| 메트로베니아 탐험 | 수평적 성장(능력 획득)이 탐험 범위를 확장한다. 새 능력을 얻을 때마다 이전에는 막혔던 공간이 열리는 것이 가장 강력한 보상이다 |
| 아이템계 야리코미 | 수직적 성장(스탯 상승)이 야리코미의 연료다. 레어리티가 높을수록 지층이 깊어지고, 이노센트 슬롯이 늘어나며, 가능성의 폭이 넓어진다 |
| 온라인 멀티플레이 | 파티 플레이가 성장 속도를 가속한다. 4인이 함께하면 솔로 불가능한 지층에 도달하여 더 강한 이노센트와 더 높은 레어리티 장비를 노릴 수 있다 |

## 3. 성장 곡선 철학 (Growth Curve Philosophy)

### 3.1. 3단계 성장 곡선

Project Abyss의 성장 경험은 세 단계로 나뉜다. 각 단계는 고유한 감정 색을 가지며, 단계가 전환될 때 플레이어가 명확히 "변화"를 인지해야 한다.

| 단계 | 구간 | 성장 속도 | 핵심 감정 | 설계 목표 |
| :--- | :--- | :--- | :--- | :--- |
| 1단계: 급성장기 | 게임 시작 ~ 스토리 중반 | 빠름 (레벨업 빈번, 능력 연속 획득) | "나는 계속 강해지고 있다" | 매 세션마다 눈에 띄는 변화로 초반 이탈 방지 |
| 2단계: 안정기 | 스토리 중반 ~ 클리어 | 보통 (성장하되 점진적) | "나는 세계를 이해하고 있다" | 전략적 선택의 폭 확장, 빌드 방향 확정 |
| 3단계: 야리코미기 | 스토리 클리어 ~ 영구 엔드게임 | 수치상 느리지만 질적 도약 | "나는 이 세계의 한계에 도전한다" | 전생 루프와 아이템계 최심층 도달이 동기 |

> **설계 원칙:** 급성장기가 너무 길면 성장의 의미가 희석된다. 야리코미기 진입이 너무 늦으면 도달하기 전에 이탈이 발생한다. 스토리 클리어가 야리코미기의 입구여야 한다.

이 세 단계는 `Design_Difficulty_Progression.md`의 두 난이도 축(숙련도/수치)과 정렬된다. 급성장기에는 숙련도 축이, 야리코미기에는 수치 축이 주 도전원이 된다.

### 3.2. 체감 성장률 원칙

> "숫자가 올라도 느껴지지 않으면 성장이 아니다."

GDC 강연(Skylanders 밸런스)의 핵심 통찰인 "Action Count" 원칙을 채택한다. STR이 10% 오른 것보다, 같은 적을 한 번 더 적게 공격해서 죽이는 것이 체감이다.

| 체감 성장의 계층 | 설계 방법 | 피드백 유형 |
| :--- | :--- | :--- |
| 즉각 체감 | 데미지 숫자 상승, 적 처치 속도 변화 | 숫자(시각), 사운드(청각) |
| 단기 체감 | 이전에 막혔던 스탯 게이트 해금 | 탐험 범위 확장 (공간적) |
| 중기 체감 | 아이템계 더 깊은 지층 생존 가능 | 도달 가능 공간 확장 |
| 장기 체감 | 전생 후 이전 플레이스루와의 속도 비교 | 자기 비교의 성취감 |

수치 인플레이션이 심화될수록 하위 계층(즉각 체감)의 의미가 희석된다. 이를 방지하기 위해 항상 "상위 계층 체감"이 존재해야 한다. 6.3절의 수치 인플레이션 관리와 연동된다.

### 3.3. 성장 축의 다양화

단일 수치 성장만으로는 야리코미기를 지탱할 수 없다. Project Abyss는 세 성장 축을 동시에 운영한다.

| 성장 축 | 방향 | 대표 메커닉 | 천장 |
| :--- | :--- | :--- | :--- |
| 수직 성장 | 스탯 수치 상승 | 레벨업, 이노센트 복종, 장비 강화 | 이론상 없음 (야리코미 무한) |
| 수평 성장 | 능력 다양화 | 이단 점프, 벽 타기, 안개 변신, 수중 호흡, 역중력 | 능력 풀 완료 시 |
| 심화 성장 | 플레이어 지식 축적 | 아이템계 미스터리 룸 활용법, 이노센트 배치 최적화, 보스 패턴 숙지 | 없음 (메타 지식) |

엑스트라 크레딧의 "깊이(Depth) vs 복잡성(Complexity)" 원칙에 따라, 세 축은 복잡성을 최소화하면서도 의미 있는 선택지를 최대화하도록 설계한다. 수직 성장은 직관적으로, 수평 성장은 명확한 피드백으로, 심화 성장은 커뮤니티 발견의 즐거움으로 제공한다.

### 3.4. 3-Space별 성장 경험

각 공간은 플레이어에게 다른 종류의 성장 경험을 제공한다.

| 공간 | 주 성장 유형 | 성장 체감 방식 | 보상 원천 |
| :--- | :--- | :--- | :--- |
| 월드 (World) | 능력 성장 (수평) | 게이트가 열림, 새 구역 진입 | 능력 렐릭, 장비 아이템 |
| 아이템계 (Item World) | 수치 성장 (수직) | 데미지 숫자 상승, 깊은 지층 도달 | 이노센트, 강화 소재, 레어리티 승급 |
| 허브 (Hub) | 준비와 최적화 | 빌드 완성감, 장비 비교와 선택 | 거래, 합성, 이노센트 팜 |

## 4. 보상 심리학 (Reward Psychology)

### 4.1. 보상 스케줄 유형

엑스트라 크레딧과 GDC 인사이트에서 도출한 네 가지 보상 스케줄을 Project Abyss의 설계에 적용한다.

| 보상 스케줄 | 원리 | Project Abyss 적용 사례 | 지속성 |
| :--- | :--- | :--- | :--- |
| 고정비율 (Fixed Ratio) | N번 행동 후 보상 | 아이템계 10층마다 보스 보상, 레벨업 경험치 | 보통 — 패턴이 예측되면 기계적이 됨 |
| 고정간격 (Fixed Interval) | N분마다 보상 | 데일리 퀘스트, 오프라인 자동사냥 보상 | 낮음 — 주어진 시간만 접속 후 이탈 |
| 가변비율 (Variable Ratio) | 평균 N번이지만 불규칙 | 레어리티별 드랍, 이노센트 미발견 룸 등장 | 매우 높음 — 다음 행동에서 나올 수 있다는 기대 |
| 가변간격 (Variable Interval) | 평균 N분이지만 불규칙 | 미스터리 룸 출현 타이밍, 레어 몬스터 등장 | 높음 — 탐험 중 언제든 발생할 수 있다는 긴장 |

> **설계 원칙:** 가변비율 강화가 가장 강력한 지속 동기를 만든다. 아이템계의 핵심 보상(높은 레어리티 드랍, 귀한 이노센트)은 가변비율로 설계한다. 고정비율 보상은 성장의 흐름을 보장하는 "바닥"으로 사용한다.

### 4.2. 니어미스 효과와 "거의 됐다" 감각

니어미스(Near Miss)는 실패이지만 성공에 근접했다는 감각을 주어 재시도 동기를 강화한다. 이 효과는 조작 없이도 설계할 수 있다.

Project Abyss에서 니어미스는 두 곳에서 자연적으로 발생한다.

**스탯 게이트에서의 니어미스:**
현재 STR이 게이트 요구치의 85% 이상이면 UI에 "거의 충족됨" 상태로 표시한다. 플레이어는 "조금만 더 강화하면"이라는 구체적 목표를 인식한다.

**아이템계 탈출 판단에서의 니어미스:**
깊은 지층에서 전멸 직전 탈출한 경험은 다음 진입 시 "이번엔 조금만 더 준비하면 된다"는 재도전 의지를 만든다. 이것은 `Design_Yarikomi_Philosophy.md`의 탈출 판단 메커닉과 연동된다.

> **주의:** 수치를 조작하여 인위적인 니어미스를 만드는 것은 장기적으로 신뢰를 손상시킨다. Project Abyss의 모든 니어미스는 실제 확률과 실제 수치에서 발생해야 한다.

### 4.3. 보상의 즉시성 원칙

행동과 보상 사이의 지연 시간이 길어질수록 행동-보상 연결이 약해진다. GDC 인사이트(Peak-End Rule)에 따르면 보상의 "종료(End)" 연출이 전체 경험의 인상을 결정한다.

| 보상 유형 | 허용 지연 | 연출 요소 |
| :--- | :--- | :--- |
| 이노센트 복종 성공 | 즉각 (0초) | 복종 이펙트 + 효과 수치 표시 |
| 레어리티별 드랍 | 0.5초 이내 | 레어리티 색상 파티클 + 사운드 (4.4절 참조) |
| 보스 처치 보상 | 1~2초 (연출 후) | 보스 소멸 이펙트 → 보상 화면 전환 |
| 레벨업 | 즉각 | 레벨업 이펙트 + 스탯 변화량 표시 |
| 스탯 게이트 해금 | 즉각 | 게이트 해제 연출 + 새 구역 미리보기 |

### 4.4. 기대감 곡선 — 레어리티별 기대의 높낮이 설계

`System_Equipment_Rarity.md`의 레어리티 체계를 보상 심리학적 관점에서 재정의한다. 각 레어리티는 서로 다른 감정 반응을 유발해야 한다.

| 레어리티 | 드랍 빈도 | 기대감 설계 | 연출 수위 | 심리적 역할 |
| :--- | :--- | :--- | :--- | :--- |
| Normal (#FFFFFF) | 매우 자주 (60%) | 무난한 안도 | 조용한 파티클 | 지속 성장의 "비". 아이템계 진입 기회가 항상 존재함을 상기 |
| Magic (#6969FF) | 자주 (25%) | 가벼운 기대 | 청색 빛 효과 | 빌드 다양성의 시작. "이 파란 검은 어떤 이노센트를 가졌을까" |
| Rare (#FFFF00) | 가끔 (10%) | 분명한 흥분 | 황색 방사 이펙트 | 야리코미 진지한 시작. 3지층 아이템계를 열기 위한 열쇠 |
| Legendary (#FF8000) | 드묾 (4%) | 강렬한 흥분 | 주황색 폭발 + 화면 진동 | 파티가 모두 주목하는 이벤트. 6 이노센트 슬롯의 가능성 |
| Ancient (#00FF00) | 매우 드묾 (1%) | 전율 | 초록 빔 + 화면 가장자리 글로우 + 효과음 | 월드 채팅 알림. 드랍 자체가 세계적 사건 |

> "Normal은 비처럼 내리고, Ancient는 번개처럼 친다."

## 5. 드랍 설계 원칙 (Drop Design Principles)

### 5.1. 직접 드랍 vs 제작 소재 — 언제 무엇을 사용하는가

모든 보상을 직접 드랍으로 제공하면 경제가 공급 과잉이 된다. 모든 보상을 제작 소재로만 주면 목표까지 거리가 너무 멀어 동기가 희석된다. 두 방식을 목적에 따라 분리한다.

| 보상 방식 | 사용 원칙 | 예시 | 경제적 효과 |
| :--- | :--- | :--- | :--- |
| 직접 드랍 | 즉각적 동기 자극이 필요할 때 | 장비 아이템, 이노센트, 능력 렐릭 | 즉각 사용 가능. 공급 과잉 위험 있어 드랍률 신중 설계 |
| 제작 소재 | 중기/장기 목표를 위한 자원 축적 | 강화석, 전생 소재, 이노센트 합성 재료 | Faucet-Sink 모델의 Sink로 작동. 경제 안정화 |
| 혼합 | 즉각 보상 + 장기 자원 동시 필요 | 보스 처치 (장비 직접 드랍 + 희귀 소재 1개 보장) | 즉각 만족과 장기 동기를 모두 충족 |

GDC의 Faucet-Sink 모델을 적용한다. 아이템계의 각 지층 클리어는 소재(Faucet, 공급)를 만들고, 이노센트 합성과 장비 전생은 소재(Sink, 소비)를 만든다. 이 순환이 경제 인플레이션을 자연적으로 억제한다.

### 5.2. 레어리티별 드랍 철학

| 레어리티 | 드랍 철학 | 드랍 구간 | 보상 깊이 |
| :--- | :--- | :--- | :--- |
| Normal | 항상 무언가 얻는다는 안도감 | 월드 전체, 아이템계 전 지층 | 2 이노센트 슬롯, 2지층 아이템계 |
| Magic | 주력 파밍 아이템. 빌드 실험의 재료 | 월드 중반 이후, 아이템계 2지층 이상 | 3 이노센트 슬롯, 3지층 아이템계 |
| Rare | 야리코미 진지하게 시작하는 플레이어의 타깃 | 월드 보스, 아이템계 깊은 지층 | 4 이노센트 슬롯, 3지층 (4×4 고정) |
| Legendary | 파티 플레이와 장기 파밍의 주 목표 | 아이템계 최심층 보스, 월드 히든 구역 | 6 이노센트 슬롯, 4지층 아이템계 |
| Ancient | 존재 자체가 야리코미의 증표 | 전생 최심층, Ancient 전용 숨겨진 구역 | 8 이노센트 슬롯, 4지층 + 심연 |

### 5.3. 드랍 연출과 피드백 — 시청각 차등 원칙

드랍 연출의 강도는 레어리티에 비례한다. 단, 모든 드랍이 화려하면 Legendary와 Ancient의 특별함이 희석된다.

| 레어리티 | 시각 연출 | 청각 연출 | 지속 시간 |
| :--- | :--- | :--- | :--- |
| Normal | 아이템 색상 파티클 (작음) | 부드러운 획득음 | 0.5초 |
| Magic | 청색 빛 확산 | 맑은 짧은 음 | 0.8초 |
| Rare | 황색 방사 + 빛 기둥 | 금속성 징 소리 | 1.2초 |
| Legendary | 주황 폭발 + 화면 진동 (약) | 팡파르 짧은 버전 | 2초 |
| Ancient | 초록 빔 + 화면 전체 글로우 + 스톱 | 전용 팡파르 풀 버전 | 3~4초 + 월드 채팅 알림 |

> **설계 원칙:** Ancient 드랍 시 화면 일시정지 + 전용 연출로 게임 흐름을 중단시킨다. 파티 플레이 시 전원의 화면에 연출이 재생된다. 드랍이 "이벤트"가 되어야 한다.

### 5.4. 핵심 드랍 제한 — 트레이드오프와 선택의 강제

모든 것을 드랍하면 선택이 의미를 잃는다. 핵심 보상에는 트레이드오프를 부여한다.

| 제한 유형 | 메커닉 | 설계 목적 |
| :--- | :--- | :--- |
| 이노센트 슬롯 상한 | 레어리티별 고정 슬롯 수 | 고레어리티 아이템의 가치 보존. Normal을 Ancient로 끝없이 채우는 것을 방지 |
| 아이템계 지층 소비성 | 아이템계 진입 시 일정 내구도 소모 | 반복 진입에 비용을 부여하여 아이템 하나의 가치를 유지 |
| 이노센트 합성 재료 소모 | 합성 시 두 이노센트 중 하나 소멸 | 합성이 선택이 되도록. "어느 이노센트를 희생할 것인가" |
| 레어리티 승급 조건 | 아이템계 모든 지층 클리어 필요 | 상위 레어리티 아이템이 노력의 결과물로 느껴지도록 |

### 5.5. 3-Space별 드랍 분배

| 공간 | 드랍 원천 | 주요 드랍 유형 | 드랍 심리 |
| :--- | :--- | :--- | :--- |
| 월드 (World) | 적 처치, 보스 처치, 탐험 보물 | 장비 아이템 (Normal ~ Rare 중심) | 탐험의 부산물. 기대보다 발견의 감각 |
| 아이템계 (Item World) | 지층 클리어, 이노센트 복종, 미스터리 룸, 보스 처치 | 이노센트, 강화 소재, 레어리티 승급 재료, Legendary ~ Ancient | 파밍의 주무대. 확률의 긴장감이 핵심 |
| 허브 (Hub) | 거래소, 이노센트 팜, NPC 교환 | 합성 결과물, 거래 아이템 | 소셜 경제 참여. 운 없이도 목표 아이템 획득 가능한 우회로 |

## 6. 파워 판타지 밸런스 (Power Fantasy Balance)

### 6.1. "강해지는 느낌"의 구조 — 수치적 증명

파워 판타지의 본질은 이전의 나와 지금의 나를 비교하는 것이다. 이 비교가 명확할 때 강해지는 느낌이 성립한다.

| 비교 층위 | 구현 방법 | 피드백 |
| :--- | :--- | :--- |
| 즉각 비교 | 데미지 숫자가 커짐. 적이 빨리 죽음 | 전투 중 실시간 |
| 공간적 비교 | 이전에 막혔던 문이 열림 | 탐험 중 발견의 순간 |
| 시간적 비교 | 전생 후 이전 플레이스루 속도보다 빠름 | 장기 세션 비교 |
| 사회적 비교 | 파티 기여도, 허브 장비 스펙 비교 | 온라인 맥락 |

GDC(Skylanders 밸런스)의 핵심 인사이트를 적용한다. "Action Count"를 직관적으로 전달하는 방법은 동일 적 처치에 필요한 타격 수가 눈에 띄게 줄어드는 것이다. STR 10% 증가보다 "5타에서 4타로"가 더 강력한 체감이다.

### 6.2. 도전의 유지 — 적 스케일링, 지층 보정, 레벨 차이 보정

사쿠라이의 원칙에 따르면 리스크 = 0이면 게임성 = 0이다. 파워 판타지를 제공하면서도 도전이 사라지지 않도록 두 가지 장치를 운영한다.

**선별적 스케일링 (엑스트라 크레딧 레벨 스케일링 교훈 적용):**
- 이미 정복한 월드 구역의 일반 적은 고정 스탯 유지. 플레이어가 "압도"를 체감하는 공간이 된다.
- 아이템계는 지층이 깊어질수록 적 스탯이 비선형 상승. 강화해도 항상 새로운 도전이 존재한다.
- 스토리 보스는 첫 도전 시 플레이어 수치를 참조하여 적정 난이도를 구성한다.

**아이템계 지층 보정:**

| 아이템계 지층 | 기본 적 스탯 배율 | 설계 의도 |
| :--- | :--- | :--- |
| 1지층 (4×4) | 현재 장비 수준의 80~100% | 진입 직후 안정적 탐험 |
| 2지층 (4×4) | 현재 장비 수준의 110~130% | 장비 강화가 필요한 압박 시작 |
| 3지층 (4×4) | 현재 장비 수준의 140~170% | 파티 플레이 또는 추가 강화 유도 |
| 4지층 (4×4) + 심연 | 현재 장비 수준의 200% 이상 | 극한 야리코미. 개인 성취의 최정점 |

### 6.3. 수치 인플레이션 관리

야리코미 게임에서 수치 인플레이션은 불가피하다. 디스가이아는 10억 단위 수치를 허용하는 방향을 선택했고, Project Abyss는 "수치 폭발을 허용하되 체감 격차를 제한"하는 방향을 선택한다.

GDC의 "Low Power Gap + High Rarity Scaling" 공식을 적용한다.

> 레어리티 배율 차이는 크지 않지만(Normal x1.0 → Ancient x3.0), 레어리티 획득 난이도 차이는 극도로 크다. 희귀함이 파워의 주된 가치가 되도록.

| 인플레이션 억제 메커닉 | 원리 | 적용 위치 |
| :--- | :--- | :--- |
| 이노센트 슬롯 상한 | 무한한 이노센트 누적을 레어리티로 제한 | 장비 시스템 |
| 전생 리셋 | 일정 강화 후 리셋하여 새 사이클 시작 | 아이템 전생 |
| 지층 스탯 비선형 상승 | 수치가 높아도 더 깊은 지층은 항상 도전적 | 아이템계 |
| 소재 Sink | 강화와 합성에 소재를 소비 | 경제 시스템 |

### 6.4. "압도의 순간" 설계 — 아이템계 클리어 후 월드 복귀 시 카타르시스

파워 판타지의 절정은 이전에 고전하던 상황을 압도적으로 제압하는 순간이다. 이 순간을 의도적으로 설계한다.

**카타르시스 사이클:**
1. 월드에서 막힌 스탯 게이트를 인식한다 (장벽 인지)
2. 아이템계에서 집중 파밍으로 스탯을 달성한다 (도전과 극복)
3. 월드로 복귀하여 게이트를 돌파한다 (카타르시스)
4. 새 구역에서 더 강한 장비를 발견한다 (다음 목표 제시)

이 카타르시스는 스탯 게이트가 열리는 순간 특수 연출(게이트 해제 이펙트 + 새 구역 음악 변화)로 강화한다. 사쿠라이의 "스트레스와 해소의 사이클(Squeeze and Release)"이 이 순간에 완성된다.

## 7. 전생(Reincarnation) 철학 (Reincarnation Philosophy)

### 7.1. 리셋의 고통과 보상 — "잃어야 더 강해진다"

전생은 현재 강화 상태를 리셋하는 행위이다. 고통이 없는 리셋은 의미가 없고, 고통만 있고 보상이 불분명한 리셋은 수용되지 않는다.

| 전생 시 잃는 것 | 전생 시 얻는 것 | 고통/보상 비율 |
| :--- | :--- | :--- |
| 현재 이노센트 배치 상태 | 적성치(Aptitude) 영구 누적 | 단기 손실 / 장기 이득 |
| 아이템계 현재 진행도 | 전생 전 미발견 구역 접근 가능 | 즉각 손실 / 즉각 새 경험 |
| 강화 소재 일부 | 더 강한 초기 스탯 베이스 | 자원 손실 / 효율 향상 |

디스가이아 7의 아이템 전생 루프를 참조한다. 리셋의 핵심은 "더 빠르게 더 멀리 가기 위한 도구"라는 인식이다. 전생 후 10지층까지 도달하는 속도가 전생 전보다 눈에 띄게 빨라야 한다.

### 7.2. 전생의 동기 설계 — 왜 플레이어가 자발적으로 리셋하는가

전생은 강제가 아닌 선택이어야 한다. 플레이어가 자발적으로 리셋을 원하게 만드는 두 가지 동기가 필요하다.

**내재적 동기 (수치적 합리성):**
아이템의 현재 강화 수준이 한계에 가까워졌을 때, 전생이 "더 강해지는 유일한 경로"가 된다. 이노센트 슬롯이 모두 채워지고, 지층 스탯 상한에 도달하면 전생만이 다음 단계를 연다.

**외재적 동기 (서사적 완성감):**
아이템의 전생 횟수가 쌓일수록 아이템의 설명 텍스트가 변화한다. "낡은 검"이 "전승된 검"이 되고, 최종적으로 "기억의 결정체"가 된다. 전생은 아이템의 역사가 깊어지는 서사적 행위이다.

### 7.3. 전생 루프의 장기 목표 — 적성치 누적, 잠재력 상승

GDC 아이들 게임 강연의 Prestige 시스템 원칙을 적용한다. 전생으로 획득하는 영구 자원은 다음 사이클의 진행 속도를 높인다.

| 전생 횟수 | 누적 적성치 보너스 | 해금 콘텐츠 |
| :--- | :--- | :--- |
| 1회 | +5% 전 스탯 기반 | 전생 전용 이노센트 슬롯 1 추가 |
| 3회 | +15% + 특수 이노센트 합성 가능 | 전생 아이템계 전용 지층 해금 |
| 5회 | +25% + Ancient 레어리티 승급 자격 | 아이템계 심연 지층 접근 |
| 10회 이상 | +스탯 상한 해제 (야리코미 극한) | 전생 기념 시각 이펙트 변화 |

### 7.4. 전생과 서사 연결 — 아이템 전생 = 아이템계 리셋 + 서사 진화

Project Abyss의 전생은 디스가이아의 수치 리셋에 서사적 의미를 더한다.

> "아이템이 전생한다는 것은, 그 아이템이 품고 있던 기억의 지층이 더 깊은 층으로 재구성된다는 것이다."

전생 이전과 이후의 아이템은 동일한 형태를 유지하지만, 아이템 설명에 전생 기록이 누적된다. 이것은 단순한 플레이버 텍스트가 아니라, 플레이어가 이 아이템과 함께한 야리코미의 기록이다. 전생을 통해 아이템은 소비재에서 "동반자"가 된다.

## 8. 진행 마일스톤 설계 (Progression Milestones)

### 8.1. 단기/중기/장기 목표 계층

엑스트라 크레딧의 플레이어 매슬로우 위계에 따라, 목표 계층이 동시에 존재해야 플레이어는 항상 다음 행동의 이유를 가진다.

| 목표 계층 | 시간 범위 | 예시 목표 | 달성 보상 유형 |
| :--- | :--- | :--- | :--- |
| 단기 목표 | 현재 세션 내 (30분 이내) | 아이템계 지층 3개 클리어, 이노센트 2개 복종 | 즉각 보상 (이노센트, 소재) |
| 중기 목표 | 수 세션 (1~3일) | Rare 장비 아이템계 전 지층 클리어, 스탯 게이트 하나 해금 | 성취 보상 (레어리티 승급, 새 구역) |
| 장기 목표 | 수 주 이상 | Legendary 전생 3회 달성, Ancient 첫 드랍 | 극한 보상 (서사적 기록, 특수 이펙트) |

세 계층의 목표가 항상 동시에 존재해야 한다. 단기 목표만 있으면 게임이 단순하고, 장기 목표만 있으면 지치고, 중기가 없으면 이탈 시점이 발생한다.

### 8.2. 마일스톤 피드백 — 달성 연출과 보상

GDC(Rational to Emotional 강연)의 Peak-End Rule을 적용한다. 마일스톤 달성의 "종료" 순간이 기억에 남아야 리텐션이 형성된다.

| 마일스톤 | 연출 방식 | 보상 형태 |
| :--- | :--- | :--- |
| 첫 아이템계 진입 | 아이템계 진입 애니메이션 (특별 버전) | 튜토리얼 완료 이노센트 선물 |
| 첫 Legendary 드랍 | 전용 드랍 연출 + "처음으로 Legendary를 발견했습니다" 팝업 | 즉각 아이템계 진입 제안 |
| 스탯 게이트 첫 해금 | 게이트 해제 풀 연출 + 새 구역 첫 진입 음악 | 새 구역 탐험 경험 자체 |
| 첫 전생 완료 | 전생 완료 특수 연출 + 아이템 설명 변화 | 적성치 보너스 + 전생 기념 칭호 |
| 아이템계 심연 첫 도달 | 심연 진입 전용 연출 | 시스템 공지 + 칭호 |

### 8.3. 엔드게임 진입점 — "스토리 클리어가 시작이다"

Project Abyss의 핵심 메시지는 스토리 클리어가 게임의 끝이 아니라 야리코미의 시작이라는 것이다.

스토리 클리어 시 특수 연출과 함께 다음 내용을 전달한다.

> "심연의 진짜 깊이는 이제부터입니다."

클리어 이후 해금되는 콘텐츠:
- Legendary/Ancient 드랍률 상승 구역 해금
- 아이템계 4지층 및 심연 접근 가능
- 전생 시스템 전면 해금
- 파티 플레이 전용 심층 던전 해금

## 9. 저주받은 문제 (Cursed Problems)

성장과 보상 설계에서 피할 수 없는 근본적 상충 관계 다섯 가지를 정의한다.

| 상충 | 문제의 본질 | 해결 방향 | 타협점 |
| :--- | :--- | :--- | :--- |
| 강해지는 느낌 vs 도전의 유지 | 플레이어가 강해지면 도전이 쉬워지고, 도전을 유지하면 성장이 무의미해진다 | 선별적 스케일링. 이미 정복한 구역은 쉬운 채로 두고, 새 구역과 아이템계 깊은 지층만 도전적으로 유지 | 월드 구역은 고정, 아이템계 지층은 비선형 상승 |
| 드랍 확률 관대함 vs 희소성의 가치 | 드랍이 너무 잦으면 흥분이 사라지고, 너무 드물면 의욕이 꺾인다 | 레어리티 계층으로 분리. Normal은 자주, Ancient는 극도로 드물게. Pity 시스템으로 연속 미획득의 좌절 완화 | 레어리티별 드랍률 명확히 차등 (60% / 25% / 10% / 4% / 1%) |
| 전생 리셋 고통 vs 접근성 | 리셋이 강렬할수록 신규/라이트 플레이어가 이탈하고, 리셋이 가벼우면 의미가 없다 | 전생은 선택지일 뿐 강제하지 않는다. 전생 없이도 콘텐츠를 즐길 수 있되, 전생이 야리코미의 진정한 입구임을 커뮤니티가 발견하도록 | 전생 = 자발적 선택. 강제 전생 구간 없음 |
| 수치 인플레이션 vs 야리코미 쾌감 | 수치를 제한하면 야리코미의 무한 성장 환상이 깨지고, 제한하지 않으면 경제와 밸런스가 붕괴한다 | 수치 자체는 무한하게 허용하되, 의미 있는 비교 단위(스탯 게이트, 지층 생존 기준)는 고정 | 수치 인플레이션 허용 + 의미 단위 고정 |
| 자동사냥 보상 vs 수동 플레이 동기 | 자동사냥이 수동보다 효율적이면 수동 플레이 동기가 사라지고, 수동이 월등하면 자동사냥이 무의미해진다 | 수동 정복 선행 원칙. 자동사냥은 이미 실력으로 정복한 구역에서만 가능. 자동사냥은 효율의 수단이지 정복의 수단이 아니다 | 자동 = 리소스 획득, 수동 = 진행과 탐험 |

## 10. 검증 체크리스트 (Verification Checklist)

### 성장 곡선 검증

- [ ] 게임 시작 후 첫 10분 내에 레벨업 또는 능력 획득이 발생하는가
- [ ] 첫 아이템계 진입 후 장비 스탯 변화가 숫자로 즉각 확인되는가
- [ ] 스토리 클리어까지 스탯 게이트가 최소 5번 이상 새로운 구역을 열어주는가
- [ ] 야리코미기 진입 후에도 "아직 더 강해질 수 있다"는 감각이 유지되는가

### 보상 심리학 검증

- [ ] 아이템계 한 세션에서 가변비율 보상(예상 외 이노센트, 특수 룸)이 최소 1회 이상 발생하는가
- [ ] Legendary 드랍 시 파티 전원이 알아채는 연출이 작동하는가
- [ ] Normal 드랍이 과잉 공급되어 "쓸모없다"는 인식이 생기지 않는가

### 드랍 설계 검증

- [ ] 레어리티별 드랍 빈도가 설계 수치(60/25/10/4/1%)와 10% 이내 오차로 일치하는가
- [ ] 연속 100회 파밍 시 플레이어가 Rare 이상을 최소 5~8회 획득하는가 (Monte Carlo 시뮬레이션 사전 검증)
- [ ] 제작 소재의 공급(Faucet)과 소비(Sink)가 균형을 이루는가 (10시간 플레이 경제 시뮬레이션)

### 파워 판타지 밸런스 검증

- [ ] 장비 강화 후 같은 적 처치에 필요한 타격 횟수가 눈에 띄게 감소하는가
- [ ] 이미 정복한 구역에서 "압도"하는 쾌감이 존재하는가
- [ ] 최고 레어리티 장비가 최저 레어리티 대비 3배 이상의 능력치 우위를 갖지 않는가 (파워 갭 제한)

### 전생 철학 검증

- [ ] 전생 후 동일 지층 도달 속도가 전생 전보다 체감상 빠른가
- [ ] 전생 3회 후 아이템 설명 텍스트가 변화하는가
- [ ] 전생을 강제하지 않고도 야리코미 플레이어가 자발적으로 전생을 선택하는가

### Project Abyss 정렬 검증

- [ ] 대순환(월드 → 아이템계 → 강화 → 스탯 게이트 → 월드)이 스탯 게이트를 중심으로 끊김 없이 연결되는가
- [ ] 3-Space 각 공간이 고유한 보상 감정을 제공하는가 (월드=발견, 아이템계=긴장과 축적, 허브=완성)
- [ ] 스토리 클리어가 야리코미기의 시작으로 자연스럽게 연결되는가
