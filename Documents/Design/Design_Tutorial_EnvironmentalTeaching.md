## 1. 8대 환경 교습 패턴

리서치에서 추출. ECHORIS 전 구간에 재사용 가능한 규칙.

### A. 공간 유도 (Spatial Funnel)

공간 형태가 행동을 강제한다. 진행 방향이 1-2개로 제한. 목표 오브젝트가 가장 눈에 띄는 위치.

**적용 규칙:**
- 새 메커닉 교습 시, 해당 메커닉 없이는 통과 불가능한 공간을 만든다
- 선택지를 줄일수록 교습이 명확해진다
- 목표는 방 안에서 **가장 밝은 오브젝트**여야 한다

### B. 시각적 차이화 (Visual Differentiation)

색, 빛, 크기로 "중요한 것"과 "위험한 것"을 전달.

**적용 규칙:**
- 상호작용 가능 오브젝트: 주변보다 밝거나 색이 다름
- 위험: 붉은색/날카로운 형태/날카로운 사운드
- 일관성: 같은 의미는 항상 같은 색. 혼용 금지

| 요소 | 시각 신호 |
|:---|:---|
| 상호작용 가능 | 파란빛 발광, 주변보다 밝음 |
| 위험 | 붉은색, 날카로운 형태 |
| 중요 | 금색, 파티클 |
| 안전 | 녹색, 따뜻한 조명 |
| 막힘 | 붉은 섬광 + 튕김 |

### C. 행동 시연 (Behavioral Demonstration)

게임 자체가 메커닉을 직접 보여줌. 플레이어는 보고 따라함.

**적용 규칙:**
- 시연 → 즉시 같은 행동을 요구하는 상황 배치
- "본 것 → 따라한 것 → 성공"의 3단계
- 첫 경험은 자동 트리거, 이후는 플레이어 자발적 실행

### D. 죽음을 교습 도구로 (Death as Teacher)

실패가 정보 습득 수단. 재시작 비용을 낮춘다.

**적용 규칙:**
- 재시작 3초 이내
- 사망 지점에서 가까운 곳 리스폰
- 사망 원인이 시각적으로 명확 (텔레그래프 애니메이션)
- spike: 페이드아웃 → 마지막 안전 발판 리스폰 → 페이드인 (200ms/200ms)
- 보스: 보스 방 입구 리스폰, HP 유지 (첫 진입 특례)

### E. 환경 서사 (Environmental Narrative)

건축, 유물, 잔해가 세계를 설명. 텍스트 없이 이야기를 추론.

**적용 규칙:**
- 사용된 흔적, 부서진 오브젝트, 봉인된 문 = 과거의 물리적 증거
- 결론을 "말해주지 않는 것"이 더 강한 몰입
- Memory Room의 상호작용 텍스트가 이 패턴의 구체적 구현

### F. 보상의 직접성 (Immediate Reward Signal)

올바른 행동에 0.5초 이내 피드백. 숫자 + 파티클 + 사운드 3채널 동시.

**적용 규칙:**
- 히트스탑: 타격 시 3프레임 정지
- 레벨업: 화면 중앙 금색 텍스트 + 섬광
- 게이트 돌파: 균열 애니메이션 → 통로 개방
- EXP 카운터: 처치마다 HUD 숫자 상승

### G. 순차 레이어링 (Sequential Mechanic Layering)

복잡한 시스템을 시간 순서에 따라 하나씩 추가.

**적용 규칙:**
- 한 스크린/구역에서 **새 메커닉은 1개만**
- 이전 메커닉이 안정화된 후 다음 도입
- 도입 3단계: 발견(Discovery) → 연습(Practice) → 조합(Combination)

**첫 30분 레이어링 순서:**

| 시점 | 활성화 메커닉 |
|:---|:---|
| S1 대장간 | 이동, 상호작용 |
| S2 봉인된 복도 | + 점프 |
| S3 첫 전투 | + 공격 (Z) |
| S4 에코 타격 | + 에코 (자동 트리거) |
| S6 아이템계 | + EXP 카운터 |
| S7 보스전 | + 패턴 읽기 |
| S11 2차 다이브 | 전체 메커닉 자발적 조합 |

### H. 선택적 고급 교습 (Optional Advanced Teaching)

필수 경로 밖에 숨겨진 고급 메커닉. 탐험한 플레이어만 발견.

**적용 규칙:**
- 기본 루트에서 벗어난 선택적 공간에 배치
- 발견하지 못해도 진행에 지장 없음
- 발견하면 강력한 우위 또는 깊은 내러티브
- Memory Room, 숨겨진 breakable 벽 뒤 보상이 이 패턴

## 3. Phase 1+ 교습 확장 규칙

첫 30분 이후 새 능력/시스템이 도입될 때마다 동일한 패턴을 적용한다.

### 3.1. 렐릭(능력) 해금 교습 템플릿

모든 렐릭 해금은 다음 3단계를 거친다:

```
1. 발견 (Discovery)
   → 렐릭이 필요한 장소를 먼저 보여줌 (패턴 A: 공간 유도)
   → "여기에 뭔가 있는데 못 간다" (패턴 B: 시각 차이화)

2. 획득 (Acquisition)
   → 렐릭 획득 직후, 안전한 환경에서 즉시 사용 (패턴 C: 행동 시연)
   → 획득 연출은 시각+사운드 3채널 (패턴 F: 즉각 보상)

3. 적용 (Application)
   → 획득 직후 1개 구간에서 새 능력만 테스트 (패턴 G: 순차 레이어링)
   → 이전에 막혔던 곳으로 돌아가면 통과 가능 (패턴 A: 공간 유도 재활용)
```

### 3.2. 아이템계 절차적 생성 내 교습

아이템계는 매 진입마다 구조가 달라지므로 "고정된 교습 순서"가 불가. 해결:

| 규칙 | 내용 |
|:---|:---|
| **Start 방 = 고정 레이아웃** | 첫 방은 항상 안전. 적 소수 또는 0. 공간 감각 확보 |
| **RoomType이 교습을 대행** | Rest 방 = "여기는 안전" 학습, Treasure 방 = "엘리트전" 학습, Memory 방 = "여기는 서사" 학습. 타입 자체가 반복 교습 |
| **Boss 방 = 항상 마지막 행** | 크리티컬 패스의 끝. "아래로 가면 보스"는 플레이어가 자연스럽게 학습 |
| **새 타일(magma/ice/charged) 첫 등장 시** | 해당 타일이 있는 첫 방은 "1개만 배치 + 안전 우회 가능" → 교습 후 다음 방부터 본격 배치 |
