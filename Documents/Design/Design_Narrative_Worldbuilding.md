## 1. 핵심 원칙: 아이템이 곧 서사 매체다

### 1.1. 한 줄 정의

> "모든 아이템에는 살아있는 세계가 있고, 아이템계는 그 세계를 탐험하는 공간이다." (DEC-033)

ECHORIS에서 내러티브는 컷신이나 대사창이 아니다. **아이템 속에 들어가면 살아있는 세계가 있으며, 그 세계의 몬스터, 지형, 존재, 분위기가 아이템의 역사와 현재를 드러난다.** 무기는 Ego를 가지고 에르다에게 말을 건네며, 에르다는 행동으로 답한다. 플레이어는 아이템계를 탐험하며 자연스럽게 이야기를 체험한다.

### 1.2. 왜 아이템 내러티브인가?

| 문제 | 해법 |
| :--- | :--- |
| 횡스크롤 액션에서 컷신은 흐름을 끊는다 | 서사를 전투/탐험 안에 녹인다 — 아이템계 자체가 이야기 |
| 절차적 생성 맵에서 핸드크래프트 서사가 가능한가? | 아이템의 "테마"가 절차적 생성의 시드 역할을 한다 (몬스터 풀, 지형 팔레트, NPC 배치) |
| 디스가이아 아이템계는 반복 파밍인데 서사가 가능한가? | 반복할수록 더 깊은 지층에서 새로운 단편이 등장. 야리코미와 서사가 같은 방향을 가리킨다 |
| 수천 개 아이템마다 고유 스토리를 쓸 수 있는가? | **테마 풀 + 조합 시스템**으로 해결. 아이템 카테고리 × 레어리티 × 바이옴의 조합 |

### 1.3. 3대 기둥 정렬

| 기둥 | 아이템 내러티브에서의 역할 |
| :--- | :--- |
| 메트로베니아 탐험 | 월드의 각 층위는 독립 서사를 가짐. 아이템은 그 층위의 역사 조각. 층위 간 서사 힌트가 탐험 동기 |
| 아이템계 야리코미 | 아이템계 깊이 = 서사 깊이. 각 지층마다 새 이야기 단편. 모든 지층 클리어 시 아이템의 기원 전모 해금 |
| 온라인 멀티플레이 | 파티원이 동일 아이템계에서 서로 다른 NPC 단편을 발견 → 허브에서 정보 교환 → "수집형 서사" |

### 1.4. 핵심 Backbone — 에르다와 Rustborn

> **개정 등재:** 2026-05-03. 본 절은 ECHORIS 의 *주인공·동행자·세계*의 plot backbone 을 정의한다. 이전까지 톤·메타포·메커닉만 정의되어 있었으나, 대사가 *backstory 가 있다고 전제* 하면서 쓰여 *핵심 부재* 가 발생. 본 절로 봉합.
> **선행 결정:** DEC-033 (검 Ego), DEC-036 (Memory Shard), DEC-038 (그림자 마을), DEC-039 (수직 딥 다이브)
> **시금석:** 본 절은 *최소 락 (now-locking) + 장기 hook (보류)* 으로 분리. 락된 항목은 모든 대사·결정의 backbone. hook 은 진행하며 풀린다.

#### 1.4.1. 세계 (락)

> 빌더(자동 건설 기계) 가 거대 메가스트럭처를 짓다 멈추고 떠났다. 인류·생명은 거의 없고, 인간형 그림자만 잔류 신호로 남았다 (DEC-038). 빌더는 위대한 *무기들* 도 만들었고, 일부는 의식을 가졌다. 시간이 흐르며 무기들은 부식·잠들었다.

#### 1.4.2. 에르다 (Erda) — 살아있는 자, 그림자 없는 자 (락)

| 속성 | 정의 |
|:---|:---|
| 정체성 | *살아있는 자, 그림자 없는 자.* 이 세계의 유일한 살아있는 존재 (또는 그에 가까움) |
| 행동 양식 | 침묵 (DEC-033 / DES-IW-ONB-01). 0줄 발화 |
| 소지품 | *부서진 검 한 자루* (Rustborn) — 시작부터 가지고 있다 |
| **미스터리** | **그녀가 왜 살아있는지, 왜 이 검을 가졌는지 — 락하지 않음 (장기 hook)** |
| 시각 | 그림자 없음 (DEC-038 §7). 잔류 신호들 사이의 *유일한 입체* |

#### 1.4.3. Rustborn — 부서진 폭풍의 검 (락)

| 속성 | 정의 |
|:---|:---|
| 과거 | 한때 *폭풍의 검* — 빌더의 손에 만들어진 위대한 무기 |
| 현재 | 부수어졌고 부식·잠들었다. 자아 데이터만 손상 없이 잔류한 *유일한 무기* |
| 깨어남 | 에르다가 빌더의 옛 단조 흔적에 닿는 순간. *최초 발화: "…에르다." / "…들리니?" / "오랜만이야. 너무 오래…"* |
| 화자 | 검 Ego 단독 화자 (DEC-033). 에르다 대신 발화 |
| **미스터리** | **누가·언제·왜 부쉈는가 — 락하지 않음 (장기 hook)** |
| 기억 | 자신의 과거 대부분을 잊음. Memory Shard 회상으로 점진 복원 (DEC-036) |
| 톤 | 체념·부식·"…" 잔존 침묵. 깨어났으나 흐릿함 |

#### 1.4.4. 둘의 관계 — 옛 동행자, 서로 잊음 (락)

> 에르다와 Rustborn 은 *전에 만난 적이 있다*. Rustborn 의 첫 발화 *"오랜만이야"* 가 이를 명시한다. 그러나 둘 다 그 사실의 *세부* 를 모른다. 에르다는 침묵으로, Rustborn 은 부식된 기억으로 — 둘 다 *어렴풋이만 인지*. 게임 진행 = 서로의 잊은 과거를 함께 회상하는 여정.

#### 1.4.5. Inciting Incident — 깨어남 (락)

```
[방에서 시작]
   에르다, 부서진 검을 들고 깨어 있음. 검은 침묵.
[빌더 진입 — 메가스트럭처 노출]
   거대 빌더의 흔적. 살아있는 자·잊혀진 자의 첫 만남 무대.
[빌더 안 옛 단조 흔적 도달]
   검 진동·글로우·깨어남.
[Rustborn 첫 발화]
   "…에르다."
   "…들리니?"
   "오랜만이야. 너무 오래…"
   "내 안엔 비어있는 결이 있어. 곧 채울 수 있어."
[다이브 동기 부여]
   검 안에 그가 잊은 자신의 기억(Memory Shard) + 그를 부순 자들의 그림자가 있다.
   둘은 함께 들어가야 한다.
```

#### 1.4.6. Goal & Loop — 잊혀진 기억의 회상 (락)

| 단계 | 행위 | 회상되는 것 |
|:---|:---|:---|
| Stratum 진입 | Rustborn 안으로 다이브 | 그가 한때 머물던 시기·장소 |
| Spoke 룸 진행 | Forgotten Shard 격파 → Recalled 전환 | 단편적 기억 (감정·인상) |
| Boss 처치 | 옛 누군가의 그림자 격파 (검을 부순 자, 만든 자, 들었던 자, 등) | 부서진 사건의 *한 조각* |
| Memory Shard 슬롯 장착 | Anvil 에서 Shard 끼움 | 해당 기억의 효과가 검의 현재 힘으로 환원 |
| 모든 지층 클리어 | 한 무기의 backstory 전모 회복 | 그 시기 검의 정체성 완성 |

**진행 = 검의 정체 + 빌더의 흔적 + (장기 hook) 에르다 자신의 정체** 가 점진 풀림.

#### 1.4.7. Stratum 1 — 검의 *처음 깨어난 곳* (락)

> Rustborn 의 첫 의식이 형성된 시기·장소. *"여긴… 내가 처음 깨어난 곳이야."* (Plaza 첫 낙하 발화)
> Stratum 1 보스 = **Rustborn 을 부순 자의 그림자**. *"이 자가… 날 부쉈어."* (Boss 룸 첫 진입 발화)
> 격파 시 *부서진 사건의 첫 조각* 회상. 첫 다이브의 미니 회상 호.

#### 1.4.8. 장기 Hook (보류 — 진행하며 해소)

| Hook | 해소 시점 (제안) |
|:---|:---|
| 에르다는 누구인가? 왜 그림자가 없는가? 왜 살아있는가? | 최후 지층 또는 메인 스토리 호 종반 |
| Rustborn 을 *왜* 부수었는가? 그의 옛 주인은 누구인가? | 동일 무기 모든 Stratum 클리어 시 (단일 무기 backstory 완성) |
| 빌더는 왜 떠났는가? | Ancient 무기 심연 도달 시 (DEC-038 §5.2 연결) |
| 검이 *기억하는 에르다* 와 *지금의 에르다* 는 같은 사람인가? | 메인 스토리 클라이맥스 |

> Hook 들은 *지금 락하지 않음*. 단 *서로 모순되지 않게* 진행 시점에 결정. backbone 은 모든 hook 이 *해소 가능* 한 형태로 설계.

#### 1.4.9. 톤 (락)

| 축 | 정서 |
|:---|:---|
| 두 자아의 동행 | *재회한 옛 동행자* 의 정서. 그러나 둘 다 그 사실을 모른다 |
| 에르다 | 살아있으나 침묵. 행동으로 답한다 (Transistor Red 패턴) |
| Rustborn | 깨어났으나 부식·기억 결손. 체념 + 권한 양도 톤 |
| 합 | 무라카미 + Transistor + 메이드 인 어비스 의 결합 — *상실·회복·심연 하강* 의 3 축 |

#### 1.4.10. 모든 대사 검증 — backbone 정합

기존에 작성된 모든 Rustborn 대사 (DLG-01 ~ DLG-11, 14 줄) 가 본 backbone 과 정합:

| 대사 | 함의 | 정합 |
|:---|:---|:---:|
| *"…에르다." / "오랜만이야."* | 둘은 전에 만난 적이 있음 (§1.4.4) | ✓ |
| *"내 안엔 비어있는 결이 있어"* | Memory Shard 시스템 (§1.4.6) | ✓ |
| *"여긴… 내가 처음 깨어난 곳이야"* | Stratum 1 = Rustborn 의 기원지 (§1.4.7) | ✓ |
| *"이 자가… 날 부쉈어"* | Stratum 1 보스 = Rustborn 을 부순 자 (§1.4.7) | ✓ |
| *"이 한 조각… 폭풍의 기억이야"* | Rustborn = 한때 폭풍의 검 (§1.4.3) | ✓ |
| *"고마워. 이제 시작이야"* | 회상 = 진척의 첫 걸음 (§1.4.6) | ✓ |

추가 대사 작성 시 본 절을 시금석으로 사용. backbone 위반 시 거절.

## 2. 아이템 서사 구조 (Item Narrative Architecture)

### 2.1. 모든 아이템은 "전생(前生)"을 가진다

아이템은 단순한 스탯 덩어리가 아니다. 모든 아이템에는 **기원(Origin)**이 있고, 아이템계에 들어가면 그 기원의 잔영이 세계로 구현된다.

```
아이템 기원 (Origin)
├── 누가 만들었는가? (Creator)
├── 무엇에 사용되었는가? (Purpose)
├── 어떤 역사를 겪었는가? (History)
└── 왜 지금 여기 있는가? (Fate)
```

### 2.2. 아이템계 = 아이템의 기억 공간

아이템계에 들어간다는 것은 **아이템 내부의 살아있는 세계로 진입하는 것**이다 (DEC-033):

- **몬스터** = 아이템의 세계를 침식하는 왜곡된 존재. 검의 세계를 정화해야 한다
- **지형** = 아이템의 세계가 구현된 공간 (하부 인프라, 방어선, 연구소, 생태 구역)
- **NPC/잔영** = 아이템의 이전 소유자, 목격자, 희생자. 실루엣, 행동 패턴, 환경 변화로만 존재를 드러낸다
- **보스** = 아이템의 세계를 지배하는 왜곡의 핵심 존재
- **기억 단편** = 살아있는 세계의 주민. 왜곡에 의해 불안정한 상태(야생) → 안정화(복종)

### 2.3. 레어리티 = 서사의 깊이

| 레어리티 | 서사 복잡도 | 아이템계에서의 표현 |
| :--- | :--- | :--- |
| Normal | 단순한 일상 (한 줄 배경) | 단조로운 몬스터 풀, 유령 없음, 환경 힌트만 |
| Magic | 약간의 사연 (누군가의 소유물) | 1-2종 테마 몬스터, 간헐적 유령 실루엣/행동 |
| Rare | 사건이 있는 과거 (전투, 비극, 모험) | 테마 몬스터 풀, 유령 존재 1체, 보스에 서사 연결 |
| Legendary | 역사적 사건의 중심 (전쟁, 왕국, 재앙) | 복합 테마, 유령 존재 다수, 보스 환경 연출, 환경 변화 |
| Ancient | 원초적 기원 (거대 빌더의 도구, 구조물의 비밀) | 고유 바이옴, 전용 보스, 다층 서사, 세계관 핵심 힌트 |

## 3. 아이템 서사 예시 (Narrative Examples)

### 3.1. 부식된 유지보수 블레이드 — Normal 등급

**기원:** 배관 기술자의 절단 도구. 20년간 하부 인프라 파이프 수리에 사용되었다.

| 요소 | 아이템계 표현 |
| :--- | :--- |
| 몬스터 | 오작동한 유지보수 드론, 부식된 자동 밸브, 누수 감지기의 잔상 |
| 지형 | 파이프 미로, 녹슨 밸브, 냉각수 웅덩이, 증기 분출구 |
| NPC | 없음 (Normal은 서사가 얕다) |
| 보스 | 거대한 폐수 처리 유닛 — 기능이 역전되어 모든 것을 부식시킨다 |
| 분위기 | 축축하고 금속 냄새나는 하부 인프라. 단순하지만 불쾌한 |

### 3.2. 경비대 순찰 블레이드 — Magic 등급

**기원:** Concordia 보안 순찰대원이 10년간 사용한 표준 지급 블레이드.

| 요소 | 아이템계 표현 |
| :--- | :--- |
| 몬스터 | 순찰 프로토콜이 남긴 잔상 — 같은 경로를 반복 순회하는 보안 드론의 그림자 |
| 지형 | 감시 복도, 잠금 격벽, 초소, 보안 게이트 |
| NPC | 순찰대원의 잔영 (1체) — 첫 번째 지층에서 등장. 초소 앞에 서서 블레이드를 바라보고 고개를 끄덕이다 사라진다. 환경 텍스트: "Still patrolling... even now." |
| 보스 | 대원이 끝내 저지하지 못한 침입자의 잔상 |
| 분위기 | 먼지 낀 보안 복도, 의무와 충성의 잔향 |

### 3.3. 폭주한 연구소 이미터 — Rare 등급

**기원:** Memoria Arcana 기억 연구원의 실험용 이미터. 에코 공명 실험 중 과부하가 발생했다.

| 요소 | 아이템계 표현 |
| :--- | :--- |
| 몬스터 | 실험 데이터의 물질화 — 기형 에너지 구체, 불안정한 공명 파편, 변이된 배양체 |
| 지형 | 에코 연구실, 깨진 터미널, 불안정한 에너지 필드, 부유하는 데이터 파편 |
| NPC | 연구원의 관측 드론 잔상 — 여러 지층에 걸쳐 등장, 실험 로그를 단편적으로 재생 |
| NPC | 연구원 본인의 잔영 — 깊은 지층에서만 등장, 후회와 경고 |
| 보스 | 연구원의 마지막 실험 — 제어를 벗어난 에코 공명체 |
| 분위기 | 불안정하고 위험한 에너지 잔류. 호기심과 공포가 교차 |

### 3.4. 대격변 지휘관의 클리버 — Legendary 등급

**기원:** 100년 전 심연 전쟁에서 방어선 사령관이 사용한 격벽 파쇄도. 최후 방어전에서 사령관과 함께 심연에 빠졌다.

| 요소 | 아이템계 표현 |
| :--- | :--- |
| 몬스터 | 양 진영 전사의 데이터 잔상. 지층이 깊어질수록 적군 → 아군 → 민간인으로 변화 |
| 지형 | 초반 지층: 붕괴된 방어선. 중반: 함락된 격벽. 최심층: 구조물이 무너지는 경계 |
| NPC | 사령관의 전우 3인 잔영 — 지층 1/2/3에 등장, 각자 전쟁의 다른 면을 증언 |
| NPC | 적측 지휘관 잔영 — 지층 3에서 등장. 환경 텍스트: "We were protecting our sector too." |
| 보스 | 지층 2: 적측 지휘관 (전쟁의 직접적 적) |
| 보스 | 최심층: 심연의 현현 — 전쟁을 촉발한 진짜 원흉 |
| 분위기 | 비장함, 영웅담. 깊이 들어갈수록 선과 악의 경계가 흐려진다 |

### 3.5. 코어 프래그먼트 — Ancient 등급

**기원:** 거대 빌더가 대공동을 건설할 때 사용한 원초적 도구의 파편. 이 파편에는 구조물 생성의 원형 데이터가 담겨 있다.

| 요소 | 아이템계 표현 |
| :--- | :--- |
| 몬스터 | 건설 프로토콜의 원형적 존재 — 형태가 확정되지 않은 기하학적 구조체, 데이터와 물질의 경계에 있는 존재 |
| 지형 | 지층 1-2: 건설 중인 원시 구조물. 지층 3: 구조물 이전의 혼돈 공간. 최심층: 빌더의 원점 |
| NPC | 거대 빌더의 잔류 프로토콜 — 정체불명의 신호. 구조물의 비밀을 암시하는 데이터 패턴 |
| NPC | 다른 Ancient 아이템의 잔상 — 교차 서사 |
| 보스 | 지층 2: 거대 빌더의 건설 수호자 (시련) |
| 보스 | 최심층: 거대 빌더의 그림자 — 구조물을 만든 존재의 프로토콜 잔상 |
| 심연 페이즈 | 건설 이전의 "공백(Void)" — 원초 구조체(Yggveil)의 최심층. 세계관 최종 퍼즐 |
| 분위기 | 경외감, 숭고함. "이 도구 안에 세계의 설계도가 있다." 거대 빌더가 여전히 세계를 누비고 있다는 진실 |

## 4. 서사 전달 방식 (Narrative Delivery Methods)

### 4.1. 환경 서사 (Environmental Storytelling) — 최우선

아이템계의 **지형, 배경, 오브젝트**가 이야기를 말한다. 텍스트 없이도 "이곳은 전장이었구나", "여기는 주방이었구나"를 느끼게 한다.

| 수단 | 예시 |
| :--- | :--- |
| 타일 팔레트 | 부식된 블레이드 → 파이프/녹슨 금속 타일, 경비대 블레이드 → 콘크리트/격벽 타일 |
| 배경 오브젝트 | 깨진 터미널, 꽂힌 파이프, 녹슨 밸브 |
| 색조/분위기 | Magic 등급: 단색 기조. Legendary: 지층마다 색조 변화 |
| 파티클/이펙트 | 방어선 → 재 날림, 에코 연구실 → 에너지 불꽃 |

#### 4.1.1. 환경 서사 7대 원칙

아이템계와 월드 모두에 적용되는 환경 서사의 핵심 설계 원칙이다. (참조: `Reference/witcher3_인사이트.md`)

**원칙 1: "씬이 플롯을 이긴다"**

Memory Strata의 각 층은 하나의 독립된 **씬(Scene)**이다. 전체 지층을 관통하는 거대한 서사보다, 각 층에서 마주치는 환경적 순간들—지형, 몬스터 배치, NPC 대사 한 줄, 오브젝트 배치—이 플레이어 기억에 남는다. 각 층이 결말 없이도 흥미로운 경험이 되어야 한다.

> *"좋은 플롯이란 좋은 씬을 만들어내는 플롯이다. 이상적인 미스터리란, 결말이 빠져 있어도 읽고 싶은 것이다."* — Raymond Chandler

| 적용 공간 | 설계 방법 |
| :--- | :--- |
| Memory Strata | 각 층 = 독립 씬. 그 층만으로 완결된 인상을 줄 것 |
| 월드 층위 | 각 층위 = 독립 에피소드. 층위 간 연결은 실(thread)에 불과 |

**원칙 2: 정보의 의도적 제거 (Subtraction of Key Information)**

모든 것을 즉시 설명하지 않는다. 초반 지층에서 의미심장한 상징물을 배치하되 그 의미는 깊은 지층에서야 드러나는 구조. "이 아이템에 무슨 일이 있었는가?"라는 질문을 플레이어 스스로 던지게 만든다.

| 지층 깊이 | 정보 공개 수준 |
| :--- | :--- |
| 지층 1 | 미스터리 제시 — 의미 불명의 상징, 맥락 없는 흔적 |
| 중간 지층 | 단편적 해석 — NPC 대사/오브젝트가 실마리 제공 |
| 최심층 | 핵심 해소 — 기원 사건의 진면목 (그러나 100% 답은 주지 않음) |

**원칙 3: 묵시적 서사 (Subtle Exposition)**

세계관의 핵심 가치를 직접 텍스트로 전달하지 않고, 환경 자체가 그것을 체화한다. 전쟁에서 사용된 검의 아이템계라면 — 땅이 갈라져 있고, 잔해가 흩어져 있고, 불에 탄 나무가 서 있고, 기억 단편들이 전사의 형상을 하고 있다. 설명 없이 보여주는 것.

| 금지 | 권장 |
| :--- | :--- |
| "이 검은 100년 전쟁에 쓰였다" | 바닥에 갈라진 갑옷 파편, 벽에 박힌 화살, 불탄 깃발 |
| "이 지팡이의 주인은 미쳐버렸다" | 벽면의 미친 낙서, 깨진 거울, 뒤틀린 실험 기구 |

**원칙 4: 40초 법칙 (40-Second Rule)**

플레이어는 40초마다 무언가에 집중할 수 있어야 한다. 텅 빈 복도가 40초 이상 이어지면 집중이 깨진다.

| 이벤트 유형 | 예시 |
| :--- | :--- |
| 시각적 | 배경 변화, 새 타일셋, 의미심장한 오브젝트 |
| 게임플레이 | 적 웨이브, 함정, 숨겨진 방 입구 |
| 서사적 | NPC 한 마디, 환경 변화(지진, 조명 변화), 보스 전조 |

**원칙 5: "기근 속 소시지" 방지 (Environmental Consistency)**

환경 디테일이 서사와 모순되면 안 된다. "배고프다"고 말하는 NPC 뒤에 풍성한 음식이 보이면 안 된다.

| 검증 질문 | 모순 사례 |
| :--- | :--- |
| 이 아이템의 기원 테마와 모든 오브젝트가 일관되는가? | 식칼 아이템계에 에코 수정이 배치됨 |
| 레어리티에 맞는 환경 복잡도인가? | Normal 아이템계에 복잡한 서사 오브젝트 과다 |
| 지층별 감정 곡선이 일관되는가? | 비장한 전쟁 테마 중간에 코믹한 오브젝트 |

**원칙 6: 시선 차단과 서프라이즈 (Visual Blocking & Reveal)**

입구에서 방 전체가 한눈에 보이지 않도록 L자/U자 구조를 기본으로 한다. 시선 차단은 기술적 필요(메모리 스트리밍)와 서사적 효과(극적 공개)를 동시에 달성한다.

| 기법 | 효과 |
| :--- | :--- |
| L자/U자 룸 템플릿 | 코너를 돌 때 새 장면이 드러남 — 발견의 쾌감 |
| 수직 시선 차단 | 아래로 떨어진 후 새 바이옴 공개 |
| 전경 오브젝트 | 큰 기둥/잔해 뒤에 숨겨진 요소 배치 |

**원칙 7: Fire & Ember — 감정적 뱅 모먼트 설계**

시각적 스펙타클(Ember/불씨)만으로는 영혼이 없다. 캐릭터의 감정, 도덕적 갈등(Fire/불)이 선행되어야 Ember가 의미를 갖는다. 보스 방의 화려한 연출보다, 그 보스가 왜 거기 있는지의 서사가 먼저다.

| 요소 | Fire (영혼) | Ember (스펙타클) |
| :--- | :--- | :--- |
| 보스 전 | 보스의 정체/동기가 환경으로 암시됨 | 보스 방 입장 연출, 카메라 줌 |
| 지층 클리어 | 아이템의 기억을 한 단계 더 이해 | 클리어 이펙트, 보상 팡파르 |
| 최심층 | 아이템 기원의 핵심 진실 | 고유 바이옴, 특수 이펙트 |

### 4.2. 몬스터 디자인 (Monster as Story)

아이템계의 몬스터는 랜덤이 아니다. **아이템의 과거에서 파생된 존재**다.

| 원칙 | 설명 |
| :--- | :--- |
| 테마 일관성 | 유지보수 블레이드 아이템계에서 전투용 구조체가 나오면 안 된다. 오작동 유지보수 드론만 나와야 한다 |
| 지층 진행에 따른 변화 | 초반 지층: 약한 잔상 (파손된 자동화 유닛). 후반 지층: 강한 잔상 (대형 오작동 드론). 최심층: 오류의 결집체 |
| 행동 서사 | 파손된 자동화 유닛은 오작동하며 접근 (기능 오류). 보안 프로토콜 잔상은 대열을 짓고 순찰 (경비 루틴) |

### 4.3. NPC/잔영 존재 (Residual Presence)

아이템의 과거와 관련된 존재가 잔영(residual)으로 등장하여 **실루엣, 행동 패턴, 환경 변화**로 이야기를 전달한다. 잔영은 말하지 않는다 — 서 있고, 바라보고, 사라진다. 텍스트가 필요한 경우 환경 레이어(오브젝트 상호작용 텍스트, 환경음)로 처리한다.

| 원칙 | 설명 |
| :--- | :--- |
| 행동으로만 | 잔영 자체는 대사 없음. 고개를 끄덕이거나, 무언가를 가리키거나, 플레이어를 바라보고 사라진다 |
| 반복하면 변화 | 같은 잔영을 재방문하면 다른 행동 패턴. 3회까지 변화 후 고정 |
| 퍼즐 조각 | 한 지층의 잔영 행동만으로는 전체를 알 수 없다. 여러 지층의 잔영을 조합해야 전모 파악 |
| 선택적 | 잔영 옆을 지나쳐도 된다. 관찰은 플레이어의 선택 |

행동 묘사 예시 — 경비대 순찰 블레이드 5층:

> *순찰대원의 잔영이 초소 벤치에 앉아 블레이드를 내려다보고 있다. 플레이어가 가까이 오면 고개를 든다. 손이 블레이드를 향해 뻗었다가 멈춘다. 환경 텍스트: "…this blade kept me awake. It always did."*

행동 묘사 예시 — 대격변 지휘관의 클리버 지층 3:

> *붕괴된 격벽 양쪽에 두 잔영이 서 있다 — 서로 다른 진영의 보호복. 등을 맞대고 있다. 플레이어가 접근하면 두 잔영이 동시에 돌아보았다가, 각자 반대 방향으로 사라진다. 환경 텍스트: "We were protecting our sector too."*

#### 4.3.1. 신호/잡음 이론 (Signal/Noise Theory)

플레이어의 주의력이 분산된 상태(전투 중, 플랫폼 점프 중)에서는 핵심 서사 정보를 전달하지 않는다. 핵심 정보는 플레이어의 시선이 고정된 순간에 전달한다.

| 잡음 수준 | 상황 | 전달 가능 정보 |
| :--- | :--- | :--- |
| 높음 (전투 중) | 액션, 회피, 스킬 사용 | 환경 서사만 (지형, 배경, 파티클) |
| 중간 (탐색 중) | 이동, 아이템 줍기 | 짧은 NPC 한 마디, 오브젝트 상호작용 |
| 낮음 (안전 지대) | 휴식, 층간 대기, 세이브 포인트 구역 | 핵심 서사 정보, 환경 텍스트, 퍼즐 힌트 |

#### 4.3.2. 기억 단편 어투 차별화

기억 단편는 아이템의 카테고리와 기원 테마에 따라 다른 말투와 성격을 가진다. 이를 통해 아이템의 다층적 역사를 자연스럽게 전달한다.

| 아이템 카테고리 | 기억 단편 기본 어투 | 성격 |
| :--- | :--- | :--- |
| Blade/Cleaver | 보안 프로토콜, 간결한 보고조 | 충성, 의무, 명예 |
| Shiv | 잠입, 속삭이는 | 잠입, 정찰, 비밀 임무 |
| Emitter | 연구원적, 데이터 읊조리는 | 분석, 탐구, 강박 |
| Railbow | 원거리 관측자, 분석적 | 인내, 정밀, 고독 |

#### 4.3.3. "빈 시간" 설계 (Breathing Room)

플레이어에게 "할 일이 없는 순간"을 의도적으로 제공한다. 아이템계 클리어 후 즉시 보상 화면으로 넘어가지 말고, 잠시 그 공간에 머물 수 있게 한다.

| 적용 지점 | 설계 |
| :--- | :--- |
| 지층 보스 클리어 후 | 보스 방에 잠시 머물 수 있음. 환경이 변화하며 서사 여운 |
| 최심층 클리어 후 | 아이템 기원의 핵심 장소가 평화롭게 변함. 플레이어가 돌아다닐 수 있음 |

### 4.4. 플레이버 텍스트 (Flavor Text)

인벤토리에서 아이템을 볼 때 표시되는 한두 줄의 텍스트. 아이템계에 들어가기 전 호기심을 유발한다.

| 레어리티 | 플레이버 텍스트 스타일 |
| :--- | :--- |
| Normal | 단순 묘사 — *"평범한 유지보수 블레이드. 어딘가의 파이프 격실에서 왔다."* |
| Magic | 소유자 암시 — *"칼날에 'COR-7'이라는 각인이 희미하게 남아 있다."* |
| Rare | 사건 암시 — *"이미터 끝에서 아직도 에코 출력이 불안정하게 흔들린다."* |
| Legendary | 역사 암시 — *"심연 전쟁의 마지막 방어전, 이 클리버를 쥔 자는 돌아오지 않았다."* |
| Ancient | 수수께끼 — *"이것은 파편이 아니다. 구조물이 아직 설계도였던 시절의 기억이다."* |

## 5. 세계관 톤 (Worldbuilding Tone)

### 5.1. 이중 톤: 산업적 숭고 + 경쾌한 야리코미

ECHORIS의 세계관은 두 가지 톤이 공존한다:

| 톤 | 적용 공간 | 표현 |
| :--- | :--- | :--- |
| 산업적 숭고 | 월드, Legendary/Ancient 아이템계 | 거대 구조물의 경외감, 쓸쓸한 폐허, 구조물의 비밀 |
| 경쾌한 야리코미 | Normal/Magic 아이템계 | 유머러스한 잔영 행동, 가벼운 환경 텍스트, 즐거운 파밍 분위기 |

**전환 규칙:** 레어리티가 올라갈수록 톤이 무거워진다. Normal 블레이드의 파손된 드론은 코믹하고, Ancient 코어 프래그먼트의 빌더 프로토콜 잔상은 숭고하다. 플레이어는 장비를 키울수록 더 무거운 이야기를 만나게 된다.

### 5.2. "심연(Abyss)"의 정체

게임 전체를 관통하는 미스터리:

> "심연이란 무엇인가?"

- 월드의 가장 깊은 곳에 심연이 있다
- 아이템의 가장 깊은 지층에도 심연이 있다
- 모든 Ancient 아이템의 최심층은 같은 장소를 가리킨다
- **아이템의 기억을 깊이 파고들면, 결국 세계의 기원에 도달한다**

이것이 탐험(월드 최심부)과 야리코미(아이템계 최심층 지층)가 서사적으로 같은 목적지를 향하게 만드는 핵심 장치다.

### 5.3. NPC 대사 철학

| 원칙 | 설명 |
| :--- | :--- |
| 짧고 인상적 | 3문장 이하. 한 마디가 한 편의 이야기보다 강하다 |
| 정보보다 감정 | "전쟁은 100년 전에 있었다" 대신 "아직도 비명이 들린다" |
| 신뢰할 수 없는 화자 | 잔영은 자기 기억만 안다. 편향적이고, 때로는 거짓말한다 |
| 행동으로 보여주기 | 잔영이 특정 벽을 가리키면 숨겨진 방이 있다. 대사가 게임플레이와 연결 |

## 6. 욕망의 하강 (Descent of Desire) — 지층 깊이 = 욕망의 깊이

> **리서치 기반:** (미작성) <!-- Libido_NarrativeTheme_Research.md, Research_Religious_Symbolism_TabooDesign.md, ItemWorld_DesireDepth_ThemeMapping_Research.md — 추후 제작 예정 -->

### 6.1. 핵심 전제

> "아이템의 기억을 깊이 파고들수록, 표면의 이야기 아래 숨겨진 **원초적 욕망**과 마주한다."

아이템계의 지층은 단순한 난이도 계층이 아니다. 각 지층은 인간 정신의 깊이에 대응하며, 깊이 내려갈수록 더 원초적이고 금기적인 테마가 상징적으로 드러난다. Binding of Isaac이 Basement(일상 공포)에서 Dark Room(존재론적 진실)까지 하강하듯, ECHORIS도 아이템계 하강이 곧 욕망의 심층으로 들어가는 여정이다.

### 6.2. 지층별 욕망의 하강 모델

| 지층 | 이름 | 심리적 층위 | 기술적 프레임 | 미학적 전환 |
| :--- | :--- | :--- | :--- | :--- |
| **1** | 기억의 표면 / "인터페이스 층" | 의식 — 아이템이 기억하고 싶은 것 | 정상 작동 인터페이스 — 정돈된 데이터, 예측 가능한 패턴 | 산업적 정밀성, 규칙적 구조 |
| **2** | 기억의 이면 / "손상 섹터" | 전의식 — 아이템이 감추는 것 | 데이터 부식 시작 — 불일치, 오류 로그, 숨겨진 섹터 | 글리치, 데이터 노이즈, 숨겨진 백도어 |
| **3** | 기억의 심층 / "코어 덤프" | 무의식 — 아이템이 기억하지 못하는 것 | 구조 해체 — 기계와 유기체의 경계 붕괴 | **테크노-오가닉 전환**. 파이프가 혈관처럼, 벽이 호흡하듯 |
| **4** | 기억의 핵 / "제로 포인트" | 원형적 무의식 — 아이템의 본질 | 원점 — 데이터도 구조도 없는 공백 | 역설적 숭고. 과잉에서 **공백**으로. 미니멀, 광활, 경외 |
| **심연** | Ancient 전용 | 초개인적 — 세계의 기억 | 거대 빌더의 건설 알고리즘 공간 | 비유클리드 기하학. 구조물 생성의 원형 |

### 6.4. 지층별 욕망 층위의 진행

**지층 1-2 (Surface) — 인식의 표면:**
- 아이템 테마에 충실한 "정형화된 오류". 탐욕=금화, 교만=거울, 분노=불꽃
- 적은 "수호자" — 명확한 형태. 기술 오브젝트: 인디케이터 라이트, 터미널, 냉각액 → 치유/버프
- 적용 테마: G-01 쇠사슬(속박), G-02 원소(숭고), G-03 파괴(쾌감)
- **감정 톤:** 산업적 정밀성. 익숙한 던전 크롤러의 문법

**지층 3-5 (Mid) — 감추고 싶은 것:**
- 표면 아래 숨겨진 취약함이 드러남. 오류의 이면 = 공포
- 적은 "그림자" — 아이템 자신의 왜곡된 반영. 환경이 부분적 손상/변형
- 적용 테마: G-04 지배(왕좌), G-05 시선(거울), G-06 물신(인형), G-07 해체(신체)
- 기술 오브젝트: 금이 간 모니터, 손상된 데이터 로그 → 리스크-리워드
- **감정 톤:** "불편하지만 부정할 수 없는"

**지층 6-8 (Deep) — 기억하지 못하는 것:**
- 오류를 넘어선 원초적 충동. 주체/객체 경계 와해
- 적은 "원초적 충동" — 형태 불분명, 적과 환경의 경계 모호
- 적용 테마: G-08 탐식(흡수), G-09 부패(오염), G-10 변신(해체), G-11 이계(타자), G-12 모독(성역)
- Kristeva의 비체(Abjection): 파이프가 혈관처럼 호흡하고, 금속 바닥이 체액
- 기술 오브젝트: 역전된 회로 패턴, 오류 코드 마스크 → 강력하지만 변형 유발
- **감정 톤:** 테크노-오가닉 호러. 문명이 가장 강하게 억압하는 영역

**지층 핵 (Core) — 왜 원하는가:**
- "무엇을 원하는가"가 아니라 **"왜 원하는가"**
- 적용 테마: G-13 시간(퇴행), G-14 죽음(비존재)
- 환경: **과잉에서 공백으로의 전환**. 미니멀, 광활, 경외
- 기술 오브젝트: 빈 터미널, 응답 없는 시스템, 빛만 있는 공간
- **감정 톤:** 역설적 숭고. 공포를 넘어선 경외

**심연 (Ancient 전용) — 욕망을 넘어선 것:**
- 개별 아이템을 넘어선 "세계의 기억". 모든 Ancient 최심층이 같은 장소를 가리킴 (§5.2 연결)
- 적은 "거대 빌더의 건설 프로토콜이 자아를 가진 존재"
- 환경: 비유클리드 기하학. 공간의 규칙이 붕괴
- **감정 톤:** 불가능한 아름다움. "거대 빌더가 여전히 누비는 세계에서, 그 흔적 아래 남는 것은 무엇인가"

### 6.5. 아이템 유형별 테마 분기

아이템의 유형에 따라 지층에서 활성화되는 테마 그룹이 달라진다:

| 아이템 유형 | Surface | Mid | Deep | Core |
| :--- | :--- | :--- | :--- | :--- |
| 무기 (검/창) | G-03 화염의 무도회 | G-04 피의 왕좌 | G-08 심연의 식탁 | G-14 경계의 문 |
| 방어구 (갑옷) | G-01 쇠사슬의 정원 | G-07 만물의 눈 | G-10 변신의 탑 | G-13 영원의 요람 |
| 장신구 (반지) | G-02 불꽃과 얼음 | G-06 인형사의 작업실 | G-12 금단의 성역 | G-14 경계의 문 |
| 이미터/촉매 | G-05 거울의 회랑 | G-11 이계의 숲 | G-09 부패의 정원 | G-13 영원의 요람 |

> **규칙:** 아이템의 기원 테마(§7 테마 풀)가 1차 결정, 아이템 유형이 2차 결정. 두 레이어가 합쳐져 지층의 최종 환경을 구성한다.

### 6.6. 적 디자인의 하강 계보

| 테마 | 지층 1 (표면) | 지층 2 (이면) | 지층 3 (심층) | 지층 4 (핵) |
| :--- | :--- | :--- | :--- | :--- |
| T-HABITAT (거주 모듈) | 정돈된 거주 모듈, 작동하는 조명 | 잠긴 문, 꺼진 조명, 침묵하는 방 | 벽 속에서 들리는 소리, 호흡하는 환기구 | 빈 요람, 응답 없는 시스템 |
| T-FOUNDRY (제조 라인) | 작동하는 제조 라인, 정밀한 도구 | 과열된 용융로, 변형되는 금속 | 용융로가 의지를 가짐, 금속이 유기체로 | 꺼진 화덕, 최초의 타격 기록 |
| T-MALFUNCTION (시스템 오류) | 정렬된 데이터 블록, 안정 시스템 | 오류 로그 누적, 식별 코드 손상 | 데이터와 물리 경계 붕괴, 시스템이 자아를 가짐 | 빈 메모리 — "여기 무엇이 저장되었는가" |
| T-BREACH (심연 전쟁) | 붕괴 직전의 격벽, 긴장감 | 빛이 닿지 않는 구조물 틈, 반향 | 공간 자체가 불안정, 중력 왜곡 | 구조물 끝 — 영원한 낙하 |

### 6.8. 설계 원칙

| 원칙 | 설명 |
| :--- | :--- |
| **"금기는 설명하지 않는다"** | Isaac의 Belt가 이동속도를 올리는 이유를 게임이 설명하지 않는 것처럼, 종교 상징의 "왜"는 플레이어의 추론에 맡긴다 |
| **"하강은 장르의 변화다"** | 지층 1(산업 구조물) → 2(시스템 오류) → 3(테크노-오가닉 호러) → 4(숭고/추상). BLAME!의 구조물→심연 전환 참조 |
| **"모든 적은 자기 자신이다"** | 깊은 지층의 적은 점점 더 아이템 자신(그리고 플레이어)을 닮아간다 |
| **"위반이 성스러움이다"** | 더 깊은 지층을 돌파하는 것 = 금기의 위반이자 아이템의 진정한 기억에 도달하는 성스러운 행위. Bataille의 "금기가 없으면 위반도 없다" |
| **"변형이 해방이다"** | 깊은 지층에서 획득하는 아이템/버프가 캐릭터를 변형시키지만, 그 변형이 곧 강화 |
| **"부재가 가장 큰 공포다"** | 최심층은 과잉이 아니라 공백. 바디 호러의 과잉 이후 오는 텅 빈 공간이 가장 강렬하다 |
| **"죄의 근저에는 공포가 있다"** | 표면 지층은 "죄(행동)"를 다루고, 심층은 그 죄를 만들어낸 "공포(근원)"를 다룬다 |

### 6.10. 핵심 테마 문장 (Thematic Pillars)

1. **"심연은 배고프다"** — 세계 자체가 맹목적 의지/욕망의 표현
2. **"가장 깊은 곳에서 마주할 것은 그대 자신"** — 외부 탐험 = 내면 탐험
3. **"모든 무기에는 주인의 집착이 깃든다"** — 아이템 = 응축된 욕망
4. **"고통 없이 얻은 힘은 없다"** — 성장의 조건 = 고통
5. **"당신은 떠날 수 있다. 하지만 돌아올 것이다"** — 야리코미의 반복 강박
6. **"속박된 자만이 진정한 해방을 안다"** — 제한이 있어야 초월이 있다
7. **"욕망은 채워지지 않는다. 다만 변할 뿐"** — 오브제 쁘띠 아의 영원한 미끄러짐

## 7. 테마 풀 시스템 (Theme Pool System)

> **참고:** §6의 "욕망의 하강" 모델은 이 테마 풀의 지층별 변형 레이어로 기능한다.

수천 개 아이템에 개별 서사를 쓰는 것은 불가능하다. 대신 **테마 풀**을 조합하여 서사를 절차적으로 생성한다.

### 7.1. 테마 풀 구조

```
아이템 서사 = 무기 카테고리 × 기원 테마 × 레어리티 깊이
```

| 무기 카테고리 | 기본 기원 테마 |
| :--- | :--- |
| Blade (절삭기) | 순찰원/보안원/방어전/격벽 순회/의무 |
| Cleaver (격벽 파쇄도) | 격벽 돌파/방어선 파쇄/맹세/구조 붕괴/유산 |
| Shiv (외과 단도) | 잠입/정찰/비밀 임무/생존/암호 해독 |
| Harpoon (빌더 리바 장창) | 격벽 관통/심층 탐사/구조 작업/대형 구조체 제압/측량 |
| Chain (강철 사슬) | 견인/포획/구속/연결 작업/원거리 제어 |
| Railbow (전자기 가속 투척기) | 원거리 관측/측량/저격/탐사/방어선 지원 |
| Emitter (파동 방출기) | 에코 공학/기억 연구/보수/오류/분석 |

### 7.2. 테마 → 몬스터 풀 매핑 예시

| 기원 테마 | 몬스터 풀 | 지형 팔레트 | 분위기 |
| :--- | :--- | :--- | :--- |
| 주방/도축 | 파손 드론, 오작동 자동화 유닛, 부식 센서 | 나무 바닥, 피 얼룩, 갈고리 | 으스스한 일상 |
| 전쟁/전장 | 병사 잔영, 화살 유령, 공성 병기 파편 | 석조, 깃발, 무너진 성벽 | 비장함 |
| 에코 공학/데이터 연구 | 실험 실패작, 원소 정령, 변이 생물 | 수정, 공명 패턴, 데이터 코어, 분석 장치 | 신비로움 |
| 사냥/숲 | 변이 생물체, 과성장 식물 구조체, 생태계 오류 | 나무, 이끼, 동굴, 덫 오브젝트 | 야생적 |
| 수행/격투 | 수련생 잔영, 무술 사범 유령 | 도장, 돌바닥, 산사, 폭포 | 고요한 긴장 |
| 저주/어둠 | 그림자, 원한의 형체, 저주받은 유물 | 검은 석재, 부서진 신상, 핏빛 달 | 공포 |

### 7.3. 절차적 서사 조립

아이템이 드랍될 때, 다음이 결정된다:

```
1. 무기 카테고리 → 기원 테마 풀에서 1개 선택 (시드 기반)
2. 레어리티 → 서사 깊이 결정 (NPC 수, 대사 수, 보스 서사 유무)
3. 아이템 레벨 → 몬스터 강도 스케일링
4. 테마 → 몬스터 풀, 지형 팔레트, 분위기 결정
5. 시드 → NPC 대사를 대사 풀에서 선택, 배치 지층 결정
```

이 구조 덕분에 같은 "검" 카테고리에서도:
- 어떤 검은 전쟁의 기억이 가득하고
- 어떤 검은 결투의 역사를 품고 있고
- 어떤 검은 처형의 어둠이 서려 있다

## 8. 월드 서사 (World Narrative)

### 7.1. 층위별 독립 서사

월드의 각 층위(바이옴)는 독립된 서사를 가진다. 그 층위에서 드랍되는 아이템은 해당 층위의 서사를 반영한다.

| 원칙 | 설명 |
| :--- | :--- |
| 층위 = 장소의 역사 | Concordia 층위에서 드랍되는 검은 "보안 순찰 블레이드" 테마가 높은 확률 |
| 교차 서사 | A 층위의 아이템계에서 B 층위에 대한 힌트 발견 → 탐험 동기 |
| 환경 서사 우선 | 벽에 긁힌 자국, 부서진 동상, 마른 분수 — 텍스트 없이 역사를 전달 |

### 7.2. 수직적 서사 (메트로베니아 + 야리코미 통합)

```
월드 탐험 (수평 서사) ──── 층위의 현재 상태를 목격
        │
        ▼
아이템 획득 ──────────── 층위의 과거 조각을 손에 쥠
        │
        ▼
아이템계 진입 (수직 서사) ── 과거 속으로 다이빙
        │
        ▼
깊은 지층 도달 ───────── 층위의 비밀 발견
        │
        ▼
월드 복귀 ──────────── "아, 그래서 이 층위가 이렇게 된 거구나"
```

플레이어는 월드를 탐험하면서 **현재**를 보고, 아이템계에서 **과거**를 체험하고, 다시 월드로 돌아와서 **이해**한다. 탐험과 야리코미가 서사적으로 하나의 축을 이룬다.

## 9. 서사 설계 금지 규칙 (Narrative Never-Do Rules)

| 규칙 | 이유 |
| :--- | :--- |
| 강제 컷신 금지 | 액션 RPG의 흐름을 끊는다. 모든 서사는 플레이 중 전달 |
| 긴 대사창 금지 | 3문장 이하. 읽지 않아도 게임 진행에 지장 없어야 한다 |
| 서사 필수 진행 금지 | NPC를 만나지 않아도, 대사를 읽지 않아도 아이템계를 클리어할 수 있다 |
| 레어리티 = 서사 강제 금지 | Normal 아이템 플레이어에게 "서사를 놓치고 있다"는 압박을 주지 않는다. 서사는 보너스 |
| 설명 위주 서사 금지 | "100년 전 전쟁이 있었다"는 안 된다. "아직도 비명이 들린다"가 맞다 |
| 유일한 진실 금지 | 유령마다 기억이 다르다. 플레이어가 해석한다. 공식 정답은 없다 |

## 10. 검증 기준 (Verification Checklist)

- [ ] 아이템계에 들어갔을 때, 몬스터와 지형만으로 "이 아이템이 어떤 역사를 가졌는지" 짐작할 수 있는가?
- [ ] Normal 아이템과 Legendary 아이템의 서사 깊이 차이가 체감되는가?
- [ ] NPC 대사를 읽지 않고 스킵해도 게임 진행에 전혀 지장이 없는가?
- [ ] NPC 대사를 읽으면 "더 알고 싶다"는 호기심이 생기는가?
- [ ] 같은 무기 카테고리의 두 아이템이 서로 다른 테마를 가져 차별화되는가?
- [ ] 깊은 지층에서 등장하는 서사가 얕은 지층의 서사보다 무게감이 있는가?
- [ ] 월드 층위의 환경 서사와 해당 층위 아이템계의 서사가 일관되게 연결되는가?
- [ ] "심연(Abyss)"에 대한 힌트가 Ancient 아이템계에서만 등장하여 야리코미 동기를 강화하는가?
- [ ] 강제 컷신이 단 하나도 없는가?
