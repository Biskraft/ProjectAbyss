## 1. 핵심 원칙: 아이템이 곧 서사 매체다

### 1.1. 한 줄 정의

> "모든 아이템에는 과거가 있고, 아이템계는 그 과거를 체험하는 공간이다."

Project Abyss에서 내러티브는 컷신이나 대사창이 아니다. **아이템 속에 들어가면 그 아이템의 역사가 몬스터로, 지형으로, NPC로, 분위기로 드러난다.** 플레이어는 아이템계를 탐험하며 자연스럽게 이야기를 체험한다.

### 1.2. 왜 아이템 내러티브인가?

| 문제 | 해법 |
| :--- | :--- |
| 횡스크롤 액션에서 컷신은 흐름을 끊는다 | 서사를 전투/탐험 안에 녹인다 — 아이템계 자체가 이야기 |
| 절차적 생성 맵에서 핸드크래프트 서사가 가능한가? | 아이템의 "테마"가 절차적 생성의 시드 역할을 한다 (몬스터 풀, 지형 팔레트, NPC 배치) |
| 디스가이아 아이템계는 반복 파밍인데 서사가 가능한가? | 반복할수록 더 깊은 지층에서 새로운 단편이 등장. 야리코미와 서사가 같은 방향을 가리킨다 |
| 수천 개 아이템마다 고유 스토리를 쓸 수 있는가? | **테마 풀 + 조합 시스템**으로 해결. 아이템 카테고리 × 레어리티 × 바이옴의 조합 |

### 1.3. 3대 기둥 정렬

| 기둥 | 아이템 내러티브에서의 역할 |
| :--- | :--- |
| 메트로베니아 탐험 | 월드의 각 층위는 독립 서사를 가짐. 아이템은 그 층위의 역사 조각. 층위 간 서사 힌트가 탐험 동기 |
| 아이템계 야리코미 | 아이템계 깊이 = 서사 깊이. 각 지층마다 새 이야기 단편. 모든 지층 클리어 시 아이템의 기원 전모 해금 |
| 온라인 멀티플레이 | 파티원이 동일 아이템계에서 서로 다른 NPC 단편을 발견 → 허브에서 정보 교환 → "수집형 서사" |

## 2. 아이템 서사 구조 (Item Narrative Architecture)

### 2.1. 모든 아이템은 "전생(前生)"을 가진다

아이템은 단순한 스탯 덩어리가 아니다. 모든 아이템에는 **기원(Origin)**이 있고, 아이템계에 들어가면 그 기원의 잔영이 세계로 구현된다.

```
아이템 기원 (Origin)
├── 누가 만들었는가? (Creator)
├── 무엇에 사용되었는가? (Purpose)
├── 어떤 역사를 겪었는가? (History)
└── 왜 지금 여기 있는가? (Fate)
```

### 2.2. 아이템계 = 아이템의 기억 공간

아이템계에 들어간다는 것은 **아이템의 기억 속으로 다이빙하는 것**이다:

- **몬스터** = 아이템이 겪은 과거의 잔영 (살해한 것, 살해당한 것, 두려워한 것)
- **지형** = 아이템이 존재했던 장소의 기억 (주방, 전장, 마탑, 숲)
- **NPC/유령** = 아이템의 이전 주인, 목격자, 희생자. 실루엣, 행동 패턴, 환경 변화로만 존재를 드러낸다.
- **보스** = 아이템 기원의 핵심 사건을 상징하는 존재
- **이노센트** = 아이템에 깃든 의지. 야생(포획 전)은 기억의 혼란, 복종(포획 후)은 정화된 의지

### 2.3. 레어리티 = 서사의 깊이

| 레어리티 | 서사 복잡도 | 아이템계에서의 표현 |
| :--- | :--- | :--- |
| Normal | 단순한 일상 (한 줄 배경) | 단조로운 몬스터 풀, 유령 없음, 환경 힌트만 |
| Magic | 약간의 사연 (누군가의 소유물) | 1~2종 테마 몬스터, 간헐적 유령 실루엣/행동 |
| Rare | 사건이 있는 과거 (전투, 비극, 모험) | 테마 몬스터 풀, 유령 존재 1체, 보스에 서사 연결 |
| Legendary | 역사적 사건의 중심 (전쟁, 왕국, 재앙) | 복합 테마, 유령 존재 다수, 보스 환경 연출, 환경 변화 |
| Ancient | 신화/전설급 기원 (신의 유물, 세계의 비밀) | 고유 바이옴, 전용 보스, 다층 서사, 세계관 핵심 힌트 |

## 3. 아이템 서사 예시 (Narrative Examples)

### 3.1. 녹슨 식칼 — Normal 등급

**기원:** 어느 마을 정육점 주인의 식칼. 매일 가축을 잡는 데 쓰였다.

| 요소 | 아이템계 표현 |
| :--- | :--- |
| 몬스터 | 돼지, 소, 닭의 원혼 (작고 약한 동물형 몬스터). 칼에 잘린 것들의 분노 |
| 지형 | 나무 도마 질감의 바닥, 피가 마른 벽, 갈고리가 걸린 천장 |
| NPC | 없음 (Normal은 서사가 얕다) |
| 보스 | 거대한 돼지 원혼 — 가장 많이 잡힌 가축의 집합체 |
| 분위기 | 축축하고 비린내 나는 주방. 단순하지만 약간 으스스한 |

### 3.2. 기사의 장검 — Magic 등급

**기원:** 왕국 수비대 기사가 10년간 써온 검. 수많은 전투를 함께했다.

| 요소 | 아이템계 표현 |
| :--- | :--- |
| 몬스터 | 검에 쓰러진 도적, 마물 잔영. 갑옷 파편이 떠다니는 유령 병사 |
| 지형 | 성벽, 초소, 훈련장 — 기사가 근무했던 장소들의 파편 |
| NPC | 기사의 유령 (1체) — 첫 번째 지층에서 등장. 초소 앞에 서서 플레이어의 검을 바라보고 고개를 끄덕이다 사라진다. 환경 텍스트: "아직도 이 검이 쓰이고 있나..." |
| 보스 | 기사가 끝내 쓰러뜨리지 못한 도적 두목의 잔영 |
| 분위기 | 먼지 낀 성채, 충성과 의무의 잔향 |

### 3.3. 마녀의 지팡이 — Rare 등급

**기원:** 숲 깊은 곳의 마녀가 연구에 사용한 지팡이. 실험 중 폭주가 일어나 마녀와 함께 버려졌다.

| 요소 | 아이템계 표현 |
| :--- | :--- |
| 몬스터 | 실험 실패작 — 기형 동물, 합성 생물체, 변이된 식물 |
| 지형 | 마법 연구실, 끓는 가마솥, 부서진 서가, 보라빛 수정 |
| NPC | 마녀의 고양이 유령 — 여러 지층에 걸쳐 등장, 마녀의 실험 일지를 단편적으로 전달 |
| NPC | 마녀 본인의 잔영 — 깊은 지층에서만 등장, 후회와 경고 |
| 보스 | 마녀의 마지막 실험 — 제어를 벗어난 합성 마물 |
| 분위기 | 신비롭고 위험한 마법의 잔류. 호기심과 공포가 교차 |

### 3.4. 심연의 검 — Legendary 등급

**기원:** 100년 전 심연 전쟁에서 영웅이 사용한 검. 전쟁의 마지막 전투에서 영웅과 함께 심연에 빠졌다.

| 요소 | 아이템계 표현 |
| :--- | :--- |
| 몬스터 | 심연 전쟁의 양 진영 병사 잔영. 지층이 깊어질수록 적 병사 → 아군 병사 → 민간인 희생자로 변화 |
| 지형 | 초반 지층: 전장 평원. 중반 지층: 함락된 성. 최심층: 심연의 경계 — 현실이 무너지는 지형 |
| NPC | 영웅의 전우 3인 — 지층 1/2/3에 등장, 각자 전쟁의 다른 면을 증언 |
| NPC | 적 장수의 잔영 — 지층 3에서 등장, "우리도 지키려 했을 뿐이다" |
| 보스 | 지층 2: 적국 장수 (전쟁의 직접적 적) |
| 보스 | 최심층: 심연의 현신 — 전쟁을 촉발한 진짜 원흉 |
| 분위기 | 비장함, 영웅담. 깊이 들어갈수록 "선과 악의 경계"가 흐려지는 서사 |

### 3.5. 세계수의 가지 — Ancient 등급

**기원:** 세계가 만들어질 때 세계수에서 떨어진 가지. 이 가지에는 세계 창조의 기억이 담겨 있다.

| 요소 | 아이템계 표현 |
| :--- | :--- |
| 몬스터 | 태초의 원소 정령, 형태가 확정되지 않은 원초적 존재, 빛과 어둠의 단편 |
| 지형 | 지층 1~2: 원시 숲 (세계수 주변). 지층 3: 혼돈의 공간 (세계가 형성되기 전). 최심층: 심연 그 자체 |
| NPC | 세계수의 의지 — 정체불명의 목소리. 세계의 비밀을 암시하는 수수께끼 |
| NPC | 다른 Ancient 아이템의 잔영 — 교차 서사 (이 아이템과 다른 Ancient 아이템이 같은 사건에 관여) |
| 보스 | 지층 2: 세계수의 수호자 (시련) |
| 보스 | 최심층: 세계를 만든 존재의 그림자 — Project Abyss 전체 세계관의 핵심 힌트 |
| 심연 페이즈 | 세계 창조 이전의 "무(無)" — 끝없는 심연. 여기서만 등장하는 이노센트가 세계관의 최종 퍼즐 조각 |
| 분위기 | 경외감, 숭고함. "이 아이템 안에 세계의 비밀이 있다" |

## 4. 서사 전달 방식 (Narrative Delivery Methods)

### 4.1. 환경 서사 (Environmental Storytelling) — 최우선

아이템계의 **지형, 배경, 오브젝트**가 이야기를 말한다. 텍스트 없이도 "이곳은 전장이었구나", "여기는 주방이었구나"를 느끼게 한다.

| 수단 | 예시 |
| :--- | :--- |
| 타일 팔레트 | 녹슨 식칼 → 나무/피 타일, 기사의 검 → 석조/갑옷 타일 |
| 배경 오브젝트 | 깨진 가마솥, 꽂힌 화살, 녹슨 갈고리 |
| 색조/분위기 | Magic 등급: 단색 기조. Legendary: 지층마다 색조 변화 |
| 파티클/이펙트 | 전장 → 재 날림, 마법 연구실 → 마력 불꽃 |

#### 4.1.1. 환경 서사 7대 원칙

아이템계와 월드 모두에 적용되는 환경 서사의 핵심 설계 원칙이다. (참조: `Reference/witcher3_인사이트.md`)

**원칙 1: "씬이 플롯을 이긴다"**

Memory Strata의 각 층은 하나의 독립된 **씬(Scene)**이다. 전체 지층을 관통하는 거대한 서사보다, 각 층에서 마주치는 환경적 순간들—지형, 몬스터 배치, NPC 대사 한 줄, 오브젝트 배치—이 플레이어 기억에 남는다. 각 층이 결말 없이도 흥미로운 경험이 되어야 한다.

> *"좋은 플롯이란 좋은 씬을 만들어내는 플롯이다. 이상적인 미스터리란, 결말이 빠져 있어도 읽고 싶은 것이다."* — Raymond Chandler

| 적용 공간 | 설계 방법 |
| :--- | :--- |
| Memory Strata | 각 층 = 독립 씬. 그 층만으로 완결된 인상을 줄 것 |
| 월드 층위 | 각 층위 = 독립 에피소드. 층위 간 연결은 실(thread)에 불과 |

**원칙 2: 정보의 의도적 제거 (Subtraction of Key Information)**

모든 것을 즉시 설명하지 않는다. 초반 지층에서 의미심장한 상징물을 배치하되 그 의미는 깊은 지층에서야 드러나는 구조. "이 아이템에 무슨 일이 있었는가?"라는 질문을 플레이어 스스로 던지게 만든다.

| 지층 깊이 | 정보 공개 수준 |
| :--- | :--- |
| 지층 1 | 미스터리 제시 — 의미 불명의 상징, 맥락 없는 흔적 |
| 중간 지층 | 단편적 해석 — NPC 대사/오브젝트가 실마리 제공 |
| 최심층 | 핵심 해소 — 기원 사건의 진면목 (그러나 100% 답은 주지 않음) |

**원칙 3: 묵시적 서사 (Subtle Exposition)**

세계관의 핵심 가치를 직접 텍스트로 전달하지 않고, 환경 자체가 그것을 체화한다. 전쟁에서 사용된 검의 아이템계라면 — 땅이 갈라져 있고, 잔해가 흩어져 있고, 불에 탄 나무가 서 있고, 이노센트들이 전사의 형상을 하고 있다. 설명 없이 보여주는 것.

| 금지 | 권장 |
| :--- | :--- |
| "이 검은 100년 전쟁에 쓰였다" | 바닥에 갈라진 갑옷 파편, 벽에 박힌 화살, 불탄 깃발 |
| "이 지팡이의 주인은 미쳐버렸다" | 벽면의 미친 낙서, 깨진 거울, 뒤틀린 실험 기구 |

**원칙 4: 40초 법칙 (40-Second Rule)**

플레이어는 40초마다 무언가에 집중할 수 있어야 한다. 텅 빈 복도가 40초 이상 이어지면 집중이 깨진다.

| 이벤트 유형 | 예시 |
| :--- | :--- |
| 시각적 | 배경 변화, 새 타일셋, 의미심장한 오브젝트 |
| 게임플레이 | 적 웨이브, 함정, 숨겨진 방 입구 |
| 서사적 | NPC 한 마디, 환경 변화(지진, 조명 변화), 보스 전조 |

**원칙 5: "기근 속 소시지" 방지 (Environmental Consistency)**

환경 디테일이 서사와 모순되면 안 된다. "배고프다"고 말하는 NPC 뒤에 풍성한 음식이 보이면 안 된다.

| 검증 질문 | 모순 사례 |
| :--- | :--- |
| 이 아이템의 기원 테마와 모든 오브젝트가 일관되는가? | 식칼 아이템계에 마법 수정이 배치됨 |
| 레어리티에 맞는 환경 복잡도인가? | Normal 아이템계에 복잡한 서사 오브젝트 과다 |
| 지층별 감정 곡선이 일관되는가? | 비장한 전쟁 테마 중간에 코믹한 오브젝트 |

**원칙 6: 시선 차단과 서프라이즈 (Visual Blocking & Reveal)**

입구에서 방 전체가 한눈에 보이지 않도록 L자/U자 구조를 기본으로 한다. 시선 차단은 기술적 필요(메모리 스트리밍)와 서사적 효과(극적 공개)를 동시에 달성한다.

| 기법 | 효과 |
| :--- | :--- |
| L자/U자 룸 템플릿 | 코너를 돌 때 새 장면이 드러남 — 발견의 쾌감 |
| 수직 시선 차단 | 아래로 떨어진 후 새 바이옴 공개 |
| 전경 오브젝트 | 큰 기둥/잔해 뒤에 숨겨진 요소 배치 |

**원칙 7: Fire & Ember — 감정적 뱅 모먼트 설계**

시각적 스펙타클(Ember/불씨)만으로는 영혼이 없다. 캐릭터의 감정, 도덕적 갈등(Fire/불)이 선행되어야 Ember가 의미를 갖는다. 보스 방의 화려한 연출보다, 그 보스가 왜 거기 있는지의 서사가 먼저다.

| 요소 | Fire (영혼) | Ember (스펙타클) |
| :--- | :--- | :--- |
| 보스 전 | 보스의 정체/동기가 환경으로 암시됨 | 보스 방 입장 연출, 카메라 줌 |
| 지층 클리어 | 아이템의 기억을 한 단계 더 이해 | 클리어 이펙트, 보상 팡파르 |
| 최심층 | 아이템 기원의 핵심 진실 | 고유 바이옴, 특수 이펙트 |

### 4.2. 몬스터 디자인 (Monster as Story)

아이템계의 몬스터는 랜덤이 아니다. **아이템의 과거에서 파생된 존재**다.

| 원칙 | 설명 |
| :--- | :--- |
| 테마 일관성 | 식칼 아이템계에서 드래곤이 나오면 안 된다. 동물 원혼만 나와야 한다 |
| 지층 진행에 따른 변화 | 초반 지층: 약한 잔영 (돼지). 후반 지층: 강한 잔영 (거대 돼지 원혼). 최심층: 원한의 결집체 |
| 행동 서사 | 동물 몬스터는 도망치다 공격 (겁에 질린 가축). 병사 몬스터는 대열을 짓고 돌진 (전투 훈련) |

### 4.3. NPC/유령 존재 (Ghost Presence)

아이템의 과거와 관련된 존재가 유령으로 등장하여 **실루엣, 행동 패턴, 환경 변화**로 이야기를 전달한다. 유령은 말하지 않는다 — 서 있고, 바라보고, 사라진다. 텍스트가 필요한 경우 환경 레이어(오브젝트 상호작용 텍스트, 환경음)로 처리한다.

| 원칙 | 설명 |
| :--- | :--- |
| 행동으로만 | 유령 자체는 대사 없음. 고개를 끄덕이거나, 무언가를 가리키거나, 플레이어를 바라보고 사라진다 |
| 반복하면 변화 | 같은 유령을 재방문하면 다른 행동 패턴. 3회까지 변화 후 고정 |
| 퍼즐 조각 | 한 지층의 유령 행동만으로는 전체를 알 수 없다. 여러 지층의 잔영을 조합해야 전모 파악 |
| 선택적 | 유령 옆을 지나쳐도 된다. 관찰은 플레이어의 선택 |

행동 묘사 예시 — 기사의 장검 5층:

> *기사의 잔영이 침상 가장자리에 앉아 검을 내려다보고 있다. 플레이어가 가까이 오면 고개를 든다. 손이 검을 향해 뻗었다가 멈춘다. 환경 텍스트: "…이 검을 잡으면 잠이 안 오지. 내가 그랬거든."*

행동 묘사 예시 — 심연의 검 지층 3:

> *전장 폐허 중앙에 두 잔영이 서 있다 — 서로 다른 진영의 군복. 등을 맞대고 있다. 플레이어가 접근하면 두 잔영이 동시에 돌아보았다가, 각자 반대 방향으로 사라진다. 환경 텍스트: "우리도 지키려 했을 뿐이다. 네 쪽도, 우리 쪽도."*

#### 4.3.1. 신호/잡음 이론 (Signal/Noise Theory)

플레이어의 주의력이 분산된 상태(전투 중, 플랫폼 점프 중)에서는 핵심 서사 정보를 전달하지 않는다. 핵심 정보는 플레이어의 시선이 고정된 순간에 전달한다.

| 잡음 수준 | 상황 | 전달 가능 정보 |
| :--- | :--- | :--- |
| 높음 (전투 중) | 액션, 회피, 스킬 사용 | 환경 서사만 (지형, 배경, 파티클) |
| 중간 (탐색 중) | 이동, 아이템 줍기 | 짧은 NPC 한 마디, 오브젝트 상호작용 |
| 낮음 (안전 지대) | 휴식, 층간 대기, 세이브 포인트 구역 | 핵심 서사 정보, 환경 텍스트, 퍼즐 힌트 |

#### 4.3.2. 이노센트 어투 차별화

이노센트는 아이템의 카테고리와 기원 테마에 따라 다른 말투와 성격을 가진다. 이를 통해 아이템의 다층적 역사를 자연스럽게 전달한다.

| 아이템 카테고리 | 이노센트 기본 어투 | 성격 |
| :--- | :--- | :--- |
| 검/대검 | 군사적, 간결한 명령조 | 충성, 의무, 명예 |
| 단검 | 은밀한, 속삭이는 | 암살, 첩보, 비밀 |
| 지팡이 | 학자적, 중얼거리는 | 지식, 탐구, 강박 |
| 활 | 조용한, 관찰자적 | 인내, 정밀, 고독 |

#### 4.3.3. "빈 시간" 설계 (Breathing Room)

플레이어에게 "할 일이 없는 순간"을 의도적으로 제공한다. 아이템계 클리어 후 즉시 보상 화면으로 넘어가지 말고, 잠시 그 공간에 머물 수 있게 한다.

| 적용 지점 | 설계 |
| :--- | :--- |
| 지층 보스 클리어 후 | 보스 방에 잠시 머물 수 있음. 환경이 변화하며 서사 여운 |
| 최심층 클리어 후 | 아이템 기원의 핵심 장소가 평화롭게 변함. 플레이어가 돌아다닐 수 있음 |

### 4.4. 플레이버 텍스트 (Flavor Text)

인벤토리에서 아이템을 볼 때 표시되는 한두 줄의 텍스트. 아이템계에 들어가기 전 호기심을 유발한다.

| 레어리티 | 플레이버 텍스트 스타일 |
| :--- | :--- |
| Normal | 단순 묘사 — *"평범한 식칼. 누군가의 부엌에서 왔다."* |
| Magic | 소유자 암시 — *"칼날에 '충성'이라는 글자가 희미하게 새겨져 있다."* |
| Rare | 사건 암시 — *"지팡이 끝에서 아직도 마력이 불안정하게 흔들린다."* |
| Legendary | 역사 암시 — *"심연 전쟁의 마지막 밤, 이 검을 쥔 자는 돌아오지 않았다."* |
| Ancient | 수수께끼 — *"이것은 가지가 아니다. 세계가 아직 꿈꾸던 시절의 기억이다."* |

## 5. 세계관 톤 (Worldbuilding Tone)

### 5.1. 이중 톤: 고딕 비극 + 경쾌한 야리코미

Project Abyss의 세계관은 두 가지 톤이 공존한다:

| 톤 | 적용 공간 | 표현 |
| :--- | :--- | :--- |
| 고딕 비극 | 월드, Legendary/Ancient 아이템계 | 어둡고 비장한 서사, 비극적 과거, 세계의 비밀 |
| 경쾌한 야리코미 | Normal/Magic 아이템계 | 유머러스한 유령 행동, 가벼운 환경 텍스트, 즐거운 파밍 분위기 |

**전환 규칙:** 레어리티가 올라갈수록 톤이 무거워진다. Normal 식칼의 돼지 원혼은 코믹하고, Ancient 세계수 가지의 태초 정령은 숭고하다. 플레이어는 장비를 키울수록 더 무거운 이야기를 만나게 된다.

### 5.2. "심연(Abyss)"의 정체

게임 전체를 관통하는 미스터리:

> "심연이란 무엇인가?"

- 월드의 가장 깊은 곳에 심연이 있다
- 아이템의 가장 깊은 지층에도 심연이 있다
- 모든 Ancient 아이템의 최심층은 같은 장소를 가리킨다
- **아이템의 기억을 깊이 파고들면, 결국 세계의 기원에 도달한다**

이것이 탐험(월드 최심부)과 야리코미(아이템계 최심층 지층)가 서사적으로 같은 목적지를 향하게 만드는 핵심 장치다.

### 5.3. NPC 대사 철학

| 원칙 | 설명 |
| :--- | :--- |
| 짧고 인상적 | 3문장 이하. 한 마디가 한 편의 이야기보다 강하다 |
| 정보보다 감정 | "전쟁은 100년 전에 있었다" 대신 "아직도 비명이 들린다" |
| 신뢰할 수 없는 화자 | 유령은 자기 기억만 안다. 편향적이고, 때로는 거짓말한다 |
| 행동으로 보여주기 | 유령이 특정 벽을 가리키면 숨겨진 방이 있다. 대사가 게임플레이와 연결 |

## 6. 욕망의 하강 (Descent of Desire) — 지층 깊이 = 욕망의 깊이

> **리서치 기반:** (미작성) <!-- Libido_NarrativeTheme_Research.md, Research_Religious_Symbolism_TabooDesign.md, ItemWorld_DesireDepth_ThemeMapping_Research.md — 추후 제작 예정 -->

### 6.1. 핵심 전제

> "아이템의 기억을 깊이 파고들수록, 표면의 이야기 아래 숨겨진 **원초적 욕망**과 마주한다."

아이템계의 지층은 단순한 난이도 계층이 아니다. 각 지층은 인간 정신의 깊이에 대응하며, 깊이 내려갈수록 더 원초적이고 금기적인 테마가 상징적으로 드러난다. Binding of Isaac이 Basement(일상 공포)에서 Dark Room(존재론적 진실)까지 하강하듯, Project Abyss도 아이템계 하강이 곧 욕망의 심층으로 들어가는 여정이다.

### 6.2. 지층별 욕망의 하강 모델

| 지층 | 이름 | 심리적 층위 | 종교적 프레임 | 미학적 전환 |
| :--- | :--- | :--- | :--- | :--- |
| **1** | 기억의 표면 / "토템의 방" | 의식 — 아이템이 기억하고 싶은 것 | 성당의 회랑 (외부의 성스러움) | 고딕 아름다움, 질서정연, 대칭적 |
| **2** | 기억의 이면 / "고해의 방" | 전의식 — 아이템이 감추는 것 | 고해소, 참회의 공간 | 대칭이 균열, 부패 시작, 비밀 통로 증가 |
| **3** | 기억의 심층 / "자궁의 방" | 무의식 — 아이템이 기억하지 못하는 것 | 카타콤, 비밀 의식 공간 | **바디 호러 전환**. 벽이 살아있고, 환경이 유기체적 |
| **4** | 기억의 핵 / "심연의 방" | 원형적 무의식 — 아이템의 본질 | 지성소 (Holy of Holies) | 역설적 숭고. 과잉에서 **공백**으로. 미니멀, 광활, 경외 |
| **심연** | Ancient 전용 | 초개인적 — 세계의 기억 | 영지주의 데미우르고스 영역 | 비유클리드, 불가능한 아름다움 |

### 6.3. 14개 욕망 테마 그룹 (Desire Theme Groups)

| G-10 | 「변신의 탑」 | 다른 무언가가 되고 싶다 | 변신/정체성 | 크기 왜곡 방, 유체 벽, 거울 속 괴물 |
| G-11 | 「이계의 숲」 | 완전한 타자에 대한 매혹 | 변신/정체성 | 호흡하는 숲, 심해 동굴, 균류 구역 |
| G-12 | 「금단의 성역」 | 신성한 것을 더럽히는 역설 | 금기/위반 | 뒤집어진 성당, 피 묻은 제단, 카타콤 |

**Core (최심층/심연) — 실존적:**

| 그룹 | 이름 | 핵심 | 축 | 상징 환경 |
| :--- | :--- | :--- | :--- | :--- |
| G-13 | 「영원의 요람」 | 시간에 대한 공포, 퇴행 | 변신/정체성 | 시계 멈춘 보육원, 거대 요람, 몽환 구름 |
| G-14 | 「경계의 문」 | 죽음 — 모든 욕망의 궁극적 근원 | 금기/위반 | 반쪽 생동/반쪽 소멸, 끝없는 심연 |

### 6.4. 지층별 욕망 층위의 진행

**지층 1~2 (Surface) — 7대 죄악의 표면:**
- 아이템 테마에 충실한 "정형화된 악". 탐욕=금화, 교만=거울, 분노=불꽃
- 적은 "수호자" — 명확한 형태. 종교 오브젝트: 촛불, 제단, 성수 → 치유/버프
- 적용 테마: G-01 쇠사슬(속박), G-02 원소(숭고), G-03 파괴(쾌감)
- **감정 톤:** 고딕 아름다움. 익숙한 던전 크롤러의 문법

**지층 3~5 (Mid) — 감추고 싶은 것:**
- 표면 아래 숨겨진 취약함이 드러남. 죄의 이면 = 공포
- 적은 "그림자" — 아이템 자신의 왜곡된 반영. 환경이 부분적 부패/변형
- 적용 테마: G-04 지배(왕좌), G-05 시선(거울), G-06 물신(인형), G-07 해체(신체)
- 종교 오브젝트: 금이 간 성상, 피 묻은 경전 → 리스크-리워드
- **감정 톤:** "불편하지만 부정할 수 없는"

**지층 6~8 (Deep) — 기억하지 못하는 것:**
- 7대 죄악을 넘어선 원초적 충동. 주체/객체 경계 와해
- 적은 "원초적 충동" — 형태 불분명, 적과 환경의 경계 모호
- 적용 테마: G-08 탐식(흡수), G-09 부패(오염), G-10 변신(해체), G-11 이계(타자), G-12 모독(성역)
- Kristeva의 비체(Abjection): 벽이 호흡하고, 바닥이 체액
- 종교 오브젝트: 거꾸로 된 십자가, 제의 가면 → 강력하지만 변형 유발
- **감정 톤:** 바디 호러. 문명이 가장 강하게 억압하는 영역

**지층 핵 (Core) — 왜 원하는가:**
- "무엇을 원하는가"가 아니라 **"왜 원하는가"**
- 적용 테마: G-13 시간(퇴행), G-14 죽음(비존재)
- 환경: **과잉에서 공백으로의 전환**. 미니멀, 광활, 경외
- 종교 오브젝트: 빈 제단, 부재하는 신상, 빛만 있는 공간
- **감정 톤:** 역설적 숭고. 공포를 넘어선 경외

**심연 (Ancient 전용) — 욕망을 넘어선 것:**
- 개별 아이템을 넘어선 "세계의 기억". 모든 Ancient 최심층이 같은 장소를 가리킴 (§5.2 연결)
- 적은 "거짓 창조자" — 데미우르고스적 존재
- 환경: 비유클리드 기하학. 공간의 규칙이 붕괴
- **감정 톤:** 불가능한 아름다움. "신이 부재한 곳에서 남는 것은 무엇인가"

### 6.5. 아이템 유형별 테마 분기

아이템의 유형에 따라 지층에서 활성화되는 테마 그룹이 달라진다:

| 아이템 유형 | Surface | Mid | Deep | Core |
| :--- | :--- | :--- | :--- | :--- |
| 무기 (검/창) | G-03 화염의 무도회 | G-04 피의 왕좌 | G-08 심연의 식탁 | G-14 경계의 문 |
| 방어구 (갑옷) | G-01 쇠사슬의 정원 | G-07 만물의 눈 | G-10 변신의 탑 | G-13 영원의 요람 |
| 장신구 (반지) | G-02 불꽃과 얼음 | G-06 인형사의 작업실 | G-12 금단의 성역 | G-14 경계의 문 |
| 마법서/촉매 | G-05 거울의 회랑 | G-11 이계의 숲 | G-09 부패의 정원 | G-13 영원의 요람 |

> **규칙:** 아이템의 기원 테마(§7 테마 풀)가 1차 결정, 아이템 유형이 2차 결정. 두 레이어가 합쳐져 지층의 최종 환경을 구성한다.

| 테마 | 지층 1 (표면) | 지층 2 (이면) | 지층 3 (심층) | 지층 4 (핵) |
| :--- | :--- | :--- | :--- | :--- |
| T-HOME (가정) | 따뜻한 부엌, 벽난로 | 닫힌 문, 소리 없는 방 | 벽 속의 시선, 호흡하는 벽지 | 빈 요람, 부재하는 인물 |

### 6.8. 설계 원칙

| 원칙 | 설명 |
| :--- | :--- |
| **"금기는 설명하지 않는다"** | Isaac의 Belt가 이동속도를 올리는 이유를 게임이 설명하지 않는 것처럼, 종교 상징의 "왜"는 플레이어의 추론에 맡긴다 |
| **"하강은 장르의 변화다"** | 지층 1(고딕 판타지) → 2(심리적 긴장) → 3(바디 호러) → 4(숭고/추상). Bloodborne의 고딕→코스믹 전환 참조 |
| **"모든 적은 자기 자신이다"** | 깊은 지층의 적은 점점 더 아이템 자신(그리고 플레이어)을 닮아간다 |
| **"위반이 성스러움이다"** | 더 깊은 지층을 돌파하는 것 = 금기의 위반이자 아이템의 진정한 기억에 도달하는 성스러운 행위. Bataille의 "금기가 없으면 위반도 없다" |
| **"변형이 해방이다"** | 깊은 지층에서 획득하는 아이템/버프가 캐릭터를 변형시키지만, 그 변형이 곧 강화 |
| **"부재가 가장 큰 공포다"** | 최심층은 과잉이 아니라 공백. 바디 호러의 과잉 이후 오는 텅 빈 공간이 가장 강렬하다 |
| **"죄의 근저에는 공포가 있다"** | 표면 지층은 "죄(행동)"를 다루고, 심층은 그 죄를 만들어낸 "공포(근원)"를 다룬다 |

### 6.10. 핵심 테마 문장 (Thematic Pillars)

1. **"심연은 배고프다"** — 세계 자체가 맹목적 의지/욕망의 표현
2. **"가장 깊은 곳에서 마주할 것은 그대 자신"** — 외부 탐험 = 내면 탐험
3. **"모든 무기에는 주인의 집착이 깃든다"** — 아이템 = 응축된 욕망
4. **"고통 없이 얻은 힘은 없다"** — 성장의 조건 = 고통
5. **"당신은 떠날 수 있다. 하지만 돌아올 것이다"** — 야리코미의 반복 강박
6. **"속박된 자만이 진정한 해방을 안다"** — 제한이 있어야 초월이 있다
7. **"욕망은 채워지지 않는다. 다만 변할 뿐"** — 오브제 쁘띠 아의 영원한 미끄러짐

## 7. 테마 풀 시스템 (Theme Pool System)

> **참고:** §6의 "욕망의 하강" 모델은 이 테마 풀의 지층별 변형 레이어로 기능한다.

수천 개 아이템에 개별 서사를 쓰는 것은 불가능하다. 대신 **테마 풀**을 조합하여 서사를 절차적으로 생성한다.

### 7.1. 테마 풀 구조

```
아이템 서사 = 무기 카테고리 × 기원 테마 × 레어리티 깊이
```

| 무기 카테고리 | 기본 기원 테마 |
| :--- | :--- |
| 검 (Sword) | 기사/전사/전쟁/결투/명예 |
| 대검 (Greatsword) | 영웅/처형/맹세/전장/유산 |
| 단검 (Dagger) | 암살/첩보/비밀/생존/의식 |
| 활 (Bow) | 사냥/정찰/암살/스포츠/전쟁 |
| 지팡이 (Staff) | 마법/연구/치유/저주/예언 |

### 7.2. 테마 → 몬스터 풀 매핑 예시

| 기원 테마 | 몬스터 풀 | 지형 팔레트 | 분위기 |
| :--- | :--- | :--- | :--- |
| 주방/도축 | 동물 원혼 (돼지, 소, 닭, 물고기) | 나무 바닥, 피 얼룩, 갈고리 | 으스스한 일상 |
| 전쟁/전장 | 병사 잔영, 화살 유령, 공성 병기 파편 | 석조, 깃발, 무너진 성벽 | 비장함 |
| 마법/연구 | 실험 실패작, 원소 정령, 변이 생물 | 수정, 마법진, 책더미, 가마솥 | 신비로움 |
| 사냥/숲 | 숲 동물 원혼, 덫 잔영, 사냥꾼 유령 | 나무, 이끼, 동굴, 덫 오브젝트 | 야생적 |
| 수행/격투 | 수련생 잔영, 무술 사범 유령 | 도장, 돌바닥, 산사, 폭포 | 고요한 긴장 |
| 저주/어둠 | 그림자, 원한의 형체, 저주받은 유물 | 검은 석재, 부서진 신상, 핏빛 달 | 공포 |

### 7.3. 절차적 서사 조립

아이템이 드랍될 때, 다음이 결정된다:

```
1. 무기 카테고리 → 기원 테마 풀에서 1개 선택 (시드 기반)
2. 레어리티 → 서사 깊이 결정 (NPC 수, 대사 수, 보스 서사 유무)
3. 아이템 레벨 → 몬스터 강도 스케일링
4. 테마 → 몬스터 풀, 지형 팔레트, 분위기 결정
5. 시드 → NPC 대사를 대사 풀에서 선택, 배치 지층 결정
```

이 구조 덕분에 같은 "검" 카테고리에서도:
- 어떤 검은 전쟁의 기억이 가득하고
- 어떤 검은 결투의 역사를 품고 있고
- 어떤 검은 처형의 어둠이 서려 있다

## 8. 월드 서사 (World Narrative)

### 7.1. 층위별 독립 서사

월드의 각 층위(바이옴)는 독립된 서사를 가진다. 그 층위에서 드랍되는 아이템은 해당 층위의 서사를 반영한다.

| 원칙 | 설명 |
| :--- | :--- |
| 층위 = 장소의 역사 | 성채 층위에서 드랍되는 검은 "기사의 검" 테마가 높은 확률 |
| 교차 서사 | A 층위의 아이템계에서 B 층위에 대한 힌트 발견 → 탐험 동기 |
| 환경 서사 우선 | 벽에 긁힌 자국, 부서진 동상, 마른 분수 — 텍스트 없이 역사를 전달 |

### 7.2. 수직적 서사 (메트로베니아 + 야리코미 통합)

```
월드 탐험 (수평 서사) ──── 층위의 현재 상태를 목격
        │
        ▼
아이템 획득 ──────────── 층위의 과거 조각을 손에 쥠
        │
        ▼
아이템계 진입 (수직 서사) ── 과거 속으로 다이빙
        │
        ▼
깊은 지층 도달 ───────── 층위의 비밀 발견
        │
        ▼
월드 복귀 ──────────── "아, 그래서 이 층위가 이렇게 된 거구나"
```

플레이어는 월드를 탐험하면서 **현재**를 보고, 아이템계에서 **과거**를 체험하고, 다시 월드로 돌아와서 **이해**한다. 탐험과 야리코미가 서사적으로 하나의 축을 이룬다.

## 9. 서사 설계 금지 규칙 (Narrative Never-Do Rules)

| 규칙 | 이유 |
| :--- | :--- |
| 강제 컷신 금지 | 액션 RPG의 흐름을 끊는다. 모든 서사는 플레이 중 전달 |
| 긴 대사창 금지 | 3문장 이하. 읽지 않아도 게임 진행에 지장 없어야 한다 |
| 서사 필수 진행 금지 | NPC를 만나지 않아도, 대사를 읽지 않아도 아이템계를 클리어할 수 있다 |
| 레어리티 = 서사 강제 금지 | Normal 아이템 플레이어에게 "서사를 놓치고 있다"는 압박을 주지 않는다. 서사는 보너스 |
| 설명 위주 서사 금지 | "100년 전 전쟁이 있었다"는 안 된다. "아직도 비명이 들린다"가 맞다 |
| 유일한 진실 금지 | 유령마다 기억이 다르다. 플레이어가 해석한다. 공식 정답은 없다 |

## 10. 검증 기준 (Verification Checklist)

- [ ] 아이템계에 들어갔을 때, 몬스터와 지형만으로 "이 아이템이 어떤 역사를 가졌는지" 짐작할 수 있는가?
- [ ] Normal 아이템과 Legendary 아이템의 서사 깊이 차이가 체감되는가?
- [ ] NPC 대사를 읽지 않고 스킵해도 게임 진행에 전혀 지장이 없는가?
- [ ] NPC 대사를 읽으면 "더 알고 싶다"는 호기심이 생기는가?
- [ ] 같은 무기 카테고리의 두 아이템이 서로 다른 테마를 가져 차별화되는가?
- [ ] 깊은 지층에서 등장하는 서사가 얕은 지층의 서사보다 무게감이 있는가?
- [ ] 월드 층위의 환경 서사와 해당 층위 아이템계의 서사가 일관되게 연결되는가?
- [ ] "심연(Abyss)"에 대한 힌트가 Ancient 아이템계에서만 등장하여 야리코미 동기를 강화하는가?
- [ ] 강제 컷신이 단 하나도 없는가?
