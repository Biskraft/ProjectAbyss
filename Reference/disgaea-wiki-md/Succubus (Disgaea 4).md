# Succubus (Disgaea 4)

|  |

*Main article: Succubus*

The **Succubus** is a Monster class in *Disgaea 4: A Promise Unforgotten*.

## Overview
The Succubus is a versatile class with magical damage, healing and ailments. Her charm weakens nearby male units.
