# Sorcerer (Disgaea 4)

|  |

*Main article: Sorcerer*

The **Sorcerer** is a Humanoid class in *Disgaea 4: A Promise Unforgotten*.

## Overview
The Sorcerer is a support class dedicated to weakening the enemy. They learn all debuff and ailment spells, as well as various Evilities that weaken all enemies just with their presence.

## Personality types

- Generic
- Hyper
- Wicked
