# Thief (Disgaea 4)

|  |

*Main article: Thief*

The **Thief** is a Humanoid class in *Disgaea 4: A Promise Unforgotten*.

## Overview
The Thief is a support class specialized in stealing, helping the party by disarming the enemy and getting rare loot. They have low stats overall except for movement and evasion, so they prefer to fight from a long distance if necessary.

## Personality types

- Cute
- Hasty
- Mischievous
