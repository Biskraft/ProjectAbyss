# Spirit (Disgaea 4)

|  |

*Main article: Spirit*

The **Spirit** is a Monster class in *Disgaea 4: A Promise Unforgotten*.

## Overview
The Spirit is a magical attacker specialized in Fire magic. Their SP regeneration can keep them using skills for the whole battle.
