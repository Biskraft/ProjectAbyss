# Medic

|  |


The **Medic** is a Humanoid class in *Disgaea 4: A Promise Unforgotten*, and is one of the two DLC classes drawn by guest artists.

## Overview
The Medic is a healer/support class with a very complete moveset, being the only class to learn all buff spells on top of all healing spells. Also notable for the unique ability to recharge an ally's turn.

## Personality types

- Passionate Worker
- Quiet
- Magical
