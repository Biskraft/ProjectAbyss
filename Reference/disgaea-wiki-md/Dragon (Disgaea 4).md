# Dragon (Disgaea 4)

|  |

*Main article: Dragon*

The **Dragon** is a Monster class in *Disgaea 4: A Promise Unforgotten*.

## Overview
The Dragon class is an offensive class with high attack and vitality. They specialize in powerful Fire attacks.
