# Ninja (Disgaea 4)

|  |

*Main article: Ninja*

The **Ninja** is a Humanoid class in *Disgaea 4: A Promise Unforgotten*.

## Overview
The Ninja is a nimble class specialized in evasion, using it to cross the battlefield undetected and hitting without being hit back.

## Personality types

- Dark
- Cold
- Incomprehensible
