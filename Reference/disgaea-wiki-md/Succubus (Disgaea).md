# Succubus (Disgaea)

|  |

*Main article: Succubus*
The **Succubus** is a Monster class in *Disgaea: Hour of Darkness*.

A versatile unit with physical, magical and healing skills.

This class is immune to ailments from Tier 5 onwards.

## Quotes
