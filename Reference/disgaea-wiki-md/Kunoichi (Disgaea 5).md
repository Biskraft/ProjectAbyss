# Kunoichi (Disgaea 5)

|  |

*Main article: Kunoichi*

The **Kunoichi** is a Humanoid class in *Disgaea 5: Alliance of Vengeance*.

## Overview
The Kunoichi is an assassin specialized in evasion and ailments, using these traits to cancel enemy attacks before responding with her own.

## Personality types

- Loyal
- Great Veteran
- Ditzy Newbie
