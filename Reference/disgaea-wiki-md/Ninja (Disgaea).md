# Ninja (Disgaea)

|  |

*Main article: Ninja*
The **Ninja** is a Humanoid class in *Disgaea: Hour of Darkness*

## Overview
The Ninja is a very nimble unit with good evasion and movement but low defenses. He's an advanced version of the Martial Artist classes, with higher proficiency in Fists(also adding good proficiency in Axes), improved evasion and more Counters.

Passive skill: This class can dodge skills. (Warning: A bug in this effect makes them able to dodge heals and buffs received from ally units)
