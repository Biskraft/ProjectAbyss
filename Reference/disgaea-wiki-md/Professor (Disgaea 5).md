# Professor (Disgaea 5)

|  |

*Main article: Professor* 

The **Professor** is a Humanoid class from *Disgaea 5: Alliance of Vengeance*.

## Overview
The Professor is a support class specialized in strengthening allies, learning various buff skills as well as buff-enhancing Evilities. Not limited to this role, they're also somewhat proficient in Guns and magic.

## Personality types

- Eccentric
- Smart
- Forgetful
