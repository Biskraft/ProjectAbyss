# Lady Samurai (Disgaea 4)

|  |

*Main article: Lady Samurai*

The **Lady Samurai** is a Humanoid class in *Disgaea 4: A Promise Unforgotten*.

## Overview
The Lady Samurai is a very strong offensive class, simply specialized in powerful and precise attacks. She's an honest fighter who prefers one-on-one duels.

## Personality types

- Brave
- Strong-willed
- Fancy
