# Mythical Robot

|  |


  - Mythical Robot**(and later **Super Overlord**) is Pringer X's class in *Disgaea 4: A Promise Unforgotten*.

It's fought in Extra Stage 8, "Historical Spectacle!".

Later, a stronger version called Pringer X: Roar can be fought in Extra Stage 10, "Vengeance of Pringer X".

## Description

: *"A powerful robot created for unknown reasons. It's a masterpiece...apparently."* - Pringer X
: *"A powerful robot created for unknown reasons. It's a masterpiece... apparently. Your adventure... must continue!"* - Pringer X: Roar
