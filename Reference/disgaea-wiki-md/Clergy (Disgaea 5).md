# Clergy (Disgaea 5)

|  |

*Main article: Male Healer*

The **Clergy** is a Humanoid class in *Disgaea 5: Alliance of Vengeance*.

## Overview
The Clergy is a class specialized in healing, so their only notable stats are RES and SP. They learn all the healing spells, as well as some buffs and debuffs, in order to keep the party as safe as possible. The Clergy's healing tends to have useful aditional effects.

## Personality types
- Naughty
- Manly
- Optimistic
