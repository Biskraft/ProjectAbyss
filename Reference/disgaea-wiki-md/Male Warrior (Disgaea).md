# Male Warrior (Disgaea)

|  |

*Main article: Warrior*
The **Male Warrior** is a Humanoid class in *Disgaea: Hour of Darkness*.

## Overview
The Male Warrior is a standard frontliner unit, having good physical stats but poor magical ones. He's proficient in using both Swords and Axes.


## Quotes
