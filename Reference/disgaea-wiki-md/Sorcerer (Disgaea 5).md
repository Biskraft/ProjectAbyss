# Sorcerer (Disgaea 5)

|  |

*Main article: Sorcerer*

The **Sorcerer** is a DLC Humanoid class in *Disgaea 5: Alliance of Vengeance*.
## Overview
The Sorcerer is a support class specialized in incapacitating the enemy. They learn all the Debuff and Ailment spells, as well as many Evilities that weaken the enemy simply with their presence. 

## Personality types

- Novice
- Yandere
- In a Trance
