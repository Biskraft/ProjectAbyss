# Healer (Disgaea 4)

|  |

*Main article: Female Healer*

The **Healer** is a Humanoid class in *Disgaea 4: A Promise Unforgotten*.

## Overview
The Healer, as suggested by their name, is a support class specialized in healing. They learn all of the healing spells as well as some buffs and debuffs in order to keep the party healthy and safe.

## Personality types

- Loyal
- Seductive
- Ditzy
