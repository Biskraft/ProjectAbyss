# Cleric (Disgaea 5)

|  |

*Main article: Female Healer*

The **Cleric** is a Humanoid class in *Disgaea 5: Alliance of Vengeance*.

## Overview
The Cleric is a class specialized in healing, so their only notable stats are RES and SP. They learn all the healing spells, as well as some buffs and debuffs, in order to keep the party as safe as possible. The Cleric's healing is stronger than anyone else's.

## Personality types

- Saintly
