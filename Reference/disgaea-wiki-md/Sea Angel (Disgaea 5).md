# Sea Angel (Disgaea 5)

|  |

*Main article: Sea Angel*
The **Sea Angel** is a Monster class in *Disgaea 5: Alliance of Vengeance*.

## Overview
The Sea Angel is a support class with an affinity for Ice-based attacks. They strengthen allies solely with their presence.
