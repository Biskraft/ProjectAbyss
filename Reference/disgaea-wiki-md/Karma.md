# Karma

- Karma** is a stat introduced in *Disgaea 6: Defiance of Destiny*. It is used to pay for upgrades when using Super Reincarnate. Completing certain D-Merit objectives and increasing in level will build up Karma which can be used to strengthen a character's stats in the way bonus points are used in prior games, as well as enhance abilities that cannot normally be trained such as counter-attack count and movement.

## Upgrades
Karma can be used to purchase upgrades for a character when using Super Reincarnate.

## Karma limit
There is a limit on how much Karma can be accumulated for a character. It starts at 9,999 and can be increased by completing certain D-Merit objectives. Attempting to redeem D-Merit objectives which would give the character more Karma than they can store will result in a pop up warning describing that excess Karma will be permanently unobtainable.
