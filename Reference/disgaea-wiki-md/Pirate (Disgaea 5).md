# Pirate (Disgaea 5)

|  |

The **Pirate** is a Humanoid class in *Disgaea 5: Alliance of Vengeance*.

## Overview
The Pirate is a class specialized in long-range combat with Guns. As skillful explorers, they facilitate the traversal and use of the Item World.

## Personality types

- Dashing
- Piratey
- Flirty
