# List of All Starting Units in Disgaea Games

This is a list of all units that the player begins the game with in each Disgaea game. The starting units consist of both storyline characters and custom characters, the latter of which start with random names and can be renamed when the Dark Assembly is unlocked. 

## *Disgaea: Hour of Darkness*
Starting units in the original *Disgaea*, and its various ports.


Initially, the player only has Laharl and Etna, but once the first battle of the game begins, the three Pvt. Prinnies will be added to the player's party.

## *Disgaea 2: Cursed Memories*
Starting units in *Disgaea 2*, and its PSP port *Disgaea 2: Dark Hero Days*.


## *Disgaea 3: Absence of Justice*
Starting units in *Disgaea 3*.


## *Disgaea 4: A Promise Unforgotten*
Starting units in *Disgaea 4*.


Note: The Catsaber starts with the name "Sardine Thief" instead of a randomly generated name, but is otherwise a normal generic unit in every way.
