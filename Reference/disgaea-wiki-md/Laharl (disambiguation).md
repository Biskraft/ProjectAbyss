# Laharl (disambiguation)

Laharl is a male character in the *Disgaea* series

Due to appearing in multiple *Disgaea* games as playable characters, he has multiple classes in the series.

- For his class in *Disgaea*, see Demon Prince.
- For his class in *Disgaea 2*, see Overlord (Laharl) (Disgaea 2).
- For his class in *Disgaea 3*, see Overlord (Disgaea 3).
- For his class in *Disgaea 4*, see Overlord (Laharl) (Disgaea 4).
- For his class in *Disgaea D2*, see Overlord (Disgaea D2).
- For his class in *Disgaea 5*, see Overlord (Disgaea 5).
