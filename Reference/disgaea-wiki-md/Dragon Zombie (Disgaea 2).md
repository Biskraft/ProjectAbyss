# Dragon Zombie (Disgaea 2)

|  |

*Main article: Dragon Zombie*

The **Dragon Zombie** is a monster class in *Disgaea 2: Cursed Memories*.

## Overview
The Dragon Zombie is a tank class with good power and incredible physical defense, balanced by terrible magical defense and low mobility.
## Skills
