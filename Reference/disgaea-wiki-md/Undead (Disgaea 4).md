# Undead (Disgaea 4)

|  |

*Main article: Undead*

The **Undead** is a Monster class in *Disgaea 4: A Promise Unforgotten*.

## Overview
The Undead is class with relatively weak stats, which they compensate with their power-boosting Evilities and self-regeneration.

## Personality types

- Zombie-voiced
- Easygoing
- Motivated
