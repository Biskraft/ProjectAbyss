# Flonne (disambiguation)

Flonne is a female character in the *Disgaea* series.

Due to appearing in multiple *Disgaea* games as a playable character, she has multiple classes in the series.

- For her player class in *Disgaea*, see Angel (Flonne)
- For her enemy-exclusive class in *Disgaea*, see S. Detective
- For her class in *Disgaea 2*, see Fallen Angel (Disgaea 2)
- For her class in *Disgaea 3*, see Fallen Angel (Disgaea 3)
- For her class in *Disgaea 4*, see Archangel (Disgaea 4)
- For her Fallen Angel class in *Disgaea 4*, see Fallen Angel (Disgaea 4)
- For her class in *Disgaea D2*, see Fallen Angel (Disgaea D2)
- For Pure Flonne's class in *Disgaea D2*, see Awakened Angel
- For her class in *Disgaea 5*, see Fallen Angel (Disgaea 5)
