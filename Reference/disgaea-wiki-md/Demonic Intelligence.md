# Demonic Intelligence

- Demonic Intelligence**, or **D.I.**, is a system in *Disgaea 6: Defiance of Destiny* that allows players to configure the actions characters use in Auto Battle. The system can be unlocked by passing a trial in the Dark Assembly, and further trials can be used to unlock more commands for the system.

## Building a template
Players can manually build a template for D.I. in the characters menu.

If an error occurs in a D.I. template, Auto-Battle will be interrupted.
