# Felynn (Disgaea 2)

|  |

*Main article: Felynn*

The **Felynn** is a monster class in *Disgaea 2: Cursed Memories*.

## Overview
The Felynn is an offensive class with good attack and evasion but low defenses. They're a close-combat specialist who's great at counterattacks.


## Skills
