# Adell (disambiguation)

Adell is a male character in the *Disgaea* series.

Due to appearing in multiple *Disgaea* games as a playable character, he has multiple classes in the series.

- For his class in *Disgaea*, see Demon Buster
- For his class in *Disgaea 2*, see Demon Hunter (Disgaea 2)
- For his class in *Disgaea 3*, see Ex-Demon Hunter
- For his class in *Disgaea 4*, see Demon Hunter (Disgaea 4)
- For his class in *Disgaea D2*, see Demon Hunter (Disgaea D2)
- For his class in *Disgaea 5*, see Demon Hunter (Disgaea 5)
