# Lady Samurai (Disgaea 2)

|  |


*Main article: Lady Samurai*
The **Lady Samurai** is a humanoid class in *Disgaea 2: Dark Hero Days*.
## Overview
The Lady Samurai is an offensive class specialized in using Swords. She's glass cannon with terrible defenses but excellent attack power.

## Skills
### Passive
- Increase damage dealt by 30%(+1% per Lover) when fighting one-on-one.
