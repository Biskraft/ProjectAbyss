# Chimera

|  |

The **Chimera** is a Monster class in *Disgaea 5: Alliance of Vengeance*.

## Overview
The Chimera is an support class with solid defenses. They specialize in weakening enemies through ailment and debuff skills, as well as global debuffs from their Evilities.

## Personality types
- Ancient
- Taciturn
