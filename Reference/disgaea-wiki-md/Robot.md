# Robot

|  |

  - Robot** is a monster class in *Disgaea 2: Dark Hero Days*. Despite looking very humanoid, they are monster units similar to Thursday in *Disgaea: Hour of Darkness*. As such, they cannot equip humanoid weapons and can turn into a weapon using Magichange.

## Overview
The Robot is an offensive unit with balanced stats except for their high attack. While they don't start too powerful, the later Tiers possess impressive stats all around.

## Skills
### Passive
- Normal attacks happen twice if you execute them without moving.
- Lovers increase stats by 1.5% instead of the usual 1%.
