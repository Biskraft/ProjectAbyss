# Gunner (Disgaea 5)

|  |

*Main article: Gunner*
The **Gunner** is a Humanoid class in *Disgaea 5: Alliance of Vengeance*.

## Overview
The Gunner is a class with high accuracy and solid evasion but poor defenses. He's a Gun-user with excellent attack range who functions better when fighting as part of a group.
