# True Final Boss

- True Final Boss** is the exclusive enemy class of Judge Nemo, the final boss of *Disgaea 4: A Promise Unforgotten.*

He's fought in Stage 10-7, "Defying God".

## Description
:*"The collected hatred towards humans controled by Fear the Great. It creates malice, which it uses to gain more power."*
