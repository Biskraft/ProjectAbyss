# Prinny (Disgaea)

|  |

*Main article: Prinny (Class)*
The **Prinny** is a Monster class in *Disgaea: Hour of Darkness*.

This class is immune to ailments from Tier 4 onwards. This is the only instance of a class in the franchise that has 5 Tiers instead of 6.

Passive skill: This unit explodes when thrown or hit by an explosion. Hits radius-2 circle, dealing damage equal to 50% of this unit's remaining HP.
