# Kunoichi (Disgaea 4)

|  |

*Main article: Kunoichi*

The **Kunoichi** is a Humanoid class in *Disgaea 4: A Promise Unforgotten*.

## Overview
The Kunoichi is a versatile assassin, with a combination of damage, evasion and ailment infliction.
