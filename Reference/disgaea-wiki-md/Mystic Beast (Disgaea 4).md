# Mystic Beast (Disgaea 4)

|  |

*Main article: Mystic Beast*

The **Mystic Beast** is a Monster class in *Disgaea 4: A Promise Unforgotten*.

## Overview
The Mystic Beast is a fragile but nimble class specialized in Wind attacks. Their power grows as they run through the battlefield.
