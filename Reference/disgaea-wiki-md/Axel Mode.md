# Axel Mode

- Axel Mode** is an alternate set of scenarios in *Disgaea 2: Dark Hero Days* and *Disgaea 2 PC*.

Normally, Axel Mode is unlocked only after completing the game and starting a New Cycle. When starting a New Cycle, a choice will be given to start Adell Mode (the normal story for *Disgaea 2*) or Axel Mode. Whichever is chosen, the entire story must be played through and another New Cycle started in order to change from Adell Mode to Axel Mode or back again. Since it's an entirely different story, story characters such as Rozalin, Tink, and Yukimaru won't be playable in Axel Mode. Instead, Axel Mode has its own story characters that can be added to the party.

Axel's story is shorter, but more difficult than Adell's story. Magichange and Toss & Pass (which is the same as Throw and Receive from *Disgaea 3: Absence of Justice*) are enabled in Axel Mode, and these new game mechanics will carry over to Adell Mode after Axel Mode is completed.

In *Disgaea 2 PC*, there is a code that can be used on the title screen to unlock Axel Mode without starting a New Cycle. This reduces the difficulty of Axel Mode, but when starting a New Cycle and going to Adell Mode, the difficulty is unchanged. The code to start Axel Mode is:
- Highlight / hover over the "start" option on the main menu

- On an Xbox controller hit Y, X, B, Y, X, B, A

- On a Playstation controller hit Triangle, Square, O, Triangle, Square, O, X

- On a Keyboard, enter I,J,L,I,J,L,K

- You should hear Axel's voice prompt as a new game starts.
