# Winged Warrior (Disgaea D2)

|  |

*Main article: Winged Warrior*
The **Winged Warrior** is a Monster class in *Disgaea D2: A Brighter Darkness*.

In the Dark Assembly when creating or reincarnating a monster character the class is refered to as the "Mothman".

## Quotes
> An evil insect that likes to trick humans. Its dust can cause a lot of trouble.|Job Description, *Disgaea D2: A Brighter Darkness*
