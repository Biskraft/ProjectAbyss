# List of Disgaea 3 Skills

## Weapon Skills List
All Sword Skills (except legendary) can be learned from Tier 4+ Samurais (male or female)<br />All Fist Skills (except legendary) can be learned from Tier 4+ Fist Fighters (male or female)<br />All Spear Skills (except legendary) can be learned from Tier 4+ Female Warriors or Heavy Knights<br />All Axe Skills (except legendary) can be learned from Tier 4+ Male Warriors or Berserkers<br />All Bow Skills (except legendary) can be learned from Tier 4+ Archers or Rangers<br />All Gun Skills (except legendary) can be learned from Tier 4+ Gunners or Gunslingers<br />Fire Skills until Omega can be learned from Red Mages / Red Skulls<br />Ice Skills until Omega can be learned from Blue Mages / Blue Skulls<br />Wind Skills until Omega can be learned from Green Mages / Green Skulls<br />Star Skills until Omega can be learned from Star Mages / Star Skulls<br />Heal Skills until Omega can be learned from Healers (male or female)

(Note: Omega Fire, Wind and Ice Skills can only be used by Star Mages/Skulls if they are reincarnated from a earlier tier mage who learned those spells)

Legendary Skills can be learned from the Old Weapons Master (Prinny), found in the mystery gate on the 10th floor of the Class World which opens after destroying all the geoblocks on the 10th floor of the Class World.
