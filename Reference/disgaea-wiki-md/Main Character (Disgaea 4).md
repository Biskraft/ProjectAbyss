# Main Character? (Disgaea 4)

|  |


  - Main Character?** is Asagi's exclusive class in Disgaea 4: A Promise Unforgotten.

This class only has an SD sprite set in the main game, although an HD version of the class(titled **Main Character NT?**) can be obtained as DLC.

They're not two separate entities, and once both versions are unlocked a Bill must be passed to switch between them. They do, however, have their own set of stats, skills, Evilities and equipment.

## Description

:*"An admirable girl who patiently awaits the day she'll become the main character."* - Main Character?

:*"A girl who dreams of becoming a main character. Will her dream ever come true?"* - Main Character NT?
