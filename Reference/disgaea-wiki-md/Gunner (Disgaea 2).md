# Gunner (Disgaea 2)

|  |

*Main article: Gunner*
The **Gunner** is a humanoid class in *Disgaea 2: Cursed Memories*.

## Overview
The Gunner is a long-range attacker specialized in using Guns. A team player who gains power when ganging up on the same target as others.

## Skills
### Passive
- Increase damage dealt by 20%(+1% per 2 Lovers) during combo attacks.
