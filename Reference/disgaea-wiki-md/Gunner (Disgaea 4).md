# Gunner (Disgaea 4)

|  |

*Main article: Gunner*

The **Gunner** is a Humanoid class in *Disgaea 4: A Promise Unforgotten*.

## Overview
The Gunner is a long-range attacker specialized in using Guns. Rather than fighting solo, he prefers to gang up on the enemy as part of a group.
