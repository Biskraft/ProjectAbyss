# Celestial Host (Disgaea 5)

|  |

*Main article: Male Angel*

The **Celestial Host** is a Humanoid class from *Disgaea 5: Alliance of Vengeance*.

## Overview
The Celestial Host is an unusual healer/support class, as they prefer to stay near the frontlines powering up nearby allies with their Evilities. They have average offensive stats, and are moderately proficient in various weapons.

The Celestial Host is also notable for being the only class able to revive allies(though at the cost of their own life).
