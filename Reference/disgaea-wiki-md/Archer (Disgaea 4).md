# Archer (Disgaea 4)

|  |

*Main article: Archer*

The **Archer** is a Humanoid class in *Disgaea 4: A Promise Unforgotten.*

## Overview
The Archer is a long-range attacker specialized in using Bows. She supports the offensive and incapacitates enemies safely from the backlines.

## Personality types
- Somber
- Ladylike
- Timid


## Skills
### Unique
Ailment chance increases by 3% per boost level. Boosting will not increase damage.
