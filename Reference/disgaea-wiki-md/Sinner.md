# Sinner

|  |

The **Sinner** is a humanoid class in *Disgaea 2: Cursed Memories*.

## Overview
The Sinner is an offensive class specialized in using Fists. They beat down the enemy through counterattacks, relying on evasion to survive.

## Skills
### Passive
- Increase counterattack damage by 50%(+1% per Lover) when HP is below 25%.
- You have 95% base Team Attack chance with anybody.

## Trivia
- The **Sinner** class is the only class from Disgaea 2 Cursed Memories/Dark Hero Days who didn't make it to Disgaea 3. This is most likely due to the redundancy in style to the Martial Artist Class being restored from the first Disgaea game and it having better stats (1 more in Move, Range, Throw, Counter & 5 in Jump).
- The *design* for the Sinner, however, may have been the inspiration for that of the Male Healer class in Disgaea 3
