# Celestial Host (Disgaea 4)

|  |

*Main article: Celestial Host*

The **Celestial Host** is a Humanoid class in *Disgaea 4: A Promise Unforgotten*.

## Overview
The Celestial Host is a healing class with high mobility. While they lack the support spells of other healers, they have the unique ability to heal from anywhere on the map or even revive units.
