# Zombie

- Zombie** is a recurring term used in the *Disgaea* series which may refer to the following:
- For the monster class introduced in *Disgaea: Hour of Darkness*, see Undead.
- For the monster class introduced in *Disgaea 2: Cursed Memories*, see Dragon Zombie.
- For the main character in *Disgaea 6: Defiance of Destiny*, see Zed.
- For Zed's exclusive class, see Zombie (Zed).
- For a playable character in *Disgaea 6: Defiance of Destiny*, see Bieko.
- For Bieko's exclusive class, see Zombie (Bieko).
