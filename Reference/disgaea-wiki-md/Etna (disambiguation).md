# Etna (disambiguation)

Etna is a female character in the *Disgaea* series.

Due to appearing in multiple *Disgaea* games as playable characters, she has multiple classes in the series.
- For her class in *Disgaea*, see Vassal (Disgaea)
- For her class in *Disgaea 2*, see Beauty Queen (Disgaea 2)
- For her class in *Disgaea 3*, see Beauty Queen (Disgaea 3)
- For her class in *Disgaea 4*, see Demon Lord (Etna)
- For her class in *Disgaea D2*, see Overlord's Vassal
- For her class in *Disgaea 5*, see Vassal (Disgaea 5)
