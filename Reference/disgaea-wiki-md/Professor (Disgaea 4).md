# Professor (Disgaea 4)

|  |

*Main article: Professor*

The **Professor** is a Humanoid class in *Disgaea 4: A Promise Unforgotten*.

## Overview
The Professor is a support class specialized in strengthening allies, learning unique buff skills as well as buff-enhancing Evilities. Not limited to this role, she's also somewhat proficient in Guns and healing.

## Personality types

- Smart
- Mad Scientist
- Forgetful
