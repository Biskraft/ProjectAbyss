# Raspberyl (disambiguation)

Raspberyl is a female character in the *Disgaea* series.

Due to appearing in multiple *Disgaea* games as a playable character, she has multiple classes in the series.

- For her class in *Disgaea 2*, see No. 1 Delinquent (Disgaea 2)
- For her class in *Disgaea 3*, see No. 1 Delinquent (Disgaea 3)
- For her class in *Disgaea 4*, see No. 1 Delinquent (Disgaea 4)
- For her class in *Disgaea 5*, see Delinquent Teacher
