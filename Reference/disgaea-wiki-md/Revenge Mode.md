# Revenge Mode

- Revenge Mode** is a gameplay feature introduced in *Disgaea 5: Alliance of Vengeance*.

Each unit has a Revenge Gauge(visible next to their portrait), which fills every time one of the following happens:

- That unit is hit. Every character has their own base value that the gauge is filled per hit
- An ally is hit by an enemy(fills by about half the base value)
- An ally is killed by an enemy(fills by about 5 times the base value)
- That character kills an enemy in Revenge Mode(fills by 5 times the base value)

When the Gauge reaches 100% the unit enters Revenge Mode for 3 turns, granting the following benefits:

- Critical Hit Rate is increased to 100%
- SP costs are reduced to 1
- Damage taken is reduced by 25%
- Able to use their Overload, if they have one(once per battle)

Also, killing an enemy that is in Revenge Mode will reward you with a Revenge Shard, which when used on a unit will increase one of their stats permanently.
