# Dragon (Disgaea D2)

|  |

  - Dragon** is a Monster class in *Disgaea D2: A Brighter Darkness*. It is unlocked by clearing map 2-4.

## Quotes
> A dragon who rules the skies. Completely blocks out wind attacks.|Job Description, *Disgaea D2: A Brighter Darkness*

| class="DD2b" | Destroyer Dragon |
| Air Charge | Increase stats by 5% every time a unit on map uses a wind skill. |
---
| class="DD2b" | Timid Dragon |
| Wind Depression | Decrease wind element stat of enemy units on map by 15%. |
