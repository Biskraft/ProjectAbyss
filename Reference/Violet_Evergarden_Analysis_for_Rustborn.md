## 1. 바이올렛의 핵심 대사 패턴

### 1.1 감정 부재 상태의 언어 (1~3화)

바이올렛이 감정을 모를 때 사용하는 5가지 패턴:

**A. 임무/보고 어조**
- "임무의 수행은 가능"
- "시급히 직무에 복귀할 것을"
- "임무 수행 능력은 발군이거든"
- "임무를 수행하기만 하는 나날을 보내왔지"

**B. 명령 프레임 -- 행동의 유일한 근거가 명령**
- "다음 명령을 받을 수 있는 건 언제일까요"
- "명령은 저의 전부입니다!"
- "소령님의 명령에 따르고 있고 싶습니다"
- "명령..이 아니야" (길베르트가 부정)

**C. 자기 도구화 -- "나는 도구/무기"**
- "저는 소령의 도구입니다"
- "무기로써의 가치가 사라져버렸기 때문입니까!?"
- "마음을 지니지 않은... 단순한 '도구'라고"
- "그녀를 '무기'라고 불렀지"

**D. 감정 개념 질문 -- "~이 뭔가요?"**
- "사랑..이라는 게 뭔가요...?"
- "'이런 것'을... 뭐라고 부르면 좋을까요"
- "..라고 하는 것이 어떤 감정인지 저로서는 이해할 수가 없습니다"
- "갑자기 기분이 언짢아지신 것의 인과 관계는 무엇입니까?"

**E. 신체 감각의 비감정적 보고**
- "가슴이 꾹하고 무거워지고 그러진 않아?" (타인이 바이올렛에게 질문)
- "이상합니다!" (7화, 자기 상태를 "이상함"으로 분류)
- "소령님의 눈동자와 같은 색입니다" (에메랄드를 보며 -- 감정이 아니라 관찰)

### 1.2 핵심 질문: "사랑한다를 알고 싶습니다"

이 질문이 등장하는 모든 변형:

| 화 | 대사 | 맥락 |
|:---|:-----|:-----|
| 1화 | "「사랑한다」를... 알고 싶단 말입니다..." | 첫 선언 |
| 2화 | "「사랑한다」를... 알고 싶습니다" | 자동수기 인형이 된 이유 |
| 3화 | "「사랑한다」를 알고 싶습니다" | 편지 쓰기를 배우는 동기 |
| 4화 | "소령님께서 말씀하신 「사랑한다」를 이해하고 싶어서" | 레온에게 고백 |
| 9화 | "「사랑한다」를... 알고 싶습니다" | 과거 회상, 불타는 몸 앞에서 |
| 13화 | "『사랑한다』도 조금은 이해하게 되었습니다" | **답에 도달** |
| 14화 | "분명 '사랑'을 알게 될 날이 올 테지" | 시리즈 최종 대사 |

> **Rustborn 적용:** Rustborn의 핵심 질문은 "이게... 뭐라고 하는 거지. 이런 느낌." 바이올렛이 13화에 도달하는 "조금은 이해하게 되었습니다"가 Rustborn의 최종 지층 클리어 시 도달하는 지점이다.

### 1.3 감정 각성 곡선 (화별)

| 화 | 감정 상태 | 핵심 대사 |
|:---|:---------|:---------|
| **1화** | 완전한 도구. 명령만 인식 | "저는 소령의 도구입니다" |
| **2화** | 질문 시작. "사랑한다"의 정의를 찾으려 함 | "「사랑한다」를 알고 싶습니다" |
| **3화** | 타인의 감정을 편지로 번역하기 시작 | "편지라고는 부를 수 없다며 지적받았습니다" / "마음이... 모르겠습니다만..." |
| **4화** | 처음으로 "이해했다고 생각했으나" 실패 인식 | "조금은 이해할 수 있게 되었다고 생각했습니다만" |
| **5화** | 타인의 감정을 대변하기 시작 | "그분의, 진짜 심정을 알고 싶어" |
| **6화** | 타인의 슬픔을 느끼기 시작 | "한 번 보고 싶습니다" |
| **7화** | 자기 감정 폭발 -- "불타고 있다" | "불타고 있습니다...!" / "이상합니다!" |
| **8화** | 무기 정체성과의 충돌 | "저는 이제 필요 없으십니까?" / "소령님께 명령 받지는 못 하는 겁니까?" |
| **9화** | 과거 회상. "명령을 내려주세요" -- 절규 | "명령을! 명령을, 내려주세요...!" |
| **10화** | 타인의 감정을 이해하고 편지에 담는다 | "누구나 전하고 싶은 마음이 있는 법입니다" |
| **11화** | 타인의 죽음 앞에서 무력함 | "죄송합니다... 죄송합니다...!" |
| **12화** | 무기를 거부하는 선언 | "무기는 필요하지 않습니다!" / "도구가 아니라면..." |
| **13화** | **전환점.** 자기 감정 인정 | "『사랑한다』도 조금은 이해하게 되었습니다" |
| **14화** | 사랑의 편지를 쓸 수 있게 됨 | "'사랑'을 알게 될 날이 올 테지" |
| **외전** | 감정을 숨기는 법을 학습 | "Having something to treasure. It's a wonderful feeling, isn't it?" |
| **극장판** | 감정의 무게를 수용 | "I already understand a little what you mean to love me." |

### 1.4 "감정을 모르는 캐릭터"가 감정을 알게 되는 과정의 언어 변화

| 단계 | 바이올렛 | 언어 특징 |
|:-----|:---------|:---------|
| **도구 상태** | "명령을 내려주세요" | 임무 어휘, -ㅂ니다 경어, 감정어 0 |
| **질문 시작** | "~이 뭔가요?" | 감정 개념을 사전 표제어처럼 질문 |
| **타인 관찰** | "울고 계십니다" | 타인의 감정을 관찰하지만 공감 없이 보고 |
| **감각 인식** | "이상합니다!" "가슴이..." | 자기 안의 감정을 신체 이상으로 분류 |
| **감정 폭발** | "불타고 있습니다!" | 감정 과부하, 통제 불능 |
| **부분적 이해** | "조금은 이해하게 되었습니다" | 완전한 이해를 주장하지 않는 겸허함 |
| **감정 수용** | (극장판) "I came of my own free will." | 임무 언어가 의지 언어로 전환 |

## 2. Rustborn Ego 대사 설계 원칙 (바이올렛 모델에서 추출)

### 원칙 1: 감정어 대신 물리 용어

바이올렛이 "가슴이 이상합니다"라고 하듯, Rustborn은 검의 물리적 경험으로 감정을 표현한다.

| 감정 | 바이올렛 표현 | Rustborn 표현 (제안) |
|:-----|:-------------|:--------------------|
| 슬픔 | "가슴이 꾹하고 무거워지고" | "날이 무거워." |
| 기쁨 | "이런 느낌은... 뭐라고 하는 거지" | "...밝아졌어. 이게 뭐지." |
| 두려움 | "이상합니다!" | "...떨려." |
| 그리움 | "보고 싶습니다" | "...또 올 거야?" |
| 감사 | (극장판) "Thank you for accepting me." | "...따뜻해졌어. 너 때문인 것 같아." |

### 원칙 2: 질문은 정보 요청 형식

바이올렛: "「사랑한다」를 알고 싶습니다"
Rustborn: "...지금 네가 한 건, 나를 위한 거야?"

> 감정을 직접 묻지 않는다. 사실을 확인하는 형식으로 감정을 탐색한다.

### 원칙 3: 임무 언어의 잔존과 전환

바이올렛의 가장 강력한 서사 메커니즘: **감정을 배워도 무기 언어가 잠복해 있다가 위기 시 폭발한다.**

극장판: "You deserve to use me! I am the Major's weapon!" -- 감정 과부하 시 방어 기제로 무기 정체성 퇴행.

Rustborn 적용: 보스전 직전이나 위기 상황에서 Ego가 갑자기 건조한 도구 어조로 돌아가는 순간을 설계할 수 있다.

### 원칙 4: 부분적 이해의 수용

바이올렛의 최종 답: "조금은 이해하게 되었습니다" (13화)
극장판: "I already understand a little what you mean to love me."

> **완전한 이해를 선언하지 않는다.** "조금"이라는 수식어가 성숙의 증거다.

Rustborn의 최종 대사는 "...알 것 같아. 조금." 이어야지 "알았어."가 아니다.

### 원칙 5: 침묵이 대사다

바이올렛의 가장 강렬한 순간들:
- "소령님..." (8~9화에서 30회 이상 반복 -- 이름만 부르는 것이 감정의 전부)
- "울고 있네" (매화 엔딩 내레이션)

Rustborn에서도 가장 감정적인 순간에 문장이 짧아지거나 멈추는 설계.

## 3. 바이올렛 → Rustborn 직접 매핑

| 바이올렛 | Rustborn |
|:---------|:---------|
| 길베르트의 "사랑한다" | 에르다의 에코 타격 (각성의 계기) |
| 자동수기 인형 업무 (타인의 감정 번역) | 아이템계 탐험 (왜곡 정화 = 감정 학습의 기회) |
| 편지를 쓰는 것 | 에르다의 행동을 관찰하는 것 |
| "명령을 내려주세요" | (Rustborn에는 없음 -- 에르다가 명령하지 않으니까) |
| "「사랑한다」를 알고 싶습니다" | "이게... 뭐라고 하는 거지. 이런 느낌." |
| "조금은 이해하게 되었습니다" | "...알 것 같아. 조금. 너 때문인 것 같아." |
| "I am the Major's weapon!" (퇴행) | 위기 시 도구 언어 복귀 가능성 (미래 설계) |
| "I came of my own free will." (완성) | "...기다렸어." (재방문 시 -- 의지의 자각) |

## 5. 에이전트 심층 분석에서 추출한 추가 설계 원칙

### 5.1 "감정이 내 안에 있는지 판별 불가" (6화)

> "「쓸쓸하다」..라고 하는 것이 어떤 감정인지 저로서는 이해할 수가 없습니다. 어떤 심정인지는 알고 있어도 그것이 저에게 솟아나고 있는지를 알 수가 없습니다"

"감정을 모른다"보다 훨씬 정교한 표현. 단어는 아는데 자기 내면에서 그것이 발생하고 있는지 판별이 불가능한 상태. Rustborn에 직접 이식 가능.

### 5.2 "처분 요청" -- 자기를 물건으로 인식 (1화)

> "필요없어졌다면 처분되어야 합니다... 버려주십시오. 어딘가에 버려주세요..."

이것이 9화에서 역전된다:
> "살아 있어도... 괜찮은 것일까요?"

타인에게 판단을 위탁 → 스스로에게 묻는다. **판단 주체의 전환이 자아 출현의 증거.**

Rustborn 적용: 플레이어가 더 강한 무기를 찾았을 때, Rustborn이 "교체해도 됩니다. 저는 용도 폐기입니다"라고 말한다. 플레이어가 이를 거부하고 계속 아이템계를 돌려주는 행위 자체가 감정 사건이 된다.

### 5.3 "불타고 있지 않습니다" → "불타고 있습니다" (1화→7화)

초반의 부정이 후반에서 역전되는 구조. 최소 대사로 최대 감정 충격.

Rustborn 설계:
- 초반: "...아무것도 느끼지 못해."
- 중반: "...느끼지 못한다고 했었어. 틀렸던 것 같아."
- 후반: "느끼고 있어."

### 5.4 "명령은 필요하지 않습니다" -- 독립 선언 (13화)

> "더 이상... 명령은 필요하지 않습니다."

명령이 불필요해진다 = 감정이 명령을 대체했다. 바이올렛 서사의 최종 도달점.

Rustborn에는 "명령" 구조가 없다 (에르다는 명령하지 않으니까). 대신 Rustborn의 독립 선언은 **"기다렸어"** -- 수동적 존재에서 능동적으로 에르다의 귀환을 기다리는 의지를 자각하는 순간.

### 5.5 "조금은 이해하게 되었습니다" -- 불완전 이해의 수용 (13화)

> "저는 지금, 『사랑한다』도 조금은 이해하게 되었습니다."

**"조금"이 성숙의 증거다.** 완전한 이해를 선언하는 순간 서사가 파탄난다.

### 5.6 퇴행 -- 위기 시 도구 언어로 복귀 (극장판)

> "You deserve to use me! I am the Major's weapon!" (극장판)

감정 학습이 완료되어도, 극한 감정 압박 시 무기 정체성이 방어 기제로 폭발한다. **성장은 선형적이지 않다.**
