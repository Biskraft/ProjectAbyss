# Wai Wai Jockey

is a horse racing simulation arcade medal game developed and published by Konami exclusively in Japan in 1995.

## Features
This jockey simulation game features several Konami mascot characters, like Kid Dracula from *Akumajō Special: Boku Dracula-kun*, Goemon and Ebisumaru from the *Ganbare Goemon* series, and Penta from *Antarctic Adventure*.
