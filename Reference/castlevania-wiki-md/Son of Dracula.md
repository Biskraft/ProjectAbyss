# Son of Dracula

- Son of Dracula** is a set bonus given for collecting Alucard related items in *Castlevania Puzzle: Encore of the Night*. Experience is awarded upon first collecting the items for the related Honor and bonus stats are given when all items in the set are equipped. This is the only set bonus that counts as an Achievement in the Windows 7 version of the game.
