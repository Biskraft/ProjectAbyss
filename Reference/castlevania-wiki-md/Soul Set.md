# Soul Set

right|250px

  - Soul Set** is a menu page in *Castlevania: Aria of Sorrow* where you can choose which souls are currently active and also view all the souls you have collected so far, their MP cost and the effect on your attributes. Each souls has a short description that tells what effect is applied, when equipped.

All souls are divided into three categories:
    - Bullet Souls** (<span style="color:red;">red</span>) - Souls which allow to perform an instantaneous attack or action at a one time MP cost.
    - Guardian Souls** (<span style="color:blue;">blue</span>) - Souls which can be activated and deactivated at will and incur in a continuous MP cost while in use.
    - Enchanted Souls** (<span style="color:orange;">yellow</span>) - Souls which can be equipped and provide a benefit without MP drain.

:***Note:** Ability Souls cannot be managed in the "Soul Set" menu in any way.*
Category: Aria of Sorrow
