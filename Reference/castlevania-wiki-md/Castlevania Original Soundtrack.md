# Castlevania Original Soundtrack

- Castlevania Original Soundtrack** could refer to:
    - 1990**  Akumajō Dracula Famicom Best, Original Soundtrack CD for *Castlevania*, *Castlevania II: Simon's Quest*, and *Castlevania III: Dracula's Curse*.
    - 1999**  Castlevania - The Original Game Soundtrack, Original Soundtrack CD for *Castlevania* (N64).
    - 2005**  Castlevania Original Soundtrack, the Japanese name for the PlayStation 2 game *Castlevania: Lament of Innocence* Original Soundtrack CD.
