## Notes
- In *Castlevania: Symphony of the Night*, the Heart Refresh is fairly well known for a glitch where while using it and repeatedly switching rooms can result in Alucard ending up inside the floor. This is often combined with a Duplicator, but in tool-assisted runs only one Heart Refresh is used, gained by using all Richter's Hearts during the prologue. This glitch can be used to access the entrance to the Underground Garden only present in the Saturn version (only a save room is present in other versions), but in speedruns it is most commonly used to access the Library Card under the second sealed floor in the Marble Gallery, allowing Alucard to teleport to the Long Library before ever going there in the first place.
