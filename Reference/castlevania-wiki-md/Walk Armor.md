# Walk Armor

There are two different types of armors that can be called **Walk Armor**: 
- For the *Castlevania: Symphony of the Night* version, please see Scout Armor, which has a higher defense the more map coverage you have.
- For the *Castlevania: Harmony of Dissonance* version, please see Healing Mail, which improves health while walking.
