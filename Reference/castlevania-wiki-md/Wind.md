# Wind

- Wind** is one of the basic elemental attributes found in *Castlevania*.

Unlike most elements in the games (or even role-playing games in general), Wind is a rarely occurring, oddly implemented element, only occurring with select few attacks, resulting in practically no equipment that inflicts or resists this element.

In most *Castlevania* games, wind spells/skills are associated with the Slash damage type (e.g.: Gale Force, Pneuma, etc.)
