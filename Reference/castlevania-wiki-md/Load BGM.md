# Load BGM

- Load BGM** is a music theme originally composed by Konami Kukeiha Club and Sota Fujimori for *Castlevania Chronicles*. It plays whenever a file from the game is currently loading.

An arranged version later appeared in Castlevania: The Dracula X Chronicles Original Soundtrack.

In *Castlevania: The Adventure ReBirth*, it is the background theme for Stage 4.
