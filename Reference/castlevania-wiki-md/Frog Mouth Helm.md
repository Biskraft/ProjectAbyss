# Frog Mouth Helm

The **Frog Mouth Helm** is a jouster's helmet in *Castlevania: Curse of Darkness*. It is created from a Gallic Helmet by adding Aquamarine, which can first be obtained in the Mortvia Aqueduct, increasing that helmet's DEF from +10 to +12. The helmet is outclassed when another Gallic Helmet is upgraded to a Sallet (DEF +14) using an Indigo Steel, or when a Thracian Helmet is upgraded to a Barrel Helmet (DEF +16) using Carbon Steel and a Knight's Vein. The Frog Mouth Helm can gain a higher DEF than either of these, by combining it with White Steel to form a Lobster Tail Pod (STR +18), which cannot be further upgraded and can be outclassed by items a Sallet can evolve to.
