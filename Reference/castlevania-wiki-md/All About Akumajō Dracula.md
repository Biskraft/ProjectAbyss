# All About Akumajō Dracula

- All About Akumajō Dracula*** is a Japanese guidebook to the game known in America as *Super Castlevania IV*, written by Ichiro Tezuka. It features original artwork for all of the enemies in the game. The book is published by Takarajimasha (formerly JICC Publishing).
