# Leather Shield

right|68px

The **Leather Shield** is a shield with a low defense. While holding it out, it can be used to block some projectiles. It is the weakest of Alucard's shields, except for perhaps the Alucart Shield, which is utterly useless other than for obtaining a luck boost gained when equipping the rest of the Alucart equipment.

The Leather Shield is the first shield found in *Castlevania: Symphony of the Night* and can be found in a container in the Alchemy Laboratory. When combined with the Shield Rod or the Mablung Sword, it can summon a cow to give a temporary DEF boost of 20.

In *Castlevania: Harmony of Despair*, it is the first shield that can be obtained without going to the shop. It's also a flammable shield in this game and will not block fire attacks.
