# Skill Cube

right|300px

The **Skill Cube** is a relic in *Castlevania: Portrait of Ruin*. It allows the player to utilize the partner's sub-weapon or magic spell by pressing the **R** button.

Note that when controlling Jonathan, this relic does not charge Charlotte's spells. Thus, pressing the **R** button only executes the weak versions of these spells.
