# Killer Bee (Innocent Devil)

right|200px
The **Killer Bee** is a Level 3 Fairy-Type Innocent Devil. It can evolve from either the Herbest or the Leaffle. It can learn two new abilities: Heal Lv. 3, which heals Hector for 100 HP compared to Lv. 2's 50, and Curse Breaker, which heals Curse status.
