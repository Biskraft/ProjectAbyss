# NES Game Atlas

The **NES Game Atlas** is a book comprising of level maps for many Nintendo games. It has a section for the three NES *Castlevania* games that had been released at the time: *Castlevania*, *Castlevania II: Simon's Quest* and *Castlevania III: Dracula's Curse*. Its first page contains original artwork of Trevor Belmont, but contains no original artwork for any other character.
