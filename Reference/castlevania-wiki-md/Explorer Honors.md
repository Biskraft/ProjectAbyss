# Explorer Honors

- Explorer Honors** are Honors awarded for uncovering certain percentages of the total game map in *Castlevania Puzzle: Encore of the Night*. They each unlock something different in the game as an award. The Veteran Explorer honor notably unlocks the Librarian as a playable character in Arcade Mode. These Honors also count as Achievements in the Windows 7 version of the game and award an incremental amount of game points.
