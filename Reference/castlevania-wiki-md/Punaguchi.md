# Punaguchi

- Punaguchi** is an enemy in the *Castlevania* series.

## Description
A moldy growth sometimes in the shape of a hand with a mouth. It spews orbs diagonally which start ricocheting off of walls until they find a way to exit the screen.

In *Castlevania II: Belmont's Revenge*, it's the only enemy that can demote Christopher's whip when hitting him.

In the Game Boy titles, the mouth faces forward, while in *The Adventure ReBirth*, the mouth faces upward.
