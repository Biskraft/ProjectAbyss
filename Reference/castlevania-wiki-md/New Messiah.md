# New Messiah

- New Messiah** is a music theme composed by Konami Kukeiha Club for *Castlevania II: Belmont's Revenge*. It is the background theme for the Crystal Castle. An arranged version later appears in *Castlevania: The Adventure ReBirth* as the background theme for Stage 3.
