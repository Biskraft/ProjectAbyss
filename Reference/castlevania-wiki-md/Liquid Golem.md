# Liquid Golem

right|200px

  - Liquid Golem** is a Battle-Type Innocent Devil found in *Castlevania: Curse of Darkness*. It is a Level 4 Innocent Devil, and can learn Mercury Sphere and Magma Mode. It can be obtained from a Juggernaut Devil, using red, green or yellow crystals.
