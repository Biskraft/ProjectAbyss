# Akumajyo Dracula Peke

is a minigame that came with *Castlevania: Rondo of Blood*. It is also known as . This minigame played instead of *Rondo of Blood* if the user was using the wrong kind of System Card.

- Peke* is Japanese for the letter "X", substituting the usual "*ekkusu*" pronunciation used in the actual game's title and the series title is pronounced  to highlight that this is essentially a "wrong" game.

The game was later included as an unlockable feature in *Castlevania: The Dracula X Chronicles*.

It is also an Easter Egg feature in *Castlevania Requiem: Symphony of the Night/Rondo of Blood*, wherein inputting the famous Konami Code on the main menu (using 16px and 16px to substitute for **A** and **B**, respectively, and **Options** for **Start**) will cause the screen to briefly glitch similar to a CRT monitor before loading the minigame.

## Description
On the original version of *Rondo of Blood*, this minigame would trigger if the CD was booted with an incorrect System Card inserted into the PC Engine console. It simply consists of a cartoonish Richter Belmont moving from left to right and defeating purple creatures with his whip. The stage is short and ends in a large board telling players to reset the game and use the correct system card, while a similarly cartoonish Maria Renard walks about, occasionally jumping and saying "Sorry!" The track "Opus 13" plays all throughout.

When it was included in *The Dracula X Chronicles*, the message at the end now simply tells players to press **Start** to return to the main menu. The Easter Egg version in *Requiem* has a similar message, but also chastises players as cheaters by stating that the collection does not support "extra lives" codes.
