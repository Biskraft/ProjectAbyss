# Konami Classics Vol. 1

- Konami Classics Vol. 1*** is a compilation of three classic Konami games released on Xbox Live Arcade in 2009. Each game features the ability to be played in its original state or in an enhanced mode with upgraded high definition support at 720p, along with a special graphics improvement filter resulting in smoother visual game quality. In addition, the game features unlockable achievements.

## Game list
The games included are:
  - Castlevania: Symphony of the Night*
  - Super Contra*
  - Frogger*
