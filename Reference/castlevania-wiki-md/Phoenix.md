# Phoenix

right|250px


  - Phoenix** is a Bird-Type Innocent Devil found in *Castlevania: Curse of Darkness*. It is a Level 3 Innocent Devil and can learn Big Bang and Fire Bird. It can be obtained from a Skull Wing Devil, using green, yellow or white Evo Crystals.
