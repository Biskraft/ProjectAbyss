# Dracula (disambiguation)

The name **Dracula** could refer to various characters or notions.

## Characters
    - 1428/31–1476/77** — Vlad the Impaler: The real historical Dracula.
    - 1897** — Count Dracula: The main antagonist of the novel written by Bram Stoker.
    - 1986** — **Dracula**: The main antagonist of the *Castlevania* series.
    - 1986** — Count Dracula: The antagonist of the game *Vampire Killer*, set in another timeline.
    - 1988** — Dracula: The antagonist of the game *Haunted Castle*, set in another timeline.
    - 1991** — Dracula: The antagonist of the game *Super Castlevania IV*, a retelling of the first game, now set in another timeline.
    - 1993** — Count: Dracula's incarnation in *Captain N: The Game Master*.
    - 1997** — Dracula: The antagonist of the game *Castlevania Legends*, set in another timeline.
    - 1999** — Dracula: The antagonist of the two Nintendo 64 games *Castlevania* and *Castlevania: Legacy of Darkness*, set in another timeline.
    - 2001** — Dracula: The antagonist of the game *Castlevania: Circle of the Moon*, set in another timeline.
    - 2005** — Dracula: The main antagonist in the graphic novel *Castlevania: The Belmont Legacy*, set in another timeline.
    - 2009** — Dracula: The antagonist of the game *Akumajō Dracula: The Arcade*, set in another timeline.
    - 2009** — Dracula: The antagonist of the game *Castlevania: The Adventure ReBirth*, set in another timeline.
    - 2010** — Dracula: The main character of the *Lords of Shadow* reboot series, who was formerly known as Gabriel Belmont.
    - 2017** — Dracula: A version of the character based on that from the original timeline and the main antagonist of the *Castlevania* animated series.

## Novels
    - 1897** — *Dracula*: The novel written by Bram Stoker.
    - 2009** — *Dracula the Un-dead*: The sequel written by Dacre Stoker and Ian Holt.
