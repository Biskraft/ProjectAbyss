# DEF Up

- DEF Up** is an item in the *Castlevania* series. When picked up, the player's Defense stat is increased.

It appears in both PlayStation 2 titles: *Castlevania: Lament of Innocence* and *Castlevania: Curse of Darkness*. It is only available when playing an alternate mode, to make up for the lack of a level up system.
