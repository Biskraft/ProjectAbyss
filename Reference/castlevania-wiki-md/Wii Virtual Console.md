# Wii Virtual Console

- Wii Virtual Console** is a specialized section of the Wii Shop Channel which allows Nintendo Wii owners to buy downloadable games for that video game system. However, as time has passed and new systems have been released, the Virtual Console has expanded its services for other game systems, like the Nintendo 3DS and the Nintendo Wii U.

There are a few *Castlevania* games available for purchase on the Virtual Console.
