# Juggernaut

right|200px

  - Juggernaut** is a Battle-Type Innocent Devil found in *Castlevania: Curse of Darkness*. It is a Level 3 Innocent Devil, and can learn Homing Eye and Brute Force Lv.2. It can be obtained from a Golem Devil, using all but white crystals.
