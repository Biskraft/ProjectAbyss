# Khaos

right|200px


  - Khaos** is a Bird-Type Innocent Devil found in *Castlevania: Curse of Darkness*. It is a Level 3 Innocent Devil, and can learn Sphere of Darkness. It can be obtained from a Goldfinch Devil, using any type of Evo Crystals.
