# Gallic Helmet

The **Gallic Helmet** is a helmet that can be equipped by Hector in *Castlevania: Curse of Darkness*. It is the first alternate evolution path for the Leather Helm, which itself has multiple evolutionary paths, each which simply increase Defense. It is created by adding a Knight's Vein to a Leather Helm. This results in a Helmet with +10 Defense, which is 2 more than the Thracian Helm that is also evolved from the Leather Helm. It can be combined with an Aquamarine (first found in the Mortvia Aqueduct) to form a Frog Mouth Helm for an additional +2 DEF, or an Indigo Steel (first found in the Forest of Jigramunt) to create a Sallet for and additional +4 DEF, which can ultimately be upgraded to the toughest helmet in the game, the Cross Helmet.
