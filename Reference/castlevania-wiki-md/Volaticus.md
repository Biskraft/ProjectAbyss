# Volaticus

right|150px

  - Volaticus** is a back glyph that allows Shanoa to fly. Shanoa can still attack while in mid-air using the currently equipped Shoulder Glyphs. Glyph Unions however, cannot be cast while Volaticus is still active.


## Item Data


## Notes
- Connecting *Castlevania Judgment* to *Castlevania: Order of Ecclesia* unlocks the Wings of Darkness, an accessory that resembles the Volaticus wings.
