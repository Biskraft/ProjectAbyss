# Rasetz

right|150px

 is a Battle-Type Innocent Devil found in *Castlevania: Curse of Darkness*. It is a Level 4 Innocent Devil, and can learn Grand Wave and Glow Sword. It can be obtained from a Speed Mail Devil, using 90 **<span style="color:blue;">Blue</span>** or **<span style="color:white;">White</span>** Evo crystals.

## Description
Rasetz has high attack, and acts very aggressively, using strings of combos and focusing primarily on a single target until it dies. While it lacks attacks with wide hitboxes like it's Level 4 counter-part Corpsey, using the Glow Sword ability, it utilizes several glowing purple orbs (Murasame-maru) which constantly hover around him to attack and keep a myriad of foes off balance, with a moderate chance to stun. Later Generations and Higher Levels increase the use of the orbs, as well as the chance they have to stun on impact.
