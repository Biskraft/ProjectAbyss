# Orb (Bloodlines)

right|150px

The **Orb** is an item in *Castlevania: Bloodlines*. It works the same way as an Invisibility Potion in other games of the series, granting John Morris or Eric Lecarde invulnerability to enemies for a small amount of time. It can be found by breaking some candles.
