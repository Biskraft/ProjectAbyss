# Evil Swords

The **Evil Sword** is an enemy in *Castlevania: Lament of Innocence*. Much like the Ruler Sword and Bloody Sword, these are flying swords possessed by an evil spirit, but some are able to master an element property.


## Description
The **Evil Sword** is a living sword that floats around and attacks the player. It can only be found in the House of Sacred Remains.
