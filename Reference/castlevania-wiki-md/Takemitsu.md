# Takemitsu

right|73px
  - Takemitsu** is a bamboo sword. It is the only sword in the *Castlevania* series that deals out Strike damage.

In *Castlevania: Symphony of the Night*, it is a slight improvement over the Red Rust. Like the Red Rust, it is a very weak two-handed sword which is unable to swing diagonally downward. It has a token attack of +1, which is better than the Red Rust's -2; however, it doesn't lock up like the Red Rust. It deals Hit rather than Cut damage, meaning that it cannot draw out blood, preventing Alucard from healing with Dark Metamorphosis while it is equipped.
