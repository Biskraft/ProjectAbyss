# Skull Wing

right|250px


  - Skull Wing** is a Bird-Type Innocent Devil in *Castlevania: Curse of Darkness*. It is a level 2 Innocent Devil, and can learn Carpet Bombs and Bone Shot. It can be obtained from a Crow Devil, using 40 **<span style="color:red;">Red</span>** or **<span style="color:blue;">Blue</span>** Evo Crystals.

## Description
Proper positioning must be taken into consideration when using this evolution of the Bird-Type. The Skull Wing is good when it comes into distant combat since the Bone Shot ability makes this Innocent Devil vulnerable. It also executes the Carpet Bombs skill like a bombing run while flying around the vicinity.
