# Delta Spark

Delta Spark is one of Charlotte Aulin's spells in *Castlevania: Harmony of Despair* that creates an electrified triangular field of destruction, which can be learned from Astarte.

Soma Cruz is able to use the same attack by absorbing and using Astarte's soul.
