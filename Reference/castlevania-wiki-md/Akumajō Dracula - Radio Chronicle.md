# Akumajō Dracula: Radio Chronicle

thumb|right|CD cover
 is a Japanese language only internet radio program aired on Konami Station in 2008 that featured the voice actors from *Akumajō Dracula X: Tsuioku no Yasōkyoku*. Various *Castlevania* subjects were discussed, although other topics non- regarding to the series were also discussed as well.
thumb|150px|The voice actor for Richter was the host of the Radio Chronicles.
The radio drama *Akumajō Dracula X: Tsuioku no Yasōkyoku* was aired as a feature of this program. A CD containing the contents of this program in MP3 format is now available in Japan as of March of 2010.
