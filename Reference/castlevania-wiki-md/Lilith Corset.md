# Lilith Corset

The **Lilith Corset** is a corset worn by Lilith. It provides DEF +17, which is greater than an Amethyst Corset (DEF +11), and less than a Platinum Corset (DEF +25). It is also greater than the three corsets that provide DEF +16, but these also increase other statistics as well (Emerald Corset boosts INT +3, Sapphire Corset boosts CON +3, and Ruby Corset boosts STR +3).
