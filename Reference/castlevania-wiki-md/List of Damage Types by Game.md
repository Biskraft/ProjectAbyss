# List of Damage Types by Game

This page is intended to show the different damage types that are found in the different games. The Bestiary for a game is considered the "official" name for the effect. If an icon is used in the bestiary, the name of the effect is taken from the English manual of the game when available. In-game text is used if it is clear what the universally accepted name for the attribute is. Official Strategy Guides are used if this information is not otherwise available.
