# Hornet

right|200px
  - Hornet** is a Fairy-Type Innocent Devil found in *Castlevania: Curse of Darkness*. It is a level 3 Innocent Devil and can learn the skills Stone Breaker and Lucky Slot. It can only be obtained from a Herbest fairy, using Blue, Yellow or White Evo Crystals.
