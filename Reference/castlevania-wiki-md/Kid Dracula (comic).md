# Kid Dracula (comic)

Konami packaged a short Kid Dracula comic with each *Akumajō Special: Boku Dracula-kun* (*Kid Dracula* (GB)) game released in Japan for the Famicom and Game Boy systems. Each one included a story section in the instruction booklet.
