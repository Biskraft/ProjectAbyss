# Silk Cloak

The **Silk Cloak** is the weakest cloak found in *Castlevania: Harmony of Dissonance*. It gives +2 Defense, which is one less than the Velvet Cloak, but is still greater than the +1 DEF that a normal Cape has, which is usually the weakest cloak, but doesn't appear in this game.
