# Tiara (Innocent Devil)

right|200px
  - Tiara** is a Fairy-Type Innocent Devil found in *Castlevania: Curse of Darkness*. It's a level 4 Innocent Devil and can learn Healing Drop and Crystal Skull. It can only be obtained either from a Killer Bee, using Red, Green or Yellow crystals; or from a Hornet fairy, using Red, Blue or Green crystals.
