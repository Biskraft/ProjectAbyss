# Talisman

right|73px

The  is a type of amulet with supernatural powers found in the *Castlevania* series. It first appeared in *Castlevania: Symphony of the Night*, where it has a random chance of absorbing the damage dealt by an attack; this effect is increased by Luck. Whenever this item absorbs damage, a shadow version of Alucard appears to be taking the blow. If Alucard has a Luck of 999 (maximum) then he will be invincible (except for those attacks that inflict the Stone ailment).

This item returned in *Castlevania: Lament of Innocence*, where it provides the same effect.
