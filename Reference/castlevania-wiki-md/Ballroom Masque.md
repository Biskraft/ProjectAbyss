# Ballroom Masque

The **Ballroom Masque** is a mask in *Castlevania* that nobles wear at formal dances.

## Game specific information
### *Castlevania: Symphony of the Night*
Called **Ballroom Mask** in North America and **Dance Mask** in Japan. It provides DEF +2. This is an improvement over the Sunglasses (which only provide DEF +1 and deduce INT -1), but is inferior to the Stone Mask, which also provides INT +4.

### *Castlevania: Portrait of Ruin*
Jonathan can equip the **Ballroom Masque**, which grants him DEF +4 and INT +1. It is dropped by Ghost Dancers. This is compared to Jonathan's Stone Mask, which increases Luck instead of Intelligence, or the cursed Skull Mask, which increases DEF greatly by 13, but at the cost of 3 INT, MND and LCK.

### *Castlevania Judgment*
This game features two types of ballroom masques: the **Gold Masquerade Mask** and the **Feathered Mask**. They are face accessories that can be equipped to customize the appearance of a character.
