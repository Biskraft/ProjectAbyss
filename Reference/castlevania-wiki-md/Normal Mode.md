# Normal Mode

- Normal Mode**, or **Standard Mode**, is a game set at a normal difficulty level. It is the default mode most games start with.

## Game specific information
### *Castlevania Legends*
Standard Mode is distinguished from Light Mode in *Legends* in that the player starts the game in a normal state and with a Normal Whip. Crystals must be obtained in order to raise the power of the whip. The whip is reduced to a Normal Whip after each death.
