# Underground

- Underground** is a music theme composed by Konami Kukeiha Club for *Castlevania*. It is the background music for the name entry/file select screen, which only appears in the original Famicom Disk System version of the game.

Arrangements of this theme later appeared in *Castlevania: Harmony of Dissonance* and *Akumajō Dracula: The Arcade*, also serving as background music for the name entry and menu screens, respectively.
