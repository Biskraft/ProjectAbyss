# Stone Whip

right|50px

The **Stone Whip** is a whip with an added Stone element.

In *Castlevania: Circle of the Moon*, this whip causes enemies that would otherwise die to become petrified briefly, which can then be used as platforms.

The **Medusa Whip** is a whip weapon which can be wielded by Jonathan Morris in *Castlevania: Portrait of Ruin*. As it name suggests, it has a chance of petrifying enemies that it strikes, and is a common drop from the Medusa Heads. Aside from its Stone attribute, it also provides +30 to ATK.
