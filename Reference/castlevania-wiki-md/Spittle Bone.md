# Spittle Bone

- Spittle Bone** is an enemy in the *Castlevania* series. It is a four-legged skeleton that crawls on walls.

## Description
A four-legged crawling skeleton that incessantly moves along the surface it is affixed to, making loops endlessly unless destroyed. In some games it drips poison, while in others it breathes flames in a steady stream outward. 

The Vice Beetle is a similar enemy that breathes out clouds of poison.
