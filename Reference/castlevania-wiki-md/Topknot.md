# Topknot

The **Topknot** is a head protection in *Castlevania: Curse of Darkness*. It can be created by combining Nodachi with a Tengu Mask. It only gives a DEF boost of +1 and is basically just something that needs to be created before evolving a Stone Mask into a Masakado's Helm. The Topknot can evolve into a Masakado's Helm by adding Adamantine.
