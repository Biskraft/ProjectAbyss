# Scissor Rod

right|200px**Scissor Rod** is a Mage-Type Innocent Devil found in *Castlevania: Curse of Darkness*. It's a level 2 Innocent Devil and can learn Floating B and Sorcery Flame. It can be obtained from a Wood Rod Mage, using red or yellow crystals.
