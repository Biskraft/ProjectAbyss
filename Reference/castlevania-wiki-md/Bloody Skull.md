# Bloody Skull

right|250px

The **Bloody Skull** is a special item in *Castlevania: Lament of Innocence* that only appears in Joachim Mode. It can clear all the enemies on the screen on pickup.

This item has the same function as the Rosario in the same game.


## Item Data


## Trivia
- It uses the same model as the Crystal Skull relic, but has different color.
- It has the same function that the Rosary, but since it's only available to a malevolent character (Joachim Armster), the skull is probably used instead to imply evil.
