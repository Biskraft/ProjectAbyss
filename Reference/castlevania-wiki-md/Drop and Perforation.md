# Drop and Perforation

*(This article is a stub)***Drop and Perforation** is a skill for the Void Sword and Crissaegrim. Dive with the Void Sword/Crissaegrim to generate an explosion capable of elevating enemies from the ground. Useful to buy some time when surrounded.
