# Crystal Rod

right|200px

  - Crystal Rod** is a Mage-Type Innocent Devil found in *Castlevania: Curse of Darkness*. It's a Level 4 Innocent Devil and can learn Shield and Meteo. It can be obtained from a Goat Head Innocent Devil by using any type of Evo Crystals except white, or from an Ogre Rod by using blue, yellow or white crystals.
