# Queen of Hearts

right|50px

The **Queen of Hearts** is a unique helmet which reduces Heart consumption by half in *Castlevania: Order of Ecclesia*. Offensively, the Queen of Hearts has the best statistics for any head-gear in the game (ATK +10, DEF +12, STR 7+, INT +7, LCK +7), although if Shanoa wants to be more specialized in the favor of Luck, she might be able to get by with a Ribbon. It's normally gained as a reward for defeating Hard Mode in the game; however, it can also be unlocked by connecting the Nintendo DS to the Wii with the game *Castlevania Judgment*.
