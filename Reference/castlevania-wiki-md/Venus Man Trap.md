# Venus Man Trap

The **Venus Man Trap** is an enemy in the Saturn version of *Castlevania: Symphony of the Night*. It is a giant venus fly trap that preys not on flies but humans. 

## Description
A demonically overgrown venus flytrap that will consume any (human) intruders by eating them up whole and using the energy to create offspring.
