# Monster Tome

The **Monster Tome** is a relic in *Castlevania: Harmony of Dissonance*. It contains information about the enemies Juste has battled. A new entry will be added to the "Encyclopedia" for each new enemy he has defeated.

For a list of all Encyclopedia entries and where they can be obtained, see Harmony of Dissonance Bestiary.
