# Stinger

right|250px

  - Stinger** is a Guardian Soul in *Castlevania: Dawn of Sorrow*. It can be obtaining by gaining dominance over a Manticore's soul. It allows Soma to grow a tail with which to attack enemies, hitting in conjunction with his current weapon.


## Item Data


## Notes
- It must be noted that, unlike the Manticore (from which this ability is obtained), this tail is not poisonous. For that effect, see Poison Stinger.
