# Bestiary

right|350px

  - Bestiary** is a skill in *Castlevania: Curse of Darkness*. It contains information about the monsters that Hector has battled. A new entry will appear in the bestiary for each new monster that Hector faces.

For a list of all bestiary entries and where they can be obtained, see Curse of Darkness Bestiary.
