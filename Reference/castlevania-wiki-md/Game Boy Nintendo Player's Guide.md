# Game Boy Nintendo Player's Guide

The companion to the NES Game Atlas, this time covering Game Boy games. It covers *Castlevania: The Adventure* and contains original artwork for many of its enemies. Being made before the release of *Castlevania II: Belmont's Revenge*, it indicates that the hero of this game is Simon Belmont, not Christopher Belmont.
