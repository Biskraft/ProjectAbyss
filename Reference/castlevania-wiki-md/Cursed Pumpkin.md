# Cursed Pumpkin

right|200px

  - Cursed Pumpkin** is a Pumpkin-Type Innocent Devil found in *Castlevania: Curse of Darkness*. It's a level 3 Innocent Devil and can only learn Pose. It can be obtained from a Bloody using Death's Scythe.

## Trivia
