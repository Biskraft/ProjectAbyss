# Castlevania (mobile)

Konami released three versions of the original *Castlevania* in Japan, specially formatted for play on mobile phones. The second version was also released in Europe and North America.

## Related products
  - Castlevania* — The original version of this game.
  - Castlevania: Order of Shadows* — An original mobile phone game created by the people who ported this game to the United States.
