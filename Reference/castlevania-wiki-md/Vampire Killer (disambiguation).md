# Vampire Killer (disambiguation)

- Vampire Killer** may refer to:
    - 1986** — Vampire Killer, the famous theme first introduced in the game *Castlevania*.
    - 1986** — Vampire Killer, the 1986 MSX2 game.
    - 1994** — Vampire Killer, the Japanese name for the Sega Mega Drive game *Castlevania: Bloodlines*.
    - 1994** — Vampire Killer, the mysterious whip owned by the Belmont Clan. Its name is first introduced in *Castlevania: Bloodlines*.
    - 2002** — Vampire Killer, a vampire hunter skateboarder in the game *Evolution Skateboarding*.
    - 2010** — Vampire Killer, the original Combat Cross, used by Gabriel Belmont in *Castlevania: Lords of Shadow*.
    - 2018** — Vampire Killer, the debut trailer for Simon and Richter Belmont in *Super Smash Bros. Ultimate*.
