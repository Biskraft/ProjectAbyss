# True Story Mode

- True Story Mode** is an alternate mode in *Castlevania Judgment*. It is only unlocked when the player beats the whole normal Story Mode with every character, excluding Aeon. The differences are a new character selection screen, which now features the Time Reaper, a second battle against Aeon in the Clock Tower, and a final boss battle against the Time Reaper in the Dimensional Rift. Completing this mode for the first time unlocks that character's respective artwork and theme song in the gallery. Beating this mode in Brutal Mode will unlock that respective character's Mask of the Time Reaper.
