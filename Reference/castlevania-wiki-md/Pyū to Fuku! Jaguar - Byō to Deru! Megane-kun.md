# Pyū to Fuku! Jaguar: Byō to Deru! Megane-kun

is a game developed and published by Konami based on the *Pyū to Fuku! Jaguar* manga by SHONEN JUMP. It was released for the Game Boy Advance handheld system on April 29, 2004 only in Japan.

## Overview
The game is conformed by many different game types, like shooters, side scrollers, overhead view, etc. One of the stages is Dracula's Castle and plays like *Castlevania*.
