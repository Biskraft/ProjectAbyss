# DEF Boost

right|73px

  - DEF Boost** is a stat boost spell or ability that provides a temporary boost in DEF by a constant amount that may consume some sort of resource. A related boost is the CON Boost.

In *Castlevania: Lament of Innocence*, the **White Bishop** is a relic that raises Leon's DEF temporarily by 30, and can be purchased from Rinaldo's shop for $3,000 each.
