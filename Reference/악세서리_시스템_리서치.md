# 악세서리/장신구 시스템 리서치

> ECHORIS 악세서리 슬롯 설계를 위한 레퍼런스 분석
> Sigil x2 (소형 Builder 문양 반지) + Seal x1 (대형 권위 아뮬렛)

## 1. Diablo 시리즈 (Ring x2 + Amulet x1)

### 총 수량
- **Diablo 2:** Unique Ring 59개, Unique Amulet 75개 (총 134개)
- **Diablo 4:** 클래스별 Malignant Ring 5종 + 다수의 Unique Ring/Amulet (지속 추가)

### 효과 카테고리
| 카테고리 | 예시 |
|----------|------|
| 스탯 부스트 | All Stats +5, All Resistances +30 (Mara's Kaleidoscope) |
| 스킬 부스트 | +2 To All Skills |
| 원소 면역/흡수 | Cold Absorb (Raven Frost), Cannot Be Frozen |
| 리소스 관리 | +1 Mana per Kill, Life Leech |
| 조건부 트리거 | 적 처치 시 소환, 특정 HP 이하 시 버프 |
| 클래스 전용 메커닉 | Malignant Ring (D4): 클래스별 고유 능력 부여 |

### 빌드 다양성 창출 방식
- **슬롯 제한(Ring 2 + Amulet 1):** 총 3개의 악세서리 슬롯으로 최소한의 선택 강제
- **Cannot Be Frozen** 같은 유틸리티 vs 순수 DPS 간 트레이드오프
- D4에서는 클래스별 전용 Ring으로 빌드 정체성 강화
- Set Item과의 시너지로 조합 선택지 확대

### 설계 인사이트
- Ring은 범용적 스탯 + 유틸리티, Amulet은 강력한 스킬/고유 효과 담당
- "Stone of Jordan" 같은 아이코닉 아이템이 경제/메타를 정의함

## 2. Path of Exile (Ring x2 + Amulet x1 + Belt x1)

### 총 수량
- Unique Ring: **110개**
- Unique Amulet: **85개**
- (PoE2 Early Access: Ring 24개, Amulet 21개)

### 효과 카테고리
| 카테고리 | 예시 |
|----------|------|
| 메커닉 변환 | Nimis (Projectile 방향 무작위화 + 데미지 증폭) |
| 원소 상호작용 | Polaric Devastation (Fire/Cold 전환 시너지) |
| 상태이상 확장 | Chill된 적을 Shatter 가능하게 변경 |
| 저항 보완 | Resistance gap 메꾸기 |
| DPS 극대화 | Crit Multi, Attack Speed |
| 방어/유지력 | Life Gain on Hit, Energy Shield |

### 빌드 다양성 창출 방식
- **"어떤 니치 빌드든 반드시 맞는 Unique Ring이 존재한다"** 는 설계 철학
- Ring 하나가 빌드의 핵심 메커닉을 활성화하는 "Build-Enabling" 디자인
- 일반 Ring의 Crafting + Unique Ring의 고유 효과 간 선택
- Thief's Torment: Ring 1개만 장착 가능하지만 강력한 효과 (슬롯 트레이드오프)

### 설계 인사이트
- **Build-Enabling Unique:** 악세서리 하나가 전체 빌드 방향을 결정하는 설계가 핵심
- 일반 아이템 Crafting과 Unique의 역할 분리가 명확함
- 슬롯이 적을수록 각 슬롯의 선택 무게가 증가

## 3. Elden Ring - Talisman (4 슬롯)

### 총 수량
- 기본 게임: **115개**
- Shadow of the Erdtree DLC 추가: **39개**
- **총 154개**

### 효과 카테고리
| 카테고리 | 예시 |
|----------|------|
| 스탯 부스트 | Erdtree's Favor (HP/Stamina/Equip Load 동시 증가) |
| 피해 증폭 | 연속 공격 시 데미지 증가, 크리티컬 후 공격력 상승 |
| 피해 경감 | Dragoncrest Greatshield (물리 데미지 20% 감소) |
| 원소 방어 | Pearldrake (비물리 데미지 경감) |
| 스태미나 회복 | Green Turtle Talisman (회복 속도 +8) |
| 조건부 트리거 | Ritual Sword (HP 최대 시 공격력 10% 증가) |
| 상태 내성 | Immunity, Robustness, Focus 관련 |
| 무기/마법 특화 | 주문, 기도, 활, 근접 무기별 전용 Talisman |
| 기동성 | 백스텝/구르기 강화 |

### 빌드 다양성 창출 방식
- **4슬롯 시스템:** 충분한 조합 공간이면서도 모든 것을 장착할 수 없는 제한
- 공격적 Talisman vs 방어적 Talisman 간 트레이드오프
- 보스/상황별 Talisman 교체 전략 (원소 방어 Talisman 등)
- +1/+2 변형으로 NG+에서도 성장감 유지

### 설계 인사이트
- 154개라는 대량의 Talisman이 다양한 빌드 조합을 가능하게 함
- "조건부 트리거" 카테고리가 스킬 기반 플레이를 보상하는 핵심 장치
- 범용 vs 특화의 축이 명확 (Erdtree's Favor vs Ritual Sword)

## 4. Dark Souls 3 - Ring (4 슬롯)

### 총 수량
- **107개** (기본 + NG+의 +1/+2/+3 변형 포함)
- 고유 효과 기준 약 40종

### 효과 카테고리
| 카테고리 | 예시 |
|----------|------|
| HP/방어 | Life Ring (HP +7%), Ring of Steel Protection |
| 스태미나/장비 하중 | Havel's Ring (장비 하중 +15%) |
| HP 재생 | Sun Princess Ring (점진적 HP 회복) |
| 스탯 부스트 | Prisoner's Chain (VIG/END/VIT +5, 피해 +10%) |
| 소울 획득 | Silver Serpent Ring (+10% Souls) |
| 주문 슬롯 | Darkmoon Ring (추가 Attunement 슬롯) |
| 전투 유지력 | Pontiff's Left Eye (연속 공격 시 HP 회복) |
| 스텔스/유틸리티 | Obscuring Ring (원거리에서 투명) |

### 빌드 다양성 창출 방식
- **리스크-리워드 Ring:** Prisoner's Chain (스탯 +5 대신 피해 +10%)
- NG+에서 강화된 버전을 획득하여 성장감 유지
- PvP와 PvE에서 다른 Ring 선택 (Obscuring Ring = PvP 전용)
- 4슬롯이므로 "핵심 2개 + 상황 대응 2개" 구성이 가능

### 설계 인사이트
- **리스크-리워드 트레이드오프:** 강력한 효과에는 반드시 페널티 부여
- +1/+2/+3 변형은 "같은 효과의 점진적 강화"로 성장감 제공
- 상황별 교체가 전략적 깊이를 추가

## 5. Hollow Knight - Charm (45개, 11 Notch)

### 총 수량
- **45개** Charm (40개만 동시 보유 가능, 5개는 상호 교체)

### Notch 시스템 (용량 관리)
- 시작: 3 Notch
- 최대: **11 Notch** (게임 진행으로 8개 추가 획득)
- Charm별 비용: **1~4 Notch**
- **Overcharming:** Notch 초과 장착 가능하나 받는 피해 2배

### 효과 카테고리
| 카테고리 | Notch 비용 | 예시 |
|----------|-----------|------|
| 전투 강화 | 2~3 | Quick Slash (공격 속도), Unbreakable Strength (데미지) |
| 마법 강화 | 3 | Shaman Stone (주문 데미지 증가) |
| 범위/사거리 | 2~3 | Mark of Pride (네일 사거리 증가) |
| 방어/회복 | 1~3 | Quick Focus (회복 속도), Stalwart Shell (무적 프레임) |
| SOUL 관리 | 1~2 | Grubsong (피격 시 SOUL 획득), Soul Eater |
| 탐험 유틸리티 | 1~2 | Wayward Compass (지도 표시), Sprintmaster (이동 속도) |
| 변신/특수 | 2~3 | Shape of Unn (회복 중 이동 가능) |

### 시너지 시스템
- **Charm Synergy:** 특정 조합 시 숨겨진 효과 발동
  - Spore Shroud + Defender's Crest = 독성 구름
  - Shape of Unn + Quick Focus = 회복 중 빠른 이동
- 전투/탐험/보스별 Charm 세트 교체 전략

### 빌드 다양성 창출 방식
- **Notch 비용:** 강력한 Charm은 비싸서 조합 제한 (3 Notch Charm 3개 = 9/11)
- 저비용 Charm 여러 개 vs 고비용 Charm 소수의 선택
- Overcharming이라는 극단적 리스크-리워드 옵션
- Bench에서만 교체 가능 = 상황 대응이 아닌 사전 계획 강제

### 설계 인사이트
- **가변 비용(Notch) 시스템이 고정 슬롯보다 풍부한 조합 공간 제공**
- 시너지 발견의 재미가 탐험적 플레이를 유도
- 교체 제한(Bench)으로 "빌드 커밋"을 강제하되, 완전 고정은 아님
- ECHORIS에 적용 시: Sigil/Seal에 "무게(Weight)" 개념 도입 가능성

## 7. Dead Cells - Mutation (3 슬롯)

### 총 수량
- **56개** Mutation
- Brutality 12 / Tactics 12 / Survival 12 / Colorless 20

### 슬롯 관리
- 레벨 클리어마다 1개씩 선택, **최대 3개** 장착
- 스탯 색상(Brutality/Tactics/Survival)에 따라 효과 스케일링

### 효과 카테고리
| 카테고리 | 색상 | 예시 |
|----------|------|------|
| 피해 증폭 | Brutality | 연속 킬 시 데미지 증가 |
| 생존/회복 | Survival | Necromancy (적 처치 시 HP 회복) |
| 유틸리티 | Tactics | 함정/소환물 강화 |
| 범용 | Colorless | 모든 빌드에서 사용 가능한 효과 |

### 빌드 다양성 창출 방식
- **색상 시너지:** 같은 색상 스탯에 투자할수록 해당 Mutation 효과 증가
- 3슬롯이라는 극도의 제한으로 매 런마다 다른 선택 강제
- Roguelike 구조상 "이번 런에 얻은 무기"에 맞춰 Mutation 선택

### 설계 인사이트
- **스탯 색상과 Mutation의 시너지** 시스템이 빌드 일관성을 강화
- 적은 슬롯(3개) + 많은 선택지(56개) = 매 런마다 신선한 조합
- ECHORIS의 3슬롯(Sigil 2 + Seal 1)과 구조적으로 유사

## 8. Hades - Keepsake (1 슬롯, 25개)

### 총 수량
- Hades 1: **25개** (2개 시크릿 포함)
- Hades 2: **33개**

### 슬롯 관리
- **1개만 장착 가능**
- Region 간 이동 시 교체 가능 (House Contractor 업그레이드 필요)
- 3단계 강화 시스템 (장착 상태로 전투하면 자동 레벨업)

### 효과 카테고리
| 카테고리 | 예시 |
|----------|------|
| HP 관련 | Old Spiked Collar (최대 HP +25/38/50) |
| 부활/보험 | Lucky Tooth (HP 0 시 50/75/100 HP로 부활, 1회) |
| 무적/방어 | Broken Spearpoint (피격 후 1~1.5초 무적, 7초 쿨다운) |
| Boon 유도 | 특정 신의 Boon을 첫 방에서 보장 |
| 보스 특화 | 보스전 피해 증가 |
| 유틸리티 | 상점 할인, 추가 선택지 |

### 빌드 다양성 창출 방식
- 1슬롯이므로 **"이번 런의 방향성"을 선언하는 역할**
- Boon 유도 Keepsake = 빌드 계획 도구
- Region 간 교체로 "초반 계획 + 후반 실행" 전략
- NPC 관계 시스템과 연동 (Nectar 선물로 획득)

### 설계 인사이트
- **악세서리 = 빌드 방향 선언 도구**라는 역할이 명확
- 내러티브(NPC 관계)와 게임플레이(효과)가 연결된 우아한 설계
- 강화 시스템이 "사용할수록 강해진다"는 성장 판타지 제공

## 9. Monster Hunter - Decoration + Charm

### 시스템 구조
- **Charm:** 1개만 장착 가능, 스킬 포인트 제공, 제작/강화
- **Decoration:** 무기/방어구/Charm의 빈 슬롯에 삽입
- **Decoration 레벨:** 1~4 (높은 레벨 = 강력한 스킬)
- 장비당 최대 3개의 Decoration 슬롯

### 효과 카테고리
| 카테고리 | 예시 |
|----------|------|
| 공격 스킬 | Attack Boost, Critical Eye, Weakness Exploit |
| 방어 스킬 | Divine Blessing, Health Boost |
| 원소 공격/내성 | Fire Attack, Thunder Resistance |
| 유틸리티 | Speed Eating, Free Meal, Item Prolonger |
| 무기 특화 | Focus (차지 속도), Artillery (포격 강화) |

### 빌드 다양성 창출 방식
- **슬롯 레벨 시스템:** 큰 슬롯에 작은 Deco는 넣을 수 있지만 반대는 불가
- Charm + Decoration + 방어구 스킬의 퍼즐식 조합
- "마지막 스킬 포인트를 어디서 채울 것인가"의 최적화 게임
- 랜덤 드롭 Decoration이 파밍 동기 부여

### 설계 인사이트
- **슬롯 크기(레벨) 시스템:** 물리적 공간 제약이 조합의 깊이를 추가
- "퍼즐식 빌딩": 모든 장비의 슬롯을 합산하여 원하는 스킬 세트를 맞추는 게임
- ECHORIS의 기억 단편 슬롯 시스템과 유사한 철학

## 10. Terraria - Accessory (5~7 슬롯)

### 총 수량
- **약 512개** (Vanity, Music Box, Golf Ball 포함)
- 실질 전투/이동 악세서리: 약 200~250개

### 슬롯 관리
- Classic: **5 슬롯**
- Expert Mode: **6 슬롯** (Demon Heart 소비)
- 3개 Loadout으로 상황별 전환
- **중복 장착 불가**

### 효과 카테고리
| 카테고리 | 예시 |
|----------|------|
| 이동 | Wings (비행), Hermes Boots (달리기), Shield of Cthulhu (대시) |
| 전투 | Warrior/Ranger/Mage/Summoner Emblem (클래스 데미지 +15%) |
| 방어/Shield | Ankh Shield (넉백 면역 + 상태이상 면역) |
| 상태이상 면역 | 개별 면역 악세서리 조합 -> Ankh Shield |
| 정보 | 시계, 나침반, 기상계 등 (슬롯 미사용) |
| 낚시 | Tackle Box, Angler Earring |
| 하이브리드 | Master Ninja Gear (대시 + 벽타기 + 회피) |

### Tinkerer's Workshop (조합 시스템)
- 여러 악세서리를 **합성하여 상위 악세서리 제작**
- 예: Obsidian Shield + Ankh Charm = Ankh Shield
- 조합 트리가 매우 깊음 (4~5단계 재료 체인)

### 빌드 다양성 창출 방식
- **조합 트리:** 여러 약한 악세서리를 합쳐 강력한 하나로 (슬롯 절약)
- 이동/전투/방어 간 슬롯 배분 전략
- Expert Mode의 추가 슬롯이 보상으로 기능
- Modifier(접두사)로 같은 악세서리도 다른 보너스 획득

### 설계 인사이트
- **조합(Crafting) 시스템으로 슬롯 효율 최적화 게임 생성**
- "이동 능력"이 악세서리로 제공됨 = 탐험 진행과 직결
- 정보 악세서리가 슬롯을 차지하지 않는 설계 = 유틸리티와 전투의 분리
- ECHORIS 적용: 기억 단편 조합 + Sigil/Seal 합성 가능성

## 11. Disgaea - Accessory (장비 슬롯)

### 시스템 구조
- 장비 슬롯: 무기 + 방어구 3개 (일반 장비 중 악세서리 선택)
- 모든 장비가 **Item World(아이템계)** 진입 가능

### 악세서리 종류
| 종류 | 효과 |
|------|------|
| Armor | DEF/RES 증가 |
| Belt | 특정 스탯 부스트 |
| Shoes | 이동력(Move) + 점프력(Jump) 증가 |
| Glasses | HIT/SPD 증가 |
| Weight | Max SP +25%, 단 HP 25%로 제한 (사망 시 자동 부활하나 소멸) |
| Emblem | 다양한 부가 효과 |

### Disgaea 7 악세서리 특수 효과
| 효과 | 수치 |
|------|------|
| Defense Power | 방어 스탯 +3% |
| Attack Power | 공격 스탯 +2% |
| Movement Up | Move +1, Jump +2 |
| SP Up | Max SP +5% |
| Critical | 크리티컬률 +1% |
| HP Up | Max HP +5% |

### 빌드 다양성 창출 방식
- **아이템계(Item World):** 모든 악세서리를 무한히 강화 가능
- 기억 단편(Memory Shard) 배치로 악세서리 커스터마이징
- Weight의 극단적 리스크-리워드 (HP 25% 제한 대신 SP 보너스 + 부활)
- 같은 종류라도 레어리티/강화 정도에 따라 완전히 다른 성능

### 설계 인사이트
- **악세서리 자체가 야리코미 대상** = 강화할수록 애착 형성
- Weight는 "극단적 트레이드오프" 아이템의 좋은 예시
- ECHORIS의 핵심 스파이크와 직접 연결: Sigil/Seal도 아이템계 진입 대상

## 12. Final Fantasy 시리즈 - Accessory (1~2 슬롯)

### 시리즈 전통 악세서리 카테고리
| 카테고리 | 대표 아이템 | 효과 |
|----------|------------|------|
| 완전 면역 | Ribbon | 모든 상태이상 면역 (시리즈 전통) |
| 즉사 방지 | Safety Bit | 즉사/석화/좀비 면역 |
| 스탯 부스트 | Power Wrist, Earring | STR/MAG 증가 |
| 원소 강화/흡수 | Fire Ring, Ice Ring | 원소 피해 흡수/강화 |
| 상태 부여 | Fury Ring | 전투 개시 시 Berserk (공격력 증가, 조작 불가) |
| 드롭/경험치 | Moogle's Amulet | 아이템 드롭률 증가 |
| 회복 강화 | Healing Carcanet | 치유 효과 증가 |

### 빌드 다양성 창출 방식
- **Ribbon의 존재:** "최강 방어 악세서리"가 명확하여 그 외 슬롯에서 선택 강제
- 보스별 상태이상 대응 악세서리 교체
- FF Tactics 등에서는 악세서리가 이동/점프/스탯에 영향

### 설계 인사이트
- **"Ribbon 문제":** 하나의 아이템이 너무 강력하면 선택의 다양성이 사라짐
- 상태이상 면역을 개별 악세서리로 분산시키는 것이 다양성에 유리
- ECHORIS에서는 Seal(대형)에 강력한 단일 효과, Sigil(소형)에 분산 효과 배분 가능

## 종합 분석: 효과 카테고리 대분류

모든 게임을 종합했을 때, 악세서리 효과는 다음 **8대 카테고리**로 분류됩니다:

| # | 카테고리 | 설명 | 대표 게임 예시 |
|---|----------|------|----------------|
| 1 | **스탯 부스트** | 순수 수치 증가 (ATK/DEF/HP 등) | 모든 게임 |
| 2 | **조건부 트리거** | "X일 때 Y 발동" 패턴 | Elden Ring Ritual Sword, DS3 Pontiff's Eye |
| 3 | **메커닉 변환** | 게임 규칙 자체를 변경 | PoE Nimis, HK Shape of Unn |
| 4 | **면역/내성** | 상태이상/원소 방어 | FF Ribbon, Terraria Ankh Shield |
| 5 | **리소스 관리** | HP/MP/Soul 회복/효율 | HK Grubsong, Hades Lucky Tooth |
| 6 | **이동/탐험** | 이동 능력 확장 | Terraria Wings, Disgaea Shoes |
| 7 | **유틸리티** | 드롭률/경험치/정보 등 비전투 | MH Free Meal, Terraria 정보 악세서리 |
| 8 | **리스크-리워드** | 강력한 효과 + 페널티 | Disgaea Weight, HK Overcharming, DS3 Prisoner's Chain |

## 종합 분석: 빌드 다양성 핵심 패턴

### 패턴 1: Build-Enabling (빌드 정의형)
- **PoE, Diablo:** 하나의 악세서리가 전체 빌드 방향을 결정
- 적용: Seal(대형)에 Build-Enabling급 효과 부여

### 패턴 2: Puzzle Fitting (퍼즐 맞추기형)
- **Monster Hunter, Terraria:** 여러 슬롯의 조합으로 원하는 스킬 세트 완성
- 적용: 기억 단편 슬롯과 Sigil/Seal의 이중 레이어

### 패턴 3: Risk-Reward Trade-off (리스크-리워드형)
- **Dark Souls, Disgaea, Hollow Knight:** 강력한 효과에 페널티 부여
- 적용: 고위 Seal에 강력한 효과 + 디메리트 부여

### 패턴 4: Contextual Swap (상황 교체형)
- **Elden Ring, Hades:** 보스/지역에 따라 악세서리를 교체하는 전략
- 적용: 아이템계 지층별 유리한 Sigil/Seal이 다르도록 설계

### 패턴 5: Capacity Budget (용량 예산형)
- **Hollow Knight:** 가변 비용으로 "강력하지만 비싼" vs "약하지만 많이 장착" 선택
- 적용: Sigil에 Weight 개념 도입 시 고려

### 패턴 6: Synergy Discovery (시너지 발견형)
- **Hollow Knight, Terraria:** 특정 조합 시 숨겨진 효과 발동
- 적용: Sigil 2개의 조합 보너스, 또는 Sigil+Seal 세트 효과

## ECHORIS 적용 제안

### 현재 구조: Sigil x2 + Seal x1

| 슬롯 | 역할 제안 | 설계 방향 |
|------|-----------|-----------|
| **Sigil (소형, 2개)** | 스탯 부스트 + 조건부 트리거 + 유틸리티 | 범용적, 조합 자유도 높음, Notch 비용 가능 |
| **Seal (대형, 1개)** | Build-Enabling + 메커닉 변환 + 리스크-리워드 | 강력하고 독보적, 플레이스타일 정의 |

### 핵심 설계 원칙 (리서치 기반)

1. **Seal은 Build-Enabling이어야 한다** (PoE/Diablo의 핵심 Unique처럼)
   - 예: "공중 공격 데미지 2배, 지상 공격 데미지 50% 감소"
   - 예: "피격 시 기억 단편 하나가 랜덤 발동"

2. **Sigil은 조합의 재미를 제공해야 한다** (Hollow Knight/Terraria처럼)
   - 같은 종류 2개 장착 시 추가 효과 (세트 보너스)
   - 서로 다른 Sigil 조합 시 시너지 발동

3. **아이템계 연동이 스파이크를 강화해야 한다** (Disgaea처럼)
   - Sigil/Seal 모두 아이템계 진입 가능
   - 아이템계에서 Sigil/Seal의 기억 단편 교체/강화

4. **상황별 교체 전략이 존재해야 한다** (Elden Ring/Hades처럼)
   - 특정 지층에서 유리한 Seal이 다름
   - 보스전 vs 탐험 시 다른 Sigil 세트

5. **리스크-리워드 옵션이 야리코미를 유도해야 한다** (DS3/Disgaea처럼)
   - 디메리트가 있는 초강력 Seal
   - 아이템계 강화로 디메리트 제거 가능 (야리코미 동기)

### 수량 가이드라인 (레퍼런스 기반)

| 항목 | 추천 수량 | 근거 |
|------|-----------|------|
| Sigil 총 종류 | 30~50개 | SotN(31) + DS3 고유(40) 참고, 3슬롯 대비 충분한 선택지 |
| Seal 총 종류 | 15~25개 | Hades Keepsake(25) 참고, 각각이 독보적이어야 하므로 적정 수 |
| 효과 카테고리 배분 | 8대 카테고리 균등 분포 | 어떤 빌드든 선택지가 있도록 |
