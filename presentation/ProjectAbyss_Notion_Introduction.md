> **빌더가 남긴 거대한 수직 구조물을 탐험하고, 획득한 아이템의 기억 속으로 들어가 끝없이 강화하며, 다른 플레이어와 함께 싸우는 횡스크롤 온라인 액션 RPG**

## 핵심 판타지

이 게임은 세 가지 핵심 판타지를 제공합니다. 모든 시스템은 이 세 가지 중 하나 이상에 반드시 정렬됩니다.

### ⚔️ 탐험가

> 능력을 하나씩 얻으며 갈 수 없던 곳을 뚫어 세계의 비밀을 밝혀낸다

- 능력 게이트와 스탯 게이트의 이중 구조
- 비선형 탐험과 재방문 보상
- 핵심 경험: *"저 절벽 위에 뭔가 보이는데 아직 못 간다"* → 능력 획득 → *"드디어!"*

### ⚒️ 장인

> 아이템 속에 들어가 이노센트를 사냥하고, 세상에 하나뿐인 최강 장비를 만든다

- 모든 장비 아이템 내부에 절차적 던전 — **기억의 지층**
- 이노센트를 사냥하여 스탯 보너스 부여
- 핵심 경험: *"이 검의 모든 지층을 클리어하면 Ancient으로 승급할 수 있다"*

### ⚡ 모험가

> 친구와 함께 끝없는 심연의 던전에 도전하고, 위기를 함께 극복한다

- 혼자서도 재미있고, 함께하면 더 재미있는 설계
- URL 링크 하나로 아이템계에 직접 합류
- 핵심 경험: *"혼자선 최심층 보스를 못 잡는다"* → 파티 구성 → 협동 클리어

### The Shaft — 수직 대공동

빌더가 남기고 떠난 거대한 수직 구조물. 아무도 그 끝을 본 적이 없다.

7개의 Tier가 수직으로 이어지며, 아래로 내려갈수록 진실에 가까워진다.

| Tier | 이름 | 분위기 |
| --- | --- | --- |
| Tier 0 | 천공의 정원 | 바람, 빛, 야생 식물 |
| Tier 1 | 상층 거주구 | 빈 집, 녹슨 배관, 먼지 |
| **Tier 2** | **중앙 성채 (시작점)** | **고딕 건축, 촛불, 정적** |
| Tier 3 | 묘지 | 자주빛 안개, 고목, 비석 |
| Tier 4 | 수로 | 침수된 아치, 청록빛 수면 |
| Tier 5 | 연구소 | 산업 잔해, 녹슨 기계 |
| Tier 6 | 빙결 보존소 | 빙벽, 동결된 무언가 |
| ??? | 심연의 구 | 알 수 없음 |

> 하강은 곧 진실에의 접근이다. 각 Tier의 격벽은 세계수의 가지 — 능력 게이트로 봉인되어 있다.

## 핵심 시스템

### 2-Space 분리 모델

게임 세계는 두 개의 공간으로 분리됩니다. 이 분리가 "메트로베니아의 솔로 탐험감"과 "온라인 협동 플레이"를 양립시키는 핵심 해법입니다.

<!-- 🖼️ Notion: 2-Space 다이어그램 또는 고딕 성채 이미지 삽입 -->
<!-- 파일: Bisk_pixel_art_side-scrolling_game_screenshot_gothic_castle_*.png -->

| | 월드 (World) | 아이템계 (Item World) |
| --- | --- | --- |
| **목적** | 탐험, 능력 획득, 스토리 | 아이템 강화, 야리코미 |
| **인원** | 솔로 (1인) | 1~4인 협동 |
| **맵** | 핸드크래프트 + 절차적 혼합 | 완전 절차적 생성 |
| **핵심** | 능력 게이트 + 스탯 게이트 | 기억의 지층 + 이노센트 사냥 |

### 순환 구조 — Core Loop

이 게임의 핵심 루프입니다. 플레이어는 이 순환을 반복하며 점점 더 깊은 곳으로 내려갑니다.

```
월드 탐험 → 아이템 획득 → 아이템계 진입 → 장비 강화 → 스탯 게이트 돌파 → 새로운 층위 탐험 ↩️
```

> **첫 30분의 경험 설계:**
대사 0줄, NPC 0명, UI 힌트 없음.
"막힘 → 강화 → 돌파" 사이클을 2회 반복시켜, 플레이어가 자연스럽게 루프를 체득합니다.

### 아이템계 — 기억의 지층

<!-- 🖼️ Notion: 차원 전환 이미지 삽입 -->
<!-- 파일: Bisk_pixel_art_game_screenshot_moment_of_dimensional_shift_*.png -->

모든 장비 아이템은 내부에 **기억의 지층(Memory Strata)**을 보유합니다. 에코로 모루 위의 아이템을 타격하면, 바닥이 붕괴하며 아이템 내부의 절차적 던전으로 진입합니다.

**레어리티에 따른 지층 구조:**

| 등급 | 색상 | 스탯 배율 | 이노센트 슬롯 | 지층 수 |
| --- | --- | --- | --- | --- |
| Normal | ⬜ 흰색 | x1.0 | 2 | 2 지층 |
| Magic | 🟦 파란 | x1.3 | 3 | 3 지층 |
| Rare | 🟨 노란 | x1.7 | 4 | 3 지층 |
| Legendary | 🟧 주황 | x2.2 | 6 | 4 지층 |
| Ancient | 🟩 초록 | x3.0 | 8 | 4 지층 + 심연 |

**지층 보스 계보:** 아이템 장군 → 아이템 왕 → 아이템 신 → 아이템 대신

### 이노센트 시스템

아이템 내부에 거주하며 보너스 스탯을 부여하는 존재. 야생 상태(50% 효율)로 아이템계에서 등장하며, 사냥하면 복종 상태(100% 효율)가 됩니다.

**4개 분류, 12종:**

| 분류 | 이노센트 | 효과 |
| --- | --- | --- |
| **기본** | Gladiator / Tutor / Dietician | ATK / INT / HP 증가 |
| **행동** | Berserker / Vampire / Ironclad / Ghost / Sprinter / Leech | 전투 중 특수 효과 |
| **원소** | Burner / Freezer / Shocker | 화 / 빙 / 뇌 원소 부여 |
| **파밍** | Statistician / Scavenger | 경험치 / 드롭률 증가 |

> 동종 이노센트는 최대 3개까지 중첩 가능 (선형 누적)
> 분해 시 Remnant Fragment 획득 → 승급 재료로 사용

### 스탯 & 게이트

심플한 3스탯 시스템. 빌드의 정체성이 명확합니다.

| 스탯 | 역할 | 게이트 |
| --- | --- | --- |
| **ATK** | 물리 피해 | ATK 게이트 — 물리 장벽 돌파 |
| **INT** | 원소 피해 + 상태이상 지속시간 | INT 게이트 — 마법 봉인 해제 |
| **HP** | 체력 | — |

**원소:** 화(Burn) / 빙(Freeze) / 뇌(Shock) + 무속성. 적중률 100%, 상태이상 항상 발동.

**무기 5종:**

| 무기 | 특성 | 주 스탯 |
| --- | --- | --- |
| 검 | 균형잡힌 기본기 | ATK |
| 대검 | 느리지만 강력한 일격 | ATK |
| 단검 | 빠른 연타 | ATK |
| 활 | 원거리 견제 | ATK |

## 레퍼런스 분석

이 프로젝트는 체계적인 레퍼런스 분석을 토대로 설계되었습니다.

### 월하의 야상곡 → 탐험 & 전투

능력 게이트 기반 비선형 월드. 고딕 비주얼 아이덴티티. 넉백과 히트스탑이 있는 타격감 있는 근접 전투. 캐슬바니아 위키 7,400+ 문서 분석.

### 디스가이아 → 아이템계 & 야리코미

아이템 내부가 절차적 던전. 이노센트 시스템으로 스탯 보너스. 무한 리플레이 루프. 디스가이아 위키 1,500+ 문서 분석. 아이템계 역기획서 작성 완료.

### 스펠렁키 → 절차적 생성

룸 템플릿 + 청크 시스템. 보장된 크리티컬 패스. 반복해도 놀라운 레벨 디자인. 레벨 생성 역기획서 작성 완료.

**리서치 규모:**

| 자료 | 수량 |
| --- | --- |
| 위키 문서 분석 | 9,000+ |
| 유튜브 트랜스크립트 | 100+ |
| 역기획서 | 3건 |
| 시스템 GDD | 19건 |

## 수익화 전략

### 퍼널: itch.io → Steam

이 게임의 수익화는 **"무료로 입소문, 유료로 수확"** 구조입니다.

```
itch.io (무료 데모) → 커뮤니티 형성 → Steam 위시리스트 → 유료 전환
```

**1단계 — itch.io로 모객**

itch.io에 무료 웹 빌드를 퍼블리싱합니다. 브라우저에서 바로 플레이할 수 있으므로 진입 장벽이 제로입니다. itch.io는 인디 게임 커뮤니티의 허브이며, 독특한 컨셉의 게임이 자연스럽게 확산되는 생태계입니다. "아이템 속에 들어가는 메트로베니아"라는 스파이크는 itch.io 유저들이 가장 좋아하는 종류의 실험입니다.

- 웹 빌드로 설치 없이 즉시 플레이
- devlog를 통한 개발 과정 공유 — 커뮤니티와 함께 성장
- 피드백 수집과 밸런스 데이터 확보
- 자연스러운 바이럴 — itch.io 추천 알고리즘 + SNS 공유

**2단계 — Steam으로 판매**

itch.io에서 검증된 핵심 루프와 형성된 커뮤니티를 기반으로 Steam 얼리 액세스 또는 정식 출시합니다. Steam 버전은 $15~20 프리미엄으로, 웹 버전 대비 확장된 콘텐츠를 제공합니다.

- itch.io 커뮤니티를 Steam 위시리스트로 유도
- Steam 독점: 추가 Tier, 독점 레어리티, 오프라인 모드, 확장 사운드트랙
- 웹 버전은 계속 무료 유지 — 지속적 신규 유저 유입 경로

| 채널 | 모델 | 역할 |
| --- | --- | --- |
| **itch.io** | 무료 (웹 빌드) | 모객 — 진입 장벽 제로, 커뮤니티 형성, 바이럴 엔진 |
| **Steam** | $15~20 프리미엄 | 수익화 — 독점 콘텐츠, 확장 경험 |

> **금지 원칙:** P2W, 가챠, 에너지 시스템, 광고 — 일체 사용하지 않습니다. 플레이어가 지갑이 아닌 실력과 시간으로 성장하는 게임입니다.

## 개발 방법론

### 멀티 에이전트 워크플로우

AI 에이전트를 역할별로 분리하여 운영합니다.

| 역할 | 담당 | 원칙 |
| --- | --- | --- |
| 🟡 **PM** | 계획, 스펙, 정합성 검증 | 코드 작성 금지 |
| 🔵 **DEV** | 코드 구현 | PM 태스크 브리프 기반 |
| 🟢 **RES** | 리서치, 문서 작성 | 독립 조사 |

> 역할 분리가 스코프 크리프를 방지하고, 각 세션의 집중도를 극대화합니다.
